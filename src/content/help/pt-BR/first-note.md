# Criando Sua Primeira Nota

Comece a usar o Classic criando sua primeira nota. É simples e intuitivo.

## Criando uma Nova Nota

### Usando a Barra Lateral

1. Clique no botão **+** na barra lateral
2. Uma nova nota é criada
3. Comece a digitar seu conteúdo

### Usando Atalho de Teclado

| Plataforma | Atalho |
| ---------- | ------ |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Usando a Paleta de Comandos

1. Abra a Paleta de Comandos (`Cmd+P` ou `Ctrl+P`)
2. Digite "Nova Nota"
3. Pressione Enter

## Escrevendo Sua Nota

### Edição Básica

Simplesmente comece a digitar! O Classic salva automaticamente enquanto você escreve.

### Adicionando um Título

A primeira linha da sua nota se torna o título. Use um título para melhores resultados:

```markdown
# Minha Primeira Nota

Este é o conteúdo da minha nota.
```

### Formatando Texto

Use Markdown para formatar seu texto:

- **Negrito**: `**texto**` ou `Cmd/Ctrl+B`
- **Itálico**: `*texto*` ou `Cmd/Ctrl+I`
- **Tachado**: `~~texto~~`
- **Código**: `` `código` ``

## Salvamento

O Classic salva suas notas automaticamente enquanto você digita. Não é necessário salvar manualmente.

### Salvamento Manual

Se necessário, você pode forçar um salvamento:

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## Organizando Sua Nota

### Adicionando Tags

Adicione tags à sua nota para encontrá-la facilmente depois:

1. Digite `#` seguido do nome da tag
2. Ou use o botão de tag na barra de ferramentas
3. Múltiplas tags são suportadas

Exemplo:

```markdown
# Notas de Reunião

Discutido planejamento do Q4 #trabalho #reunioes
```

### Movendo para uma Pasta

1. Clique com o botão direito na nota na barra lateral
2. Selecione "Mover para Pasta"
3. Escolha ou crie uma pasta de destino

### Fixando Notas Importantes

1. Clique com o botão direito na nota
2. Selecione "Fixar no Topo"
3. A nota aparece no topo da sua lista

## Visualize Sua Nota

Veja como sua nota formatada aparece:

- Clique no botão **Visualizar**
- Ou use `Cmd/Ctrl+Shift+P`
- Alterne entre os modos Edição e Visualização

## Dicas

### Títulos de Notas

- Mantenha os títulos descritivos
- Primeira linha = título da nota
- Use H1 (`#`) para clareza

### Salvamento Automático

- Nunca se preocupe em perder trabalho
- As alterações são salvas automaticamente
- Verifique a barra de status para o status do salvamento

### Captura Rápida

Precisa anotar algo rápido?

- Use a Captura Rápida (`Cmd/Ctrl+Shift+N`)
- Cria uma nota instantaneamente
- Ótimo para pensamentos fugidios
