# Temas e Aparência

Personalize a aparência do Classic para combinar com seu estilo e reduzir a fadiga visual.

## Opções de Tema

### Modo Claro

O tema padrão do Classic apresenta:

- Interface limpa e brilhante
- Ideal para ambientes bem iluminados
- Alto contraste para legibilidade

### Modo Escuro

Perfeito para corujas noturnas e trabalho em ambientes com pouca luz:

- Reduz a fadiga visual
- Menor consumo de bateria em telas OLED
- Fácil para os olhos em ambientes escuros

### Sistema (Automático)

Siga o tema do seu sistema operacional:

- Alterna automaticamente baseado nas configurações do SO
- Sem necessidade de troca manual
- Consistente com outros apps

## Alternando Temas

### Via Configurações

1. Abra as Configurações (`Cmd+,` ou `Ctrl+,`)
2. Navegue para **Aparência**
3. Selecione seu tema preferido

### Via Paleta de Comandos

1. Abra a Paleta de Comandos (`Cmd+P` ou `Ctrl+P`)
2. Digite "Tema"
3. Selecione seu tema preferido

### Atalho de Teclado

| Plataforma | Atalho |
| ---------- | ------ |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Cicla através de: Claro → Escuro → Sistema

## Configurações de Fonte

### Fonte do Editor

Personalize a experiência de escrita:

1. Abra as Configurações
2. Vá para **Aparência** → **Editor**
3. Configure as opções de fonte

| Configuração | Opções | Padrão |
| ------------ | ------ | ------ |
| Família da Fonte | Fontes do sistema + personalizadas | UI do Sistema |
| Tamanho da Fonte | 12-24px | 16px |
| Altura da Linha | 1.0-2.5 | 1.75 |
| Espaçamento de Letras | -2 a 4 | 0 |

### Fonte da UI

Ajuste a fonte da interface:

1. Abra as Configurações
2. Vá para **Aparência** → **Interface**
3. Escolha a fonte e o tamanho

### Fonte Monospace

Para blocos de código e conteúdo técnico:

1. Abra as Configurações
2. Vá para **Aparência** → **Código**
3. Selecione a fonte monospace

## Opções de Interface

### Barra Lateral

Personalize o comportamento da barra lateral:

| Configuração | Descrição |
| ------------ | --------- |
| **Ocultar automaticamente** | Ocultar barra lateral quando não estiver em uso |
| **Largura** | Ajustar largura da barra lateral (200-400px) |
| **Mostrar Ícones** | Alternar visibilidade dos ícones |

### Layout do Editor

Escolha sua experiência de edição:

- **Visualização Dividida**: Edite e visualize lado a lado
- **Apenas Visualização**: Veja apenas a saída renderizada
- **Apenas Edição**: Foque no Markdown bruto

### Lista de Notas

Configure como as notas aparecem:

- **Visualização em Lista**: Compacta, uma linha por nota
- **Visualização em Cartões**: Cartões de pré-visualização com trechos
- **Visualização em Árvore**: Estrutura hierárquica de pastas

## Cores de Destaque

### Cores Disponíveis

Escolha sua cor de destaque:

- Azul Clássico (padrão)
- Roxo
- Verde
- Laranja
- Vermelho
- Rosa
- Turquesa

### Mudando a Cor de Destaque

1. Abra as Configurações
2. Vá para **Aparência** → **Cores**
3. Selecione seu destaque preferido

A cor de destaque afeta:

- Links e botões
- Destaques de seleção
- Elementos ativos
- Indicadores de foco

## Acessibilidade

### Modo de Alto Contraste

Para melhor visibilidade:

1. Abra as Configurações
2. Vá para **Aparência** → **Acessibilidade**
3. Habilite **Alto Contraste**

### Movimento Reduzido

Minimize animações:

1. Abra as Configurações
2. Vá para **Aparência** → **Acessibilidade**
3. Habilite **Movimento Reduzido**

### Ajustes de Fonte

- Aumente o tamanho mínimo da fonte
- Use fontes amigáveis para dislexia
- Ajuste taxas de contraste

## CSS Personalizado (Avançado)

Para usuários avançados que querem controle completo:

1. Abra as Configurações
2. Vá para **Aparência** → **Avançado**
3. Clique em **Abrir CSS Personalizado**
4. Adicione suas personalizações

### Exemplos de Personalização

```css
/* Títulos maiores */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Cor de link personalizada */
a { color: #3b82f6; }

/* Espaçamento de linha mais amplo */
.prose p { line-height: 2; }
```

## Sincronização de Tema

Quando você entra (opcional), suas preferências de tema sincronizam entre dispositivos:

- Sincronização automática de tema
- Configurações de fonte sincronizam
- CSS personalizado sincroniza

## Dicas

### Combinando com Seu Fluxo de Trabalho

- Use **Claro** durante o dia, **Escuro** à noite
- O modo **Sistema** lida com isso automaticamente
- Considere sua iluminação ambiente

### Reduzindo Fadiga Visual

- Use o modo Escuro com pouca luz
- Aumente o tamanho da fonte para sessões longas
- Habilite Alto Contraste se necessário
- Faça pausas regulares

### Produtividade

- Aprenda o atalho de alternância de tema
- Configure horários de troca automática
- Use o modo Sistema para consistência
