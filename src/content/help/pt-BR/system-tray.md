# Bandeja do Sistema

O Classic vive na sua bandeja do sistema para acesso rápido. Capture pensamentos instantaneamente sem interromper seu fluxo de trabalho.

## O Que É a Bandeja do Sistema?

A bandeja do sistema (Windows/Linux) ou barra de menu (macOS) é onde o Classic roda em segundo plano, sempre pronto para captura rápida.

### Benefícios

- **Sempre disponível**: A um clique de distância
- **Captura rápida**: Anote pensamentos instantaneamente
- **Execução em segundo plano**: Não precisa manter o app aberto
- **Notificações**: Seja lembrado de tarefas

## Habilitando a Bandeja do Sistema

### macOS

1. Abra o Classic
2. Vá para Configurações → Geral
3. Habilite "Mostrar na Barra de Menu"
4. O ícone do Classic aparece na barra de menu

### Windows

1. Abra o Classic
2. Vá para Configurações → Geral
3. Habilite "Mostrar na Bandeja do Sistema"
4. O ícone do Classic aparece na bandeja do sistema

### Linux

1. Abra o Classic
2. Vá para Configurações → Geral
3. Habilite "Ícone da Bandeja do Sistema"
4. Depende do ambiente de desktop

## Captura Rápida

### Abrindo a Captura Rápida

Clique no ícone do Classic na bandeja do sistema:

- **macOS**: Ícone da barra de menu
- **Windows**: Ícone da bandeja do sistema
- **Linux**: Ícone da bandeja do sistema

### Usando a Captura Rápida

1. Clique no ícone da bandeja do sistema
2. A janela de captura rápida abre
3. Digite sua nota
4. Pressione Enter para salvar
5. A janela fecha automaticamente

### Atalho de Teclado

Abra a Captura Rápida de qualquer lugar:

| Plataforma | Atalho |
|------------|--------|
| **macOS** | `Cmd+Shift+N` (global) |
| **Windows** | `Ctrl+Shift+N` (global) |
| **Linux** | `Ctrl+Shift+N` (global) |

## Menu da Bandeja do Sistema

Clique com o botão direito no ícone da bandeja para opções:

### Opções do Menu

| Opção | Ação |
|-------|------|
| **Nova Nota** | Abrir Captura Rápida |
| **Nova Nota Diária** | Abrir nota diária de hoje |
| **Pesquisar** | Abrir janela de pesquisa |
| **Abrir Classic** | Abrir janela principal |
| **Configurações** | Abrir configurações |
| **Sair** | Fechar Classic |

## Configurações da Bandeja

### Configure nas Configurações

| Configuração | Descrição |
|--------------|-----------|
| **Mostrar Ícone** | Alternar visibilidade do ícone da bandeja |
| **Iniciar Minimizado** | Iniciar Classic apenas na bandeja |
| **Fechar para Bandeja** | Manter rodando quando a janela for fechada |
| **Notificações** | Habilitar/desabilitar notificações |

### Iniciar no Login

Faça o Classic iniciar automaticamente:

1. Abra as Configurações
2. Vá para Geral
3. Habilite "Iniciar no Login"
4. O Classic abre na inicialização

## Notificações

### Habilitar Notificações

1. Abra as Configurações
2. Vá para Notificações
3. Alterne "Habilitar Notificações"

### Tipos de Notificação

- **Lembretes**: Lembretes agendados
- **Nota Diária**: Prompts de nota diária
- **Atualizações**: Atualização do app disponível

## Comportamento da Janela

### Comportamento ao Fechar

Escolha o que acontece quando você fecha a janela:

- **Minimizar para Bandeja**: Manter rodando, ocultar janela
- **Sair**: Fechar o aplicativo
- **Perguntar**: Perguntar cada vez

### Comportamento ao Minimizar

- **Minimizar para Bandeja**: Ocultar na bandeja do sistema
- **Minimizar para Barra de Tarefas**: Minimizar normalmente

## Dicas

### Captura Rápida

Use o atalho global para captura mais rápida:

1. Pressione `Cmd/Ctrl+Shift+N` em qualquer lugar
2. Digite seu pensamento
3. Pressione Enter
4. Continue seu trabalho

### Acesso Rápido

Clique no ícone da bandeja para ver:

- Notas recentes
- Ações rápidas
- Nota diária de hoje

### Execução em Segundo Plano

Mantenha o Classic rodando para:

- Inicialização mais rápida
- Captura rápida
- Notificações
- Backup automático

## Solução de Problemas

### Ícone da Bandeja Faltando

**macOS**

1. Verifique se o ícone está oculto
2. Procure na Central de Controle (macOS Big Sur+)
3. Reinicie o Classic

**Windows**

1. Verifique ícones ocultos (seta na barra de tarefas)
2. Arraste o ícone do Classic para a área visível
3. Reinicie o Classic

**Linux**

1. Verifique o suporte do ambiente de desktop
2. Instale a extensão de bandeja do sistema se necessário
3. Reinicie o Classic

### Atalho Global Não Funciona

- Verifique conflitos com outros apps
- Re-habilite o atalho nas Configurações
- Reinicie o Classic
- Verifique as permissões do SO

### Notificações Não Aparecem

**macOS**

1. Preferências do Sistema → Notificações
2. Encontre o Classic
3. Habilite notificações

**Windows**

1. Configurações → Sistema → Notificações
2. Habilite notificações do Classic

**Linux**

1. Verifique o daemon de notificação
2. Instale libnotify se necessário

### Alto Uso de Memória/CPU

Se o ícone da bandeja usa muitos recursos:

1. Desabilite animações nas Configurações
2. Reduza a frequência de notificações
3. Verifique atualizações do app
