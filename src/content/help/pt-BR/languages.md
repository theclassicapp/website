# Idiomas e Localização

O Classic fala seu idioma. Use o app no seu idioma preferido e escreva notas em qualquer idioma que escolher.

## Idiomas Suportados

A interface do Classic está disponível em múltiplos idiomas:

| Idioma | Código |
|--------|--------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| E mais... | |

## Mudando o Idioma da Interface

### Via Configurações

1. Abra as Configurações (`Cmd+,` ou `Ctrl+,`)
2. Vá para **Geral** → **Idioma**
3. Selecione seu idioma preferido
4. Reinicie o Classic para aplicar

### Via Paleta de Comandos

1. Abra a Paleta de Comandos (`Cmd+P` ou `Ctrl+P`)
2. Digite "Idioma"
3. Selecione seu idioma preferido
4. Reinicie para aplicar as mudanças

## Escrevendo em Múltiplos Idiomas

O Classic suporta escrever notas em qualquer idioma.

### Suporte Unicode

- Suporte Unicode completo
- Idiomas RTL (Direita para Esquerda) suportados
- Caracteres CJK (Chinês, Japonês, Coreano) funcionam perfeitamente
- Suporte a Emoji

### Detecção de Idioma

O Classic pode detectar o idioma das suas notas:

- Detecção automática para verificação ortográfica
- Configuração de idioma por nota
- Suporte a idiomas mistos

## Verificação Ortográfica

### Habilitando Verificação Ortográfica

1. Abra as Configurações
2. Vá para **Editor** → **Verificação Ortográfica**
3. Habilite "Verificação Ortográfica"

### Idiomas de Verificação Ortográfica

1. Abra as Configurações
2. Vá para **Editor** → **Verificação Ortográfica**
3. Clique em "Idiomas"
4. Selecione os idiomas para verificar

### Adicionando Palavras ao Dicionário

1. Clique com o botão direito em uma palavra sinalizada
2. Selecione "Adicionar ao Dicionário"
3. A palavra não será mais sinalizada

## Suporte RTL

Para Árabe, Hebraico e outros idiomas RTL:

### Detecção RTL Automática

O Classic detecta conteúdo RTL automaticamente:

- A direção do texto se ajusta
- O movimento do cursor se adapta
- O alinhamento muda

### Alternância RTL Manual

Force a direção RTL ou LTR:

1. Abra a Paleta de Comandos
2. Digite "RTL" ou "LTR"
3. Selecione sua preferência

## Métodos de Entrada

O Classic funciona com os métodos de entrada do seu sistema:

### Métodos de Entrada CJK

- Funciona com Pinyin, Zhuyin, etc.
- Suporta IME Japonês
- Métodos de entrada Coreanos suportados

### Caracteres Especiais

Insira caracteres especiais facilmente:

- Use o seletor de caracteres do seu sistema
- Copie/cole de mapas de caracteres
- Use entrada Unicode

## Formatos de Data e Hora

As datas se adaptam à sua localidade:

### Opções de Formato

| Configuração | Opções |
|--------------|--------|
| Formato de Data | MM/DD/AAAA, DD/MM/AAAA, AAAA-MM-DD |
| Formato de Hora | 12 horas, 24 horas |
| Primeiro Dia da Semana | Domingo, Segunda, Sábado |

Configure em Configurações → Geral → Data e Hora

## Formatos de Números

Os números são exibidos de acordo com sua localidade:

- **Separador decimal**: `.` ou `,`
- **Separador de milhares**: `,` ou `.` ou espaço
- **Símbolos de moeda**: Baseado na região

## Layouts de Teclado

O Classic funciona com qualquer layout de teclado:

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Layouts personalizados

## Configurações de Localização

### Em Configurações → Geral → Idioma

| Configuração | Descrição |
|--------------|-----------|
| **Idioma da Interface** | Idioma da UI do app |
| **Formato de Data** | Como as datas aparecem |
| **Formato de Hora** | 12 ou 24 horas |
| **Primeiro Dia da Semana** | Dia de início do calendário |
| **Formato de Número** | Separadores decimal/milhar |

## Dicas

### Notas com Idiomas Mistos

Escreva em múltiplos idiomas em uma nota:

```markdown
# Projeto Internacional

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Fontes Específicas por Idioma

Alguns idiomas ficam melhores com fontes específicas:

1. Abra as Configurações
2. Vá para Aparência → Fontes
3. Defina fontes para scripts diferentes

### Alternando Idiomas Rapidamente

Crie um atalho de teclado:

1. Abra Configurações → Atalhos
2. Encontre "Alternar Idioma"
3. Defina seu atalho preferido

## Solução de Problemas

### Idioma Não Disponível

- Verifique atualizações do app
- O idioma pode estar em desenvolvimento
- Contribua com traduções no GitHub

### Caracteres Não Exibindo

- Instale as fontes necessárias
- Verifique as configurações de fonte
- Tente uma fonte diferente

### Verificação Ortográfica Não Funciona

- Verifique se a verificação ortográfica está habilitada
- Adicione o idioma à verificação ortográfica
- Reinicie o Classic

### RTL Não Funciona Corretamente

- Certifique-se de que a detecção RTL está ligada
- Tente a alternância RTL manual
- Verifique problemas de conteúdo misto
