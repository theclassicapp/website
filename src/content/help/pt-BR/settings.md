# Configurações

Personalize o Classic para funcionar exatamente do jeito que você quer. Da aparência ao comportamento do editor, tudo é configurável.

## Abrindo as Configurações

### Atalho de Teclado

| Plataforma | Atalho |
|------------|--------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Via Menu

- **macOS**: Classic → Configurações
- **Windows**: Arquivo → Configurações
- **Linux**: Editar → Configurações

### Via Paleta de Comandos

1. Abra a Paleta de Comandos (`Cmd+P` ou `Ctrl+P`)
2. Digite "Configurações"
3. Pressione Enter

## Categorias de Configurações

### Geral

Configurações básicas do aplicativo:

| Configuração | Descrição |
|--------------|-----------|
| **Idioma** | Idioma da interface |
| **Tema** | Claro, Escuro ou Sistema |
| **Iniciar no Login** | Abrir na inicialização |
| **Verificar Atualizações** | Comportamento de atualização automática |

### Aparência

Personalize a aparência:

| Configuração | Opções |
|--------------|--------|
| **Tema** | Claro / Escuro / Sistema |
| **Cor de Destaque** | Azul, Roxo, Verde, etc. |
| **Fonte do Editor** | Família e tamanho da fonte |
| **Fonte da UI** | Fonte da interface |
| **Fonte Monospace** | Fonte de código |
| **Largura da Barra Lateral** | 200-400px |
| **Altura da Linha** | 1.0-2.5 |

### Editor

Configure o comportamento de escrita:

| Configuração | Descrição |
|--------------|-----------|
| **Verificação Ortográfica** | Habilitar/desabilitar verificação ortográfica |
| **Salvamento Automático** | Intervalo de salvamento automático |
| **Quebra de Linha** | Habilitar/desabilitar quebra de linha |
| **Números de Linha** | Mostrar/ocultar números de linha |
| **Tamanho do Tab** | 2, 4 ou 8 espaços |
| **Auto-correção** | Aspas inteligentes, substituição automática |

### Histórico de Versões

Controle o rastreamento de versões:

| Configuração | Opções | Padrão |
|--------------|--------|--------|
| **Habilitar Histórico** | Ligado/Desligado | Ligado |
| **Período de Retenção** | 30-180 dias | 90 dias |
| **Máximo de Versões** | 50-500 | 100 |

### Backup e Restauração

Configurações de proteção de dados:

| Configuração | Descrição |
|--------------|-----------|
| **Backup Automático** | Habilitar backups automáticos |
| **Frequência de Backup** | 15min / 30min / 1h / 4h |
| **Localização do Backup** | Caminho personalizado |
| **Manter Backups** | 5 / 10 / 20 / 50 |
| **Criptografar Backups** | Proteção por senha |

### Atalhos de Teclado

Visualize e personalize atalhos:

- Veja todos os atalhos disponíveis
- Personalize qualquer atalho
- Reset para padrões
- Importar/exportar atalhos

### Importar e Exportar

| **Relatórios de Falhas** | Enviar relatórios de falhas |

### Notificações

Controle as notificações:

|--------------|-----------|

### Avançado

Configurações para usuários avançados:

| Configuração | Descrição |

## Pesquisando Configurações

Não consegue encontrar uma configuração? Use a barra de pesquisa:

1. Abra as Configurações
2. Digite na caixa de pesquisa
3. Configurações correspondentes aparecem
4. Clique para pular para aquela configuração

## Resetando Configurações

### Resetar Todas as Configurações

Volte para os padrões:

1. Abra as Configurações
2. Role até o final
3. Clique em "Resetar Todas as Configurações"
4. Confirme a ação

### Resetar Categoria

Reset categoria específica:

1. Vá para a categoria
2. Clique em "Resetar para Padrões"
3. Apenas aquela categoria é resetada

## Exportando/Importando Configurações

### Exportar Configurações

Salve sua configuração:

1. Abra as Configurações
2. Clique em "Exportar Configurações"
3. Escolha a localização
4. Salve como arquivo JSON

### Importar Configurações

Carregue uma configuração salva:

1. Abra as Configurações
2. Clique em "Importar Configurações"
3. Selecione o arquivo JSON
4. Confirme a importação

## Localização das Configurações

As configurações são armazenadas localmente:

| Plataforma | Localização |
|------------|-------------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Sincronizar Configurações (Opcional)

Quando conectado, as configurações podem sincronizar entre dispositivos:

- **Sincronizar**: Habilite em Configurações → Sincronização
- **O Que Sincroniza**: Tema, fontes, atalhos
- **O Que Não Sincroniza**: Caminhos locais, chaves de criptografia

## Personalizações Comuns

### Aumentar Tamanho da Fonte

1. Configurações → Aparência
2. Tamanho da Fonte do Editor → Aumentar
3. Mudanças aplicam imediatamente

### Mudar Tema

1. Configurações → Aparência
2. Tema → Selecione Claro/Escuro/Sistema
3. App atualiza instantaneamente

### Habilitar Backup Automático

1. Configurações → Backup e Restauração
2. Habilite "Backup Automático"
3. Defina frequência e localização

### Personalizar Atalhos

1. Configurações → Atalhos de Teclado
2. Encontre a ação
3. Clique no atalho atual
4. Pressione novas teclas
5. Salve

## Dicas

### Acesso Rápido

- Use `Cmd/Ctrl+,` para abrir configurações rapidamente
- Pesquise configurações pelo nome
- Use categorias para navegar

### Exporte Antes de Mudanças

Antes de mudanças importantes, exporte suas configurações:

- Fácil de restaurar se algo der errado
- Compartilhe configuração com outros

### Verifique Novas Configurações

Atualizações podem adicionar novas configurações:

- Verifique após atualizar
- Explore novas opções

## Solução de Problemas

### Configurações Não Salvam

- Verifique as permissões do arquivo
- Verifique o espaço em disco
- Tente resetar e reconfigurar

### Configurações Resetaram Inesperadamente

- Pode indicar corrupção de arquivo
- Exporte configurações regularmente
- Verifique conflitos de apps

### Não Consigo Encontrar uma Configuração

- Use a barra de pesquisa
- Verifique categorias diferentes
- Pode estar nas configurações Avançadas
