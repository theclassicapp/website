# Captura Rápida

Capture ideias instantaneamente sem interromper seu fluxo de trabalho. A Captura Rápida abre uma janela minimalista de anotações de qualquer lugar do seu sistema.

## Atalho Global

Pressione o atalho designado para abrir a Captura Rápida:

| Plataforma | Atalho Padrão |
| ---------- | ------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

Você pode personalizar este atalho nas Configurações.

## Usando a Captura Rápida

1. Pressione o atalho global (em qualquer lugar do seu sistema)
2. Digite sua nota na janela popup
3. Pressione `Enter` para salvar
4. Pressione `Escape` para cancelar

## Recursos

### Interface Minimalista

- Janela pequena e focada
- Suporte a Markdown
- Foco automático na entrada
- Design priorizando teclado

### Ações Rápidas

- `# ` + espaço = título
- `#inbox` = adicionar à Inbox
- `#hoje` = adicionar à nota de hoje
- Tags com `#nomedatag`

### Destino Padrão

As notas da Captura Rápida vão para sua **Inbox** por padrão. Você pode mudar isso:

1. Abra as Configurações
2. Vá para Captura Rápida
3. Selecione o destino padrão

## Dicas

### Fluxo de Trabalho de Captura

1. **Capturar** - Anote a ideia rapidamente
2. **Processar** - Revise a Inbox diariamente
3. **Organizar** - Mova para a pasta/nota apropriada

### Melhores Práticas

- Mantenha breve - uma ideia por captura
- Use tags para contexto
- Não se preocupe com formatação - refine depois
- Revise e processe capturas regularmente

### Integração

A Captura Rápida funciona bem com:

- **Notas Diárias**: Comece cada dia revisando capturas
- **Inbox**: Use como área de espera
- **Tags**: Adicione contexto imediatamente

## Solução de Problemas

### Atalho Não Funciona?

**macOS**
1. Verifique Preferências do Sistema > Teclado > Atalhos
2. Procure por conflitos
3. Reinicie o Classic

**Windows**
1. Execute o Classic como Administrador
2. Verifique atalhos conflitantes
3. Reinicie o Classic

### Janela Não Aparece?

- Verifique se o Classic está rodando
- Verifique o atalho
- Verifique problemas de foco
- Reinicie o Classic

### Capturas Não Salvam?

- Verifique o destino padrão
- Verifique as permissões de escrita
- Verifique o espaço em disco
- Reinicie o Classic
