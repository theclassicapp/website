# Comandos com Barra

Os comandos com barra são uma maneira poderosa de inserir rapidamente formatação e conteúdo especial sem sair do teclado. Digite `/` no editor para abrir a paleta de comandos.

## Comandos Disponíveis

### Títulos

| Comando | Resultado |
| ------- | --------- |
| `/h1` | Título nível 1 |
| `/h2` | Título nível 2 |
| `/h3` | Título nível 3 |
| `/h4` | Título nível 4 |
| `/h5` | Título nível 5 |
| `/h6` | Título nível 6 |

### Formatação de Texto

| Comando | Resultado |
| ------- | --------- |
| `/bold` | **Texto em negrito** |
| `/italic` | *Texto em itálico* |
| `/strikethrough` | ~~Tachado~~ |
| `/code` | Bloco de código inline |
| `/quote` | Citação em bloco |

### Listas

| Comando | Resultado |
| ------- | --------- |
| `/bullet` | • Lista com marcadores |
| `/numbered` | 1. Lista numerada |
| `/todo` | - [ ] Item de tarefa |
| `/check` | - [x] Item concluído |

### Inserir Conteúdo

| Comando | Resultado |
| ------- | --------- |
| `/image` | Inserir imagem de URL ou arquivo |
| `/link` | Inserir hyperlink |
| `/table` | Inserir tabela |
| `/hr` | Linha horizontal |
| `/code` | Bloco de código com seleção de linguagem |

### Blocos Especiais

| Comando | Resultado |
| ------- | --------- |
| `/mermaid` | Bloco de diagrama Mermaid |
| `/katex` | Bloco matemático KaTeX |
| `/callout` | Bloco de destaque/aviso |

## Usando Comandos com Barra

1. Digite `/` no editor
2. Um menu aparecerá com todos os comandos disponíveis
3. Comece a digitar para filtrar comandos
4. Pressione `Enter` ou clique para inserir

## Dicas Pro

- **Acesso rápido**: Digite `/` e comece a digitar o nome do comando
- **Busca fuzzy**: Você não precisa digitar o nome exato do comando - o Classic encontrará a correspondência mais próxima
- **Navegação por teclado**: Use as setas para navegar no menu
- **Comandos recentes**: Comandos usados frequentemente aparecem no topo da lista
