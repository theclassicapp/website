# Paleta de Comandos

A paleta de comandos é sua porta de entrada para cada ação no Classic. Pressione um simples atalho de teclado e acesse qualquer recurso instantaneamente.

## Abrindo a Paleta de Comandos

| Plataforma | Atalho |
| ---------- | ------ |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Como Usar

1. Abra a paleta de comandos (`Cmd+P` ou `Ctrl+P`)
2. Digite para pesquisar um comando
3. Use as setas para navegar
4. Pressione `Enter` para executar

## Categorias de Comandos

### Operações de Arquivo

- **Nova Nota**: Criar uma nova nota
- **Nova Nota Diária**: Criar/abrir a nota diária de hoje
- **Abrir Nota**: Pesquisar e abrir notas existentes
- **Salvar Nota**: Salvar a nota atual
- **Excluir Nota**: Excluir a nota atual
- **Renomear Nota**: Renomear a nota atual

### Navegação

- **Ir para Nota**: Pular para qualquer nota
- **Ir para Nota Diária**: Abrir uma data específica
- **Ir para Tag**: Navegar notas por tag
- **Voltar**: Navegar para trás no histórico
- **Avançar**: Navegar para frente no histórico

### Editor

- **Alternar Visualização**: Mudar entre modos de edição e visualização
- **Alternar Barra Lateral**: Mostrar/ocultar a barra lateral
- **Modo Foco**: Entrar em escrita sem distrações
- **Inserir Data**: Adicionar data/hora atual
- **Inserir Template**: Inserir um template

### Formatação

- **Negrito**: Colocar texto em negrito
- **Itálico**: Colocar texto em itálico
- **Tachado**: Tachar texto
- **Código**: Formatar como código
- **Link**: Inserir um link
- **Título**: Aplicar formato de título

### Organização

- **Adicionar Tag**: Taguear a nota atual
- **Mover para Pasta**: Mover nota para uma pasta
- **Arquivar Nota**: Arquivar nota atual
- **Fixar Nota**: Fixar nota no topo da lista

### Visualização

- **Alternar Tema**: Mudar entre claro/escuro
- **Aumentar Zoom**: Aumentar tamanho da fonte
- **Diminuir Zoom**: Diminuir tamanho da fonte
- **Resetar Zoom**: Voltar ao tamanho padrão

## Dicas

### Acesso Rápido

- Comandos são pesquisados com fuzzy matching
- Comandos recentes aparecem primeiro
- Fixe comandos usados frequentemente

### Navegação por Teclado

- `↑`/`↓` - Navegar resultados
- `Enter` - Executar comando
- `Escape` - Fechar paleta
- `Tab` - Circular através das categorias

### Comandos Personalizados

1. Abra as Configurações
2. Vá para Paleta de Comandos
3. Adicione comandos personalizados
4. Defina atalhos de teclado

## Comandos Frequentemente Usados

Aqui estão alguns comandos que você usará frequentemente:

| Comando | O que faz |
| ------- | --------- |
| `Nova Nota` | Criar nota em branco |
| `Abrir Nota` | Pesquisar todas as notas |
| `Nova Nota Diária` | Diário de hoje |
| `Alternar Visualização` | Visualização vs edição |
| `Adicionar Tag` | Taguear nota atual |
| `Captura Rápida` | Captura global |

## Personalizando Comandos

### Mudando Atalhos

1. Abra as Configurações (`Cmd+,` ou `Ctrl+,`)
2. Navegue para Atalhos de Teclado
3. Encontre o comando
4. Clique no atalho atual
5. Pressione seu novo atalho
6. Salve

### Criando Comandos Personalizados

Usuários avançados podem criar comandos personalizados:

1. Abra as Configurações
2. Vá para Comandos
3. Clique em "Adicionar Comando Personalizado"
4. Defina o nome do comando
5. Adicione a(s) ação(ões)
6. Defina um atalho (opcional)
