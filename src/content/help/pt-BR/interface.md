# Entendendo a Interface

A interface do Classic foi projetada para ser limpa, intuitiva e não atrapalhar seu trabalho. Aqui está um tour pelos principais componentes.

## Layout Principal

O Classic possui um layout de três painéis:

1. **Barra Lateral** (esquerda) - Navegação e organização de notas
2. **Editor** (centro) - Seu espaço de escrita
3. **Visualização/Propriedades** (direita) - Painéis opcionais

## Barra Lateral

A barra lateral contém várias seções importantes:

### Lista de Notas

- **Todas as Notas**: Visualize todas as suas notas em uma lista
- **Notas Diárias**: Acesso rápido ao diário de hoje
- **Favoritos**: Marque notas usadas frequentemente
- **Pastas**: Organize notas em hierarquias

### Navegação

- **Barra de Pesquisa**: Encontre notas instantaneamente
- **Tags**: Navegue por tags
- **Backlinks**: Veja conexões entre notas

### Ações

- **Nova Nota**: Crie uma nova nota
- **Nova Pasta**: Organize com pastas
- **Importar**: Traga conteúdo externo

## Editor

O editor é onde você escreve e formata suas notas.

### Recursos

- **Suporte Markdown**: Sintaxe Markdown completa
- **Visualização em Tempo Real**: Veja o texto formatado enquanto digita
- **Destaque de Sintaxe**: Blocos de código são destacados
- **Comandos com Barra**: Digite `/` para ações rápidas

### Modos do Editor

| Modo | Descrição |
|------|-----------|
| **Apenas Edição** | Veja o Markdown bruto |
| **Apenas Visualização** | Veja a saída renderizada |
| **Visualização Dividida** | Edite e visualize lado a lado |

### Barra de Ferramentas

Ações rápidas de formatação:

- Negrito, Itálico, Tachado
- Títulos (H1-H6)
- Listas (com marcadores, numeradas, tarefas)
- Links e imagens
- Blocos de código
- Tabelas

## Barra de Status

Na parte inferior do editor:

- **Contagem de Palavras/Caracteres**: Acompanhe sua escrita
- **Último Salvamento**: Quando a nota foi salva pela última vez
- **Caminho da Nota**: Onde a nota está localizada
- **Status de Sincronização**: Se estiver usando sincronização opcional

## Menus de Contexto

Clique com o botão direito para ações contextuais:

### Menu de Contexto da Nota

- Renomear
- Mover para pasta
- Duplicar
- Excluir
- Exportar
- Copiar link

### Menu de Contexto do Editor

- Recortar, Copiar, Colar
- Opções de formatação
- Inserir link/imagem
- Localizar e substituir

## Navegação por Teclado

Navegue pela interface com seu teclado:

| Ação | Atalho |
|------|--------|
| Alternar Barra Lateral | `Cmd/Ctrl + M` |
| Focar na Pesquisa | `Cmd/Ctrl + F` |
| Paleta de Comandos | `Cmd/Ctrl + P` |
| Nova Nota | `Cmd/Ctrl + N` |

## Personalização

Ajuste a interface ao seu gosto:

### Largura da Barra Lateral

1. Passe o mouse sobre a borda da barra lateral
2. Arraste para redimensionar
3. Ou defina a largura exata nas Configurações

### Tema

Alterne entre os modos Claro e Escuro:

- Use o alternador de tema no cabeçalho
- Ou configure para seguir a preferência do sistema

### Tamanho da Fonte

1. Abra as Configurações
2. Vá para Aparência
3. Ajuste o Tamanho da Fonte do Editor

## Dicas

### Modo Foco

Oculte todos os painéis para escrita sem distrações:

1. Abra a Paleta de Comandos
2. Digite "Modo Foco"
3. Pressione Enter

### Acesso Rápido

- Fixe notas usadas frequentemente aos Favoritos
- Use atalhos de teclado para ações comuns
- Personalize a barra de ferramentas nas Configurações
