# Pesquisa

Encontre qualquer nota instantaneamente com a poderosa pesquisa do Classic. Pesquisa de texto completo, filtros e operadores ajudam você a localizar exatamente o que precisa.

## Pesquisa Básica

### Abrindo a Pesquisa

| Plataforma | Atalho |
|------------|--------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

Ou clique na barra de pesquisa na barra lateral.

### Como Funciona

1. Abra a pesquisa
2. Comece a digitar
3. Os resultados aparecem instantaneamente
4. Clique para abrir uma nota

## Recursos de Pesquisa

### Pesquisa de Texto Completo

O Classic pesquisa através de:

- Títulos de notas
- Conteúdo de notas
- Tags
- Nomes de arquivos

### Resultados em Tempo Real

Os resultados são atualizados enquanto você digita:

- Não precisa pressionar Enter
- Feedback instantâneo
- Correspondência fuzzy habilitada

### Destaque de Pesquisa

Termos encontrados são destacados nos resultados:

- Veja o contexto ao redor das correspondências
- Identifique notas relevantes rapidamente

## Operadores de Pesquisa

### Correspondência Exata

Use aspas para frases exatas:

```
"planejamento de projeto"
"notas de reuniao"
```

### Pesquisa por Tag

Encontre notas com tags específicas:

```
tag:trabalho
tag:reunioes
tag:urgente
```

### Excluir Termos

Use `-` para excluir:

```
projeto -arquivado
reuniao -cancelada
```

### Pesquisa OU

Encontre notas com qualquer termo:

```
projeto OU iniciativa
```

### Intervalo de Datas

Pesquise por data:

```
criado:2024-01-01..2024-12-31
modificado:ultima-semana
```

### Tipo de Arquivo

Filtre por extensão:

```
ext:md
ext:txt
```

## Pesquisa Avançada

### Combinando Operadores

Construa consultas complexas:

```
tag:trabalho -tag:arquivado criado:ultima-semana
"reuniao" tag:projeto-alfa
```

### Pesquisa por Caminho

Pesquise dentro de pastas:

```
path:Projetos/
path:Arquivo/
```

### Pesquisa por Título

Pesquise apenas títulos:

```
title:Roadmap
title:Reuniao
```

### Pesquisa por Conteúdo

Pesquise apenas conteúdo (exclui títulos):

```
content:importante
content:prazo
```

## Filtros de Pesquisa

### Por Tag

1. Digite sua pesquisa
2. Clique em uma tag para filtrar
3. Os resultados são limitados às notas com tags

### Por Data

Filtre por data de modificação:

- Hoje
- Esta semana
- Este mês
- Intervalo personalizado

### Por Pasta

Limite a pesquisa a uma pasta:

1. Clique com o botão direito em uma pasta
2. Selecione "Pesquisar na Pasta"
3. Resultados limitados a essa pasta

## Resultados da Pesquisa

### Ordenação

Ordene os resultados por:

- **Relevância** (padrão)
- **Data de Modificação** (mais recentes primeiro)
- **Data de Criação** (mais recentes primeiro)
- **Título** (alfabética)

### Visualização de Resultados

Passe o mouse sobre os resultados para visualizar:

- Primeiras linhas mostradas
- Termos correspondentes destacados
- Tags exibidas

### Contexto

Veja onde a correspondência ocorreu:

- Mostra o texto ao redor
- Destaca a correspondência
- Clique para pular para o local

## Histórico de Pesquisa

### Pesquisas Recentes

Acesse pesquisas anteriores:

1. Clique na barra de pesquisa
2. Veja pesquisas recentes
3. Clique para executar novamente

### Limpando o Histórico

1. Abra as Configurações
2. Vá para Privacidade
3. Clique em "Limpar Histórico de Pesquisa"

## Dicas

### Use Termos Específicos

Termos mais específicos = melhores resultados:

```
✅ "orçamento de marketing Q4"
❌ orçamento
```

### Combine com Tags

Reduza os resultados rapidamente:

```
projeto tag:urgente
```

### Use Exclusões

Filtre o ruído:

```
reuniao -tag:arquivado
```

### Salve Pesquisas Frequentes

Crie pesquisas salvas:

1. Digite sua pesquisa
2. Clique em "Salvar Pesquisa"
3. Dê um nome a ela
4. Acesse pela barra lateral

## Navegação por Teclado

| Ação | Atalho |
|------|--------|


## Solução de Problemas

### Nenhum Resultado Encontrado

- Verifique a ortografia
- Remova filtros
- Tente termos mais amplos
- Verifique se a nota está arquivada

### Pesquisa Lenta

- Grandes coleções de notas podem demorar mais
- Desative diferenciação de maiúsculas
- Reduza o escopo da pesquisa

### Notas Faltando

- A nota pode estar em uma pasta diferente
- Verifique se a tag está correta
- Verifique se a nota não está arquivada
