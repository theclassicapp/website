# Поиск

Находите любую заметку мгновенно с мощным поиском Classic. Полнотекстовый поиск, фильтры и операторы помогут найти именно то, что нужно.

## Базовый поиск

### Открытие поиска

| Платформа | Горячая клавиша |
|----------|----------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

Или нажмите на строку поиска в боковой панели.

### Как это работает

1. Откройте поиск
2. Начните вводить
3. Результаты появляются мгновенно
4. Нажмите для открытия заметки

## Возможности поиска

### Полнотекстовый поиск

Classic ищет по:

- Заголовкам заметок
- Содержимому заметок
- Тегам
- Именам файлов

### Результаты в реальном времени

Результаты обновляются при вводе:

- Не нужно нажимать Enter
- Мгновенная обратная связь
- Нечёткое соответствие включено

### Подсветка поиска

Найденные термины подсвечиваются в результатах:

- Смотрите контекст вокруг совпадений
- Быстро определяйте релевантные заметки

## Операторы поиска

### Точное совпадение

Используйте кавычки для точных фраз:

```
"планирование проекта"
"заметки с встречи"
```

### Поиск по тегам

Находите заметки с конкретными тегами:

```
tag:работа
tag:встречи
tag:срочно
```

### Исключение терминов

Используйте `-` для исключения:

```
проект -архив
встреча -отменена
```

### Поиск ИЛИ

Находите заметки с любым термином:

```
проект OR инициатива
```

### Диапазон дат

Поиск по дате:

```
created:2024-01-01..2024-12-31
modified:last-week
```

### Тип файла

Фильтрация по расширению:

```
ext:md
ext:txt
```

## Расширенный поиск

### Комбинирование операторов

Создавайте сложные запросы:

```
tag:работа -tag:архив created:last-week
"встреча" tag:проект-альфа
```

### Поиск по пути

Поиск в папках:

```
path:Проекты/
path:Архив/
```

### Поиск по заголовку

Поиск только по заголовкам:

```
title:Дорожная_карта
title:Встреча
```

### Поиск по содержимому

Поиск только по содержимому (исключая заголовки):

```
content:важно
content:дедлайн
```

## Фильтры поиска

### По тегу

1. Введите поисковый запрос
2. Нажмите тег для фильтрации
3. Результаты сужаются до заметок с тегами

### По дате

Фильтрация по дате изменения:

- Сегодня
- Эта неделя
- Этот месяц
- Пользовательский диапазон

### По папке

Ограничение поиска папкой:

1. Кликните правой кнопкой на папке
2. Выберите "Искать в папке"
3. Результаты ограничены этой папкой

## Результаты поиска

### Сортировка

Сортировка результатов по:

- **Релевантности** (по умолчанию)
- **Дате изменения** (сначала новые)
- **Дате создания** (сначала новые)
- **Заголовку** (по алфавиту)

### Предпросмотр результата

Наведите на результаты для предпросмотра:

- Показаны первые несколько строк
- Подсвечены совпадающие термины
- Отображаются теги

### Контекст

Смотрите, где произошло совпадение:

- Показывает окружающий текст
- Подсвечивает совпадение
- Нажмите для перехода к месту

## История поиска

### Недавние поиски

Доступ к предыдущим поискам:

1. Нажмите в строке поиска
2. Просмотрите недавние поиски
3. Нажмите для повторного запуска

### Очистка истории

1. Откройте настройки
2. Перейдите в Приватность
3. Нажмите "Очистить историю поиска"

## Советы

### Используйте конкретные термины

Более конкретные = лучшие результаты:

```
✅ "Бюджет маркетинга Q4"
❌ бюджет
```

### Комбинируйте с тегами

Быстро сужайте результаты:

```
проект tag:срочно
```

### Используйте исключения

Фильтруйте шум:

```
встреча -tag:архив
```

### Сохраняйте частые поиски

Создавайте сохранённые поиски:

1. Введите поисковый запрос
2. Нажмите "Сохранить поиск"
3. Назовите его
4. Доступ из боковой панели

## Навигация с клавиатуры

| Действие | Горячая клавиша |
|--------|----------|
| Открыть поиск | `Cmd/Ctrl+F` |
| Следующий результат | `Enter` или `Down` |
| Предыдущий результат | `Shift+Enter` или `Up` |
| Закрыть поиск | `Escape` |

## Настройки поиска

### Настройка в настройках

| Параметр | Опции |
|---------|---------|
| С учётом регистра | Вкл/Выкл |
| Поиск по заголовкам | Вкл/Выкл |
| Поиск по содержимому | Вкл/Выкл |
| Поиск по тегам | Вкл/Выкл |
| Включая архив | Вкл/Выкл |

## Примеры использования

### Поиск старых заметок

```
"встреча" date:last-year
```

### Исследование проекта

```
path:Проекты/ tag:исследование
```

### Элементы действий

```
"TODO" OR "FIXME" -tag:готово
```

### Недавняя работа

```
modified:this-week
```

## Устранение неполадок

### Результаты не найдены

- Проверьте правописание
- Уберите фильтры
- Попробуйте более широкие термины
- Проверьте, не заархивирована ли заметка

### Медленный поиск

- Большие коллекции заметок могут занимать больше времени
- Отключите учёт регистра
- Уменьшите область поиска

### Отсутствующие заметки

- Заметка может быть в другой папке
- Проверьте правильность тега
- Убедитесь, что заметка не заархивирована
