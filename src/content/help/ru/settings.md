# Настройки

Настройте Classic так, чтобы он работал именно так, как вы хотите. От внешнего вида до поведения редактора — всё настраивается.

## Открытие настроек

### Горячая клавиша

| Платформа | Горячая клавиша |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Через меню

- **macOS**: Classic → Настройки
- **Windows**: Файл → Настройки
- **Linux**: Правка → Настройки

### Через палитру команд

1. Откройте палитру команд (`Cmd+P` или `Ctrl+P`)
2. Введите "Настройки"
3. Нажмите Enter

## Категории настроек

### Общие

Базовые настройки приложения:

| Параметр | Описание |
|---------|-------------|
| **Язык** | Язык интерфейса |
| **Тема** | Светлая, тёмная или системная |
| **Запускать при входе** | Запуск при старте |
| **Проверять обновления** | Поведение автообновления |

### Внешний вид

Настройка внешнего вида:

| Параметр | Опции |
|---------|---------|
| **Тема** | Светлая / Тёмная / Системная |
| **Акцентный цвет** | Синий, фиолетовый, зелёный и т.д. |
| **Шрифт редактора** | Семейство и размер шрифта |
| **Шрифт интерфейса** | Шрифт интерфейса |
| **Моноширинный шрифт** | Шрифт кода |
| **Ширина боковой панели** | 200-400px |
| **Межстрочный интервал** | 1.0-2.5 |

### Редактор

Настройка поведения письма:

| Параметр | Описание |
|---------|-------------|
| **Проверка орфографии** | Включить/отключить проверку орфографии |
| **Автосохранение** | Интервал автоматического сохранения |
| **Перенос слов** | Включить/отключить перенос слов |
| **Номера строк** | Показать/скрыть номера строк |
| **Размер табуляции** | 2, 4 или 8 пробелов |
| **Автозамена** | Умные кавычки, автозамена |

### История версий

Управление отслеживанием версий:

| Параметр | Опции | По умолчанию |
|---------|---------|---------|
| **Включить историю** | Вкл/Выкл | Вкл |
| **Период хранения** | 30-180 дней | 90 дней |
| **Максимум версий** | 50-500 | 100 |

### Резервное копирование и восстановление

Настройки защиты данных:

| Параметр | Описание |
|---------|-------------|
| **Автосохранение** | Включить автоматические резервные копии |
| **Частота резервного копирования** | 15мин / 30мин / 1ч / 4ч |
| **Расположение резервных копий** | Пользовательский путь |
| **Хранить резервные копии** | 5 / 10 / 20 / 50 |
| **Шифрование резервных копий** | Защита паролем |

### Горячие клавиши

Просмотр и настройка горячих клавиш:

- Просмотр всех доступных горячих клавиш

## Поиск по настройкам

Не можете найти настройку? Используйте строку поиска:

1. Откройте настройки
2. Введите в поле поиска
3. Появятся соответствующие настройки
4. Нажмите для перехода к этой настройке

## Сброс настроек

### Сбросить все настройки

Вернуться к значениям по умолчанию:

1. Откройте настройки
2. Прокрутите вниз
3. Нажмите "Сбросить все настройки"
4. Подтвердите действие

### Сброс категории

Сброс определённой категории:

1. Перейдите в категорию
2. Нажмите "Сбросить до значений по умолчанию"
3. Только эта категория сбрасывается

## Экспорт/импорт настроек

### Экспорт настроек

Сохраните конфигурацию:

1. Откройте настройки
2. Нажмите "Экспортировать настройки"
3. Выберите расположение
4. Сохраните как JSON-файл

### Импорт настроек

Загрузите сохранённую конфигурацию:

1. Откройте настройки
2. Нажмите "Импортировать настройки"
3. Выберите JSON-файл
4. Подтвердите импорт

## Расположение настроек

Настройки хранятся локально:

| Платформа | Расположение |
|----------|----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Синхронизация настроек (опционально)

При входе настройки могут синхронизироваться между устройствами:

- **Синхронизация**: Включите в Настройки → Синхронизация
- **Что синхронизируется**: Тема, шрифты, горячие клавиши
- **Что не синхронизируется**: Локальные пути, ключи шифрования

## Распространённые настройки

### Увеличить размер шрифта

1. Настройки → Внешний вид
2. Размер шрифта редактора → Увеличить
3. Изменения применяются немедленно

### Изменить тему

1. Настройки → Внешний вид
2. Тема → Выберите Светлая/Тёмная/Системная
3. Приложение обновляется мгновенно

### Включить автосохранение

1. Настройки → Резервное копирование и восстановление
2. Включите "Автосохранение"
3. Установите частоту и расположение

### Настроить горячие клавиши

1. Настройки → Горячие клавиши
2. Найдите действие
3. Нажмите текущую горячую клавишу
4. Нажмите новые клавиши
5. Сохраните

## Советы

### Быстрый доступ

- Используйте `Cmd/Ctrl+,` для быстрого открытия настроек
- Ищите настройки по имени
- Используйте категории для навигации

### Экспорт перед изменениями

Перед значительными изменениями экспортируйте настройки:

- Легко восстановить, если что-то пойдёт не так
- Делитесь конфигурацией с другими

### Проверяйте новые настройки

Обновления могут добавлять новые настройки:

- Проверяйте после обновления
- Изучайте новые опции

## Устранение неполадок

### Настройки не сохраняются

- Проверьте права файлов
- Верифицируйте место на диске
- Попробуйте сбросить и перенастроить

### Настройки сбрасываются неожиданно

- Может указывать на повреждение файла
- Экспортируйте настройки регулярно
- Проверьте конфликты приложений

### Не могу найти настройку

- Используйте строку поиска
- Проверьте разные категории
- Может быть в продвинутых настройках
