# Математика и уравнения

Classic поддерживает математическую нотацию с использованием синтаксиса LaTeX. Пишите красивые уравнения в заметках.

## Включение математики

Поддержка математики включена в Classic по умолчанию.

### Встроенная математика

Используйте одинарные знаки доллара для встроенных уравнений:

```markdown
Формула $E = mc^2$ навсегда изменила физику.
```

Отображается как: Формула E=mc² навсегда изменила физику.

### Блочная математика

Используйте двойные знаки доллара для центрированных блочных уравнений:

```markdown
$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

## Распространённая нотация

### Базовые операции

| Синтаксис | Результат |
| ------ | ------ |
| `x + y` | x + y |
| `x - y` | x - y |
| `x \times y` | x × y |
| `x \div y` | x ÷ y |
| `x / y` | x/y |
| `x^2` | x² |
| `x_{n}` | xₙ |

### Дроби

```markdown
$\frac{числитель}{знаменатель}$
```

Примеры:

- `$\frac{1}{2}$` → ½
- `$\frac{a+b}{c+d}$` → (a+b)/(c+d)

### Квадратные корни

```markdown
$\sqrt{x}$
$\sqrt[n]{x}$
```

Примеры:

- `$\sqrt{2}$` → √2
- `$\sqrt[3]{8}$` → ³√8

### Нижние и верхние индексы

```markdown
$x_1$          → нижний индекс
$x^2$          → верхний индекс
$x_1^2$        → оба
$x_{10}$       → многосимвольный нижний индекс
$x^{n+1}$      → многосимвольный верхний индекс
```

## Продвинутая математика

### Суммы

```markdown
$$
\sum_{i=1}^{n} x_i
$$
```

### Интегралы

```markdown
$$
\int_{a}^{b} f(x) \, dx
$$
```

### Пределы

```markdown
$$
\lim_{x \to \infty} f(x)
$$
```

### Матрицы

```markdown
$$
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$
```

### Кусочные функции

```markdown
$$
f(x) = \begin{cases}
  1 & \text{если } x > 0 \\
  0 & \text{если } x = 0 \\
  -1 & \text{если } x < 0
\end{cases}
$$
```

## Советы

### Пробелы

Используйте `\,` `\;` `\:` `\quad` `\qquad` для настраиваемых пробелов:

```markdown
$a \, b$     → маленький пробел
$a \quad b$  → большой пробел
```

### Текст в математике

Используйте `\text{}` для обычного текста внутри уравнений:

```markdown
$\text{когда } x > 0$
```

### Цвет

Добавьте цвет к частям уравнений:

```markdown
$\textcolor{red}{важно}$
```

### Выравнивание

Используйте окружение `align` для многострочных уравнений:

```markdown
$$
\begin{align}
a &= b + c \\
d &= e + f
\end{align}
$$
```
