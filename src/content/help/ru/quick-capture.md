# Быстрый захват

Захватывайте идеи мгновенно, не прерывая рабочий процесс. Быстрый захват открывает минималистичное окно для заметок из любого места системы.

## Глобальная горячая клавиша

Нажмите назначенную горячую клавишу для открытия быстрого захвата:

| Платформа | Горячая клавиша по умолчанию |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

Вы можете настроить эту горячую клавишу в настройках.

## Использование быстрого захвата

1. Нажмите глобальную горячую клавишу (из любого места системы)
2. Введите заметку во всплывающем окне
3. Нажмите `Enter` для сохранения
4. Нажмите `Escape` для отмены

## Возможности

### Минималистичный интерфейс

- Маленькое, сфокусированное окно
- Поддержка Markdown
- Автофокус на вводе
- Дизайн, ориентированный на клавиатуру

### Быстрые действия

- `# ` + пробел = заголовок
- `#входящие` = добавить во входящие
- `#сегодня` = добавить в сегодняшнюю заметку
- Теги с `#имятега`

### Назначение по умолчанию

Заметки быстрого захвата по умолчанию попадают в **Входящие**. Вы можете изменить это:

1. Откройте настройки
2. Перейдите в Быстрый захват
3. Выберите назначение по умолчанию

## Советы

### Рабочий процесс захвата

1. **Захват** - Быстро запишите идею
2. **Обработка** - Ежедневно просматривайте входящие
3. **Организация** - Перемещайте в соответствующую папку/заметку

### Лучшие практики

- Будьте краткими - одна идея на захват
- Используйте теги для контекста
- Не беспокойтесь о форматировании - уточните позже
- Регулярно просматривайте и обрабатывайте захваты

### Интеграция

Быстрый захват отлично работает с:

- **Ежедневными заметками**: Начинайте каждый день с просмотра захватов
- **Входящими**: Используйте как зону ожидания
- **Тегами**: Добавляйте контекст немедленно

## Устранение неполадок

### Горячая клавиша не работает?

**macOS**
1. Проверьте Системные настройки > Клавиатура > Горячие клавиши
2. Ищите конфликты
3. Перезапустите Classic

**Windows**
1. Запустите Classic от имени администратора
2. Проверьте конфликтующие горячие клавиши
3. Перезапустите Classic

### Окно не появляется?

- Проверьте, запущен ли Classic
- Проверьте горячую клавишу
- Проверьте проблемы с фокусом
- Перезапустите Classic

### Захваты не сохраняются?

- Проверьте назначение по умолчанию
- Проверьте права на запись
- Проверьте место на диске
- Перезапустите Classic
