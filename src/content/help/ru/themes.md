# Темы и внешний вид

Персонализируйте внешний вид Classic в соответствии со своим стилем и уменьшите нагрузку на глаза.

## Опции тем

### Светлый режим

Тема Classic по умолчанию включает:

- Чистый, яркий интерфейс
- Оптимально для хорошо освещённых помещений
- Высокая контрастность для читаемости

### Тёмный режим

Идеально для сов и работы при слабом освещении:

- Уменьшает нагрузку на глаза
- Меньшее потребление батареи на OLED-экранах
- Комфортно для глаз в тёмных помещениях

### Системная (Авто)

Следуйте теме операционной системы:

- Автоматическое переключение на основе настроек ОС
- Не нужно переключать вручную
- Согласованно с другими приложениями

## Переключение тем

### Через настройки

1. Откройте настройки (`Cmd+,` или `Ctrl+,`)
2. Перейдите в **Внешний вид**
3. Выберите предпочтительную тему

### Через палитру команд

1. Откройте палитру команд (`Cmd+P` или `Ctrl+P`)
2. Введите "Тема"
3. Выберите предпочтительную тему

### Горячая клавиша

| Платформа | Горячая клавиша |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Цикл: Светлая → Тёмная → Системная

## Настройки шрифтов

### Шрифт редактора

Настройте опыт письма:

1. Откройте настройки
2. Перейдите в **Внешний вид** → **Редактор**
3. Настройте опции шрифта

| Параметр | Опции | По умолчанию |
| ------- | ------- | ------- |
| Семейство шрифтов | Системные + пользовательские | System UI |
| Размер шрифта | 12-24px | 16px |
| Межстрочный интервал | 1.0-2.5 | 1.75 |
| Межбуквенный интервал | -2 до 4 | 0 |

### Шрифт интерфейса

Настройте шрифт интерфейса:

1. Откройте настройки
2. Перейдите в **Внешний вид** → **Интерфейс**
3. Выберите шрифт и размер

### Моноширинный шрифт

Для блоков кода и технического контента:

1. Откройте настройки
2. Перейдите в **Внешний вид** → **Код**
3. Выберите моноширинный шрифт

## Опции интерфейса

### Боковая панель

Настройте поведение боковой панели:

| Параметр | Описание |
| ------- | ----------- |
| **Автоскрытие** | Скрыть боковую панель, когда не используется |
| **Ширина** | Настройка ширины боковой панели (200-400px) |
| **Показывать значки** | Переключение видимости значков |

### Макет редактора

Выберите опыт редактирования:

- **Разделённый вид**: Редактирование и предпросмотр рядом
- **Только предпросмотр**: Только отрисованный результат
- **Только редактирование**: Фокус на сыром Markdown

### Список заметок

Настройте отображение заметок:

- **Вид списком**: Компактный, одна строка на заметку
- **Вид карточками**: Карточки предпросмотра с фрагментами
- **Древовидный вид**: Иерархическая структура папок

## Акцентные цвета

### Доступные цвета

Выберите акцентный цвет:

- Классический синий (по умолчанию)
- Фиолетовый
- Зелёный
- Оранжевый
- Красный
- Розовый
- Бирюзовый

### Изменение акцентного цвета

1. Откройте настройки
2. Перейдите в **Внешний вид** → **Цвета**
3. Выберите предпочтительный акцент

Акцентный цвет влияет на:

- Ссылки и кнопки
- Подсветку выделения
- Активные элементы
- Индикаторы фокуса

## Доступность

### Режим высокой контрастности

Для лучшей видимости:

1. Откройте настройки
2. Перейдите в **Внешний вид** → **Доступность**
3. Включите **Высокая контрастность**

### Уменьшение анимации

Минимизируйте анимации:

1. Откройте настройки
2. Перейдите в **Внешний вид** → **Доступность**
3. Включите **Уменьшение движения**

### Настройки шрифтов

- Увеличьте минимальный размер шрифта
- Используйте шрифты, удобные для дислексии
- Настройте коэффициенты контрастности

## Пользовательский CSS (продвинуто)

Для опытных пользователей, желающих полный контроль:

1. Откройте настройки
2. Перейдите в **Внешний вид** → **Продвинуто**
3. Нажмите **Открыть пользовательский CSS**
4. Добавьте свои настройки

### Примеры настроек

```css
/* Большие заголовки */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Пользовательский цвет ссылок */
a { color: #3b82f6; }

/* Широкий межстрочный интервал */
.prose p { line-height: 2; }
```

## Синхронизация тем

При входе (опционально) настройки темы синхронизируются между устройствами:

- Автоматическая синхронизация темы
- Синхронизация настроек шрифтов
- Пользовательский CSS синхронизируется

## Советы

### Соответствие рабочему процессу

- Используйте **Светлую** днём, **Тёмную** ночью
- **Системный** режим обрабатывает это автоматически
- Учитывайте окружающее освещение

### Уменьшение нагрузки на глаза

- Используйте тёмный режим при слабом освещении
- Увеличьте размер шрифта для долгих сессий
- Включите высокую контрастность при необходимости
- Делайте регулярные перерывы

### Продуктивность

- Изучите горячую клавишу переключения темы
- Настройте время автоматического переключения
- Используйте системный режим для согласованности
