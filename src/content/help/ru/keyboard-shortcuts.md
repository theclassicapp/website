# Горячие клавиши

Работайте быстрее с горячими клавишами Classic. Все горячие клавиши настраиваются в настройках.

## Глобальные горячие клавиши

Работают в любом месте приложения.

### Общие

| Горячая клавиша (Windows/Linux) | Горячая клавиша (macOS) | Действие |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | Новая заметка |
| `Ctrl+P` | `Cmd+P` | Палитра команд |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Глобальный быстрый захват |
| `Ctrl+S` | `Cmd+S` | Сохранить заметку |
| `Ctrl+F` | `Cmd+F` | Поиск |
| `Ctrl+,` | `Cmd+,` | Настройки |

### Навигация

| Горячая клавиша (Windows/Linux) | Горячая клавиша (macOS) | Действие |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | Перейти к первой заметке |
| `Ctrl+2` | `Cmd+2` | Перейти ко второй заметке |
| `Ctrl+Left` | `Cmd+Left` | Навигация назад |
| `Ctrl+Right` | `Cmd+Right` | Навигация вперёд |
| `Ctrl+Up` | `Cmd+Up` | Перейти к родительской заметке |
| `Ctrl+Down` | `Cmd+Down` | Развернуть/фокус на первой подзаметке |

### Редактор

| Горячая клавиша (Windows/Linux) | Горячая клавиша (macOS) | Действие |
| ------------------------ | ----------------- | ------ |
| `Ctrl+B` | `Cmd+B` | Жирный |
| `Ctrl+I` | `Cmd+I` | Курсив |
| `Ctrl+K` | `Cmd+K` | Вставить ссылку |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | Зачёркнутый |

### Управление окном

| Горячая клавиша (Windows/Linux) | Горячая клавиша (macOS) | Действие |
| ------------------------ | ----------------- | ------ |
| `Ctrl+W` | `Cmd+W` | Закрыть текущую заметку |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | Закрыть все заметки |
| `F11` | `F11` | Полноэкранный режим |

## Быстрый захват

| Горячая клавиша (Windows/Linux) | Горячая клавиша (macOS) | Действие |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Открыть быстрый захват |
| `Escape` | `Escape` | Закрыть быстрый захват |
| `Ctrl+Enter` | `Cmd+Enter` | Сохранить и закрыть |

## Настройка горячих клавиш

1. Откройте **Настройки** (`Ctrl+,` или `Cmd+,`)
2. Перейдите в **Горячие клавиши**
3. Нажмите на любую горячую клавишу для изменения
4. Нажмите новую комбинацию клавиш
5. Нажмите **Сохранить**

## Советы

- Большинство горячих клавиш настраиваются
- Конфликты с системными горячими клавишами будут подсвечены
- Сбросьте до значений по умолчанию, нажав кнопку сброса рядом с каждой горячей клавишей
