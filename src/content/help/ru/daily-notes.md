# Ежедневные заметки

Ежедневные заметки автоматически создают новую заметку для каждого дня — идеально для ведения журнала, ежедневного планирования и фиксации информации по времени.

## Как это работает

### Автоматическое создание

- Новая заметка создаётся для каждого дня
- Заметки именуются с использованием формата даты (например, `2024-01-15.md`)
- Заметки предыдущих дней сохраняются и доступны для поиска

### Доступ к ежедневным заметкам

1. Нажмите **Ежедневные заметки** в боковой панели
2. Или нажмите `Ctrl/Cmd+D`
3. Сегодняшняя заметка откроется автоматически

## Шаблон ежедневной заметки

Настройте шаблон для новых ежедневных заметок.

### Шаблон по умолчанию

```markdown
# {{date}}

## Задачи
- [ ]
- [ ]

## Заметки


## Журнал

```

### Настройка

1. Откройте настройки
2. Перейдите в Ежедневные заметки
3. Отредактируйте шаблон
4. Используйте `{{date}}` для текущей даты

### Переменные шаблона

| Переменная | Описание |
| -------- | ----------- |
| `{{date}}` | Текущая дата (YYYY-MM-DD) |
| `{{time}}` | Текущее время (HH:mm) |
| `{{weekday}}` | Название дня (Понедельник и т.д.) |

## Возможности

### Навигация по датам

- **Предыдущая/Следующая**: Навигация между днями
- **Календарь**: Переход к конкретным датам
- **Поиск**: Поиск по всем ежедневным заметкам

### Автоматические ссылки

- Заметки, созданные в один день, автоматически связываются
- Обратные ссылки показывают связи между днями
- Теги работают по всем ежедневным заметкам

### Интеграция

- **Быстрый захват**: Добавьте в сегодняшнюю заметку с `#сегодня`
- **Календарный вид**: Визуальный обзор ежедневных заметок
- **Статистика**: Отслеживание серий письма

## Советы

### Последовательность

- Создавайте заметки в одно и то же время каждый день
- Используйте согласованный формат
- Просматривайте и обновляйте в течение дня

### Организация

- Используйте заголовки для разных разделов
- Добавляйте теги для лёгкого поиска
- Ссылайтесь на связанные заметки

### Ревью

- Еженедельный обзор ежедневных заметок
- Извлекайте важные элементы в постоянные заметки
- Архивируйте или обобщайте старые заметки
