# Палитра команд

Палитра команд — ваш шлюз к каждому действию в Classic. Нажмите простую горячую клавишу и мгновенно получите доступ к любой функции.

## Открытие палитры команд

| Платформа | Горячая клавиша |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Как использовать

1. Откройте палитру команд (`Cmd+P` или `Ctrl+P`)
2. Введите для поиска команды
3. Используйте стрелки для навигации
4. Нажмите `Enter` для выполнения

## Категории команд

### Файловые операции

- **Новая заметка**: Создать новую заметку
- **Новая ежедневная заметка**: Создать/открыть сегодняшнюю ежедневную заметку
- **Открыть заметку**: Поиск и открытие существующих заметок
- **Сохранить заметку**: Сохранить текущую заметку
- **Удалить заметку**: Удалить текущую заметку
- **Переименовать заметку**: Переименовать текущую заметку

### Навигация

- **Перейти к заметке**: Переход к любой заметке
- **Перейти к ежедневной заметке**: Открыть конкретную дату
- **Перейти к тегу**: Просмотр заметок по тегу
- **Назад**: Навигация назад по истории
- **Вперёд**: Навигация вперёд по истории

### Редактор

- **Переключить предпросмотр**: Переключение между режимами редактирования и предпросмотра
- **Переключить боковую панель**: Показать/скрыть боковую панель
- **Режим концентрации**: Войти в режим письма без отвлечений
- **Вставить дату**: Добавить текущую дату/время
- **Вставить шаблон**: Вставить шаблон

### Форматирование

- **Жирный**: Сделать текст жирным
- **Курсив**: Сделать текст курсивным
- **Зачёркнутый**: Зачеркнуть текст
- **Код**: Форматировать как код
- **Ссылка**: Вставить ссылку
- **Заголовок**: Применить формат заголовка

### Организация

- **Добавить тег**: Тегировать текущую заметку
- **Переместить в папку**: Переместить заметку в папку
- **Архивировать заметку**: Архивировать текущую заметку
- **Закрепить заметку**: Закрепить заметку вверху списка

### Вид

- **Переключить тему**: Переключение между светлой/тёмной
- **Увеличить**: Увеличить размер шрифта
- **Уменьшить**: Уменьшить размер шрифта
- **Сбросить масштаб**: Сбросить до размера по умолчанию

## Советы

### Быстрый доступ

- Команды используют нечёткий поиск
- Недавние команды появляются первыми
- Закрепляйте часто используемые команды

### Навигация с клавиатуры

- `↑`/`↓` - Навигация по результатам
- `Enter` - Выполнить команду
- `Escape` - Закрыть палитру
- `Tab` - Циклический переход по категориям

### Пользовательские команды

1. Откройте настройки
2. Перейдите в Палитра команд
3. Добавьте пользовательские команды
4. Определите горячие клавиши

## Часто используемые команды

Вот некоторые команды, которые вы будете использовать часто:

| Команда | Что делает |
| ------- | ------------- |
| `Новая заметка` | Создать пустую заметку |
| `Открыть заметку` | Поиск по всем заметкам |
| `Новая ежедневная заметка` | Сегодняшний журнал |
| `Переключить предпросмотр` | Предпросмотр или редактирование |
| `Добавить тег` | Тегировать текущую заметку |
| `Быстрый захват` | Глобальный захват |

## Настройка команд

### Изменение горячих клавиш

1. Откройте настройки (`Cmd+,` или `Ctrl+,`)
2. Перейдите в Горячие клавиши
3. Найдите команду
4. Нажмите текущую горячую клавишу
5. Нажмите новую комбинацию
6. Сохраните

### Создание пользовательских команд

Продвинутые пользователи могут создавать пользовательские команды:

1. Откройте настройки
2. Перейдите в Команды
3. Нажмите "Добавить пользовательскую команду"
4. Определите имя команды
5. Добавьте действие(я)
6. Установите горячую клавишу (опционально)
