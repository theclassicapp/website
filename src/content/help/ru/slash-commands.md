# Команды с косой чертой

Команды с косой чертой — это мощный способ быстрого добавления форматирования и специального содержимого без отрыва от клавиатуры. Введите `/` в редакторе, чтобы открыть палитру команд.

## Доступные команды

### Заголовки

| Команда | Результат |
| ------- | ------ |
| `/h1` | Заголовок уровня 1 |
| `/h2` | Заголовок уровня 2 |
| `/h3` | Заголовок уровня 3 |
| `/h4` | Заголовок уровня 4 |
| `/h5` | Заголовок уровня 5 |
| `/h6` | Заголовок уровня 6 |

### Форматирование текста

| Команда | Результат |
| ------- | ------ |
| `/bold` | **Жирный текст** |
| `/italic` | *Курсивный текст* |
| `/strikethrough` | ~~Зачёркнутый~~ |
| `/code` | Встроенный блок кода |
| `/quote` | Цитата |

### Списки

| Команда | Результат |
| ------- | ------ |
| `/bullet` | • Маркированный список |
| `/numbered` | 1. Нумерованный список |
| `/todo` | - [ ] Элемент задачи |
| `/check` | - [x] Выполненный элемент |

### Вставка содержимого

| Команда | Результат |
| ------- | ------ |
| `/image` | Вставить изображение из URL или файла |
| `/link` | Вставить гиперссылку |
| `/table` | Вставить таблицу |
| `/hr` | Горизонтальная линия |
| `/code` | Блок кода с выбором языка |

### Специальные блоки

| Команда | Результат |
| ------- | ------ |
| `/mermaid` | Блок диаграммы Mermaid |
| `/katex` | Блок математики KaTeX |
| `/callout` | Блок примечания/предупреждения |

## Использование команд с косой чертой

1. Введите `/` в редакторе
2. Появится меню со всеми доступными командами
3. Начните вводить для фильтрации команд
4. Нажмите `Enter` или кликните для вставки

## Профессиональные советы

- **Быстрый доступ**: Просто введите `/` и начните вводить название команды
- **Нечёткий поиск**: Не нужно вводить точное название команды — Classic найдёт ближайшее совпадение
- **Навигация с клавиатуры**: Используйте стрелки для навигации по меню
- **Недавние команды**: Часто используемые команды появляются вверху списка
