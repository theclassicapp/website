# История версий

Classic отслеживает каждое изменение ваших заметок. Просматривайте, сравнивайте и восстанавливайте предыдущие версии когда угодно.

## Как работает история версий

### Автоматическое отслеживание

Classic автоматически сохраняет версии заметок:

- **Каждое редактирование**: Каждое сохранение создаёт версию
- **С временными метками**: Знайте, когда произошли изменения
- **Полное содержимое**: Полные снимки сохранены

### Хранение

| Параметр | По умолчанию |
|---------|---------|
| **Хранение истории** | 90 дней |
| **Максимум версий** | 100 на заметку |
| **Автоочистка** | Да |

Настройте это в Настройки → Редактор → История версий.

## Просмотр истории

### Открытие истории версий

1. Откройте любую заметку
2. Нажмите значок истории на панели инструментов
3. Или используйте `Cmd/Ctrl+H`

### Панель истории

Панель истории показывает:

- **Временную шкалу**: Список всех версий
- **Временные метки**: Когда каждая версия была сохранена
- **Предпросмотр**: Смотрите содержимое выбранной версии

### Навигация по версиям

1. Нажмите любую версию в списке
2. Предпросмотр показывает эту версию
3. Прокрутите изменения
4. Сравните с текущей версией

## Сравнение версий

### Вид рядом

Смотрите различия между версиями:

1. Откройте историю версий
2. Выберите версию
3. Нажмите "Сравнить"
4. Смотрите добавления (зелёным) и удаления (красным)

### Вид различий

Текстовые различия подсвечены:

- **Зелёным**: Добавленный контент
- **Красным**: Удалённый контент
- **Без изменений**: Обычный текст

## Восстановление версий

### Восстановить всю заметку

Замените текущее содержимое предыдущей версией:

1. Откройте историю версий
2. Выберите версию
3. Нажмите "Восстановить"
4. Подтвердите действие

### Восстановить выборку

Скопируйте только часть предыдущей версии:

1. Откройте историю версий
2. Выберите версию
3. Выделите нужный текст
4. Нажмите "Копировать в буфер обмена"
5. Вставьте в текущую заметку

### Создать новую заметку из версии

Сохраните обе версии:

1. Откройте историю версий
2. Выберите версию
3. Нажмите "Создать заметку из версии"
4. Новая заметка создана

## Действия с историей версий

| Действие | Описание |
|--------|-------------|
| **Просмотр** | Предпросмотр версии |
| **Сравнить** | Сравнить с текущей |
| **Восстановить** | Заменить текущую на эту |
| **Копировать** | Копировать содержимое в буфер |
| **Удалить** | Удалить эту версию |

## Горячие клавиши

| Действие | Горячая клавиша |
|--------|----------|
| Открыть историю | `Cmd/Ctrl+H` |
| Предыдущая версия | `Up` |
| Следующая версия | `Down` |
| Восстановить | `Cmd/Ctrl+R` |
| Закрыть историю | `Escape` |

## Настройки

### Настройка в Настройки → Редактор → История версий

| Параметр | Опции | По умолчанию |
|---------|---------|---------|
| Включить историю | Вкл/Выкл | Вкл |
| Период хранения | 30/60/90/180 дней | 90 дней |
| Максимум версий | 50/100/200/500 | 100 |
| Интервал автосохранения | 1/5/10/30 минут | 5 минут |

### Расположение хранилища

История версий хранится локально:

- **macOS**: `~/Library/Application Support/Classic/history/`
- **Windows**: `%APPDATA%/Classic/history/`
- **Linux**: `~/.config/Classic/history/`

## Примеры использования

### Восстановление случайных изменений

1. Откройте историю версий
2. Найдите версию до ошибки
3. Восстановите эту версию

### Сравнение черновиков

1. Откройте историю
2. Сравните разные версии
3. Смотрите, как заметка развивалась

### Поиск удалённого содержимого

1. Просматривайте старые версии
2. Ищите удалённый текст
3. Скопируйте обратно

### Отслеживание прогресса

Смотрите, как заметка развивалась со временем:

- Когда идеи были добавлены
- Как структура изменилась
- Эволюция письма

## Советы

### Регулярный обзор

Периодически просматривайте историю версий:

- Очищайте ненужные версии
- Отмечайте, как контент развивался
- Учитесь на своих паттернах редактирования

### Перед значительными изменениями

Создайте контрольную точку:

1. Сделайте небольшое редактирование
2. Сохраните (создаёт версию)
3. Теперь делайте значительные изменения
4. Легко вернуться, если нужно

### Называйте важные версии

Добавляйте маркеры в заметки:

```markdown
<!-- ВЕРСИЯ: Финальный черновик -->
<!-- ВЕРСИЯ: До реструктуризации -->
```

Это помогает идентифицировать версии позже.

## Устранение неполадок

### История не сохраняется

- Проверьте, включена ли история версий
- Верифицируйте, что в расположении хранилища есть место
- Проверьте права файлов

### Не могу найти старую версию

- Версия могла быть автоматически очищена
- Проверьте настройки хранения
- Увеличьте период хранения

### История занимает слишком много места

1. Уменьшите период хранения
2. Уменьшите максимум версий на заметку
3. Запустите ручную очистку

### Медленная загрузка истории

Большие заметки со множеством версий могут загружаться медленно:

- Будьте терпеливы с большими историями
- Рассмотрите уменьшение максимума версий
- Архивируйте старые заметки

## Приватность и безопасность

### Локальное хранение

История версий хранится локально:

- Никогда не отправляется на серверы
- Ваши данные остаются приватными
- Только у вас есть доступ

### Шифрование резервных копий

Если вы используете шифрование резервных копий:

- История версий включена
- Защищена паролем резервной копии
- Безопасна даже в файлах резервных копий
