# Локальная архитектура

Classic создан с приватностью в качестве основы. Ваши данные остаются на вашем устройстве, всегда.

## Что означает "локальный подход"?

Локальный подход означает:

1. **Ваши данные хранятся локально** на вашем устройстве. Все заметки, вложения и настройки хранятся в папке на вашем компьютере.
2. **Облако не требуется** Classic работает без интернет-соединения.
3. **Вы владеете своими данными** У вас полный контроль над данными.

## Хранение данных

### Расположение по умолчанию

Ваши данные хранятся в выделенной папке:

| Платформа | Расположение |
| -------- | -------- |
| **macOS** | `~/Library/Application Support/Classic/notes` |
| **Windows** | `%APPDATA%\Classic\notes` |
| **Linux** | `~/.config/Classic/notes` |

### Формат данных

- Заметки хранятся как текстовые файлы (`.md`)
- Вложения хранятся в подпапке `attachments`
- Настройки хранятся в `settings.json`

### Структура файлов

```
notes/
├── Входящие/
├── Ежедневные заметки/
│   └── 2024-01-15.md
├── Проекты/
│   ├── Мой проект.md
│   └── подзаметка.md
└── attachments/
    └── image.png
settings.json
```

## Преимущества приватности

### Учётная запись не требуется

- Регистрация не нужна
- Верификация email не требуется
- Персональная информация не собирается

### Без отслеживания

- Без аналитики
- Без телеметрии
- Данные об использовании не отправляются на внешние серверы

### Работа офлайн

- Работает без интернета
- Синхронизация только по вашему выбору
- Данные всегда доступны

## Безопасность данных

### Ваша ответственность

- **Резервные копии**: Настройте автоматические резервные копии в настройках
- **Экспорт**: Регулярно экспортируйте данные
- **Обновления**: Держите Classic обновлённым для исправлений безопасности

### Наша ответственность

- **Локальное хранение**: Мы никогда не видим и не касаемся ваших данных
- **Без бэкдоров**: Без скрытого сбора данных
- **Прозрачность**: Откровенность в обработке данных

## Синхронизация (опционально)

Classic не включает встроенную облачную синхронизацию. Однако вы можете синхронизировать данные:

### Использование iCloud/OneDrive/Dropbox

1. Откройте настройки Classic
2. Измените расположение папки заметок на вашу облачную папку
3. Classic будет синхронизироваться через облачного провайдера

### Использование Git

1. Инициализируйте git-репозиторий в папке заметок
2. Коммитьте и отправляйте изменения вручную или с автоматизацией

## Восстановление данных

### Из резервной копии

1. Откройте настройки
2. Перейдите в Резервное копирование и восстановление
3. Выберите файл резервной копии
4. Нажмите Восстановить

### Ручное восстановление

1. Найдите папку заметок
2. Скопируйте файлы в безопасное место
3. Импортируйте в новую установку Classic

## Часто задаваемые вопросы

### Действительно ли мои данные приватны?

**Да.** Ваши данные никогда не покидают ваше устройство, если вы явно не синхронизируете их с помощью стороннего сервиса.

### Могу ли я перенести данные на другой компьютер?

**Да.** Просто скопируйте папку заметок на новый компьютер и укажите на неё в настройках Classic.

### Что произойдёт при удалении Classic?

**Ваши данные остаются.** Удаление Classic не удаляет папку заметок. Вы можете безопасно удалить и переустановить без потери данных.

### Может ли Classic получить доступ к моим данным?

**Нет.** Classic — это локальное приложение без внешнего подключения для сбора данных.
