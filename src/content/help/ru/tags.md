# Теги

Теги — это мощный способ организации и категоризации заметок в Classic. Мгновенно находите связанный контент.

## Создание тегов

### Встроенные теги

Добавляйте теги прямо в содержимое заметки:

```markdown
# Заметки с встречи

Обсудили дорожную карту Q4 #работа #встречи #планирование
```

### Использование кнопки тега

1. Нажмите значок тега на панели инструментов
2. Введите имя тега
3. Нажмите Enter

### Горячая клавиша

| Платформа | Горячая клавиша |
|----------|----------|
| **macOS** | `Cmd+T` |
| **Windows/Linux** | `Ctrl+T` |

## Синтаксис тегов

### Базовые теги

```markdown
#проект #работа #важно
```

### Вложенные теги

Создавайте иерархии тегов с помощью `/`:

```markdown
#работа/встречи
#работа/проекты
#личное/хобби/чтение
```

### Многословные теги

Используйте дефисы или подчёркивания:

```markdown
#проект-альфа
#заметки_встречи
#планирование-q4
```

## Управление тегами

### Просмотр всех тегов

1. Откройте боковую панель
2. Найдите раздел "Теги"
3. Просмотрите все теги

### Облако тегов

Визуализируйте наиболее используемые теги:

- Большие теги = больше заметок
- Нажмите любой тег для фильтрации
- Смотрите количество заметок на тег

### Переименование тегов

1. Кликните правой кнопкой на теге
2. Выберите "Переименовать"
3. Введите новое имя
4. Все заметки обновятся автоматически

### Удаление тегов

1. Кликните правой кнопкой на теге
2. Выберите "Удалить"
3. Подтвердите удаление
4. Тег удалён из всех заметок

## Поиск по тегам

### Панель тегов

Нажмите тег в боковой панели для фильтрации заметок.

### Синтаксис поиска

Используйте `tag:` в поиске:

```
tag:работа
tag:работа/встречи
```

### Комбинирование тегов

Находите заметки с несколькими тегами:

```
tag:работа tag:срочно
```

### Исключение тегов

Находите заметки без тега:

```
-tag:архив
```

## Лучшие практики работы с тегами

### Будьте проще

- Используйте 3-5 тегов на заметку
- Будьте последовательны в именовании
- Избегайте разрастания тегов

### Используйте иерархии

Организуйте связанные теги:

```
#работа
  #работа/встречи
  #работа/проекты
  #работа/дедлайны

#личное
  #личное/здоровье
  #личное/финансы
```

### Установите соглашения

Определите стандарты тегов для себя:

- **Статус**: `#сделать` `#в-процессе` `#готово`
- **Приоритет**: `#срочно` `#высокий` `#низкий`
- **Тип**: `#встреча` `#справка` `#идея`

## Предложения тегов

При вводе `#` Classic предлагает существующие теги:

- Недавние теги появляются первыми
- Поддерживается нечёткое соответствие
- Нажмите Tab для принятия предложения

## Статистика тегов

Просмотр использования тегов в настройках:

- Наиболее используемые теги
- Неиспользуемые теги
- Тренды тегов по времени

## Примеры использования

### Управление проектами

```markdown
# Запуск проекта #проект-альфа #встреча #2024-01-15

Участники: Команда Альфа
Статус: #сделать
```

### База знаний

```markdown
# Замыкания JavaScript #программирование #javascript #справка #продвинутое

Замыкания — это функции, которые имеют доступ к...
```

### Ежедневные заметки

```markdown
# 2024-01-15

- Утренний стендап #работа #встречи
- Код-ревью #работа #разработка
- Тренировка в зале #личное #здоровье
```

### Заметки о чтении

```markdown
# Заметки по книге "Атомные привычки" #чтение #книги #продуктивность

Ключевые выводы из книги...
```

## Горячие клавиши

| Действие | Горячая клавиша |
|--------|----------|
| Добавить тег | `Cmd/Ctrl+T` |
| Поиск по тегам | `Cmd/Ctrl+F`, затем введите `tag:` |
| Показать панель тегов | `Cmd/Ctrl+Shift+T` |

## Советы

### Согласованное именование

Придерживайтесь соглашения:

```
✅ #заметки-встречи
✅ #заметки_встречи
❌ #Заметки Встречи (пробелы не разрешены)
❌ #заметкиВстречи (несогласованно)
```

### Избегайте избыточного тегирования

Слишком много тегов становятся неуправляемыми:

```
✅ #работа #срочно #проект-альфа
❌ #работа #срочно #важно #сегодня #понедельник #офис #стол #компьютер
```

### Периодическая очистка

Регулярно просматривайте и консолидируйте теги:

- Объединяйте похожие теги
- Удаляйте неиспользуемые теги
- Обновляйте соглашения

## Расширенные возможности

### Умные папки на основе тегов

Создавайте папки, которые автоматически заполняются на основе тегов:

1. Создайте новую папку
2. Установите фильтр: `tag:проект`
3. Все заметки с тегами появятся

### Теги в YAML

Добавляйте теги во фронтматтер заметки:

```yaml
---
tags:
  - работа
  - встречи
  - проект-альфа
---
```

### Псевдонимы тегов

Настройте псевдонимы тегов в настройках:

- `#сделать` = `#задача` = `#действие`
- Консолидируйте похожие теги
