# Подзаметки

Подзаметки позволяют создавать вложенные, иерархические заметки в Classic. Организуйте сложные темы в управляемые части.

## Что такое подзаметки?

Подзаметки — это заметки внутри заметок, способ создания иерархических структур контента.

### Преимущества

- **Организация**: Разбиение сложных тем на меньшие части
- **Вложенность**: Создание нескольких уровней иерархии
- **Навигация**: Лёгкая навигация между связанными заметками
- **Гибкость**: Развёртывание и свёртывание по необходимости

## Создание подзаметок

### Способ 1: Перетаскивание

Создайте подзаметки, перетаскивая содержимое под родительскую заметку:

1. Создайте родительскую заметку
2. Создайте новую заметку
3. Перетащите новую заметку на родительскую
4. Она станет подзаметкой

### Способ 2: Контекстное меню

1. Кликните правой кнопкой на заметке
2. Выберите "Новая подзаметка"
3. Вложенная заметка создана

### Способ 3: Горячая клавиша

1. Выберите заметку
2. Нажмите `Cmd/Ctrl+Shift+N`
3. Подзаметка создана

## Управление подзаметками

### Развёртывание и свёртывание

Нажмите стрелку рядом с заметкой, содержащей подзаметки:

- **Развёртывание**: Показать все вложенные заметки
- **Свёртывание**: Скрыть вложенные заметки

### Навигация

- Кликните для открытия подзаметки
- Используйте стрелки для навигации по иерархии
- Хлебные крошки показывают ваше местоположение

### Перемещение подзаметок

Перетащите для реорганизации:

1. Нажмите и удерживайте подзаметку
2. Перетащите в новое место
3. Отпустите для перемещения

### Преобразование в основную заметку

Повысьте подзаметку до верхнего уровня:

1. Кликните правой кнопкой на подзаметке
2. Выберите "Переместить в корень"
3. Заметка станет основной

## Глубина подзаметок

Classic поддерживает несколько уровней вложенности:

```
📁 Проект
  📄 Обзор
  📁 Исследование
    📄 Пользовательские интервью
    📄 Результаты опроса
    📁 Анализ
      📄 Ключевые выводы
      📄 Рекомендации
  📁 Планирование
    📄 Временная шкала
    📄 Ресурсы
```

### Лучшие практики

- **Ограничьте глубину**: Максимум 3-4 уровня для ясности
- **Логическая группировка**: Группируйте связанный контент вместе
- **Чёткие названия**: Используйте описательные имена для подзаметок

## Связи с подзаметками

### Обратные ссылки

Подзаметки автоматически создают обратные ссылки на родительские заметки:

- Смотрите раздел "Связанные ссылки"
- Навигация между связанными заметками
- Понимание связей между заметками

### Вики-ссылки

Ссылайтесь на подзаметки из любого места:

```markdown
См. [[Проект/Исследование/Анализ]] для подробностей.
```

## Советы

### Соглашение об именах

Используйте чёткие, согласованные имена:

```
✅ Хорошо:
Проект/Требования/Функциональные
Проект/Требования/Нефункциональные

❌ Избегайте:
Проект/треб/функ
Проект/тр/нф
```

### Навигация по хлебным крошкам

Используйте хлебные крошки для понимания вашего местоположения:

```
Проект > Исследование > Анализ
```

### Поиск

Поиск автоматически включает подзаметки:

- Поиск контента на всех уровнях
- Фильтрация по глубине заметки
- Быстрая навигация

### Шаблоны

Применяйте шаблоны к подзаметкам:

1. Создайте подзаметку
2. Используйте команду с косой чертой `/template`
3. Выберите шаблон

## Горячие клавиши

| Действие | Горячая клавиша |
|--------|----------|
| Новая подзаметка | `Cmd/Ctrl+Shift+N` |
| Развёрнуть все | `Cmd/Ctrl+Down` |
| Свёрнуть все | `Cmd/Ctrl+Up` |
| Переместить вверх | `Opt/Alt+Up` |
| Переместить вниз | `Opt/Alt+Down` |
| Повысить уровень | `Opt/Alt+Left` |
| Понизить уровень | `Opt/Alt+Right` |

## Устранение неполадок

### Подзаметка не отображается

- Проверьте, не свёрнута ли родительская заметка
- Убедитесь, что заметка не была перемещена
- Обновите боковую панель

### Невозможно создать подзаметку

- Убедитесь, что родительская заметка существует
- Проверьте права доступа
- Попробуйте метод через контекстное меню

### Потерянные подзаметки

Используйте поиск для поиска потерянных заметок:

1. Откройте поиск (`Cmd/Ctrl+F`)
2. Введите имя заметки
3. Просмотрите полный путь в результатах
