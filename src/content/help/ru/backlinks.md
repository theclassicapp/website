# Обратные ссылки

Обратные ссылки показывают связи между заметками, создавая сеть связанных идей. Открывайте связи, о которых вы не знали.

## Что такое обратные ссылки?

Обратные ссылки показывают, какие заметки ссылаются на текущую заметку. Это связи в вашем графе знаний.

### Преимущества

- **Открывайте связи**: Автоматически находите связанные заметки
- **Контекст**: Понимайте, как идеи соотносятся
- **Навигация**: Переходите между связанными заметками
- **Граф знаний**: Визуализируйте своё мышление

## Как работают обратные ссылки

Когда вы создаёте ссылку на другую заметку, Classic автоматически создаёт обратную ссылку.

```markdown
В заметке A:
См. [[Планирование проекта]] для подробностей.

Результат:
- Заметка A ссылается на "Планирование проекта"
- "Планирование проекта" показывает обратную ссылку на заметку A
```

## Просмотр обратных ссылок

### Панель обратных ссылок

Откройте панель обратных ссылок для просмотра всех ссылок:

1. Откройте любую заметку
2. Нажмите значок обратных ссылок в боковой панели
3. Просмотрите все заметки, ссылающиеся на эту

### Встроенные обратные ссылки

Обратные ссылки появляются внизу каждой заметки:

- Показывает имя исходной заметки
- Отображает фрагмент контекста
- Нажмите для навигации

## Создание ссылок

### Синтаксис вики-ссылок

Используйте двойные скобки для ссылок на заметки:

```markdown
[[Имя заметки]]
[[Другая заметка]]
```

### Текст отображения

Настройте текст ссылки:

```markdown
[[Проект Альфа|проект альфа]]
[[Заметки с встречи|вчерашняя встреча]]
```

### Ссылки на заголовки

Ссылайтесь на конкретные разделы:

```markdown
[[Имя заметки#Заголовок]]
[[Имя заметки#Раздел|перейти к разделу]]
```

## Типы обратных ссылок

### Прямые ссылки

Явные вики-ссылки, которые вы создаёте:

```markdown
Связано: [[Похожая заметка]]
```

### Несвязанные упоминания

Заметки, которые упоминают заголовок, но не связаны:

```markdown
Заметка упоминает "Проект Альфа", но не связана
```

Classic предлагает преобразовать их в ссылки.

## Панель обратных ссылок

### Возможности

- **Фильтр**: Поиск по обратным ссылкам
- **Сортировка**: По дате, имени или релевантности
- **Предпросмотр**: Наведите для предпросмотра содержимого
- **Контекст**: Смотрите окружающий текст

### Действия

- Нажмите для открытия связанной заметки
- Кликните правой кнопкой для опций
- Преобразуйте несвязанные упоминания

## Примеры использования

### Управление знаниями

Создайте персональную вики:

```markdown
# Машинное обучение

Связано с [[Искусственный интеллект]]
Основано на [[Статистика]]
Используется в [[Обработка естественного языка]]
```

### Документация проекта

Связывайте связанные документы:

```markdown
# Дизайн API

Ссылается на [[Схема базы данных]]
Зависит от [[Поток аутентификации]]
См. также [[Интеграция с фронтендом]]
```

### Исследования

Связывайте концепции и источники:

```markdown
# Исследовательские заметки

Основано на [[Исходная статья A]]
Противоречит [[Исходная статья B]]
Поддерживает [[Гипотеза 1]]
```

### Заметки с встреч

Связывайте встречи с проектами:

```markdown
# Встреча: Планирование Q4

Проект: [[Проект Альфа]]
Предыдущая: [[Обзор Q3]]
Следующая: [[Запуск Q4]]
```

## Представление графа

Визуализируйте обратные ссылки как сеть:

1. Откройте представление графа (`Cmd/Ctrl+G`)
2. Просмотрите все заметки как узлы
3. Линии показывают связи
4. Нажимайте узлы для навигации

### Управление графом

- **Масштаб**: Прокручивайте для масштабирования
- **Панорамирование**: Перетаскивайте для перемещения
- **Фильтр**: Показать/скрыть по тегу
- **Фокус**: Подсветите связи заметки

## Советы

### Создавайте много ссылок

Создавайте ссылки при каждом упоминании другой концепции:

```markdown
✅ Хорошо:
Обсудили [[Дорожная карта Q4]] и [[Распределение бюджета]].

❌ Избегайте:
Обсудили дорожную карту Q4 и распределение бюджета.
```

### Используйте согласованные имена

Точно совпадайте с заголовками заметок для надёжных ссылок:

```markdown
✅ [[Заметки с встречи]]
❌ [[заметки с встречи]] (регистр важен)
```

### Просматривайте обратные ссылки

Периодически проверяйте обратные ссылки для:

- Обнаружения забытых связей
- Поиска заметок для объединения
- Выявления пробелов в знаниях

### Предпросмотр ссылок

Наведите на ссылки для предпросмотра без открытия:

- Просмотрите первые несколько строк
- Проверьте, правильная ли заметка
- Быстрая справка

## Горячие клавиши

| Действие | Горячая клавиша |
|--------|----------|
| Открыть обратные ссылки | `Cmd/Ctrl+Shift+B` |
| Создать ссылку | `[[` (автозаполнение) |
| Открыть представление графа | `Cmd/Ctrl+G` |
| Навигация по ссылкам | `Tab` по ссылкам |

## Расширенные возможности

### Встраивание

Встраивайте содержимое из других заметок:

```markdown
![[Имя заметки]]
![[Имя заметки#Раздел]]
```

### Трансклюзия

Включайте содержимое встроенно:

```markdown
## Связанный контент
![[Общий шаблон#Заголовок]]
```

### Цветовая кодировка ссылок

Ссылки меняют цвет в зависимости от:

- **Синий**: Непосещённая ссылка
- **Фиолетовый**: Посещённая ссылка
- **Красный**: Сломанная ссылка (заметка не существует)

## Устранение неполадок

### Отсутствующие обратные ссылки

- Проверьте синтаксис ссылки `[[Имя заметки]]`
- Убедитесь, что заголовок заметки точно совпадает
- Обновите панель обратных ссылок

### Сломанные ссылки

- Заметка могла быть переименована
- Заметка могла быть удалена
- Используйте поиск для нахождения правильной заметки

### Дублирующиеся ссылки

- Консолидируйте похожие заметки
- Используйте перенаправления для переименованных заметок
- Проверьте вариации заголовков
