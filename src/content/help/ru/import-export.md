# Импорт и экспорт

Перенесите заметки в Classic из других приложений или экспортируйте их для обмена или резервного копирования. Classic поддерживает несколько форматов для бесшовной миграции данных.

## Импорт заметок

### Поддерживаемые форматы

| Формат | Расширение | Примечания |
| ------ | --------- | ----- |
| Markdown | `.md` | ✅ Полная поддержка |
| Простой текст | `.txt` | ✅ Импортируется как заметка |
| JSON | `.json` | ✅ Формат Classic |
| HTML | `.html` | ⚠️ Конвертируется в Markdown |
| Evernote | `.enex` | ⚠️ Экспериментально |
| Notion | Экспорт Notion | ⚠️ Экспериментально |
| Obsidian | Экспорт Markdown | ✅ Полная поддержка |

### Процесс импорта

1. Откройте **Файл** → **Импорт** (или `Cmd/Ctrl+I`)
2. Выберите файл(ы) для импорта
3. Выберите папку назначения
4. Просмотрите предпросмотр импорта
5. Подтвердите импорт

### Импорт из буфера обмена

Быстрый импорт из любого источника:

1. Скопируйте текст в буфер обмена
2. Откройте палитру команд
3. Введите "Импорт из буфера обмена"
4. Новая заметка создана автоматически

### Массовый импорт

Импорт нескольких файлов одновременно:

1. Откройте **Файл** → **Импорт**
2. Выберите **Импортировать папку**
3. Выберите папку, содержащую заметки
4. Все поддерживаемые файлы импортируются

## Экспорт заметок

### Форматы экспорта

| Формат | Случай использования |
| ------ | ------- |
| **Markdown** | Обмен, резервные копии, другие приложения |
| **HTML** | Веб-публикация |
| **PDF** | Архивирование, печать |
| **Простой текст** | Простой обмен |
| **JSON** | Интеграция с API |

### Экспорт одной заметки

1. Откройте заметку
2. Нажмите **Файл** → **Экспорт**
3. Выберите формат
4. Выберите расположение
5. Сохраните

### Экспорт нескольких заметок

1. Выберите заметки в боковой панели (Cmd/Ctrl+клик)
2. Кликните правой кнопкой → **Экспортировать выбранные**
3. Выберите формат
4. Выберите расположение
5. Сохраните

### Экспорт всех заметок

1. Откройте **Файл** → **Экспортировать всё**
2. Выберите формат
3. Выберите расположение
4. Дождитесь завершения экспорта

## Руководства по миграции

### Из Obsidian

Classic может импортировать хранилища Obsidian напрямую:

1. Откройте **Файл** → **Импорт**
2. Выберите **Хранилище Obsidian**
3. Перейдите в папку хранилища
4. Выберите для импорта

**Что переносится**

- ✅ Заметки и папки
- ✅ Теги
- ✅ Ссылки (конвертируются в обратные ссылки)
- ✅ Метаданные
- ⚠️ Плагины (не поддерживаются)

### Из Notion

1. Экспортируйте из Notion как Markdown
2. Импортируйте в Classic
3. Проверьте форматирование
4. Исправьте сломанные ссылки

**Советы по миграции из Notion**

- Используйте "Экспорт как Markdown" в Notion
- Проверьте форматирование чекбоксов
- Верифицируйте ссылки на базы данных
- Воссоздайте связанные базы данных вручную

### Из Evernote

1. Экспортируйте из Evernote как HTML
2. Используйте импорт HTML в Classic
3. Проверьте конвертированный контент

**Советы по миграции из Evernote**

- Блокноты становятся папками
- Теги сохраняются
- Вложения импортируются
- Проверьте форматирование

## Лучшие практики

### Перед импортом

- Сделайте резервную копию существующих заметок
- Просмотрите предпросмотр импорта
- Проверьте дубликаты
- Верифицируйте форматирование

### После импорта

- Просмотрите импортированные заметки
- Исправьте проблемы форматирования
- Обновите внутренние ссылки
- Добавьте теги по необходимости

### Регулярный экспорт

- Экспортируйте резервные копии ежемесячно
- Используйте разные форматы для разных целей
- Держите экспорт организованным по дате
- Храните в облаке для внешнего резервного копирования

## Устранение неполадок

### Ошибка импорта

**Распространённые причины**

- Неподдерживаемый формат
- Повреждённый файл
- Проблемы с правами
- Ограничения памяти

**Решения**

1. Попробуйте другой формат
2. Проверьте целостность файла
3. Верифицируйте права на чтение
4. Импортируйте меньше файлов за раз

### Потеря форматирования

**Распространённые причины**

- Сложное форматирование
- Неподдерживаемый синтаксис
- Ограничения конвертации

**Решения**

1. Используйте экспорт Markdown из исходного приложения
2. Упростите форматирование перед импортом
3. Исправьте вручную после импорта

### Отсутствующие вложения

**Распространённые причины**

- Импорт не включал вложения
- Неправильный путь к вложениям
- Ограничения размера

**Решения**

1. Повторно импортируйте с вложениями
2. Проверьте папку вложений
3. Импортируйте вложения отдельно
