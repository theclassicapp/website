# Знакомство с интерфейсом

Интерфейс Classic создан чистым, интуитивным и ненавязчивым. Вот обзор основных компонентов.

## Основной макет

Classic имеет трёхпанельный макет:

1. **Боковая панель** (слева) — Навигация и организация заметок
2. **Редактор** (центр) — Ваше пространство для письма
3. **Предпросмотр/Свойства** (справа) — Опциональные панели

## Боковая панель

Боковая панель содержит несколько ключевых разделов:

### Список заметок

- **Все заметки**: Просмотр всех заметок в одном списке
- **Ежедневные заметки**: Быстрый доступ к сегодняшнему журналу
- **Избранное**: Закладки часто используемых заметок
- **Папки**: Организация заметок в иерархии

### Навигация

- **Строка поиска**: Мгновенный поиск заметок
- **Теги**: Просмотр по тегам
- **Обратные ссылки**: Просмотр связей между заметками

### Действия

- **Новая заметка**: Создать новую заметку
- **Новая папка**: Организация с помощью папок
- **Импорт**: Импорт внешнего контента

## Редактор

Редактор — это место, где вы пишете и форматируете заметки.

### Возможности

- **Поддержка Markdown**: Полный синтаксис Markdown
- **Живой предпросмотр**: Форматированный текст при вводе
- **Подсветка синтаксиса**: Блоки кода подсвечиваются
- **Команды с косой чертой**: Введите `/` для быстрых действий

### Режимы редактора

| Режим | Описание |
|------|-------------|
| **Только редактирование** | Сырой Markdown |
| **Только предпросмотр** | Отрисованный результат |
| **Разделённый вид** | Редактирование и предпросмотр рядом |

### Панель инструментов

Быстрые действия форматирования:

- Жирный, курсив, зачёркнутый
- Заголовки (H1-H6)
- Списки (маркированные, нумерованные, задачи)
- Ссылки и изображения
- Блоки кода
- Таблицы

## Строка состояния

Внизу редактора:

- **Количество слов/символов**: Отслеживание письма
- **Последнее сохранение**: Когда заметка была сохранена
- **Путь заметки**: Где находится заметка
- **Статус синхронизации**: При использовании опциональной синхронизации

## Контекстные меню

Кликните правой кнопкой для контекстных действий:

### Контекстное меню заметки

- Переименовать
- Переместить в папку
- Дублировать
- Удалить
- Экспортировать
- Копировать ссылку

### Контекстное меню редактора

- Вырезать, копировать, вставить
- Параметры форматирования
- Вставить ссылку/изображение
- Найти и заменить

## Навигация с клавиатуры

Навигация по интерфейсу с клавиатурой:

| Действие | Горячая клавиша |
|--------|----------|
| Переключить боковую панель | `Cmd/Ctrl + M` |
| Фокус на поиск | `Cmd/Ctrl + F` |
| Палитра команд | `Cmd/Ctrl + P` |
| Новая заметка | `Cmd/Ctrl + N` |

## Настройка

Настройте интерфейс по своему вкусу:

### Ширина боковой панели

1. Наведите курсор на край боковой панели
2. Перетащите для изменения размера
3. Или установите точную ширину в настройках

### Тема

Переключение между светлым и тёмным режимом:

- Используйте переключатель темы в заголовке
- Или настройте следование системным предпочтениям

### Размер шрифта

1. Откройте настройки
2. Перейдите в Внешний вид
3. Настройте размер шрифта редактора

## Советы

### Режим концентрации

Скрыть все панели для письма без отвлечений:

1. Откройте палитру команд
2. Введите "Режим концентрации"
3. Нажмите Enter

### Быстрый доступ

- Закрепите часто используемые заметки в Избранное
- Используйте горячие клавиши для частых действий
- Настройте панель инструментов в настройках
