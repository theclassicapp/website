# Системный трей

Classic живёт в вашем системном трее для быстрого доступа. Захватывайте мысли мгновенно, не прерывая рабочий процесс.

## Что такое системный трей?

Системный трей (Windows/Linux) или строка меню (macOS) — это место, где Classic работает в фоновом режиме, всегда готовый для быстрого захвата.

### Преимущества

- **Всегда доступен**: Один клик до доступа
- **Быстрый захват**: Мгновенно записывайте мысли
- **Фоновая работа**: Не нужно держать приложение открытым
- **Уведомления**: Получайте напоминания о задачах

## Включение системного трея

### macOS

1. Откройте Classic
2. Перейдите в Настройки → Общие
3. Включите "Показывать в строке меню"
4. Значок Classic появится в строке меню

### Windows

1. Откройте Classic
2. Перейдите в Настройки → Общие
3. Включите "Показывать в системном трее"
4. Значок Classic появится в системном трее

### Linux

1. Откройте Classic
2. Перейдите в Настройки → Общие
3. Включите "Значок системного трея"
4. Зависит от окружения рабочего стола

## Быстрый захват

### Открытие быстрого захвата

Нажмите значок Classic в системном трее:

- **macOS**: Значок в строке меню
- **Windows**: Значок в системном трее
- **Linux**: Значок в системном трее

### Использование быстрого захвата

1. Нажмите значок системного трея
2. Откроется окно быстрого захвата
3. Введите заметку
4. Нажмите Enter для сохранения
5. Окно закроется автоматически

### Горячая клавиша

Откройте быстрый захват из любого места:

| Платформа | Горячая клавиша |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (глобальная) |
| **Windows** | `Ctrl+Shift+N` (глобальная) |
| **Linux** | `Ctrl+Shift+N` (глобальная) |

## Меню системного трея

Кликните правой кнопкой на значок трея для опций:

### Опции меню

| Опция | Действие |
|--------|--------|
| **Новая заметка** | Открыть быстрый захват |
| **Новая ежедневная заметка** | Открыть сегодняшнюю ежедневную заметку |
| **Поиск** | Открыть окно поиска |
| **Открыть Classic** | Открыть главное окно |
| **Настройки** | Открыть настройки |
| **Выход** | Закрыть Classic |

## Настройки трея

### Настройка в настройках

| Параметр | Описание |
|---------|-------------|
| **Показывать значок** | Переключение видимости значка трея |
| **Запускать свёрнутым** | Запускать Classic только в трее |
| **Сворачивать в трей** | Продолжать работу при закрытии окна |
| **Уведомления** | Включить/отключить уведомления |

### Запуск при входе в систему

Настройте автоматический запуск Classic:

1. Откройте настройки
2. Перейдите в Общие
3. Включите "Запускать при входе"
4. Classic будет запускаться при старте

## Уведомления

### Включение уведомлений

1. Откройте настройки
2. Перейдите в Уведомления
3. Переключите "Включить уведомления"

### Типы уведомлений

- **Напоминания**: Запланированные напоминания
- **Ежедневная заметка**: Подсказки для ежедневных заметок
- **Обновления**: Доступно обновление приложения

## Поведение окна

### Поведение при закрытии

Выберите, что происходит при закрытии окна:

- **Свернуть в трей**: Продолжить работу, скрыть окно
- **Выход**: Закрыть приложение
- **Спросить**: Запрос каждый раз

### Поведение при сворачивании

- **Свернуть в трей**: Скрыть в системном трее
- **Свернуть в панель задач**: Обычное сворачивание

## Советы

### Быстрый захват

Используйте глобальную горячую клавишу для самого быстрого захвата:

1. Нажмите `Cmd/Ctrl+Shift+N` из любого места
2. Введите мысль
3. Нажмите Enter
4. Продолжайте работу

### Быстрый доступ

Нажмите значок трея для просмотра:

- Недавние заметки
- Быстрые действия
- Сегодняшняя ежедневная заметка

### Фоновая работа

Держите Classic запущенным для:

- Быстрого запуска
- Быстрого захвата
- Уведомлений
- Автосохранения

## Устранение неполадок

### Значок трея отсутствует

**macOS**

1. Проверьте, не скрыт ли значок
2. Посмотрите в Центре управления (macOS Big Sur+)
3. Перезапустите Classic

**Windows**

1. Проверьте скрытые значки (стрелка в панели задач)
2. Перетащите значок Classic в видимую область
3. Перезапустите Classic

**Linux**

1. Проверьте поддержку окружения рабочего стола
2. Установите расширение системного трея, если нужно
3. Перезапустите Classic

### Глобальная горячая клавиша не работает

- Проверьте конфликты с другими приложениями
- Повторно включите горячую клавишу в настройках
- Перезапустите Classic
- Проверьте разрешения ОС

### Уведомления не отображаются

**macOS**

1. Системные настройки → Уведомления
2. Найдите Classic
3. Включите уведомления

**Windows**

1. Настройки → Система → Уведомления
2. Включите уведомления Classic

**Linux**

1. Проверьте демон уведомлений
2. Установите libnotify, если нужно

### Высокое использование памяти/ЦП

Если значок трея использует слишком много ресурсов:

1. Отключите анимации в настройках
2. Уменьшите частоту уведомлений
3. Проверьте обновления приложения
