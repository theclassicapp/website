# Создание первой заметки

Начните работу с Classic, создав первую заметку. Это просто и интуитивно понятно.

## Создание новой заметки

### Через боковую панель

1. Нажмите кнопку **+** в боковой панели
2. Новая заметка создана
3. Начните вводить содержимое

### С помощью горячей клавиши

| Платформа | Горячая клавиша |
| -------- | -------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Через палитру команд

1. Откройте палитру команд (`Cmd+P` или `Ctrl+P`)
2. Введите "Новая заметка"
3. Нажмите Enter

## Написание заметки

### Базовое редактирование

Просто начните вводить! Classic автоматически сохраняет по мере написания.

### Добавление заголовка

Первая строка заметки становится заголовком. Используйте заголовок для лучших результатов:

```markdown
# Моя первая заметка

Это содержимое моей заметки.
```

### Форматирование текста

Используйте Markdown для форматирования текста:

- **Жирный**: `**текст**` или `Cmd/Ctrl+B`
- **Курсив**: `*текст*` или `Cmd/Ctrl+I`
- **Зачёркнутый**: `~~текст~~`
- **Код**: `` `код` ``

## Сохранение

Classic автоматически сохраняет заметки при вводе. Не нужно сохранять вручную.

### Ручное сохранение

При необходимости можно принудительно сохранить:

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## Организация заметки

### Добавление тегов

Добавьте теги для лёгкого поиска:

1. Введите `#`, затем название тега
2. Или используйте кнопку тега на панели инструментов
3. Поддерживается несколько тегов

Пример:

```markdown
# Заметки с встречи

Обсудили планирование Q4 #работа #встречи
```

### Перемещение в папку

1. Кликните правой кнопкой на заметке в боковой панели
2. Выберите "Переместить в папку"
3. Выберите или создайте папку назначения

### Закрепление важных заметок

1. Кликните правой кнопкой на заметке
2. Выберите "Закрепить сверху"
3. Заметка появится вверху списка

## Предпросмотр заметки

Посмотрите, как выглядит отформатированная заметка:

- Нажмите кнопку **Предпросмотр**
- Или используйте `Cmd/Ctrl+Shift+P`
- Переключайтесь между режимами редактирования и предпросмотра

## Следующие шаги

- Настройте [ежедневные заметки](/help/daily-notes)

## Советы

### Заголовки заметок

- Делайте заголовки описательными
- Первая строка = заголовок заметки
- Используйте H1 (`#`) для ясности

### Автосохранение

- Не беспокойтесь о потере работы
- Изменения сохраняются автоматически
- Проверьте строку состояния для статуса сохранения

### Быстрый захват

Нужно быстро что-то записать?

- Используйте быстрый захват (`Cmd/Ctrl+Shift+N`)
- Мгновенно создаёт заметку
- Отлично подходит для мимолётных мыслей
