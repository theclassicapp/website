# Языки и локализация

Classic говорит на вашем языке. Используйте приложение на предпочитаемом языке и пишите заметки на любом языке по вашему выбору.

## Поддерживаемые языки

Интерфейс Classic доступен на нескольких языках:

| Язык | Код |
|----------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| И другие... | |

## Изменение языка интерфейса

### Через настройки

1. Откройте настройки (`Cmd+,` или `Ctrl+,`)
2. Перейдите в **Общие** → **Язык**
3. Выберите предпочтительный язык
4. Перезапустите Classic для применения

### Через палитру команд

1. Откройте палитру команд (`Cmd+P` или `Ctrl+P`)
2. Введите "Язык"
3. Выберите предпочтительный язык
4. Перезапустите для применения изменений

## Письмо на нескольких языках

Classic поддерживает написание заметок на любом языке.

### Поддержка Unicode

- Полная поддержка Unicode
- Поддержка RTL (справа налево) языков
- CJK (китайский, японский, корейский) символы работают идеально
- Поддержка эмодзи

### Определение языка

Classic может определять язык заметок:

- Автоопределение для проверки орфографии
- Настройка языка для каждой заметки
- Поддержка смешанных языков

## Проверка орфографии

### Включение проверки орфографии

1. Откройте настройки
2. Перейдите в **Редактор** → **Проверка орфографии**
3. Включите "Проверка орфографии"

### Языки проверки орфографии

1. Откройте настройки
2. Перейдите в **Редактор** → **Проверка орфографии**
3. Нажмите "Языки"
4. Выберите языки для проверки

### Добавление слов в словарь

1. Кликните правой кнопкой на отмеченном слове
2. Выберите "Добавить в словарь"
3. Слово больше не будет отмечаться

## Поддержка RTL

Для арабского, иврита и других RTL-языков.

### Автоматическое определение RTL

Classic автоматически определяет RTL-контент:

- Направление текста адаптируется
- Движение курсора адаптируется
- Выравнивание изменяется

### Ручное переключение RTL

Принудительное направление RTL или LTR:

1. Откройте палитру команд
2. Введите "RTL" или "LTR"
3. Выберите предпочтение

## Методы ввода

Classic работает с системными методами ввода.

### Методы ввода CJK

- Работает с Pinyin, Zhuyin и т.д.
- Поддерживает японский IME
- Корейские методы ввода поддерживаются

### Специальные символы

Легко вставляйте специальные символы:

- Используйте системный выборщик символов
- Копируйте/вставляйте из карт символов
- Используйте ввод Unicode

## Форматы даты и времени

Даты адаптируются к вашей локали.

### Опции формата

| Параметр | Опции |
|---------|---------|
| Формат даты | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| Формат времени | 12-часовой, 24-часовой |
| Первый день недели | Воскресенье, Понедельник, Суббота |

Настройка в Настройки → Общие → Дата и время

## Форматы чисел

Числа отображаются в соответствии с вашей локалью:

- **Десятичный разделитель**: `.` или `,`
- **Разделитель тысяч**: `,` или `.` или пробел
- **Символы валюты**: На основе региона

## Раскладки клавиатуры

Classic работает с любой раскладкой клавиатуры:

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Пользовательские раскладки

## Настройки локализации

### В Настройки → Общие → Язык

| Параметр | Описание |
|---------|-------------|
| **Язык интерфейса** | Язык UI приложения |
| **Формат даты** | Как отображаются даты |
| **Формат времени** | 12 или 24 часа |
| **Первый день недели** | День начала календаря |
| **Формат чисел** | Десятичные/тысячные разделители |

## Советы

### Заметки на смешанных языках

Пишите на нескольких языках в одной заметке:

```markdown
# Международный проект

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Шрифты для конкретных языков

Некоторые языки выглядят лучше с определёнными шрифтами:

1. Откройте настройки
2. Перейдите в Внешний вид → Шрифты
3. Установите шрифты для разных скриптов

### Быстрое переключение языков

Создайте горячую клавишу:

1. Откройте Настройки → Горячие клавиши
2. Найдите "Переключить язык"
3. Установите предпочтительную горячую клавишу

## Устранение неполадок

### Язык недоступен

- Проверьте обновления приложения
- Язык может быть в разработке
- Внесите переводы на GitHub

### Символы не отображаются

- Установите необходимые шрифты
- Проверьте настройки шрифтов
- Попробуйте другой шрифт

### Проверка орфографии не работает

- Убедитесь, что проверка орфографии включена
- Добавьте язык в проверку орфографии
- Перезапустите Classic

### RTL работает некорректно

- Убедитесь, что определение RTL включено
- Попробуйте ручное переключение RTL
- Проверьте проблемы смешанного контента
