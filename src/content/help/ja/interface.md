# インターフェースについて

Classicのインターフェースは、シンプルで直感的、そして邪魔にならないよう設計されています。メインコンポーネントのツアーをご紹介します。

## メインレイアウト

Classicは3パネルレイアウトを採用しています

1. **サイドバー**（左） - ナビゲーションとノート整理
2. **エディター**（中央） - 執筆スペース
3. **プレビュー/プロパティ**（右） - オプションパネル

## サイドバー

サイドバーにはいくつかの重要なセクションが含まれています

### ノートリスト

- **すべてのノート**: すべてのノートを一覧表示
- **デイリーノート**: 今日のジャーナルに素早くアクセス
- **お気に入り**: よく使うノートにスター
- **フォルダー**: 階層でノートを整理

### ナビゲーション

- **検索バー**: ノートを即座に検索
- **タグ**: タグでブラウズ
- **バックリンク**: ノート間のつながりを表示

### アクション

- **新規ノート**: 新しいノートを作成
- **新規フォルダー**: フォルダーで整理
- **インポート**: 外部コンテンツを取り込む

## エディター

エディターはノートを書いてフォーマットする場所です。

### 機能

- **Markdownサポート**: 完全なMarkdown構文
- **ライブプレビュー**: 入力中にフォーマットされたテキストを表示
- **シンタックスハイライト**: コードブロックがハイライト表示
- **スラッシュコマンド**: `/` を入力してクイックアクション

### エディターモード

| モード | 説明 |
|------|-------------|
| **編集のみ** | 生のMarkdownを表示 |
| **プレビューのみ** | レンダリングされた出力を表示 |
| **分割ビュー** | 編集とプレビューを並べて表示 |

### ツールバー

クイックフォーマットアクション

- 太字、斜体、取り消し線
- 見出し (H1-H6)
- リスト（箇条書き、番号付き、タスク）
- リンクと画像
- コードブロック
- テーブル

## ステータスバー

エディターの下部に表示

- **文字数/文字カウント**: 執筆を追跡
- **最終保存**: ノートが最後に保存された時刻
- **ノートパス**: ノートの場所
- **同期ステータス**: オプションの同期を使用している場合

## コンテキストメニュー

右クリックでコンテキストアクション

### ノートコンテキストメニュー

- 名前変更
- フォルダーに移動
- 複製
- 削除
- エクスポート
- リンクをコピー

### エディターコンテキストメニュー

- 切り取り、コピー、貼り付け
- フォーマットオプション
- リンク/画像を挿入
- 検索と置換

## キーボードナビゲーション

キーボードでインターフェースをナビゲート

| アクション | ショートカット |
|--------|----------|
| サイドバー切り替え | `Cmd/Ctrl + M` |
| 検索にフォーカス | `Cmd/Ctrl + F` |
| コマンドパレット | `Cmd/Ctrl + P` |
| 新規ノート | `Cmd/Ctrl + N` |

## カスタマイズ

インターフェースをお好みに調整

### サイドバー幅

1. サイドバーの端にホバー
2. ドラッグしてリサイズ
3. または設定で正確な幅を設定

### テーマ

ライトモードとダークモードを切り替え

- ヘッダーのテーマ切り替えを使用
- またはシステム設定に従うよう設定

### フォントサイズ

1. 設定を開く
2. 外観に移動
3. エディターフォントサイズを調整

## ヒント

### フォーカスモード

集中して書くためにすべてのパネルを非表示

1. コマンドパレットを開く
2. 「フォーカスモード」と入力
3. Enterを押す

### クイックアクセス

- よく使うノートをお気に入りにピン留め
- 一般的なアクションにキーボードショートカットを使用
- 設定でツールバーをカスタマイズ
