# スラッシュコマンド

スラッシュコマンドは、キーボードから離れることなくフォーマットや特別なコンテンツを素早く挿入する強力な方法です。エディターで `/` を入力してコマンドパレットを開きます。

## 利用可能なコマンド

### 見出し

| コマンド | 結果 |
| ------- | ------ |
| `/h1` | レベル1見出し |
| `/h2` | レベル2見出し |
| `/h3` | レベル3見出し |
| `/h4` | レベル4見出し |
| `/h5` | レベル5見出し |
| `/h6` | レベル6見出し |

### テキストフォーマット

| コマンド | 結果 |
| ------- | ------ |
| `/bold` | **太字テキスト** |
| `/italic` | *斜体テキスト* |
| `/strikethrough` | ~~取り消し線~~ |
| `/code` | インラインコードブロック |
| `/quote` | 引用 |

### リスト

| コマンド | 結果 |
| ------- | ------ |
| `/bullet` | • 箇条書きリスト |
| `/numbered` | 1. 番号付きリスト |
| `/todo` | - [ ] Todo項目 |
| `/check` | - [x] 完了項目 |

### コンテンツの挿入

| コマンド | 結果 |
| ------- | ------ |
| `/image` | URLまたはファイルから画像を挿入 |
| `/link` | ハイパーリンクを挿入 |
| `/table` | テーブルを挿入 |
| `/hr` | 水平線 |
| `/code` | 言語選択付きコードブロック |

### 特殊ブロック

| コマンド | 結果 |
| ------- | ------ |
| `/mermaid` | Mermaidダイアグラムブロック |
| `/katex` | KaTeX数式ブロック |
| `/callout` | コールアウト/注釈ブロック |

## スラッシュコマンドの使用

1. エディターで `/` を入力
2. 利用可能なすべてのコマンドのメニューが表示
3. 入力してコマンドをフィルタリング
4. `Enter` を押すかクリックして挿入

## プロのヒント

- **クイックアクセス**: `/` を入力してコマンド名を入力開始
- **あいまい検索**: 正確なコマンド名を入力する必要はありません - Classicが最も近い一致を見つけます
- **キーボードナビゲーション**: 矢印キーでメニューをナビゲート
- **最近のコマンド**: 頻繁に使用されるコマンドがリストのトップに表示
