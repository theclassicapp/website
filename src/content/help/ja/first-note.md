# 最初のノートを作成

Classicで最初のノートを作成して始めましょう。シンプルで直感的です。

## 新規ノートの作成

### サイドバーを使用

1. サイドバーの**+**ボタンをクリック
2. 新しいノートが作成される
3. コンテンツの入力を開始

### キーボードショートカットを使用

| プラットフォーム | ショートカット |
| -------- | -------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### コマンドパレットを使用

1. コマンドパレットを開く（`Cmd+P` または `Ctrl+P`）
2. 「新規ノート」と入力
3. Enterを押す

## ノートを書く

### 基本的な編集

すぐに入力を開始！Classicは書きながら自動保存します。

### タイトルを追加

ノートの最初の行がタイトルになります。見出しを使用すると最適です

```markdown
# 初めてのノート

これはノートの内容です。
```

### テキストのフォーマット

Markdownを使用してテキストをフォーマット

- **太字**: `**テキスト**` または `Cmd/Ctrl+B`
- **斜体**: `*テキスト*` または `Cmd/Ctrl+I`
- **取り消し線**: `~~テキスト~~`
- **コード**: `` `コード` ``

## 保存

Classicは入力中にノートを自動保存します。手動で保存する必要はありません。

### 手動保存

必要に応じて、強制保存が可能

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## ノートの整理

### タグを追加

後で簡単に見つけられるようノートにタグを付け

1. `#` に続けてタグ名を入力
2. またはツールバーのタグボタンを使用
3. 複数のタグがサポートされています

例

```markdown
# 会議ノート

Q4計画について議論 #仕事 #会議
```

### フォルダーに移動

1. サイドバーのノートを右クリック
2. 「フォルダーに移動」を選択
3. 移動先フォルダーを選択または作成

### 重要なノートをピン留め

1. ノートを右クリック
2. 「トップにピン留め」を選択
3. ノートがリストのトップに表示

## ノートをプレビュー

フォーマットされたノートの見た目を確認

- **プレビュー**ボタンをクリック
- または `Cmd/Ctrl+Shift+P` を使用
- 編集モードとプレビューモードを切り替え

## 次のステップ

最初のノートを作成したので

- [Markdown](/help/markdown)についてもっと学ぶ
- [タグ](/help/tags)で整理
- [デイリーノート](/help/daily-notes)を設定
- [キーボードショートカット](/help/keyboard-shortcuts)を探索

## ヒント

### ノートタイトル

- タイトルは説明的に
- 最初の行 = ノートタイトル
- 明確にするためにH1（`#`）を使用

### 自動保存

- 作業を失う心配はありません
- 変更は自動的に保存
- ステータスバーで保存状態を確認

### クイックキャプチャ

素早くメモする必要がある？

- クイックキャプチャを使用（`Cmd/Ctrl+Shift+N`）
- ノートを即座に作成
- ちょっとしたアイデアに最適
