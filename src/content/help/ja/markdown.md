# Markdownリファレンス

Classicはライブプレビュー付きの完全なMarkdown構文をサポートしています。サポートされているすべてのフォーマットオプションの包括的なリファレンスです。

## 基本的なフォーマット

| 構文 | 結果 |
|-------|--------|
| `**太字**` | **太字** |
| `*斜体*` | *斜体* |
| `~~取り消し線~~` | ~~取り消し線~~ |
| `==見出し 1==` | 見出し 1 |
| `### 見出し 2` | ### 見出し 2 |
| `#### 見出し 3` | #### 見出し 3 |

## リンク

```markdown
[インラインリンク](https://classic.app)

[参照スタイルリンク][https://classic.app]
```

## リスト

```markdown
- 項目 1
- 項目 2
  - ネストされた項目 2a
    - ネストされた項目 2a
- 項目 3

1. 最初の項目
2. 2番目の項目
3. 3番目の項目
```

## コードブロック

インライン `コード`:

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

言語付きコードブロック:

```javascript
```python
def greet(name):
    return f"Hello, {name}!"

print(greet("Classic"))
```
