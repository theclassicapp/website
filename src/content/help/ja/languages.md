# 言語とローカライズ

Classicはあなたの言語を話します。好みの言語でアプリを使用し、任意の言語でノートを書くことができます。

## サポートされている言語

Classicのインターフェースは複数の言語で利用可能

| 言語 | コード |
|----------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| その他... | |

## インターフェース言語の変更

### 設定から

1. 設定を開く（`Cmd+,` または `Ctrl+,`）
2. **一般** → **言語**に移動
3. 好みの言語を選択
4. 適用するためにClassicを再起動

### コマンドパレットから

1. コマンドパレットを開く（`Cmd+P` または `Ctrl+P`）
2. 「言語」と入力
3. 好みの言語を選択
4. 変更を適用するために再起動

## 複数言語での執筆

Classicは任意の言語でのノート執筆をサポート

### Unicodeサポート

- 完全なUnicodeサポート
- RTL（右から左）言語をサポート
- CJK（中国語、日本語、韓国語）文字は完璧に動作
- 絵文字サポート

### 言語検出

Classicはノートの言語を検出可能

- スペルチェックの自動検出
- ノートごとの言語設定
- 混合言語サポート

## スペルチェック

### スペルチェックの有効化

1. 設定を開く
2. **エディター** → **スペルチェック**に移動
3. 「スペルチェック」を有効化

### スペルチェック言語

1. 設定を開く
2. **エディター** → **スペルチェック**に移動
3. 「言語」をクリック
4. チェックする言語を選択

### 辞書に単語を追加

1. フラグ付きの単語を右クリック
2. 「辞書に追加」を選択
3. 単語は再度フラグされません

## RTLサポート

アラビア語、ヘブライ語、その他のRTL言語用

### 自動RTL検出

ClassicはRTLコンテンツを自動的に検出

- テキストの方向が調整
- カーソルの動きが適応
- 配置が変更

### 手動RTL切り替え

RTLまたはLTRの方向を強制

1. コマンドパレットを開く
2. 「RTL」または「LTR」と入力
3. 好みを選択

## 入力メソッド

Classicはシステムの入力メソッドで動作

### CJK入力メソッド

- Pinyin、Zhuyinなどで動作
- 日本語IMEをサポート
- 韓国語入力メソッドをサポート

### 特殊文字

特殊文字を簡単に挿入

- システムの文字ピッカーを使用
- 文字マップからコピー/貼り付け
- Unicode入力を使用

## 日付と時刻の形式

日付はロケールに適応

### 形式オプション

| 設定 | オプション |
|---------|---------|
| 日付形式 | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| 時刻形式 | 12時間制, 24時間制 |
| 週の最初の日 | 日曜日, 月曜日, 土曜日 |

設定 → 一般 → 日付と時刻で構成

## 数値形式

数値はロケールに従って表示

- **小数点**: `.` または `,`
- **桁区切り**: `,` または `.` またはスペース
- **通貨記号**: 地域に基づく

## キーボードレイアウト

Classicは任意のキーボードレイアウトで動作

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- カスタムレイアウト

## ローカライズ設定

### 設定 → 一般 → 言語

| 設定 | 説明 |
|---------|-------------|
| **インターフェース言語** | アプリUI言語 |
| **日付形式** | 日付の表示方法 |
| **時刻形式** | 12または24時間 |
| **週の最初の日** | カレンダーの開始日 |
| **数値形式** | 小数/桁区切り |

## ヒント

### 混合言語ノート

1つのノートで複数の言語で書く

```markdown
# 国際プロジェクト

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### 言語固有のフォント

一部の言語は特定のフォントでより良く見える

1. 設定を開く
2. 外観 → フォントに移動
3. 異なるスクリプトのフォントを設定

### 言語を素早く切り替える

キーボードショートカットを作成

1. 設定 → ショートカットを開く
2. 「言語を切り替え」を見つける
3. 好みのショートカットを設定

## トラブルシューティング

### 言語が利用できない

- アプリの更新を確認
- 言語は開発中の可能性
- GitHubで翻訳に貢献

### 文字が表示されない

- 必要なフォントをインストール
- フォント設定を確認
- 別のフォントを試す

### スペルチェックが動作しない

- スペルチェックが有効かどうか確認
- スペルチェックに言語を追加
- Classicを再起動

### RTLが正しく動作しない

- RTL検出がオンになっていることを確認
- 手動RTL切り替えを試す
- 混合コンテンツの問題を確認
