# 検索

Classicの強力な検索で任意のノートを即座に見つけます。全文検索、フィルター、演算子で必要なものを正確に特定できます。

## 基本検索

### 検索を開く

| プラットフォーム | ショートカット |
|----------|----------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

またはサイドバーの検索バーをクリック。

### 仕組み

1. 検索を開く
2. 入力を開始
3. 結果が即座に表示
4. クリックしてノートを開く

## 検索機能

### 全文検索

Classicは以下を検索

- ノートタイトル
- ノート内容
- タグ
- ファイル名

### リアルタイム結果

入力中に結果が更新

- Enterを押す必要なし
- 即座のフィードバック
- あいまいマッチングが有効

### 検索ハイライト

見つかった用語が結果でハイライト

- マッチ周辺のコンテキストを表示
- 関連ノートを素早く特定

## 検索演算子

### 完全一致

完全なフレーズには引用符を使用

```
"プロジェクト計画"
"会議ノート"
```

### タグ検索

特定のタグを持つノートを見つける

```
tag:仕事
tag:会議
tag:緊急
```

### 用語の除外

除外に `-` を使用

```
プロジェクト -アーカイブ
会議 -キャンセル
```

### OR検索

いずれかの用語を持つノートを見つける

```
プロジェクト OR イニシアチブ
```

### 日付範囲

日付で検索

```
created:2024-01-01..2024-12-31
modified:先週
```

### ファイルタイプ

拡張子でフィルタリング

```
ext:md
ext:txt
```

## 高度な検索

### 演算子の組み合わせ

複雑なクエリを構築

```
tag:仕事 -tag:アーカイブ created:先週
"会議" tag:プロジェクト-alpha
```

### パス検索

フォルダー内で検索

```
path:プロジェクト/
path:アーカイブ/
```

### タイトル検索

タイトルのみを検索

```
title:ロードマップ
title:会議
```

### コンテンツ検索

コンテンツのみを検索（タイトルを除外）

```
content:重要
content:期限
```

## 検索フィルター

### タグで

1. 検索を入力
2. タグをクリックしてフィルタリング
3. タグ付きノートに結果が絞り込まれる

### 日付で

更新日でフィルタリング

- 今日
- 今週
- 今月
- カスタム範囲

### フォルダーで

検索をフォルダーに制限

1. フォルダーを右クリック
2. 「フォルダー内で検索」を選択
3. 結果がそのフォルダーに制限

## 検索結果

### ソート

結果を以下でソート

- **関連性**（デフォルト）
- **更新日**（新しい順）
- **作成日**（新しい順）
- **タイトル**（アルファベット順）

### 結果プレビュー

結果にホバーしてプレビュー

- 最初の数行を表示
- マッチした用語をハイライト
- タグを表示

### コンテキスト

マッチが発生した場所を表示

- 周囲のテキストを表示
- マッチをハイライト
- クリックして場所にジャンプ

## 検索履歴

### 最近の検索

過去の検索にアクセス

1. 検索バーをクリック
2. 最近の検索を表示
3. クリックして再実行

### 履歴のクリア

1. 設定を開く
2. プライバシーに移動
3. 「検索履歴をクリア」をクリック

## ヒント

### 具体的な用語を使用

より具体的 = より良い結果

```
✅ "Q4マーケティング予算"
❌ 予算
```

### タグと組み合わせる

結果を素早く絞り込む

```
プロジェクト tag:緊急
```

### 除外を使用

ノイズをフィルタリング

```
会議 -tag:アーカイブ
```

### 頻繁な検索を保存

保存した検索を作成

1. 検索を入力
2. 「検索を保存」をクリック
3. 名前を付ける
4. サイドバーからアクセス

## キーボードナビゲーション

| アクション | ショートカット |
|--------|----------|
| 検索を開く | `Cmd/Ctrl+F` |
| 次の結果 | `Enter` または `Down` |
| 前の結果 | `Shift+Enter` または `Up` |
| 検索を閉じる | `Escape` |

## 検索設定

### 設定でカスタマイズ

| 設定 | オプション |
|---------|---------|
| 大文字小文字を区別 | オン/オフ |
| タイトルを検索 | オン/オフ |
| コンテンツを検索 | オン/オフ |
| タグを検索 | オン/オフ |
| アーカイブを含める | オン/オフ |

## 使用例

### 古いノートを見つける

```
"会議" date:昨年
```

### プロジェクト研究

```
path:プロジェクト/ tag:研究
```

### アクションアイテム

```
"TODO" OR "FIXME" -tag:完了
```

### 最近の作業

```
modified:今週
```

## トラブルシューティング

### 結果が見つからない

- スペルを確認
- フィルターを削除
- より広い用語を試す
- ノートがアーカイブされていないか確認

### 検索が遅い

- 大きなノートコレクションは時間がかかる可能性
- 大文字小文字の区別を無効にする
- 検索範囲を減らす

### ノートが見つからない

- ノートが別のフォルダーにある可能性
- タグが正しいか確認
- ノートがアーカイブされていないか確認
