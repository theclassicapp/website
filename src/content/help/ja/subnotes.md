# サブノート

サブノートを使用すると、Classic内でネストされた階層的なノートを作成できます。複雑なトピックを管理しやすい部分に整理できます。

## サブノートとは？

サブノートはノート内のノート - 階層的なコンテンツ構造を作成する方法です。

### 利点

- **整理**: 複雑なトピックを小さな部分に分割
- **ネスト**: 複数レベルの階層を作成
- **ナビゲーション**: 関連ノート間を簡単に移動
- **柔軟性**: 必要に応じて展開・折りたたみ

## サブノートの作成

### 方法1: インデント

親ノートの下にコンテンツをインデントしてサブノートを作成

1. 親ノートを作成
2. 新しいノートを作成
3. 新しいノートを親にドラッグ
4. サブノートになります

### 方法2: コンテキストメニュー

1. ノートを右クリック
2. 「新規サブノート」を選択
3. ネストされたノートが作成されます

### 方法3: キーボードショートカット

1. ノートを選択
2. `Cmd/Ctrl+Shift+N` を押す
3. サブノートが作成されます

## サブノートの管理

### 展開と折りたたみ

サブノートのあるノートの横の矢印をクリック

- **展開**: すべてのネストされたノートを表示
- **折りたたみ**: ネストされたノートを非表示

### ナビゲーション

- クリックしてサブノートを開く
- 矢印キーで階層をナビゲート
- パンくずで現在地を表示

### サブノートの移動

ドラッグアンドドロップで再編成

1. サブノートをクリックして保持
2. 新しい場所にドラッグ
3. ドロップして位置を変更

### メインノートに変換

サブノートをトップレベルに昇格

1. サブノートを右クリック
2. 「ルートに移動」を選択
3. ノートがメインノートになります

## サブノートの深さ

Classicは複数レベルのネストをサポート

```
📁 プロジェクト
  📄 概要
  📁 調査
    📄 ユーザーインタビュー
    📄 アンケート結果
    📁 分析
      📄 主要な発見
      📄 推奨事項
  📁 計画
    📄 タイムライン
    📄 リソース
```

### ベストプラクティス

- **深さを制限**: 明確にするために最大3-4レベル
- **論理的なグループ化**: 関連コンテンツをまとめる
- **明確な命名**: サブノートに説明的な名前を使用

## サブノートとのリンク

### バックリンク

サブノートは自動的に親ノートへのバックリンクを作成

- 「リンクされた参照」セクションを表示
- 関連ノート間をナビゲート
- ノートの関係を理解

### ウィキリンク

どこからでもサブノートにリンク

```markdown
詳細は [[プロジェクト/調査/分析]] を参照。
```

## ヒント

### 命名規則

明確で一貫した命名を使用

```
✅ 良い:
プロジェクト/要件/機能的
プロジェクト/要件/非機能的

❌ 避ける:
プロジェクト/req/func
プロジェクト/rq/nf
```

### パンくずナビゲーション

パンくずを使用して現在地を理解

```
プロジェクト > 調査 > 分析
```

### 検索

検索には自動的にサブノートが含まれます

- すべてのレベルでコンテンツを検索
- ノートの深さでフィルタリング
- クイックナビゲーション

### テンプレート

サブノートにテンプレートを適用

1. サブノートを作成
2. スラッシュコマンド `/template` を使用
3. テンプレートを選択

## キーボードショートカット

| アクション | ショートカット |
|--------|----------|
| 新規サブノート | `Cmd/Ctrl+Shift+N` |
| すべて展開 | `Cmd/Ctrl+Down` |
| すべて折りたたみ | `Cmd/Ctrl+Up` |
| 上に移動 | `Opt/Alt+Up` |
| 下に移動 | `Opt/Alt+Down` |
| 昇格 | `Opt/Alt+Left` |
| 降格 | `Opt/Alt+Right` |

## トラブルシューティング

### サブノートが表示されない

- 親が折りたたまれていないか確認
- ノートが移動されていないか確認
- サイドバーを更新

### サブノートを作成できない

- 親ノートが存在することを確認
- 権限を確認
- コンテキストメニューの方法を試す

### サブノートが見つからない

検索を使用して見つからないノートを見つける

1. 検索を開く（`Cmd/Ctrl+F`）
2. ノート名を入力
3. 結果でフルパスを表示
