# コマンドパレット

コマンドパレットはClassicのすべてのアクションへの入り口です。簡単なキーボードショートカットで任意の機能に即座にアクセスできます。

## コマンドパレットを開く

| プラットフォーム | ショートカット |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## 使い方

1. コマンドパレットを開く（`Cmd+P` または `Ctrl+P`）
2. 入力してコマンドを検索
3. 矢印キーでナビゲート
4. `Enter` を押して実行

## コマンドカテゴリ

### ファイル操作

- **新規ノート**: 新しいノートを作成
- **新規デイリーノート**: 今日のデイリーノートを作成/開く
- **ノートを開く**: 既存のノートを検索して開く
- **ノートを保存**: 現在のノートを保存
- **ノートを削除**: 現在のノートを削除
- **ノートの名前変更**: 現在のノートの名前を変更

### ナビゲーション

- **ノートに移動**: 任意のノートにジャンプ
- **デイリーノートに移動**: 特定の日付を開く
- **タグに移動**: タグでノートをブラウズ
- **戻る**: 履歴で戻る
- **進む**: 履歴で進む

### エディター

- **プレビュー切り替え**: 編集モードとプレビューモードを切り替え
- **サイドバー切り替え**: サイドバーの表示/非表示
- **フォーカスモード**: 集中して書くモードに入る
- **日付を挿入**: 現在の日付/時刻を追加
- **テンプレートを挿入**: テンプレートを挿入

### フォーマット

- **太字**: テキストを太字に
- **斜体**: テキストを斜体に
- **取り消し線**: テキストに取り消し線
- **コード**: コードとしてフォーマット
- **リンク**: リンクを挿入
- **見出し**: 見出しフォーマットを適用

### 整理

- **タグを追加**: 現在のノートにタグを付ける
- **フォルダーに移動**: ノートをフォルダーに移動
- **ノートをアーカイブ**: 現在のノートをアーカイブ
- **ノートをピン留め**: ノートをリストのトップにピン留め

### 表示

- **テーマ切り替え**: ライト/ダークを切り替え
- **ズームイン**: フォントサイズを増やす
- **ズームアウト**: フォントサイズを減らす
- **ズームリセット**: デフォルトサイズにリセット

## ヒント

### クイックアクセス

- コマンドはあいまい検索されます
- 最近のコマンドが最初に表示
- 頻繁に使用するコマンドをピン留め

### キーボードナビゲーション

- `↑`/`↓` - 結果をナビゲート
- `Enter` - コマンドを実行
- `Escape` - パレットを閉じる
- `Tab` - カテゴリをサイクル

### カスタムコマンド

1. 設定を開く
2. コマンドパレットに移動
3. カスタムコマンドを追加
4. キーボードショートカットを定義

## 頻繁に使用するコマンド

よく使用するいくつかのコマンド

| コマンド | 機能 |
| ------- | ------------- |
| `新規ノート` | 空白ノートを作成 |
| `ノートを開く` | すべてのノートを検索 |
| `新規デイリーノート` | 今日のジャーナル |
| `プレビュー切り替え` | プレビュー vs 編集 |
| `タグを追加` | 現在のノートにタグ |
| `クイックキャプチャ` | グローバルキャプチャ |

## コマンドのカスタマイズ

### ショートカットの変更

1. 設定を開く（`Cmd+,` または `Ctrl+,`）
2. キーボードショートカットに移動
3. コマンドを見つける
4. 現在のショートカットをクリック
5. 新しいショートカットを押す
6. 保存

### カスタムコマンドの作成

上級ユーザーはカスタムコマンドを作成可能

1. 設定を開く
2. コマンドに移動
3. 「カスタムコマンドを追加」をクリック
4. コマンド名を定義
5. アクションを追加
6. ショートカットを設定（オプション）
