# タグ

タグはClassicでノートを整理・分類する強力な方法です。関連コンテンツを即座に見つけられます。

## タグの作成

### インラインタグ

ノートの内容に直接タグを追加

```markdown
# 会議ノート

Q4ロードマップについて議論 #仕事 #会議 #計画
```

### タグボタンを使用

1. ツールバーのタグアイコンをクリック
2. タグ名を入力
3. Enterを押す

### キーボードショートカット

| プラットフォーム | ショートカット |
|----------|----------|
| **macOS** | `Cmd+T` |
| **Windows/Linux** | `Ctrl+T` |

## タグ構文

### 基本タグ

```markdown
#プロジェクト #仕事 #重要
```

### ネストされたタグ

`/` でタグの階層を作成

```markdown
#仕事/会議
#仕事/プロジェクト
#個人/趣味/読書
```

### 複数単語タグ

ハイフンまたはアンダースコアを使用

```markdown
#プロジェクト-alpha
#会議_ノート
#q4-計画
```

## タグの管理

### すべてのタグを表示

1. サイドバーを開く
2. 「タグ」セクションを見つける
3. すべてのタグを表示

### タグクラウド

最も使用されるタグを視覚化

- 大きいタグ = より多くのノート
- 任意のタグをクリックしてフィルタリング
- タグごとのノート数を表示

### タグの名前変更

1. タグを右クリック
2. 「名前変更」を選択
3. 新しい名前を入力
4. すべてのノートが自動的に更新

### タグの削除

1. タグを右クリック
2. 「削除」を選択
3. 削除を確認
4. タグがすべてのノートから削除

## タグで検索

### タグパネル

サイドバーのタグをクリックしてノートをフィルタリング

### 検索構文

検索で `tag:` を使用

```
tag:仕事
tag:仕事/会議
```

### タグの組み合わせ

複数のタグを持つノートを見つける

```
tag:仕事 tag:緊急
```

### タグの除外

タグを持たないノートを見つける

```
-tag:アーカイブ
```

## タグのベストプラクティス

### シンプルに保つ

- ノートあたり3-5個のタグを使用
- 命名を一貫させる
- タグの乱立を避ける

### 階層を使用

関連タグを整理

```
#仕事
  #仕事/会議
  #仕事/プロジェクト
  #仕事/期限

#個人
  #個人/健康
  #個人/財務
```

### 規約を確立

自分用のタグ標準を定義

- **ステータス**: `#todo` `#進行中` `#完了`
- **優先度**: `#緊急` `#高` `#低`
- **タイプ**: `#会議` `#参照` `#アイデア`

## タグの提案

`#` を入力すると、Classicは既存のタグを提案

- 最近のタグが最初に表示
- あいまいマッチングをサポート
- Tabを押して提案を受け入れる

## タグの統計

設定でタグの使用状況を表示

- 最も使用されるタグ
- 未使用のタグ
- 時間経過によるタグの傾向

## 使用例

### プロジェクト管理

```markdown
# プロジェクトキックオフ #プロジェクト-alpha #会議 #2024-01-15

参加者: チームAlpha
ステータス: #todo
```

### ナレッジベース

```markdown
# JavaScriptクロージャ #プログラミング #javascript #参照 #上級

クロージャは...にアクセスできる関数です
```

### デイリーノート

```markdown
# 2024-01-15

- 朝のスタンドアップ #仕事 #会議
- コードレビュー #仕事 #開発
- ジムセッション #個人 #健康
```

### 読書ノート

```markdown
# アトミックハビットのノート #読書 #本 #生産性

本からの重要なポイント...
```

## キーボードショートカット

| アクション | ショートカット |
|--------|----------|
| タグを追加 | `Cmd/Ctrl+T` |
| タグを検索 | `Cmd/Ctrl+F`、そして `tag:` を入力 |
| タグパネルを表示 | `Cmd/Ctrl+Shift+T` |

## ヒント

### 一貫した命名

規約に従う

```
✅ #会議-ノート
✅ #会議_ノート
❌ #会議 ノート (スペースは許可されていません)
❌ #会議ノート (一貫性がない)
```

### 過剰なタグ付けを避ける

多すぎるタグは管理不能になります

```
✅ #仕事 #緊急 #プロジェクト-alpha
❌ #仕事 #緊急 #重要 #今日 #月曜日 #オフィス #デスク #コンピュータ
```

### 定期的なクリーンアップ

定期的にタグを見直して統合

- 類似タグをマージ
- 未使用タグを削除
- 規約を更新

## 高度な機能

### タグベースのスマートフォルダー

タグに基づいて自動的に入力されるフォルダーを作成

1. 新規フォルダーを作成
2. フィルターを設定: `tag:プロジェクト`
3. タグ付きのすべてのノートが表示

### YAMLのタグ

ノートのフロントマターにタグを追加

```yaml
---
tags:
  - 仕事
  - 会議
  - プロジェクト-alpha
---
```

### タグエイリアス

設定でタグエイリアスを設定

- `#todo` = `#タスク` = `#アクション`
- 類似タグを統合
