# クイックキャプチャ

ワークフローを中断せずにアイデアを即座にキャプチャ。クイックキャプチャはシステムのどこからでも最小限のノート取りウィンドウを開きます。

## グローバルホットキー

指定のホットキーを押してクイックキャプチャを開く

| プラットフォーム | デフォルトショートカット |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

このショートカットは設定でカスタマイズできます。

## クイックキャプチャの使用

1. グローバルホットキーを押す（システムのどこでも）
2. ポップアップウィンドウにノートを入力
3. `Enter` を押して保存
4. `Escape` を押してキャンセル

## 機能

### ミニマルインターフェース

- 小さく、集中したウィンドウ
- Markdownサポート
- 入力に自動フォーカス
- キーボードファースト設計

### クイックアクション

- `# ` + スペース = 見出し
- `#inbox` = 受信トレイに追加
- `#today` = 今日のノートに追加
- `#タグ名` でタグ

### デフォルトの保存先

クイックキャプチャのノートはデフォルトで**受信トレイ**に保存されます。これは変更可能

1. 設定を開く
2. クイックキャプチャに移動
3. デフォルトの保存先を選択

## ヒント

### キャプチャワークフロー

1. **キャプチャ** - アイデアを素早くメモ
2. **処理** - 受信トレイを毎日レビュー
3. **整理** - 適切なフォルダー/ノートに移動

### ベストプラクティス

- 簡潔に - キャプチャごとに1つのアイデア
- コンテキストにタグを使用
- フォーマットを気にしない - 後で洗練
- キャプチャを定期的にレビューして処理

### 統合

クイックキャプチャは以下と相性が良い

- **デイリーノート**: 毎朝キャプチャをレビューして開始
- **受信トレイ**: 保留エリアとして使用
- **タグ**: すぐにコンテキストを追加

## トラブルシューティング

### ホットキーが動作しない？

**macOS**
1. システム設定 > キーボード > ショートカットを確認
2. 競合を探す
3. Classicを再起動

**Windows**
1. Classicを管理者として実行
2. 競合するショートカットを確認
3. Classicを再起動

### ウィンドウが表示されない？

- Classicが実行されているか確認
- ショートカットを確認
- フォーカスの問題を確認
- Classicを再起動

### キャプチャが保存されない？

- デフォルトの保存先を確認
- 書き込み権限を確認
- ディスク容量を確認
- Classicを再起動
