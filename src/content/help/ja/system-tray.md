# システムトレイ

Classicはシステムトレイで常にすぐに使えます。ワークフローを中断せずにアイデアを即座にキャプチャ。

## システムトレイとは？

システムトレイ（Windows/Linux）またはメニューバー（macOS）は、Classicがバックグラウンドで実行され、常にクイックキャプチャの準備ができている場所です。

### 利点

- **常に利用可能**: ワンクリック
- **クイックキャプチャ**: アイデアを即座にメモ
- **バックグラウンド実行**: アプリを開いておく必要がない
- **通知**: タスクのリマインダー

## システムトレイの有効化

### macOS

1. Classicを開く
2. 設定 → 一般に移動
3. 「メニューバーに表示」を有効化
4. Classicアイコンがメニューバーに表示

### Windows

1. Classicを開く
2. 設定 → 一般に移動
3. 「システムトレイに表示」を有効化
4. Classicアイコンがシステムトレイに表示

### Linux

1. Classicを開く
2. 設定 → 一般に移動
3. 「システムトレイアイコン」を有効化
4. デスクトップ環境に依存

## クイックキャプチャ

### クイックキャプチャを開く

システムトレイのClassicアイコンをクリック

- **macOS**: メニューバーアイコン
- **Windows**: システムトレイアイコン
- **Linux**: システムトレイアイコン

### クイックキャプチャの使用

1. システムトレイアイコンをクリック
2. クイックキャプチャウィンドウが開く
3. ノートを入力
4. Enterを押して保存
5. ウィンドウが自動的に閉じる

### キーボードショートカット

どこからでもクイックキャプチャを開く

| プラットフォーム | ショートカット |
|----------|----------|
| **macOS** | `Cmd+Shift+N` （グローバル） |
| **Windows** | `Ctrl+Shift+N` （グローバル） |
| **Linux** | `Ctrl+Shift+N` （グローバル） |

## システムトレイメニュー

トレイアイコンを右クリックしてオプション

### メニューオプション

| オプション | アクション |
|--------|--------|
| **新規ノート** | クイックキャプチャを開く |
| **新規デイリーノート** | 今日のデイリーノートを開く |
| **検索** | 検索ウィンドウを開く |
| **Classicを開く** | メインウィンドウを開く |
| **設定** | 設定を開く |
| **終了** | Classicを終了 |

## トレイ設定

### 設定で構成

| 設定 | 説明 |
|---------|-------------|
| **アイコンを表示** | トレイアイコンの表示を切り替え |
| **最小化で起動** | Classicをトレイのみで起動 |
| **トレイに閉じる** | ウィンドウを閉じても実行を継続 |
| **通知** | 通知の有効/無効 |

### ログイン時に起動

Classicが自動的に起動するよう設定

1. 設定を開く
2. 一般に移動
3. 「ログイン時に起動」を有効化
4. Classicが起動時に起動

## 通知

### 通知を有効化

1. 設定を開く
2. 通知に移動
3. 「通知を有効化」を切り替え

### 通知タイプ

- **リマインダー**: スケジュールされたリマインダー
- **デイリーノート**: デイリーノートのプロンプト
- **更新**: アプリの更新が利用可能

## ウィンドウの動作

### 閉じる動作

ウィンドウを閉じたときの動作を選択

- **トレイに最小化**: 実行を継続、ウィンドウを非表示
- **終了**: アプリケーションを終了
- **確認**: 毎回プロンプト

### 最小化動作

- **トレイに最小化**: システムトレイに非表示
- **タスクバーに最小化**: 通常通り最小化

## ヒント

### 高速キャプチャ

最速のキャプチャにグローバルショートカットを使用

1. どこでも `Cmd/Ctrl+Shift+N` を押す
2. 思考を入力
3. Enterを押す
4. 仕事を続ける

### クイックアクセス

トレイアイコンをクリックして表示

- 最近のノート
- クイックアクション
- 今日のデイリーノート

### バックグラウンド実行

Classicを実行し続ける利点

- より高速な起動
- クイックキャプチャ
- 通知
- 自動バックアップ

## トラブルシューティング

### トレイアイコンが見つからない

**macOS**

1. アイコンが非表示になっていないか確認
2. コントロールセンターを確認（macOS Big Sur以降）
3. Classicを再起動

**Windows**

1. 非表示のアイコンを確認（タスクバーの矢印）
2. Classicアイコンを表示エリアにドラッグ
3. Classicを再起動

**Linux**

1. デスクトップ環境のサポートを確認
2. 必要に応じてシステムトレイ拡張機能をインストール
3. Classicを再起動

### グローバルショートカットが動作しない

- 他のアプリとの競合を確認
- 設定でショートカットを再有効化
- Classicを再起動
- OSの権限を確認

### 通知が表示されない

**macOS**

1. システム設定 → 通知
2. Classicを見つける
3. 通知を有効化

**Windows**

1. 設定 → システム → 通知
2. Classicの通知を有効化

**Linux**

1. 通知デーモンを確認
2. 必要に応じてlibnotifyをインストール

### メモリ/CPU使用量が高い

トレイアイコンがリソースを使いすぎている場合

1. 設定でアニメーションを無効化
2. 通知頻度を減らす
3. アプリの更新を確認
