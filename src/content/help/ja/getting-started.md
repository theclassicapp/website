# クイックスタートガイド

Classicへようこそ！このガイドは、数分でClassicを使い始めるのに役立ちます。

## システム要件

- **macOS**: macOS 12 (Monterey) 以降
- **Windows**: Windows 10 以降
- **Linux**: Ubuntu 20.04+ / Fedora 36+ 以降

## インストール

1. [ウェブサイト](/)からClassicをダウンロード
2. アプリケーションを開く
3. 書き始める！

## 初回起動時の体験

Classicを初めて開くと、執筆に集中できるよう設計されたシンプルでミニマルなインターフェースが表示されます。

![Classic App Screenshot]

### サイドバー

左側には以下が表示されます

- **デイリーノート** - 自動の日次ジャーナル
- **検索** - すべてのノートから素早く検索
- **コマンドパレット** - `Cmd/Ctrl+P` (Windows/Linux) または `Cmd+Shift+P` (macOS) を押してコマンドパレットを開く
- **アウトライン** - ツリー構造でノートを参照・整理

### エディター

中央のエリアは**Markdownエディター**で、以下の機能があります：
- ライブプレビュー
- シンタックスハイライト
- 豊富なフォーマットツール

### 最初のノートを作成

1. サイドバーの**新規ノート**ボタンをクリック、または `Ctrl/Cmd+N` (Windows/Linux) または `Cmd+N` (macOS) を押す
2. ノートにタイトルを付ける
3. Markdownで書き始める！

```markdown
# 初めてのノート

Classicへようこそ！ここで思考やアイデアを記録します。
```
