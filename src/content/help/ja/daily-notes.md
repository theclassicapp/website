# デイリーノート

デイリーノートは毎日新しいノートを自動的に作成し、ジャーナリング、日次計画、または時間ベースの情報を記録するのに最適です。

## 仕組み

### 自動作成

- 毎日新しいノートが作成されます
- ノートは日付形式で命名されます（例: `2024-01-15.md`）
- 過去のノートは保存され、検索可能

### デイリーノートへのアクセス

1. サイドバーの**デイリーノート**をクリック
2. または `Ctrl/Cmd+D` を押す
3. 今日のノートが自動的に開きます

## デイリーノートテンプレート

新しいデイリーノートのテンプレートをカスタマイズ

### デフォルトテンプレート

```markdown
# {{date}}

## タスク
- [ ]
- [ ]

## ノート


## ジャーナル

```

### カスタマイズ

1. 設定を開く
2. デイリーノートに移動
3. テンプレートを編集
4. 現在の日付に `{{date}}` を使用

### テンプレート変数

| 変数 | 説明 |
| -------- | ----------- |
| `{{date}}` | 現在の日付 (YYYY-MM-DD) |
| `{{time}}` | 現在の時刻 (HH:mm) |
| `{{weekday}}` | 曜日名 (月曜日など) |

## 機能

### 日付ナビゲーション

- **前/次**: 日間を移動
- **カレンダー**: 特定の日付にジャンプ
- **検索**: すべてのデイリーノートでコンテンツを検索

### 自動リンク

- 同じ日に作成されたノートは自動的にリンク
- バックリンクは日間のつながりを表示
- タグはすべてのデイリーノートで機能

### 統合

- **クイックキャプチャ**: `#today` で今日のノートに追加
- **カレンダービュー**: デイリーノートの視覚的な概要
- **統計**: 執筆ストリークを追跡

## ベストプラクティス

### 毎日のジャーナリング

```markdown
# 2024-01-15

## 朝
- 6:30に起床
- 気分: 元気

## タスク
- [x] 朝の運動
- [ ] PRをレビュー
- [ ] 午後2時にチームミーティング

## 学び
- 新しいキーボードショートカットを発見
- Mermaidダイアグラムについて学んだ

## 明日
- ドキュメントに集中
- コードレビューをスケジュール
```

### 会議ノート

```markdown
# 2024-01-15

## チームスタンドアップ

**参加者**: Alice, Bob, Carol

### 更新
- Alice: API統合を完了
- Bob: UIコンポーネントに取り組み中
- Carol: テスト進行中

### ブロッカー
- デザインレビューを待機中

### アクションアイテム
- [ ] Alice: ステージングにデプロイ
- [ ] Bob: コンポーネントライブラリを完了
- [ ] Carol: 統合テストを執筆
```

### 学習ログ

```markdown
# 2024-01-15

## 今日学んだこと

### Mermaidダイアグラム
- Markdownでフローチャートを作成可能
- 構文は簡単
- ライブプレビューが役立つ

### リソース
- [Mermaidドキュメント](https://mermaid.js.org/)

## 質問
- 複雑なダイアグラムを作成するには？
- ダイアグラム整理のベストプラクティスは？
```

## ヒント

### 一貫性

- 毎日同じ時間にノートを作成
- 一貫したフォーマットを使用
- 一日を通して見直しと更新

### 整理

- 異なるセクションに見出しを使用
- 簡単に検索できるようタグを追加
- 関連ノートにリンク

### レビュー

- デイリーノートの週次レビュー
- 重要なアイテムを永続ノートに抽出
- 古いノートをアーカイブまたは要約
