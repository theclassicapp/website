# キーボードショートカット

Classicのキーボードショートカットでより速く作業。すべてのショートカットは設定でカスタマイズ可能です。

## グローバルショートカット

これらはアプリのどこでも動作します。

### 一般

| ショートカット (Windows/Linux) | ショートカット (macOS) | アクション |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | 新規ノート |
| `Ctrl+P` | `Cmd+P` | コマンドパレット |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | グローバルクイックキャプチャ |
| `Ctrl+S` | `Cmd+S` | ノートを保存 |
| `Ctrl+F` | `Cmd+F` | 検索 |
| `Ctrl+,` | `Cmd+,` | 設定 |

### ナビゲーション

| ショートカット (Windows/Linux) | ショートカット (macOS) | アクション |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | 最初のノートに移動 |
| `Ctrl+2` | `Cmd+2` | 2番目のノートに移動 |
| `Ctrl+Left` | `Cmd+Left` | 戻る |
| `Ctrl+Right` | `Cmd+Right` | 進む |
| `Ctrl+Up` | `Cmd+Up` | 親ノートに移動 |
| `Ctrl+Down` | `Cmd+Down` | 最初のサブノートを展開/フォーカス |

### エディター

| ショートカット (Windows/Linux) | ショートカット (macOS) | アクション |
| ------------------------ | ----------------- | ------ |
| `Ctrl+B` | `Cmd+B` | 太字 |
| `Ctrl+I` | `Cmd+I` | 斜体 |
| `Ctrl+K` | `Cmd+K` | リンクを挿入 |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | 取り消し線 |
| `Ctrl+`` | `Cmd+`` | インラインコード |
| `Ctrl+Shift+C` | `Cmd+Shift+C` | コードブロック |
| `Ctrl+/` | `Cmd+/` | スラッシュコマンド |
| `Tab` | `Tab` | インデント |
| `Shift+Tab` | `Shift+Tab` | インデント解除 |

### ウィンドウ管理

| ショートカット (Windows/Linux) | ショートカット (macOS) | アクション |
| ------------------------ | ----------------- | ------ |
| `Ctrl+W` | `Cmd+W` | 現在のノートを閉じる |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | すべてのノートを閉じる |
| `F11` | `F11` | フルスクリーン切り替え |
| `Ctrl+M` | `Cmd+M` | サイドバー切り替え |

## クイックキャプチャ

| ショートカット (Windows/Linux) | ショートカット (macOS) | アクション |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | クイックキャプチャを開く |
| `Escape` | `Escape` | クイックキャプチャを閉じる |
| `Ctrl+Enter` | `Cmd+Enter` | 保存して閉じる |

## ショートカットのカスタマイズ

1. **設定**を開く（`Ctrl+,` または `Cmd+,`）
2. **キーボードショートカット**に移動
3. 任意のショートカットをクリックして変更
4. 新しいキーの組み合わせを押す
5. **保存**をクリック

## ヒント

- ほとんどのショートカットはカスタマイズ可能
- システムショートカットとの競合はハイライト表示
- 各ショートカットの横にあるリセットボタンでデフォルトにリセット
