# 設定

Classicを自分の望む通りに動作するようカスタマイズ。外観からエディターの動作まで、すべてが設定可能です。

## 設定を開く

### キーボードショートカット

| プラットフォーム | ショートカット |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### メニューから

- **macOS**: Classic → 設定
- **Windows**: ファイル → 設定
- **Linux**: 編集 → 設定

### コマンドパレットから

1. コマンドパレットを開く（`Cmd+P` または `Ctrl+P`）
2. 「設定」と入力
3. Enterを押す

## 設定カテゴリ

### 一般

基本的なアプリケーション設定

| 設定 | 説明 |
|---------|-------------|
| **言語** | インターフェース言語 |
| **テーマ** | ライト、ダーク、またはシステム |
| **ログイン時に起動** | 起動時に起動 |
| **更新を確認** | 自動更新動作 |

### 外観

ルックアンドフィールをカスタマイズ

| 設定 | オプション |
|---------|---------|
| **テーマ** | ライト / ダーク / システム |
| **アクセントカラー** | ブルー、パープル、グリーンなど |
| **エディターフォント** | フォントファミリーとサイズ |
| **UIフォント** | インターフェースフォント |
| **等幅フォント** | コードフォント |
| **サイドバー幅** | 200-400px |
| **行の高さ** | 1.0-2.5 |

### エディター

執筆動作を設定

| 設定 | 説明 |
|---------|-------------|
| **スペルチェック** | スペルチェックの有効/無効 |
| **自動保存** | 自動保存間隔 |
| **ワードラップ** | ワードラップの有効/無効 |
| **行番号** | 行番号の表示/非表示 |
| **タブサイズ** | 2, 4, または8スペース |
| **オートコレクト** | スマート引用符、自動置換 |

### バージョン履歴

バージョン追跡を制御

| 設定 | オプション | デフォルト |
|---------|---------|---------|
| **履歴を有効化** | オン/オフ | オン |
| **保持期間** | 30-180日 | 90日 |
| **最大バージョン数** | 50-500 | 100 |

### バックアップと復元

データ保護設定

| 設定 | 説明 |
|---------|-------------|
| **自動バックアップ** | 自動バックアップを有効化 |
| **バックアップ頻度** | 15分 / 30分 / 1時間 / 4時間 |
| **バックアップ場所** | カスタムパス |
| **バックアップを保持** | 5 / 10 / 20 / 50 |
| **バックアップを暗号化** | パスワード保護 |

### キーボードショートカット

ショートカットを表示してカスタマイズ

- すべての利用可能なショートカットを表示
- 任意のショートカットをカスタマイズ
- デフォルトにリセット
- ショートカットのインポート/エクスポート

### インポートとエクスポート

データ移行設定

- 他のアプリからインポート
- エクスポート形式オプション
- 一括操作

### プライバシー

プライバシーとセキュリティ設定

| 設定 | 説明 |
|---------|-------------|
| **アナリティクス** | 使用アナリティクスの有効/無効 |
| **クラッシュレポート** | クラッシュレポートを送信 |
| **履歴をクリア** | 検索/履歴をクリア |

### 通知

通知を制御

| 設定 | 説明 |
|---------|---------|
| **通知を有効化** | オン/オフ |
| **デイリーノートリマインダー** | リマインド時刻 |
| **サウンド** | 通知音 |

### 詳細

パワーユーザー設定

| 設定 | 説明 |
|---------|-------------|
| **カスタムCSS** | カスタムスタイルを追加 |
| **デバッグモード** | デバッグログを有効化 |
| **実験的機能** | ベータ機能 |
| **データの場所** | ノートフォルダーを変更 |

## 設定の検索

設定が見つからない？検索バーを使用

1. 設定を開く
2. 検索ボックスに入力
3. 一致する設定が表示
4. クリックしてその設定にジャンプ

## 設定のリセット

### すべての設定をリセット

デフォルトに戻す

1. 設定を開く
2. 一番下までスクロール
3. 「すべての設定をリセット」をクリック
4. アクションを確認

### カテゴリをリセット

特定のカテゴリをリセット

1. カテゴリに移動
2. 「デフォルトにリセット」をクリック
3. そのカテゴリのみリセット

## 設定のエクスポート/インポート

### 設定をエクスポート

設定を保存

1. 設定を開く
2. 「設定をエクスポート」をクリック
3. 場所を選択
4. JSONファイルとして保存

### 設定をインポート

保存した設定を読み込む

1. 設定を開く
2. 「設定をインポート」をクリック
3. JSONファイルを選択
4. インポートを確認

## 設定の場所

設定はローカルに保存

| プラットフォーム | 場所 |
|----------|----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## 設定の同期（オプション）

サインイン時、設定はデバイス間で同期可能

- **同期**: 設定 → 同期で有効化
- **同期されるもの**: テーマ、フォント、ショートカット
- **同期されないもの**: ローカルパス、暗号化キー

## 一般的なカスタマイズ

### フォントサイズを大きく

1. 設定 → 外観
2. エディターフォントサイズ → 増やす
3. 変更は即座に適用

### テーマを変更

1. 設定 → 外観
2. テーマ → ライト/ダーク/システムを選択
3. アプリは即座に更新

### 自動バックアップを有効化

1. 設定 → バックアップと復元
2. 「自動バックアップ」を有効化
3. 頻度と場所を設定

### ショートカットをカスタマイズ

1. 設定 → キーボードショートカット
2. アクションを見つける
3. 現在のショートカットをクリック
4. 新しいキーを押す
5. 保存

## ヒント

### クイックアクセス

- `Cmd/Ctrl+,` で設定を素早く開く
- 名前で設定を検索
- カテゴリでナビゲート

### 変更前にエクスポート

大きな変更の前に設定をエクスポート

- 何か問題が発生した場合に簡単に復元
- 設定を他人と共有

### 新しい設定を確認

更新で新しい設定が追加される可能性

- 更新後に確認
- 新しいオプションを探索

## トラブルシューティング

### 設定が保存されない

- ファイル権限を確認
- ディスク容量を確認
- リセットして再設定を試す

### 設定が予期せずリセット

- ファイルの破損を示す可能性
- 定期的に設定をエクスポート
- アプリの競合を確認

### 設定が見つからない

- 検索バーを使用
- 異なるカテゴリを確認
- 詳細設定にある可能性
