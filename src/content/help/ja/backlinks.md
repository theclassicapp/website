# バックリンク

バックリンクはノート間のつながりを表示し、関連アイデアのネットワークを作成します。知らなかった関係を発見できます。

## バックリンクとは？

バックリンクはどのノートが現在のノートにリンクしているかを表示します。ナレッジグラフ内のつながりです。

### 利点

- **つながりを発見**: 関連ノートを自動的に見つける
- **コンテキスト**: アイデアがどのように関連しているかを理解
- **ナビゲーション**: 接続されたノート間をジャンプ
- **ナレッジグラフ**: 思考を視覚化

## バックリンクの仕組み

別のノートへのリンクを作成すると、Classicは自動的にバックリンクを作成します。

```markdown
ノートAで:
詳細は [[プロジェクト計画]] を参照。

結果:
- ノートAは「プロジェクト計画」にリンク
- 「プロジェクト計画」はノートAへのバックリンクを表示
```

## バックリンクの表示

### バックリンクパネル

すべての参照を表示するにはバックリンクパネルを開く

1. 任意のノートを開く
2. サイドバーのバックリンクアイコンをクリック
3. このノートにリンクしているすべてのノートを表示

### インラインバックリンク

バックリンクは各ノートの下部に表示

- ソースノート名を表示
- コンテキストスニペットを表示
- クリックしてナビゲート

## リンクの作成

### ウィキリンク構文

二重角括弧を使用してノートにリンク

```markdown
[[ノート名]]
[[別のノート]]
```

### 表示テキスト

リンクテキストをカスタマイズ

```markdown
[[プロジェクトAlpha|アルファプロジェクト]]
[[会議ノート|昨日の会議]]
```

### 見出しへのリンク

特定のセクションにリンク

```markdown
[[ノート名#見出し]]
[[ノート名#セクション|セクションにジャンプ]]
```

## バックリンクのタイプ

### 直接リンク

作成した明示的なウィキリンク

```markdown
関連: [[類似ノート]]
```

### リンクされていない言及

タイトルに言及しているがリンクされていないノート

```markdown
ノートは「プロジェクトAlpha」に言及しているがリンクされていない
```

Classicはこれらをリンクに変換することを提案します。

## バックリンクパネル

### 機能

- **フィルター**: バックリンクを検索
- **ソート**: 日付、名前、関連性順
- **プレビュー**: ホバーしてコンテンツをプレビュー
- **コンテキスト**: 周囲のテキストを表示

### アクション

- クリックしてリンクされたノートを開く
- 右クリックでオプション
- リンクされていない言及を変換

## 使用例

### ナレッジ管理

パーソナルWikiを構築

```markdown
# 機械学習

[[人工知能]]に関連
[[統計学]]に基づく
[[自然言語処理]]で使用
```

### プロジェクトドキュメント

関連ドキュメントを接続

```markdown
# API設計

[[データベーススキーマ]]を参照
[[認証フロー]]に依存
[[フロントエンド統合]]も参照
```

### 研究

概念とソースをリンク

```markdown
# 研究ノート

[[ソース論文A]]に基づく
[[ソース論文B]]と矛盾
[[仮説1]]をサポート
```

### 会議ノート

会議をプロジェクトに接続

```markdown
# 会議: Q4計画

プロジェクト: [[プロジェクトAlpha]]
前回: [[Q3レビュー]]
次回: [[Q4キックオフ]]
```

## グラフビュー

バックリンクをネットワークとして視覚化

1. グラフビューを開く（`Cmd/Ctrl+G`）
2. すべてのノートをノードとして表示
3. 線がつながりを表示
4. ノードをクリックしてナビゲート

### グラフコントロール

- **ズーム**: スクロールしてズーム
- **パン**: ドラッグして移動
- **フィルター**: タグで表示/非表示
- **フォーカス**: ノートのつながりをハイライト

## ヒント

### 惜しみなくリンク

別の概念を参照するたびにリンクを作成

```markdown
✅ 良い:
[[Q4ロードマップ]]と[[予算配分]]について議論。

❌ 避ける:
Q4ロードマップと予算配分について議論。
```

### 一貫した名前を使用

信頼性の高いリンクのためにノートタイトルと正確に一致

```markdown
✅ [[会議ノート]]
❌ [[会議ノート]]（大文字小文字が重要）
```

### バックリンクを見直す

定期的にバックリンクを確認

- 忘れていたつながりを発見
- マージするノートを見つける
- ナレッジのギャップを特定

### リンクプレビュー

開かずにプレビューするにはリンクにホバー

- 最初の数行を表示
- 正しいノートかどうか確認
- クイックリファレンス

## キーボードショートカット

| アクション | ショートカット |
|--------|----------|
| バックリンクを開く | `Cmd/Ctrl+Shift+B` |
| リンクを作成 | `[[` （自動補完） |
| グラフビューを開く | `Cmd/Ctrl+G` |
| リンクをナビゲート | `Tab` でリンクを移動 |

## 高度な機能

### 埋め込み

他のノートからコンテンツを埋め込む

```markdown
![[ノート名]]
![[ノート名#セクション]]
```

### トランスクルージョン

コンテンツをインラインで含める

```markdown
## 関連コンテンツ
![[共有テンプレート#ヘッダー]]
```

### リンクの色分け

リンクは以下に基づいて色が変わる

- **青**: 未訪問リンク
- **紫**: 訪問済みリンク
- **赤**: リンク切れ（ノートが存在しない）

## トラブルシューティング

### バックリンクが見つからない

- リンク構文 `[[ノート名]]` を確認
- ノートタイトルが正確に一致することを確認
- バックリンクパネルを更新

### リンク切れ

- ノートの名前が変更された可能性
- ノートが削除された可能性
- 検索を使用して正しいノートを見つける

### 重複リンク

- 類似ノートを統合
- 名前変更されたノートにリダイレクトを使用
- タイトルのバリエーションを確認
