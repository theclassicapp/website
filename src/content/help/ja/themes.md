# テーマと外観

Classicの外観をパーソナライズしてスタイルに合わせ、目の疲れを軽減。

## テーマオプション

### ライトモード

Classicのデフォルトテーマの特徴

- クリーンで明るいインターフェース
- 明るい環境に最適
- 読みやすさのために高コントラスト

### ダークモード

夜型人間や薄暗い作業に最適

- 目の疲れを軽減
- OLED画面でバッテリー消費を低減
- 暗い環境で目に優しい

### システム（自動）

オペレーティングシステムのテーマに従う

- OS設定に基づいて自動的に切り替え
- 手動切り替え不要
- 他のアプリと一貫性

## テーマの切り替え

### 設定から

1. 設定を開く（`Cmd+,` または `Ctrl+,`）
2. **外観**に移動
3. 好みのテーマを選択

### コマンドパレットから

1. コマンドパレットを開く（`Cmd+P` または `Ctrl+P`）
2. 「テーマ」と入力
3. 好みのテーマを選択

### キーボードショートカット

| プラットフォーム | ショートカット |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

ライト → ダーク → システムの順に切り替え

## フォント設定

### エディターフォント

執筆体験をカスタマイズ

1. 設定を開く
2. **外観** → **エディター**に移動
3. フォントオプションを設定

| 設定 | オプション | デフォルト |
| ------- | ------- | ------- |
| フォントファミリー | システムフォント + カスタム | System UI |
| フォントサイズ | 12-24px | 16px |
| 行の高さ | 1.0-2.5 | 1.75 |
| 文字間隔 | -2 to 4 | 0 |

### UIフォント

インターフェースフォントを調整

1. 設定を開く
2. **外観** → **インターフェース**に移動
3. フォントとサイズを選択

### 等幅フォント

コードブロックと技術コンテンツ用

1. 設定を開く
2. **外観** → **コード**に移動
3. 等幅フォントを選択

## インターフェースオプション

### サイドバー

サイドバーの動作をカスタマイズ

| 設定 | 説明 |
| ------- | ----------- |
| **自動非表示** | 使用していないときにサイドバーを非表示 |
| **幅** | サイドバー幅を調整（200-400px） |
| **アイコンを表示** | アイコンの表示を切り替え |

### エディターレイアウト

編集体験を選択

- **分割ビュー**: 編集とプレビューを並べて表示
- **プレビューのみ**: レンダリングされた出力のみを表示
- **編集のみ**: 生のMarkdownに集中

### ノートリスト

ノートの表示方法を設定

- **リストビュー**: コンパクト、ノートごとに1行
- **カードビュー**: スニペット付きのプレビューカード
- **ツリービュー**: 階層フォルダー構造

## アクセントカラー

### 利用可能な色

アクセントカラーを選択

- クラシックブルー（デフォルト）
- パープル
- グリーン
- オレンジ
- レッド
- ピンク
- ティール

### アクセントカラーの変更

1. 設定を開く
2. **外観** → **色**に移動
3. 好みのアクセントを選択

アクセントカラーは以下に影響

- リンクとボタン
- 選択ハイライト
- アクティブ要素
- フォーカスインジケーター

## アクセシビリティ

### ハイコントラストモード

より良い可視性のため

1. 設定を開く
2. **外観** → **アクセシビリティ**に移動
3. **ハイコントラスト**を有効化

### モーションを減らす

アニメーションを最小限に

1. 設定を開く
2. **外観** → **アクセシビリティ**に移動
3. **モーションを減らす**を有効化

### フォント調整

- 最小フォントサイズを増やす
- ディスレクシアフレンドリーフォントを使用
- コントラスト比を調整

## カスタムCSS（上級）

完全なコントロールを望むパワーユーザー向け

1. 設定を開く
2. **外観** → **詳細**に移動
3. **カスタムCSSを開く**をクリック
4. カスタマイズを追加

### カスタマイズ例

```css
/* より大きな見出し */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* カスタムリンク色 */
a { color: #3b82f6; }

/* より広い行間隔 */
.prose p { line-height: 2; }
```

## テーマ同期

サインイン時（オプション）、テーマ設定はデバイス間で同期

- 自動テーマ同期
- フォント設定の同期
- カスタムCSSの同期

## ヒント

### ワークフローに合わせる

- 日中は**ライト**、夜は**ダーク**を使用
- **システム**モードはこれを自動的に処理
- 環境光を考慮

### 目の疲れを軽減

- 薄暗い場所ではダークモードを使用
- 長時間のセッションではフォントサイズを大きく
- 必要に応じてハイコントラストを有効化
- 定期的に休憩を取る

### 生産性

- テーマ切り替えショートカットを学ぶ
- 自動切り替え時間を設定
- 一貫性のためにシステムモードを使用
