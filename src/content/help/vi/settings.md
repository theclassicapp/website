# Cài đặt

Tùy chỉnh Classic để hoạt động chính xác theo cách bạn muốn. Từ vẻ ngoài đến hành vi trình soạn thảo, mọi thứ đều có thể cấu hình.

## Mở Cài đặt

### Phím tắt

| Nền tảng | Phím tắt |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Qua Menu

- **macOS**: Classic → Cài đặt
- **Windows**: Tệp → Cài đặt
- **Linux**: Chỉnh sửa → Cài đặt

### Qua Bảng lệnh

1. Mở Bảng lệnh (`Cmd+P` hoặc `Ctrl+P`)
2. Gõ "Cài đặt"
3. Nhấn Enter

## Danh mục Cài đặt

### Chung

Cài đặt ứng dụng cơ bản:

| Cài đặt | Mô tả |
|---------|-------------|
| **Ngôn ngữ** | Ngôn ngữ giao diện |
| **Giao diện** | Sáng, Tối hoặc Hệ thống |
| **Khởi động cùng Đăng nhập** | Khởi chạy khi khởi động |
| **Kiểm tra Cập nhật** | Hành vi tự động cập nhật |

### Vẻ ngoài

Tùy chỉnh vẻ và cảm:

| Cài đặt | Tùy chọn |
|---------|---------|
| **Giao diện** | Sáng / Tối / Hệ thống |
| **Màu Nhấn** | Xanh dương, Tím, Xanh lá, v.v. |
| **Phông chữ Trình soạn thảo** | Họ phông chữ và cỡ |
| **Phông chữ UI** | Phông chữ giao diện |
| **Phông chữ Monospace** | Phông chữ mã |
| **Độ rộng Thanh bên** | 200-400px |
| **Chiều cao Dòng** | 1.0-2.5 |

### Trình soạn thảo

Cấu hình hành vi viết:

| Cài đặt | Mô tả |
|---------|-------------|
| **Kiểm tra Chính tả** | Bật/tắt kiểm tra chính tả |
| **Tự động lưu** | Khoảng thời gian tự động lưu |
| **Gói từ** | Bật/tắt gói từ |
| **Số dòng** | Hiển thị/ẩn số dòng |
| **Cỡ Tab** | 2, 4 hoặc 8 khoảng trắng |
| **Tự động sửa** | Trích dẫn thông minh, tự động thay thế |

### Lịch sử Phiên bản

Kiểm soát theo dõi phiên bản:

| Cài đặt | Tùy chọn | Mặc định |
|---------|---------|---------|
| **Kích hoạt Lịch sử** | Bật/Tắt | Bật |
| **Thời gian Lưu trữ** | 30-180 ngày | 90 ngày |
| **Phiên bản Tối đa** | 50-500 | 100 |

### Sao lưu & Khôi phục

Cài đặt bảo vệ dữ liệu:

| Cài đặt | Mô tả |
|---------|-------------|
| **Tự động Sao lưu** | Kích hoạt sao lưu tự động |
| **Tần suất Sao lưu** | 15phút / 30phút / 1giờ / 4giờ |
| **Vị trí Sao lưu** | Đường dẫn tùy chỉnh |
| **Giữ Sao lưu** | 5 / 10 / 20 / 50 |
| **Mã hóa Sao lưu** | Bảo vệ mật khẩu |

### Phím tắt Bàn phím

Xem và tùy chỉnh phím tắt:

- Xem tất cả các phím tắt khả dụng
- Tùy chỉnh bất kỳ phím tắt nào
- Đặt lại về mặc định
- Nhập/xuất phím tắt

### Nhập & Xuất

Cài đặt di chuyển dữ liệu:

- Nhập từ các ứng dụng khác
- Tùy chọn định dạng xuất
- Thao tác hàng loạt

### Quyền riêng tư

Cài đặt quyền riêng tư và bảo mật:

| Cài đặt | Mô tả |
|---------|-------------|
| **Phân tích** | Bật/tắt phân tích sử dụng |
| **Báo cáo Lỗi** | Gửi báo cáo lỗi |
| **Xóa Lịch sử** | Xóa tìm kiếm/lịch sử |

### Thông báo

Kiểm soát thông báo:

| Cài đặt | Mô tả |
|---------|-------------|
| **Kích hoạt Thông báo** | Bật/Tắt |
| **Nhắc nhở Ghi chú Hàng ngày** | Thời gian nhắc nhở |
| **Âm thanh** | Âm thanh thông báo |

### Nâng cao

Cài đặt người dùng nâng cao:

| Cài đặt | Mô tả |
|---------|-------------|
| **CSS Tùy chỉnh** | Thêm kiểu tùy chỉnh |
| **Chế độ Gỡ lỗi** | Kích hoạt ghi gỡ lỗi |
| **Tính năng Thử nghiệm** | Tính năng beta |
| **Vị trí Dữ liệu** | Thay đổi thư mục ghi chú |

## Tìm kiếm Cài đặt

Không tìm thấy một cài đặt? Sử dụng thanh tìm kiếm:

1. Mở Cài đặt
2. Gõ vào ô tìm kiếm
3. Các cài đặt khớp xuất hiện
4. Nhấp để nhảy đến cài đặt đó

## Đặt lại Cài đặt

### Đặt lại Tất cả Cài đặt

Trở về mặc định:

1. Mở Cài đặt
2. Cuộn xuống dưới cùng
3. Nhấp "Đặt lại Tất cả Cài đặt"
4. Xác nhận hành động

### Đặt lại Danh mục

Đặt lại danh mục cụ thể:

1. Đi đến danh mục
2. Nhấp "Đặt lại về Mặc định"
3. Chỉ danh mục đó được đặt lại

## Xuất/Nhập Cài đặt

### Xuất Cài đặt

Lưu cấu hình của bạn:

1. Mở Cài đặt
2. Nhấp "Xuất Cài đặt"
3. Chọn vị trí
4. Lưu dưới dạng tệp JSON

### Nhập Cài đặt

Tải một cấu hình đã lưu:

1. Mở Cài đặt
2. Nhấp "Nhập Cài đặt"
3. Chọn tệp JSON
4. Xác nhận nhập

## Vị trí Cài đặt

Cài đặt được lưu trữ cục bộ:

| Nền tảng | Vị trí |
|----------|----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Đồng bộ Cài đặt (Tùy chọn)

Khi đăng nhập, cài đặt có thể đồng bộ qua các thiết bị:

- **Đồng bộ**: Kích hoạt trong Cài đặt → Đồng bộ
- **Những gì Đồng bộ**: Giao diện, phông chữ, phím tắt
- **Những gì Không Đồng bộ**: Đường dẫn cục bộ, khóa mã hóa

## Các Tùy chỉnh Phổ biến

### Tăng Cỡ Phông chữ

1. Cài đặt → Vẻ ngoài
2. Cỡ Phông chữ Trình soạn thảo → Tăng
3. Thay đổi áp dụng ngay lập tức

### Thay đổi Giao diện

1. Cài đặt → Vẻ ngoài
2. Giao diện → Chọn Sáng/Tối/Hệ thống
3. Ứng dụng cập nhật ngay lập tức

### Kích hoạt Tự động Sao lưu

1. Cài đặt → Sao lưu & Khôi phục
2. Kích hoạt "Tự động Sao lưu"
3. Đặt tần suất và vị trí

### Tùy chỉnh Phím tắt

1. Cài đặt → Phím tắt Bàn phím
2. Tìm hành động
3. Nhấp vào phím tắt hiện tại
4. Nhấn phím mới
5. Lưu

## Mẹo

### Truy cập Nhanh

- Sử dụng `Cmd/Ctrl+,` để mở cài đặt nhanh chóng
- Tìm kiếm cài đặt theo tên
- Sử dụng danh mục để điều hướng

### Xuất trước Khi Thay đổi

Trước các thay đổi lớn, xuất cài đặt của bạn:

- Dễ dàng khôi phục nếu có gì đó sai
- Chia sẻ cấu hình với người khác

### Kiểm tra Cài đặt Mới

Cập nhật có thể thêm cài đặt mới:

- Kiểm tra sau khi cập nhật
- Khám phá các tùy chọn mới

## Khắc phục sự cố

### Cài đặt Không Lưu

- Kiểm tra quyền tệp
- Xác minh dung lượng đĩa
- Thử đặt lại và cấu hình lại

### Cài đặt Đặt lại Bất ngờ

- Có thể chỉ ra hỏng tệp
- Xuất cài đặt thường xuyên
- Kiểm tra xung đột ứng dụng

### Không thể Tìm một Cài đặt

- Sử dụng thanh tìm kiếm
- Kiểm tra các danh mục khác nhau
- Có thể trong cài đặt Nâng cao
