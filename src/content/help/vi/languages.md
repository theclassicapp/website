# Ngôn ngữ & Bản địa hóa

Classic nói ngôn ngữ của bạn. Sử dụng ứng dụng bằng ngôn ngữ ưa thích của bạn và viết ghi chú bằng bất kỳ ngôn ngữ nào bạn chọn.

## Các Ngôn ngữ Được hỗ trợ

Giao diện của Classic có sẵn bằng nhiều ngôn ngữ:

| Ngôn ngữ | Mã |
|----------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| Và nhiều hơn... | |

## Thay đổi Ngôn ngữ Giao diện

### Qua Cài đặt

1. Mở Cài đặt (`Cmd+,` hoặc `Ctrl+,`)
2. Đi đến **Chung** → **Ngôn ngữ**
3. Chọn ngôn ngữ ưa thích của bạn
4. Khởi động lại Classic để áp dụng

### Qua Bảng lệnh

1. Mở Bảng lệnh (`Cmd+P` hoặc `Ctrl+P`)
2. Gõ "Ngôn ngữ"
3. Chọn ngôn ngữ ưa thích của bạn
4. Khởi động lại để áp dụng các thay đổi

## Viết bằng Nhiều Ngôn ngữ

Classic hỗ trợ viết ghi chú bằng bất kỳ ngôn ngữ nào:

### Hỗ trợ Unicode

- Hỗ trợ Unicode đầy đủ
- Các ngôn ngữ RTL (Phải-sang-Trái) được hỗ trợ
- Các ký tự CJK (Trung, Nhật, Hàn) hoạt động hoàn hảo
- Hỗ trợ Emoji

### Phát hiện Ngôn ngữ

Classic có thể phát hiện ngôn ngữ của ghi chú của bạn:

- Tự động phát hiện cho kiểm tra chính tả
- Cài đặt ngôn ngữ cho mỗi ghi chú
- Hỗ trợ ngôn ngữ hỗn hợp

## Kiểm tra Chính tả

### Kích hoạt Kiểm tra Chính tả

1. Mở Cài đặt
2. Đi đến **Trình soạn thảo** → **Kiểm tra Chính tả**
3. Kích hoạt "Kiểm tra Chính tả"

### Các Ngôn ngữ Kiểm tra Chính tả

1. Mở Cài đặt
2. Đi đến **Trình soạn thảo** → **Kiểm tra Chính tả**
3. Nhấp "Ngôn ngữ"
4. Chọn các ngôn ngữ để kiểm tra

### Thêm Từ vào Từ điển

1. Nhấp chuột phải vào một từ được đánh dấu
2. Chọn "Thêm vào Từ điển"
3. Từ sẽ không bị đánh dấu lại

## Hỗ trợ RTL

Cho tiếng Ả Rập, tiếng Do Thái và các ngôn ngữ RTL khác:

### Phát hiện RTL Tự động

Classic phát hiện nội dung RTL tự động:

- Hướng văn bản điều chỉnh
- Di chuyển con trúc thích ứng
- Căn chỉnh thay đổi

### Chuyển đổi RTL Thủ công

Ép buộc hướng RTL hoặc LTR:

1. Mở Bảng lệnh
2. Gõ "RTL" hoặc "LTR"
3. Chọn tùy chọn của bạn

## Phương thức Nhập

Classic hoạt động với các phương thức nhập của hệ thống của bạn:

### Các Phương thức Nhập CJK

- Hoạt động với Pinyin, Zhuyin, v.v.
- Hỗ trợ IME tiếng Nhật
- Các phương thức nhập tiếng Hàn được hỗ trợ

### Các Ký tự Đặc biệt

Chèn các ký tự đặc biệt dễ dàng:

- Sử dụng công cụ chọn ký tự của hệ thống
- Sao chép/dán từ bản đồ ký tự
- Sử dụng nhập Unicode

## Định dạng Ngày & Giờ

Các ngày thích ứng với locale của bạn:

### Các Tùy chọn Định dạng

| Cài đặt | Tùy chọn |
|---------|---------|
| Định dạng Ngày | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| Định dạng Giờ | 12-giờ, 24-giờ |
| Ngày đầu tuần | Chủ nhật, Thứ hai, Thứ bảy |

Cấu hình trong Cài đặt → Chung → Ngày & Giờ

## Định dạng Số

Các số hiển thị theo locale của bạn:

- **Dấu phân cách thập phân**: `.` hoặc `,`
- **Dấu phân cách hàng nghìn**: `,` hoặc `.` hoặc khoảng trắng
- **Ký hiệu tiền tệ**: Dựa trên khu vực

## Bố cục Bàn phím

Classic hoạt động với bất kỳ bố cục bàn phím nào:

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Các bố cục tùy chỉnh

## Cài đặt Bản địa hóa

### Trong Cài đặt → Chung → Ngôn ngữ

| Cài đặt | Mô tả |
|---------|-------------|
| **Ngôn ngữ Giao diện** | Ngôn ngữ UI ứng dụng |
| **Định dạng Ngày** | Cách ngày xuất hiện |
| **Định dạng Giờ** | 12 hoặc 24 giờ |
| **Ngày đầu tuần** | Ngày bắt đầu lịch |
| **Định dạng Số** | Dấu phân cách thập phân/hàng nghìn |

## Mẹo

### Ghi chú Ngôn ngữ Hỗn hợp

Viết bằng nhiều ngôn ngữ trong một ghi chú:

```markdown
# Dự án Quốc tế

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Phông chữ Theo Ngôn ngữ

Một số ngôn ngữ trông tốt hơn với các phông chữ cụ thể:

1. Mở Cài đặt
2. Đi đến Vẻ ngoài → Phông chữ
3. Đặt phông chữ cho các hệ thống chữ khác nhau

### Chuyển đổi Ngôn ngữ Nhanh

Tạo một phím tắt:

1. Mở Cài đặt → Phím tắt
2. Tìm "Chuyển đổi Ngôn ngữ"
3. Đặt phím tắt ưa thích của bạn

## Khắc phục sự cố

### Ngôn ngữ Không Khả dụng

- Kiểm tra cập nhật ứng dụng
- Ngôn ngữ có thể đang được phát triển
- Đóng góp bản dịch trên GitHub

### Ký tự Không Hiển thị

- Cài đặt các phông chữ cần thiết
- Kiểm tra cài đặt phông chữ
- Thử một phông chữ khác

### Kiểm tra Chính tả Không Hoạt động

- Xác minh kiểm tra chính tả được kích hoạt
- Thêm ngôn ngữ vào kiểm tra chính tả
- Khởi động lại Classic

### RTL Không Hoạt động Đúng

- Đảm bảo phát hiện RTL được bật
- Thử chuyển đổi RTL thủ công
- Kiểm tra các vấn đề nội dung hỗn hợp

## Đóng góp Bản dịch

Giúp dịch Classic:

1. Ghé thăm cộng đồng Classic
2. Tham gia nhóm dịch
3. Gửi bản dịch
4. Giúp cải thiện bản địa hóa
