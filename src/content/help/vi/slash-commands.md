# Lệnh Slash

Lệnh slash là một cách mạnh mẽ để nhanh chóng chèn định dạng và nội dung đặc biệt mà không cần rời bàn phím. Gõ `/` trong trình soạn thảo để mở bảng lệnh.

## Các Lệnh Khả dụng

### Tiêu đề

| Lệnh | Kết quả |
| ---- | ------- |
| `/h1` | Tiêu đề cấp 1 |
| `/h2` | Tiêu đề cấp 2 |
| `/h3` | Tiêu đề cấp 3 |
| `/h4` | Tiêu đề cấp 4 |
| `/h5` | Tiêu đề cấp 5 |
| `/h6` | Tiêu đề cấp 6 |

### Định dạng Văn bản

| Lệnh | Kết quả |
| ---- | ------- |
| `/bold` | **Văn bản in đậm** |
| `/italic` | *Văn bản in nghiêng* |
| `/strikethrough` | ~~Gạch ngang~~ |
| `/code` | Khối mã nội tuyến |
| `/quote` | Trích dẫn khối |

### Danh sách

| Lệnh | Kết quả |
| ---- | ------- |
| `/bullet` | • Danh sách gạch đầu dòng |
| `/numbered` | 1. Danh sách đánh số |
| `/todo` | - [ ] Mục việc cần làm |
| `/check` | - [x] Mục đã hoàn thành |

### Chèn Nội dung

| Lệnh | Kết quả |
| ---- | ------- |
| `/image` | Chèn hình ảnh từ URL hoặc tệp |
| `/link` | Chèn siêu liên kết |
| `/table` | Chèn bảng |
| `/hr` | Quy tắc ngang |
| `/code` | Khối mã với lựa chọn ngôn ngữ |

### Khối Đặc biệt

| Lệnh | Kết quả |
| ---- | ------- |
| `/mermaid` | Khối sơ đồ Mermaid |
| `/katex` | Khối toán học KaTeX |
| `/callout` | Khối callout/admonition |

## Sử dụng Lệnh Slash

1. Gõ `/` trong trình soạn thảo
2. Một menu sẽ xuất hiện với tất cả các lệnh khả dụng
3. Bắt đầu gõ để lọc các lệnh
4. Nhấn `Enter` hoặc nhấp để chèn

## Mẹo Chuyên nghiệp

- **Truy cập nhanh**: Chỉ cần gõ `/` và bắt đầu gõ tên lệnh
- **Tìm kiếm mờ**: Bạn không cần gõ chính xác tên lệnh - Classic sẽ tìm kết quả phù hợp nhất
- **Điều hướng bàn phím**: Sử dụng các phím mũi tên để điều hướng menu
- **Lệnh gần đây**: Các lệnh thường xuyên sử dụng xuất hiện ở đầu danh sách
