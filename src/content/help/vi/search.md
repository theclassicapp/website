# Tìm kiếm

Tìm bất kỳ ghi chú nào ngay lập tức với tìm kiếm mạnh mẽ của Classic. Tìm kiếm toàn văn, bộ lọc và toán tử giúp bạn định vị chính xác những gì bạn cần.

## Tìm kiếm Cơ bản

### Mở Tìm kiếm

| Nền tảng | Phím tắt |
|----------|----------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

Hoặc nhấp vào thanh tìm kiếm trong thanh bên.

### Cách Hoạt động

1. Mở tìm kiếm
2. Bắt đầu gõ
3. Kết quả xuất hiện ngay lập tức
4. Nhấp để mở một ghi chú

## Tính năng Tìm kiếm

### Tìm kiếm Toàn văn

Classic tìm kiếm qua:

- Tiêu đề ghi chú
- Nội dung ghi chú
- Thẻ
- Tên tệp

### Kết quả Thời gian thực

Kết quả cập nhật khi bạn gõ:

- Không cần nhấn Enter
- Phản hồi tức thì
- Khớp mờ được kích hoạt

### Tô sáng Tìm kiếm

Các thuật ngữ tìm thấy được tô sáng trong kết quả:

- Xem bối cảnh xung quanh các khớp
- Xác định ghi chú liên quan nhanh chóng

## Toán tử Tìm kiếm

### Khớp Chính xác

Sử dụng ngoặc kép cho các cụm từ chính xác:

```
"lập kế hoạch dự án"
"ghi chú cuộc họp"
```

### Tìm kiếm Thẻ

Tìm ghi chú với thẻ cụ thể:

```
tag:côngviệc
tag:cuộchọp
tag:khẩn
```

### Loại trừ Thuật ngữ

Sử dụng `-` để loại trừ:

```
dựán -lưu_trữ
cuộchọp -đã_hủy
```

### Tìm kiếm OR

Tìm ghi chú với bất kỳ thuật ngữ nào:

```
dựán OR sáng_kiến
```

### Phạm vi Ngày

Tìm kiếm theo ngày:

```
created:2024-01-01..2024-12-31
modified:tuần_trước
```

### Loại Tệp

Lọc theo phần mở rộng:

```
ext:md
ext:txt
```

## Tìm kiếm Nâng cao

### Kết hợp Toán tử

Xây dựng các truy vấn phức tạp:

```
tag:côngviệc -tag:lưu_trữ created:tuần_trước
"cuộc họp" tag:dựán-alpha
```

### Tìm kiếm Đường dẫn

Tìm kiếm trong các thư mục:

```
path:Dựán/
path:Lưu_trữ/
```

### Tìm kiếm Tiêu đề

Tìm kiếm chỉ tiêu đề:

```
title:Lộtrình
title:Cuộchọp
```

### Tìm kiếm Nội dung

Tìm kiếm chỉ nội dung (loại trừ tiêu đề):

```
content:quantrọng
content:hạnchót
```

## Bộ lọc Tìm kiếm

### Theo Thẻ

1. Gõ tìm kiếm của bạn
2. Nhấp vào một thẻ để lọc
3. Kết quả thu hẹp đến các ghi chú được gắn thẻ

### Theo Ngày

Lọc theo ngày sửa đổi:

- Hôm nay
- Tuần này
- Tháng này
- Phạm vi tùy chỉnh

### Theo Thư mục

Giới hạn tìm kiếm vào một thư mục:

1. Nhấp chuột phải vào một thư mục
2. Chọn "Tìm kiếm trong Thư mục"
3. Kết quả giới hạn vào thư mục đó

## Kết quả Tìm kiếm

### Sắp xếp

Sắp xếp kết quả theo:

- **Mức độ liên quan** (mặc định)
- **Ngày Sửa đổi** (mới nhất trước)
- **Ngày Tạo** (mới nhất trước)
- **Tiêu đề** (theo bảng chữ cái)

### Xem trước Kết quả

Di chuột qua kết quả để xem trước:

- Hiển thị một vài dòng đầu tiên
- Các thuật ngữ khớp được tô sáng
- Thẻ được hiển thị

### Bối cảnh

Xem nơi khớp xảy ra:

- Hiển thị văn bản xung quanh
- Tô sáng khớp
- Nhấp để nhảy đến vị trí

## Lịch sử Tìm kiếm

### Tìm kiếm Gần đây

Truy cập các tìm kiếm trước:

1. Nhấp vào thanh tìm kiếm
2. Xem các tìm kiếm gần đây
3. Nhấp để chạy lại

### Xóa Lịch sử

1. Mở Cài đặt
2. Đi đến Quyền riêng tư
3. Nhấp "Xóa Lịch sử Tìm kiếm"

## Mẹo

### Sử dụng Thuật ngữ Cụ thể

Cụ thể hơn = kết quả tốt hơn:

```
✅ "Ngân sách tiếp thị Q4"
❌ ngân sách
```

### Kết hợp với Thẻ

Thu hẹp kết quả nhanh chóng:

```
dựán tag:khẩn
```

### Sử dụng Loại trừ

Lọc bỏ nhiễu:

```
cuộchọp -tag:lưu_trữ
```

### Lưu Tìm kiếm Thường xuyên

Tạo các tìm kiếm đã lưu:

1. Nhập tìm kiếm của bạn
2. Nhấp "Lưu Tìm kiếm"
3. Đặt tên cho nó
4. Truy cập từ thanh bên

## Điều hướng Bàn phím

| Hành động | Phím tắt |
|----------|----------|
| Mở Tìm kiếm | `Cmd/Ctrl+F` |
| Kết quả Tiếp theo | `Enter` hoặc `Down` |
| Kết quả Trước | `Shift+Enter` hoặc `Up` |
| Đóng Tìm kiếm | `Escape` |

## Cài đặt Tìm kiếm

### Tùy chỉnh trong Cài đặt

| Cài đặt | Tùy chọn |
|---------|---------|
| Phân biệt chữ hoa/thường | Bật/Tắt |
| Tìm kiếm Tiêu đề | Bật/Tắt |
| Tìm kiếm Nội dung | Bật/Tắt |
| Tìm kiếm Thẻ | Bật/Tắt |
| Bao gồm Lưu trữ | Bật/Tắt |

## Trường hợp Sử dụng

### Tìm Ghi chú Cũ

```
"cuộchọp" date:năm_trước
```

### Nghiên cứu Dự án

```
path:Dựán/ tag:nghiên_cứu
```

### Mục Hành động

```
"TODO" OR "FIXME" -tag:hoàn_thành
```

### Công việc Gần đây

```
modified:tuần_này
```

## Khắc phục sự cố

### Không tìm thấy Kết quả

- Kiểm tra chính tả
- Xóa bộ lọc
- Thử các thuật ngữ rộng hơn
- Kiểm tra xem ghi chú có được lưu trữ không

### Tìm kiếm Chậm

- Bộ sưu tập ghi chú lớn có thể mất nhiều thời gian hơn
- Tắt phân biệt chữ hoa/thường
- Giảm phạm vi tìm kiếm

### Thiếu Ghi chú

- Ghi chú có thể ở trong một thư mục khác
- Kiểm tra xem thẻ có đúng không
- Xác minh ghi chú không được lưu trữ
