# Giao diện & Vẻ ngoài

Cá nhân hóa vẻ ngoài của Classic để phù hợp với phong cách của bạn và giảm mỏi mắt.

## Các Tùy chọn Giao diện

### Chế độ Sáng

Giao diện mặc định của Classic có:

- Giao diện sạch, sáng
- Tối ưu cho môi trường có ánh sáng tốt
- Độ tương phản cao để dễ đọc

### Chế độ Tối

Hoàn hảo cho những người làm việc đêm và làm việc trong ánh sáng yếu:

- Giảm mỏi mắt
- Tiêu thụ pin thấp hơn trên màn hình OLED
- Dễ chịu cho mắt trong môi trường tối

### Hệ thống (Tự động)

Theo giao diện hệ điều hành của bạn:

- Tự động chuyển đổi dựa trên cài đặt hệ điều hành
- Không cần chuyển đổi thủ công
- Nhất quán với các ứng dụng khác

## Chuyển đổi Giao diện

### Qua Cài đặt

1. Mở Cài đặt (`Cmd+,` hoặc `Ctrl+,`)
2. Điều hướng đến **Vẻ ngoài**
3. Chọn giao diện ưa thích của bạn

### Qua Bảng lệnh

1. Mở Bảng lệnh (`Cmd+P` hoặc `Ctrl+P`)
2. Gõ "Giao diện"
3. Chọn giao diện ưa thích của bạn

### Phím tắt

| Nền tảng | Phím tắt |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Chu kỳ qua: Sáng → Tối → Hệ thống

## Cài đặt Phông chữ

### Phông chữ Trình soạn thảo

Tùy chỉnh trải nghiệm viết:

1. Mở Cài đặt
2. Đi đến **Vẻ ngoài** → **Trình soạn thảo**
3. Cấu hình các tùy chọn phông chữ

| Cài đặt | Tùy chọn | Mặc định |
| ------- | -------- | -------- |
| Họ Phông chữ | Phông chữ hệ thống + tùy chỉnh | System UI |
| Cỡ Phông chữ | 12-24px | 16px |
| Chiều cao Dòng | 1.0-2.5 | 1.75 |
| Khoảng cách Chữ | -2 đến 4 | 0 |

### Phông chữ UI

Điều chỉnh phông chữ giao diện:

1. Mở Cài đặt
2. Đi đến **Vẻ ngoài** → **Giao diện**
3. Chọn phông chữ và cỡ

### Phông chữ Monospace

Cho các khối mã và nội dung kỹ thuật:

1. Mở Cài đặt
2. Đi đến **Vẻ ngoài** → **Mã**
3. Chọn phông chữ monospace

## Tùy chọn Giao diện

### Thanh bên

Tùy chỉnh hành vi thanh bên:

| Cài đặt | Mô tả |
| ------- | ----------- |
| **Tự động ẩn** | Ẩn thanh bên khi không sử dụng |
| **Độ rộng** | Điều chỉnh độ rộng thanh bên (200-400px) |
| **Hiển thị Biểu tượng** | Bật/tắt hiển thị biểu tượng |

### Bố cục Trình soạn thảo

Chọn trải nghiệm chỉnh sửa của bạn:

- **Chia đôi**: Chỉnh sửa và xem trước cạnh nhau
- **Chỉ Xem trước**: Xem chỉ kết quả được hiển thị
- **Chỉ Chỉnh sửa**: Tập trung vào Markdown thô

### Danh sách Ghi chú

Cấu hình cách ghi chú xuất hiện:

- **Chế độ xem Danh sách**: Compact, một dòng cho mỗi ghi chú
- **Chế độ xem Thẻ**: Thẻ xem trước với các đoạn
- **Chế độ xem Cây**: Cấu trúc thư mục phân cấp

## Màu Nhấn

### Các Màu Khả dụng

Chọn màu nhấn của bạn:

- Xanh dương Classic (mặc định)
- Tím
- Xanh lá
- Cam
- Đỏ
- Hồng
- Xanh ngọc

### Thay đổi Màu Nhấn

1. Mở Cài đặt
2. Đi đến **Vẻ ngoài** → **Màu sắc**
3. Chọn màu nhấn ưa thích của bạn

Màu nhấn ảnh hưởng đến:

- Liên kết và nút
- Tô sáng lựa chọn
- Các phần tử hoạt động
- Chỉ báo tập trung

## Khả năng Truy cập

### Chế độ Độ tương phản Cao

Để hiển thị tốt hơn:

1. Mở Cài đặt
2. Đi đến **Vẻ ngoài** → **Khả năng Truy cập**
3. Kích hoạt **Độ tương phản Cao**

### Giảm Chuyển động

Giảm thiểu hoạt hình:

1. Mở Cài đặt
2. Đi đến **Vẻ ngoài** → **Khả năng Truy cập**
3. Kích hoạt **Giảm Chuyển động**

### Điều chỉnh Phông chữ

- Tăng kích thước phông chữ tối thiểu
- Sử dụng phông chữ thân thiện với dyslexia
- Điều chỉnh tỷ lệ tương phản

## CSS Tùy chỉnh (Nâng cao)

Đối với người dùng nâng cao muốn kiểm soát hoàn toàn:

1. Mở Cài đặt
2. Đi đến **Vẻ ngoài** → **Nâng cao**
3. Nhấp **Mở CSS Tùy chỉnh**
4. Thêm các tùy chỉnh của bạn

### Ví dụ Tùy chỉnh

```css
/* Tiêu đề lớn hơn */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Màu liên kết tùy chỉnh */
a { color: #3b82f6; }

/* Khoảng cách dòng rộng hơn */
.prose p { line-height: 2; }
```

## Đồng bộ Giao diện

Khi bạn đăng nhập (tùy chọn), tùy chọn giao diện của bạn đồng bộ qua các thiết bị:

- Tự động đồng bộ giao diện
- Đồng bộ cài đặt phông chữ
- CSS tùy chỉnh đồng bộ

## Mẹo

### Phù hợp với Quy trình làm việc của bạn

- Sử dụng **Sáng** vào ban ngày, **Tối** vào ban đêm
- Chế độ **Hệ thống** xử lý điều này tự động
- Cân nhắc ánh sáng xung quanh của bạn

### Giảm Mỏi mắt

- Sử dụng chế độ Tối trong ánh sáng yếu
- Tăng cỡ phông chữ cho các phiên dài
- Kích hoạt Độ tương phản Cao nếu cần
- Nghỉ ngơi thường xuyên

### Năng suất

- Học phím tắt chuyển đổi giao diện
- Thiết lập thời gian tự động chuyển đổi
- Sử dụng chế độ Hệ thống để nhất quán
