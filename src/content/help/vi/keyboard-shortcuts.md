# Phím tắt

Làm việc nhanh hơn với các phím tắt của Classic. Tất cả các phím tắt có thể tùy chỉnh trong Cài đặt.

## Phím tắt Toàn cục

Các phím tắt này hoạt động ở bất kỳ đâu trong ứng dụng.

### Chung

| Phím tắt (Windows/Linux) | Phím tắt (macOS) | Hành động |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | Ghi chú mới |
| `Ctrl+P` | `Cmd+P` | Bảng lệnh |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Ghi nhanh toàn cục |
| `Ctrl+S` | `Cmd+S` | Lưu ghi chú |
| `Ctrl+F` | `Cmd+F` | Tìm kiếm |
| `Ctrl+,` | `Cmd+,` | Cài đặt |

### Điều hướng

| Phím tắt (Windows/Linux) | Phím tắt (macOS) | Hành động |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | Đi đến ghi chú đầu tiên |
| `Ctrl+2` | `Cmd+2` | Đi đến ghi chú thứ hai |
| `Ctrl+Left` | `Cmd+Left` | Điều hướng lại |
| `Ctrl+Right` | `Cmd+Right` | Điều hướng tiếp |
| `Ctrl+Up` | `Cmd+Up` | Điều hướng đến ghi chú cha |
| `Ctrl+Down` | `Cmd+Down` | Mở rộng/tập trung ghi chú con đầu tiên |

### Trình soạn thảo

| Phím tắt (Windows/Linux) | Phím tắt (macOS) | Hành động |
| ------------------------ | ----------------- | ------ |
| `Ctrl+B` | `Cmd+B` | In đậm |
| `Ctrl+I` | `Cmd+I` | In nghiêng |
| `Ctrl+K` | `Cmd+K` | Chèn liên kết |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | Gạch ngang |
| `Ctrl+`` | `Cmd+`` | Mã nội tuyến |
| `Ctrl+Shift+C` | `Cmd+Shift+C` | Khối mã |
| `Ctrl+/` | `Cmd+/` | Lệnh slash |
| `Tab` | `Tab` | Thụt lề |
| `Shift+Tab` | `Shift+Tab` | Bỏ thụt lề |

### Quản lý Cửa sổ

| Phím tắt (Windows/Linux) | Phím tắt (macOS) | Hành động |
| ------------------------ | ----------------- | ------ |
| `Ctrl+W` | `Cmd+W` | Đóng ghi chú hiện tại |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | Đóng tất cả ghi chú |
| `F11` | `F11` | Bật/tắt toàn màn hình |
| `Ctrl+M` | `Cmd+M` | Bật/tắt thanh bên |

## Ghi nhanh

| Phím tắt (Windows/Linux) | Phím tắt (macOS) | Hành động |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Mở ghi nhanh |
| `Escape` | `Escape` | Đóng ghi nhanh |
| `Ctrl+Enter` | `Cmd+Enter` | Lưu và đóng |

## Tùy chỉnh Phím tắt

1. Mở **Cài đặt** (`Ctrl+,` hoặc `Cmd+,`)
2. Điều hướng đến **Phím tắt Bàn phím**
3. Nhấp vào bất kỳ phím tắt nào để thay đổi nó
4. Nhấn tổ hợp phím mới của bạn
5. Nhấp **Lưu**

## Mẹo

- Hầu hết các phím tắt có thể tùy chỉnh
- Các xung đột với phím tắt hệ thống sẽ được tô sáng
- Đặt lại về mặc định bằng cách nhấp vào nút đặt lại bên cạnh mỗi phím tắt
