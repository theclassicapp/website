# 서브노트

서브노트를 사용하면 Classic 내에서 중첩되고 계층적인 노트를 만들 수 있습니다. 복잡한 주제를 관리하기 쉬운 조각으로 정리하세요.

## 서브노트란?

서브노트는 노트 안의 노트입니다—계층적 콘텐츠 구조를 만드는 방법입니다.

### 장점

- **정리**: 복잡한 주제를 더 작은 조각으로 나누기
- **중첩**: 여러 수준의 계층 구조 만들기
- **탐색**: 관련 노트 간에 쉽게 탐색
- **유연성**: 필요에 따라 확장 및 축소

## 서브노트 만들기

### 방법 1: 들여쓰기

부모 노트 아래에 콘텐츠를 들여쓰기하여 서브노트 만들기

1. 부모 노트 만들기
2. 새 노트 만들기
3. 새 노트를 부모 위로 드래그
4. 서브노트가 됩니다

### 방법 2: 컨텍스트 메뉴

1. 노트를 마우스 오른쪽 버튼으로 클릭
2. "새 서브노트" 선택
3. 중첩된 노트가 생성됩니다

### 방법 3: 키보드 단축키

1. 노트 선택
2. `Cmd/Ctrl+Shift+N` 누르기
3. 서브노트가 생성됩니다

## 서브노트 관리

### 확장 및 축소

서브노트가 있는 노트 옆의 화살표 클릭

- **확장**: 모든 중첩된 노트 표시
- **축소**: 중첩된 노트 숨기기

### 탐색

- 클릭하여 서브노트 열기
- 화살표 키를 사용하여 계층 구조 탐색
- 브레드크럼이 위치를 표시합니다

### 서브노트 이동

드래그 앤 드롭으로 재구성

1. 서브노트를 클릭하고 유지
2. 새 위치로 드래그
3. 드롭하여 위치 변경

### 메인 노트로 변환

서브노트를 최상위 수준으로 승격

1. 서브노트를 마우스 오른쪽 버튼으로 클릭
2. "루트로 이동" 선택
3. 노트가 메인 노트가 됩니다

## 서브노트 깊이

Classic은 여러 수준의 중첩을 지원합니다

```
📁 프로젝트
  📄 개요
  📁 연구
    📄 사용자 인터뷰
    📄 설문 조사 결과
    📁 분석
      📄 핵심 발견
      📄 권장 사항
  📁 계획
    📄 타임라인
    📄 리소스
```

### 모범 사례

- **깊이 제한**: 명확성을 위해 최대 3-4 수준
- **논리적 그룹화**: 관련 콘텐츠를 함께 그룹화
- **명확한 이름**: 서브노트에 설명적인 이름 사용

## 사용 사례

### 프로젝트 문서화

```
프로젝트 알파
├── 요구 사항
├── 아키텍처
│   ├── 프론트엔드
│   ├── 백엔드
│   └── 데이터베이스
├── 회의 노트
└── 진행 보고서
```

### 지식 기반

```
프로그래밍
├── JavaScript
│   ├── 기본
│   ├── 고급
│   └── 모범 사례
├── Python
│   ├── 기본
│   └── 라이브러리
└── 리소스
```

### 학습 노트

```
생물학 101
├── 세포 구조
│   ├── 막
│   ├── 핵
│   └── 세포소기관
├── 유전학
│   ├── DNA
│   └── RNA
└── 진화
```

## 서브노트와 연결

### 백링크

서브노트는 자동으로 부모 노트에 대한 백링크를 생성합니다

- "연결된 참조" 섹션 보기
- 관련 노트 간 탐색
- 노트 관계 이해

### 위키링크

어디서든 서브노트에 연결

```markdown
자세한 내용은 [[프로젝트/연구/분석]]을 참조하세요.
```

## 팁

### 이름 규칙

명확하고 일관된 이름 사용

```
✅ 좋음:
프로젝트/요구사항/기능적
프로젝트/요구사항/비기능적

❌ 피하세요:
프로젝트/req/func
프로젝트/rq/nf
```

### 브레드크럼 탐색

브레드크럼을 사용하여 위치 이해

```
프로젝트 > 연구 > 분석
```

### 검색

검색은 자동으로 서브노트를 포함합니다

- 모든 수준에서 콘텐츠 찾기
- 노트 깊이로 필터링
- 빠른 탐색

### 템플릿

서브노트에 템플릿 적용

1. 서브노트 만들기
2. 슬래시 명령 `/template` 사용
3. 템플릿 선택

## 키보드 단축키

| 작업 | 단축키 |
|------|--------|
| 새 서브노트 | `Cmd/Ctrl+Shift+N` |
| 모두 확장 | `Cmd/Ctrl+Down` |
| 모두 축소 | `Cmd/Ctrl+Up` |
| 위로 이동 | `Opt/Alt+Up` |
| 아래로 이동 | `Opt/Alt+Down` |
| 승격 | `Opt/Alt+Left` |
| 강등 | `Opt/Alt+Right` |

## 문제 해결

### 서브노트가 표시되지 않음

- 부모가 축소되어 있는지 확인
- 노트가 이동되지 않았는지 확인
- 사이드바 새로고침

### 서브노트를 만들 수 없음

- 부모 노트가 있는지 확인
- 권한 확인
- 컨텍스트 메뉴 방법 시도

### 서브노트 분실

검색을 사용하여 누락된 노트 찾기

1. 검색 열기 (`Cmd/Ctrl+F`)
2. 노트 이름 입력
3. 결과에서 전체 경로 보기
