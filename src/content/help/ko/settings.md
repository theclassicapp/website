# 설정

Classic이 원하는 대로 작동하도록 사용자 정의하세요. 모양부터 에디터 동작까지 모든 것이 구성 가능합니다.

## 설정 열기

### 키보드 단축키

| 플랫폼 | 단축키 |
|--------|--------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### 메뉴를 통해

- **macOS**: Classic → 설정
- **Windows**: 파일 → 설정
- **Linux**: 편집 → 설정

### 명령 팔레트를 통해

1. 명령 팔레트 열기 (`Cmd+P` 또는 `Ctrl+P`)
2. "설정" 입력
3. Enter 누르기

## 설정 검색

설정을 찾을 수 없나요? 검색 바를 사용하세요

1. 설정 열기
2. 검색 상자에 입력
3. 일치하는 설정이 나타납니다
4. 클릭하여 해당 설정으로 이동

## 설정 재설정

### 모든 설정 재설정

기본값으로 되돌리기

1. 설정 열기
2. 하단으로 스크롤
3. "모든 설정 재설정" 클릭
4. 작업 확인

### 카테고리 재설정

특정 카테고리 재설정

1. 카테고리로 이동
2. "기본값으로 재설정" 클릭
3. 해당 카테고리만 재설정됩니다

## 설정 내보내기/가져오기

### 설정 내보내기

구성 저장

1. 설정 열기
2. "설정 내보내기" 클릭
3. 위치 선택
4. JSON 파일로 저장

### 설정 가져오기

저장된 구성 불러오기

1. 설정 열기
2. "설정 가져오기" 클릭
3. JSON 파일 선택
4. 가져오기 확인

## 설정 위치

설정은 로컬에 저장됩니다

| 플랫폼 | 위치 |
|--------|------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## 설정 동기화 (선택 사항)

로그인하면 설정이 기기 간에 동기화될 수 있습니다

- **동기화**: 설정 → 동기화에서 활성화
- **동기화되는 것**: 테마, 글꼴, 단축키
- **동기화되지 않는 것**: 로컬 경로, 암호화 키

## 일반적인 사용자 정의

### 글꼴 크기 증가

1. 설정 → 모양
2. 에디터 글꼴 크기 → 증가
3. 변경 사항이 즉시 적용됩니다

### 테마 변경

1. 설정 → 모양
2. 테마 → 라이트/다크/시스템 선택
3. 앱이 즉시 업데이트됩니다

### 자동 백업 활성화

1. 설정 → 백업 및 복원
2. "자동 백업" 활성화
3. 빈도 및 위치 설정

### 단축키 사용자 정의

1. 설정 → 키보드 단축키
2. 작업 찾기
3. 현재 단축키 클릭
4. 새 키 누르기
5. 저장

## 팁

### 빠른 접근

- `Cmd/Ctrl+,`를 사용하여 설정을 빠르게 열기
- 이름으로 설정 검색
- 카테고리를 사용하여 탐색

### 변경 전 내보내기

주요 변경 전에 설정 내보내기

- 문제가 발생하면 쉽게 복원 가능
- 다른 사람과 구성 공유

### 새 설정 확인

업데이트로 새 설정이 추가될 수 있습니다

- 업데이트 후 확인
- 새 옵션 탐색

## 문제 해결

### 설정이 저장되지 않음

- 파일 권한 확인
- 디스크 공간 확인
- 재설정 및 재구성 시도

### 설정이 예기치 않게 재설정됨

- 파일 손상을 나타낼 수 있음
- 정기적으로 설정 내보내기
- 앱 충돌 확인

### 설정을 찾을 수 없음

- 검색 바 사용
- 다른 카테고리 확인
- 고급 설정에 있을 수 있음
