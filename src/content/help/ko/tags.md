# 태그

태그는 Classic에서 노트를 정리하고 분류하는 강력한 방법입니다. 관련 콘텐츠를 즉시 찾으세요.

## 태그 만들기

### 인라인 태그

노트 콘텐츠에 직접 태그 추가

```markdown
# 회의 노트

Q4 로드맵 논의 #업무 #회의 #계획
```

### 태그 버튼 사용

1. 도구 모음의 태그 아이콘 클릭
2. 태그 이름 입력
3. Enter 누르기

### 키보드 단축키

| 플랫폼 | 단축키 |
|--------|--------|
| **macOS** | `Cmd+T` |
| **Windows/Linux** | `Ctrl+T` |

## 태그 구문

### 기본 태그

```markdown
#프로젝트 #업무 #중요
```

### 중첩 태그

`/`로 태그 계층 구조 만들기

```markdown
#업무/회의
#업무/프로젝트
#개인/취미/독서
```

### 여러 단어 태그

하이픈 또는 밑줄 사용

```markdown
#프로젝트-알파
#회의_노트
#q4-계획
```

## 태그 관리

### 모든 태그 보기

1. 사이드바 열기
2. "태그" 섹션 찾기
3. 모든 태그 보기

### 태그 클라우드

가장 많이 사용되는 태그 시각화

- 더 큰 태그 = 더 많은 노트
- 태그를 클릭하여 필터링
- 태그당 노트 수 보기

### 태그 이름 바꾸기

1. 태그를 마우스 오른쪽 버튼으로 클릭
2. "이름 바꾸기" 선택
3. 새 이름 입력
4. 모든 노트가 자동으로 업데이트됩니다

### 태그 삭제

1. 태그를 마우스 오른쪽 버튼으로 클릭
2. "삭제" 선택
3. 삭제 확인
4. 모든 노트에서 태그 제거됩니다

## 태그로 검색

### 태그 패널

사이드바에서 태그를 클릭하여 노트 필터링

### 검색 구문

검색에 `tag:` 사용

```
tag:업무
tag:업무/회의
```

### 태그 결합

여러 태그가 있는 노트 찾기

```
tag:업무 tag:긴급
```

### 태그 제외

태그가 없는 노트 찾기

```
-tag:보관됨
```

## 태그 모범 사례

### 단순하게 유지

- 노트당 3-5개 태그 사용
- 이름 지정에 일관성 유지
- 태그 남발 피하기

### 계층 구조 사용

관련 태그 정리

```
#업무
  #업무/회의
  #업무/프로젝트
  #업무/마감일

#개인
  #개인/건강
  #개인/재정
```

### 규칙 수립

자신을 위한 태그 표준 정의

- **상태**: `#할일` `#진행중` `#완료`
- **우선순위**: `#긴급` `#높음` `#낮음`
- **유형**: `#회의` `#참조` `#아이디어`

## 태그 제안

`#`를 입력하면 Classic이 기존 태그를 제안합니다

- 최근 태그가 먼저 표시됨
- 퍼지 매칭 지원
- Tab을 눌러 제안 수락

## 태그 통계

설정에서 태그 사용량 보기

- 가장 많이 사용되는 태그
- 사용되지 않는 태그
- 시간에 따른 태그 추세

## 사용 사례

### 프로젝트 관리

```markdown
# 프로젝트 킥오프 #프로젝트-알파 #회의 #2024-01-15

참석자: 팀 알파
상태: #할일
```

### 지식 기반

```markdown
# JavaScript 클로저 #프로그래밍 #javascript #참조 #고급

클로저는 ...에 접근할 수 있는 함수입니다.
```

### 데일리 노트

```markdown
# 2024-01-15

- 오전 스탠드업 #업무 #회의
- 코드 리뷰 #업무 #개발
- 헬스장 세션 #개인 #건강
```

### 독서 노트

```markdown
# 아토믹 해빗 노트 #독서 #책 #생산성

책에서 얻은 핵심 내용...
```

|------|--------|

| 태그 검색 | `Cmd/Ctrl+F`, then type `tag:` |
| 태그 패널 표시 | `Cmd/Ctrl+Shift+T` |

## 팁

### 일관된 이름 지정

규칙을 따르세요

```
✅ #회의-노트
✅ #회의_노트
❌ #회의 노트 (공백 허용되지 않음)
❌ #회의노트 (일관성 없음)
```

### 과도한 태그 지정 피하기

너무 많은 태그는 관리할 수 없게 됩니다

```
✅ #업무 #긴급 #프로젝트-알파
❌ #업무 #긴급 #중요 #오늘 #월요일 #사무실 #책상 #컴퓨터
```

### 정기적인 정리

정기적으로 태그 검토 및 통합

- 유사한 태그 병합
- 사용되지 않는 태그 제거
- 규칙 업데이트

## 고급 기능

### 태그 기반 스마트 폴더

태그를 기반으로 자동 채워지는 폴더 만들기

1. 새 폴더 만들기
2. 필터 설정: `tag:프로젝트`
3. 태그가 지정된 모든 노트가 표시됩니다

### YAML의 태그

노트 프론트매터에 태그 추가

```yaml
---
tags:
  - 업무
  - 회의
  - 프로젝트-알파
---
```

### 태그 별칭

설정에서 태그 별칭 설정

- `#할일` = `#작업` = `#액션`
- 유사한 태그 통합
