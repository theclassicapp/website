# 첫 번째 노트 만들기

Classic에서 첫 번째 노트를 만들어 보세요. 간단하고 직관적입니다.

## 새 노트 만들기

### 사이드바 사용

1. 사이드바에서 **+** 버튼 클릭
2. 새 노트가 생성됩니다
3. 내용 입력 시작

### 키보드 단축키 사용

| 플랫폼 | 단축키 |
| ------ | ------ |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### 명령 팔레트 사용

1. 명령 팔레트 열기 (`Cmd+P` 또는 `Ctrl+P`)
2. "새 노트" 입력
3. Enter 누르기

## 노트 작성하기

### 기본 편집

그냥 입력을 시작하세요! Classic은 작성하는 동안 자동 저장합니다.

### 제목 추가

노트의 첫 번째 줄이 제목이 됩니다. 최상의 결과를 위해 제목을 사용하세요.

```markdown
# 나의 첫 번째 노트

이것은 노트의 내용입니다.
```

### 텍스트 서식

Markdown을 사용하여 텍스트 서식 지정

- **굵게**: `**텍스트**` 또는 `Cmd/Ctrl+B`
- **기울임꼴**: `*텍스트*` 또는 `Cmd/Ctrl+I`
- **취소선**: `~~텍스트~~`
- **코드**: `` `코드` ``

## 저장

Classic은 입력하는 대로 노트를 자동 저장합니다. 수동으로 저장할 필요가 없습니다.

### 수동 저장

필요한 경우 강제 저장 가능

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## 노트 정리하기

### 태그 추가

나중에 쉽게 찾을 수 있도록 노트에 태그 지정

1. `#` 다음에 태그 이름 입력
2. 또는 도구 모음의 태그 버튼 사용
3. 여러 태그 지원

예시

```markdown
# 회의 노트

Q4 계획 논의 #업무 #회의
```

### 폴더로 이동

1. 사이드바에서 노트를 마우스 오른쪽 버튼으로 클릭
2. "폴더로 이동" 선택
3. 대상 폴더 선택 또는 생성

### 중요한 노트 고정

1. 노트를 마우스 오른쪽 버튼으로 클릭
2. "상단에 고정" 선택
3. 노트가 목록 상단에 표시됩니다

## 노트 미리보기

서식이 적용된 노트가 어떻게 보이는지 확인

- **미리보기** 버튼 클릭
- 또는 `Cmd/Ctrl+Shift+P` 사용
- 편집 모드와 미리보기 모드 전환

## 팁

### 노트 제목

- 설명적인 제목 사용
- 첫 번째 줄 = 노트 제목
- 명확성을 위해 H1 (`#`) 사용

### 자동 저장

- 작업 손실 걱정 없음
- 변경 사항이 자동으로 저장됨
- 상태 표시줄에서 저장 상태 확인

### 빠른 캡처

빠르게 무언가를 적어야 하나요?

- 빠른 캡처 사용 (`Cmd/Ctrl+Shift+N`)
- 즉시 노트 생성
- 순간적인 생각에 적합
