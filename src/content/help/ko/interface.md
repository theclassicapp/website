# 인터페이스 이해하기

Classic의 인터페이스는 깔끔하고 직관적이며 방해받지 않도록 설계되었습니다. 주요 구성 요소를 둘러보겠습니다.

## 메인 레이아웃

Classic은 3개 패널 레이아웃을 가지고 있습니다:

1. **사이드바** (왼쪽) - 탐색 및 노트 정리
2. **에디터** (중앙) - 글쓰기 공간
3. **미리보기/속성** (오른쪽) - 선택적 패널

## 사이드바

사이드바에는 여러 주요 섹션이 있습니다.

### 노트 목록

- **모든 노트**: 모든 노트를 한 목록에서 보기
- **데일리 노트**: 오늘의 저널에 빠르게 접근
- **즐겨찾기**: 자주 사용하는 노트 즐겨찾기
- **폴더**: 계층 구조로 노트 정리

### 탐색

- **검색 바**: 노트를 즉시 찾기
- **태그**: 태그로 탐색
- **백링크**: 노트 간 연결 보기

### 작업

- **새 노트**: 새 노트 만들기
- **새 폴더**: 폴더로 정리
- **가져오기**: 외부 콘텐츠 가져오기

## 에디터

에디터는 노트를 작성하고 서식을 지정하는 곳입니다.

### 기능

- **Markdown 지원**: 전체 Markdown 구문
- **실시간 미리보기**: 입력할 때 서식이 적용된 텍스트 보기
- **구문 강조**: 코드 블록 강조
- **슬래시 명령**: 빠른 작업을 위해 `/` 입력

### 에디터 모드

| 모드 | 설명 |
|------|------|
| **편집만** | 원본 Markdown 보기 |
| **미리보기만** | 렌더링된 결과 보기 |
| **분할 보기** | 편집과 미리보기를 나란히 |

### 도구 모음

빠른 서식 작업

- 굵게, 기울임꼴, 취소선
- 제목 (H1-H6)
- 목록 (글머리 기호, 번호, 할 일)
- 링크 및 이미지
- 코드 블록
- 표

## 상태 표시줄

에디터 하단에서:

- **단어/문자 수**: 글쓰기 추적
- **마지막 저장**: 노트가 마지막으로 저장된 시간
- **노트 경로**: 노트가 위치한 곳
- **동기화 상태**: 선택적 동기화 사용 시

## 컨텍스트 메뉴

마우스 오른쪽 버튼으로 상황에 맞는 작업

### 노트 컨텍스트 메뉴

- 이름 바꾸기
- 폴더로 이동
- 복제
- 삭제
- 내보내기
- 링크 복사

### 에디터 컨텍스트 메뉴

- 잘라내기, 복사, 붙여넣기
- 서식 옵션
- 링크/이미지 삽입
- 찾기 및 바꾸기

## 키보드 탐색

키보드로 인터페이스 탐색

| 작업 | 단축키 |
|------|--------|
| 사이드바 토글 | `Cmd/Ctrl + M` |
| 검색 포커스 | `Cmd/Ctrl + F` |
| 명령 팔레트 | `Cmd/Ctrl + P` |
| 새 노트 | `Cmd/Ctrl + N` |

## 사용자 정의

인터페이스를 원하는 대로 조정하세요.

### 사이드바 너비

1. 사이드바 가장자리에 마우스를 올리세요.
2. 드래그하여 크기 조정
3. 또는 설정에서 정확한 너비 지정

### 테마

라이트 모드와 다크 모드 전환

- 헤더의 테마 토글 사용
- 또는 시스템 환경설정 따르기 설정

### 글꼴 크기

1. 설정 열기
2. 모양으로 이동
3. 에디터 글꼴 크기 조정

## 팁

### 집중 모드

모든 패널을 숨겨 방해 없는 글쓰기

1. 명령 팔레트 열기
2. "집중 모드" 입력
3. Enter 누르기

### 빠른 접근

- 자주 사용하는 노트를 즐겨찾기에 고정
- 일반 작업에 키보드 단축키 사용
- 설정에서 도구 모음 사용자 정의
