# 슬래시 명령

슬래시 명령은 키보드를 벗어나지 않고 빠르게 서식과 특수 콘텐츠를 삽입하는 강력한 방법입니다. 에디터에서 `/`를 입력하여 명령 팔레트를 엽니다.

## 사용 가능한 명령

### 제목

| 명령 | 결과 |
| ---- | ---- |
| `/h1` | 1단계 제목 |
| `/h2` | 2단계 제목 |
| `/h3` | 3단계 제목 |
| `/h4` | 4단계 제목 |
| `/h5` | 5단계 제목 |
| `/h6` | 6단계 제목 |

### 텍스트 서식

| 명령 | 결과 |
| ---- | ---- |
| `/bold` | **굵은 텍스트** |
| `/italic` | *기울임꼴 텍스트* |
| `/strikethrough` | ~~취소선~~ |
| `/code` | 인라인 코드 블록 |
| `/quote` | 인용문 |

### 목록

| 명령 | 결과 |
| ---- | ---- |
| `/bullet` | • 글머리 기호 목록 |
| `/numbered` | 1. 번호 매기기 목록 |
| `/todo` | - [ ] 할 일 항목 |
| `/check` | - [x] 완료된 항목 |

### 콘텐츠 삽입

| 명령 | 결과 |
| ---- | ---- |
| `/image` | URL 또는 파일에서 이미지 삽입 |
| `/link` | 하이퍼링크 삽입 |
| `/table` | 표 삽입 |
| `/hr` | 수평선 |
| `/code` | 언어 선택이 있는 코드 블록 |

### 특수 블록

| 명령 | 결과 |
| ---- | ---- |
| `/mermaid` | Mermaid 다이어그램 블록 |
| `/katex` | KaTeX 수학 블록 |
| `/callout` | 콜아웃/주의 블록 |

## 슬래시 명령 사용법

1. 에디터에서 `/` 입력
2. 사용 가능한 모든 명령이 있는 메뉴가 나타납니다
3. 입력하여 명령 필터링
4. `Enter`를 누르거나 클릭하여 삽입

## 프로 팁

- **빠른 접근**: `/`를 입력하고 명령 이름 입력 시작
- **퍼지 검색**: 정확한 명령 이름을 입력할 필요가 없습니다 - Classic이 가장 가까운 일치 항목을 찾습니다
- **키보드 탐색**: 화살표 키를 사용하여 메뉴 탐색
- **최근 명령**: 자주 사용하는 명령이 목록 상단에 표시됩니다
