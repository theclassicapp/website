# 검색

Classic의 강력한 검색으로 모든 노트를 즉시 찾으세요. 전체 텍스트 검색, 필터 및 연산자가 필요한 것을 정확하게 찾는 데 도움이 됩니다.

## 기본 검색

### 검색 열기

| 플랫폼 | 단축키 |
|--------|--------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

또는 사이드바의 검색 바를 클릭하세요.

### 작동 방식

1. 검색 열기
2. 입력 시작
3. 결과가 즉시 나타납니다
4. 클릭하여 노트 열기

## 검색 기능

### 전체 텍스트 검색

Classic은 다음을 검색합니다:

- 노트 제목
- 노트 콘텐츠
- 태그
- 파일 이름

### 실시간 결과

입력하는 대로 결과 업데이트

- Enter를 누를 필요 없음
- 즉각적인 피드백
- 퍼지 매칭 활성화

### 검색 강조

발견된 용어가 결과에서 강조됩니다

- 일치 항목 주변의 컨텍스트 보기
- 관련 노트를 빠르게 식별

## 검색 연산자

### 정확한 일치

정확한 문구에 따옴표 사용

```
"프로젝트 계획"
"회의 노트"
```

### 태그 검색

특정 태그가 있는 노트 찾기

```
tag:업무
tag:회의
tag:긴급
```

### 용어 제외

제외하려면 `-` 사용

```
프로젝트 -보관됨
회의 -취소됨
```

### OR 검색

용어가 있는 노트 찾기

```
프로젝트 OR 이니셔티브
```

### 날짜 범위

날짜로 검색

```
created:2024-01-01..2024-12-31
modified:last-week
```

### 파일 유형

확장자로 필터링

```
ext:md
ext:txt
```

## 고급 검색

### 연산자 결합

복잡한 쿼리 작성

```
tag:업무 -tag:보관됨 created:last-week
"회의" tag:프로젝트-알파
```

### 경로 검색

폴더 내에서 검색

```
path:프로젝트/
path:보관함/
```

### 제목 검색

제목만 검색

```
title:로드맵
title:회의
```

### 콘텐츠 검색

콘텐츠만 검색 (제목 제외)

```
content:중요
content:마감일
```

## 검색 필터

### 태그별

1. 검색어 입력
2. 태그를 클릭하여 필터링
3. 태그가 지정된 노트로 결과 좁히기

### 날짜별

수정 날짜로 필터링

- 오늘
- 이번 주
- 이번 달
- 사용자 정의 범위

### 폴더별

폴더로 검색 제한

1. 폴더를 마우스 오른쪽 버튼으로 클릭
2. "폴더에서 검색" 선택
3. 해당 폴더로 결과 제한

## 검색 결과

### 정렬

결과 정렬 기준:

- **관련성** (기본값)
- **수정 날짜** (최신순)
- **생성 날짜** (최신순)
- **제목** (알파벳순)

### 결과 미리보기

결과 위에 마우스를 올려 미리보기

- 처음 몇 줄 표시
- 일치하는 용어 강조
- 태그 표시

### 컨텍스트

일치가 발생한 위치 보기

- 주변 텍스트 표시
- 일치 항목 강조
- 클릭하여 위치로 이동

## 검색 기록

### 최근 검색

이전 검색 액세스

1. 검색 바 클릭
2. 최근 검색 보기
3. 클릭하여 다시 실행

### 기록 지우기

1. 설정 열기
2. 개인정보로 이동
3. "검색 기록 지우기" 클릭

## 팁

### 구체적인 용어 사용

더 구체적일수록 더 나은 결과

```
✅ "Q4 마케팅 예산"
❌ 예산
```

### 태그와 결합

결과를 빠르게 좁히기

```
프로젝트 tag:긴급
```

### 제외 사용

노이즈 필터링

```
회의 -tag:보관됨
```

### 자주 사용하는 검색 저장

저장된 검색 만들기

1. 검색어 입력
2. "검색 저장" 클릭
3. 이름 지정
4. 사이드바에서 액세스

## 키보드 탐색

| 작업 | 단축키 |
|------|--------|
| 검색 열기 | `Cmd/Ctrl+F` |
| 다음 결과 | `Enter` 또는 `Down` |
| 이전 결과 | `Shift+Enter` 또는 `Up` |
| 검색 닫기 | `Escape` |

## 검색 설정

### 설정에서 사용자 정의

| 설정 | 옵션 |
|------|------|
| 대소문자 구분 | 켜기/끄기 |
| 제목 검색 | 켜기/끄기 |
| 콘텐츠 검색 | 켜기/끄기 |
| 태그 검색 | 켜기/끄기 |
| 보관됨 포함 | 켜기/끄기 |

## 사용 사례

### 오래된 노트 찾기

```
"회의" date:last-year
```

### 프로젝트 연구

```
path:프로젝트/ tag:연구
```

### 실행 항목

```
"TODO" OR "FIXME" -tag:완료
```

### 최근 작업

```
modified:this-week
```

## 문제 해결

### 결과 없음

- 철자 확인
- 필터 제거
- 더 넓은 용어 시도
- 노트가 보관되었는지 확인

### 느린 검색

- 큰 노트 컬렉션은 더 오래 걸릴 수 있음
- 대소문자 구분 비활성화
- 검색 범위 줄이기

### 노트 누락

- 노트가 다른 폴더에 있을 수 있음
- 태그가 올바른지 확인
- 노트가 보관되지 않았는지 확인
