# 빠른 시작 가이드

Classic에 오신 것을 환영합니다! 이 가이드는 몇 분 안에 Classic을 시작하는 데 도움을 드립니다.

## 시스템 요구 사항

- **macOS**: macOS 12 (Monterey) 이상
- **Windows**: Windows 10 이상
- **Linux**: Ubuntu 20.04+ / Fedora 36+ 이상

## 설치

1. [웹사이트](/)에서 Classic을 다운로드하세요.
2. 애플리케이션을 실행하세요.
3. 글쓰기를 시작하세요!

## 첫 실행 경험

Classic을 처음 열면 글쓰기에 집중할 수 있도록 깔끔하고 미니멀한 인터페이스가 표시됩니다.

![Classic 앱 스크린샷]

### 사이드바

왼쪽에서 다음을 찾을 수 있습니다:
- **데일리 노트** - 자동 일일 저널
- **검색** - 모든 노트에서 빠르게 찾기
- **명령 팔레트** - `Cmd/Ctrl+P` (Windows/Linux) 또는 `Cmd+Shift+P` (macOS)를 눌러 명령 팔레트 열기
- **아웃라인** - 트리 구조로 노트 탐색 및 정리

### 에디터

중앙 영역은 **Markdown 에디터**입니다:
- 실시간 미리보기
- 구문 강조
- 풍부한 서식 도구

### 첫 번째 노트 만들기

1. 사이드바에서 **새 노트** 버튼을 클릭하거나 `Ctrl/Cmd+N` (Windows/Linux) 또는 `Cmd+N` (macOS)를 누르세요.
2. 노트에 제목을 지정하세요.
3. Markdown으로 글쓰기를 시작하세요!

```markdown
# 나의 첫 번째 노트

Classic에 오신 것을 환영합니다! 여기에 생각과 아이디어를 기록하겠습니다.
```
