# 백링크

백링크는 노트 간의 연결을 보여주어 관련 아이디어의 네트워크를 만듭니다. 몰랐던 관계를 발견하세요.

## 백링크란?

백링크는 현재 노트에 연결되는 노트를 보여줍니다. 지식 그래프의 연결입니다.

### 장점

- **연결 발견**: 자동으로 관련 노트 찾기
- **컨텍스트**: 아이디어가 어떻게 관련되어 있는지 이해
- **탐색**: 연결된 노트 간 이동
- **지식 그래프**: 사고 과정 시각화

## 백링크 작동 방식

다른 노트에 대한 링크를 만들면 Classic이 자동으로 백링크를 만듭니다.

```markdown
노트 A에서:
자세한 내용은 [[프로젝트 계획]]을 참조하세요.

결과:
- 노트 A가 "프로젝트 계획"에 연결됨
- "프로젝트 계획"이 노트 A에 대한 백링크를 표시함
```

## 백링크 보기

### 백링크 패널

모든 참조를 보려면 백링크 패널 열기

1. 노트 열기
2. 사이드바의 백링크 아이콘 클릭
3. 이 노트에 연결된 모든 노트 보기

### 인라인 백링크

백링크는 각 노트 하단에 나타납니다

- 소스 노트 이름 표시
- 컨텍스트 스니펫 표시
- 클릭하여 탐색

## 링크 만들기

### 위키링크 구문

이중 대괄호를 사용하여 노트에 연결

```markdown
[[노트 이름]]
[[다른 노트]]
```

### 표시 텍스트

링크 텍스트 사용자 정의

```markdown
[[프로젝트 알파|알파 프로젝트]]
[[회의 노트|어제 회의]]
```

### 제목에 연결

특정 섹션에 연결

```markdown
[[노트 이름#제목]]
[[노트 이름#섹션|섹션으로 이동]]
```

## 백링크 유형

### 직접 링크

명시적으로 만든 위키링크

```markdown
관련: [[유사한 노트]]
```

### 연결되지 않은 언급

제목을 언급하지만 연결되지 않은 노트

```markdown
노트가 "프로젝트 알파"를 언급하지만 연결되지 않음
```

Classic은 이들을 링크로 변환할 것을 제안합니다.

## 백링크 패널

### 기능

- **필터**: 백링크 검색
- **정렬**: 날짜, 이름 또는 관련성별
- **미리보기**: 마우스를 올려 콘텐츠 미리보기
- **컨텍스트**: 주변 텍스트 보기

### 작업

- 클릭하여 연결된 노트 열기
- 마우스 오른쪽 버튼으로 옵션 보기
- 연결되지 않은 언급 변환

## 사용 사례

### 지식 관리

개인 위키 구축

```markdown
# 머신러닝

[[인공지능]]과 관련됨
[[통계]]를 기반으로 함
[[자연어 처리]]에서 사용됨
```

### 프로젝트 문서화

관련 문서 연결

```markdown
# API 디자인

[[데이터베이스 스키마]] 참조
[[인증 흐름]]에 의존
[[프론트엔드 통합]]도 참조
```

### 연구

개념 및 출처 연결

```markdown
# 연구 노트

[[출처 논문 A]]를 기반으로 함
[[출처 논문 B]]와 모순됨
[[가설 1]]을 지지함
```

### 회의 노트

회의와 프로젝트 연결

```markdown
# 회의: Q4 계획

프로젝트: [[프로젝트 알파]]
이전: [[Q3 리뷰]]
다음: [[Q4 킥오프]]
```

## 그래프 보기

백링크를 네트워크로 시각화

1. 그래프 보기 열기 (`Cmd/Ctrl+G`)
2. 모든 노트를 노드로 보기
3. 선이 연결을 표시
4. 노드를 클릭하여 탐색

### 그래프 컨트롤

- **확대**: 스크롤하여 확대
- **이동**: 드래그하여 이동
- **필터**: 태그별 표시/숨기기
- **포커스**: 노트의 연결 강조

## 팁

### 넉넉하게 연결

다른 개념을 참조할 때마다 링크 만들기

```markdown
✅ 좋음:
[[Q4 로드맵]]과 [[예산 할당]]을 논의했습니다.

❌ 피하세요:
Q4 로드맵과 예산 할당을 논의했습니다.
```

### 일관된 이름 사용

신뢰할 수 있는 연결을 위해 노트 제목과 정확히 일치

```markdown
✅ [[회의 노트]]
❌ [[회의 노트]] (대소문자 중요)
```

### 백링크 검토

정기적으로 백링크 확인:

- 잊혀진 연결 발견
- 병합할 노트 찾기
- 지식 격차 식별

### 링크 미리보기

열지 않고 미리보려면 링크 위에 마우스 올리기

- 처음 몇 줄 보기
- 올바른 노트인지 확인
- 빠른 참조

## 키보드 단축키

| 작업 | 단축키 |
|------|--------|
| 백링크 열기 | `Cmd/Ctrl+Shift+B` |
| 링크 만들기 | `[[` (자동 완성) |
| 그래프 보기 열기 | `Cmd/Ctrl+G` |
| 링크 탐색 | 링크 간 `Tab` |

## 고급 기능

### 임베딩

다른 노트에서 콘텐츠 임베드

```markdown
![[노트 이름]]
![[노트 이름#섹션]]
```

### 트랜스클루전

콘텐츠를 인라인으로 포함

```markdown
## 관련 콘텐츠
![[공유 템플릿#헤더]]
```

### 링크 색상

링크는 다음에 따라 색상이 변경됩니다:

- **파란색**: 방문하지 않은 링크
- **보라색**: 방문한 링크
- **빨간색**: 끊어진 링크 (노트가 존재하지 않음)

## 문제 해결

### 백링크 누락

- 링크 구문 `[[노트 이름]]` 확인
- 노트 제목이 정확히 일치하는지 확인
- 백링크 패널 새로고침

### 끊어진 링크

- 노트 이름이 변경되었을 수 있음
- 노트가 삭제되었을 수 있음
- 검색을 사용하여 올바른 노트 찾기

### 중복 링크

- 유사한 노트 통합
- 이름이 변경된 노트에 리디렉션 사용
- 제목 변형 확인
