# 명령 팔레트

명령 팔레트는 Classic의 모든 작업에 대한 게이트웨이입니다. 간단한 키보드 단축키를 누르고 모든 기능에 즉시 액세스하세요.

## 명령 팔레트 열기

| 플랫폼 | 단축키 |
| ------ | ------ |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## 사용 방법

1. 명령 팔레트 열기 (`Cmd+P` 또는 `Ctrl+P`)
2. 검색할 명령 입력
3. 화살표 키로 탐색
4. `Enter`를 눌러 실행

## 명령 카테고리

### 파일 작업

- **새 노트**: 새 노트 만들기
- **새 데일리 노트**: 오늘의 데일리 노트 만들기/열기
- **노트 열기**: 기존 노트 검색 및 열기
- **노트 저장**: 현재 노트 저장
- **노트 삭제**: 현재 노트 삭제
- **노트 이름 바꾸기**: 현재 노트 이름 바꾸기

### 탐색

- **노트로 이동**: 노트로 점프
- **데일리 노트로 이동**: 특정 날짜 열기
- **태그로 이동**: 태그별로 노트 탐색
- **뒤로 가기**: 기록에서 뒤로 탐색
- **앞으로 가기**: 기록에서 앞으로 탐색

### 에디터

- **미리보기 토글**: 편집 및 미리보기 모드 전환
- **사이드바 토글**: 사이드바 표시/숨기기
- **집중 모드**: 방해 없는 글쓰기 시작
- **날짜 삽입**: 현재 날짜/시간 추가
- **템플릿 삽입**: 템플릿 삽입

### 서식

- **굵게**: 텍스트를 굵게 만들기
- **기울임꼴**: 텍스트를 기울임꼴로 만들기
- **취소선**: 텍스트에 취소선
- **코드**: 코드로 서식 지정
- **링크**: 링크 삽입
- **제목**: 제목 서식 적용

### 정리

- **태그 추가**: 현재 노트에 태그 지정
- **폴더로 이동**: 노트를 폴더로 이동
- **노트 보관**: 현재 노트 보관
- **노트 고정**: 목록 상단에 노트 고정

### 보기

- **테마 토글**: 라이트/다크 전환
- **확대**: 글꼴 크기 증가
- **축소**: 글꼴 크기 감소
- **확대/축소 재설정**: 기본 크기로 재설정

## 팁

### 빠른 접근

- 명령은 퍼지 검색됩니다
- 최근 명령이 먼저 표시됩니다
- 자주 사용하는 명령 고정

### 키보드 탐색

- `↑`/`↓` - 결과 탐색
- `Enter` - 명령 실행
- `Escape` - 팔레트 닫기
- `Tab` - 카테고리 순환

### 사용자 정의 명령

1. 설정 열기
2. 명령 팔레트로 이동
3. 사용자 정의 명령 추가
4. 키보드 단축키 정의

## 자주 사용하는 명령

자주 사용하게 될 몇 가지 명령:

| 명령 | 기능 |
| ---- | ---- |
| `새 노트` | 빈 노트 만들기 |
| `노트 열기` | 모든 노트 검색 |
| `새 데일리 노트` | 오늘의 저널 |
| `미리보기 토글` | 미리보기 vs 편집 |
| `태그 추가` | 현재 노트에 태그 지정 |
| `빠른 캡처` | 전역 캡처 |

## 명령 사용자 정의

### 단축키 변경

1. 설정 열기 (`Cmd+,` 또는 `Ctrl+,`)
2. 키보드 단축키로 이동
3. 명령 찾기
4. 현재 단축키 클릭
5. 새 단축키 누르기
6. 저장

### 사용자 정의 명령 만들기

고급 사용자는 사용자 정의 명령을 만들 수 있습니다

1. 설정 열기
2. 명령으로 이동
3. "사용자 정의 명령 추가" 클릭
4. 명령 이름 정의
5. 작업 추가
6. 단축키 설정 (선택 사항)
