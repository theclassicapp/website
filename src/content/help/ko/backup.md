# 백업 및 복원

Classic의 내장 백업 시스템으로 노트를 안전하게 보호하세요. 작업을 다시는 잃지 마세요.

## 자동 백업

### 작동 방식

Classic은 정기적으로 노트의 백업을 자동으로 만듭니다

- **기본 빈도**: 30분마다
- **백업 위치**: 노트와 동일한 폴더
- **보존**: 마지막 10개 백업 유지
- **형식**: 압축 아카이브 (`.cbak`)

### 자동 백업 구성

1. 설정 열기
2. **백업 및 복원**으로 이동
3. 다음 구성

| 설정 | 옵션 | 기본값 |
| ---- | ---- | ------ |
| 자동 백업 활성화 | 켜기/끄기 | 켜기 |
| 백업 빈도 | 15분 / 30분 / 1시간 / 4시간 | 30분 |
| 백업 보관 | 5 / 10 / 20 / 50 | 10 |
| 백업 위치 | 사용자 정의 경로 | 노트 폴더 |

## 수동 백업

### 지금 백업 만들기

1. 명령 팔레트 열기 (`Cmd/Ctrl+P`)
2. "백업" 입력
3. **지금 백업 만들기** 선택
4. 위치 선택 (선택 사항)
5. 백업이 생성됩니다

### 백업 파일 형식

백업은 다음을 포함하는 `.cbak` 파일로 저장됩니다

- 모든 노트 (Markdown 파일)
- 첨부 파일
- 설정
- 메타데이터 (태그, 링크 등)

## 백업에서 복원

### 복원 프로세스

1. 설정 열기
2. **백업 및 복원**으로 이동
3. **백업에서 복원** 클릭
4. 백업 파일 선택
5. 복원 옵션 선택
6. 복원 확인

### 복원 옵션

| 옵션 | 설명 |
| ---- | ---- |
| **모두 교체** | 현재 노트를 백업으로 완전히 교체 |
| **병합** | 백업을 기존 노트와 결합 |
| **미리보기** | 복원 전 백업 내용 보기 |

### 복원되는 것

- ✅ 모든 노트 및 폴더
- ✅ 첨부 파일
- ✅ 태그 및 메타데이터
- ✅ 노트 기록
- ⚠️ 설정 (선택 사항)
- ❌ 애플리케이션 환경설정

## 백업 위치

### 기본 위치

```
~/Classic/backups/
├── backup-2024-01-15-10-30.cbak
├── backup-2024-01-15-11-00.cbak
└── backup-2024-01-15-11-30.cbak
```

### 사용자 정의 위치

1. 설정 열기
2. 백업 및 복원으로 이동
3. "백업 위치 변경" 클릭
4. 선호하는 폴더 선택

### 클라우드 백업

백업 위치를 클라우드 동기화 폴더로 지정

- **iCloud**: `~/Library/Mobile Documents/com~apple~CloudDocs/Classic Backups`
- **Dropbox**: `~/Dropbox/Classic Backups`
- **Google Drive**: `~/Google Drive/Classic Backups`
- **OneDrive**: `~/OneDrive/Classic Backups`

## 모범 사례

### 백업 일정

- **자동 백업**: 30분 간격으로 활성화 유지
- **수동 백업**: 주요 변경 전 생성
- **오프사이트 백업**: 정기적으로 외부 저장소에 백업 복사

### 백업 검증

정기적으로 백업 검증

1. 테스트 복원 생성
2. 노트 내용 확인
3. 첨부 파일 확인
4. 메타데이터 확인

### 백업 순환

- 최근 백업은 로컬에 유지
- 오래된 백업은 외부 저장소에 보관
- 공간 절약을 위해 매우 오래된 백업 삭제

## 문제 해결

### 백업 실패

**가능한 원인**

- 디스크 공간 부족
- 권한 문제
- 파일 사용 중

**해결 방법**

1. 사용 가능한 디스크 공간 확인
2. 폴더 권한 확인
3. 다른 애플리케이션 닫기
4. 다른 위치 시도

### 복원 실패

**가능한 원인**

- 손상된 백업 파일
- 호환되지 않는 버전
- 공간 부족

**해결 방법**

1. 오래된 백업 시도
2. 백업 파일 무결성 확인
3. 디스크 공간 확보
4. 지원팀에 문의

### 백업 누락

**가능한 원인**

- 자동 백업 비활성화
- 백업 위치 변경
- 백업 삭제됨

**해결 방법**

1. 백업 설정 확인
2. 백업 위치 확인
3. 자동 백업 활성화
4. 지금 수동 백업 생성

## 고급 옵션

### 암호화된 백업

추가 보안을 위해

1. 설정 열기
2. 백업 및 복원으로 이동
3. "백업 암호화" 활성화
4. 암호 설정
5. 이 암호를 기억하세요 - 복구할 수 없습니다

### 증분 백업

증분 백업으로 공간 절약

- 마지막 백업 이후의 변경 사항만 저장
- 전체 복원이 완전한 상태를 재구성
- 백업 시간 및 크기 감소

### 백업 제외

백업에서 특정 콘텐츠 제외

1. 설정 열기
2. 백업 및 복원으로 이동
3. "제외" 클릭
4. 제외할 폴더 또는 파일 추가
