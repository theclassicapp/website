# 테마 및 모양

Classic의 모양을 개인화하여 스타일에 맞추고 눈의 피로를 줄이세요.

## 테마 옵션

### 라이트 모드

Classic의 기본 테마 기능:

- 깔끔하고 밝은 인터페이스
- 잘 조명된 환경에 최적
- 가독성을 위한 높은 대비

### 다크 모드

올빼미족과 저조도 작업에 완벽

- 눈의 피로 감소
- OLED 화면에서 배터리 소모 감소
- 어두운 환경에서 눈에 편안함

### 시스템 (자동)

운영 체제의 테마 따르기

- OS 설정에 따라 자동 전환
- 수동 전환 불필요
- 다른 앱과 일관성 유지

## 테마 전환

### 설정을 통해

1. 설정 열기 (`Cmd+,` 또는 `Ctrl+,`)
2. **모양**으로 이동
3. 선호하는 테마 선택

### 명령 팔레트를 통해

1. 명령 팔레트 열기 (`Cmd+P` 또는 `Ctrl+P`)
2. "테마" 입력
3. 선호하는 테마 선택

### 키보드 단축키

| 플랫폼 | 단축키 |
| ------ | ------ |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

순환: 라이트 → 다크 → 시스템

## 글꼴 설정

### 에디터 글꼴

글쓰기 경험 사용자 정의

1. 설정 열기
2. **모양** → **에디터**로 이동
3. 글꼴 옵션 구성

| 설정 | 옵션 | 기본값 |
| ---- | ---- | ------ |
| 글꼴 패밀리 | 시스템 글꼴 + 사용자 정의 | 시스템 UI |
| 글꼴 크기 | 12-24px | 16px |
| 줄 높이 | 1.0-2.5 | 1.75 |
| 문자 간격 | -2 ~ 4 | 0 |

### UI 글꼴

인터페이스 글꼴 조정

1. 설정 열기
2. **모양** → **인터페이스**로 이동
3. 글꼴 및 크기 선택

### 고정폭 글꼴

코드 블록 및 기술 콘텐츠용

1. 설정 열기
2. **모양** → **코드**로 이동
3. 고정폭 글꼴 선택

## 인터페이스 옵션

### 사이드바

사이드바 동작 사용자 정의

| 설정 | 설명 |
| ---- | ---- |
| **자동 숨기기** | 사용하지 않을 때 사이드바 숨기기 |
| **너비** | 사이드바 너비 조정 (200-400px) |
| **아이콘 표시** | 아이콘 가시성 토글 |

### 에디터 레이아웃

편집 경험 선택

- **분할 보기**: 편집 및 미리보기를 나란히
- **미리보기만**: 렌더링된 결과만 보기
- **편집만**: 원본 Markdown에 집중

### 노트 목록

노트 표시 방법 구성

- **목록 보기**: 노트당 한 줄로 간결하게
- **카드 보기**: 스니펫이 있는 미리보기 카드
- **트리 보기**: 계층적 폴더 구조

## 강조 색상

### 사용 가능한 색상

강조 색상 선택

- 클래식 블루 (기본값)
- 보라색
- 녹색
- 주황색
- 빨간색
- 분홍색
- 청록색

### 강조 색상 변경

1. 설정 열기
2. **모양** → **색상**으로 이동
3. 선호하는 강조 선택

강조 색상은 다음에 영향을 줍니다:

- 링크 및 버튼
- 선택 강조
- 활성 요소
- 포커스 표시기

## 접근성

### 고대비 모드

더 나은 가시성을 위해

1. 설정 열기
2. **모양** → **접근성**으로 이동
3. **고대비** 활성화

### 모션 감소

애니메이션 최소화

1. 설정 열기
2. **모양** → **접근성**으로 이동
3. **모션 감소** 활성화

### 글꼴 조정

- 최소 글꼴 크기 증가
- 난독증 친화적 글꼴 사용
- 대비 비율 조정

## 사용자 정의 CSS (고급)

완전한 제어를 원하는 파워 유저용

1. 설정 열기
2. **모양** → **고급**으로 이동
3. **사용자 정의 CSS 열기** 클릭
4. 사용자 정의 추가

### 사용자 정의 예시

```css
/* 더 큰 제목 */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* 사용자 정의 링크 색상 */
a { color: #3b82f6; }

/* 더 넓은 줄 간격 */
.prose p { line-height: 2; }
```

## 테마 동기화

로그인하면 (선택 사항), 테마 환경설정이 기기 간에 동기화됩니다

- 자동 테마 동기화
- 글꼴 설정 동기화
- 사용자 정의 CSS 동기화

## 팁

### 워크플로에 맞추기

- 낮에는 **라이트**, 밤에는 **다크** 사용
- **시스템** 모드가 자동으로 처리
- 주변 조명 고려

### 눈의 피로 감소

- 저조도에서 다크 모드 사용
- 긴 세션을 위해 글꼴 크기 증가
- 필요한 경우 고대비 활성화
- 정기적인 휴식

### 생산성

- 테마 토글 단축키 학습
- 자동 전환 시간 설정
- 일관성을 위해 시스템 모드 사용
