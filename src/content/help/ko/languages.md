# 언어 및 현지화

Classic은 사용자의 언어로 말합니다. 선호하는 언어로 앱을 사용하고 원하는 언어로 노트를 작성하세요.

## 지원되는 언어

Classic의 인터페이스는 여러 언어로 제공됩니다

| 언어 | 코드 |
|------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| 그리고 더... | |

## 인터페이스 언어 변경

### 설정을 통해

1. 설정 열기 (`Cmd+,` 또는 `Ctrl+,`)
2. **일반** → **언어**로 이동
3. 선호하는 언어 선택
4. 적용하려면 Classic 재시작

### 명령 팔레트를 통해

1. 명령 팔레트 열기 (`Cmd+P` 또는 `Ctrl+P`)
2. "언어" 입력
3. 선호하는 언어 선택
4. 변경 사항을 적용하려면 재시작

## 여러 언어로 글쓰기

Classic은 모든 언어로 노트 작성을 지원합니다

### 유니코드 지원

- 전체 유니코드 지원
- RTL (오른쪽에서 왼쪽) 언어 지원
- CJK (중국어, 일본어, 한국어) 문자 완벽 작동
- 이모지 지원

### 언어 감지

Classic은 노트의 언어를 감지할 수 있습니다

- 맞춤법 검사를 위한 자동 감지
- 노트별 언어 설정
- 혼합 언어 지원

## 맞춤법 검사

### 맞춤법 검사 활성화

1. 설정 열기
2. **에디터** → **맞춤법 검사**로 이동
3. "맞춤법 검사" 활성화

### 맞춤법 검사 언어

1. 설정 열기
2. **에디터** → **맞춤법 검사**로 이동
3. "언어" 클릭
4. 검사할 언어 선택

### 사전에 단어 추가

1. 표시된 단어를 마우스 오른쪽 버튼으로 클릭
2. "사전에 추가" 선택
3. 단어가 다시 표시되지 않습니다

## RTL 지원

아랍어, 히브리어 및 기타 RTL 언어용

### 자동 RTL 감지

Classic은 RTL 콘텐츠를 자동으로 감지합니다

- 텍스트 방향 조정
- 커서 이동 적응
- 정렬 변경

### 수동 RTL 토글

RTL 또는 LTR 방향 강제

1. 명령 팔레트 열기
2. "RTL" 또는 "LTR" 입력
3. 선호하는 것 선택

## 입력 방법

Classic은 시스템의 입력 방법과 함께 작동합니다

### CJK 입력 방법

- 병음, 주음 등과 함께 작동
- 일본어 IME 지원
- 한국어 입력 방법 지원

### 특수 문자

특수 문자를 쉽게 삽입

- 시스템의 문자 선택기 사용
- 문자 맵에서 복사/붙여넣기
- 유니코드 입력 사용

## 날짜 및 시간 형식

날짜는 로케일에 맞게 조정됩니다

### 형식 옵션

| 설정 | 옵션 |
|------|------|
| 날짜 형식 | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| 시간 형식 | 12시간, 24시간 |
| 주의 첫 날 | 일요일, 월요일, 토요일 |

설정 → 일반 → 날짜 및 시간에서 구성

## 숫자 형식

숫자는 로케일에 따라 표시됩니다

- **소수점 구분자**: `.` 또는 `,`
- **천 단위 구분자**: `,` 또는 `.` 또는 공백
- **통화 기호**: 지역 기반

## 키보드 레이아웃

Classic은 모든 키보드 레이아웃과 함께 작동합니다

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- 사용자 정의 레이아웃

## 현지화 설정

### 설정 → 일반 → 언어에서

| 설정 | 설명 |
|------|------|
| **인터페이스 언어** | 앱 UI 언어 |
| **날짜 형식** | 날짜 표시 방법 |
| **시간 형식** | 12 또는 24시간 |
| **주의 첫 날** | 달력 시작일 |
| **숫자 형식** | 소수점/천 단위 구분자 |

## 팁

### 혼합 언어 노트

한 노트에 여러 언어로 작성

```markdown
# 국제 프로젝트

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### 언어별 글꼴

일부 언어는 특정 글꼴에서 더 잘 보입니다

1. 설정 열기
2. 모양 → 글꼴로 이동
3. 다른 스크립트에 대한 글꼴 설정

### 빠른 언어 전환

키보드 단축키 만들기

1. 설정 → 단축키로 이동
2. "언어 전환" 찾기
3. 선호하는 단축키 설정

## 문제 해결

### 언어를 사용할 수 없음

- 앱 업데이트 확인
- 언어가 개발 중일 수 있음
- GitHub에서 번역 기여

### 문자가 표시되지 않음

- 필요한 글꼴 설치
- 글꼴 설정 확인
- 다른 글꼴 시도

### 맞춤법 검사가 작동하지 않음

- 맞춤법 검사가 활성화되어 있는지 확인
- 맞춤법 검사에 언어 추가
- Classic 재시작

### RTL이 올바르게 작동하지 않음

- RTL 감지가 켜져 있는지 확인
- 수동 RTL 토글 시도
- 혼합 콘텐츠 문제 확인
