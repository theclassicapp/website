# 시스템 트레이

Classic은 빠른 액세스를 위해 시스템 트레이에 상주합니다. 워크플로를 중단하지 않고 생각을 즉시 캡처하세요.

## 시스템 트레이란?

시스템 트레이 (Windows/Linux) 또는 메뉴 바 (macOS)는 Classic이 백그라운드에서 실행되며 항상 빠른 캡처 준비가 된 곳입니다.

### 장점

- **항상 사용 가능**: 한 번의 클릭
- **빠른 캡처**: 생각을 즉시 적기
- **백그라운드 실행**: 앱을 열어둘 필요 없음
- **알림**: 작업 알림 받기

## 시스템 트레이 활성화

### macOS

1. Classic 열기
2. 설정 → 일반으로 이동
3. "메뉴 바에 표시" 활성화
4. Classic 아이콘이 메뉴 바에 나타납니다

### Windows

1. Classic 열기
2. 설정 → 일반으로 이동
3. "시스템 트레이에 표시" 활성화
4. Classic 아이콘이 시스템 트레이에 나타납니다

### Linux

1. Classic 열기
2. 설정 → 일반으로 이동
3. "시스템 트레이 아이콘" 활성화
4. 데스크톱 환경에 따라 다름

## 빠른 캡처

### 빠른 캡처 열기

시스템 트레이의 Classic 아이콘 클릭

- **macOS**: 메뉴 바 아이콘
- **Windows**: 시스템 트레이 아이콘
- **Linux**: 시스템 트레이 아이콘

### 빠른 캡처 사용법

1. 시스템 트레이 아이콘 클릭
2. 빠른 캡처 창이 열립니다
3. 노트 입력
4. Enter를 눌러 저장
5. 창이 자동으로 닫힙니다

### 키보드 단축키

어디서나 빠른 캡처 열기

| 플랫폼 | 단축키 |
|--------|--------|
| **macOS** | `Cmd+Shift+N` (전역) |
| **Windows** | `Ctrl+Shift+N` (전역) |
| **Linux** | `Ctrl+Shift+N` (전역) |

## 시스템 트레이 메뉴

옵션을 보려면 트레이 아이콘을 마우스 오른쪽 버튼으로 클릭

### 메뉴 옵션

| 옵션 | 작업 |
|------|------|
| **새 노트** | 빠른 캡처 열기 |
| **새 데일리 노트** | 오늘의 데일리 노트 열기 |
| **검색** | 검색 창 열기 |
| **Classic 열기** | 메인 창 열기 |
| **설정** | 설정 열기 |
| **종료** | Classic 종료 |

## 트레이 설정

### 설정에서 구성

| 설정 | 설명 |
|------|------|
| **아이콘 표시** | 트레이 아이콘 가시성 토글 |
| **최소화된 상태로 시작** | 트레이에서만 Classic 시작 |
| **트레이로 닫기** | 창이 닫힐 때 계속 실행 |
| **알림** | 알림 활성화/비활성화 |

### 로그인 시 시작

Classic이 자동으로 시작되도록 설정

1. 설정 열기
2. 일반으로 이동
3. "로그인 시 시작" 활성화
4. Classic이 시작 시 실행됩니다

## 알림

### 알림 활성화

1. 설정 열기
2. 알림으로 이동
3. "알림 활성화" 토글

### 알림 유형

- **알림**: 예약된 알림
- **데일리 노트**: 데일리 노트 프롬프트
- **업데이트**: 앱 업데이트 사용 가능

## 창 동작

### 닫기 동작

창을 닫을 때 발생하는 작업 선택

- **트레이로 최소화**: 계속 실행, 창 숨기기
- **종료**: 애플리케이션 종료
- **물어보기**: 매번 확인

### 최소화 동작

- **트레이로 최소화**: 시스템 트레이에 숨기기
- **작업 표시줄로 최소화**: 일반적으로 최소화

## 팁

### 빠른 캡처

가장 빠른 캡처를 위해 전역 단축키 사용

1. 어디서나 `Cmd/Ctrl+Shift+N` 누르기
2. 생각 입력
3. Enter 누르기
4. 작업 계속

### 빠른 접근

트레이 아이콘을 클릭하여 보기

- 최근 노트
- 빠른 작업
- 오늘의 데일리 노트

### 백그라운드 실행

Classic을 계속 실행하여:

- 더 빠른 시작
- 빠른 캡처
- 알림
- 자동 백업

## 문제 해결

### 트레이 아이콘 누락

**macOS**

1. 아이콘이 숨겨져 있는지 확인
2. 제어 센터 확인 (macOS Big Sur+)
3. Classic 재시작

**Windows**

1. 숨겨진 아이콘 확인 (작업 표시줄의 화살표)
2. Classic 아이콘을 보이는 영역으로 드래그
3. Classic 재시작

**Linux**

1. 데스크톱 환경 지원 확인
2. 필요한 경우 시스템 트레이 확장 설치
3. Classic 재시작

### 전역 단축키가 작동하지 않음

- 다른 앱과의 충돌 확인
- 설정에서 단축키 다시 활성화
- Classic 재시작
- OS 권한 확인

### 알림이 표시되지 않음

**macOS**

1. 시스템 환경설정 → 알림
2. Classic 찾기
3. 알림 활성화

**Windows**

1. 설정 → 시스템 → 알림
2. Classic 알림 활성화

**Linux**

1. 알림 데몬 확인
2. 필요한 경우 libnotify 설치

### 높은 메모리/CPU 사용량

트레이 아이콘이 너무 많은 리소스를 사용하는 경우:

1. 설정에서 애니메이션 비활성화
2. 알림 빈도 줄이기
3. 앱 업데이트 확인
