# 키보드 단축키

Classic의 키보드 단축키로 더 빠르게 작업하세요. 모든 단축키는 설정에서 사용자 정의할 수 있습니다.

## 전역 단축키

앱의 어디에서나 작동합니다.

### 일반

| 단축키 (Windows/Linux) | 단축키 (macOS) | 작업 |
| ---------------------- | --------------- | ---- |
| `Ctrl+N` | `Cmd+N` | 새 노트 |
| `Ctrl+P` | `Cmd+P` | 명령 팔레트 |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | 전역 빠른 캡처 |
| `Ctrl+S` | `Cmd+S` | 노트 저장 |
| `Ctrl+F` | `Cmd+F` | 검색 |
| `Ctrl+,` | `Cmd+,` | 설정 |

### 탐색

| 단축키 (Windows/Linux) | 단축키 (macOS) | 작업 |
| ---------------------- | --------------- | ---- |
| `Ctrl+1` | `Cmd+1` | 첫 번째 노트로 이동 |
| `Ctrl+2` | `Cmd+2` | 두 번째 노트로 이동 |
| `Ctrl+Left` | `Cmd+Left` | 뒤로 탐색 |
| `Ctrl+Right` | `Cmd+Right` | 앞으로 탐색 |
| `Ctrl+Up` | `Cmd+Up` | 부모 노트로 탐색 |
| `Ctrl+Down` | `Cmd+Down` | 첫 번째 서브노트 확장/포커스 |

### 에디터

| 단축키 (Windows/Linux) | 단축키 (macOS) | 작업 |
| ---------------------- | --------------- | ---- |
| `Ctrl+B` | `Cmd+B` | 굵게 |
| `Ctrl+I` | `Cmd+I` | 기울임꼴 |
| `Ctrl+K` | `Cmd+K` | 링크 삽입 |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | 취소선 |
| `Ctrl+`` | `Cmd+`` | 인라인 코드 |
| `Ctrl+Shift+C` | `Cmd+Shift+C` | 코드 블록 |
| `Ctrl+/` | `Cmd+/` | 슬래시 명령 |
| `Tab` | `Tab` | 들여쓰기 |
| `Shift+Tab` | `Shift+Tab` | 내어쓰기 |

### 창 관리

| 단축키 (Windows/Linux) | 단축키 (macOS) | 작업 |
| ---------------------- | --------------- | ---- |
| `Ctrl+W` | `Cmd+W` | 현재 노트 닫기 |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | 모든 노트 닫기 |
| `F11` | `F11` | 전체 화면 토글 |
| `Ctrl+M` | `Cmd+M` | 사이드바 토글 |

## 빠른 캡처

| 단축키 (Windows/Linux) | 단축키 (macOS) | 작업 |
| ---------------------- | --------------- | ---- |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | 빠른 캡처 열기 |
| `Escape` | `Escape` | 빠른 캡처 닫기 |
| `Ctrl+Enter` | `Cmd+Enter` | 저장 및 닫기 |

## 단축키 사용자 정의

1. **설정** 열기 (`Ctrl+,` 또는 `Cmd+,`)
2. **키보드 단축키**로 이동
3. 변경할 단축키 클릭
4. 새 키 조합 누르기
5. **저장** 클릭

## 팁

- 대부분의 단축키는 사용자 정의 가능
- 시스템 단축키와의 충돌은 강조 표시됨
- 각 단축키 옆의 재설정 버튼을 클릭하여 기본값으로 재설정
