# Creating Your First Note

Get started with Classic by creating your first note. It's simple and intuitive.

## Creating a New Note

### Using the Sidebar

1. Click the **+** button in the sidebar
2. A new note is created
3. Start typing your content

### Using Keyboard Shortcut

| Platform | Shortcut |
| -------- | -------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Using Command Palette

1. Open Command Palette (`Cmd+P` or `Ctrl+P`)
2. Type "New Note"
3. Press Enter

## Writing Your Note

### Basic Editing

Just start typing! Classic auto-saves as you write.

### Adding a Title

The first line of your note becomes the title. Use a heading for best results

```markdown
# My First Note

This is the content of my note.
```

### Formatting Text

Use Markdown to format your text

- **Bold**: `**text**` or `Cmd/Ctrl+B`
- **Italic**: `*text*` or `Cmd/Ctrl+I`
- **Strikethrough**: `~~text~~`
- **Code**: `` `code` ``

## Saving

Classic auto-saves your notes as you type. No need to manually save.

### Manual Save

If needed, you can force a save

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## Organizing Your Note

### Adding Tags

Tag your note for easy finding later

1. Type `#` followed by your tag name
2. Or use the tag button in the toolbar
3. Multiple tags are supported

Example

```markdown
# Meeting Notes

Discussed Q4 planning #work #meetings
```

### Moving to a Folder

1. Right-click the note in the sidebar
2. Select "Move to Folder"
3. Choose or create a destination folder

### Pinning Important Notes

1. Right-click the note
2. Select "Pin to Top"
3. The note appears at the top of your list

## Preview Your Note

See how your formatted note looks

- Click the **Preview** button
- Or use `Cmd/Ctrl+Shift+P`
- Toggle between Edit and Preview modes

## Tips

### Note Titles

- Keep titles descriptive
- First line = note title
- Use H1 (`#`) for clarity

### Auto-Save

- Never worry about losing work
- Changes save automatically
- Check status bar for save status

### Quick Capture

Need to jot something fast?

- Use Quick Capture (`Cmd/Ctrl+Shift+N`)
- Creates a note instantly
- Great for fleeting thoughts
