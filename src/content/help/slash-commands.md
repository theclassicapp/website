# Slash Commands

Slash commands are a powerful way to quickly insert formatting and special content without leaving the keyboard. Type `/` in the editor to open the command palette.

## Available Commands

### Headings

| Command | Result |
| ------- | ------ |
| `/h1` | Level 1 heading |
| `/h2` | Level 2 heading |
| `/h3` | Level 3 heading |
| `/h4` | Level 4 heading |
| `/h5` | Level 5 heading |
| `/h6` | Level 6 heading |

### Text Formatting

| Command | Result |
| ------- | ------ |
| `/bold` | **Bold text** |
| `/italic` | *Italic text* |
| `/strikethrough` | ~~Strikethrough~~ |
| `/code` | Inline code block |
| `/quote` | Blockquote |

### Lists

| Command | Result |
| ------- | ------ |
| `/bullet` | • Bullet list |
| `/numbered` | 1. Numbered list |
| `/todo` | - [ ] Todo item |
| `/check` | - [x] Completed item |

### Insert Content

| Command | Result |
| ------- | ------ |
| `/image` | Insert image from URL or file |
| `/link` | Insert hyperlink |
| `/table` | Insert table |
| `/hr` | Horizontal rule |
| `/code` | Code block with language selection |

### Special Blocks

| Command | Result |
| ------- | ------ |
| `/mermaid` | Mermaid diagram block |
| `/katex` | KaTeX math block |
| `/callout` | Callout/admonition block |

## Using Slash Commands

1. Type `/` in the editor
2. A menu will appear with all available commands
3. Start typing to filter commands
4. Press `Enter` or click to insert

## Pro Tips

- **Quick access**: Just type `/` and start typing the command name
- **Fuzzy search**: You don't need to type the exact command name - Classic will find the closest match
- **Keyboard navigation**: Use arrow keys to navigate the menu
- **Recent commands**: Frequently used commands appear at the top of the list
