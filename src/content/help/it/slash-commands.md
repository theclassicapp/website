# Comandi Slash

I comandi slash sono un modo potente per inserire rapidamente formattazione e contenuti speciali senza lasciare la tastiera. Digita `/` nell'editor per aprire la palette comandi.

## Comandi Disponibili

### Intestazioni

| Comando | Risultato |
|---------|-----------|
| `/h1` | Intestazione livello 1 |
| `/h2` | Intestazione livello 2 |
| `/h3` | Intestazione livello 3 |
| `/h4` | Intestazione livello 4 |
| `/h5` | Intestazione livello 5 |
| `/h6` | Intestazione livello 6 |

### Formattazione Testo

| Comando | Risultato |
|---------|-----------|
| `/bold` | **Testo in grassetto** |
| `/italic` | *Testo in corsivo* |
| `/strikethrough` | ~~Testo barrato~~ |
| `/code` | Blocco di codice inline |
| `/quote` | Citazione |

### Elenchi

| Comando | Risultato |
|---------|-----------|
| `/bullet` | • Elenco puntato |
| `/numbered` | 1. Elenco numerato |
| `/todo` | - [ ] Elemento da fare |
| `/check` | - [x] Elemento completato |

### Inserisci Contenuto

| Comando | Risultato |
|---------|-----------|
| `/image` | Inserisci immagine da URL o file |
| `/link` | Inserisci hyperlink |
| `/table` | Inserisci tabella |
| `/hr` | Linea orizzontale |
| `/code` | Blocco di codice con selezione linguaggio |

### Blocchi Speciali

| Comando | Risultato |
|---------|-----------|
| `/mermaid` | Blocco diagramma Mermaid |
| `/katex` | Blocco matematico KaTeX |
| `/callout` | Blocco callout/ammonimento |

## Usare i Comandi Slash

1. Digita `/` nell'editor
2. Apparirà un menu con tutti i comandi disponibili
3. Inizia a digitare per filtrare i comandi
4. Premi `Invio` o clicca per inserire

## Suggerimenti Pro

- **Accesso rapido**: Basta digitare `/` e iniziare a digitare il nome del comando
- **Ricerca fuzzy**: Non devi digitare il nome esatto del comando - Classic troverà la corrispondenza più vicina
- **Navigazione da tastiera**: Usa i tasti freccia per navigare il menu
- **Comandi recenti**: I comandi usati frequentemente appaiono in cima alla lista
