# Scorciatoie da Tastiera

Lavora più velocemente con le scorciatoie da tastiera di Classic. Tutte le scorciatoie sono personalizzabili nelle Impostazioni.

## Scorciatoie Globali

Queste funzionano ovunque nell'app.

### Generale

| Scorciatoia (Windows/Linux) | Scorciatoia (macOS) | Azione |
|----------------------------|---------------------|--------|
| `Ctrl+N` | `Cmd+N` | Nuova nota |
| `Ctrl+P` | `Cmd+P` | Palette comandi |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Cattura rapida globale |
| `Ctrl+S` | `Cmd+S` | Salva nota |
| `Ctrl+F` | `Cmd+F` | Ricerca |
| `Ctrl+,` | `Cmd+,` | Impostazioni |

### Navigazione

| Scorciatoia (Windows/Linux) | Scorciatoia (macOS) | Azione |
|----------------------------|---------------------|--------|
| `Ctrl+1` | `Cmd+1` | Vai alla prima nota |
| `Ctrl+2` | `Cmd+2` | Vai alla seconda nota |
| `Ctrl+Sinistra` | `Cmd+Sinistra` | Naviga indietro |
| `Ctrl+Destra` | `Cmd+Destra` | Naviga avanti |
| `Ctrl+Su` | `Cmd+Su` | Naviga alla nota genitore |
| `Ctrl+Giù` | `Cmd+Giù` | Espandi/focus prima sottonota |

| `Tab` | `Tab` | Indenta |

### Gestione Finestra

|----------------------------|---------------------|--------|

## Cattura Rapida

| Scorciatoia (Windows/Linux) | Scorciatoia (macOS) | Azione |
|----------------------------|---------------------|--------|
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Apri cattura rapida |
| `Escape` | `Escape` | Chiudi cattura rapida |
| `Ctrl+Invio` | `Cmd+Invio` | Salva e chiudi |

## Personalizzare le Scorciatoie

1. Apri **Impostazioni** (`Ctrl+,` o `Cmd+,`)
2. Naviga su **Scorciatoie da Tastiera**
3. Clicca su qualsiasi scorciatoia per cambiarla
4. Premi la tua nuova combinazione di tasti
5. Clicca **Salva**

## Suggerimenti

- La maggior parte delle scorciatoie sono personalizzabili
- I conflitti con le scorciatoie di sistema saranno evidenziati
- Ripristina il default cliccando il pulsante reset accanto a ogni scorciatoia
