# Creare la Tua Prima Nota

Inizia con Classic creando la tua prima nota. È semplice e intuitivo.

## Creare una Nuova Nota

### Usando la Barra Laterale

1. Clicca il pulsante **+** nella barra laterale
2. Viene creata una nuova nota
3. Inizia a digitare il contenuto

### Usando la Scorciatoia da Tastiera

| Piattaforma | Scorciatoia |
|-------------|-------------|
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Usando la Palette Comandi

1. Apri Palette Comandi (`Cmd+P` o `Ctrl+P`)
2. Digita "Nuova Nota"
3. Premi Invio

## Scrivere la Tua Nota

### Modifica Base

Basta iniziare a digitare! Classic salva automaticamente mentre scrivi.

### Aggiungere un Titolo

La prima riga della tua nota diventa il titolo. Usa un'intestazione per migliori risultati:

```markdown
# La Mia Prima Nota

Questo è il contenuto della mia nota.
```

### Formattare il Testo

Usa Markdown per formattare il testo:

- **Grassetto**: `**testo**` o `Cmd/Ctrl+B`
- **Corsivo**: `*testo*` o `Cmd/Ctrl+I`
- **Barrato**: `~~testo~~`
- **Codice**: `` `codice` ``

## Salvataggio

Classic salva automaticamente le tue note mentre digiti. Non è necessario salvare manualmente.

### Salvataggio Manuale

Se necessario, puoi forzare il salvataggio:

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## Organizzare la Tua Nota

### Aggiungere Tag

Tagga la tua nota per trovarla facilmente in seguito:

1. Digita `#` seguito dal nome del tag
2. Oppure usa il pulsante tag nella barra degli strumenti
3. Sono supportati tag multipli

Esempio:

```markdown
# Note Riunione

Discussione pianificazione Q4 #lavoro #riunioni
```

### Spostare in una Cartella

1. Clic con il tasto destro sulla nota nella barra laterale
2. Seleziona "Sposta in Cartella"
3. Scegli o crea una cartella di destinazione

### Aggiungere Note Importanti in Alto

1. Clic con il tasto destro sulla nota
2. Seleziona "Fissa in Alto"
3. La nota appare in cima alla tua lista

## Anteprima della Tua Nota

Vedi come appare la tua nota formattata:

- Clicca il pulsante **Anteprima**
- Oppure usa `Cmd/Ctrl+Shift+P`
- Passa tra modalità Modifica e Anteprima

## Prossimi Passi

Ora che hai creato la tua prima nota:

- Scopri di più su [Markdown](/help/markdown)
- Organizza con [Tag](/help/tags)
- Configura le [Note Giornaliere](/help/daily-notes)
- Esplora le [Scorciatoie da Tastiera](/help/keyboard-shortcuts)

## Suggerimenti

### Titoli delle Note

- Mantieni i titoli descrittivi
- Prima riga = titolo della nota
- Usa H1 (`#`) per chiarezza

### Salvataggio Automatico

- Non preoccuparti mai di perdere il lavoro
- Le modifiche si salvano automaticamente
- Controlla la barra di stato per lo stato del salvataggio

### Cattura Rapida

Hai bisogno di annotare qualcosa velocemente?

- Usa Cattura Rapida (`Cmd/Ctrl+Shift+N`)
- Crea una nota istantaneamente
- Ottimo per pensieri fugaci
