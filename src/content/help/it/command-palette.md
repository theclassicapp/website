# Palette Comandi

La palette comandi è il tuo gateway per ogni azione in Classic. Premi una semplice scorciatoia da tastiera e accedi a qualsiasi funzionalità istantaneamente.

## Aprire la Palette Comandi

| Piattaforma | Scorciatoia |
|-------------|-------------|
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Come Usare

1. Apri la palette comandi (`Cmd+P` o `Ctrl+P`)
2. Digita per cercare un comando
3. Usa i tasti freccia per navigare
4. Premi `Invio` per eseguire

## Categorie di Comandi

### Operazioni File

- **Nuova Nota**: Crea una nuova nota
- **Nuova Nota Giornaliera**: Crea/apri la nota giornaliera di oggi
- **Apri Nota**: Cerca e apri note esistenti
- **Salva Nota**: Salva nota corrente
- **Elimina Nota**: Elimina nota corrente
- **Rinomina Nota**: Rinomina nota corrente

### Navigazione

- **Vai a Nota**: Salta a qualsiasi nota
- **Vai a Nota Giornaliera**: Apri una data specifica
- **Vai a Tag**: Sfoglia note per tag
- **Vai Indietro**: Naviga indietro nella cronologia
- **Vai Avanti**: Naviga avanti nella cronologia

### Editor

- **Attiva/Disattiva Anteprima**: Passa tra modalità modifica e anteprima
- **Attiva/Disattiva Barra Laterale**: Mostra/nascondi la barra laterale
- **Modalità Focus**: Entra in scrittura senza distrazioni
- **Inserisci Data**: Aggiungi data/ora corrente
- **Inserisci Template**: Inserisci un template

### Formattazione

- **Grassetto**: Rendi il testo grassetto
- **Corsivo**: Rendi il testo corsivo
- **Barrato**: Barrato il testo
- **Codice**: Formatta come codice
- **Link**: Inserisci un link
- **Intestazione**: Applica formato intestazione

### Organizzazione

- **Aggiungi Tag**: Tagga la nota corrente
- **Sposta in Cartella**: Sposta nota in una cartella
- **Archivia Nota**: Archivia nota corrente
- **Fissa Nota**: Fissa nota in cima alla lista

### Vista

- **Attiva/Disattiva Tema**: Passa tra chiaro/scuro
- **Zoom In**: Aumenta dimensione font
- **Zoom Out**: Diminuisci dimensione font
- **Reset Zoom**: Ripristina dimensione predefinita

## Suggerimenti

### Accesso Rapido

- I comandi sono ricercati in modo fuzzy
- I comandi recenti appaiono per primi
- Fissa i comandi usati frequentemente

### Navigazione da Tastiera

- `↑`/`↓` - Naviga risultati
- `Invio` - Esegui comando
- `Escape` - Chiudi palette
- `Tab` - Scorri le categorie

### Comandi Personalizzati

1. Apri Impostazioni
2. Vai su Palette Comandi
3. Aggiungi comandi personalizzati
4. Definisci scorciatoie da tastiera

## Comandi Usati Frequentemente

Ecco alcuni comandi che userai spesso:

| Comando | Cosa fa |
|---------|---------|
| `Nuova Nota` | Crea nota vuota |
| `Apri Nota` | Cerca tutte le note |
| `Nuova Nota Giornaliera` | Diario di oggi |
| `Attiva Anteprima` | Anteprima vs modifica |
| `Aggiungi Tag` | Tagga nota corrente |
| `Cattura Rapida` | Cattura globale |

## Personalizzare i Comandi

### Cambiare Scorciatoie

1. Apri Impostazioni (`Cmd+,` o `Ctrl+,`)
2. Naviga su Scorciatoie da Tastiera
3. Trova il comando
4. Clicca la scorciatoia corrente
5. Premi la nuova scorciatoia
6. Salva

### Creare Comandi Personalizzati

Gli utenti avanzati possono creare comandi personalizzati:

1. Apri Impostazioni
2. Vai su Comandi
3. Clicca "Aggiungi Comando Personalizzato"
4. Definisci il nome del comando
5. Aggiungi l'azione/i azioni
6. Imposta una scorciatoia (opzionale)
