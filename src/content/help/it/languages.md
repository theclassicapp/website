# Lingue e Localizzazione

Classic parla la tua lingua. Usa l'app nella tua lingua preferita e scrivi note in qualsiasi lingua tu scelga.

## Lingue Supportate

L'interfaccia di Classic è disponibile in più lingue:

| Lingua | Codice |
|--------|--------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| E altre... | |

## Cambiare la Lingua dell'Interfaccia

### Via Impostazioni

1. Apri Impostazioni (`Cmd+,` o `Ctrl+,`)
2. Vai su **Generale** → **Lingua**
3. Seleziona la tua lingua preferita
4. Riavvia Classic per applicare

### Via Palette Comandi

1. Apri Palette Comandi (`Cmd+P` o `Ctrl+P`)
2. Digita "Lingua"
3. Seleziona la tua lingua preferita
4. Riavvia per applicare le modifiche

## Scrivere in Lingue Multiple

Classic supporta la scrittura di note in qualsiasi lingua:

### Supporto Unicode

- Supporto Unicode completo
- Lingue RTL (da destra a sinistra) supportate
- Caratteri CJK (Cinese, Giapponese, Coreano) funzionano perfettamente
- Supporto Emoji

### Rilevamento Lingua

Classic può rilevare la lingua delle tue note:

- Auto-rilevamento per controllo ortografico
- Impostazione lingua per nota
- Supporto lingua mista

## Controllo Ortografico

### Abilitare Controllo Ortografico

1. Apri Impostazioni
2. Vai su **Editor** → **Controllo Ortografico**
3. Abilita "Controllo Ortografico"

### Lingue Controllo Ortografico

1. Apri Impostazioni
2. Vai su **Editor** → **Controllo Ortografico**
3. Clicca "Lingue"
4. Seleziona lingue da controllare

### Aggiungere Parole al Dizionario

1. Clic con il tasto destro su una parola segnalata
2. Seleziona "Aggiungi al Dizionario"
3. La parola non verrà più segnalata

## Supporto RTL

Per Arabo, Ebraico e altre lingue RTL:

### Rilevamento RTL Automatico

Classic rileva contenuti RTL automaticamente:

- La direzione del testo si adatta
- Il movimento del cursore si adatta
- L'allineamento cambia

### Toggle RTL Manuale

Forza direzione RTL o LTR:

1. Apri Palette Comandi
2. Digita "RTL" o "LTR"
3. Seleziona la tua preferenza

## Metodi di Input

Classic funziona con i metodi di input del tuo sistema:

### Metodi di Input CJK

- Funziona con Pinyin, Zhuyin, ecc.
- Supporta IME Giapponese
- Metodi di input Coreano supportati

### Caratteri Speciali

Inserisci caratteri speciali facilmente:

- Usa il selettore caratteri del tuo sistema
- Copia/incolla da mappe caratteri
- Usa input Unicode

## Formati Data e Ora

Le date si adattano alla tua locale:

### Opzioni Formato

| Impostazione | Opzioni |
|--------------|---------|
| Formato Data | GG/MM/AAAA, MM/GG/AAAA, AAAA-MM-GG |
| Formato Ora | 12 ore, 24 ore |
| Primo Giorno della Settimana | Domenica, Lunedì, Sabato |

Configura in Impostazioni → Generale → Data e Ora

## Formati Numeri

I numeri vengono visualizzati secondo la tua locale:

- **Separatore decimale**: `.` o `,`
- **Separatore migliaia**: `,` o `.` o spazio
- **Simboli valuta**: Basati sulla regione

## Layout Tastiera

Classic funziona con qualsiasi layout tastiera:

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Layout personalizzati

## Impostazioni Localizzazione

### In Impostazioni → Generale → Lingua

| Impostazione | Descrizione |
|--------------|-------------|
| **Lingua Interfaccia** | Lingua UI app |
| **Formato Data** | Come appaiono le date |
| **Formato Ora** | 12 o 24 ore |
| **Primo Giorno della Settimana** | Giorno inizio calendario |
| **Formato Numeri** | Separatori decimali/migliaia |

## Suggerimenti

### Note in Lingua Mista

Scrivi in più lingue in una nota:

```markdown
# Progetto Internazionale

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Font Specifici per Lingua

Alcune lingue appaiono meglio con font specifici:

1. Apri Impostazioni
2. Vai su Aspetto → Font
3. Imposta font per script diversi

### Cambiare Lingua Rapidamente

Crea una scorciatoia da tastiera:

1. Apri Impostazioni → Scorciatoie
2. Trova "Cambia Lingua"
3. Imposta la tua scorciatoia preferita

## Risoluzione Problemi

### Lingua Non Disponibile

- Controlla aggiornamenti app
- La lingua potrebbe essere in sviluppo
- Contribuisci traduzioni su GitHub

### Caratteri Non Visualizzati

- Installa font richiesti
- Controlla impostazioni font
- Prova un font diverso

### Controllo Ortografico Non Funziona

- Verifica che il controllo ortografico sia abilitato
- Aggiungi la lingua al controllo ortografico
- Riavvia Classic

### RTL Non Funziona Correttamente

- Assicurati che il rilevamento RTL sia attivo
- Prova il toggle RTL manuale
- Controlla problemi di contenuto misto
