# Guida Rapida

Benvenuto in Classic! Questa guida ti aiuterà a iniziare con Classic in pochi minuti.

## Requisiti di Sistema

- **macOS**: macOS 12 (Monterey) o successivo
- **Windows**: Windows 10 o successivo
- **Linux**: Ubuntu 20.04+ / Fedora 36+ o successivo

## Installazione

1. Scarica Classic dal [sito web](/).
2. Apri l'applicazione
3. Inizia a scrivere!

## Prima Esperienza di Avvio

Quando apri Classic per la prima volta, vedrai un'interfaccia pulita e minimale progettata per mantenerti concentrato sulla scrittura.

![Screenshot Classic App]

### La Barra Laterale

Sul lato sinistro puoi trovare:
- **Note Giornaliere** - Il tuo diario automatico quotidiano
- **Ricerca** - Trova rapidamente qualsiasi cosa in tutte le tue note
- **Palette Comandi** - Premi `Cmd/Ctrl+P` (Windows/Linux) o `Cmd+Shift+P` (macOS) per aprire la palette comandi
- **Struttura** - Sfoglia e organizza le tue note in una struttura ad albero

### Editor

L'area centrale è l'**editor Markdown** con:
- Anteprima in tempo reale
- Evidenziazione della sintassi
- Ricchi strumenti di formattazione

### Creare la Tua Prima Nota

1. Clicca il pulsante **Nuova Nota** nella barra laterale, o premi `Ctrl/Cmd+N` (Windows/Linux) o `Cmd+N` (macOS)
2. Dai un titolo alla tua nota
3. Inizia a scrivere in Markdown!

```markdown
# La Mia Prima Nota

Benvenuto in Classic! Qui catturerò i miei pensieri e le mie idee.
```

## Prossimi Passi

- Scopri la [sintassi Markdown](/help/markdown)
- Esplora i [Comandi Slash](/help/slash-commands)
- Configura le [Note Giornaliere](/help/daily-notes)
