# Tray di Sistema

Classic risiede nel tray di sistema per accesso rapido. Cattura pensieri istantaneamente senza interrompere il tuo flusso di lavoro.

## Cos'è il Tray di Sistema?

Il tray di sistema (Windows/Linux) o la barra dei menu (macOS) è dove Classic gira in background, sempre pronto per la cattura rapida.

### Vantaggi

- **Sempre disponibile**: A un clic di distanza
- **Cattura rapida**: Annota pensieri istantaneamente
- **Esecuzione in background**: Non serve mantenere l'app aperta
- **Notifiche**: Ricevi promemoria per le attività

## Abilitare il Tray di Sistema

### macOS

1. Apri Classic
2. Vai su Impostazioni → Generale
3. Abilita "Mostra nella Barra dei Menu"
4. L'icona Classic appare nella barra dei menu

### Windows

1. Apri Classic
2. Vai su Impostazioni → Generale
3. Abilita "Mostra nel Tray di Sistema"
4. L'icona Classic appare nel tray di sistema

### Linux

1. Apri Classic
2. Vai su Impostazioni → Generale
3. Abilita "Icona Tray di Sistema"
4. Dipende dall'ambiente desktop

## Cattura Rapida

### Aprire Cattura Rapida

Clicca l'icona Classic nel tray di sistema:

- **macOS**: Icona barra menu
- **Windows**: Icona tray di sistema
- **Linux**: Icona tray di sistema

### Usare Cattura Rapida

1. Clicca icona tray di sistema
2. Si apre la finestra di cattura rapida
3. Digita la tua nota
4. Premi Invio per salvare
5. La finestra si chiude automaticamente

### Scorciatoia da Tastiera

Apri Cattura Rapida da ovunque:

| Piattaforma | Scorciatoia |
|-------------|-------------|
| **macOS** | `Cmd+Shift+N` (globale) |
| **Windows** | `Ctrl+Shift+N` (globale) |
| **Linux** | `Ctrl+Shift+N` (globale) |

## Menu Tray di Sistema

Clic con il tasto destro sull'icona tray per opzioni:

### Opzioni Menu

| Opzione | Azione |
|---------|--------|
| **Nuova Nota** | Apri Cattura Rapida |
| **Nuova Nota Giornaliera** | Apri nota giornaliera di oggi |
| **Ricerca** | Apri finestra di ricerca |
| **Apri Classic** | Apri finestra principale |
| **Impostazioni** | Apri impostazioni |
| **Esci** | Chiudi Classic |

## Impostazioni Tray

### Configura nelle Impostazioni

| Impostazione | Descrizione |
|--------------|-------------|
| **Mostra Icona** | Attiva/disattiva visibilità icona tray |
| **Avvia Minimizzato** | Avvia Classic solo nel tray |
| **Chiudi nel Tray** | Mantieni in esecuzione quando la finestra è chiusa |
| **Notifiche** | Abilita/disabilita notifiche |

### Avvia al Login

Fai avviare Classic automaticamente:

1. Apri Impostazioni
2. Vai su Generale
3. Abilita "Avvia al Login"
4. Classic si avvia all'accensione

## Notifiche

### Abilita Notifiche

1. Apri Impostazioni
2. Vai su Notifiche
3. Attiva "Abilita Notifiche"

### Tipi di Notifica

- **Promemoria**: Promemoria pianificati
- **Nota Giornaliera**: Prompt nota giornaliera
- **Aggiornamenti**: Aggiornamento app disponibile

## Comportamento Finestra

### Comportamento Chiusura

Scegli cosa succede quando chiudi la finestra:

- **Minimizza nel Tray**: Mantieni in esecuzione, nascondi finestra
- **Esci**: Chiudi l'applicazione
- **Chiedi**: Chiedi ogni volta

### Comportamento Minimizzazione

- **Minimizza nel Tray**: Nascondi nel tray di sistema
- **Minimizza nella Taskbar**: Minimizza normalmente

## Suggerimenti

### Cattura Veloce

Usa la scorciatoia globale per cattura più veloce:

1. Premi `Cmd/Ctrl+Shift+N` ovunque
2. Digita il tuo pensiero
3. Premi Invio
4. Continua il tuo lavoro

### Accesso Rapido

Clicca icona tray per vedere:

- Note recenti
- Azioni rapide
- Nota giornaliera di oggi

### Esecuzione in Background

Mantieni Classic in esecuzione per:

- Avvio più veloce
- Cattura rapida
- Notifiche
- Auto-backup

## Risoluzione Problemi

### Icona Tray Mancante

**macOS**

1. Controlla se l'icona è nascosta
2. Cerca nel Centro di Controllo (macOS Big Sur+)
3. Riavvia Classic

**Windows**

1. Controlla icone nascoste (freccia nella taskbar)
2. Trascina l'icona Classic nell'area visibile
3. Riavvia Classic

**Linux**

1. Controlla il supporto dell'ambiente desktop
2. Installa estensione tray di sistema se necessario
3. Riavvia Classic

### Scorciatoia Globale Non Funziona

- Controlla conflitti con altre app
- Ri-abilita la scorciatoia nelle Impostazioni
- Riavvia Classic
- Controlla permessi OS

### Notifiche Non Appaiono

**macOS**

1. Preferenze di Sistema → Notifiche
2. Trova Classic
3. Abilita notifiche

**Windows**

1. Impostazioni → Sistema → Notifiche
2. Abilita notifiche Classic

**Linux**

1. Controlla il daemon di notifica
2. Installa libnotify se necessario

### Uso Eccessivo Memoria/CPU

Se l'icona tray usa troppe risorse:

1. Disabilita animazioni nelle Impostazioni
2. Riduci frequenza notifiche
3. Controlla aggiornamenti app
