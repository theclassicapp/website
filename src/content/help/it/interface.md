# Comprendere l'Interfaccia

L'interfaccia di Classic è progettata per essere pulita, intuitiva e non intralciare. Ecco una panoramica dei componenti principali.

## Layout Principale

Classic ha un layout a tre pannelli:

1. **Barra Laterale** (sinistra) - Navigazione e organizzazione delle note
2. **Editor** (centro) - Il tuo spazio di scrittura
3. **Anteprima/Proprietà** (destra) - Pannelli opzionali

## Barra Laterale

La barra laterale contiene diverse sezioni chiave:

### Elenco Note

- **Tutte le Note**: Visualizza tutte le tue note in un unico elenco
- **Note Giornaliere**: Accesso rapido al diario di oggi
- **Preferiti**: Aggiungi alle preferite le note usate frequentemente
- **Cartelle**: Organizza le note in gerarchie

### Navigazione

- **Barra di Ricerca**: Trova note istantaneamente
- **Tag**: Sfoglia per tag
- **Backlink**: Vedi le connessioni tra le note

### Azioni

- **Nuova Nota**: Crea una nuova nota
- **Nuova Cartella**: Organizza con cartelle
- **Importa**: Porta contenuti esterni

## Editor

L'editor è dove scrivi e formatti le tue note.

### Funzionalità

- **Supporto Markdown**: Sintassi Markdown completa
- **Anteprima in Tempo Reale**: Vedi il testo formattato mentre digiti
- **Evidenziazione Sintassi**: I blocchi di codice sono evidenziati
- **Comandi Slash**: Digita `/` per azioni rapide

### Modalità Editor

| Modalità | Descrizione |
|----------|-------------|
| **Solo Modifica** | Vedi Markdown grezzo |
| **Solo Anteprima** | Vedi l'output renderizzato |
| **Vista Divisa** | Modifica e anteprima affiancate |

### Barra degli Strumenti

Azioni di formattazione rapida:

- Grassetto, Corsivo, Barrato
- Intestazioni (H1-H6)
- Elenchi (puntati, numerati, attività)
- Link e immagini
- Blocchi di codice
- Tabelle

## Barra di Stato

In basso nell'editor:

- **Conteggio Parole/Caratteri**: Traccia la tua scrittura
- **Ultimo Salvataggio**: Quando la nota è stata salvata l'ultima volta
- **Percorso Nota**: Dove si trova la nota
- **Stato Sincronizzazione**: Se usi la sincronizzazione opzionale

## Menu Contestuali

Clic con il tasto destro per azioni contestuali:

### Menu Contestuale Nota

- Rinomina
- Sposta in cartella
- Duplica
- Elimina
- Esporta
- Copia link

### Menu Contestuale Editor

- Taglia, Copia, Incolla
- Opzioni di formattazione
- Inserisci link/immagine
- Trova e sostituisci

## Navigazione da Tastiera

Naviga l'interfaccia con la tastiera:

| Azione | Scorciatoia |
|--------|-------------|
| Attiva/Disattiva Barra Laterale | `Cmd/Ctrl + M` |
| Focus Ricerca | `Cmd/Ctrl + F` |
| Palette Comandi | `Cmd/Ctrl + P` |
| Nuova Nota | `Cmd/Ctrl + N` |

## Personalizzazione

Regola l'interfaccia a tuo piacimento:

### Larghezza Barra Laterale

1. Passa il mouse sul bordo della barra laterale
2. Trascina per ridimensionare
3. Oppure imposta la larghezza esatta nelle Impostazioni

### Tema

Passa tra modalità Chiara e Scura:

- Usa l'interruttore tema nell'intestazione
- Oppure imposta per seguire la preferenza di sistema

### Dimensione Carattere

1. Apri Impostazioni
2. Vai su Aspetto
3. Regola Dimensione Carattere Editor

## Suggerimenti

### Modalità Focus

Nascondi tutti i pannelli per una scrittura senza distrazioni:

1. Apri Palette Comandi
2. Digita "Modalità Focus"
3. Premi Invio

### Accesso Rapido

- Aggiungi le note usate frequentemente ai Preferiti
- Usa scorciatoie da tastiera per azioni comuni
- Personalizza la barra degli strumenti nelle Impostazioni
