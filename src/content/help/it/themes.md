# Temi e Aspetto

Personalizza l'aspetto di Classic per adattarlo al tuo stile e ridurre l'affaticamento degli occhi.

## Opzioni Tema

### Modalità Chiara

Il tema predefinito di Classic presenta:

- Interfaccia pulita e luminosa
- Ottimale per ambienti ben illuminati
- Alto contrasto per leggibilità

### Modalità Scura

Perfetta per nottambuli e lavoro in bassa luce:

- Riduce l'affaticamento degli occhi
- Minore consumo batteria su schermi OLED
- Riposa gli occhi in ambienti scuri

### Sistema (Auto)

Segui il tema del tuo sistema operativo:

- Cambia automaticamente in base alle impostazioni OS
- Nessun cambio manuale necessario
- Coerente con altre app

## Cambiare Tema

### Via Impostazioni

1. Apri Impostazioni (`Cmd+,` o `Ctrl+,`)
2. Naviga su **Aspetto**
3. Seleziona il tuo tema preferito

### Via Palette Comandi

1. Apri Palette Comandi (`Cmd+P` o `Ctrl+P`)
2. Digita "Tema"
3. Seleziona il tuo tema preferito

### Scorciatoia da Tastiera

| Piattaforma | Scorciatoia |
|-------------|-------------|
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Cicla attraverso: Chiara → Scura → Sistema

## Impostazioni Font

### Font Editor

Personalizza l'esperienza di scrittura:

1. Apri Impostazioni
2. Vai su **Aspetto** → **Editor**
3. Configura opzioni font

| Impostazione | Opzioni | Predefinito |
|--------------|---------|-------------|
| Famiglia Font | Font di sistema + personalizzati | UI di Sistema |
| Dimensione Font | 12-24px | 16px |
| Altezza Linea | 1.0-2.5 | 1.75 |
| Spaziatura Lettere | -2 a 4 | 0 |

### Font UI

Regola il font dell'interfaccia:

1. Apri Impostazioni
2. Vai su **Aspetto** → **Interfaccia**
3. Scegli font e dimensione

### Font Monospace

Per blocchi di codice e contenuti tecnici:

1. Apri Impostazioni
2. Vai su **Aspetto** → **Codice**
3. Seleziona font monospace

## Opzioni Interfaccia

### Barra Laterale

Personalizza il comportamento della barra laterale:

| Impostazione | Descrizione |
|--------------|-------------|
| **Auto-nascondi** | Nascondi barra laterale quando non in uso |
| **Larghezza** | Regola larghezza barra laterale (200-400px) |
| **Mostra Icone** | Attiva/disattiva visibilità icone |

### Layout Editor

Scegli la tua esperienza di modifica:

- **Vista Divisa**: Modifica e anteprima affiancate
- **Solo Anteprima**: Vedi solo output renderizzato
- **Solo Modifica**: Focus su Markdown grezzo

### Elenco Note

Configura come appaiono le note:

- **Vista Lista**: Compatta, una riga per nota
- **Vista Schede**: Schede anteprima con snippet
- **Vista Albero**: Struttura cartelle gerarchica

## Colori Accent

### Colori Disponibili

Scegli il tuo colore accent:

- Blu Classico (predefinito)
- Viola
- Verde
- Arancione
- Rosso
- Rosa
- Turchese

### Cambiare Colore Accent

1. Apri Impostazioni
2. Vai su **Aspetto** → **Colori**
3. Seleziona il tuo accent preferito

Il colore accent influisce su:

- Link e pulsanti
- Evidenziazioni selezione
- Elementi attivi
- Indicatori di focus

## Accessibilità

### Modalità Alto Contrasto

Per migliore visibilità:

1. Apri Impostazioni
2. Vai su **Aspetto** → **Accessibilità**
3. Abilita **Alto Contrasto**

### Movimento Ridotto

Minimizza animazioni:

1. Apri Impostazioni
2. Vai su **Aspetto** → **Accessibilità**
3. Abilita **Movimento Ridotto**

### Regolazioni Font

- Aumenta dimensione minima font
- Usa font friendly per dislessia
- Regola rapporti contrasto

## CSS Personalizzato (Avanzato)

Per utenti esperti che vogliono controllo completo:

1. Apri Impostazioni
2. Vai su **Aspetto** → **Avanzate**
3. Clicca **Apri CSS Personalizzato**
4. Aggiungi le tue personalizzazioni

### Esempi di Personalizzazione

```css
/* Intestazioni più grandi */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Colore link personalizzato */
a { color: #3b82f6; }

/* Spaziatura linee più ampia */
.prose p { line-height: 2; }
```

## Sincronizzazione Tema

Quando accedi (opzionale), le tue preferenze tema si sincronizzano tra dispositivi:

- Sincronizzazione automatica tema
- Sincronizzazione impostazioni font
- CSS personalizzato si sincronizza

## Suggerimenti

### Abbinare al Tuo Flusso di Lavoro

- Usa **Chiara** durante il giorno, **Scura** di notte
- La modalità **Sistema** gestisce questo automaticamente
- Considera l'illuminazione ambientale

### Ridurre l'Affaticamento degli Occhi

- Usa modalità Scura in bassa luce
- Aumenta dimensione font per sessioni lunghe
- Abilita Alto Contrasto se necessario
- Fai pause regolari

### Produttività

- Impara la scorciatoia per cambiare tema
- Imposta orari di cambio automatico
- Usa modalità Sistema per coerenza
