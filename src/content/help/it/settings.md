# Impostazioni

Personalizza Classic per funzionare esattamente come vuoi. Dall'aspetto al comportamento dell'editor, tutto è configurabile.

## Aprire Impostazioni

### Scorciatoia da Tastiera

| Piattaforma | Scorciatoia |
|-------------|-------------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Via Menu

- **macOS**: Classic → Impostazioni
- **Windows**: File → Impostazioni
- **Linux**: Modifica → Impostazioni

### Via Palette Comandi

1. Apri Palette Comandi (`Cmd+P` o `Ctrl+P`)
2. Digita "Impostazioni"
3. Premi Invio

## Cercare Impostazioni

Non trovi un'impostazione? Usa la barra di ricerca:

1. Apri Impostazioni
2. Digita nella casella di ricerca
3. Le impostazioni corrispondenti appaiono
4. Clicca per saltare a quell'impostazione

## Ripristinare Impostazioni

### Ripristina Tutte le Impostazioni

Torna ai predefiniti:

1. Apri Impostazioni
2. Scorri in fondo
3. Clicca "Ripristina Tutte le Impostazioni"
4. Conferma l'azione

### Ripristina Categoria

Ripristina categoria specifica:

1. Vai alla categoria
2. Clicca "Ripristina Predefiniti"
3. Solo quella categoria viene ripristinata

## Esportare/Importare Impostazioni

### Esporta Impostazioni

Salva la tua configurazione:

1. Apri Impostazioni
2. Clicca "Esporta Impostazioni"
3. Scegli posizione
4. Salva come file JSON

### Importa Impostazioni

Carica una configurazione salvata:

1. Apri Impostazioni
2. Clicca "Importa Impostazioni"
3. Seleziona il file JSON
4. Conferma importazione

## Posizione Impostazioni

Le impostazioni sono salvate localmente:

| Piattaforma | Posizione |
|-------------|-----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Sincronizza Impostazioni (Opzionale)

Quando hai effettuato l'accesso, le impostazioni possono sincronizzarsi tra dispositivi:

- **Sincronizza**: Abilita in Impostazioni → Sincronizzazione
- **Cosa Si Sincronizza**: Tema, font, scorciatoie
- **Cosa Non Si Sincronizza**: Percorsi locali, chiavi di crittografia

## Personalizzazioni Comuni

### Aumenta Dimensione Font

1. Impostazioni → Aspetto
2. Dimensione Font Editor → Aumenta
3. Le modifiche si applicano immediatamente

### Cambia Tema

1. Impostazioni → Aspetto
2. Tema → Seleziona Chiara/Scura/Sistema
3. L'app si aggiorna istantaneamente

### Abilita Auto-Backup

1. Impostazioni → Backup e Ripristino
2. Abilita "Auto Backup"
3. Imposta frequenza e posizione

### Personalizza Scorciatoie

1. Impostazioni → Scorciatoie da Tastiera
2. Trova l'azione
3. Clicca scorciatoia corrente
4. Premi nuovi tasti
5. Salva

## Suggerimenti

### Accesso Rapido

- Usa `Cmd/Ctrl+,` per aprire impostazioni rapidamente
- Cerca impostazioni per nome
- Usa categorie per navigare

### Esporta Prima di Modifiche

Prima di modifiche importanti, esporta le tue impostazioni:

- Facile da ripristinare se qualcosa va storto
- Condividi configurazione con altri

### Controlla Nuove Impostazioni

Gli aggiornamenti potrebbero aggiungere nuove impostazioni:

- Controlla dopo l'aggiornamento
- Esplora nuove opzioni

## Risoluzione Problemi

### Impostazioni Non Si Salvano

- Controlla permessi file
- Verifica spazio su disco
- Prova a ripristinare e riconfigurare

### Impostazioni Ripristinate Inaspettatamente

- Potrebbe indicare corruzione file
- Esporta impostazioni regolarmente
- Controlla conflitti app

### Non Trovo un'Impostazione

- Usa la barra di ricerca
- Controlla categorie diverse
- Potrebbe essere nelle Impostazioni Avanzate
