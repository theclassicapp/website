# Quick Capture

Capture ideas instantly without interrupting your workflow. Quick Capture opens a minimal note-taking window from anywhere on your system.

## Global Hotkey

Press the designated hotkey to open Quick Capture

| Platform | Default Shortcut |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

You can customize this shortcut in Settings.

## Using Quick Capture

1. Press the global hotkey (anywhere on your system)
2. Type your note in the popup window
3. Press `Enter` to save
4. Press `Escape` to cancel

## Features

### Minimal Interface

- Small, focused window
- Markdown support
- Auto-focus on input
- Keyboard-first design

### Quick Actions

- `# ` + space = heading
- `#inbox` = add to Inbox
- `#today` = add to Today's note
- Tags with `#tagname`

### Default Destination

Quick Capture notes go to your **Inbox** by default. You can change this

1. Open Settings
2. Go to Quick Capture
3. Select default destination

## Tips

### Capture Workflow

1. **Capture** - Jot down the idea quickly
2. **Process** - Review Inbox daily
3. **Organize** - Move to appropriate folder/note

### Best Practices

- Keep it brief - one idea per capture
- Use tags for context
- Don't worry about formatting - refine later
- Review and process captures regularly

### Integration

Quick Capture works great with

- **Daily Notes**: Start each day reviewing captures
- **Inbox**: Use as a holding area
- **Tags**: Add context immediately

## Troubleshooting

### Hotkey Not Working?

**macOS**
1. Check System Preferences > Keyboard > Shortcuts
2. Look for conflicts
3. Restart Classic

**Windows**
1. Run Classic as Administrator
2. Check for conflicting shortcuts
3. Restart Classic

### Window Not Appearing?

- Check if Classic is running
- Verify the shortcut
- Check for focus issues
- Restart Classic

### Captures Not Saving?

- Check default destination
- Verify write permissions
- Check for disk space
- Restart Classic
