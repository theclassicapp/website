# Slash-commando's

Slash-commando's zijn een krachtige manier om snel opmaak en speciale content in te voegen zonder het toetsenbord te verlaten. Typ `/` in de editor om het commandopalet te openen.

## Beschikbare Commando's

### Koppen

| Commando | Resultaat |
| ------- | ------ |
| `/h1` | Kop niveau 1 |
| `/h2` | Kop niveau 2 |
| `/h3` | Kop niveau 3 |
| `/h4` | Kop niveau 4 |
| `/h5` | Kop niveau 5 |
| `/h6` | Kop niveau 6 |

### Tekstopmaak

| Commando | Resultaat |
| ------- | ------ |
| `/bold` | **Vette tekst** |
| `/italic` | *Cursieve tekst* |
| `/strikethrough` | ~~Doorhalen~~ |
| `/code` | Inline codeblok |
| `/quote` | Blokcitaat |

### Lijsten

| Commando | Resultaat |
| ------- | ------ |
| `/bullet` | • Opsommingslijst |
| `/numbered` | 1. Genummerde lijst |
| `/todo` | - [ ] Taakitem |
| `/check` | - [x] Voltooid item |

### Content Invoegen

| Commando | Resultaat |
| ------- | ------ |
| `/image` | Afbeelding invoegen van URL of bestand |
| `/link` | Hyperlink invoegen |
| `/table` | Tabel invoegen |
| `/hr` | Horizontale lijn |
| `/code` | Codeblok met taalselectie |

### Speciale Blokken

| Commando | Resultaat |
| ------- | ------ |
| `/mermaid` | Mermaid diagramblok |
| `/katex` | KaTeX wiskundeblok |
| `/callout` | Callout/aandachtblok |

## Slash-commando's Gebruiken

1. Typ `/` in de editor
2. Er verschijnt een menu met alle beschikbare commando's
3. Begin met typen om commando's te filteren
4. Druk op `Enter` of klik om in te voegen

## Pro Tips

- **Snelle toegang**: Typ gewoon `/` en begin de commandonaam te typen
- **Fuzzy zoeken**: Je hoeft niet de exacte commandonaam te typen - Classic vindt de beste match
- **Toetsenbord navigatie**: Gebruik pijltjestoetsen om door het menu te navigeren
- **Recente commando's**: Vaak gebruikte commando's verschijnen bovenaan de lijst
