# Thema's & Weergave

Personaliseer het uiterlijk van Classic om aan jouw stijl te voldoen en oogvermoeidheid te verminderen.

## Thema Opties

### Licht Modus

Het standaard thema van Classic bevat:

- Schone, heldere interface
- Optimaal voor goed verlichte omgevingen
- Hoog contrast voor leesbaarheid

### Donker Modus

Perfect voor nachtuilen en werken bij weinig licht:

- Vermindert oogvermoeidheid
- Lager batterijverbruik op OLED schermen
- Zachter voor de ogen in donkere omgevingen

### Systeem (Auto)

Volg het thema van je besturingssysteem:

- Schakelt automatisch op basis van OS-instellingen
- Geen handmatige schakeling nodig
- Consistent met andere apps

## Thema's Wisselen

### Via Instellingen

1. Open Instellingen (`Cmd+,` of `Ctrl+,`)
2. Navigeer naar **Weergave**
3. Selecteer je voorkeursthema

### Via Commandopalet

1. Open Commandopalet (`Cmd+P` of `Ctrl+P`)
2. Typ "Thema"
3. Selecteer je voorkeursthema

### Sneltoets

| Platform | Sneltoets |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Schakelt door: Licht → Donker → Systeem

## Lettertype Instellingen

### Editor Lettertype

Pas de schrijfervaring aan:

1. Open Instellingen
2. Ga naar **Weergave** → **Editor**
3. Configureer lettertype opties

| Instelling | Opties | Standaard |
| ------- | ------- | ------- |
| Lettertype Familie | Systeem lettertypen + aangepast | Systeem UI |
| Lettergrootte | 12-24px | 16px |
| Regelhoogte | 1.0-2.5 | 1.75 |
| Letterspatiëring | -2 tot 4 | 0 |

### UI Lettertype

Pas het interface lettertype aan:

1. Open Instellingen
2. Ga naar **Weergave** → **Interface**
3. Kies lettertype en grootte

### Monospace Lettertype

Voor codeblokken en technische content:

1. Open Instellingen
2. Ga naar **Weergave** → **Code**
3. Selecteer monospace lettertype

## Interface Opties

### Zijbalk

Pas het zijbalkgedrag aan:

| Instelling | Beschrijving |
| ------- | ----------- |
| **Auto-verbergen** | Verberg zijbalk indien niet in gebruik |
| **Breedte** | Pas zijbalkbreedte aan (200-400px) |
| **Pictogrammen Tonen** | Schakel pictogram zichtbaarheid in/uit |

### Editor Indeling

Kies je bewerkingservaring:

- **Gesplitste Weergave**: Bewerken en voorbeeld zij aan zij
- **Alleen Voorbeeld**: Bekijk alleen gerenderde uitvoer
- **Alleen Bewerken**: Focus op ruwe Markdown

### Notitielijst

Configureer hoe notities verschijnen:

- **Lijstweergave**: Compact, één regel per notitie
- **Kaartweergave**: Voorbeeldkaarten met fragmenten
- **Boomweergave**: Hiërarchische mapstructuur

## Accentkleuren

### Beschikbare Kleuren

Kies je accentkleur:

- Klassiek Blauw (standaard)
- Paars
- Groen
- Oranje
- Rood
- Roze
- Turquoise

### Accentkleur Wijzigen

1. Open Instellingen
2. Ga naar **Weergave** → **Kleuren**
3. Selecteer je voorkeursaccent

De accentkleur beïnvloedt:

- Links en knoppen
- Selectie markeringen
- Actieve elementen
- Focus indicatoren

## Toegankelijkheid

### Hoog Contrast Modus

Voor betere zichtbaarheid:

1. Open Instellingen
2. Ga naar **Weergave** → **Toegankelijkheid**
3. Schakel **Hoog Contrast** in

### Verminderde Beweging

Minimaliseer animaties:

1. Open Instellingen
2. Ga naar **Weergave** → **Toegankelijkheid**
3. Schakel **Verminderde Beweging** in

### Lettertype Aanpassingen

- Verhoog minimum lettergrootte
- Gebruik dyslexie-vriendelijke lettertypen
- Pas contrastverhoudingen aan

## Aangepaste CSS (Geavanceerd)

Voor power users die volledige controle willen:

1. Open Instellingen
2. Ga naar **Weergave** → **Geavanceerd**
3. Klik **Open Aangepaste CSS**
4. Voeg je aanpassingen toe

### Voorbeeld Aanpassingen

```css
/* Grotere koppen */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Aangepaste link kleur */
a { color: #3b82f6; }

/* Ruimere regelafstand */
.prose p { line-height: 2; }
```

## Thema Synchronisatie

Wanneer je inlogt (optioneel), synchroniseren je themavoorkeuren tussen apparaten:

- Automatische thema synchronisatie
- Lettertype instellingen synchronisatie
- Aangepaste CSS synchroniseert

## Tips

### Passend bij Je Workflow

- Gebruik **Licht** overdag, **Donker** 's nachts
- **Systeem** modus handelt dit automatisch af
- Overweeg je omgevingslicht

### Oogvermoeidheid Verminderen

- Gebruik Donker modus bij weinig licht
- Verhoog lettergrootte voor lange sessies
- Schakel Hoog Contrast in indien nodig
- Neem regelmatig pauzes

### Productiviteit

- Leer de thema wissel sneltoets
- Stel auto-wissel tijden in
- Gebruik Systeem modus voor consistentie
