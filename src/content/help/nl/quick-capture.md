# Snelle Vastlegging

Leg ideeën direct vast zonder je workflow te onderbreken. Snelle Vastlegging opent een minimalistisch notitievenster vanaf elke locatie op je systeem.

## Globale Sneltoets

Druk op de toegewezen sneltoets om Snelle Vastlegging te openen:

| Platform | Standaard Sneltoets |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

Je kunt deze sneltoets aanpassen in Instellingen.

## Snelle Vastlegging Gebruiken

1. Druk op de globale sneltoets (overal op je systeem)
2. Typ je notitie in het popup-venster
3. Druk op `Enter` om op te slaan
4. Druk op `Escape` om te annuleren

## Functies

### Minimale Interface

- Klein, gefocust venster
- Markdown-ondersteuning
- Autofocus op invoer
- Toetsenbord-eerst ontwerp

### Snelle Acties

- `# ` + spatie = kop
- `#inbox` = toevoegen aan Inbox
- `#vandaag` = toevoegen aan notitie van vandaag
- Labels met `#labelnaam`

### Standaard Bestemming

Snelle Vastlegging notities gaan standaard naar je **Inbox**. Je kunt dit aanpassen:

1. Open Instellingen
2. Ga naar Snelle Vastlegging
3. Selecteer standaard bestemming

## Tips

### Vastlegging Workflow

1. **Vastleggen** - Noteer het idee snel
2. **Verwerken** - Beoordeel Inbox dagelijks
3. **Organiseren** - Verplaats naar de juiste map/notitie

### Best Practices

- Houd het kort - één idee per vastlegging
- Gebruik labels voor context
- Maak je geen zorgen over opmaak - verfijn later
- Beoordeel en verwerk vastleggingen regelmatig

### Integratie

Snelle Vastlegging werkt geweldig met:

- **Dagelijkse Notities**: Begin elke dag met het beoordelen van vastleggingen
- **Inbox**: Gebruik als verzamelplaats
- **Labels**: Voeg direct context toe

## Probleemoplossing

### Sneltoets Werkt Niet?

**macOS**
1. Controleer Systeemvoorkeuren > Toetsenbord > Sneltoetsen
2. Zoek naar conflicten
3. Herstart Classic

**Windows**
1. Voer Classic uit als Administrator
2. Controleer op conflicterende sneltoetsen
3. Herstart Classic

### Venster Verschijnt Niet?

- Controleer of Classic draait
- Verifieer de sneltoets
- Controleer op focusproblemen
- Herstart Classic

### Vastleggingen Worden Niet Opgeslagen?

- Controleer standaard bestemming
- Verifieer schrijfmachtigingen
- Controleer schijfruimte
- Herstart Classic
