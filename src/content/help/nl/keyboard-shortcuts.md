# Sneltoetsen

Werk sneller met de sneltoetsen van Classic. Alle sneltoetsen zijn aanpasbaar in Instellingen.

## Globale Sneltoetsen

Deze werken overal in de app.

### Algemeen

| Sneltoets (Windows/Linux) | Sneltoets (macOS) | Actie |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | Nieuwe notitie |
| `Ctrl+P` | `Cmd+P` | Commandopalet |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Globale snelle vastlegging |
| `Ctrl+S` | `Cmd+S` | Notitie opslaan |
| `Ctrl+F` | `Cmd+F` | Zoeken |
| `Ctrl+,` | `Cmd+,` | Instellingen |

### Navigatie

| Sneltoets (Windows/Linux) | Sneltoets (macOS) | Actie |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | Ga naar eerste notitie |
| `Ctrl+2` | `Cmd+2` | Ga naar tweede notitie in tabs |
| `Ctrl+Left` | `Cmd+Left` | Navigeer terug |
| `Ctrl+Right` | `Cmd+Right` | Navigeer vooruit |
| `Ctrl+Up` | `Cmd+Up` | Navigeer naar bovenliggende notitie |
| `Ctrl+Down` | `Cmd+Down` | Vouw uit/focus eerste subnotitie |

## Snelle Vastlegging

| Sneltoets (Windows/Linux) | Sneltoets (macOS) | Actie |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Open snelle vastlegging |
| `Escape` | `Escape` | Sluit snelle vastlegging |
| `Ctrl+Enter` | `Cmd+Enter` | Opslaan en sluiten |

## Sneltoetsen Aanpassen

1. Open **Instellingen** (`Ctrl+,` of `Cmd+,`)
2. Navigeer naar **Sneltoetsen**
3. Klik op een sneltoets om deze te wijzigen
4. Druk je nieuwe toetsencombinatie in
5. Klik **Opslaan**

## Tips

- De meeste sneltoetsen zijn aanpasbaar
- Conflicten met systeemsneltoetsen worden gemarkeerd
- Reset naar standaard door op de resetknop naast elke sneltoets te klikken
