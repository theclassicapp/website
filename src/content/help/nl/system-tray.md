# Systeemvak

Classic leeft in je systeemvak voor snelle toegang. Leg gedachten direct vast zonder je workflow te onderbreken.

## Wat Is Het Systeemvak?

Het systeemvak (Windows/Linux) of menubalk (macOS) is waar Classic op de achtergrond draait, altijd klaar voor snelle vastlegging.

### Voordelen

- **Altijd beschikbaar**: Een klik verwijderd
- **Snelle vastlegging**: Noteer gedachten direct
- **Achtergrond draaien**: App open houden is niet nodig
- **Meldingen**: Wordt herinnerd aan taken

## Systeemvak Inschakelen

### macOS

1. Open Classic
2. Ga naar Instellingen → Algemeen
3. Schakel "Toon in Menubalk" in
4. Classic pictogram verschijnt in menubalk

### Windows

1. Open Classic
2. Ga naar Instellingen → Algemeen
3. Schakel "Toon in Systeemvak" in
4. Classic pictogram verschijnt in systeemvak

### Linux

1. Open Classic
2. Ga naar Instellingen → Algemeen
3. Schakel "Systeemvak Pictogram" in
4. Afhankelijk van desktopomgeving

## Snelle Vastlegging

### Snelle Vastlegging Openen

Klik op het Classic pictogram in het systeemvak:

- **macOS**: Menubalk pictogram
- **Windows**: Systeemvak pictogram
- **Linux**: Systeemvak pictogram

### Snelle Vastlegging Gebruiken

1. Klik op systeemvak pictogram
2. Snelle vastlegging venster opent
3. Typ je notitie
4. Druk op Enter om op te slaan
5. Venster sluit automatisch

### Sneltoets

Open Snelle Vastlegging vanaf elke locatie:

| Platform | Sneltoets |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (globaal) |
| **Windows** | `Ctrl+Shift+N` (globaal) |
| **Linux** | `Ctrl+Shift+N` (globaal) |

## Systeemvak Menu

Klik met de rechtermuisknop op het vak-pictogram voor opties:

### Menu Opties

| Optie | Actie |
|--------|--------|
| **Nieuwe Notitie** | Open Snelle Vastlegging |
| **Nieuwe Dagelijkse Notitie** | Open dagelijkse notitie van vandaag |
| **Zoeken** | Open zoekvenster |
| **Open Classic** | Open hoofdvenster |
| **Instellingen** | Open instellingen |
| **Afsluiten** | Sluit Classic |

## Vak Instellingen

### Configureer in Instellingen

| Instelling | Beschrijving |
|---------|-------------|
| **Pictogram Tonen** | Schakel vak-pictogram zichtbaarheid in/uit |
| **Geminimaliseerd Starten** | Start Classic alleen in vak |
| **Sluiten naar Vak** | Blijf draaien bij sluiten venster |
| **Meldingen** | Meldingen in-/uitschakelen |

### Starten bij Login

Laat Classic automatisch starten:

1. Open Instellingen
2. Ga naar Algemeen
3. Schakel "Starten bij Login" in
4. Classic start bij opstarten

## Meldingen

### Meldingen Inschakelen

1. Open Instellingen
2. Ga naar Meldingen
3. Schakel "Meldingen Inschakelen" in

### Meldingstypen

- **Herinneringen**: Geplande herinneringen
- **Dagelijkse Notitie**: Dagelijkse notitie herinneringen
- **Updates**: App-update beschikbaar

## Venstergedrag

### Sluit Gedrag

Kies wat er gebeurt bij sluiten van het venster:

- **Minimaliseren naar Vak**: Blijf draaien, verberg venster
- **Afsluiten**: Sluit de applicatie
- **Vragen**: Vraag elke keer

### Minimaliseer Gedrag

- **Minimaliseren naar Vak**: Verberg in systeemvak
- **Minimaliseren naar Taakbalk**: Minimaliseer normaal

## Tips

### Snelle Vastlegging

Gebruik globale sneltoets voor snelste vastlegging:

1. Druk op `Cmd/Ctrl+Shift+N` overal
2. Typ je gedachte
3. Druk op Enter
4. Ga door met je werk

### Snelle Toegang

Klik op vak-pictogram om te zien:

- Recente notities
- Snelle acties
- Dagelijkse notitie van vandaag

### Achtergrond Draaien

Houd Classic draaiende voor:

- Snellere start
- Snelle vastlegging
- Meldingen
- Automatische back-up

## Probleemoplossing

### Vak-pictogram Ontbreekt

**macOS**

1. Controleer of pictogram is verborgen
2. Kijk in Controle Centrum (macOS Big Sur+)
3. Herstart Classic

**Windows**

1. Controleer verborgen pictogrammen (pijl in taakbalk)
2. Sleep Classic pictogram naar zichtbaar gebied
3. Herstart Classic

**Linux**

1. Controleer desktopomgeving ondersteuning
2. Installeer systeemvak extensie indien nodig
3. Herstart Classic

### Globale Sneltoets Werkt Niet

- Controleer op conflicten met andere apps
- Schakel de sneltoets opnieuw in bij Instellingen
- Herstart Classic
- Controleer OS-machtigingen

### Meldingen Verschijnen Niet

**macOS**

1. Systeemvoorkeuren → Meldingen
2. Zoek Classic
3. Schakel meldingen in

**Windows**

1. Instellingen → Systeem → Meldingen
2. Schakel Classic meldingen in

**Linux**

1. Controleer meldingen daemon
2. Installeer libnotify indien nodig

### Hoog Geheugen/CPU Gebruik

Als vak-pictogram te veel bronnen gebruikt:

1. Schakel animaties uit bij Instellingen
2. Verminder meldingsfrequentie
3. Controleer op app-updates
