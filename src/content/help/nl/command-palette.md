# Commandopalet

Het commandopalet is je toegangspoort tot elke actie in Classic. Druk op een eenvoudige sneltoets en heb direct toegang tot elke functie.

## Het Commandopalet Openen

| Platform | Sneltoets |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Hoe Te Gebruiken

1. Open het commandopalet (`Cmd+P` of `Ctrl+P`)
2. Typ om een commando te zoeken
3. Gebruik pijltjestoetsen om te navigeren
4. Druk op `Enter` om uit te voeren

## Commandocategorieën

### Bestandsbewerkingen

- **Nieuwe Notitie**: Maak een nieuwe notitie
- **Nieuwe Dagelijkse Notitie**: Maak/open de dagelijkse notitie van vandaag
- **Open Notitie**: Zoek en open bestaande notities
- **Notitie Opslaan**: Sla huidige notitie op
- **Notitie Verwijderen**: Verwijder huidige notitie
- **Notitie Hernoemen**: Hernoem huidige notitie

### Navigatie

- **Ga naar Notitie**: Spring naar een notitie
- **Ga naar Dagelijkse Notitie**: Open een specifieke datum
- **Ga naar Label**: Blader notities op label
- **Ga Terug**: Navigeer terug in geschiedenis
- **Ga Vooruit**: Navigeer vooruit in geschiedenis

### Editor

- **Voorbeeld In-/Uitschakelen**: Schakel tussen bewerk- en voorbeeldmodus
- **Zijbalk In-/Uitschakelen**: Toon/verberg de zijbalk
- **Focusmodus**: Start afleidingsvrij schrijven
- **Datum Invoegen**: Voeg huidige datum/tijd toe
- **Sjabloon Invoegen**: Voeg een sjabloon in

### Opmaak

- **Vet**: Maak tekst vet
- **Cursief**: Maak tekst cursief
- **Doorhalen**: Streep tekst door
- **Code**: Formatteer als code
- **Link**: Voeg een link in
- **Kop**: Pas kopformaat toe

### Organisatie

- **Label Toevoegen**: Label de huidige notitie
- **Verplaatsen naar Map**: Verplaats notitie naar een map
- **Notitie Archiveren**: Archiveer huidige notitie
- **Notitie Vastmaken**: Maak notitie vast aan bovenkant van lijst

### Weergave

- **Thema In-/Uitschakelen**: Schakel tussen licht/donker
- **Inzoomen**: Vergroot lettergrootte
- **Uitzoomen**: Verklein lettergrootte
- **Zoom Resetten**: Reset naar standaardgrootte

## Tips

### Snelle Toegang

- Commando's worden fuzzy doorzocht
- Recente commando's verschijnen eerst
- Maak vaak gebruikte commando's vast

### Toetsenbord Navigatie

- `↑`/`↓` - Navigeer resultaten
- `Enter` - Voer commando uit
- `Escape` - Sluit palet
- `Tab` - Blader door categorieën

### Aangepaste Commando's

1. Open Instellingen
2. Ga naar Commandopalet
3. Voeg aangepaste commando's toe
4. Definieer sneltoetsen

## Veelgebruikte Commando's

Hier zijn enkele commando's die je vaak zult gebruiken:

| Commando | Wat het doet |
| ------- | ------------- |
| `Nieuwe Notitie` | Maak lege notitie |
| `Open Notitie` | Doorzoek alle notities |
| `Nieuwe Dagelijkse Notitie` | Journaal van vandaag |
| `Voorbeeld In-/Uitschakelen` | Voorbeeld vs bewerken |
| `Label Toevoegen` | Label huidige notitie |
| `Snelle Vastlegging` | Globale vastlegging |

## Commando's Aanpassen

### Sneltoetsen Wijzigen

1. Open Instellingen (`Cmd+,` of `Ctrl+,`)
2. Navigeer naar Sneltoetsen
3. Zoek het commando
4. Klik op de huidige sneltoets
5. Druk je nieuwe sneltoets in
6. Sla op

### Aangepaste Commando's Maken

Geavanceerde gebruikers kunnen aangepaste commando's maken:

1. Open Instellingen
2. Ga naar Commando's
3. Klik "Aangepast Commando Toevoegen"
4. Definieer de commandonaam
5. Voeg de actie(s) toe
6. Stel een sneltoets in (optioneel)
