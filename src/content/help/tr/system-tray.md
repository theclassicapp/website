# Sistem Tepsisi

Classic, hızlı erişim için sistem tepsinizde yaşar. İş akışınızı kesmeden düşünceleri anında yakalayın.

## Sistem Tepsi Nedir?

Sistem tepsi (Windows/Linux) veya menü çubuğu (macOS), Classic'in her zaman hızlı yakalamaya hazır olduğu arka planda çalıştığı yerdir.

### Faydalar

- **Her zaman mevcut**: Bir tık uzakta
- **Hızlı yakalama**: Düşünceleri anında not edin
- **Arka planda çalışma**: Uygulamayı açık tutmanıza gerek yok
- **Bildirimler**: Görevler hakkında hatırlatma alın

## Sistem Tepsini Etkinleştirme

### macOS

1. Classic'i açın
2. Ayarlar → Genel'e gidin
3. "Menü Çubuğunda Göster"i etkinleştirin
4. Classic simgesi menü çubuğunda görünür

### Windows

1. Classic'i açın
2. Ayarlar → Genel'e gidin
3. "Sistem Tepsisinde Göster"i etkinleştirin
4. Classic simgesi sistem tepsisinde görünür

### Linux

1. Classic'i açın
2. Ayarlar → Genel'e gidin
3. "Sistem Tepsi Simgesi"ni etkinleştirin
4. Masaüstü ortamına bağlıdır

## Hızlı Yakalama

### Hızlı Yakalama'yı Açma

Sistem tepsindeki Classic simgesine tıklayın:

- **macOS**: Menü çubuğu simgesi
- **Windows**: Sistem tepsi simgesi
- **Linux**: Sistem tepsi simgesi

### Hızlı Yakalama'yı Kullanma

1. Sistem tepsi simgesine tıklayın
2. Hızlı yakalama penceresi açılır
3. Notunuzu yazın
4. Kaydetmek için Enter'a basın
5. Pencere otomatik olarak kapanır

### Klavye Kısayolu

Her yerden Hızlı Yakalama'yı açın:

| Platform | Kısayol |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (global) |
| **Windows** | `Ctrl+Shift+N` (global) |
| **Linux** | `Ctrl+Shift+N` (global) |

## Sistem Tepsi Menüsü

Seçenekler için tepsi simgesine sağ tıklayın

### Menü Seçenekleri

| Seçenek | Eylem |
|--------|--------|
| **Yeni Not** | Hızlı Yakalama'yı aç |
| **Yeni Günlük Not** | Bugünün günlük notunu aç |
| **Ara** | Arama penceresini aç |
| **Classic'i Aç** | Ana pencereyi aç |
| **Ayarlar** | Ayarları aç |
| **Çık** | Classic'ten çık |

## Tepsi Ayarları

### Ayarlar'da Yapılandırma

| Ayar | Açıklama |
|---------|-------------|
| **Simgeyi Göster** | Tepsi simgesi görünürlüğünü aç/kapat |
| **Simge Durumunda Başlat** | Classic'i sadece tepside başlat |
| **Tepsiye Kapat** | Pencere kapatıldığında çalışmaya devam et |
| **Bildirimler** | Bildirimleri etkinleştir/devre dışı bırak |

### Girişte Başlat

Classic'in otomatik olarak başlamasını sağlayın:

1. Ayarlar'ı açın
2. Genel'e gidin
3. "Girişte Başlat"ı etkinleştirin
4. Classic başlangıçta açılır

## Bildirimler

### Bildirimleri Etkinleştirme

1. Ayarlar'ı açın
2. Bildirimler'e gidin
3. "Bildirimleri Etkinleştir"i aç/kapat

### Bildirim Türleri

- **Hatırlatıcılar**: Zamanlanmış hatırlatıcılar
- **Günlük Not**: Günlük not istemleri
- **Güncellemeler**: Uygulama güncellemesi mevcut

## Pencere Davranışı

### Kapatma Davranışı

Pencereyi kapattığınızda ne olacağını seçin:

- **Tepsiye Küçült**: Çalışmaya devam et, pencereyi gizle
- **Çık**: Uygulamadan çık
- **Sor**: Her seferinde sor

### Küçültme Davranışı

- **Tepsiye Küçült**: Sistem tepsisinde gizle
- **Görev Çubuğuna Küçült**: Normal şekilde küçült

## İpuçları

### Hızlı Yakalama

En hızlı yakalama için global kısayolu kullanın:

1. Her yerde `Cmd/Ctrl+Shift+N`'e basın
2. Düşüncenizi yazın
3. Enter'a basın
4. Çalışmanıza devam edin

### Hızlı Erişim

Tepsi simgesine tıklayarak görün:

- Son notlar
- Hızlı işlemler
- Bugünün günlük notu

### Arka Planda Çalışma

Classic'in çalışmasını sürdürün:

- Daha hızlı başlangıç
- Hızlı yakalama
- Bildirimler
- Otomatik yedekleme

## Sorun Giderme

### Tepsi Simgesi Eksik

**macOS**

1. Simgenin gizli olup olmadığını kontrol edin
2. Kontrol Merkezi'ne bakın (macOS Big Sur+)
3. Classic'i yeniden başlatın

**Windows**

1. Gizli simgeleri kontrol edin (görev çubuğundaki ok)
2. Classic simgesini görünür alana sürükleyin
3. Classic'i yeniden başlatın

**Linux**

1. Masaüstü ortamı desteğini kontrol edin
2. Gerekirse sistem tepsi uzantısını yükleyin
3. Classic'i yeniden başlatın

### Global Kısayol Çalışmıyor

- Diğer uygulamalarla çakışmaları kontrol edin
- Kısayolu Ayarlar'da yeniden etkinleştirin
- Classic'i yeniden başlatın
- İşletim sistemi izinlerini kontrol edin

### Bildirimler Gösterilmiyor

**macOS**

1. Sistem Tercihleri → Bildirimler
2. Classic'i bulun
3. Bildirimleri etkinleştirin

**Windows**

1. Ayarlar → Sistem → Bildirimler
2. Classic bildirimlerini etkinleştirin

**Linux**

1. Bildirim daemon'unu kontrol edin
2. Gerekirse libnotify'ı yükleyin

### Yüksek Bellek/CPU Kullanımı

Tepsi simgesi çok fazla kaynak kullanıyorsa:

1. Ayarlar'daki animasyonları devre dışı bırakın
2. Bildirim sıklığını azaltın
3. Uygulama güncellemelerini kontrol edin
