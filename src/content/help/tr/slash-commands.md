# Bölü Komutları

Bölü komutları, klavyeyi terk etmeden hızlıca biçimlendirme ve özel içerik eklemenin güçlü bir yoludur. Komut paletini açmak için editörde `/` yazın.

## Mevcut Komutlar

### Başlıklar

| Komut | Sonuç |
| ----- | ----- |
| `/h1` | 1. seviye başlık |
| `/h2` | 2. seviye başlık |
| `/h3` | 3. seviye başlık |
| `/h4` | 4. seviye başlık |
| `/h5` | 5. seviye başlık |
| `/h6` | 6. seviye başlık |

### Metin Biçimlendirme

| Komut | Sonuç |
| ----- | ----- |
| `/bold` | **Kalın metin** |
| `/italic` | *Eğik metin* |
| `/strikethrough` | ~~Üstü çizili~~ |
| `/code` | Satır içi kod bloğu |
| `/quote` | Alıntı bloğu |

### Listeler

| Komut | Sonuç |
| ----- | ----- |
| `/bullet` | • Madde işaretli liste |
| `/numbered` | 1. Numaralı liste |
| `/todo` | - [ ] Yapılacak öğe |
| `/check` | - [x] Tamamlanmış öğe |

### İçerik Ekle

| Komut | Sonuç |
| ----- | ----- |
| `/image` | URL veya dosyadan resim ekle |
| `/link` | Köprü ekle |
| `/table` | Tablo ekle |
| `/hr` | Yatay çizgi |
| `/code` | Dil seçimi ile kod bloğu |

### Özel Bloklar

| Komut | Sonuç |
| ----- | ----- |
| `/mermaid` | Mermaid diyagram bloğu |
| `/katex` | KaTeX matematik bloğu |
| `/callout` | Çağrı/uyarı bloğu |

## Bölü Komutlarını Kullanma

1. Editörde `/` yazın
2. Mevcut tüm komutlarla bir menü görünür
3. Komutları filtrelemek için yazmaya başlayın
4. Eklemek için `Enter`'a basın veya tıklayın

## Profesyonel İpuçları

- **Hızlı erişim**: Sadece `/` yazın ve komut adını yazmaya başlayın
- **Bulanık arama**: Tam komut adını yazmanız gerekmez - Classic en yakın eşleşmeyi bulur
- **Klavye gezinme**: Menüde gezinmek için ok tuşlarını kullanın
- **Son komutlar**: Sık kullanılan komutlar listenin en üstünde görünür
