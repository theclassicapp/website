# Klavye Kısayolları

Classic'in klavye kısayollarıyla daha hızlı çalışın. Tüm kısayollar Ayarlar'da özelleştirilebilir.

## Global Kısayollar

Bunlar uygulamada her yerde çalışır.

### Genel

| Kısayol (Windows/Linux) | Kısayol (macOS) | Eylem |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | Yeni not |
| `Ctrl+P` | `Cmd+P` | Komut paleti |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Global hızlı yakalama |
| `Ctrl+S` | `Cmd+S` | Notu kaydet |
| `Ctrl+F` | `Cmd+F` | Ara |
| `Ctrl+,` | `Cmd+,` | Ayarlar |

### Gezinme

| Kısayol (Windows/Linux) | Kısayol (macOS) | Eylem |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | İlk nota git |
| `Ctrl+2` | `Cmd+2` | İkinci nota git |
| `Ctrl+Left` | `Cmd+Left` | Geriye git |
| `Ctrl+Right` | `Cmd+Right` | İleriye git |
| `Ctrl+Up` | `Cmd+Up` | Ana nota git |
| `Ctrl+Down` | `Cmd+Down` | İlk alt notu genişlet/odakla |

## Hızlı Yakalama

| Kısayol (Windows/Linux) | Kısayol (macOS) | Eylem |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Hızlı yakalamayı aç |
| `Escape` | `Escape` | Hızlı yakalamayı kapat |
| `Ctrl+Enter` | `Cmd+Enter` | Kaydet ve kapat |

## Kısayolları Özelleştirme

1. **Ayarlar**'ı açın (`Ctrl+,` veya `Cmd+,`)
2. **Klavye Kısayolları**'na gidin
3. Değiştirmek için herhangi bir kısayola tıklayın
4. Yeni tuş kombinasyonunuza basın
5. **Kaydet**'e tıklayın

## İpuçları

- Çoğu kısayol özelleştirilebilir
- Sistem kısayollarıyla çakışmalar vurgulanacaktır
- Her kısayolun yanındaki sıfırlama düğmesine tıklayarak varsayılana sıfırlayın
