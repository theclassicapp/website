# Tworzenie Pierwszej Notatki

Rozpocznij pracę z Classic, tworząc swoją pierwszą notatkę. Jest to proste i intuicyjne.

## Tworzenie Nowej Notatki

### Używając Paska Bocznego

1. Kliknij przycisk **+** na pasku bocznym
2. Nowa notatka zostanie utworzona
3. Zacznij pisać treść

### Używając Skrótu Klawiaturowego

| Platforma | Skrót |
| -------- | -------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Używając Palety Poleceń

1. Otwórz Paletę Poleceń (`Cmd+P` lub `Ctrl+P`)
2. Wpisz "Nowa Notatka"
3. Naciśnij Enter

## Pisanie Notatki

### Podstawowa Edycja

Po prostu zacznij pisać! Classic automatycznie zapisuje podczas pisania.

### Dodawanie Tytułu

Pierwsza linia Twojej notatki staje się tytułem. Użyj nagłówka dla najlepszych efektów

```markdown
# Moja Pierwsza Notatka

To jest treść mojej notatki.
```

### Formatowanie Tekstu

Użyj Markdown do formatowania tekstu

- **Pogrubienie**: `**tekst**` lub `Cmd/Ctrl+B`
- **Kursywa**: `*tekst*` lub `Cmd/Ctrl+I`
- **Przekreślenie**: `~~tekst~~`
- **Kod**: `` `kod` ``

## Zapisywanie

Classic automatycznie zapisuje Twoje notatki podczas pisania. Nie musisz ręcznie zapisywać.

### Ręczne Zapisywanie

Jeśli potrzebujesz, możesz wymusić zapis

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## Organizowanie Notatki

### Dodawanie Tagów

Otaguj notatkę dla łatwego późniejszego znalezienia

1. Wpisz `#` następnie nazwę tagu
2. Lub użyj przycisku tagu na pasku narzędzi
3. Wiele tagów jest obsługiwane

Przykład

```markdown
# Notatki ze Spotkania

Omówiono planowanie Q4 #praca #spotkania
```

### Przenoszenie do Folderu

1. Kliknij prawym przyciskiem myszy notatkę na pasku bocznym
2. Wybierz "Przenieś do Folderu"
3. Wybierz lub utwórz folder docelowy

### Przypinanie Ważnych Notatek

1. Kliknij prawym przyciskiem myszy notatkę
2. Wybierz "Przypnij na Górę"
3. Notatka pojawia się na górze listy

## Podgląd Notatki

Zobacz jak wygląda Twoja sformatowana notatka

- Kliknij przycisk **Podgląd**
- Lub użyj `Cmd/Ctrl+Shift+P`
- Przełączaj między trybami Edycji i Podglądu

## Wskazówki

### Tytuły Notatek

- Utrzymuj tytuły opisowe
- Pierwsza linia = tytuł notatki
- Użyj H1 (`#`) dla jasności

### Auto-Zapis

- Nigdy nie martw się o utratę pracy
- Zmiany zapisują się automatycznie
- Sprawdź pasek stanu dla statusu zapisu

### Szybkie Przechwytywanie

Potrzebujesz szybko coś zapisać?

- Użyj Szybkiego Przechwytywania (`Cmd/Ctrl+Shift+N`)
- Tworzy notatkę natychmiast
- Świetne dla ulotnych myśli
