# Języki i lokalizacja

Classic mówi Twoim językiem. Używaj aplikacji w preferowanym języku i pisz notatki w dowolnym języku, który wybierzesz.

## Obsługiwane języki

Interfejs Classic jest dostępny w wielu językach
| Język | Kod |
|----------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| Polski | `pl` |
| I więcej... | |
## Zmiana języka interfejsu
### Przez Ustawienia
1. Otwórz Ustawienia (`Cmd+,` lub `Ctrl+,`)
2. Przejdź do **Ogólne** → **Język**
3. Wybierz preferowany język
4. Uruchom ponownie Classic, aby zastosować
### Przez paletę poleceń
1. Otwórz paletę poleceń (`Cmd+P` lub `Ctrl+P`)
2. Wpisz "Język"
3. Wybierz preferowany język
4. Uruchom ponownie, aby zastosować zmiany
## Pisanie w wielu językach
Classic obsługuje pisanie notatek w dowolnym języku
### Obsługa Unicode
- Pełna obsługa Unicode
- Obsługiwane języki RTL (od prawej do lewej)
- Znaki CJK (chińskie, japońskie, koreańskie) działają idealnie
- Obsługa emoji
### Wykrywanie języka
Classic może wykrywać język Twoich notatek
- Automatyczne wykrywanie dla sprawdzania pisowni
- Ustawienie języka dla pojedynczej notatki
- Obsługa mieszanego języków
## Sprawdzanie pisowni
### Włączanie sprawdzania pisowni
1. Otwórz Ustawienia
2. Przejdź do **Edytor** → **Sprawdzanie pisowni**
3. Włącz "Sprawdzanie pisowni"
### Języki sprawdzania pisowni
1. Otwórz Ustawienia
2. Przejdź do **Edytor** → **Sprawdzanie pisowni**
3. Kliknij "Języki"
4. Wybierz języki do sprawdzania
### Dodawanie słów do słownika
1. Kliknij prawym przyciskiem myszy zaznaczzone słowo
2. Wybierz "Dodaj do słownika"
3. Słowo nie będzie więcej zaznaczone
## Obsługa RTL
Dla arabskiego, he hebrajskiego i innych języków RTL
### Automatyczne wykrywanie RTL
Classic automatycznie wykrywa treść RTL
- Dostosowuje kierunek tekstu
- Dostosowuje ruch kursora
- Zmienia się wyrównanie
### Ręczne przełączanie RTL
Wymuś kierunek RTL lub LTR
1. Otwórz paletę poleceń
2. Wpisz "RTL" lub "LTR"
3. Wybierz preferencję
## Metody wprowadzania
Classic współpracuje z metodami wprowadzania Twojego systemu
### Metody wprowadzania CJK
- Działa z Pinyin, Zhuyin, itp.
- Obsługuje japoński IME
- Obsługiwane koreańskie metody wprowadzania
### Znaki specjalne
Łatwo wstawiaj znaki specjalne
- Użyj selektora znaków systemu
- Kopiuj/wklejaj z map znaków
- Użyj wprowadzania Unicode
## Formaty daty i czasu
Daty dostosowują się do Twoich ustawień regionalnych
### Opcje formatu
| Ustawienie | Opcje |
|---------|---------|
| Format daty | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| Format czasu | 12-godzinny, 24-godzinny |
| Pierwszy dzień tygodnia | Niedziela, Poniedziałek, Sobota |
Skonfiguruj w Ustawienia → Ogólne → Data i czas
## Formaty liczb
Liczby wyświetlają się zgodnie z Twoimi ustawieniami regionalnymi
- **Separator dziesiętny**: `.` lub `,`
- **Separator tysięcy**: `,` lub `.` lub spacja
- **Symbole walutowe**: Na podstawie regionu
## Układy klawiatury
Classic współpracuje z dowolnym układem klawiatury
- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Układy niestandardowe
## Ustawienia lokalizacji
### W Ustawienia → Ogólne → Język
| Ustawienie | Opis |
|---------|-------------|
| **Język interfejsu** | Język UI aplikacji |
| **Format daty** | Jak pojawiają się daty |
| **Format czasu** | 12 lub 24 godziny |
| **Pierwszy dzień tygodnia** | Dzień rozpoczęcia kalendarza |
| **Format liczb** | Separatory dziesiętne/tysięcy |
## Wskazówki
### Notatki w wielu językach
Pisz w wielu językach w jednej notatce
```markdown
# Projekt miiedzynarodowy
English: Witamy w projekcie
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```
### Czcionki dla konkretnych języków
Niektóre języki wyglądają lepiej z konkretnymi czcionkami
1. Otwórz Ustawienia
2. Przejdź do Wygląd → Czcionki
3. Ustaw czcionki dla różnych skryptów
### Szybkie przełączanie języków
Utwórz skrót klawiaturowy
1. Otwórz Ustawienia → Skróty
2. Znajdź "Przełącz język"
3. Ustaw preferowany skrót
## Rozwiązywanie problemów
### Język niedostępny
- Sprawdź aktualizacje aplikacji
- Język może być w opracowy
- Wspomóż tłumaczenia na GitHub
### Znaki się nie wyświetlają
- Zainstaluj wymagane czcionki
- Sprawdź ustawienia czcionek
- Spróbuj innej czcionki
### Sprawdzanie pisowni nie działa
- Zweryfikuj, czy sprawdzanie pisowni jest włączone
- Dodaj język do sprawdzania pisowni
- Uruchom ponownie Classic
### RTL nie działa poprawnie
- Upewnij się, że wykrywanie RTL jest włączone
- Spróbuj ręczne przełączanie RTL
- Sprawdź problemy z mieszaną treścią
## Wspomaganie tłumaczeń
Pomóż tłumaczyć Classic
1. Odwiedź społeczność Classic
2. Dołącz do zespołu tłumaczy
3. Prześlij tłumaczenia
4. Pomóż ulepszyć lokalizację
