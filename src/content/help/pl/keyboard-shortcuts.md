# Skróty Klawiaturowe

Pracuj szybciej ze skrótami klawiaturowymi Classic. Wszystkie skróty są konfigurowalne w Ustawieniach.

## Globalne Skróty

Te działają gdziekolwiek w aplikacji.

### Ogólne

| Skrót (Windows/Linux) | Skrót (macOS) | Akcja |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | Nowa notatka |
| `Ctrl+P` | `Cmd+P` | Paleta poleceń |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Globalne szybkie przechwytywanie |
| `Ctrl+S` | `Cmd+S` | Zapisz notatkę |
| `Ctrl+F` | `Cmd+F` | Wyszukiwanie |
| `Ctrl+,` | `Cmd+,` | Ustawienia |

### Nawigacja

| Skrót (Windows/Linux) | Skrót (macOS) | Akcja |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | Przejdź do pierwszej notatki |
| `Ctrl+2` | `Cmd+2` | Przejdź do drugiej notatki w tab |
| `Ctrl+Left` | `Cmd+Left` | Nawiguj wstecz |
| `Ctrl+Right` | `Cmd+Right` | Nawiguj naprzód |
| `Ctrl+Up` | `Cmd+Up` | Nawiguj do notatki nadrzędnej |
| `Ctrl+Down` | `Cmd+Down` | Rozwiń/fokusuj pierwszą podnotatkę |

### Edytor

| ------------------------ | ----------------- | ------ |

| `Ctrl+I` | `Cmd+I` | Kursywa |
| `Ctrl+K` | `Cmd+K` | Wstaw link |

| `Tab` | `Tab` | Wcięcie |
| `Shift+Tab` | `Shift+Tab` | Usuń wcięcie |

### Zarządzanie Oknami

| Skrót (Windows/Linux) | Skrót (macOS) | Akcja |
| ------------------------ | ----------------- | ------ |
| `Ctrl+W` | `Cmd+W` | Zamknij bieżącą notatkę |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | Zamknij wszystkie notatki |
| `F11` | `F11` | Przełącz pełny ekran |
| `Ctrl+M` | `Cmd+M` | Przełącz pasek boczny |

## Szybkie Przechwytywanie

| Skrót (Windows/Linux) | Skrót (macOS) | Akcja |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Otwórz szybkie przechwytywanie |
| `Escape` | `Escape` | Zamknij szybkie przechwytywanie |
| `Ctrl+Enter` | `Cmd+Enter` | Zapisz i zamknij |

## Dostosowywanie Skrótów

1. Otwórz **Ustawienia** (`Ctrl+,` lub `Cmd+,`)
2. Nawiguj do **Skróty Klawiaturowe**
3. Kliknij dowolny skrót, aby go zmienić
4. Naciśnij swoją nową kombinację klawiszy
5. Kliknij **Zapisz**

## Wskazówki

- Większość skrótów jest konfigurowalna
- Konflikty ze skrótami systemowymi będą podświetlone
- Zresetuj do domyślnych, klikając przycisk resetowania obok każdego skrótu
