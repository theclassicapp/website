# Motywy i Wygląd

Personalizuj wygląd Classic, aby pasował do Twojego stylu i zmniejszy zmęczenie oczu.

## Opcje motywów
### Tryb jasny
Domyślny motyw Classic zawiera
- Czysty, jasny interfejs
- Optymalny dla dobrze oświetlonych pomieszczeń
- Wysoki kontrast dla czytelności
### Tryb ciemny
Idealny dla nocnych marek i pracy w słabym świetle
- Zmniejsza zmęczenie oczu
- Niższe zużycie baterii na ekranach OLED
- Łagodny dla oczu w ciemnych pomieszczeniach
### Systemowy (auto)
Podążaj za motywem systemu operacyjnego
- Automatycznie przełącza się na podstawie ustawień systemu
- Brak konieczności ręcznego przełączania
- Spójność z innymi aplikacjami
## Przełączanie motywów
### Przez Ustawienia
1. Otwórz Ustawienia (`Cmd+,` lub `Ctrl+,`)
2. Przejdź do **Wygląd**
3. Wybierz preferowany motyw
### Przez paletę poleceń
1. Otwórz paletę poleceń (`Cmd+P` lub `Ctrl+P`)
2. Wpisz "Motyw"
3. Wybierz preferowany motyw
### Skrót klawiaturowy
| Platforma | Skrót |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |
Cykl: Jasny → Ciemny → Systemowy
## Ustawienia czcionek
### Czcionka edytora
Dostosuj doświadczenie pisania
1. Otwórz Ustawienia
2. Przejdź do **Wygląd** → **Edytor**
3. Skonfiguruj opcje czcionki
| Ustawienie | Opcje | Domyślnie |
| ------- | ------- | ------- |
| Rodzina czcionek | Czcionki systemowe + niestandardowe | System UI |
| Rozmiar czcionki | 12-24px | 16px |
| Wysokość linii | 1.0-2.5 | 1.75 |
| Odstęp między literami | -2 do 4 | 0 |
### Czcionka interfejsu
Dostosuj czcionkę interfejsu
1. Otwórz Ustawienia
2. Przejdź do **Wygląd** → **Interfejs**
3. Wybierz czcionkę i rozmiar
### Czcionka o stałej szerokości
Dla bloków kodu i treści technicznych
1. Otwórz Ustawienia
2. Przejdź do **Wygląd** → **Kod**
3. Wybierz czcionkę o stałej szerokości
## Opcje interfejsu
### Pasek boczny
Dostosuj zachowanie paska bocznego
| Ustawienie | Opis |
| ------- | ----------- |
| **Automatyczne ukrywanie** | Ukryj pasek boczny, gdy nie jest używany |
| **Szerokość** | Dostosuj szerokość paska bocznego (200-400px) |
| **Pokazuj ikony** | Przełącz widoczność ikon |
### Układ edytora
Wybierz doświadczenie edycji
- **Widok dzielony**: Edytuj i podglądaj obok siebie
- **Tylko podgląd**: Zobacz tylko wyrenderowane wyjście
- **Tylko edycja**: Skup się na surowym Markdown
### Lista notatek
Skonfiguruj, jak pojawiają się notatki
- **Widok listy**: Kompaktowe, jedna linia na notatkę
- **Widok kart**: Karty podglądu z fragmentami
- **Widok drzewa**: Hierarchiczna struktura folderów
## Kolory akcentu
### Dostępne kolory
Wybierz swój kolor akcentu
- Klasyczny niebieski (domyślny)
- Fioletowy
- Zielony
- Pomarańczowy
- Czerwony
- Różowy
- Morski
### Zmiana koloru akcentu
1. Otwórz Ustawienia
2. Przejdź do **Wygląd** → **Kolory**
3. Wybierz preferowany akcent
Kolor akcentu wpływa na
- Linki i przyciski
- Podświetlenia zaznaczenia
- Aktywne elementy
- Wskaźniki fokusu
## Dostępność
### Tryb wysokiego kontrastu
Dla lepszej widoczności
1. Otwórz Ustawienia
2. Przejdź do **Wygląd** → **Dostępność**
3. Włącz **Wysoki kontrast**
### Zmniejszone ruch
Zminimalizuj animacje
1. Otwórz Ustawienia
2. Przejdź do **Wygląd** → **Dostępność**
3. Włącz **Zmniejszone ruch**
### Dostosowania czcionek
- Zwiększ minimalny rozmiar czcionki
- Używaj czcionek przyjaznych dla dysleksji
- Dostosuj wskaźniki kontrastu
## Niestandardowe CSS (zaawansowane)
Dla zaawansowanych użytkowników, którzy chcą pełnej kontroli
1. Otwórz Ustawienia
2. Przejdź do **Wygląd** → **Zaawansowane**
3. Kliknij **Otwórz niestandardowe CSS**
4. Dodaj swoje dostosowania
### Przykładowe dostosowania
```css
/* Większe nagłówki */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }
/* Niestandardowy kolor linków */
a { color: #3b82f6; }
/* Szersze odstępy między liniami */
.prose p { line-height: 2; }
```
## Synchronizacja motywów
Gdy się zalogujesz (opcjonalnie), preferencje motywu synchronizują się między urządzeniami
- Automatyczna synchronizacja motywu
- Synchronizacja ustawień czcionek
- Niestandardowe CSS się synchronizuje
## Wskazówki
### Dopasowanie do przepływu pracy
- Używaj **Jasny** w dzień, **Ciemny** w nocy
- Tryb **Systemowy** obsługuje to automatycznie
- Rozważ oświetlenie otoczenia
### Zmniejszanie zmęczenia oczu
- Używaj trybu ciemnego w słabym świetle
- Zwiększ rozmiar czcionki dla długich sesji
- Włącz wysoki kontrast, jeśli potrzeba
- Rób regularne przerwy
### Produktywność
- Naucz się skrót przełączania motywu
- Skonfiguruj czasy automatycznego przełączania
- Używaj trybu systemowego dla spójności
