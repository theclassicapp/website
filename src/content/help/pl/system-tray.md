# Zasobnik Systemowy

Classic żyje w Twoim zasobniku systemowym dla szybkiego dostępu. Przechwytuj myśli natychmiast bez przerywania swojego przepływu pracy.

## Czym Jest Zasobnik Systemowy?

Zasobnik systemowy (Windows/Linux) lub pasek menu (macOS) to miejsce, gdzie Classic działa w tle, zawsze gotowy do szybkiego przechwytywania.

### Korzyści

- **Zawsze dostępny**: Jeden klik z dala
- **Szybkie przechwytywanie**: Zapisz myśli natychmiast
- **Działanie w tle**: Nie musisz trzymać aplikacji otwartej
- **Powiadomienia**: Bądź przypominany o zadaniach

## Włączanie Zasobnika Systemowego

### macOS

1. Otwórz Classic
2. Przejdź do Ustawienia → Ogólne
3. Włącz "Pokaż w Pasku Menu"
4. Ikona Classic pojawia się w pasku menu

### Windows

1. Otwórz Classic
2. Przejdź do Ustawienia → Ogólne
3. Włącz "Pokaż w Zasobniku Systemowym"
4. Ikona Classic pojawia się w zasobniku systemowym

### Linux

1. Otwórz Classic
2. Przejdź do Ustawienia → Ogólne
3. Włącz "Ikona Zasobnika Systemowego"
4. Zależy od środowiska graficznego

## Szybkie Przechwytywanie

### Otwieranie Szybkiego Przechwytywania

Kliknij ikonę Classic w zasobniku systemowym

- **macOS**: Ikona paska menu
- **Windows**: Ikona zasobnika systemowego
- **Linux**: Ikona zasobnika systemowego

### Używanie Szybkiego Przechwytywania

1. Kliknij ikonę zasobnika systemowego
2. Otwiera się okno szybkiego przechwytywania
3. Wpisz swoją notatkę
4. Naciśnij Enter, aby zapisać
5. Okno zamyka się automatycznie

### Skrót Klawiaturowy

Otwórz Szybkie Przechwytywanie z dowolnego miejsca

| Platforma | Skrót |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (globalny) |
| **Windows** | `Ctrl+Shift+N` (globalny) |
| **Linux** | `Ctrl+Shift+N` (globalny) |

## Menu Zasobnika Systemowego

Kliknij prawym przyciskiem myszy ikonę zasobnika dla opcji

### Opcje Menu

| Opcja | Akcja |
|--------|--------|
| **Nowa Notatka** | Otwórz Szybkie Przechwytywanie |
| **Nowa Notatka Dzienna** | Otwórz dzisiejszą notatkę dzienną |
| **Wyszukaj** | Otwórz okno wyszukiwania |
| **Otwórz Classic** | Otwórz główne okno |
| **Ustawienia** | Otwórz ustawienia |
| **Wyjdź** | Zamknij Classic |

## Ustawienia Zasobnika

### Konfiguracja w Ustawieniach

| Ustawienie | Opis |
|---------|-------------|
| **Pokaż Ikonę** | Przełącz widoczność ikony zasobnika |
| **Uruchom Zminimalizowany** | Uruchom Classic tylko w zasobniku |
| **Zamknij do Zasobnika** | Utrzymuj działanie po zamknięciu okna |
| **Powiadomienia** | Włącz/wyłącz powiadomienia |

### Uruchamianie przy Logowaniu

Spraw, aby Classic uruchamiał się automatycznie

1. Otwórz Ustawienia
2. Przejdź do Ogólne
3. Włącz "Uruchom przy Logowaniu"
4. Classic uruchamia się przy starcie

## Powiadomienia

### Włączanie Powiadomień

1. Otwórz Ustawienia
2. Przejdź do Powiadomienia
3. Przełącz "Włącz Powiadomienia"

### Typy Powiadomień

- **Przypomnienia**: Zaplanowane przypomnienia
- **Notatka Dzienna**: Monity o notatce dziennej
- **Aktualizacje**: Dostępna aktualizacja aplikacji

## Zachowanie Okna

### Zachowanie przy Zamknięciu

Wybierz, co dzieje się po zamknięciu okna

- **Zminimalizuj do Zasobnika**: Utrzymuj działanie, ukryj okno
- **Wyjdź**: Zamknij aplikację
- **Zapytaj**: Pytaj za każdym razem

### Zachowanie przy Minimalizacji

- **Zminimalizuj do Zasobnika**: Ukryj w zasobniku systemowym
- **Zminimalizuj do Paska Zadań**: Zminimalizuj normalnie

## Wskazówki

### Szybkie Przechwytywanie

Użyj globalnego skrótu dla najszybszego przechwytywania

1. Naciśnij `Cmd/Ctrl+Shift+N` gdziekolwiek
2. Wpisz swoją myśl
3. Naciśnij Enter
4. Kontynuuj swoją pracę

### Szybki Dostęp

Kliknij ikonę zasobnika, aby zobaczyć

- Ostatnie notatki
- Szybkie akcje
- Dzisiejszą notatkę dzienną

### Działanie w Tle

Utrzymuj Classic uruchomiony dla

- Szybszego uruchamiania
- Szybkiego przechwytywania
- Powiadomień
- Auto-kopii zapasowej

## Rozwiązywanie Problemów

### Brakująca Ikona Zasobnika

**macOS**

1. Sprawdź, czy ikona jest ukryta
2. Szukaj w Centrum Kontroli (macOS Big Sur+)
3. Zrestartuj Classic

**Windows**

1. Sprawdź ukryte ikony (strzałka w pasku zadań)
2. Przeciągnij ikonę Classic do widocznego obszaru
3. Zrestartuj Classic

**Linux**

1. Sprawdź obsługę środowiska graficznego
2. Zainstaluj rozszerzenie zasobnika systemowego, jeśli potrzebne
3. Zrestartuj Classic

### Globalny Skrót Nie Działa

- Sprawdź konflikty z innymi aplikacjami
- Ponownie włącz skrót w Ustawieniach
- Zrestartuj Classic
- Sprawdź uprawnienia systemu operacyjnego

### Powiadomienia Nie Pojawiają Się

**macOS**

1. Preferencje Systemowe → Powiadomienia
2. Znajdź Classic
3. Włącz powiadomienia

**Windows**

1. Ustawienia → System → Powiadomienia
2. Włącz powiadomienia Classic

**Linux**

1. Sprawdź demona powiadomień
2. Zainstaluj libnotify, jeśli potrzebne

### Wysokie Użycie Pamięci/CPU

Jeśli ikona zasobnika zużywa zbyt wiele zasobów

1. Wyłącz animacje w Ustawieniach
2. Zmniejsz częstotliwość powiadomień
3. Sprawdź aktualizacje aplikacji
