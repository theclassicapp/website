# Paleta Poleceń

Paleta poleceń to Twoja brama do każdej akcji w Classic. Naciśnij prosty skrót klawiaturowy i uzyskaj dostęp do dowolnej funkcji natychmiast.

## Otwieranie Palety Poleceń

| Platforma | Skrót |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Jak Używać

1. Otwórz paletę poleceń (`Cmd+P` lub `Ctrl+P`)
2. Wpisz, aby wyszukać polecenie
3. Użyj klawiszy strzałek do nawigacji
4. Naciśnij `Enter`, aby wykonać

## Kategorie Poleceń

### Operacje na Plikach

- **Nowa Notatka**: Utwórz nową notatkę
- **Nowa Notatka Dzienna**: Utwórz/otwórz dzisiejszą notatkę dzienną
- **Otwórz Notatkę**: Wyszukaj i otwórz istniejące notatki
- **Zapisz Notatkę**: Zapisz bieżącą notatkę
- **Usuń Notatkę**: Usuń bieżącą notatkę
- **Zmień Nazwę Notatki**: Zmień nazwę bieżącej notatki

### Nawigacja

- **Przejdź do Notatki**: Skocz do dowolnej notatki
- **Przejdź do Notatki Dziennej**: Otwórz konkretną datę
- **Przejdź do Tagu**: Przeglądaj notatki według tagu
- **Wstecz**: Nawiguj wstecz w historii
- **Naprzód**: Nawiguj naprzód w historii

### Edytor

- **Przełącz Podgląd**: Przełączaj między trybami edycji i podglądu
- **Przełącz Pasek Boczny**: Pokaż/ukryj pasek boczny
- **Tryb Skupienia**: Wejdź w tryb pisania bez rozpraszaczy
- **Wstaw Datę**: Dodaj bieżącą datę/czas
- **Wstaw Szablon**: Wstaw szablon

### Formatowanie

- **Pogrubienie**: Pogrub tekst
- **Kursywa**: Pochyl tekst
- **Przekreślenie**: Przekreśl tekst
- **Kod**: Sformatuj jako kod
- **Link**: Wstaw link
- **Nagłówek**: Zastosuj format nagłówka

### Organizacja

- **Dodaj Tag**: Otaguj bieżącą notatkę
- **Przenieś do Folderu**: Przenieś notatkę do folderu
- **Zarchiwizuj Notatkę**: Zarchiwizuj bieżącą notatkę
- **Przypnij Notatkę**: Przypnij notatkę na górze listy

### Widok

- **Przełącz Motyw**: Przełączaj między jasnym/ciemnym
- **Przybliż**: Zwiększ rozmiar czcionki
- **Oddal**: Zmniejsz rozmiar czcionki
- **Resetuj Powiększenie**: Przywróć domyślny rozmiar

## Wskazówki

### Szybki Dostęp

- Polecenia są wyszukiwane rozmycie
- Ostatnie polecenia pojawiają się jako pierwsze
- Przypnij często używane polecenia

### Nawigacja Klawiaturą

- `↑`/`↓` - Nawiguj wyniki
- `Enter` - Wykonaj polecenie
- `Escape` - Zamknij paletę
- `Tab` - Przełączaj między kategoriami

### Niestandardowe Polecenia

1. Otwórz Ustawienia
2. Przejdź do Paleta Poleceń
3. Dodaj niestandardowe polecenia
4. Zdefiniuj skróty klawiaturowe

## Często Używane Polecenia

Oto kilka poleceń, których będziesz często używać

| Polecenie | Co robi |
| ------- | ------------- |
| `Nowa Notatka` | Utwórz pustą notatkę |
| `Otwórz Notatkę` | Przeszukaj wszystkie notatki |
| `Nowa Notatka Dzienna` | Dzisiejszy dziennik |
| `Przełącz Podgląd` | Podgląd vs edycja |
| `Dodaj Tag` | Otaguj bieżącą notatkę |
| `Szybkie Przechwytywanie` | Globalne przechwytywanie |

## Dostosowywanie Poleceń

### Zmiana Skrótów

1. Otwórz Ustawienia (`Cmd+,` lub `Ctrl+,`)
2. Nawiguj do Skróty Klawiaturowe
3. Znajdź polecenie
4. Kliknij bieżący skrót
5. Naciśnij swój nowy skrót
6. Zapisz

### Tworzenie Niestandardowych Poleceń

Zaawansowani użytkownicy mogą tworzyć niestandardowe polecenia

1. Otwórz Ustawienia
2. Przejdź do Polecenia
3. Kliknij "Dodaj Niestandardowe Polecenie"
4. Zdefiniuj nazwę polecenia
5. Dodaj akcje
6. Ustaw skrót (opcjonalnie)
