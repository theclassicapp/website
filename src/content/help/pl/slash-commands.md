# Polecenia Ukośnikowe

Polecenia ukośnikowe to potężny sposób na szybkie wstawianie formatowania i specjalnej treści bez odrywania rąk od klawiatury. Wpisz `/` w edytorze, aby otworzyć paletę poleceń.

## Dostępne Polecenia

### Nagłówki

| Polecenie | Wynik |
| ------- | ------ |
| `/h1` | Nagłówek poziomu 1 |
| `/h2` | Nagłówek poziomu 2 |
| `/h3` | Nagłówek poziomu 3 |
| `/h4` | Nagłówek poziomu 4 |
| `/h5` | Nagłówek poziomu 5 |
| `/h6` | Nagłówek poziomu 6 |

### Formatowanie Tekstu

| Polecenie | Wynik |
| ------- | ------ |
| `/bold` | **Tekst pogrubiony** |
| `/italic` | *Tekst kursywą* |
| `/strikethrough` | ~~Przekreślenie~~ |
| `/code` | Blok kodu w linii |
| `/quote` | Cytat blokowy |

### Listy

| Polecenie | Wynik |
| ------- | ------ |
| `/bullet` | • Lista punktowana |
| `/numbered` | 1. Lista numerowana |
| `/todo` | - [ ] Element do zrobienia |
| `/check` | - [x] Ukończony element |

### Wstawianie Treści

| Polecenie | Wynik |
| ------- | ------ |
| `/image` | Wstaw obraz z URL lub pliku |
| `/link` | Wstaw hiperlink |
| `/table` | Wstaw tabelę |
| `/hr` | Pozioma linia |
| `/code` | Blok kodu z wyborem języka |

### Bloki Specjalne

| Polecenie | Wynik |
| ------- | ------ |
| `/mermaid` | Blok diagramu Mermaid |
| `/katex` | Blok matematyczny KaTeX |
| `/callout` | Blok callout/admonition |

## Używanie Poleceń Ukośnikowych

1. Wpisz `/` w edytorze
2. Pojawi się menu ze wszystkimi dostępnymi poleceniami
3. Zacznij pisać, aby filtrować polecenia
4. Naciśnij `Enter` lub kliknij, aby wstawić

## Wskazówki dla Zaawansowanych

- **Szybki dostęp**: Po prostu wpisz `/` i zacznij pisać nazwę polecenia
- **Wyszukiwanie rozmyte**: Nie musisz wpisywać dokładnej nazwy polecenia - Classic znajdzie najbliższe dopasowanie
- **Nawigacja klawiaturą**: Użyj klawiszy strzałek do nawigacji po menu
- **Ostatnie polecenia**: Często używane polecenia pojawiają się na górze listy
