# 反向链接

反向链接显示笔记之间的连接，创建相关想法的网络。发现你不知道存在的关系。

## 什么是反向链接？

反向链接显示哪些笔记链接到当前笔记。它们是你知识图谱中的连接。

### 好处

- **发现连接**：自动找到相关笔记
- **上下文**：理解想法之间的关系
- **导航**：在连接的笔记之间跳转
- **知识图谱**：可视化你的思维

## 反向链接如何工作

当你创建指向另一个笔记的链接时，Classic 会自动创建一个反向链接。

```markdown
在笔记 A 中：
详见 [[项目计划]]。

结果：
- 笔记 A 链接到"项目计划"
- "项目计划"显示指向笔记 A 的反向链接
```

## 查看反向链接

### 反向链接面板

打开反向链接面板查看所有引用

1. 打开任意笔记
2. 点击侧边栏中的反向链接图标
3. 查看所有链接到此笔记的笔记

### 行内反向链接

反向链接显示在每个笔记的底部

- 显示源笔记名称
- 显示上下文片段
- 点击导航

## 创建链接

### Wiki 链接语法

使用双方括号链接到笔记

```markdown
[[笔记名称]]
[[另一个笔记]]
```

### 显示文本

自定义链接文本

```markdown
[[项目阿尔法|阿尔法项目]]
[[会议笔记|昨天的会议]]
```

### 链接到标题

链接到特定章节

```markdown
[[笔记名称#标题]]
[[笔记名称#章节|跳转到章节]]
```

## 反向链接类型

### 直接链接

你创建的显式 wiki 链接

```markdown
相关：[[相似笔记]]
```

### 未链接提及

提到标题但未链接的笔记

```markdown
笔记提到"项目阿尔法"但未链接
```

Classic 建议将这些转换为链接。

## 反向链接面板

### 功能

- **筛选**：搜索反向链接
- **排序**：按日期、名称或相关性
- **预览**：悬停预览内容
- **上下文**：查看周围文本

### 操作

- 点击打开链接的笔记
- 右键查看选项
- 转换未链接的提及

## 使用案例

### 知识管理

构建个人维基

```markdown
# 机器学习

与 [[人工智能]] 相关
建立在 [[统计学]] 基础上
应用于 [[自然语言处理]]
```

### 项目文档

连接相关文档

```markdown
# API 设计

参考 [[数据库架构]]
依赖于 [[认证流程]]
另见 [[前端集成]]
```

### 研究

链接概念和来源

```markdown
# 研究笔记

基于 [[来源论文 A]]
与 [[来源论文 B]] 矛盾
支持 [[假设 1]]
```

### 会议笔记

将会议连接到项目

```markdown
# 会议：Q4 计划

项目：[[项目阿尔法]]
上一次：[[Q3 回顾]]
下一次：[[Q4 启动]]
```

## 图谱视图

将反向链接可视化为网络

1. 打开图谱视图（`Cmd/Ctrl+G`）
2. 查看所有笔记作为节点
3. 线条显示连接
4. 点击节点导航

### 图谱控制

- **缩放**：滚动缩放
- **平移**：拖动移动
- **筛选**：按标签显示/隐藏
- **聚焦**：高亮笔记的连接

## 提示

### 慷慨链接

每当引用另一个概念时创建链接

```markdown
✅ 好：
讨论了 [[Q4 路线图]] 和 [[预算分配]]。

❌ 避免：
讨论了 Q4 路线图和预算分配。
```

### 使用一致的名称

完全匹配笔记标题以获得可靠的链接

```markdown
✅ [[会议笔记]]
❌ [[会议 笔记]] (区分大小写)
```

### 审查反向链接

定期检查反向链接以

- 发现被遗忘的连接
- 找到要合并的笔记
- 识别知识空白

### 链接预览

悬停在链接上预览而不打开

- 查看前几行
- 检查是否是正确的笔记
- 快速参考

## 键盘快捷键

| 操作 | 快捷键 |
|--------|----------|
| 打开反向链接 | `Cmd/Ctrl+Shift+B` |
| 创建链接 | `[[`（自动完成） |
| 打开图谱视图 | `Cmd/Ctrl+G` |
| 导航链接 | `Tab` 切换链接 |

## 高级功能

### 嵌入

嵌入其他笔记的内容

```markdown
![[笔记名称]]
![[笔记名称#章节]]
```

### 包含

行内包含内容

```markdown
## 相关内容
![[共享模板#标题]]
```

### 链接着色

链接根据以下情况改变颜色

- **蓝色**：未访问的链接
- **紫色**：已访问的链接
- **红色**：断开的链接（笔记不存在）

## 故障排除

### 缺少反向链接

- 检查链接语法 `[[笔记名称]]`
- 验证笔记标题完全匹配
- 刷新反向链接面板

### 断开的链接

- 笔记可能已被重命名
- 笔记可能已被删除
- 使用搜索找到正确的笔记

### 重复链接

- 合并相似的笔记
- 为重命名的笔记使用重定向
- 检查标题变体
