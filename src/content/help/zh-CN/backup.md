# 备份与恢复

使用 Classic 内置的备份系统保护你的笔记安全。再也不丢失工作成果。

## 自动备份

### 工作原理

Classic 会定期自动创建笔记备份

- **默认频率**：每 30 分钟
- **备份位置**：与笔记相同的文件夹
- **保留期限**：保留最近 10 个备份
- **格式**：压缩存档（`.cbak`）

### 配置自动备份

1. 打开设置
2. 进入**备份与恢复**
3. 配置以下选项

| 设置 | 选项 | 默认值 |
| ------- | ------- | ------- |
| 启用自动备份 | 开/关 | 开 |
| 备份频率 | 15分钟 / 30分钟 / 1小时 / 4小时 | 30分钟 |
| 保留备份数 | 5 / 10 / 20 / 50 | 10 |
| 备份位置 | 自定义路径 | 笔记文件夹 |

## 手动备份

### 立即创建备份

1. 打开命令面板（`Cmd/Ctrl+P`）
2. 输入"备份"
3. 选择**立即创建备份**
4. 选择位置（可选）
5. 备份已创建

### 备份文件格式

备份存储为 `.cbak` 文件，包含

- 所有笔记（Markdown 文件）
- 附件
- 设置
- 元数据（标签、链接等）

## 从备份恢复

### 恢复过程

1. 打开设置
2. 进入**备份与恢复**
3. 点击**从备份恢复**
4. 选择备份文件
5. 选择恢复选项
6. 确认恢复

### 恢复选项

| 选项 | 描述 |
| ------ | ----------- |
| **全部替换** | 完全用备份替换当前笔记 |
| **合并** | 将备份与现有笔记合并 |
| **预览** | 恢复前查看备份内容 |

### 恢复的内容

- ✅ 所有笔记和文件夹
- ✅ 附件
- ✅ 标签和元数据
- ✅ 笔记历史
- ⚠️ 设置（可选）
- ❌ 应用程序首选项

## 备份位置

### 默认位置

```
~/Classic/backups/
├── backup-2024-01-15-10-30.cbak
├── backup-2024-01-15-11-00.cbak
└── backup-2024-01-15-11-30.cbak
```

### 自定义位置

1. 打开设置
2. 进入备份与恢复
3. 点击"更改备份位置"
4. 选择你喜欢的文件夹

### 云备份

将备份位置指向云同步文件夹

- **iCloud**: `~/Library/Mobile Documents/com~apple~CloudDocs/Classic Backups`
- **Dropbox**: `~/Dropbox/Classic Backups`
- **Google Drive**: `~/Google Drive/Classic Backups`
- **OneDrive**: `~/OneDrive/Classic Backups`

## 最佳实践

### 备份计划

- **自动备份**：保持启用，间隔 30 分钟
- **手动备份**：在进行重大更改前创建
- **异地备份**：定期将备份复制到外部存储

### 备份验证

定期验证你的备份

1. 创建测试恢复
2. 检查笔记内容
3. 验证附件
4. 确认元数据

### 备份轮换

- 在本地保留最近备份
- 将较旧备份归档到外部存储
- 删除非常旧的备份以节省空间

## 故障排除

### 备份失败

**可能原因**

- 磁盘空间不足
- 权限问题
- 文件正在使用中

**解决方案**

1. 检查可用磁盘空间
2. 验证文件夹权限
3. 关闭其他应用程序
4. 尝试不同位置

### 恢复失败

**可能原因**

- 备份文件损坏
- 版本不兼容
- 空间不足

**解决方案**

1. 尝试较旧的备份
2. 检查备份文件完整性
3. 释放磁盘空间
4. 联系支持

### 备份丢失

**可能原因**

- 自动备份已禁用
- 备份位置已更改
- 备份已删除

**解决方案**

1. 检查备份设置
2. 验证备份位置
3. 启用自动备份
4. 立即创建手动备份

## 高级选项

### 加密备份

增加安全性

1. 打开设置
2. 进入备份与恢复
3. 启用"加密备份"
4. 设置密码
5. 记住此密码 - 无法恢复

### 增量备份

使用增量备份节省空间

- 仅存储自上次备份以来的更改
- 完整恢复可重建完整状态
- 减少备份时间和大小

### 备份排除

从备份中排除特定内容

1. 打开设置
2. 进入备份与恢复
3. 点击"排除项"
4. 添加要排除的文件夹或文件
