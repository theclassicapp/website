# 语言与本地化

Classic 会说你的语言。使用你偏好的语言使用应用程序，并用你选择的任何语言编写笔记。

## 支持的语言

Classic 的界面支持多种语言

| 语言 | 代码 |
|----------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| 更多... | |

## 更改界面语言

### 通过设置

1. 打开设置（`Cmd+,` 或 `Ctrl+,`）
2. 进入**通用** → **语言**
3. 选择你偏好的语言
4. 重启 Classic 以应用

### 通过命令面板

1. 打开命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 输入"语言"
3. 选择你偏好的语言
4. 重启以应用更改

## 用多种语言写作

Classic 支持用任何语言编写笔记

### Unicode 支持

- 完整的 Unicode 支持
- 支持 RTL（从右到左）语言
- CJK（中文、日文、韩文）字符完美工作
- Emoji 支持

### 语言检测

Classic 可以检测你笔记的语言

- 拼写检查自动检测
- 每笔记语言设置
- 混合语言支持

## 拼写检查

### 启用拼写检查

1. 打开设置
2. 进入**编辑器** → **拼写检查**
3. 启用"拼写检查"

### 拼写检查语言

1. 打开设置
2. 进入**编辑器** → **拼写检查**
3. 点击"语言"
4. 选择要检查的语言

### 添加词到词典

1. 右键点击标记的词
2. 选择"添加到词典"
3. 该词不再被标记

## RTL 支持

对于阿拉伯语、希伯来语和其他 RTL 语言

### 自动 RTL 检测

Classic 自动检测 RTL 内容

- 文本方向自动调整
- 光标移动适应
- 对齐方式改变

### 手动 RTL 切换

强制 RTL 或 LTR 方向

1. 打开命令面板
2. 输入"RTL"或"LTR"
3. 选择你的偏好

## 输入法

Classic 与你系统的输入法配合工作

### CJK 输入法

- 与拼音、注音等配合工作
- 支持日文 IME
- 支持韩文输入法

### 特殊字符

轻松插入特殊字符

- 使用系统的字符选择器
- 从字符映射复制/粘贴
- 使用 Unicode 输入

## 日期与时间格式

日期适应你的区域设置

### 格式选项

| 设置 | 选项 |
|---------|---------|
| 日期格式 | MM/DD/YYYY、DD/MM/YYYY、YYYY-MM-DD |
| 时间格式 | 12 小时制、24 小时制 |
| 一周的第一天 | 周日、周一、周六 |

在设置 → 通用 → 日期与时间中配置

## 数字格式

数字根据你的区域设置显示

- **小数分隔符**：`.` 或 `,`
- **千位分隔符**：`,` 或 `.` 或空格
- **货币符号**：基于地区

## 键盘布局

Classic 与任何键盘布局配合工作

- QWERTY、AZERTY、QWERTZ
- Dvorak、Colemak
- 自定义布局

## 本地化设置

### 在设置 → 通用 → 语言中

| 设置 | 描述 |
|---------|-------------|
| **界面语言** | 应用 UI 语言 |
| **日期格式** | 日期显示方式 |
| **时间格式** | 12 或 24 小时 |
| **一周的第一天** | 日历起始日 |
| **数字格式** | 小数/千位分隔符 |

## 提示

### 混合语言笔记

在一个笔记中用多种语言编写

```markdown
# 国际项目

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### 特定语言字体

某些语言使用特定字体会更好看

1. 打开设置
2. 进入外观 → 字体
3. 为不同文字设置字体

### 快速切换语言

创建键盘快捷键

1. 打开设置 → 快捷键
2. 找到"切换语言"
3. 设置你偏好的快捷键

## 故障排除

### 语言不可用

- 检查应用更新
- 语言可能正在开发中
- 在 GitHub 上贡献翻译

### 字符不显示

- 安装所需字体
- 检查字体设置
- 尝试不同字体

### 拼写检查不工作

- 验证拼写检查已启用
- 将语言添加到拼写检查
- 重启 Classic

### RTL 不正常工作

- 确保 RTL 检测已开启
- 尝试手动 RTL 切换
- 检查混合内容问题
