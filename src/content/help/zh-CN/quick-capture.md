# 快速捕捉

无需打断工作流程即可即时捕捉想法。快速捕捉可在系统任意位置打开一个简洁的笔记窗口。

## 全局热键

按指定热键打开快速捕捉

| 平台 | 默认快捷键 |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

你可以在设置中自定义此快捷键。

## 使用快速捕捉

1. 按全局热键（系统任意位置）
2. 在弹出窗口中输入笔记
3. 按 `Enter` 保存
4. 按 `Escape` 取消

## 功能

### 简洁界面

- 小巧、专注的窗口
- 支持 Markdown
- 自动聚焦输入
- 键盘优先设计

### 快捷操作

- `# ` + 空格 = 标题
- `#收件箱` = 添加到收件箱
- `#今天` = 添加到今天的笔记
- 使用 `#标签名` 添加标签

### 默认目标

快速捕捉的笔记默认保存到**收件箱**。你可以更改此设置

1. 打开设置
2. 进入快速捕捉
3. 选择默认目标

## 提示

### 捕捉工作流

1. **捕捉** - 快速记下想法
2. **处理** - 每天审查收件箱
3. **整理** - 移动到适当的文件夹/笔记

### 最佳实践

- 保持简洁 - 每次捕捉一个想法
- 使用标签添加上下文
- 不要担心格式 - 稍后完善
- 定期审查和处理捕捉内容

### 集成

快速捕捉与以下功能配合良好

- **日记**：每天开始时审查捕捉内容
- **收件箱**：用作暂存区
- **标签**：立即添加上下文

## 故障排除

### 热键不工作？

**macOS**
1. 检查系统偏好设置 > 键盘 > 快捷键
2. 查找冲突
3. 重启 Classic

**Windows**
1. 以管理员身份运行 Classic
2. 检查快捷键冲突
3. 重启 Classic

### 窗口不显示？

- 检查 Classic 是否正在运行
- 验证快捷键
- 检查焦点问题
- 重启 Classic

### 捕捉内容未保存？

- 检查默认目标
- 验证写入权限
- 检查磁盘空间
- 重启 Classic
