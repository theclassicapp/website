# 系统托盘

Classic 常驻系统托盘，便于快速访问。无需打断工作流程即可即时捕捉想法。

## 什么是系统托盘？

系统托盘（Windows/Linux）或菜单栏（macOS）是 Classic 在后台运行的位置，随时准备快速捕捉。

### 好处

- **始终可用**：一键即达
- **快速捕捉**：即时记录想法
- **后台运行**：无需保持应用打开
- **通知**：获得任务提醒

## 启用系统托盘

### macOS

1. 打开 Classic
2. 进入设置 → 通用
3. 启用"在菜单栏显示"
4. Classic 图标出现在菜单栏

### Windows

1. 打开 Classic
2. 进入设置 → 通用
3. 启用"在系统托盘显示"
4. Classic 图标出现在系统托盘

### Linux

1. 打开 Classic
2. 进入设置 → 通用
3. 启用"系统托盘图标"
4. 取决于桌面环境

## 快速捕捉

### 打开快速捕捉

点击系统托盘中的 Classic 图标

- **macOS**：菜单栏图标
- **Windows**：系统托盘图标
- **Linux**：系统托盘图标

### 使用快速捕捉

1. 点击系统托盘图标
2. 快速捕捉窗口打开
3. 输入笔记
4. 按 Enter 保存
5. 窗口自动关闭

### 键盘快捷键

从任意位置打开快速捕捉

| 平台 | 快捷键 |
|----------|----------|
| **macOS** | `Cmd+Shift+N`（全局） |
| **Windows** | `Ctrl+Shift+N`（全局） |
| **Linux** | `Ctrl+Shift+N`（全局） |

## 系统托盘菜单

右键点击托盘图标查看选项

### 菜单选项

| 选项 | 操作 |
|--------|--------|
| **新建笔记** | 打开快速捕捉 |
| **新建日记** | 打开今天的日记 |
| **搜索** | 打开搜索窗口 |
| **打开 Classic** | 打开主窗口 |
| **设置** | 打开设置 |
| **退出** | 退出 Classic |

## 托盘设置

### 在设置中配置

| 设置 | 描述 |
|---------|-------------|
| **显示图标** | 切换托盘图标可见性 |
| **启动时最小化** | 仅在托盘启动 Classic |
| **关闭到托盘** | 关闭窗口时保持运行 |
| **通知** | 启用/禁用通知 |

### 登录时启动

让 Classic 自动启动

1. 打开设置
2. 进入通用
3. 启用"登录时启动"
4. Classic 在启动时启动

## 通知

### 启用通知

1. 打开设置
2. 进入通知
3. 切换"启用通知"

### 通知类型

- **提醒**：计划的提醒
- **日记**：日记提示
- **更新**：有应用更新可用

## 窗口行为

### 关闭行为

选择关闭窗口时发生什么

- **最小化到托盘**：保持运行，隐藏窗口
- **退出**：退出应用程序
- **询问**：每次提示

### 最小化行为

- **最小化到托盘**：在系统托盘隐藏
- **最小化到任务栏**：正常最小化

## 提示

### 快速捕捉

使用全局快捷键进行最快捕捉

1. 在任意位置按 `Cmd/Ctrl+Shift+N`
2. 输入想法
3. 按 Enter
4. 继续工作

### 快速访问

点击托盘图标查看

- 最近笔记
- 快捷操作
- 今天的日记

### 后台运行

保持 Classic 运行以

- 更快启动
- 快速捕捉
- 通知
- 自动备份

## 故障排除

### 托盘图标丢失

**macOS**

1. 检查图标是否隐藏
2. 查看控制中心（macOS Big Sur+）
3. 重启 Classic

**Windows**

1. 检查隐藏图标（任务栏箭头）
2. 将 Classic 图标拖到可见区域
3. 重启 Classic

**Linux**

1. 检查桌面环境支持
2. 如需要，安装系统托盘扩展
3. 重启 Classic

### 全局快捷键不工作

- 检查与其他应用的冲突
- 在设置中重新启用快捷键
- 重启 Classic
- 检查操作系统权限

### 通知不显示

**macOS**

1. 系统偏好设置 → 通知
2. 找到 Classic
3. 启用通知

**Windows**

1. 设置 → 系统 → 通知
2. 启用 Classic 通知

**Linux**

1. 检查通知守护进程
2. 如需要，安装 libnotify

### 内存/CPU 使用率高

如果托盘图标使用过多资源

1. 在设置中禁用动画
2. 减少通知频率
3. 检查应用更新
