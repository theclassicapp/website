# 子笔记

子笔记让您在 Classic 中创建嵌套、分层的笔记。将复杂主题分解为易于管理的部分。

## 什么是子笔记？

子笔记是笔记中的笔记——一种创建分层内容结构的方式。

### 优势

- **组织**：将复杂主题分解为更小的部分
- **嵌套**：创建多个层次结构
- **导航**：轻松在相关笔记间跳转
- **灵活性**：按需展开和折叠

## 创建子笔记

### 方法 1：拖放

1. 创建父笔记
2. 创建新笔记
3. 将新笔记拖到父笔记上
4. 它成为子笔记

### 方法 2：上下文菜单

1. 右键点击笔记
2. 选择"新建子笔记"
3. 嵌套笔记被创建

### 方法 3：键盘快捷键

1. 选择笔记
2. 按 `Cmd+Shift+N` (macOS) 或 `Ctrl+Shift+N` (Windows/Linux)
3. 子笔记被创建

## 管理子笔记

### 展开和折叠

点击有子笔记的笔记旁边的箭头

- **展开**：显示所有嵌套笔记
- **折叠**：隐藏嵌套笔记

### 移动子笔记

拖放以重新组织

1. 点击并按住子笔记
2. 拖到新位置
3. 放置以重新定位

### 转换为主笔记

将子笔记提升到顶层

1. 右键点击子笔记
2. 选择"移到根目录"
3. 笔记成为主笔记

## 子笔记深度

Classic 支持多层嵌套

```
📁 项目
  📄 概述
  📁 研究
    📄 用户访谈
    📄 调查结果
    📁 分析
      📄 关键发现
      📄 建议
  📁 计划
    📄 时间线
    📄 资源
```

### 最佳实践

- **限制深度**：最多 3-4 层以保持清晰
- **逻辑分组**：将相关内容分组在一起
- **清晰命名**：为子笔记使用描述性名称

## 键盘快捷键

| 操作 | 快捷键 |
|------|--------|
| 新建子笔记 | `Cmd/Ctrl+Shift+N` |
| 全部展开 | `Cmd/Ctrl+Down` |
| 全部折叠 | `Cmd/Ctrl+Up` |
| 上移 | `Opt/Alt+Up` |
| 下移 | `Opt/Alt+Down` |
| 提升 | `Opt/Alt+Left` |
| 降低 | `Opt/Alt+Right` |
