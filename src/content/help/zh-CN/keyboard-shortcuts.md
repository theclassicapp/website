# 键盘快捷键

使用 Classic 的键盘快捷键加快工作速度。所有快捷键都可在设置中自定义。

## 全局快捷键

这些快捷键在应用任意位置都有效。

### 通用

| 快捷键 (Windows/Linux) | 快捷键 (macOS) | 操作 |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | 新建笔记 |
| `Ctrl+P` | `Cmd+P` | 命令面板 |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | 全局快速捕捉 |
| `Ctrl+S` | `Cmd+S` | 保存笔记 |
| `Ctrl+F` | `Cmd+F` | 搜索 |
| `Ctrl+,` | `Cmd+,` | 设置 |

### 导航

| 快捷键 (Windows/Linux) | 快捷键 (macOS) | 操作 |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | 跳转到第一个笔记 |
| `Ctrl+2` | `Cmd+2` | 跳转到第二个笔记 |
| `Ctrl+Left` | `Cmd+Left` | 后退导航 |
| `Ctrl+Right` | `Cmd+Right` | 前进导航 |
| `Ctrl+Up` | `Cmd+Up` | 导航到父笔记 |
| `Ctrl+Down` | `Cmd+Down` | 展开/聚焦第一个子笔记 |

### 编辑器

| 快捷键 (Windows/Linux) | 快捷键 (macOS) | 操作 |
| ------------------------ | ----------------- | ------ |
| `Ctrl+B` | `Cmd+B` | 粗体 |
| `Ctrl+I` | `Cmd+I` | 斜体 |
| `Ctrl+K` | `Cmd+K` | 插入链接 |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | 删除线 |
| `Ctrl+`` | `Cmd+`` | 行内代码 |
| `Ctrl+Shift+C` | `Cmd+Shift+C` | 代码块 |
| `Ctrl+/` | `Cmd+/` | 斜杠命令 |
| `Tab` | `Tab` | 缩进 |
| `Shift+Tab` | `Shift+Tab` | 取消缩进 |

### 窗口管理

| 快捷键 (Windows/Linux) | 快捷键 (macOS) | 操作 |
| ------------------------ | ----------------- | ------ |
| `Ctrl+W` | `Cmd+W` | 关闭当前笔记 |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | 关闭所有笔记 |
| `F11` | `F11` | 切换全屏 |
| `Ctrl+M` | `Cmd+M` | 切换侧边栏 |

## 快速捕捉

| 快捷键 (Windows/Linux) | 快捷键 (macOS) | 操作 |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | 打开快速捕捉 |
| `Escape` | `Escape` | 关闭快速捕捉 |
| `Ctrl+Enter` | `Cmd+Enter` | 保存并关闭 |

## 自定义快捷键

1. 打开**设置**（`Ctrl+,` 或 `Cmd+,`）
2. 进入**键盘快捷键**
3. 点击任意快捷键进行更改
4. 按新的组合键
5. 点击**保存**

## 提示

- 大多数快捷键都可以自定义
- 与系统快捷键冲突会高亮显示
- 点击每个快捷键旁边的重置按钮可恢复默认值
