# 快速入门指南

欢迎使用 Classic！本指南将帮助您在几分钟内快速上手。

## 系统要求

- **macOS**: macOS 12 (Monterey) 或更高版本
- **Windows**: Windows 10 或更高版本
- **Linux**: Ubuntu 20.04+ / Fedora 36+ 或类似发行版

## 安装

1. 从[官网](/)下载 Classic
2. 打开应用程序
3. 开始写作！

## 首次启动

首次打开 Classic 时，您将看到一个简洁、极简的界面，旨在让您专注于写作。

### 侧边栏

在左侧，您可以找到：
- **每日笔记** - 自动化的每日日记
- **搜索** - 快速查找所有笔记中的任何内容
- **命令面板** - 按 `Cmd+P` (macOS) 或 `Ctrl+P` (Windows/Linux) 打开命令面板
- **大纲** - 以树状结构浏览和组织您的笔记

### 编辑器

中间区域是 **Markdown 编辑器**，具有：
- 实时预览
- 语法高亮
- 丰富的格式化工具

### 创建您的第一个笔记

1. 点击侧边栏中的 **新建笔记** 按钮，或按 `Cmd+N` (macOS) 或 `Ctrl+N` (Windows/Linux)
2. 为笔记命名
3. 开始用 Markdown 写作！

```markdown
# 我的第一篇笔记

欢迎使用 Classic！这里是我记录想法和灵感的地方。
```
