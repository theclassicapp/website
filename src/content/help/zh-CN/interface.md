# 了解界面

Classic 的界面设计简洁、直观，不会干扰您的工作。以下是主要组件的介绍。

## 主布局

Classic 采用三栏布局：

1. **侧边栏**（左侧）- 导航和笔记组织
2. **编辑器**（中间）- 您的写作空间
3. **预览/属性**（右侧）- 可选面板

## 侧边栏

侧边栏包含几个关键部分：

### 笔记列表

- **所有笔记**：在一个列表中查看所有笔记
- **每日笔记**：快速访问今天的日记
- **收藏**：收藏常用的笔记
- **文件夹**：在层次结构中组织笔记

### 导航

- **搜索栏**：即时查找笔记
- **标签**：按标签浏览
- **反向链接**：查看笔记之间的连接

### 操作

- **新建笔记**：创建新笔记
- **新建文件夹**：用文件夹组织
- **导入**：导入外部内容

## 编辑器

编辑器是您编写和格式化笔记的地方。

### 功能

- **Markdown 支持**：完整的 Markdown 语法
- **实时预览**：输入时查看格式化文本
- **语法高亮**：代码块高亮显示
- **斜杠命令**：输入 `/` 快速执行操作

### 编辑器模式

| 模式 | 描述 |
|------|------|
| **仅编辑** | 查看原始 Markdown |
| **仅预览** | 查看渲染输出 |
| **分屏视图** | 并排编辑和预览 |

### 工具栏

快速格式化操作：

- 粗体、斜体、删除线
- 标题（H1-H6）
- 列表（无序、有序、任务）
- 链接和图片
- 代码块
- 表格

## 状态栏

在编辑器底部：

- **字数/字符数**：跟踪您的写作
- **上次保存**：笔记上次保存的时间
- **笔记路径**：笔记所在位置
- **同步状态**：如果使用可选同步

## 上下文菜单

右键单击获取上下文操作：

### 笔记上下文菜单

- 重命名
- 移动到文件夹
- 复制
- 删除
- 导出
- 复制链接

### 编辑器上下文菜单

- 剪切、复制、粘贴
- 格式选项
- 插入链接/图片
- 查找和替换

## 键盘导航

使用键盘导航界面：

| 操作 | 快捷键 |
|------|--------|
| 切换侧边栏 | `Cmd/Ctrl + M` |
| 聚焦搜索 | `Cmd/Ctrl + F` |
| 命令面板 | `Cmd/Ctrl + P` |
| 新建笔记 | `Cmd/Ctrl + N` |

## 自定义

根据您的喜好调整界面：

### 侧边栏宽度

1. 将鼠标悬停在侧边栏边缘
2. 拖动调整大小
3. 或在设置中设置精确宽度

### 主题

在浅色和深色模式之间切换：

- 使用顶部的主题切换按钮
- 或设置为跟随系统偏好

### 字体大小

1. 打开设置
2. 转到外观
3. 调整编辑器字体大小

## 技巧

### 专注模式

隐藏所有面板进行无干扰写作：

1. 打开命令面板
2. 输入"专注模式"
3. 按 Enter

### 快速访问

- 将常用笔记固定到收藏
- 使用键盘快捷键执行常见操作
- 在设置中自定义工具栏
