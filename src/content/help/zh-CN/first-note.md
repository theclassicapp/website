# 创建您的第一篇笔记

开始在 Classic 中创建您的第一篇笔记。简单直观。

## 创建新笔记

### 使用侧边栏

1. 点击侧边栏中的 **+** 按钮
2. 创建新笔记
3. 开始输入内容

### 使用键盘快捷键

| 平台 | 快捷键 |
|------|--------|
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### 使用命令面板

1. 打开命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 输入"新建笔记"
3. 按 Enter

## 编写笔记

### 基本编辑

直接开始输入！Classic 会在您编写时自动保存。

### 添加标题

笔记的第一行成为标题。使用标题以获得最佳效果：

```markdown
# 我的第一篇笔记

这是笔记的内容。
```

### 格式化文本

使用 Markdown 格式化文本：

- **粗体**：`**文本**` 或 `Cmd/Ctrl+B`
- **斜体**：`*文本*` 或 `Cmd/Ctrl+I`
- **删除线**：`~~文本~~`
- **代码**：`` `代码` ``

## 保存

Classic 会在您输入时自动保存笔记。无需手动保存。

### 手动保存

如果需要，可以强制保存：

- **macOS**：`Cmd+S`
- **Windows/Linux**：`Ctrl+S`

## 组织笔记

### 添加标签

为笔记添加标签以便日后查找：

1. 输入 `#` 后跟标签名称
2. 或使用工具栏中的标签按钮
3. 支持多个标签

示例：

```markdown
# 会议记录

讨论第四季度规划 #工作 #会议
```

### 移动到文件夹

1. 在侧边栏中右键点击笔记
2. 选择"移动到文件夹"
3. 选择或创建目标文件夹

### 固定重要笔记

1. 右键点击笔记
2. 选择"固定到顶部"
3. 笔记显示在列表顶部

## 预览笔记

查看格式化笔记的效果：

- 点击 **预览** 按钮
- 或使用 `Cmd/Ctrl+Shift+P`
- 在编辑和预览模式之间切换

## 下一步

现在您已经创建了第一篇笔记：

- 了解更多关于 [Markdown](/help/markdown) 的内容
- 用 [标签](/help/tags) 组织
- 设置 [每日笔记](/help/daily-notes)
- 探索 [键盘快捷键](/help/keyboard-shortcuts)

## 技巧

### 笔记标题

- 保持标题描述性
- 第一行 = 笔记标题
- 使用 H1（`#`）以获得清晰度

### 自动保存

- 永远不用担心丢失工作
- 更改自动保存
- 检查状态栏查看保存状态

### 快速捕获

需要快速记录？

- 使用快速捕获（`Cmd/Ctrl+Shift+N`）
- 立即创建笔记
- 非常适合记录稍纵即逝的想法
