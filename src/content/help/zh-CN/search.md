# 搜索

使用 Classic 强大的搜索功能即时找到任何笔记。全文搜索、筛选器和操作符帮助你准确定位所需内容。

## 基本搜索

### 打开搜索

| 平台 | 快捷键 |
|----------|----------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

或点击侧边栏中的搜索栏。

### 工作原理

1. 打开搜索
2. 开始输入
3. 结果即时显示
4. 点击打开笔记

## 搜索功能

### 全文搜索

Classic 搜索以下内容

- 笔记标题
- 笔记内容
- 标签
- 文件名

### 实时结果

输入时结果实时更新

- 无需按 Enter
- 即时反馈
- 启用模糊匹配

### 搜索高亮

找到的词在结果中高亮显示

- 查看匹配周围的上下文
- 快速识别相关笔记

## 搜索操作符

### 精确匹配

使用引号搜索精确短语

```
"项目计划"
"会议笔记"
```

### 标签搜索

查找带有特定标签的笔记

```
tag:工作
tag:会议
tag:紧急
```

### 排除词

使用 `-` 排除

```
项目 -已归档
会议 -已取消
```

### OR 搜索

查找包含任意词的笔记

```
项目 OR 计划
```

### 日期范围

按日期搜索

```
created:2024-01-01..2024-12-31
modified:last-week
```

### 文件类型

按扩展名筛选

```
ext:md
ext:txt
```

## 高级搜索

### 组合操作符

构建复杂查询

```
tag:工作 -tag:已归档 created:last-week
"会议" tag:项目-阿尔法
```

### 路径搜索

在文件夹内搜索

```
path:项目/
path:归档/
```

### 标题搜索

仅搜索标题

```
title:路线图
title:会议
```

### 内容搜索

仅搜索内容（排除标题）

```
content:重要
content:截止日期
```

## 搜索筛选器

### 按标签

1. 输入搜索内容
2. 点击标签进行筛选
3. 结果缩小到带标签的笔记

### 按日期

按修改日期筛选

- 今天
- 本周
- 本月
- 自定义范围

### 按文件夹

限制搜索到特定文件夹

1. 右键点击文件夹
2. 选择"在文件夹中搜索"
3. 结果限制在该文件夹

## 搜索结果

### 排序

按以下方式排序结果

- **相关性**（默认）
- **修改日期**（最新优先）
- **创建日期**（最新优先）
- **标题**（字母顺序）

### 结果预览

悬停结果进行预览

- 显示前几行
- 匹配词高亮
- 显示标签

### 上下文

查看匹配发生的位置

- 显示周围文本
- 高亮匹配
- 点击跳转到位置

## 搜索历史

### 最近搜索

访问之前的搜索

1. 点击搜索栏
2. 查看最近搜索
3. 点击重新运行

### 清除历史

1. 打开设置
2. 进入隐私
3. 点击"清除搜索历史"

## 提示

### 使用具体词语

更具体 = 更好的结果

```
✅ "Q4 营销预算"
❌ 预算
```

### 与标签结合

快速缩小结果

```
项目 tag:紧急
```

### 使用排除

过滤噪音

```
会议 -tag:已归档
```

### 保存常用搜索

创建保存的搜索

1. 输入搜索内容
2. 点击"保存搜索"
3. 命名
4. 从侧边栏访问

## 键盘导航

| 操作 | 快捷键 |
|--------|----------|
| 打开搜索 | `Cmd/Ctrl+F` |
| 下一个结果 | `Enter` 或 `Down` |
| 上一个结果 | `Shift+Enter` 或 `Up` |
| 关闭搜索 | `Escape` |

## 搜索设置

### 在设置中自定义

| 设置 | 选项 |
|---------|---------|
| 区分大小写 | 开/关 |
| 搜索标题 | 开/关 |
| 搜索内容 | 开/关 |
| 搜索标签 | 开/关 |
| 包含已归档 | 开/关 |

## 使用案例

### 查找旧笔记

```
"会议" date:last-year
```

### 项目研究

```
path:项目/ tag:研究
```

### 行动项目

```
"待办" OR "修复" -tag:完成
```

### 最近工作

```
modified:this-week
```

## 故障排除

### 未找到结果

- 检查拼写
- 移除筛选器
- 尝试更广泛的词
- 检查笔记是否已归档

### 搜索缓慢

- 大量笔记集合可能需要更长时间
- 禁用区分大小写
- 减少搜索范围

### 缺少笔记

- 笔记可能在不同文件夹
- 检查标签是否正确
- 验证笔记未归档
