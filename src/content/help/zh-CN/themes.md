# 主题与外观

个性化 Classic 的外观以匹配你的风格并减少眼睛疲劳。

## 主题选项

### 亮色模式

Classic 的默认主题特点

- 简洁明亮的界面
- 适合光线充足的环境
- 高对比度以提高可读性

### 暗色模式

适合夜猫子和低光环境工作

- 减少眼睛疲劳
- OLED 屏幕上更省电
- 在暗色环境中对眼睛友好

### 系统（自动）

跟随操作系统的主题

- 根据 OS 设置自动切换
- 无需手动切换
- 与其他应用保持一致

## 切换主题

### 通过设置

1. 打开设置（`Cmd+,` 或 `Ctrl+,`）
2. 进入**外观**
3. 选择你偏好的主题

### 通过命令面板

1. 打开命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 输入"主题"
3. 选择你偏好的主题

### 键盘快捷键

| 平台 | 快捷键 |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

循环切换：亮色 → 暗色 → 系统

## 字体设置

### 编辑器字体

自定义写作体验

1. 打开设置
2. 进入**外观** → **编辑器**
3. 配置字体选项

| 设置 | 选项 | 默认值 |
| ------- | ------- | ------- |
| 字体族 | 系统字体 + 自定义 | System UI |
| 字体大小 | 12-24px | 16px |
| 行高 | 1.0-2.5 | 1.75 |
| 字符间距 | -2 到 4 | 0 |

### 界面字体

调整界面字体

1. 打开设置
2. 进入**外观** → **界面**
3. 选择字体和大小

### 等宽字体

用于代码块和技术内容

1. 打开设置
2. 进入**外观** → **代码**
3. 选择等宽字体

## 界面选项

### 侧边栏

自定义侧边栏行为

| 设置 | 描述 |
| ------- | ----------- |
| **自动隐藏** | 不使用时隐藏侧边栏 |
| **宽度** | 调整侧边栏宽度（200-400px） |
| **显示图标** | 切换图标可见性 |

### 编辑器布局

选择你的编辑体验

- **分栏视图**：编辑和预览并排
- **仅预览**：仅查看渲染输出
- **仅编辑**：专注于原始 Markdown

### 笔记列表

配置笔记显示方式

- **列表视图**：紧凑，每笔记一行
- **卡片视图**：带片段的预览卡片
- **树状视图**：层级文件夹结构

## 强调色

### 可用颜色

选择你的强调色

- 经典蓝（默认）
- 紫色
- 绿色
- 橙色
- 红色
- 粉色
- 青色

### 更改强调色

1. 打开设置
2. 进入**外观** → **颜色**
3. 选择你偏好的强调色

强调色影响

- 链接和按钮
- 选择高亮
- 活动元素
- 焦点指示器

## 无障碍

### 高对比度模式

为了更好的可见性

1. 打开设置
2. 进入**外观** → **无障碍**
3. 启用**高对比度**

### 减少动态

最小化动画

1. 打开设置
2. 进入**外观** → **无障碍**
3. 启用**减少动态**

### 字体调整

- 增加最小字体大小
- 使用阅读障碍友好字体
- 调整对比度比率

## 自定义 CSS（高级）

为想要完全控制的高级用户

1. 打开设置
2. 进入**外观** → **高级**
3. 点击**打开自定义 CSS**
4. 添加你的自定义样式

### 自定义示例

```css
/* 更大的标题 */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* 自定义链接颜色 */
a { color: #3b82f6; }

/* 更宽的行距 */
.prose p { line-height: 2; }
```

## 主题同步

当你登录（可选）时，主题首选项在设备间同步

- 自动主题同步
- 字体设置同步
- 自定义 CSS 同步

## 提示

### 匹配你的工作流程

- 白天使用**亮色**，晚上使用**暗色**
- **系统**模式自动处理
- 考虑你的环境光线

### 减少眼睛疲劳

- 在低光环境中使用暗色模式
- 长时间会话时增加字体大小
- 如需要启用高对比度
- 定期休息

### 提高生产力

- 学习主题切换快捷键
- 设置自动切换时间
- 使用系统模式保持一致性
