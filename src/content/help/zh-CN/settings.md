# 设置

自定义 Classic 完全按照你想要的方式工作。从外观到编辑器行为，一切都是可配置的。

## 打开设置

### 键盘快捷键

| 平台 | 快捷键 |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### 通过菜单

- **macOS**: Classic → 设置
- **Windows**: 文件 → 设置
- **Linux**: 编辑 → 设置

### 通过命令面板

1. 打开命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 输入"设置"
3. 按 Enter

## 设置类别

### 通用

基本应用程序设置

| 设置 | 描述 |
|---------|-------------|
| **语言** | 界面语言 |
| **主题** | 亮色、暗色或系统 |
| **登录时启动** | 启动时运行 |
| **检查更新** | 自动更新行为 |

### 外观

自定义外观

| 设置 | 选项 |
|---------|---------|
| **主题** | 亮色 / 暗色 / 系统 |
| **强调色** | 蓝色、紫色、绿色等 |
| **编辑器字体** | 字体族和大小 |
| **界面字体** | 界面字体 |
| **等宽字体** | 代码字体 |
| **侧边栏宽度** | 200-400px |
| **行高** | 1.0-2.5 |

### 编辑器

配置写作行为

| 设置 | 描述 |
|---------|-------------|
| **拼写检查** | 启用/禁用拼写检查 |
| **自动保存** | 自动保存间隔 |
| **自动换行** | 启用/禁用自动换行 |
| **行号** | 显示/隐藏行号 |
| **制表符大小** | 2、4 或 8 个空格 |
| **自动更正** | 智能引号、自动替换 |

### 版本历史

控制版本追踪

| 设置 | 选项 | 默认值 |
|---------|---------|---------|
| **启用历史** | 开/关 | 开 |
| **保留期限** | 30-180 天 | 90 天 |
| **最大版本数** | 50-500 | 100 |

### 备份与恢复

数据保护设置

| 设置 | 描述 |
|---------|-------------|
| **自动备份** | 启用自动备份 |
| **备份频率** | 15分钟 / 30分钟 / 1小时 / 4小时 |
| **备份位置** | 自定义路径 |
| **保留备份数** | 5 / 10 / 20 / 50 |
| **加密备份** | 密码保护 |

### 键盘快捷键

查看和自定义快捷键

- 查看所有可用快捷键
- 自定义任意快捷键
- 重置为默认值
- 导入/导出快捷键

### 导入与导出

数据迁移设置

- 从其他应用导入
- 导出格式选项
- 批量操作

### 隐私

隐私和安全设置

| 设置 | 描述 |
|---------|-------------|
| **分析** | 启用/禁用使用分析 |
| **崩溃报告** | 发送崩溃报告 |
| **清除历史** | 清除搜索/历史 |

### 通知

控制通知

| 设置 | 描述 |
|---------|-------------|
| **启用通知** | 开/关 |
| **日记提醒** | 提醒时间 |
| **声音** | 通知声音 |

### 高级

高级用户设置

| 设置 | 描述 |
|---------|-------------|
| **自定义 CSS** | 添加自定义样式 |
| **调试模式** | 启用调试日志 |
| **实验性功能** | Beta 功能 |
| **数据位置** | 更改笔记文件夹 |

## 搜索设置

找不到设置？使用搜索栏

1. 打开设置
2. 在搜索框中输入
3. 显示匹配的设置
4. 点击跳转到该设置

## 重置设置

### 重置所有设置

恢复默认值

1. 打开设置
2. 滚动到底部
3. 点击"重置所有设置"
4. 确认操作

### 重置类别

重置特定类别

1. 进入该类别
2. 点击"重置为默认值"
3. 仅该类别重置

## 导出/导入设置

### 导出设置

保存你的配置

1. 打开设置
2. 点击"导出设置"
3. 选择位置
4. 保存为 JSON 文件

### 导入设置

加载已保存的配置

1. 打开设置
2. 点击"导入设置"
3. 选择 JSON 文件
4. 确认导入

## 设置位置

设置存储在本地

| 平台 | 位置 |
|----------|----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## 同步设置（可选）

登录后，设置可以在设备间同步

- **同步**：在设置 → 同步中启用
- **同步内容**：主题、字体、快捷键
- **不同步内容**：本地路径、加密密钥

## 常见自定义

### 增大字体

1. 设置 → 外观
2. 编辑器字体大小 → 增大
3. 更改立即应用

### 更改主题

1. 设置 → 外观
2. 主题 → 选择亮色/暗色/系统
3. 应用立即更新

### 启用自动备份

1. 设置 → 备份与恢复
2. 启用"自动备份"
3. 设置频率和位置

### 自定义快捷键

1. 设置 → 键盘快捷键
2. 找到操作
3. 点击当前快捷键
4. 按新键
5. 保存

## 提示

### 快速访问

- 使用 `Cmd/Ctrl+,` 快速打开设置
- 按名称搜索设置
- 使用类别导航

### 更改前导出

在进行重大更改前，导出你的设置

- 如果出现问题易于恢复
- 与他人分享配置

### 检查新设置

更新可能会添加新设置

- 更新后检查
- 探索新选项

## 故障排除

### 设置不保存

- 检查文件权限
- 验证磁盘空间
- 尝试重置并重新配置

### 设置意外重置

- 可能表示文件损坏
- 定期导出设置
- 检查应用冲突

### 找不到设置

- 使用搜索栏
- 检查不同类别
- 可能在高级设置中
