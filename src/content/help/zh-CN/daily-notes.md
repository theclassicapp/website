# 日记

日记会为每一天自动创建新笔记，非常适合写日志、每日计划或记录基于时间的信息。

## 工作原理

### 自动创建

- 每天创建一个新笔记
- 笔记使用日期格式命名（如 `2024-01-15.md`）
- 以前日期的笔记会被保留并可搜索

### 访问日记

1. 点击侧边栏中的 **日记**
2. 或按 `Ctrl/Cmd+D`
3. 今天的笔记自动打开

## 日记模板

自定义新日记的模板

### 默认模板

```markdown
# {{date}}

## 任务
- [ ]
- [ ]

## 笔记


## 日志

```

### 自定义

1. 打开设置
2. 进入日记
3. 编辑模板
4. 使用 `{{date}}` 表示当前日期

### 模板变量

| 变量 | 描述 |
| -------- | ----------- |
| `{{date}}` | 当前日期 (YYYY-MM-DD) |
| `{{time}}` | 当前时间 (HH:mm) |
| `{{weekday}}` | 星期名称（星期一等） |

## 功能

### 日期导航

- **上一天/下一天**：在日期之间导航
- **日历**：跳转到特定日期
- **搜索**：在所有日记中查找内容

### 自动链接

- 同一天创建的笔记自动链接
- 反向链接显示日期之间的连接
- 标签在所有日记中有效

### 集成

- **快速捕捉**：用 `#今天` 添加到今天的笔记
- **日历视图**：日记的可视化概览
- **统计**：追踪你的写作连续天数

## 提示

### 一致性

- 每天在同一时间创建笔记
- 使用一致的格式
- 全天审查和更新

### 组织

- 使用标题分隔不同部分
- 添加标签便于搜索
- 链接到相关笔记

### 审查

- 每周审查日记
- 将重要项目提取到永久笔记
- 归档或总结旧笔记
