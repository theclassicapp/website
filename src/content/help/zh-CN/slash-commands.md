# 斜杠命令

在 Classic 中使用斜杠命令快速执行操作。输入 `/` 即可开始。

## 使用斜杠命令

1. 在笔记中的任意位置输入 `/`
2. 命令菜单会自动出现
3. 输入以筛选命令
4. 使用箭头键导航
5. 按 Enter 执行

## 文本格式

| 命令 | 结果 |
|------|------|
| `/bold` | **粗体** 文本 |
| `/italic` | *斜体* 文本 |
| `/strikethrough` | ~~删除线~~ 文本 |
| `/code` | `行内代码` |
| `/highlight` | ==高亮== 文本 |
| `/comment` | 添加评论 |

## 技巧

### 快速筛选

输入 `/` 后紧跟关键词：

- `/b` → 快速访问粗体
- `/li` → 快速访问链接
- `/cb` → 快速访问代码块

### 键盘导航

- `↑`/`↓` - 导航命令
- `Enter` - 选择命令
- `Escape` - 关闭菜单
- `Tab` - 在类别间切换

### 最近使用

最近使用的命令显示在顶部以便快速访问。
