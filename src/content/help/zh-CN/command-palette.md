# 命令面板

命令面板是你访问 Classic 中每个操作的入口。按一个简单的键盘快捷键即可即时访问任何功能。

## 打开命令面板

| 平台 | 快捷键 |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## 如何使用

1. 打开命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 输入搜索命令
3. 使用方向键导航
4. 按 `Enter` 执行

## 命令类别

### 文件操作

- **新建笔记**：创建新笔记
- **新建日记**：创建/打开今天的日记
- **打开笔记**：搜索并打开现有笔记
- **保存笔记**：保存当前笔记
- **删除笔记**：删除当前笔记
- **重命名笔记**：重命名当前笔记

### 导航

- **跳转到笔记**：跳转到任意笔记
- **跳转到日记**：打开特定日期
- **跳转到标签**：按标签浏览笔记
- **后退**：在历史中后退
- **前进**：在历史中前进

### 编辑器

- **切换预览**：在编辑和预览模式间切换
- **切换侧边栏**：显示/隐藏侧边栏
- **专注模式**：进入无干扰写作
- **插入日期**：添加当前日期/时间
- **插入模板**：插入模板

### 格式化

- **粗体**：将文本加粗
- **斜体**：将文本倾斜
- **删除线**：给文本添加删除线
- **代码**：格式化为代码
- **链接**：插入链接
- **标题**：应用标题格式

### 组织

- **添加标签**：为当前笔记添加标签
- **移动到文件夹**：将笔记移动到文件夹
- **归档笔记**：归档当前笔记
- **置顶笔记**：将笔记置顶到列表顶部

### 视图

- **切换主题**：在亮色/暗色间切换
- **放大**：增大字体大小
- **缩小**：减小字体大小
- **重置缩放**：重置为默认大小

## 提示

### 快速访问

- 命令支持模糊搜索
- 最近使用的命令优先显示
- 可置顶常用命令

### 键盘导航

- `↑`/`↓` - 导航结果
- `Enter` - 执行命令
- `Escape` - 关闭面板
- `Tab` - 在类别间切换

### 自定义命令

1. 打开设置
2. 进入命令面板
3. 添加自定义命令
4. 定义键盘快捷键

## 常用命令

以下是你经常会用到的命令

| 命令 | 作用 |
| ------- | ------------- |
| `新建笔记` | 创建空白笔记 |
| `打开笔记` | 搜索所有笔记 |
| `新建日记` | 今天的日志 |
| `切换预览` | 预览 vs 编辑 |
| `添加标签` | 为当前笔记添加标签 |
| `快速捕捉` | 全局捕捉 |

## 自定义命令

### 更改快捷键

1. 打开设置（`Cmd+,` 或 `Ctrl+,`）
2. 进入键盘快捷键
3. 找到命令
4. 点击当前快捷键
5. 按新的快捷键
6. 保存

### 创建自定义命令

高级用户可以创建自定义命令

1. 打开设置
2. 进入命令
3. 点击"添加自定义命令"
4. 定义命令名称
5. 添加操作
6. 设置快捷键（可选）
