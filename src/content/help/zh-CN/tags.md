# 标签

标签是在 Classic 中组织和分类笔记的强大方式。即时找到相关内容。

## 创建标签

### 行内标签

直接在笔记内容中添加标签

```markdown
# 会议笔记

讨论了 Q4 路线图 #工作 #会议 #计划
```

### 使用标签按钮

1. 点击工具栏中的标签图标
2. 输入标签名称
3. 按 Enter 键

### 键盘快捷键

| 平台 | 快捷键 |
|----------|----------|
| **macOS** | `Cmd+T` |
| **Windows/Linux** | `Ctrl+T` |

## 标签语法

### 基本标签

```markdown
#项目 #工作 #重要
```

### 嵌套标签

使用 `/` 创建标签层级

```markdown
#工作/会议
#工作/项目
#个人/爱好/阅读
```

### 多词标签

使用连字符或下划线

```markdown
#项目-阿尔法
#会议_笔记
#Q4-计划
```

## 管理标签

### 查看所有标签

1. 打开侧边栏
2. 找到"标签"部分
3. 查看所有标签

### 标签云

可视化最常用的标签

- 更大的标签 = 更多笔记
- 点击任意标签进行筛选
- 查看每个标签的笔记数量

### 重命名标签

1. 右键点击标签
2. 选择"重命名"
3. 输入新名称
4. 所有笔记自动更新

### 删除标签

1. 右键点击标签
2. 选择"删除"
3. 确认删除
4. 标签从所有笔记中移除

## 按标签搜索

### 标签面板

点击侧边栏中的标签来筛选笔记

### 搜索语法

在搜索中使用 `tag:`

```
tag:工作
tag:工作/会议
```

### 组合标签

查找包含多个标签的笔记

```
tag:工作 tag:紧急
```

### 排除标签

查找没有某个标签的笔记

```
-tag:已归档
```

## 标签最佳实践

### 保持简洁

- 每个笔记使用 3-5 个标签
- 命名保持一致
- 避免标签泛滥

### 使用层级

组织相关标签

```
#工作
  #工作/会议
  #工作/项目
  #工作/截止日期

#个人
  #个人/健康
  #个人/财务
```

### 建立规范

为自己定义标签标准

- **状态**: `#待办` `#进行中` `#完成`
- **优先级**: `#紧急` `#高` `#低`
- **类型**: `#会议` `#参考` `#想法`

## 标签建议

当你输入 `#` 时，Classic 会建议现有标签

- 最近使用的标签优先显示
- 支持模糊匹配
- 按 Tab 键接受建议

## 标签统计

在设置中查看标签使用情况

- 最常用的标签
- 未使用的标签
- 标签随时间的变化趋势

| 操作 | 快捷键 |
|--------|----------|

| 搜索标签 | `Cmd/Ctrl+F`，然后输入 `tag:` |

## 提示

### 命名一致

坚持一个规范

```
✅ #会议-笔记
✅ #会议_笔记
❌ #会议 笔记 (不允许空格)
❌ #会议笔记 (不一致)
```

### 避免过度标签

太多标签会变得难以管理

```
✅ #工作 #紧急 #项目-阿尔法
❌ #工作 #紧急 #重要 #今天 #周一 #办公室 #桌子 #电脑
```

### 定期清理

定期审查和整合标签

- 合并相似的标签
- 删除未使用的标签
- 更新规范

## 高级功能

### 基于标签的智能文件夹

创建根据标签自动填充的文件夹

1. 创建新文件夹
2. 设置筛选器：`tag:项目`
3. 所有带标签的笔记都会显示

### YAML 中的标签

在笔记前置信息中添加标签

```yaml
---
tags:
  - 工作
  - 会议
  - 项目-阿尔法
---
```

### 标签别名

在设置中设置标签别名

- `#待办` = `#任务` = `#行动`
- 整合相似的标签
