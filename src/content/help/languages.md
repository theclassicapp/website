## Changing the Interface Language

### Via Settings

1. Open Settings (`Cmd+,` or `Ctrl+,`)
2. Go to **General** → **Language**
3. Select your preferred language
4. Restart Classic to apply

### Via Command Palette

1. Open Command Palette (`Cmd+P` or `Ctrl+P`)
2. Type "Language"
3. Select your preferred language
4. Restart to apply changes

## Writing in Multiple Languages

Classic supports writing notes in any language

### Unicode Support

- Full Unicode support
- RTL (Right-to-Left) languages supported
- CJK (Chinese, Japanese, Korean) characters work perfectly
- Emoji support

### Language Detection

Classic can detect the language of your notes

- Auto-detect for spell checking
- Per-note language setting
- Mixed language support

## Spell Checking

### Enabling Spell Check

1. Open Settings
2. Go to **Editor** → **Spell Check**
3. Enable "Spell Check"

### Spell Check Languages

1. Open Settings
2. Go to **Editor** → **Spell Check**
3. Click "Languages"
4. Select languages to check

### Adding Words to Dictionary

1. Right-click a flagged word
2. Select "Add to Dictionary"
3. Word won't be flagged again

## RTL Support

For Arabic, Hebrew, and other RTL languages

### Automatic RTL Detection

Classic detects RTL content automatically

- Text direction adjusts
- Cursor movement adapts
- Alignment changes

### Manual RTL Toggle

Force RTL or LTR direction

1. Open Command Palette
2. Type "RTL" or "LTR"
3. Select your preference

## Date & Time Formats

Dates adapt to your locale

### Format Options

| Setting | Options |
|---------|---------|
| Date Format | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| Time Format | 12-hour, 24-hour |
| First Day of Week | Sunday, Monday, Saturday |

Configure in Settings → General → Date & Time

## Number Formats

Numbers display according to your locale

- **Decimal separator**: `.` or `,`
- **Thousands separator**: `,` or `.` or space
- **Currency symbols**: Based on region

## Keyboard Layouts

Classic works with any keyboard layout

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Custom layouts

## Localization Settings

### In Settings → General → Language

| Setting | Description |
|---------|-------------|
| **Interface Language** | App UI language |
| **Date Format** | How dates appear |
| **Time Format** | 12 or 24 hour |
| **First Day of Week** | Calendar start day |
| **Number Format** | Decimal/thousand separators |

## Tips

### Mixed Language Notes

Write in multiple languages in one note

```markdown
# International Project

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Language-Specific Fonts

Some languages look better with specific fonts

1. Open Settings
2. Go to Appearance → Fonts
3. Set fonts for different scripts

### Switching Languages Quickly

Create a keyboard shortcut

1. Open Settings → Shortcuts
2. Find "Switch Language"
3. Set your preferred shortcut

## Troubleshooting

### Language Not Available

- Check for app updates
- Language may be in development
- Contribute translations on GitHub

### Characters Not Displaying

- Install required fonts
- Check font settings
- Try a different font

### Spell Check Not Working

- Verify spell check is enabled
- Add the language to spell check
- Restart Classic

### RTL Not Working Correctly

- Ensure RTL detection is on
- Try manual RTL toggle
- Check for mixed content issues
