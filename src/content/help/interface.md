# Understanding the Interface

Classic's interface is designed to be clean, intuitive, and get out of your way. Here's a tour of the main components.

## Main Layout

Classic has a three-panel layout

1. **Sidebar** (left) - Navigation and note organization
2. **Editor** (center) - Your writing space
3. **Preview/Properties** (right) - Optional panels

## Sidebar

The sidebar contains several key sections

### Note List

- **All Notes**: View all your notes in one list
- **Daily Notes**: Quick access to today's journal
- **Favorites**: Star frequently used notes
- **Folders**: Organize notes in hierarchies

### Navigation

- **Search Bar**: Find notes instantly
- **Tags**: Browse by tag
- **Backlinks**: See connections between notes

### Actions

- **New Note**: Create a new note
- **New Folder**: Organize with folders
- **Import**: Bring in external content

## Editor

The editor is where you write and format your notes.

### Features

- **Markdown Support**: Full Markdown syntax
- **Live Preview**: See formatted text as you type
- **Syntax Highlighting**: Code blocks are highlighted
- **Slash Commands**: Type `/` for quick actions

### Editor Modes

| Mode | Description |
|------|-------------|
| **Edit Only** | See raw Markdown |
| **Preview Only** | See rendered output |
| **Split View** | Edit and preview side by side |

### Toolbar

Quick formatting actions

- Bold, Italic, Strikethrough
- Headings (H1-H6)
- Lists (bullet, numbered, task)
- Links and images
- Code blocks
- Tables

## Status Bar

At the bottom of the editor

- **Word/Character Count**: Track your writing
- **Last Saved**: When the note was last saved
- **Note Path**: Where the note is located
- **Sync Status**: If using optional sync

## Context Menus

Right-click for contextual actions

### Note Context Menu

- Rename
- Move to folder
- Duplicate
- Delete
- Export
- Copy link

### Editor Context Menu

- Cut, Copy, Paste
- Format options
- Insert link/image
- Find and replace

## Keyboard Navigation

Navigate the interface with your keyboard

| Action | Shortcut |
|--------|----------|
| Toggle Sidebar | `Cmd/Ctrl + M` |
| Focus Search | `Cmd/Ctrl + F` |
| Command Palette | `Cmd/Ctrl + P` |
| New Note | `Cmd/Ctrl + N` |

## Customization

Adjust the interface to your liking

### Sidebar Width

1. Hover over the sidebar edge
2. Drag to resize
3. Or set exact width in Settings

### Theme

Switch between Light and Dark mode

- Use the theme toggle in the header
- Or set to follow your system preference

### Font Size

1. Open Settings
2. Go to Appearance
3. Adjust Editor Font Size

## Tips

### Focus Mode

Hide all panels for distraction-free writing

1. Open Command Palette
2. Type "Focus Mode"
3. Press Enter

### Quick Access

- Pin frequently used notes to Favorites
- Use keyboard shortcuts for common actions
- Customize the toolbar in Settings
