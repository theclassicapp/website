يتحدث كلاسيك لغتك. استخدم التطبيق بلغتك المفضلة واكتب ملاحظات بأي لغة تختارها.

## تغيير لغة الواجهة
### عبر الإعدادات
1. افتح الإعدادات (`Cmd+,` أو `Ctrl+,`)
2. انتقل إلى **عام** → **اللغة**
3. حدد لغتك المفضلة
4. أعد تشغيل كلاسيك للتطبيق
### عبر لوحة الأوامر
1. افتح لوحة الأوامر (`Cmd+P` أو `Ctrl+P`)
2. اكتب "Language"
3. حدد لغتك المفضلة
4. أعد التشغيل لتطبيق التغييرات
## الكتابة بلغات متعددة
يدعم كلاسيك الكتابة بأي لغة
### دعم Unicode
- دعم Unicode كامل
- دعم لغات RTL (من اليمين إلى اليسار)
- دعم أحرف CJK (الصينية واليابانية والكوريةية)
- دعم الرموز التعبيريةية
### كشف اللغة
يمكن لكلاسيك اكتشاف لغة ملاحظاتك
- الكشف التلقائي للمدق الإملائي
- إعداد لغة لكل ملاحظة
- دعم اللغات المختلطة
## التدقيق الإملائي
### تفعيل التدقيق الإملائي
1. افتح الإعدادات
2. انتقل إلى **المحرر** → **التدقيق الإملائي**
3. فعّل "التدقيق الإملائي"
### لغات التدقيق الإملائي
1. افتح الإعدادات
2. انتقل إلى **المحرر** → **التدقيق الإملائي**
3. انقر فوق "Languages"
4. حدد اللغات للتحقق
### إضافة كلمات إلى القاموس
1. انقر بزر الفأرة الأيمن على كلمة المميز
2. لن يتم تمييز الكلمة مرة أخرى
## دعم RTL
للعربية والعبرية ولغات RTL الأخرى
### الكشف التلقائي عن RTL
يكشف كلاسيك محتوى RTL تلقائياً
- يضبط اتجاه النص
- تتكيف حركة المؤشر
- تغيير المحاذا
### تبديل RTL اليدوي
فرض RTL أو LTR
1. افتح لوحة الأوامر
2. اكتب "RTL" أو "LTR"
3. حدد تفضيلك
## طرق الإدخال
يعمل كلاسيك مع طرق إدخال نظامك
### طرق إدخال CJK
- يعمل مع Pinyin و Zhuyin و إلخ
- يدعم IME الياباني
- دعم طرق إدخال الكورية
### أحرف خاصة
أدرج أحرف خاصة بسهولة
- استخدم منتقي أحرف النظام
- نسخ/لصق من خرائط الأحرف
- استخدام إدخال Unicode
## تنسيقات التاريخ والوقت
تتكيف التاريخ مع موقعك
| الإعداد | الخيارات |
| --------- | --------- |
| تنسيق التاريخ | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| تنسيق الوقت | 12 ساعة، 24 ساعة |
| اليوم الأول من الأسبوع | الأحد، السبت،، السبت |
 تكوين في الإعدادات ← عام ← التاريخ والوقت
## تنسيقات الأرقام
عرض الأرقام حسب موقعك
- **الفاصلة العشرية**: `.` أو `,`
- **فاصلة الآلاف**: `,` أو مسافة
- **رموز العملة**: حسب المنطقة
## تخطيطات لوحة المفاتيح
يعمل كلاسيك مع أي تخطيط لوحة المفاتيح
- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- تخطيطات مخصصة
## إعدادات الترجمة
### في الإعدادات ← عام ← اللغة
| الإعداد | الوصف |
| --------- | ----------- |
| **لغة الواجهة** | لغة واجهة المستخدم |
| **تنسيق التاريخ** | كيف تظهر التواريخ |
| **تنسيق الوقت** | 12 أو 24 ساعة |
| **اليوم الأول من الأسبوع** | بداية التقويم |
| **تنسيق الأرقام** | فواصل عشرية/فواصل الآلاف |
## نصائح
### ملاحظات متعددة اللغات
```markdown
# مشروع دولثال
English: مرحبًا بك المشروع
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```
### خطوط الخاصةةبلغات
بعض اللغات تبدو أفضل بخطوط محددة
1. افتح الإعدادات
2. انتقل إلى المظهر ← الخطوط
3. عيّن خطوط للنصوص المختلفة
### التبديل السريع بين اللغات
1. افتح الإعدادات ← الاختصارات
2. ابحث عن "تبديل اللغة"
3. عيّن اختصارك المفضل
## استكشاف الأخطاء
### اللغة غير متوفرة
- تحقق من وجود تحديثات للتطبيق
- قد تكون اللغة قيد التطوير
- ساهم في الترجمة على GitHub
### عدم عرض الأحرف
- تحقق من إعدادات الخط
- جرب خطا مختلف
- تحقق من عدم عمل التدقيق الإمللي
- أعد تشغيل كلاسيك
### RTL لا يعمل بشكل صحيح
- تأكد من تفعيل الكشف التلقائي عن RTL
- جرب التبديل اليدوي RTL
- تحقق من المحتوى المختلط
## المساهمة في الترجمة
ساعد في ترجمة كلاسيك
1. زر مجتمع كلاسيك
2. انضم إلى فريق الترجمة
3. قدم بالترجمات

4. ساعد في تحسين الترجمة
