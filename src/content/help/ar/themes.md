# السمات والمظهر
خصص مظهر كلاسيك ليتناسب أسلوبك وتقليل إجهاد العين.

## خيارات السمات
### الوضع الفاتح
يتميز السمة الافتراضية في كلاسيك بـ:
- واجهة نظيفة ومشرقة
- مثالي للبيئات جيدة الإضاءة
- تباين عالي للقراءة
### الوضع الداكن
مثالي لطيور الليل والعمل في الإضاءة المنخفضة
- يقلل إجهاد العين
- استهلاك أقل للبطارية على شاشات OLED
- مريح للعين في البيئات الداكنة
### النظام (تلقائي)
يتبع سمة نظام التشغيل
- يتبديل تلقائياً بناءً على إعدادات OS
- لا حاجة للتبديل اليدوي
- متسق مع التطبيقات الأخرى
## تبديل السمات
### عبر الإعدادات
1. افتح الإعدادات (`Cmd+,` أو `Ctrl+,`)
2. انتقل إلى **المظهر**
3. حدد سمتك المفضلة
### عبر لوحة الأوامر
1. افتح لوحة الأوامر (`Cmd+P` أو `Ctrl+P`)
2. اكتب "Theme"
3. حدد سمتك المفضلة
### اختصار لوحة المفاتيح
| platform | shortcut |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |
يت من خلال: فاتح → داكن → النظام
## إعدادات الخط
### خط المحرر
خصص تجربة الكتابة
1. افتح الإعدادات
2. انتقل إلى **المظهر** → **المحرر**
3. قم بتكوين خيارات الخط
| الإعداد | الخيارات | الافتراضي |
| ------- | ------- | ------- |
| عائلة الخط | خطوط النظام + مخصص | واجهة المستخدم |
| حجم الخط | 12-24px | 16px |
| ارتفاع السطر | 1.0-2.5 | 1.75 |
| تباعد الحروف | -2 إلى 4 | 0 |
### خط واجهة المستخدم
اضبط خط الواجهة
1. افتح الإعدادات
2. انتقل إلى **المظهر** → **الواجهة**
3. اختر الخط وحجمه
### خط ثابت العرض
لكتل الكود والمحتوى الفني
1. افتح الإعدادات
2. انتقل إلى **المظهر** → **الكود**
3. حدد خط ثابت العرض
## خيارات الواجهة
### الشريط الجانبي
خصص سلوك الشريط الجانبي
| الإعداد | الوصف |
| ------- | ----------- |
| **الإخفاء التلقائي** | إخفاء الشريط الجانبي عند عدم الاستخدام |
| **العرض** | ضبط عرض الشريط الجانبي (200-400px) |
| **إظهار الأيقونات** | تبديل رؤية الأيقونات |
### تخطيط المحرر
اختر تجربة التحرير
- **العرض المقسم**: تحرير ومعاينة جنباً إلى جنب
- **المعاينة فقط**: شاهد المخرجات المنسقة فقط
- **التحرير فقط**: ركز على Markdown الخام
### قائمة الملاحظات
تكوين كيفية ظهور الملاحظات
- **عرض القائمة**: مدمج، سطر واحد لكل ملاحظة
- **عرض البطاقات**: بطاقات معاينة مع مقتطفات
- **عرض الشجرة**: هيكل مجلدات هرمي
## ألوان التمييز
### الألوان المتاحة
اختر لون التمييز
- الأزرق الكلاسيكي (افتراضي)
- البنفسجي
- الأخضر
- البرتقالي
- الأحمر
- الوردي
- التركوازي
### تغيير لون التمييز
1. افتح الإعدادات
2. انتقل إلى **المظهر** → **الألوان**
3. حدد التمييز المفضل
يؤثر لون التمييز على
- الروابط والأزرار
- تمييز التحديد
- العناصر النشطة
- مؤشرات التركيز
## إمكانية الوصول
### وضع التباين العالي
لرؤية أفضل
1. افتح الإعدادات
2. انتقل إلى **المظهر** → **إمكانية الوصول**
3. فعّل **التباين العالي**
### الحركة المقللة
قلل الحركات
1. افتح الإعدادات
2. انتقل إلى **المظهر** → **إمكانية الوصول**
3. فعّل **الحركة المقللة**
### ضبط الخط
- زيادة الحد الأدنى لحجم الخط
- استخدام خطوط صديقة لخلل القراءة
- ضبط نسب التباين
## CSS مخصص (متقدم)
للمستخدمين المتقدمين الذين يريدون تحكماً كاملاً
1. افتح الإعدادات
2. انتقل إلى **المظهر** → **متقدم**
3. انقر فوق **فتح CSS المخصص**
4. أضف التخصيصاتك
### أمثلة التخصيص
```css
/* عناوين أكبر */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }
/* لون رابط مخصص */
a { color: #3b82f6; }
/* تباعد سطر أوسع */
.prose p { line-height: 2; }
```
## مزامنة السمات
عند تسجيل الدخول (اختياري)، تتم مزامنة تفضيللات السمات عبر الأجهزة
- مزامنة السمات تلقائية
- مزامنة إعدادات الخط
- مزامنة CSS المخصص
## نصائح
### مطابقة سير عملك
- استخدم **فاتح** خلال النهار، **داكن** في الليل
- يتولى وضع **النظام** هذا تلقائياً
- ضع في اعتبار الإضاءة المحيطة
### تقليل إجهاد العين
- استخدم الوضع الداكن في الإضاءة المنخفضة
- زد حجم الخط لجلسات طويلة
- فعّل التباين عالي إذا لزم الأمر
- خذ فترات راحة منتظمة
### الإنتاجية
- تعلم اختصار تبديل السمات
- قم بإعداد أوقات التبديل التلقائي
- استخدم وضع النظام للاتساق
