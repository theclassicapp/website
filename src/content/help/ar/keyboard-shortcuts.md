# اختصارات لوحة المفاتيح
اعمل بسرعة باست كلاسيك. جميع الاختصارات قابلة للإعدادات.

## الاختصارات العامةية
| الاختصار (Windows/Linux) | الاختصار (macOS) | الإجراء |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | ملاحظة جديدة |
| `Ctrl+P` | `Cmd+P` | لوحة الأوامر |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | الالتقاط السريع العالمي |
| `Ctrl+S` | `Cmd+S` | حفظ الملاحظة |
| `Ctrl+F` | `Cmd+F` | البحث |
| `Ctrl+,` | `Cmd+,` | الإعدادات |

## الالتقاط السريع
| الاختصار (Windows/Linux) | الاختصار (macOS) | الإجراء |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | فتح الالتقاط السريع |
| `Escape` | `Escape` | إغلاق الالتقاط السريع |
| `Ctrl+Enter` | `Cmd+Enter` | حفظ وإغلاق |
## تخصيص الاختصارات
1. افتح **الإعدادات** (`Ctrl+,` أو `Cmd+,`)
2. انتقل إلى **اختصارات لوحة المفاتيح**
3. انقر على أي اختصار لتغييره
4. اضغط **حفظ**
## نصائح
- معظم الاختصارات قابلة للإعدادات
- معظمها قابل للتخصي
 `Cmd/Ctrl+,` لفتح الإعدادات بسرعة
- استخدم البحث للإعدادات حسب الاسم
