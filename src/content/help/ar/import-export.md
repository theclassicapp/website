# الاستيراد والتصدير
أحضر ملاحظاتك إلى كلاسيك من تطبيقات أخرى، أو صدّها لمشاركتها أو نسخ احتياطي. يدعم كلاسيك تنسيقات متعددة للترحيل السلس للبيانات.

## الاستيراد
### التنسيقات المدعومة
| التنسيق | الامتداد | ملاحظات |
| ------ | --------- | ----- |
| Markdown | `.md` | ✅ دعم كامل |
| نص عادي | `.txt` | ✅ استيراد كملاحظة |
| JSON | `.json` | ✅ تنسيق كلاسيك |
| HTML | `.html` | ⚠️ تحويل إلى Markdown |
| Evernote | `.enex` | ⚠️ تجريبي |
| Notion | تصدير Notion | ⚠️ تجريبي |
| Obsidian | تصدير Markdown | ✅ دعم كامل |
### عملية الاستيراد
1. افتح **File** → **Import** (أو `Cmd/Ctrl+I`)
2. حدد الملفات للاستيراد
3. اختر مجلد الوجهة
4. راجع معاينة الاستيراد
5. تأكيد الاستيراد
### الاستيراد من الحافظة
استيراد سريع من أي مصدر
1. انسخ النص إلى الحافظة
2. افتح لوحة الأوامر
3. اكتب "Import from Clipboard"
4. يتم إنشاء ملاحظة جديدة تلقائياً
### الاستيراد المجمع
استيراد ملفات متعددة دفعة واحدة
1. افتح **File** → **Import**
2. حدد **Import Folder**
3. اختر المجلد الذي يحتوي على الملاحظات
4. يتم استيراد جميع الملفات المدعومة
## التصدير
### تنسيقات التصدير
| التنسيق | حالة الاستخدام |
| ------ | ------- |
| **Markdown** | المشاركة، النسخ الاحتياطية، تطبيقات أخرى |
| **HTML** | النشر على الويب، المعاينة |
| **PDF** | الأرشفة، الطباعة |
| **نص عادي** | مشاركة بسيطة |
| **JSON** | تكامل API |
### تصدير ملاحظة واحدة
1. افتح الملاحظة
2. انقر **File** → **Export**
3. اختر التنسيق
4. حدد الموقع
5. حفظ
### تصدير ملاحظات متعددة
1. حدد ملاحظات في الشريط الجانبي (Cmd/Ctrl+click)
2. انقر بزر الفأرة الأيمن → **Export Selected**
3. اختر التنسيق
4. حدد الموقع
5. حفظ
### تصدير جميع الملاحظات
1. افتح **File** → **Export All**
2. اختر التنسيق
3. حدد الموقع
4. انتظر اكتمال التصدير
## أدلة الترحيل
### من Obsidian
يمكن كلاسيك استيراد مخازن Obsidian مباشرة
1. افتح **File** → **Import**
2. حدد **Obsidian Vault**
3. انتقل إلى مجلد المخزن
4. حدد للاستيراد
**ما يتم نقله**
- ✅ الملاحظات والمجلدات
- ✅ الوسوم
- ✅ الروابط (محولة إلى روابط خلفية)
- ✅ البيانات التعريفية
- ⚠️ الإضافات (غير مدعومة)
### من Notion
1. صدّر من Notion كـ Markdown
2. استيراد إلى كلاسيك
3. راجع التنسيق
4. أصلح أي روابط معطلة
**نصائح لترحيل Notion**
- استخدم "Export as Markdown" في Notion
- تحقق من تنسيق مربعات الاختيار
- تحقق من مراجع قواعد البيانات
- أعد إنشاء قواعد البيانات المرتبطة يدوياً
### من Evernote
1. صدّر من Evernote كـ HTML
2. استخدم استيراد HTML في كلاسيك
3. راجع المحتوى المحول
**نصائح لترحيل Evernote**
- تصبح دفاتر الملاحظات مجلدات
- يتم الحفاظ على الوسوم
- يتم استيراد المرفقات
- تحقق من التنسيق
## أفضل الممارسات
### قبل الاستيراد
- نسخ احتياطي للملاحظات الموجودة
- راجع معاينة الاستيراد
- تحقق من التكرارات
- تحقق من التنسيق
### بعد الاستيراد
- راجع الملاحظات المستوردة
- أصلح أي مشاكل في التنسيق
- حدّث الروابط الداخلية
- أضف وسوم حسب الحاجة
### التصدير المنتظم
- صدّر نسخاً احتياطية شهرياً
- استخدم تنسيقات مختلفة لأغراض مختلفة
- حافظ على تنظيم التصديرات حسب التاريخ
- خزن في السحابة كنسخة احتياطية خارج الموقع
## استكشاف الأخطاء
### فشل الاستيراد
**الأسباب الشائعة**
- تنسيق غير مدعوم
- ملف تالف
- مشاكل في الأذونات
- قيود في الذاكرة
**الحلول**
1. جرب تنسيقاً مختلفاً
2. تحقق من سلامة الملف
3. تحقق من أذونات القراءة
4. استيراد عدد أقل من الملفات في المرة الواحدة
### فقدان التنسيق
**الأسباب الشائعة**
- تنسيق معقد
- بناء جملة غير مدعوم
- قيود التحويل
**الحلول**
1. استخدم تصدير Markdown من التطبيق المصدر
2. ببسط التنسيق قبل الاستيراد
3. أصلح يدوياً بعد الاستيراد
### المرفقات مفقودة
**الأسباب الشائعة**
- الاستيراد لم يتضمن المرفقات
- مسار مرفق خاطئ
- قيود الحجم
**الحلول**
1. أعد الاستيراد مع المرفقات
2. تحقق من مجلد المرفقات
3. استيراد المرفقات بشكل منفصل
