# الإعدادات
خصص كلاسيك ليعمل بالطريقة التي تريدها بالضبط. كل شيء قابل قابل الإعدادات.

## فتح الإعدادات
### اختصار لوحة المفاتيح
| platform | shortcut |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |
### عبر القائمة
- **macOS**: كلاسيك ← الإعدادات
- **Windows**: File → Settings
- **Linux**: تحرير ← الإعدادات
### عبر لوحة الأوامر
1. افتح لوحة الأوامر (`Cmd+P` or `Ctrl+P`)
2. اكتب "Settings"
3. اضغط Enter

## البحث في الإعدادات
لا تجد إعداداً؟ استخدم شريط البحث
1. افتح الإعدادات
2. اكتب في مربع البحث
3. تظهر الإعدادات المطابقة
4. انقر للانتقال إلى ذلك الإعداد
## إعادة تعيين الإعدادات
### إعادة تعيين جميع الإعدادات
العودة إلى الافتراضيات
1. افتح الإعدادات
2. انتقل إلى الأسفل
3. انقر فوق "Reset All Settings"
4. تأكيد الإجراء
### إعادة تعيين فئة
إعادة تعيين فئة محددة
1. انتقل إلى الفئة
2. انقر فوق "Reset to Defaults"
3. يتم إعادة تعيين تلك الفئة فقط
## تصدير/استيراد الإعدادات
### تصدير الإعدادات
احفظ تكوينك
1. افتح الإعدادات
2. انقر فوق "Export Settings"
3. اختر الموقع
4. احفظ كملف JSON
### استيراد الإعدادات
تحميل تكوين محفوظ
1. افتح الإعدادات
2. انقر فوق "Import Settings"
3. حدد ملف JSON
4. تأكيد الاستيراد
## موقع الإعدادات
يتم تخزين الإعدادات محلياً
| platform | location |
| ---------- | ---------- |
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |
## مزامنة الإعدادات (اختياري)
عند تسجيل الدخول، يمكن مزامنة الإعدادات عبر الأجهزة
- **المزامنة**: تمكين في الإعدادات ← مزامنة
- **ما الذي يُزامن**: السم،، الخطوط، الاختصارات
- **ما لا يُزامن**: المسارات المحلية، مفاتيح التشفير
## التخصيصات الشائعة
### زيادة حجم الخط
1. الإعدادات ← المظهر
2. حجم خط المحرر ← زيادة
3. يتم تطبيق التغييرات فوراً
### تغيير السمة
1. الإعدادات ← المظهر
2. السمة ← حدد فاتح/داكن/النظام
3. يتم تحديث التطبيق فوراً
### تفعيل النسخ الاحتياطي التلقائي
1. الإعدادات ← النسخ الاحتياطي والاستعادة
2. فعّل "Auto Backup"
3. عيّن التكرار والموقع
### تخصيص الاختصارات
1. الإعدادات ← اختصارات لوحة المفاتيح
2. ابحث عن الإجراء
3. انقر على الاختصار الحالي
4. اضغط على المفاتيح الجديدة
5. حفظ
## نصائح
### الوصول السريع
- استخدم `Cmd/Ctrl+,` لفتح الإعدادات بسرعة
- ابحث عن الإعدادات بالاسم
- استخدم الفئات للتنقل
### التصدير قبل التغييرات
قبل التغييرات الكبيرة، صدّر إعداداتك
- سهولة الاستعادة إذا حدث خطأ ما
- شارك التكوين مع الآخرين
### التحققق من الإعدادات الجيدة
- تحقق بعد التحديث
- استكشف الخيارات الجيدة
## استكشاف الأخطاء
### الإعدادات لا تحفظ
- تحقق من أذونات الملفات
- تحقق من مساحة القرص
- جرب إعادة التعيين وإعادة التكوين
### الإعدادات تُعاد بشكل غير متوقع
- قد تشير إلى تلف الملف
- صدّر الإعدادات بانتظام
- تحقق من تعارضات التطبيق
### لا أجد إعداداً
- استخدم شريط البحث
- تحقق من فئات مختلفة
- قد يكون في الإعدادات المتقدمة
