# تاريخ الإصدارات
يتتبع كلاسيك كل تغيير في ملاحظاتك. تصفح وقارن واستعد الإصدارات السابقة حاجةك إليها.

## كيف يعمل تاريخ الإصدارات
### التتبع التلقائي
يقوم كلاسيك بحفظ إصدارات من ملاحظاتك تلقائياً
- **كل تعديل**: كل حفظ ينشئ إصدارة
- **مختوم بالوقت**: اعرف متى حدثت التغييرات
- **محتوى كامل**: لقطات كاملة محفوظةة### الاحتفاظ
| الإعداد | الافتراضي |
| --------- | --------- |
| **فترة الاحتفاظ** | 90 يوماً |
| **الحد الأقصى للإصدارات** | 100 لكل ملاحظة |
| **التنظيف التلقائي** | نعم |
قم بتكوين هذه في الإعدادات ← المحرر → تاريخ الإصدار.
## عرض السسجل
### فتح سسجل الإصدارات
1. افتح أي ملاحظة
2. انقر على رمز السجل في شريط الأدوات
3. أو استخدم `Cmd/Ctrl+H`
### لوحة السجل
تعرض لوحة السجل:
- **الجدول الزمني**: قائمة بجميع الإصدارات
- **الطوابع الزمنية**: متى تم حفظ كل إصدارة
- **المعاينة**: شاهد محتوى الإصدارة المحددة
### التنقل بين الإصدارات
1. انقر على أي إصدارة في القائمة
2. تظهر المعاينة تلك الإصدارة
3. التمرير عبر التغييرات
4. قارن مع الإصدارة الحالية
## مقارنة الإصدارات
### العرض جنباً إلىجنب
رؤية الاختلافات بين الإصدارات
1. افتح سجل الإصدارات
2. حدد إصدارة
3. انقر فوق "Compare"
4. شاهد الإضافات (أخضر) والحذف (أحمر)
### عرض الفروق
تظهر الاختلافات النصية
- **الأخضر**: المحتوى المضاف
- **الأحمر**: المحتوى المحذوف
- **بدون تغيير**: نص عادي
## استعادة الإصدارات
### استعادة الملاحظة بالكامل
استبدل المحتوى الحالي بالإصدارة سابقة
1. افتح سجل الإصدارات
2. حدد الإصدارة
3. انقر فوق "Restore"
4. تأكيد الإجراء
### استعادة التحديد
انسخ جزء فقط فقط من إصدارة سابقة
1. افتح سجل الإصدارات
2. حدد الإصدارة
3. حدد النص الذي تريده
4. انقر فوق "Copy to Clipboard"
5. الصق في ملاحظتك الحالية
### إنشاء ملاحظة جديدة من إصدارة
احتفظ على كلت الإصدارين
1. افتح سجل الإصدارات
2. حدد الإصدارة
3. انقر فوق "Create Note from Version"
4. يتم إنشاء ملاحظة جديدة
## إجراءات السجل الإصدارات
| الإجراء | الوصف |
| -------- | ----------- |
| **عرض** | معاينة الإصدارة |
| **مقارنة** | المقارنة مع الحالي |
| **استعادة** | استبدال الحالي به هذه |
| **نسخ** | نسخ المحتوى إلى الحافظة |
| **حذف** | إزالة هذه الإصدارة |
## اختصارات لوحة المفاتيح
| الإجراء | الاختصار |
| -------- | ---------- |
| فتح السجل | `Cmd/Ctrl+H` |
| الإصدارة السابقة | `Up` |
| الإصدارة التالية | `Down` |
| استعادة | `Cmd/Ctrl+R` |
| إغلاق السجل | `Escape` |
## الإعدادات
### التكوين في الإعدادات ← المحرر → تاريخ الإصدار
| الإعداد | الخيارات | الافتراضي |
| --------- | --------- | --------- |
| تفعيل السجل | تشغيل/إيقاف | تشغيل |
| فترة الاحتفاظ | 30/60/90/180 يوماً | 90 يوماً |
| الحد الأقصى للإصدارات | 50/100/200/500 | 100 |
| فاصل الحفظ التلقائي | 1/5/10/30 دقيقة | 5 دقائق |
### موقع التخزين
يتم تخزين السجل محلياً
- **macOS**: `~/Library/Application Support/Classic/history/`
- **Windows**: `%APPDATA%/Classic/history/`
- **Linux**: `~/.config/Classic/history/`
## حالات الاستخدام
### استرداد التغييرات العرضية
1. افتح سجل الإصدارات
2. ابحث عن إصدارة قبل الخطأ
3. استعد تلك الإصدارة
### مقارنة المسودات
1. افتح السجل
2. قارن بين الإصدارات المختلفة
3. شاهد كيف تطورت الملاحظة
### البحث عن محتوى المحذوف
1. تصفح الإصدارات الأقدم
2. ابحث عن النص المحذوف
3. انسخه مرة أخرى
### تتبع التقدم
شاهد كيف تطورت ملاحظة بمرور الوقت
- متى أضيفت الأفكار
- كيف تغير الهيكل
- تطور الكتابة
## نصائح
### المراجعة الدورية
راجع السجل الإصدارات بشكل دوري
- نظف الإصدارات غير الضرورية
- لاحظ كيف تطور المحتوى
- تعلم من أنماط التحرير الخاصة بك
### قبل التغييرات الكبيرة
أنشئ نقطة مرجع
1. قم بإجراء تعديل صغير
2. احفظ (ينشئ إصدارة)
3. الآن قم بإجراء تغييراتكبيرة
4. سهولة الرجوع إذا لزم الأمر
### تسمية الإصدارات المهمة
أضف علامات في ملاحظاتك
```markdown
<!-- VERSION: Final draft -->
<!-- VERSION: Before restructuring -->
```
هذه تساعد في تحديد الإصدارات لاحقًاً.
## استكشاف الأخطاء
### السجل لا يحفظظ
- تحقق مما تم تفعيل السجل
- تحقق من موقع التخزين المتاح مساحة
- تحقق من أذونات الملفات
### لا أجد إصدارة قديمة
- تم تنظيفها تلقائياً
- قللت فترة الاحتفاظ
- زد الحد الأقصى للإصدارات ل لكل ملاحظة
### السجل بطيء في التحميل
الملاحظات الكبيرة التي تحتوي على العديد من الإصدارات قد يتم تحميلها بمرور
- فكر في تقليل الحد الأقصى للإصدارات
- أرشفة الملاحظات القدي
## الخصوصية والأمان
### التخزين المحلي
يتم تخزين السجل محلياً
- لا يُرسل أبداً إلى الخوادم
- بياناتك تظل خاصة
- أنت فقط لديك حق الوصول
- محمي بكلمة مرور النسخ الاحتياطي المشفرة،
