# الروابط الخلفية
تظهر الروابط الخلفية الروابط بين ملاحظاتك، مما ينشئ شبكة معرفة. اكتشف علاقات لم تكن تعرفها موجودة.

## ما هي الروابط الخلفية؟
تظهر الروابط الخلفية الملاحظات التي تشير إلى الملاحظة الحالية. إنها روابط في رسم المعرفة الخاص بك.
### الفوائد
- **اكتشاف الروابط**: ابحث عن الملاحظات ذات الصلة تلقائياً
- **السياق**: فهم كيف ترتبط الأفكار
- **التنقل**: انتقل بسرعة بين الملاحظات المتصلة
- **رسم المعرفة**: تصور تفكيرك بصرياً
## كيف تعمل الروابط الخلفية
عندما تنشئ رابطًا لملاحظة أخرى، ينشئ كلاسيك تلقائياً رابط خلفي.
```markdown
في الملاحظة أ:
See [[Project Planning]] for details.
النتيجة:
- الملاحظة أ تشير إلى "Project Planning"
- "Project Planning" يظهر رابط خلفي إلى الملاحظة أ
```
## عرض الروابط الخلفية
### لوحة الروابط الخلفية
افتح لوحة الروابط الخلفية لرؤية جميع المراجع
1. افتح أي ملاحظة
2. انقر على رمز الروابط الخلفية في الشريط الجانبي
3. شاهد جميع الملاحظات التي تشير إلى هذه الملاحظة
### الروابط الخلفية المضمنة
تظهر الروابط الخلفية في أسفل كل ملاحظة
- يظهر اسم الملاحظة المصدر
- يعرض مقتطف سياق
- انقر للتنقل
## إنشاء الروابط
### بناء جملة رابط ويكي
استخدم أقواس مزدوجة للربط بالملاحظات
```markdown
[[Note Name]]
[[Another Note]]
```
### نص العرض
خصص نص الرابط
```markdown
[[Project Alpha|the alpha project]]
[[Meeting Notes|yesterday's meeting]]
```
### الربط بالعناوين
اربط بأقسام محددة
```markdown
[[Note Name#Heading]]
[[Note Name#Section|jump to section]]
```
## أنواع الروابط الخلفية
### الروابط المباشرة
روابط ويكي صريحة التي تنشئها
```markdown
Related: [[Similar Note]]
```
### الإشارات غير المرتبطة
ملاحظات تذكر العنوان لكن غير مرروطة
```markdown
Note mentions "Project Alpha" but isn't linked
```
يقترح كلاسيك تحويل هذه إلى روابط.
## لوحة الروابط الخلفية
### الميزات
- **التصفية**: البحث في الروابط الخلفية
- **الترتيب**: حسب التاريخ أو الاسم أو الصلة
- **المعاينة**: مرر لمعاينة المحتوى
- **السياق**: شاهد النص المحيط
### الإجراءات
- انقر لفتح الملاحظة المرتبطة
- انقر بزر الفأرة الأيمن للخيارات
- تحويل الإشارات غير المرتبطة
## حالات الاستخدام
### إدارة المعرفة
بناء ويكي شخصي
```markdown
# Machine Learning
Related to [[Artificial Intelligence]]
Builds on [[Statistics]]
Used in [[Natural Language Processing]]
```
### توثيق المشاريع
ربط المستندات ذات الصلة
```markdown
# API Design
References [[Database Schema]]
Depends on [[Authentication Flow]]
See also [[Frontend Integration]]
```
### البحث
اربط المفاهيم والمصادر
```markdown
# Research Notes
Based on [[Source Paper A]]
Contradicts [[Source Paper B]]
Supports [[Hypothesis 1]]
```
### ملاحظات الاجتماعات
اربط الاجتماعات بالمشاريع
```markdown
# Meeting: Q4 Planning
Project: [[Project Alpha]]
Previous: [[Q3 Review]]
Next: [[Q4 Kickoff]]
```
## عرض الرسم البي
تصور الروابط الخلفية كشبكة
1. افتح عرض الرسم (`Cmd/Ctrl+G`)
2. شاهد جميع الملاحظات كعقد
3. تظهر الخطوط الروابط
4. انقر على العقد للتنقل
### تحكمات الرسم
- **التكبير**: مرر للتكبير
- **التحريك**: اسحب للتحريك
- **التصفية**: إظه/إخفاء حسب الوسم
- **التركيز**: تمييز روابط ملاحظة
## نصائح
### اربط بسخاء
أنشئ روابط كلما أشرت بمفهوم أخرى
```markdown
✅ Good:
Discussed the [[Q4 Roadmap]] and [[Budget Allocation]].
❌ Avoid:
Discussed the Q4 Roadmap and Budget Allocation.
```
### استخدم أسماء متسقة
طابق عناوين الملاحظات بالضبط للربط الموثوق
```markdown
✅ [[Meeting Notes]]
❌ [[meeting notes]] (الأحرف الكلمات مهمة)
```
### راجع الروابط الخلفية
راجع الروابط الخلفية بشكل دوري
- اكتشف روابط منسية
- ابحث عن ملاحظات لدمجها
- تحديد فجوات المعرفة
### معاينة الرابط
مرر فوق الروابط للمعاينة بدون فتحها
- شاهد الأسطر القليلة
- تحقق مما إذا كانت الملاحظة الصحيحة
- مرجع سريع
## اختصارات لوحة المفاتيح
| الإجراء | الاختصار |
| -------- | ---------- |
| فتح الروابط الخلفية | `Cmd/Ctrl+Shift+B` |
| إنشاء رابط | `[[` (يكمل تلقائياً) |
| فتح عرض الرسم | `Cmd/Ctrl+G` |
| التنقل عبر الروابط | `Tab` خلال الروابط |
## ميزات متقدمة
### التضمين
تضمين محتوى من ملاحظات أخرى
```markdown
![[Note Name]]
![[Note Name#Section]]
```
### التحويل
تضمين المحتوى مضمّن
```markdown
## Related Content
![[Shared Template#Header]]
```
### تلوين الروابط
تتغير لون الروابط بناءً على
- **الأزرق**: رابط غير مزار
- **البنفسجي**: رابط تمت زيارته
- **الأحمر**: رابط معطل (الملاحظة غير موجودة)
## استكشاف الأخطاء
### الروابط الخلفية المفقودة
- تحقق من بناء جملة الرابط `[[Note Name]]`
- تحقق من تطابق عنوان الملاحظة بالضبط
- حدّث لوحة الروابط الخلفية
### الروابط المعطلة
- قد تمت إعادة تسمية الملاحظة
- قد تم حذف الملاحظة
- استخدم البحث للعثور على الملاحظة الصحيح
### الروابط المكررة
- دمج الملاحظات المشابه
- استخدم عمليات إعادة التوجيه للملاحظات المعاد تسميةها
- تحقق من اختلافات العناوين
