# أوامر الشرطة المائلة

أوامر الشرطة المائلة هي طريقة قوية لإدراج التنسيق والمحتوى الخاص بسرعة دون مغادرة لوحة المفاتيح. اكتب `/` في المحرر لفتح لوحة الأوامر.

## الأوامر المتاحة

### العناوين

| الأمر | النتيجة |
| ------- | ------ |
| `/h1` | عنوان من المستوى 1 |
| `/h2` | عنوان من المستوى 2 |
| `/h3` | عنوان من المستوى 3 |
| `/h4` | عنوان من المستوى 4 |
| `/h5` | عنوان من المستوى 5 |
| `/h6` | عنوان من المستوى 6 |

### تنسيق النص

| الأمر | النتيجة |
| ------- | ------ |
| `/bold` | **نص عريض** |
| `/italic` | *نص مائل* |
| `/strikethrough` | ~~يتوسطه خط~~ |
| `/code` | كتلة كود مضمّنة |
| `/quote` | اقتباس |

### القوائم

| الأمر | النتيجة |
| ------- | ------ |
| `/bullet` | • قائمة نقطية |
| `/numbered` | 1. قائمة مرقمة |
| `/todo` | - [ ] عنصر مهام |
| `/check` | - [x] عنصر مكتمل |

### إدراج محتوى

| الأمر | النتيجة |
| ------- | ------ |
| `/image` | إدراج صورة من رابط أو ملف |
| `/link` | إدراج رابط تشعبي |
| `/table` | إدراج جدول |
| `/hr` | خط أفقي |
| `/code` | كتلة كود مع اختيار اللغة |

### الكتل الخاصة

| الأمر | النتيجة |
| ------- | ------ |
| `/mermaid` | كتلة مخطط Mermaid |
| `/katex` | كتلة رياضيات KaTeX |
| `/callout` | كتلة تنبيه/إشعار |

## استخدام أوامر الشرطة المائلة

1. اكتب `/` في المحرر
2. ستظهر قائمة بجميع الأوامر المتاحة
3. ابدأ بالكتابة لتصفية الأوامر
4. اضغط `Enter` أو انقر للإدراج

## نصائح احترافية

- **وصول سريع**: فقط اكتب `/` وابدأ بكتابة اسم الأمر
- **البحث الضبابي**: لا تحتاج لكتابة اسم الأمر بالضبط - سيجد كلاسيك أقرب تطابق
- **التنقل بلوحة المفاتيح**: استخدم مفاتيح الأسهم للتنقل في القائمة
- **الأوامر الأخيرة**: الأوامر المستخدمة بشكل متكرر تظهر في أعلى القائمة
