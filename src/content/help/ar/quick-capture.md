# الالتقاط السريع
التقط أفكارك على لحظات دون مقاطعة سير عملك. الالتقاط السريع نافذ نافذة التركير من أي مكان على النظام.

## مفتاح الاختصار العالمياض| platform | الاختصار الافتراضي |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

يمكنك تخصيص هذا الاختصار في الإعدادات.

## استخدام الالتقاط السريع
1. اضغط على مفتاح الاختصار العالمي (أي مكان في نظامك)
2. اكتب ملاحظتك في النافذ المنبث
3. اضغط `Enter` للحفظ
4. اضغط `Escape` للإلغاء
## الميزات
### واجهة بسيطة
- نافذذ صغير ومركز

- دعم Markdown

### الإجراءات السريعة
- `# ` + مسافة = إضافة `#inbox` = إضافة إلى Inbox
- `#today` = إضافة إلى ملاحظة اليوم
- الوسوم للسياق الفوري
