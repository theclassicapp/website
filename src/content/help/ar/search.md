# البحث
ابحث عن أي ملاحظتك في ثوانٍ باستخدام ميزة البحث القوي في كلاسيك. حدد موقع ما تبحث عنه بالضبط.

## البحث الأساسي
### opening search
| platform | shortcut |
|----------|----------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |
or click the search bar in the sidebar.
### how it works
1. open search
2. start typing
3. results appear instantly
4. click to open a note
## search features
### full-text search
Classic searches through
- note titles
- note content
- tags
- file names
### real-time results
results update as you type
- no need to press enter
- instant feedback
- fuzzy matching enabled
### search highlighting
found terms are highlighted in results
- see context around matches
- identify relevant notes quickly
## search operators
### exact match
use quotes for exact phrases
```
"project planning"
"meeting notes"
```
### tag search
find notes with specific tags
```
tag:work
tag:meetings
tag:urgent
```
### exclude terms
use `-` to exclude
```
project -archived
meeting -cancelled
```
### or search
find notes with any term
```
project or initiative
```
### date range
search by date
```
created:2024-01-01..2024-12-31
modified:last-week
```
### file type
filter by extension
```
ext:md
ext:txt
```
## advanced search
### combining operators
build complex queries
```
tag:work -tag:archived created:last-week
"meeting" tag:project-alpha
```
### path search
search within folders
```
path:Projects/
path:Archive/
```
### title search
search only titles
```
title:Roadmap
title:Meeting
```
### content search
search only content (exclude titles)
```
content:important
content:deadline
```
## search filters
### by tag
1. type your search
2. click a tag to filter
3. results narrow to tagged notes
### by date
filter by modification date
- today
- this week
- this month
- custom range
### by folder
limit search to a folder
1. right-click a folder
2. select "Search in Folder"
3. results limited to that folder
## search results
### sorting
sort results by
- **relevance** (default)
- **date modified** (newest first)
- **date created** (newest first)
- **title** (alphabetical)
### result preview
hover over results to preview
- first few lines shown
- matching terms highlighted
- tags displayed
### context
see where the match occurred
- shows surrounding text
- highlights the match
- click to jump to location
## search history
### recent searches
access previous searches
1. click in search bar
2. see recent searches
3. click to re-run
### clearing history
1. open settings
2. go to privacy
3. click "Clear Search History"
## tips
### use specific terms
more specific = better results
```
✅ "Q4 marketing budget"
❌ budget
```
### combine with tags
narrow results quickly
```
project tag:urgent
```
### use exclusions
filter out noise
```
meeting -tag:archived
```
### save frequent searches
create saved searches
1. enter your search
2. click "Save Search"
3. name it
4. access from sidebar
## keyboard navigation
| action | shortcut |
|--------|----------|
| open search | `Cmd/Ctrl+F` |
| next result | `Enter` or `Down` |
| previous result | `Shift+Enter` or `Up` |
| close search | `Escape` |
## search settings
### customize in settings
| setting | options |
|---------|---------|
| case sensitive | on/off |
| search titles | on/off |
| search content | on/off |
| search tags | on/off |
| include archived | on/off |
## use cases
### finding old notes
```
"meeting" date:last-year
```
### project research
```
path:Projects/ tag:research
```
### action items
```
"TODO" or "FIXME" -tag:done
```
### recent work
```
modified:this-week
```
## troubleshooting
### no results found
- check spelling
- remove filters
- try broader terms
- check if note is archived
### slow search
- large note collections may take longer
- disable case sensitivity
- reduce search scope
### missing notes
- note might be in a different folder
- check if tag is correct
- verify note isn't archived
