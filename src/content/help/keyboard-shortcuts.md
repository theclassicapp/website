# Keyboard Shortcuts

Work faster with Classic's keyboard shortcuts. All shortcuts are customizable in Settings.

## Global Shortcuts

These work anywhere in the app.

### General

| Shortcut (Windows/Linux) | Shortcut (macOS) | Action |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | New note |
| `Ctrl+P` | `Cmd+P` | Command palette |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Global quick capture |
| `Ctrl+S` | `Cmd+S` | Save note |
| `Ctrl+F` | `Cmd+F` | Search |
| `Ctrl+,` | `Cmd+,` | Settings |

### Navigation

| Shortcut (Windows/Linux) | Shortcut (macOS) | Action |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | Go to first note |
| `Ctrl+2` | `Cmd+2` | Go to second note in `Ctrl+Left` | `Cmd+Left` | Navigate back |
| `Ctrl+Right` | `Cmd+Right` | Navigate forward |
| `Ctrl+Up` | `Cmd+Up` | Navigate to parent note |
| `Ctrl+Down` | `Cmd+Down` | Expand/focus first subnote |

## Quick Capture

The Shortcut (Windows/Linux) | Shortcut (macOS) | Action |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Open quick capture |
| `Escape` | `Escape` | Close quick capture |
| `Ctrl+Enter` | `Cmd+Enter` | Save and close |

## Customizing Shortcuts

1. Open **Settings** (`Ctrl+,` or `Cmd+,`)
2. Navigate to **Keyboard Shortcuts**
3. Click on any shortcut to change it
4. Press your new key combination
5. Click **Save**

## Tips

- Most shortcuts are customizable
- Conflicts with system shortcuts will be highlighted
- Reset to default by clicking the reset button next to each shortcut
