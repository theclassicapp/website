# Entendiendo la Interfaz

La interfaz de Classic está diseñada para ser limpia, intuitiva y no estorbar. Aquí tienes un recorrido por los componentes principales.

## Diseño Principal

Classic tiene un diseño de tres paneles:

1. **Barra Lateral** (izquierda) - Navegación y organización de notas
2. **Editor** (centro) - Tu espacio de escritura
3. **Vista Previa/Propiedades** (derecha) - Paneles opcionales

## Barra Lateral

La barra lateral contiene varias secciones clave:

### Lista de Notas

- **Todas las Notas**: Ver todas tus notas en una lista
- **Notas Diarias**: Acceso rápido al diario de hoy
- **Favoritos**: Marcar notas usadas frecuentemente
- **Carpetas**: Organizar notas en jerarquías

### Navegación

- **Barra de Búsqueda**: Encuentra notas al instante
- **Etiquetas**: Navegar por etiqueta
- **Enlaces Inversos**: Ver conexiones entre notas

### Acciones

- **Nueva Nota**: Crear una nueva nota
- **Nueva Carpeta**: Organizar con carpetas
- **Importar**: Traer contenido externo

## Editor

El editor es donde escribes y formateas tus notas.

### Características

- **Soporte Markdown**: Sintaxis Markdown completa
- **Vista Previa en Vivo**: Ver texto formateado mientras escribes
- **Resaltado de Sintaxis**: Los bloques de código se resaltan
- **Comandos con Barra**: Escribe `/` para acciones rápidas

### Modos del Editor

| Modo | Descripción |
|------|-------------|
| **Solo Edición** | Ver Markdown sin formato |
| **Solo Vista Previa** | Ver salida renderizada |
| **Vista Dividida** | Editar y previsualizar lado a lado |

### Barra de Herramientas

Acciones de formato rápido:

- Negrita, Cursiva, Tachado
- Encabezados (H1-H6)
- Listas (con viñetas, numeradas, tareas)
- Enlaces e imágenes
- Bloques de código
- Tablas

## Barra de Estado

En la parte inferior del editor:

- **Conteo de Palabras/Caracteres**: Rastrea tu escritura
- **Último Guardado**: Cuándo se guardó la nota por última vez
- **Ruta de la Nota**: Dónde está ubicada la nota
- **Estado de Sincronización**: Si se usa sincronización opcional

## Menús Contextuales

Clic derecho para acciones contextuales:

### Menú Contextual de Nota

- Cambiar nombre
- Mover a carpeta
- Duplicar
- Eliminar
- Exportar
- Copiar enlace

### Menú Contextual del Editor

- Cortar, Copiar, Pegar
- Opciones de formato
- Insertar enlace/imagen
- Buscar y reemplazar

## Navegación con Teclado

Navega la interfaz con tu teclado:

| Acción | Atajo |
|--------|-------|
| Alternar Barra Lateral | `Cmd/Ctrl + M` |
| Enfocar Búsqueda | `Cmd/Ctrl + F` |
| Paleta de Comandos | `Cmd/Ctrl + P` |
| Nueva Nota | `Cmd/Ctrl + N` |

## Personalización

Ajusta la interfaz a tu gusto:

### Ancho de la Barra Lateral

1. Pasa el cursor sobre el borde de la barra lateral
2. Arrastra para redimensionar
3. O establece el ancho exacto en Configuración

### Tema

Cambia entre modo Claro y Oscuro:

- Usa el interruptor de tema en el encabezado
- O configura para seguir la preferencia de tu sistema

### Tamaño de Fuente

1. Abre Configuración
2. Ve a Apariencia
3. Ajusta el Tamaño de Fuente del Editor

## Consejos

### Modo de Enfoque

Oculta todos los paneles para escritura sin distracciones:

1. Abre la Paleta de Comandos
2. Escribe "Modo de Enfoque"
3. Presiona Enter

### Acceso Rápido

- Fija las notas usadas frecuentemente a Favoritos
- Usa atajos de teclado para acciones comunes
- Personaliza la barra de herramientas en Configuración
