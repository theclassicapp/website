# Idiomas y Localización

Classic habla tu idioma. Usa la aplicación en tu idioma preferido y escribe notas en cualquier idioma que elijas.

## Idiomas Soportados

La interfaz de Classic está disponible en múltiples idiomas:

| Idioma | Código |
|--------|--------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| Y más... | |

## Cambiar el Idioma de la Interfaz

### Vía Configuración

1. Abre Configuración (`Cmd+,` o `Ctrl+,`)
2. Ve a **General** → **Idioma**
3. Selecciona tu idioma preferido
4. Reinicia Classic para aplicar

### Vía Paleta de Comandos

1. Abre la Paleta de Comandos (`Cmd+P` o `Ctrl+P`)
2. Escribe "Idioma"
3. Selecciona tu idioma preferido
4. Reinicia para aplicar cambios

## Escribir en Múltiples Idiomas

Classic soporta escribir notas en cualquier idioma.

### Soporte Unicode

- Soporte Unicode completo
- Idiomas RTL (Derecha a Izquierda) soportados
- Caracteres CJK (Chino, Japonés, Coreano) funcionan perfectamente
- Soporte para emojis

### Detección de Idioma

Classic puede detectar el idioma de tus notas:

- Auto-detectar para corrección ortográfica
- Configuración de idioma por nota
- Soporte para idiomas mixtos

## Corrección Ortográfica

### Habilitar Corrector Ortográfico

1. Abre Configuración
2. Ve a **Editor** → **Corrector Ortográfico**
3. Habilita "Corrector Ortográfico"

### Idiomas del Corrector Ortográfico

1. Abre Configuración
2. Ve a **Editor** → **Corrector Ortográfico**
3. Haz clic en "Idiomas"
4. Selecciona idiomas para verificar

### Agregar Palabras al Diccionario

1. Clic derecho en una palabra marcada
2. Selecciona "Agregar al Diccionario"
3. La palabra no se marcará de nuevo

## Soporte RTL

Para Árabe, Hebreo y otros idiomas RTL:

### Detección RTL Automática

Classic detecta contenido RTL automáticamente:

- La dirección del texto se ajusta
- El movimiento del cursor se adapta
- La alineación cambia

### Alternar RTL Manual

Forzar dirección RTL o LTR:

1. Abre la Paleta de Comandos
2. Escribe "RTL" o "LTR"
3. Selecciona tu preferencia

## Métodos de Entrada

Classic funciona con los métodos de entrada de tu sistema.

### Métodos de Entrada CJK

- Funciona con Pinyin, Zhuyin, etc.
- Soporta IME japonés
- Métodos de entrada coreanos soportados

### Caracteres Especiales

Inserta caracteres especiales fácilmente:

- Usa el selector de caracteres de tu sistema
- Copia/pega de mapas de caracteres
- Usa entrada Unicode

## Formatos de Fecha y Hora

Las fechas se adaptan a tu configuración regional.

### Opciones de Formato

| Configuración | Opciones |
|---------------|----------|
| Formato de Fecha | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| Formato de Hora | 12 horas, 24 horas |
| Primer Día de la Semana | Domingo, Lunes, Sábado |

Configura en Configuración → General → Fecha y Hora

## Formatos de Números

Los números se muestran según tu configuración regional:

- **Separador decimal**: `.` o `,`
- **Separador de miles**: `,` o `.` o espacio
- **Símbolos de moneda**: Basado en región

## Distribuciones de Teclado

Classic funciona con cualquier distribución de teclado:

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Distribuciones personalizadas

## Configuración de Localización

### En Configuración → General → Idioma

| Configuración | Descripción |
|---------------|-------------|
| **Idioma de Interfaz** | Idioma de la UI de la app |
| **Formato de Fecha** | Cómo aparecen las fechas |
| **Formato de Hora** | 12 o 24 horas |
| **Primer Día de la Semana** | Día de inicio del calendario |
| **Formato de Número** | Separadores decimal/millar |

## Consejos

### Notas en Múltiples Idiomas

Escribe en múltiples idiomas en una nota:

```markdown
# Proyecto Internacional

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Fuentes Específicas por Idioma

Algunos idiomas se ven mejor con fuentes específicas:

1. Abre Configuración
2. Ve a Apariencia → Fuentes
3. Establece fuentes para diferentes scripts

### Cambiar Idiomas Rápidamente

Crea un atajo de teclado:

1. Abre Configuración → Atajos
2. Encuentra "Cambiar Idioma"
3. Establece tu atajo preferido

## Solución de Problemas

### Idioma No Disponible

- Verifica actualizaciones de la app
- El idioma puede estar en desarrollo
- Contribuye con traducciones en GitHub

### Caracteres No Se Muestran

- Instala las fuentes requeridas
- Verifica configuración de fuentes
- Intenta una fuente diferente

### Corrector Ortográfico No Funciona

- Verifica que el corrector esté habilitado
- Agrega el idioma al corrector ortográfico
- Reinicia Classic

### RTL No Funciona Correctamente

- Asegúrate de que la detección RTL esté activada
- Intenta alternar RTL manual
- Verifica problemas de contenido mixto

## Contribuir con Traducciones

Ayuda a traducir Classic:

1. Visita la comunidad de Classic
2. Únete al equipo de traducción
3. Envía traducciones
4. Ayuda a mejorar la localización
