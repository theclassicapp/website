# Atajos de Teclado

Trabaja más rápido con los atajos de teclado de Classic. Todos los atajos son personalizables en Configuración.

## Atajos Globales

Estos funcionan en cualquier lugar de la aplicación.

### General

| Atajo (Windows/Linux) | Atajo (macOS) | Acción |
| --------------------- | ------------- | ------ |
| `Ctrl+N` | `Cmd+N` | Nueva nota |
| `Ctrl+P` | `Cmd+P` | Paleta de comandos |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Captura rápida global |
| `Ctrl+S` | `Cmd+S` | Guardar nota |
| `Ctrl+F` | `Cmd+F` | Búsqueda |
| `Ctrl+,` | `Cmd+,` | Configuración |

### Navegación

| Atajo (Windows/Linux) | Atajo (macOS) | Acción |
| --------------------- | ------------- | ------ |
| `Ctrl+1` | `Cmd+1` | Ir a la primera nota |
| `Ctrl+2` | `Cmd+2` | Ir a la segunda nota |
| `Ctrl+Left` | `Cmd+Left` | Navegar atrás |
| `Ctrl+Right` | `Cmd+Right` | Navegar adelante |
| `Ctrl+Up` | `Cmd+Up` | Navegar a nota padre |
| `Ctrl+Down` | `Cmd+Down` | Expandir/enfocar primera subnota |

### Editor

| Atajo (Windows/Linux) | Atajo (macOS) | Acción |
| --------------------- | ------------- | ------ |
| `Ctrl+B` | `Cmd+B` | Negrita |
| `Ctrl+I` | `Cmd+I` | Cursiva |
| `Ctrl+K` | `Cmd+K` | Insertar enlace |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | Tachado |
| `Ctrl+`` | `Cmd+`` | Código en línea |
| `Ctrl+Shift+C` | `Cmd+Shift+C` | Bloque de código |
| `Ctrl+/` | `Cmd+/` | Comandos con barra |
| `Tab` | `Tab` | Sangrar |
| `Shift+Tab` | `Shift+Tab` | Quitar sangría |

### Gestión de Ventanas

| Atajo (Windows/Linux) | Atajo (macOS) | Acción |
| --------------------- | ------------- | ------ |
| `Ctrl+W` | `Cmd+W` | Cerrar nota actual |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | Cerrar todas las notas |
| `F11` | `F11` | Alternar pantalla completa |
| `Ctrl+M` | `Cmd+M` | Alternar barra lateral |

## Captura Rápida

| Atajo (Windows/Linux) | Atajo (macOS) | Acción |
| --------------------- | ------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Abrir captura rápida |
| `Escape` | `Escape` | Cerrar captura rápida |
| `Ctrl+Enter` | `Cmd+Enter` | Guardar y cerrar |

## Personalizar Atajos

1. Abre **Configuración** (`Ctrl+,` o `Cmd+,`)
2. Navega a **Atajos de Teclado**
3. Haz clic en cualquier atajo para cambiarlo
4. Presiona tu nueva combinación de teclas
5. Haz clic en **Guardar**

## Consejos

- La mayoría de los atajos son personalizables
- Los conflictos con atajos del sistema se resaltarán
- Restablece a valores por defecto haciendo clic en el botón de restablecer junto a cada atajo
