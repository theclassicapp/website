# Crear Tu Primera Nota

Comienza con Classic creando tu primera nota. Es simple e intuitivo.

## Crear una Nueva Nota

### Usando la Barra Lateral

1. Haz clic en el botón **+** en la barra lateral
2. Se crea una nueva nota
3. Comienza a escribir tu contenido

### Usando Atajo de Teclado

| Plataforma | Atajo |
| ---------- | ----- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Usando la Paleta de Comandos

1. Abre la Paleta de Comandos (`Cmd+P` o `Ctrl+P`)
2. Escribe "Nueva Nota"
3. Presiona Enter

## Escribir Tu Nota

### Edición Básica

¡Simplemente comienza a escribir! Classic guarda automáticamente mientras escribes.

### Agregar un Título

La primera línea de tu nota se convierte en el título. Usa un encabezado para mejores resultados:

```markdown
# Mi Primera Nota

Este es el contenido de mi nota.
```

### Formatear Texto

Usa Markdown para formatear tu texto:

- **Negrita**: `**texto**` o `Cmd/Ctrl+B`
- *Cursiva*: `*texto*` o `Cmd/Ctrl+I`
- ~~Tachado~~: `~~texto~~`
- `Código`: `` `código` ``

## Guardar

Classic guarda tus notas automáticamente mientras escribes. No necesitas guardar manualmente.

### Guardar Manual

Si es necesario, puedes forzar un guardado:

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## Organizar Tu Nota

### Agregar Etiquetas

Etiqueta tu nota para encontrarla fácilmente después:

1. Escribe `#` seguido del nombre de tu etiqueta
2. O usa el botón de etiqueta en la barra de herramientas
3. Se admiten múltiples etiquetas

Ejemplo:

```markdown
# Notas de Reunión

Discutimos la planificación del Q4 #trabajo #reuniones
```

### Mover a una Carpeta

1. Clic derecho en la nota en la barra lateral
2. Selecciona "Mover a Carpeta"
3. Elige o crea una carpeta de destino

### Fijar Notas Importantes

1. Clic derecho en la nota
2. Selecciona "Fijar al Inicio"
3. La nota aparece al inicio de tu lista

## Previsualizar Tu Nota

Ver cómo se ve tu nota formateada:

- Haz clic en el botón **Vista Previa**
- O usa `Cmd/Ctrl+Shift+P`
- Alterna entre modos de Edición y Vista Previa

## Consejos

### Títulos de Notas

- Mantén los títulos descriptivos
- Primera línea = título de la nota
- Usa H1 (`#`) para claridad

### Guardado Automático

- Nunca te preocupes por perder trabajo
- Los cambios se guardan automáticamente
- Revisa la barra de estado para el estado de guardado

### Captura Rápida

¿Necesitas anotar algo rápido?

- Usa Captura Rápida (`Cmd/Ctrl+Shift+N`)
- Crea una nota al instante
- Genial para pensamientos fugaces
