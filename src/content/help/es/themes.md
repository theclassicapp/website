# Temas y Apariencia

Personaliza la apariencia de Classic para que coincida con tu estilo y reduzca la fatiga visual.

## Opciones de Tema

### Modo Claro

El tema por defecto de Classic cuenta con:

- Interfaz limpia y brillante
- Óptimo para ambientes bien iluminados
- Alto contraste para legibilidad

### Modo Oscuro

Perfecto para noctámbulos y trabajo con poca luz:

- Reduce fatiga visual
- Menor consumo de batería en pantallas OLED
- Suave para los ojos en ambientes oscuros

### Sistema (Auto)

Sigue el tema de tu sistema operativo:

- Cambia automáticamente basado en configuración del SO
- Sin necesidad de cambiar manualmente
- Consistente con otras apps

## Cambiar Temas

### Vía Configuración

1. Abre Configuración (`Cmd+,` o `Ctrl+,`)
2. Navega a **Apariencia**
3. Selecciona tu tema preferido

### Vía Paleta de Comandos

1. Abre la Paleta de Comandos (`Cmd+P` o `Ctrl+P`)
2. Escribe "Tema"
3. Selecciona tu tema preferido

### Atajo de Teclado

| Plataforma | Atajo |
| ---------- | ----- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Cicla a través de: Claro → Oscuro → Sistema

## Configuración de Fuentes

### Fuente del Editor

Personaliza la experiencia de escritura:

1. Abre Configuración
2. Ve a **Apariencia** → **Editor**
3. Configura opciones de fuente

| Configuración | Opciones | Por Defecto |
| ------------- | -------- | ----------- |
| Familia de Fuente | Fuentes del sistema + personalizadas | UI del Sistema |
| Tamaño de Fuente | 12-24px | 16px |
| Altura de Línea | 1.0-2.5 | 1.75 |
| Espaciado de Letras | -2 a 4 | 0 |

### Fuente de UI

Ajusta la fuente de la interfaz:

1. Abre Configuración
2. Ve a **Apariencia** → **Interfaz**
3. Elige fuente y tamaño

### Fuente Monoespaciada

Para bloques de código y contenido técnico:

1. Abre Configuración
2. Ve a **Apariencia** → **Código**
3. Selecciona fuente monoespaciada

## Opciones de Interfaz

### Barra Lateral

Personaliza el comportamiento de la barra lateral:

| Configuración | Descripción |
| ------------- | ----------- |
| **Auto-ocultar** | Ocultar barra lateral cuando no está en uso |
| **Ancho** | Ajustar ancho de barra lateral (200-400px) |
| **Mostrar Íconos** | Alternar visibilidad de íconos |

### Diseño del Editor

Elige tu experiencia de edición:

- **Vista Dividida**: Editar y previsualizar lado a lado
- **Solo Vista Previa**: Ver solo salida renderizada
- **Solo Edición**: Enfocarse en Markdown sin formato

### Lista de Notas

Configura cómo aparecen las notas:

- **Vista de Lista**: Compacta, una línea por nota
- **Vista de Tarjetas**: Tarjetas de vista previa con fragmentos
- **Vista de Árbol**: Estructura de carpetas jerárquica

## Colores de Acento

### Colores Disponibles

Elige tu color de acento:

- Azul Classic (por defecto)
- Púrpura
- Verde
- Naranja
- Rojo
- Rosa
- Turquesa

### Cambiar Color de Acento

1. Abre Configuración
2. Ve a **Apariencia** → **Colores**
3. Selecciona tu acento preferido

El color de acento afecta:

- Enlaces y botones
- Resaltados de selección
- Elementos activos
- Indicadores de enfoque

## Accesibilidad

### Modo de Alto Contraste

Para mejor visibilidad:

1. Abre Configuración
2. Ve a **Apariencia** → **Accesibilidad**
3. Habilita **Alto Contraste**

### Movimiento Reducido

Minimiza animaciones:

1. Abre Configuración
2. Ve a **Apariencia** → **Accesibilidad**
3. Habilita **Movimiento Reducido**

### Ajustes de Fuente

- Aumenta tamaño mínimo de fuente
- Usa fuentes amigables para dislexia
- Ajusta ratios de contraste

## CSS Personalizado (Avanzado)

Para usuarios avanzados que quieren control completo:

1. Abre Configuración
2. Ve a **Apariencia** → **Avanzado**
3. Haz clic en **Abrir CSS Personalizado**
4. Agrega tus personalizaciones

### Ejemplo de Personalizaciones

```css
/* Encabezados más grandes */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Color de enlace personalizado */
a { color: #3b82f6; }

/* Espaciado de línea más amplio */
.prose p { line-height: 2; }
```

## Sincronización de Temas

Cuando inicias sesión (opcional), tus preferencias de tema se sincronizan entre dispositivos:

- Sincronización automática de tema
- Sincronización de configuración de fuentes
- CSS personalizado se sincroniza

## Consejos

### Coincidir con Tu Flujo de Trabajo

- Usa **Claro** durante el día, **Oscuro** de noche
- El modo **Sistema** maneja esto automáticamente
- Considera tu iluminación ambiental

### Reducir Fatiga Visual

- Usa modo Oscuro con poca luz
- Aumenta tamaño de fuente para sesiones largas
- Habilita Alto Contraste si es necesario
- Toma descansos regulares

### Productividad

- Aprende el atajo de alternar tema
- Configura tiempos de auto-cambio
- Usa modo Sistema para consistencia
