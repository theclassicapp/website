# Captura Rápida

Captura ideas al instante sin interrumpir tu flujo de trabajo. Captura Rápida abre una ventana minimal de toma de notas desde cualquier lugar de tu sistema.

## Tecla de Acceso Rápido Global

Presiona la tecla de acceso rápido designada para abrir Captura Rápida:

| Plataforma | Atajo por Defecto |
| ---------- | ----------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

Puedes personalizar este atajo en Configuración.

## Usar Captura Rápida

1. Presiona la tecla de acceso rápido global (en cualquier lugar de tu sistema)
2. Escribe tu nota en la ventana emergente
3. Presiona `Enter` para guardar
4. Presiona `Escape` para cancelar

## Características

### Interfaz Minimal

- Ventana pequeña y enfocada
- Soporte Markdown
- Auto-enfoque en entrada
- Diseño priorizando teclado

### Acciones Rápidas

- `# ` + espacio = encabezado
- `#inbox` = agregar a Bandeja de Entrada
- `#today` = agregar a la nota de Hoy
- Etiquetas con `#nombreetiqueta`

### Destino por Defecto

Las notas de Captura Rápida van a tu **Bandeja de Entrada** por defecto. Puedes cambiar esto:

1. Abre Configuración
2. Ve a Captura Rápida
3. Selecciona destino por defecto

## Consejos

### Flujo de Trabajo de Captura

1. **Capturar** - Anota la idea rápidamente
2. **Procesar** - Revisa la Bandeja de Entrada diariamente
3. **Organizar** - Mueve a la carpeta/nota apropiada

### Mejores Prácticas

- Manténlo breve - una idea por captura
- Usa etiquetas para contexto
- No te preocupes por el formato - refina después
- Revisa y procesa capturas regularmente

### Integración

Captura Rápida funciona genial con:

- **Notas Diarias**: Comienza cada día revisando capturas
- **Bandeja de Entrada**: Úsala como área de espera
- **Etiquetas**: Agrega contexto inmediatamente

## Solución de Problemas

### ¿Tecla de Acceso Rápido No Funciona?

**macOS**
1. Verifica Preferencias del Sistema > Teclado > Atajos
2. Busca conflictos
3. Reinicia Classic

**Windows**
1. Ejecuta Classic como Administrador
2. Verifica atajos en conflicto
3. Reinicia Classic

### ¿Ventana No Aparece?

- Verifica si Classic está ejecutándose
- Verifica el atajo
- Verifica problemas de enfoque
- Reinicia Classic

### ¿Capturas No Se Guardan?

- Verifica destino por defecto
- Verifica permisos de escritura
- Verifica espacio en disco
- Reinicia Classic
