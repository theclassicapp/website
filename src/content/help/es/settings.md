# Configuración

Personaliza Classic para que funcione exactamente como quieres. Desde apariencia hasta comportamiento del editor, todo es configurable.

## Abrir Configuración

### Atajo de Teclado

| Plataforma | Atajo |
|------------|-------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Vía Menú

- **macOS**: Classic → Configuración
- **Windows**: Archivo → Configuración
- **Linux**: Editar → Configuración

### Vía Paleta de Comandos

1. Abre la Paleta de Comandos (`Cmd+P` o `Ctrl+P`)
2. Escribe "Configuración"
3. Presiona Enter

## Categorías de Configuración

### General

Configuración básica de la aplicación:

| Configuración | Descripción |
|---------------|-------------|
| **Idioma** | Idioma de la interfaz |
| **Tema** | Claro, Oscuro o Sistema |
| **Iniciar al Inicio de Sesión** | Ejecutar al inicio |
| **Buscar Actualizaciones** | Comportamiento de auto-actualización |

| Configuración | Opciones |
|---------------|----------|

| Configuración | Descripción |

Controla el rastreo de versiones:

| Configuración | Opciones | Por Defecto |
|---------------|----------|-------------|
| **Habilitar Historial** | Encendido/Apagado | Encendido |
| **Período de Retención** | 30-180 días | 90 días |

## Buscar Configuración

¿No puedes encontrar una configuración? Usa la barra de búsqueda:

1. Abre Configuración
2. Escribe en el cuadro de búsqueda
3. Aparecen configuraciones coincidentes
4. Haz clic para saltar a esa configuración

## Restablecer Configuración

### Restablecer Toda la Configuración

Vuelve a valores por defecto:

1. Abre Configuración
2. Desplázate al final
3. Haz clic en "Restablecer Toda la Configuración"
4. Confirma la acción

### Restablecer Categoría

Restablece una categoría específica:

1. Ve a la categoría
2. Haz clic en "Restablecer a Valores por Defecto"
3. Solo esa categoría se restablece

## Exportar/Importar Configuración

### Exportar Configuración

Guarda tu configuración:

1. Abre Configuración
2. Haz clic en "Exportar Configuración"
3. Elige ubicación
4. Guarda como archivo JSON

### Importar Configuración

Carga una configuración guardada:

1. Abre Configuración
2. Haz clic en "Importar Configuración"
3. Selecciona el archivo JSON
4. Confirma importación

## Ubicación de Configuración

Las configuraciones se almacenan localmente:

| Plataforma | Ubicación |
|------------|-----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Sincronizar Configuración (Opcional)

Cuando inicias sesión, las configuraciones pueden sincronizarse entre dispositivos:

- **Sincronizar**: Habilita en Configuración → Sincronizar
- **Qué se Sincroniza**: Tema, fuentes, atajos
- **Qué No se Sincroniza**: Rutas locales, claves de encriptación

## Personalizaciones Comunes

### Aumentar Tamaño de Fuente

1. Configuración → Apariencia
2. Tamaño de Fuente del Editor → Aumentar
3. Los cambios se aplican inmediatamente

### Cambiar Tema

1. Configuración → Apariencia
2. Tema → Seleccionar Claro/Oscuro/Sistema
3. La app se actualiza al instante

### Habilitar Auto-Respaldo

1. Configuración → Respaldo y Restauración
2. Habilita "Auto Respaldo"
3. Establece frecuencia y ubicación

### Personalizar Atajos

1. Configuración → Atajos de Teclado
2. Encuentra la acción
3. Haz clic en el atajo actual
4. Presiona nuevas teclas
5. Guarda

## Consejos

### Acceso Rápido

- Usa `Cmd/Ctrl+,` para abrir configuración rápidamente
- Busca configuraciones por nombre
- Usa categorías para navegar

### Exportar Antes de Cambios

Antes de cambios mayores, exporta tu configuración:

- Fácil de restaurar si algo sale mal
- Comparte configuración con otros

### Verifica Nuevas Configuraciones

Las actualizaciones pueden agregar nuevas configuraciones:

- Verifica después de actualizar
- Explora nuevas opciones

## Solución de Problemas

### Configuración No Se Guarda

- Verifica permisos de archivo
- Verifica espacio en disco
- Intenta restablecer y reconfigurar

### Configuración Se Restablece Inesperadamente

- Puede indicar corrupción de archivo
- Exporta configuración regularmente
- Verifica conflictos de apps

### No Puedo Encontrar una Configuración

- Usa la barra de búsqueda
- Verifica diferentes categorías
- Puede estar en configuración Avanzada
