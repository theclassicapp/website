# Paleta de Comandos

La paleta de comandos es tu puerta de entrada a cada acción en Classic. Presiona un simple atajo de teclado y accede a cualquier característica al instante.

## Abrir la Paleta de Comandos

| Plataforma | Atajo |
| ---------- | ----- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Cómo Usar

1. Abre la paleta de comandos (`Cmd+P` o `Ctrl+P`)
2. Escribe para buscar un comando
3. Usa las teclas de flecha para navegar
4. Presiona `Enter` para ejecutar

## Categorías de Comandos

### Operaciones de Archivo

- **Nueva Nota**: Crear una nueva nota
- **Nueva Nota Diaria**: Crear/abrir la nota diaria de hoy
- **Abrir Nota**: Buscar y abrir notas existentes
- **Guardar Nota**: Guardar nota actual
- **Eliminar Nota**: Eliminar nota actual
- **Cambiar Nombre de Nota**: Cambiar nombre de nota actual

### Navegación

- **Ir a Nota**: Saltar a cualquier nota
- **Ir a Nota Diaria**: Abrir una fecha específica
- **Ir a Etiqueta**: Navegar notas por etiqueta
- **Ir Atrás**: Navegar atrás en el historial
- **Ir Adelante**: Navegar adelante en el historial

### Editor

- **Alternar Vista Previa**: Cambiar entre modos de edición y vista previa
- **Alternar Barra Lateral**: Mostrar/ocultar la barra lateral
- **Modo de Enfoque**: Entrar en escritura sin distracciones
- **Insertar Fecha**: Agregar fecha/hora actual
- **Insertar Plantilla**: Insertar una plantilla

### Formato

- **Negrita**: Poner texto en negrita
- **Cursiva**: Poner texto en cursiva
- **Tachado**: Tachar texto
- **Código**: Formatear como código
- **Enlace**: Insertar un enlace
- **Encabezado**: Aplicar formato de encabezado

### Organización

- **Agregar Etiqueta**: Etiquetar la nota actual
- **Mover a Carpeta**: Mover nota a una carpeta
- **Archivar Nota**: Archivar nota actual
- **Fijar Nota**: Fijar nota al inicio de la lista

### Vista

- **Alternar Tema**: Cambiar entre claro/oscuro
- **Acercar**: Aumentar tamaño de fuente
- **Alejar**: Disminuir tamaño de fuente
- **Restablecer Zoom**: Volver al tamaño por defecto

## Consejos

### Acceso Rápido

- Los comandos se buscan de forma difusa
- Los comandos recientes aparecen primero
- Fija comandos usados frecuentemente

### Navegación con Teclado

- `↑`/`↓` - Navegar resultados
- `Enter` - Ejecutar comando
- `Escape` - Cerrar paleta
- `Tab` - Ciclar a través de categorías

### Comandos Personalizados

1. Abre Configuración
2. Ve a Paleta de Comandos
3. Agrega comandos personalizados
4. Define atajos de teclado

Aquí hay algunos comandos que usarás a menudo:

| Comando | Qué hace |

| `Nueva Nota` | Crear nota en blanco |
| `Abrir Nota` | Buscar todas las notas |

## Personalizar Comandos

### Cambiar Atajos

1. Abre Configuración (`Cmd+,` o `Ctrl+,`)
2. Navega a Atajos de Teclado
3. Encuentra el comando
4. Haz clic en el atajo actual
5. Presiona tu nuevo atajo
6. Guarda

### Crear Comandos Personalizados

Usuarios avanzados pueden crear comandos personalizados:

1. Abre Configuración
2. Ve a Comandos
3. Haz clic en "Agregar Comando Personalizado"
4. Define el nombre del comando
5. Agrega la(s) acción(es)
6. Establece un atajo (opcional)
