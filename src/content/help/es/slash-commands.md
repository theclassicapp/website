# Comandos con Barra

Los comandos con barra son una forma poderosa de insertar rápidamente formato y contenido especial sin dejar el teclado. Escribe `/` en el editor para abrir la paleta de comandos.

## Comandos Disponibles

### Encabezados

| Comando | Resultado |
| ------- | --------- |
| `/h1` | Encabezado de nivel 1 |
| `/h2` | Encabezado de nivel 2 |
| `/h3` | Encabezado de nivel 3 |
| `/h4` | Encabezado de nivel 4 |
| `/h5` | Encabezado de nivel 5 |
| `/h6` | Encabezado de nivel 6 |

### Formato de Texto

| Comando | Resultado |
| ------- | --------- |
| `/bold` | **Texto en negrita** |
| `/italic` | *Texto en cursiva* |
| `/strikethrough` | ~~Tachado~~ |
| `/code` | Bloque de código en línea |
| `/quote` | Cita en bloque |

### Listas

| Comando | Resultado |
| ------- | --------- |
| `/bullet` | • Lista con viñetas |
| `/numbered` | 1. Lista numerada |
| `/todo` | - [ ] Elemento de tarea |
| `/check` | - [x] Elemento completado |

### Insertar Contenido

| Comando | Resultado |
| ------- | --------- |
| `/image` | Insertar imagen desde URL o archivo |
| `/link` | Insertar hipervínculo |
| `/table` | Insertar tabla |
| `/hr` | Regla horizontal |
| `/code` | Bloque de código con selección de lenguaje |

### Bloques Especiales

| Comando | Resultado |
| ------- | --------- |
| `/mermaid` | Bloque de diagrama Mermaid |
| `/katex` | Bloque matemático KaTeX |
| `/callout` | Bloque de llamada/advertencia |

## Usar Comandos con Barra

1. Escribe `/` en el editor
2. Aparecerá un menú con todos los comandos disponibles
3. Comienza a escribir para filtrar comandos
4. Presiona `Enter` o haz clic para insertar

## Consejos Pro

- **Acceso rápido**: Simplemente escribe `/` y comienza a escribir el nombre del comando
- **Búsqueda difusa**: No necesitas escribir el nombre exacto del comando - Classic encontrará la coincidencia más cercana
- **Navegación con teclado**: Usa las teclas de flecha para navegar el menú
- **Comandos recientes**: Los comandos usados frecuentemente aparecen al inicio de la lista
