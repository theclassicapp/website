# 本地優先架構

Classic 以隱私為基礎打造。您的資料始終留在您的裝置上。

## 「本地優先」是什麼意思？

本地優先意味著：

1. **您的資料儲存在本地**：您所有的筆記、附件和設定都儲存在您電腦上的資料夾中。
2. **不需要雲端**：Classic 可以在沒有網路連線的情況下運作。
3. **您擁有您的資料**：您對您的資料擁有完全的控制權。

## 資料儲存

### 預設位置

您的資料儲存在專用資料夾中：

| 平台 | 位置 |
| ---- | ---- |
| **macOS** | `~/Library/Application Support/Classic/notes` |
| **Windows** | `%APPDATA%\Classic\notes` |
| **Linux** | `~/.config/Classic/notes` |

### 資料格式

- 筆記儲存為純文字檔案（`.md`）
- 附件儲存在 `attachments` 子資料夾中
- 設定儲存在 `settings.json` 中

### 檔案結構

```
notes/
├── 收件匣/
├── 每日筆記/
│   └── 2024-01-15.md
├── 專案/
│   ├── 我的專案.md
│   └── 子筆記.md
└── attachments/
    └── image.png
settings.json
```

## 隱私優點

### 不需要帳號

- 不需要註冊
- 不需要電子郵件驗證
- 不收集個人資訊

### 沒有追蹤

- 沒有分析
- 沒有遙測
- 沒有使用資料發送到外部伺服器

### 可離線運作

- 沒有網路也能運作
- 只有在您選擇時才同步
- 您的資料隨時可存取

## 資料安全

### 您的責任

- **備份**：在設定中設定自動備份
- **匯出**：定期匯出您的資料
- **更新**：保持 Classic 更新以獲得安全性修復

### 我們的責任

- **本地儲存**：我們永遠不會看到或觸碰您的資料
- **沒有後門**：沒有隱藏的資料收集
- **透明度**：對資料處理方式公開誠實

## 同步（選用）

Classic 不包含內建的雲端同步。但是，您可以同步您的資料：

### 使用 iCloud/OneDrive/Dropbox

1. 開啟 Classic 設定
2. 將筆記資料夾位置變更為您的雲端資料夾
3. Classic 會透過您的雲端供應商同步

### 使用 Git

1. 在您的筆記資料夾中初始化 git 儲存庫
2. 手動或透過自動化提交並推送變更

## 資料復原

### 從備份復原

1. 開啟設定
2. 前往備份與還原
3. 選擇備份檔案
4. 點擊還原

### 手動復原

1. 找到您的筆記資料夾
2. 將檔案複製到安全位置
3. 匯入到全新的 Classic 安裝中

## 常見問題

### 我的資料真的私密嗎？

**是的。** 除非您明確使用第三方服務同步，否則您的資料永遠不會離開您的裝置。

### 我可以將資料移到另一台電腦嗎？

**可以。** 只需將您的筆記資料夾複製到新電腦，並在設定中指定 Classic 使用它。

### 如果我解除安裝 Classic 會怎樣？

**您的資料會保留。** 解除安裝 Classic 不會刪除您的筆記資料夾。您可以安全地解除安裝並重新安裝而不會遺失資料。

### Classic 可以存取我的資料嗎？

**不行。** Classic 是一個本地應用程式，沒有用於資料收集的外部連線。
