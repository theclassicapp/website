# 快速擷取

即時捕捉想法而不中斷工作流程。快速擷取從系統的任何地方開啟極簡的筆記視窗。

## 全域快速鍵

按指定的快速鍵開啟快速擷取：

| 平台 | 預設快速鍵 |
| ---- | ---------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

您可以在設定中自訂此快速鍵。

## 使用快速擷取

1. 按全域快速鍵（系統的任何地方）
2. 在彈出視窗中輸入您的筆記
3. 按 `Enter` 儲存
4. 按 `Escape` 取消

## 功能

### 極簡介面

- 小型、專注的視窗
- Markdown 支援
- 自動聚焦輸入
- 鍵盤優先設計

### 快速操作

- `# ` + 空格 = 標題
- `#收件匣` = 新增至收件匣
- `#今天` = 新增至今天的筆記
- 使用 `#標籤名稱` 加入標籤

### 預設目的地

快速擷取的筆記預設會進入您的 **收件匣**。您可以變更此設定：

1. 開啟設定
2. 前往快速擷取
3. 選擇預設目的地

## 提示

### 擷取工作流程

1. **擷取** - 快速記下想法
2. **處理** - 每天檢視收件匣
3. **組織** - 移動到適當的資料夾/筆記

### 最佳實踐

- 保持簡短 - 每次擷取一個想法
- 使用標籤添加情境
- 不用擔心格式 - 之後再完善
- 定期檢視和處理擷取內容

### 整合

快速擷取與以下功能搭配良好：

- **每日筆記**：每天開始時檢視擷取內容
- **收件匣**：作為暫存區
- **標籤**：立即添加情境

## 疑難排解

### 快速鍵無效？

**macOS**
1. 檢查系統偏好設定 > 鍵盤 > 快速鍵
2. 尋找衝突
3. 重新啟動 Classic

**Windows**
1. 以系統管理員身分執行 Classic
2. 檢查是否有快速鍵衝突
3. 重新啟動 Classic

### 視窗未出現？

- 檢查 Classic 是否正在執行
- 確認快速鍵
- 檢查焦點問題
- 重新啟動 Classic

### 擷取內容未儲存？

- 檢查預設目的地
- 確認寫入權限
- 檢查磁碟空間
- 重新啟動 Classic
