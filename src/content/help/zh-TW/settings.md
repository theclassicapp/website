# 設定

自訂 Classic 以完全按照您想要的方式運作。從外觀到編輯器行為，一切都可以配置。

## 開啟設定

### 快速鍵

| 平台 | 快速鍵 |
|------|--------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### 透過選單

- **macOS**：Classic → 設定
- **Windows**：檔案 → 設定
- **Linux**：編輯 → 設定

### 透過命令面板

1. 開啟命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 輸入「設定」
3. 按 Enter

## 設定類別

### 一般

基本應用程式設定：

| 設定 | 說明 |
|------|------|
| **語言** | 介面語言 |
| **佈景主題** | 淺色、深色或系統 |
| **登入時啟動** | 開機時啟動 |
| **檢查更新** | 自動更新行為 |

### 外觀

自訂外觀和風格：

| 設定 | 選項 |
|------|------|
| **佈景主題** | 淺色 / 深色 / 系統 |
| **強調色** | 藍色、紫色、綠色等 |
| **編輯器字型** | 字型系列和大小 |
| **UI 字型** | 介面字型 |
| **等寬字型** | 程式碼字型 |
| **側邊欄寬度** | 200-400px |
| **行高** | 1.0-2.5 |

### 編輯器

配置寫作行為：

| 設定 | 說明 |
|------|------|
| **拼字檢查** | 啟用/停用拼字檢查 |
| **自動儲存** | 自動儲存間隔 |
| **自動換行** | 啟用/停用自動換行 |
| **行號** | 顯示/隱藏行號 |
| **Tab 大小** | 2、4 或 8 個空格 |
| **自動校正** | 智慧引號、自動取代 |

### 版本歷史

控制版本追蹤：

| 設定 | 選項 | 預設值 |
|------|------|--------|
| **啟用歷史** | 開/關 | 開 |
| **保留期間** | 30-180 天 | 90 天 |
| **最大版本數** | 50-500 | 100 |

### 備份與還原

資料保護設定：

| 設定 | 說明 |
|------|------|
| **自動備份** | 啟用自動備份 |
| **備份頻率** | 15分鐘 / 30分鐘 / 1小時 / 4小時 |
| **備份位置** | 自訂路徑 |
| **保留備份數** | 5 / 10 / 20 / 50 |
| **加密備份** | 密碼保護 |

### 鍵盤快速鍵

檢視和自訂快速鍵：

- 查看所有可用的快速鍵
- 自訂任何快速鍵
- 重設為預設值
- 匯入/匯出快速鍵

### 匯入與匯出

資料遷移設定：

- 從其他應用程式匯入
- 匯出格式選項
- 批次操作

### 隱私

隱私和安全設定：

| 設定 | 說明 |
|------|------|
| **分析** | 啟用/停用使用分析 |
| **當機報告** | 發送當機報告 |
| **清除歷史** | 清除搜尋/歷史記錄 |

### 通知

控制通知：

| 設定 | 說明 |
|------|------|
| **啟用通知** | 開/關 |
| **每日筆記提醒** | 提醒時間 |
| **音效** | 通知音效 |

### 進階

進階使用者設定：

| 設定 | 說明 |
|------|------|
| **自訂 CSS** | 加入自訂樣式 |
| **除錯模式** | 啟用除錯日誌 |
| **實驗性功能** | Beta 功能 |
| **資料位置** | 變更筆記資料夾 |

## 搜尋設定

找不到設定？使用搜尋列：

1. 開啟設定
2. 在搜尋框中輸入
3. 符合的設定會出現
4. 點擊以跳到該設定

## 重設設定

### 重設所有設定

恢復為預設值：

1. 開啟設定
2. 捲動到底部
3. 點擊「重設所有設定」
4. 確認操作

### 重設類別

重設特定類別：

1. 前往該類別
2. 點擊「重設為預設值」
3. 只有該類別會重設

## 匯出/匯入設定

### 匯出設定

儲存您的配置：

1. 開啟設定
2. 點擊「匯出設定」
3. 選擇位置
4. 儲存為 JSON 檔案

### 匯入設定

載入已儲存的配置：

1. 開啟設定
2. 點擊「匯入設定」
3. 選擇 JSON 檔案
4. 確認匯入

## 設定位置

設定儲存在本地：

| 平台 | 位置 |
|------|------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## 同步設定（選用）

登入後，設定可以在裝置間同步：

- **同步**：在設定 → 同步中啟用
- **會同步的項目**：佈景主題、字型、快速鍵
- **不會同步的項目**：本地路徑、加密金鑰

## 常見自訂

### 增加字型大小

1. 設定 → 外觀
2. 編輯器字型大小 → 增加
3. 變更立即套用

### 變更佈景主題

1. 設定 → 外觀
2. 佈景主題 → 選擇淺色/深色/系統
3. 應用程式立即更新

### 啟用自動備份

1. 設定 → 備份與還原
2. 啟用「自動備份」
3. 設定頻率和位置

### 自訂快速鍵

1. 設定 → 鍵盤快速鍵
2. 找到操作
3. 點擊當前快速鍵
4. 按下新的按鍵
5. 儲存

## 提示

### 快速存取

- 使用 `Cmd/Ctrl+,` 快速開啟設定
- 依名稱搜尋設定
- 使用類別導航

### 變更前匯出

在重大變更前，匯出您的設定：

- 如果出問題可以輕鬆還原
- 與他人分享配置

### 檢查新設定

更新可能會加入新設定：

- 更新後檢查
- 探索新選項

## 疑難排解

### 設定未儲存

- 檢查檔案權限
- 確認磁碟空間
- 嘗試重設並重新配置

### 設定意外重設

- 可能表示檔案損壞
- 定期匯出設定
- 檢查應用程式衝突

### 找不到設定

- 使用搜尋列
- 檢查不同的類別
- 可能在進階設定中
