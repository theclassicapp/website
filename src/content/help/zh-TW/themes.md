# 佈景主題與外觀

個人化 Classic 的外觀以符合您的風格並減少眼睛疲勞。

## 佈景主題選項

### 淺色模式

Classic 的預設佈景主題特色：

- 乾淨、明亮的介面
- 適合光線充足的環境
- 高對比度以提高可讀性

### 深色模式

適合夜貓子和低光源工作：

- 減少眼睛疲勞
- 在 OLED 螢幕上耗電較低
- 在黑暗環境中對眼睛較友善

### 系統（自動）

跟隨作業系統的佈景主題：

- 根據作業系統設定自動切換
- 無需手動切換
- 與其他應用程式一致

## 切換佈景主題

### 透過設定

1. 開啟設定（`Cmd+,` 或 `Ctrl+,`）
2. 前往 **外觀**
3. 選擇您偏好的佈景主題

### 透過命令面板

1. 開啟命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 輸入「佈景主題」
3. 選擇您偏好的佈景主題

### 快速鍵

| 平台 | 快速鍵 |
| ---- | ------ |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

循環切換：淺色 → 深色 → 系統

## 字型設定

### 編輯器字型

自訂寫作體驗：

1. 開啟設定
2. 前往 **外觀** → **編輯器**
3. 配置字型選項

| 設定 | 選項 | 預設值 |
| ---- | ---- | ------ |
| 字型系列 | 系統字型 + 自訂 | 系統 UI |
| 字型大小 | 12-24px | 16px |
| 行高 | 1.0-2.5 | 1.75 |
| 字元間距 | -2 到 4 | 0 |

### UI 字型

調整介面字型：

1. 開啟設定
2. 前往 **外觀** → **介面**
3. 選擇字型和大小

### 等寬字型

用於程式碼區塊和技術內容：

1. 開啟設定
2. 前往 **外觀** → **程式碼**
3. 選擇等寬字型

## 介面選項

### 側邊欄

自訂側邊欄行為：

| 設定 | 說明 |
| ---- | ---- |
| **自動隱藏** | 不使用時隱藏側邊欄 |
| **寬度** | 調整側邊欄寬度（200-400px） |
| **顯示圖示** | 切換圖示可見性 |

### 編輯器版面配置

選擇您的編輯體驗：

- **分割檢視**：編輯和預覽並排顯示
- **僅預覽**：僅查看渲染輸出
- **僅編輯**：專注於原始 Markdown

### 筆記列表

配置筆記顯示方式：

- **列表檢視**：緊湊，每筆記一行
- **卡片檢視**：帶有片段的預覽卡片
- **樹狀檢視**：階層式資料夾結構

## 強調色

### 可用顏色

選擇您的強調色：

- Classic Blue（預設）
- 紫色
- 綠色
- 橙色
- 紅色
- 粉紅色
- 青色

### 變更強調色

1. 開啟設定
2. 前往 **外觀** → **顏色**
3. 選擇您偏好的強調色

強調色會影響：

- 連結和按鈕
- 選取高亮
- 活動元素
- 焦點指示器

## 輔助功能

### 高對比模式

為了更好的可見性：

1. 開啟設定
2. 前往 **外觀** → **輔助功能**
3. 啟用 **高對比**

### 減少動態效果

最小化動畫：

1. 開啟設定
2. 前往 **外觀** → **輔助功能**
3. 啟用 **減少動態效果**

### 字型調整

- 增加最小字型大小
- 使用閱讀障礙友善字型
- 調整對比度

## 自訂 CSS（進階）

適合想要完全控制權的進階使用者：

1. 開啟設定
2. 前往 **外觀** → **進階**
3. 點擊 **開啟自訂 CSS**
4. 加入您的自訂內容

### 自訂範例

```css
/* 較大的標題 */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* 自訂連結顏色 */
a { color: #3b82f6; }

/* 更寬的行距 */
.prose p { line-height: 2; }
```

## 佈景主題同步

當您登入（選用）時，您的佈景主題偏好會在裝置間同步：

- 自動佈景主題同步
- 字型設定同步
- 自訂 CSS 同步

## 提示

### 配合您的工作流程

- 白天使用 **淺色**，晚上使用 **深色**
- **系統** 模式會自動處理
- 考慮您的環境光線

### 減少眼睛疲勞

- 在低光源環境使用深色模式
- 長時間工作時增加字型大小
- 如需要啟用高對比
- 定期休息

### 生產力

- 學習佈景主題切換快速鍵
- 設定自動切換時間
- 使用系統模式保持一致性
