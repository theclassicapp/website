# 快速入門指南

歡迎使用 Classic！本指南將幫助您在幾分鐘內開始使用 Classic。

## 系統需求

- **macOS**：macOS 12 (Monterey) 或更新版本
- **Windows**：Windows 10 或更新版本
- **Linux**：Ubuntu 20.04+ / Fedora 36+ 或更新版本

## 安裝

1. 從[官方網站](/)下載 Classic
2. 開啟應用程式
3. 開始寫作！

## 首次啟動體驗

當您第一次開啟 Classic 時，您會看到一個簡潔、極致的介面，專為讓您專注於寫作而設計。

![Classic App Screenshot]

### 側邊欄

在左側，您可以找到：
- **每日筆記** - 您的自動日記
- **搜尋** - 快速在所有筆記中尋找任何內容
- **命令面板** - 按 `Cmd/Ctrl+P` (Windows/Linux) 或 `Cmd+Shift+P` (macOS) 開啟命令面板
- **大綱** - 以樹狀結構瀏覽和組織您的筆記

### 編輯器

中央區域是 **Markdown 編輯器**，具有：
- 即時預覽
- 語法高亮
- 豐富的格式化工具

### 建立您的第一個筆記

1. 點擊側邊欄中的 **新增筆記** 按鈕，或按 `Ctrl/Cmd+N` (Windows/Linux) 或 `Cmd+N` (macOS)
2. 為您的筆記命名
3. 使用 Markdown 開始寫作！

```markdown
# 我的第一個筆記

歡迎使用 Classic！這是我記錄想法和靈感的地方。
```
