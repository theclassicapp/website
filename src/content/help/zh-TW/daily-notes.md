# 每日筆記

每日筆記會自動為每一天建立新筆記，非常適合日記、每日規劃或記錄時間基礎的資訊。

## 運作方式

### 自動建立

- 每天會自動建立新筆記
- 筆記使用日期格式命名（例如：`2024-01-15.md`）
- 過往日期的筆記會被保留並可搜尋

### 存取每日筆記

1. 點擊側邊欄中的 **每日筆記**
2. 或按 `Ctrl/Cmd+D`
3. 今天的筆記會自動開啟

## 每日筆記範本

自訂新每日筆記的範本。

### 預設範本

```markdown
# {{date}}

## 任務
- [ ]
- [ ]

## 筆記


## 日記

```

### 自訂範本

1. 開啟設定
2. 前往每日筆記
3. 編輯範本
4. 使用 `{{date}}` 表示當前日期

### 範本變數

| 變數 | 說明 |
| ---- | ---- |
| `{{date}}` | 當前日期（YYYY-MM-DD） |
| `{{time}}` | 當前時間（HH:mm） |
| `{{weekday}}` | 星期名稱（星期一等） |

## 功能

### 日期導航

- **上一天/下一天**：在日期之間導航
- **日曆**：跳到特定日期
- **搜尋**：在所有每日筆記中搜尋內容

### 自動連結

- 同一天建立的筆記會自動連結
- 反向連結顯示日期之間的關聯
- 標籤適用於所有每日筆記

### 整合

- **快速擷取**：使用 `#今天` 新增到今天的筆記
- **日曆檢視**：每日筆記的視覺概覽
- **統計**：追蹤您的寫作連續天數

## 最佳實踐

### 每日日記

```markdown
# 2024-01-15

## 早上
- 6:30 起床
- 感覺：精力充沛

## 任務
- [x] 早上運動
- [ ] 審查 PR
- [ ] 下午 2 點團隊會議

## 學習
- 發現了一個新的快速鍵
- 學習了 Mermaid 圖表

## 明天
- 專注於文件
- 安排程式碼審查
```

### 會議筆記

```markdown
# 2024-01-15

## 團隊站會

**與會者**：Alice、Bob、Carol

### 更新
- Alice：完成 API 整合
- Bob：開發 UI 元件
- Carol：測試進行中

### 阻礙
- 等待設計審查

### 行動項目
- [ ] Alice：部署到測試環境
- [ ] Bob：完成元件庫
- [ ] Carol：撰寫整合測試
```

### 學習日誌

```markdown
# 2024-01-15

## 今日所學

### Mermaid 圖表
- 可以在 Markdown 中建立流程圖
- 語法很直接
- 即時預覽很有幫助

### 資源
- [Mermaid 文件](https://mermaid.js.org/)

## 問題
- 如何建立複雜圖表？
- 圖表組織的最佳實踐？
```

## 提示

### 一致性

- 每天在相同時間建立筆記
- 使用一致的格式
- 全天檢視和更新

### 組織

- 使用標題區分不同區塊
- 加入標籤便於搜尋
- 連結到相關筆記

### 檢視

- 每週檢視每日筆記
- 將重要項目提取到永久筆記
- 封存或彙整舊筆記
