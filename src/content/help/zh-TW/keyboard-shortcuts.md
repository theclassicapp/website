# 鍵盤快速鍵

使用 Classic 的鍵盤快速鍵加快工作速度。所有快速鍵都可以在設定中自訂。

## 全域快速鍵

這些快速鍵在應用程式的任何地方都有效。

### 一般

| 快速鍵（Windows/Linux） | 快速鍵（macOS） | 操作 |
| ---------------------- | --------------- | ---- |
| `Ctrl+N` | `Cmd+N` | 新增筆記 |
| `Ctrl+P` | `Cmd+P` | 命令面板 |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | 全域快速擷取 |
| `Ctrl+S` | `Cmd+S` | 儲存筆記 |
| `Ctrl+F` | `Cmd+F` | 搜尋 |
| `Ctrl+,` | `Cmd+,` | 設定 |

### 導航

| 快速鍵（Windows/Linux） | 快速鍵（macOS） | 操作 |
| ---------------------- | --------------- | ---- |
| `Ctrl+1` | `Cmd+1` | 前往第一個筆記 |
| `Ctrl+2` | `Cmd+2` | 前往第二個筆記 |
| `Ctrl+Left` | `Cmd+Left` | 向後導航 |
| `Ctrl+Right` | `Cmd+Right` | 向前導航 |
| `Ctrl+Up` | `Cmd+Up` | 前往父筆記 |
| `Ctrl+Down` | `Cmd+Down` | 展開/聚焦第一個子筆記 |

### 編輯器

| 快速鍵（Windows/Linux） | 快速鍵（macOS） | 操作 |
| ---------------------- | --------------- | ---- |
| `Ctrl+B` | `Cmd+B` | 粗體 |
| `Ctrl+I` | `Cmd+I` | 斜體 |
| `Ctrl+K` | `Cmd+K` | 插入連結 |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | 刪除線 |
| `Ctrl+`` | `Cmd+`` | 行內程式碼 |
| `Ctrl+Shift+C` | `Cmd+Shift+C` | 程式碼區塊 |
| `Ctrl+/` | `Cmd+/` | 斜線命令 |
| `Tab` | `Tab` | 縮排 |
| `Shift+Tab` | `Shift+Tab` | 取消縮排 |

### 視窗管理

| 快速鍵（Windows/Linux） | 快速鍵（macOS） | 操作 |
| ---------------------- | --------------- | ---- |
| `Ctrl+W` | `Cmd+W` | 關閉當前筆記 |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | 關閉所有筆記 |
| `F11` | `F11` | 切換全螢幕 |
| `Ctrl+M` | `Cmd+M` | 切換側邊欄 |

## 快速擷取

| 快速鍵（Windows/Linux） | 快速鍵（macOS） | 操作 |
| ---------------------- | --------------- | ---- |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | 開啟快速擷取 |
| `Escape` | `Escape` | 關閉快速擷取 |
| `Ctrl+Enter` | `Cmd+Enter` | 儲存並關閉 |

## 自訂快速鍵

1. 開啟 **設定**（`Ctrl+,` 或 `Cmd+,`）
2. 前往 **鍵盤快速鍵**
3. 點擊任何快速鍵以變更
4. 按下新的組合鍵
5. 點擊 **儲存**

## 提示

- 大多數快速鍵都可以自訂
- 與系統快速鍵的衝突會被高亮顯示
- 點擊每個快速鍵旁的重設按鈕可恢復預設值
