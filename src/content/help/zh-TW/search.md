# 搜尋

使用 Classic 強大的搜尋功能即時找到任何筆記。全文搜尋、篩選器和運算子幫助您精確找到所需內容。

## 基本搜尋

### 開啟搜尋

| 平台 | 快速鍵 |
|------|--------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

或點擊側邊欄中的搜尋列。

### 運作方式

1. 開啟搜尋
2. 開始輸入
3. 結果即時顯示
4. 點擊以開啟筆記

## 搜尋功能

### 全文搜尋

Classic 會搜尋：

- 筆記標題
- 筆記內容
- 標籤
- 檔案名稱

### 即時結果

結果會在您輸入時更新：

- 無需按 Enter
- 即時回饋
- 啟用模糊匹配

### 搜尋高亮

找到的詞彙會在結果中高亮顯示：

- 查看匹配項周圍的情境
- 快速識別相關筆記

## 搜尋運算子

### 精確匹配

使用引號表示精確詞組：

```
"專案規劃"
"會議筆記"
```

### 標籤搜尋

尋找帶有特定標籤的筆記：

```
tag:工作
tag:會議
tag:緊急
```

### 排除詞彙

使用 `-` 排除：

```
專案 -已封存
會議 -已取消
```

### OR 搜尋

尋找帶有任何詞彙的筆記：

```
專案 OR 計畫
```

### 日期範圍

依日期搜尋：

```
created:2024-01-01..2024-12-31
modified:上週
```

### 檔案類型

依副檔名篩選：

```
ext:md
ext:txt
```

## 進階搜尋

### 組合運算子

建立複雜查詢：

```
tag:工作 -tag:已封存 created:上週
"會議" tag:專案-alpha
```

### 路徑搜尋

在資料夾內搜尋：

```
path:專案/
path:封存/
```

### 標題搜尋

僅搜尋標題：

```
title:路線圖
title:會議
```

### 內容搜尋

僅搜尋內容（排除標題）：

```
content:重要
content:截止日
```

## 搜尋篩選器

### 依標籤

1. 輸入您的搜尋
2. 點擊標籤以篩選
3. 結果縮小到帶有標籤的筆記

### 依日期

依修改日期篩選：

- 今天
- 本週
- 本月
- 自訂範圍

### 依資料夾

限制搜尋到資料夾：

1. 右鍵點擊資料夾
2. 選擇「在資料夾中搜尋」
3. 結果限制在該資料夾

## 搜尋結果

### 排序

依以下方式排序結果：

- **相關性**（預設）
- **修改日期**（最新優先）
- **建立日期**（最新優先）
- **標題**（字母順）

### 結果預覽

懸停在結果上以預覽：

- 顯示前幾行
- 高亮匹配詞彙
- 顯示標籤

### 情境

查看匹配發生的位置：

- 顯示周圍文字
- 高亮匹配項
- 點擊跳到該位置

## 搜尋歷史

### 最近搜尋

存取之前的搜尋：

1. 點擊搜尋列
2. 查看最近搜尋
3. 點擊以重新執行

### 清除歷史

1. 開啟設定
2. 前往隱私
3. 點擊「清除搜尋歷史」

## 提示

### 使用具體詞彙

更具體 = 更好的結果：

```
✅ "Q4 行銷預算"
❌ 預算
```

### 與標籤組合

快速縮小結果：

```
專案 tag:緊急
```

### 使用排除

過濾雜訊：

```
會議 -tag:已封存
```

### 儲存常用搜尋

建立已儲存的搜尋：

1. 輸入您的搜尋
2. 點擊「儲存搜尋」
3. 為其命名
4. 從側邊欄存取

## 鍵盤導航

| 操作 | 快速鍵 |
|------|--------|
| 開啟搜尋 | `Cmd/Ctrl+F` |
| 下一個結果 | `Enter` 或 `Down` |
| 上一個結果 | `Shift+Enter` 或 `Up` |
| 關閉搜尋 | `Escape` |

## 搜尋設定

### 在設定中自訂

| 設定 | 選項 |
|------|------|
| 區分大小寫 | 開/關 |
| 搜尋標題 | 開/關 |
| 搜尋內容 | 開/關 |
| 搜尋標籤 | 開/關 |
| 包含已封存 | 開/關 |

## 使用案例

### 尋找舊筆記

```
"會議" date:去年
```

### 專案研究

```
path:專案/ tag:研究
```

### 行動項目

```
"TODO" OR "FIXME" -tag:完成
```

### 最近工作

```
modified:本週
```

## 疑難排解

### 找不到結果

- 檢查拼字
- 移除篩選器
- 嘗試更廣泛的詞彙
- 檢查筆記是否已封存

### 搜尋緩慢

- 大量筆記集合可能需要較長時間
- 停用區分大小寫
- 減少搜尋範圍

### 筆記遺失

- 筆記可能在不同的資料夾
- 檢查標籤是否正確
- 確認筆記未被封存
