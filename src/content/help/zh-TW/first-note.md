# 建立您的第一個筆記

開始使用 Classic，建立您的第一個筆記。既簡單又直觀。

## 建立新筆記

### 使用側邊欄

1. 點擊側邊欄中的 **+** 按鈕
2. 新筆記即建立
3. 開始輸入內容

### 使用快速鍵

| 平台 | 快速鍵 |
| ---- | ------ |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### 使用命令面板

1. 開啟命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 輸入「新增筆記」
3. 按 Enter

## 撰寫筆記

### 基本編輯

直接開始輸入！Classic 會在您撰寫時自動儲存。

### 新增標題

筆記的第一行會成為標題。使用標題以獲得最佳效果：

```markdown
# 我的第一個筆記

這是我筆記的內容。
```

### 格式化文字

使用 Markdown 格式化文字：

- **粗體**：`**文字**` 或 `Cmd/Ctrl+B`
- **斜體**：`*文字*` 或 `Cmd/Ctrl+I`
- **刪除線**：`~~文字~~`
- **程式碼**：`` `程式碼` ``

## 儲存

Classic 會在您輸入時自動儲存筆記。無需手動儲存。

### 手動儲存

如有需要，您可以強制儲存：

- **macOS**：`Cmd+S`
- **Windows/Linux**：`Ctrl+S`

## 組織筆記

### 新增標籤

為筆記加上標籤以便日後尋找：

1. 輸入 `#` 後接標籤名稱
2. 或使用工具列中的標籤按鈕
3. 支援多個標籤

範例：

```markdown
# 會議筆記

討論 Q4 規劃 #工作 #會議
```

### 移動到資料夾

1. 在側邊欄中右鍵點擊筆記
2. 選擇「移動到資料夾」
3. 選擇或建立目標資料夾

### 釘選重要筆記

1. 右鍵點擊筆記
2. 選擇「釘選到頂部」
3. 筆記會出現在列表頂部

## 預覽筆記

查看格式化後的筆記外觀：

- 點擊 **預覽** 按鈕
- 或使用 `Cmd/Ctrl+Shift+P`
- 在編輯和預覽模式之間切換

## 下一步

現在您已經建立了第一個筆記：

- 進一步了解 [Markdown](/help/markdown)
- 使用 [標籤](/help/tags) 組織
- 設定 [每日筆記](/help/daily-notes)
- 探索 [快速鍵](/help/keyboard-shortcuts)

## 提示

### 筆記標題

- 保持標題具描述性
- 第一行 = 筆記標題
- 使用 H1（`#`）以清晰區分

### 自動儲存

- 永遠不用擔心遺失工作
- 變更會自動儲存
- 查看狀態列了解儲存狀態

### 快速擷取

需要快速記下某些內容？

- 使用快速擷取（`Cmd/Ctrl+Shift+N`）
- 即時建立筆記
- 非常適合稍縱即逝的想法
