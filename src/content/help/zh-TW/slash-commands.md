# 斜線命令

斜線命令是快速插入格式化和特殊內容的強大方式，無需離開鍵盤。在編輯器中輸入 `/` 開啟命令面板。

## 可用命令

### 標題

| 命令 | 結果 |
| ---- | ---- |
| `/h1` | 一級標題 |
| `/h2` | 二級標題 |
| `/h3` | 三級標題 |
| `/h4` | 四級標題 |
| `/h5` | 五級標題 |
| `/h6` | 六級標題 |

### 文字格式

| 命令 | 結果 |
| ---- | ---- |
| `/bold` | **粗體文字** |
| `/italic` | *斜體文字* |
| `/strikethrough` | ~~刪除線~~ |
| `/code` | 行內程式碼區塊 |
| `/quote` | 區塊引言 |

### 列表

| 命令 | 結果 |
| ---- | ---- |
| `/bullet` | • 項目符號列表 |
| `/numbered` | 1. 編號列表 |
| `/todo` | - [ ] 待辦事項 |
| `/check` | - [x] 已完成項目 |

### 插入內容

| 命令 | 結果 |
| ---- | ---- |
| `/image` | 從 URL 或檔案插入圖片 |
| `/link` | 插入超連結 |
| `/table` | 插入表格 |
| `/hr` | 水平分隔線 |
| `/code` | 帶語言選擇的程式碼區塊 |

### 特殊區塊

| 命令 | 結果 |
| ---- | ---- |
| `/mermaid` | Mermaid 圖表區塊 |
| `/katex` | KaTeX 數學區塊 |
| `/callout` | 提示/警告區塊 |

## 使用斜線命令

1. 在編輯器中輸入 `/`
2. 會出現包含所有可用命令的選單
3. 開始輸入以篩選命令
4. 按 `Enter` 或點擊以插入

## 專業提示

- **快速存取**：只需輸入 `/` 並開始輸入命令名稱
- **模糊搜尋**：您不需要輸入確切的命令名稱 - Classic 會找到最接近的匹配
- **鍵盤導航**：使用方向鍵導航選單
- **最近使用**：經常使用的命令會出現在列表頂部
