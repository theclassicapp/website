# 命令面板

命令面板是您存取 Classic 中每個操作的入口。按簡單的快速鍵即可即時存取任何功能。

## 開啟命令面板

| 平台 | 快速鍵 |
| ---- | ------ |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## 如何使用

1. 開啟命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 輸入以搜尋命令
3. 使用方向鍵導航
4. 按 `Enter` 執行

## 命令類別

### 檔案操作

- **新增筆記**：建立新筆記
- **新增每日筆記**：建立/開啟今天的每日筆記
- **開啟筆記**：搜尋並開啟現有筆記
- **儲存筆記**：儲存當前筆記
- **刪除筆記**：刪除當前筆記
- **重新命名筆記**：重新命名當前筆記

### 導航

- **前往筆記**：跳到任何筆記
- **前往每日筆記**：開啟特定日期
- **前往標籤**：依標籤瀏覽筆記
- **返回**：在歷史記錄中向後導航
- **前進**：在歷史記錄中向前導航

### 編輯器

- **切換預覽**：在編輯和預覽模式之間切換
- **切換側邊欄**：顯示/隱藏側邊欄
- **專注模式**：進入無干擾寫作
- **插入日期**：加入當前日期/時間
- **插入範本**：插入範本

### 格式化

- **粗體**：將文字設為粗體
- **斜體**：將文字設為斜體
- **刪除線**：為文字加上刪除線
- **程式碼**：格式化為程式碼
- **連結**：插入連結
- **標題**：套用標題格式

### 組織

- **新增標籤**：為當前筆記加上標籤
- **移至資料夾**：將筆記移至資料夾
- **封存筆記**：封存當前筆記
- **釘選筆記**：將筆記釘選到列表頂部

### 檢視

- **切換佈景主題**：在淺色/深色之間切換
- **放大**：增加字型大小
- **縮小**：減少字型大小
- **重設縮放**：重設為預設大小

## 提示

### 快速存取

- 命令支援模糊搜尋
- 最近使用的命令會優先顯示
- 可釘選常用命令

### 鍵盤導航

- `↑`/`↓` - 導航結果
- `Enter` - 執行命令
- `Escape` - 關閉面板
- `Tab` - 切換類別

### 自訂命令

1. 開啟設定
2. 前往命令面板
3. 新增自訂命令
4. 定義快速鍵

## 常用命令

以下是您會經常使用的一些命令：

| 命令 | 功能 |
| ---- | ---- |
| `新增筆記` | 建立空白筆記 |
| `開啟筆記` | 搜尋所有筆記 |
| `新增每日筆記` | 今天的日記 |
| `切換預覽` | 預覽 vs 編輯 |
| `新增標籤` | 為當前筆記加上標籤 |
| `快速擷取` | 全域擷取 |

## 自訂命令

### 變更快速鍵

1. 開啟設定（`Cmd+,` 或 `Ctrl+,`）
2. 前往鍵盤快速鍵
3. 找到命令
4. 點擊當前快速鍵
5. 按下新的快速鍵
6. 儲存

### 建立自訂命令

進階使用者可以建立自訂命令：

1. 開啟設定
2. 前往命令
3. 點擊「新增自訂命令」
4. 定義命令名稱
5. 加入操作
6. 設定快速鍵（選用）
