# 標籤

標籤是在 Classic 中組織和分類筆記的強大方式。即時找到相關內容。

## 建立標籤

### 行內標籤

直接在筆記內容中加入標籤：

```markdown
# 會議筆記

討論 Q4 路線圖 #工作 #會議 #規劃
```

### 使用標籤按鈕

1. 點擊工具列中的標籤圖示
2. 輸入標籤名稱
3. 按 Enter

### 快速鍵

| 平台 | 快速鍵 |
|------|--------|
| **macOS** | `Cmd+T` |
| **Windows/Linux** | `Ctrl+T` |

## 標籤語法

### 基本標籤

```markdown
#專案 #工作 #重要
```

### 巢狀標籤

使用 `/` 建立標籤階層：

```markdown
#工作/會議
#工作/專案
#個人/興趣/閱讀
```

### 多字標籤

使用連字號或底線：

```markdown
#專案-alpha
#會議筆記
#q4-規劃
```

## 管理標籤

### 查看所有標籤

1. 開啟側邊欄
2. 找到「標籤」區塊
3. 查看所有標籤

### 標籤雲

視覺化最常用的標籤：

- 較大的標籤 = 較多筆記
- 點擊任何標籤以篩選
- 查看每個標籤的筆記數量

### 重新命名標籤

1. 右鍵點擊標籤
2. 選擇「重新命名」
3. 輸入新名稱
4. 所有筆記會自動更新

### 刪除標籤

1. 右鍵點擊標籤
2. 選擇「刪除」
3. 確認刪除
4. 標籤會從所有筆記中移除

## 依標籤搜尋

### 標籤面板

點擊側邊欄中的標籤以篩選筆記。

### 搜尋語法

在搜尋中使用 `tag:`：

```
tag:工作
tag:工作/會議
```

### 組合標籤

尋找帶有多個標籤的筆記：

```
tag:工作 tag:緊急
```

### 排除標籤

尋找不帶有某標籤的筆記：

```
-tag:已封存
```

## 標籤最佳實踐

### 保持簡單

- 每個筆記使用 3-5 個標籤
- 命名保持一致
- 避免標籤氾濫

### 使用階層

組織相關標籤：

```
#工作
  #工作/會議
  #工作/專案
  #工作/截止日

#個人
  #個人/健康
  #個人/財務
```

### 建立慣例

為自己定義標籤標準：

- **狀態**：`#待辦` `#進行中` `#完成`
- **優先級**：`#緊急` `#高` `#低`
- **類型**：`#會議` `#參考` `#想法`

## 標籤建議

當您輸入 `#` 時，Classic 會建議現有標籤：

- 最近使用的標籤會優先顯示
- 支援模糊匹配
- 按 Tab 接受建議

## 標籤統計

在設定中查看標籤使用情況：

- 最常用的標籤
- 未使用的標籤
- 標籤隨時間的趨勢

## 使用案例

### 專案管理

```markdown
# 專案啟動 #專案-alpha #會議 #2024-01-15

與會者：Alpha 團隊
狀態：#待辦
```

### 知識庫

```markdown
# JavaScript 閉包 #程式設計 #javascript #參考 #進階

閉包是可以存取...的函數
```

### 每日筆記

```markdown
# 2024-01-15

- 早上站會 #工作 #會議
- 程式碼審查 #工作 #開發
- 健身房 #個人 #健康
```

### 閱讀筆記

```markdown
# 原子習慣筆記 #閱讀 #書籍 #生產力

書中的關鍵要點...
```

## 快速鍵

| 操作 | 快速鍵 |
|------|--------|
| 新增標籤 | `Cmd/Ctrl+T` |
| 搜尋標籤 | `Cmd/Ctrl+F`，然後輸入 `tag:` |
| 顯示標籤面板 | `Cmd/Ctrl+Shift+T` |

## 提示

### 一致的命名

堅持一種慣例：

```
✅ #會議筆記
✅ #會議_筆記
❌ #Meeting Notes（不允許空格）
❌ #會議筆記（不一致）
```

### 避免過度標記

太多標籤會變得難以管理：

```
✅ #工作 #緊急 #專案-alpha
❌ #工作 #緊急 #重要 #今天 #星期一 #辦公室 #桌子 #電腦
```

### 定期清理

定期檢視和整合標籤：

- 合併相似標籤
- 移除未使用的標籤
- 更新慣例

## 進階功能

### 基於標籤的智慧資料夾

建立根據標籤自動填充的資料夾：

1. 建立新資料夾
2. 設定篩選條件：`tag:專案`
3. 所有帶有該標籤的筆記會出現

### 在 YAML 中使用標籤

在筆記的前言中加入標籤：

```yaml
---
tags:
  - 工作
  - 會議
  - 專案-alpha
---
```

### 標籤別名

在設定中設定標籤別名：

- `#待辦` = `#任務` = `#行動`
- 整合相似標籤
