# 語言與本地化

Classic 支援您的語言。使用您偏好的語言操作應用程式，並以您選擇的任何語言撰寫筆記。

## 支援的語言

Classic 的介面提供多種語言版本：

| 語言 | 代碼 |
| ---- | ---- |
| English | `en` |
| 簡體中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| 更多... | |

## 變更介面語言

### 透過設定

1. 開啟設定（`Cmd+,` 或 `Ctrl+,`）
2. 前往 **一般** → **語言**
3. 選擇您偏好的語言
4. 重新啟動 Classic 以套用

### 透過命令面板

1. 開啟命令面板（`Cmd+P` 或 `Ctrl+P`）
2. 輸入「語言」
3. 選擇您偏好的語言
4. 重新啟動以套用變更

## 多語言寫作

Classic 支援以任何語言撰寫筆記：

### Unicode 支援

- 完整的 Unicode 支援
- 支援 RTL（由右至左）語言
- CJK（中文、日文、韓文）字元完美運作
- Emoji 支援

### 語言偵測

Classic 可以偵測您筆記的語言：

- 拼字檢查自動偵測
- 每筆記語言設定
- 混合語言支援

## 拼字檢查

### 啟用拼字檢查

1. 開啟設定
2. 前往 **編輯器** → **拼字檢查**
3. 啟用「拼字檢查」

### 拼字檢查語言

1. 開啟設定
2. 前往 **編輯器** → **拼字檢查**
3. 點擊「語言」
4. 選擇要檢查的語言

### 將單字加入字典

1. 右鍵點擊標記的單字
2. 選擇「加入字典」
3. 單字不會再被標記

## RTL 支援

針對阿拉伯文、希伯來文和其他 RTL 語言：

### 自動 RTL 偵測

Classic 會自動偵測 RTL 內容：

- 文字方向自動調整
- 游標移動適應
- 對齊方式變更

### 手動 RTL 切換

強制 RTL 或 LTR 方向：

1. 開啟命令面板
2. 輸入「RTL」或「LTR」
3. 選擇您的偏好

## 輸入法

Classic 與您系統的輸入法搭配運作：

### CJK 輸入法

- 與拼音、注音等搭配運作
- 支援日文 IME
- 支援韓文輸入法

### 特殊字元

輕鬆插入特殊字元：

- 使用您系統的字元選擇器
- 從字元表複製/貼上
- 使用 Unicode 輸入

## 日期與時間格式

日期會根據您的語言環境調整：

### 格式選項

| 設定 | 選項 |
| ---- | ---- |
| 日期格式 | MM/DD/YYYY、DD/MM/YYYY、YYYY-MM-DD |
| 時間格式 | 12 小時制、24 小時制 |
| 一週的第一天 | 星期日、星期一、星期六 |

在設定 → 一般 → 日期與時間中配置

## 數字格式

數字根據您的語言環境顯示：

- **小數分隔符號**：`.` 或 `,`
- **千位分隔符號**：`,` 或 `.` 或空格
- **貨幣符號**：根據地區

## 鍵盤配置

Classic 與任何鍵盤配置搭配運作：

- QWERTY、AZERTY、QWERTZ
- Dvorak、Colemak
- 自訂配置

## 本地化設定

### 在設定 → 一般 → 語言中

| 設定 | 說明 |
| ---- | ---- |
| **介面語言** | 應用程式 UI 語言 |
| **日期格式** | 日期顯示方式 |
| **時間格式** | 12 或 24 小時制 |
| **一週的第一天** | 日曆起始日 |
| **數字格式** | 小數/千位分隔符號 |

## 提示

### 混合語言筆記

在一個筆記中以多種語言撰寫：

```markdown
# 國際專案

English: Welcome to the project
中文: 歡迎來到這個專案
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### 特定語言字型

某些語言使用特定字型效果更好：

1. 開啟設定
2. 前往外觀 → 字型
3. 為不同的文字系統設定字型

### 快速切換語言

建立快速鍵：

1. 開啟設定 → 快速鍵
2. 找到「切換語言」
3. 設定您偏好的快速鍵

## 疑難排解

### 語言無法使用

- 檢查應用程式更新
- 語言可能正在開發中
- 在 GitHub 上貢獻翻譯

### 字元無法顯示

- 安裝必要的字型
- 檢查字型設定
- 嘗試不同的字型

### 拼字檢查無效

- 確認拼字檢查已啟用
- 將語言加入拼字檢查
- 重新啟動 Classic

### RTL 無法正常運作

- 確保 RTL 偵測已開啟
- 嘗試手動 RTL 切換
- 檢查混合內容問題
