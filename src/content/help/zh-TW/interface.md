# 了解介面

Classic 的介面設計簡潔、直觀，不會干擾您的工作。以下是主要元件的導覽。

## 主要版面配置

Classic 採用三欄式版面配置：

1. **側邊欄**（左側）- 導航和筆記組織
2. **編輯器**（中間）- 您的寫作空間
3. **預覽/屬性**（右側）- 選用面板

## 側邊欄

側邊欄包含幾個重要區塊：

### 筆記列表

- **所有筆記**：在同一列表中查看所有筆記
- **每日筆記**：快速存取今日日記
- **收藏**：將常用筆記加上星號
- **資料夾**：以階層方式組織筆記

### 導航

- **搜尋列**：即時尋找筆記
- **標籤**：按標籤瀏覽
- **反向連結**：查看筆記之間的關聯

### 操作

- **新增筆記**：建立新筆記
- **新增資料夾**：使用資料夾組織
- **匯入**：引入外部內容

## 編輯器

編輯器是您撰寫和格式化筆記的地方。

### 功能

- **Markdown 支援**：完整的 Markdown 語法
- **即時預覽**：輸入時即顯示格式化文字
- **語法高亮**：程式碼區塊會被高亮顯示
- **斜線命令**：輸入 `/` 快速執行操作

### 編輯器模式

| 模式 | 說明 |
|------|------|
| **僅編輯** | 查看原始 Markdown |
| **僅預覽** | 查看渲染輸出 |
| **分割檢視** | 編輯和預覽並排顯示 |

### 工具列

快速格式化操作：

- 粗體、斜體、刪除線
- 標題（H1-H6）
- 列表（項目符號、編號、任務）
- 連結和圖片
- 程式碼區塊
- 表格

## 狀態列

在編輯器底部：

- **字數/字元數**：追蹤您的寫作
- **上次儲存**：筆記上次儲存的時間
- **筆記路徑**：筆記所在位置
- **同步狀態**：若使用選用的同步功能

## 右鍵選單

右鍵點擊以取得情境相關的操作選項。

### 筆記右鍵選單

- 重新命名
- 移動到資料夾
- 複製
- 刪除
- 匯出
- 複製連結

### 編輯器右鍵選單

- 剪下、複製、貼上
- 格式選項
- 插入連結/圖片
- 尋找和取代

## 鍵盤導航

使用鍵盤導航介面：

| 操作 | 快速鍵 |
|------|--------|
| 切換側邊欄 | `Cmd/Ctrl + M` |
| 聚焦搜尋 | `Cmd/Ctrl + F` |
| 命令面板 | `Cmd/Ctrl + P` |
| 新增筆記 | `Cmd/Ctrl + N` |

## 自訂設定

調整介面以符合您的喜好：

### 側邊欄寬度

1. 將滑鼠懸停在側邊欄邊緣
2. 拖曳以調整大小
3. 或在設定中設定確切寬度

### 佈景主題

在淺色和深色模式之間切換：

- 使用標題中的佈景主題切換按鈕
- 或設定為跟隨系統偏好

### 字型大小

1. 開啟設定
2. 前往外觀
3. 調整編輯器字型大小

## 提示

### 專注模式

隱藏所有面板以進行無干擾寫作：

1. 開啟命令面板
2. 輸入「專注模式」
3. 按 Enter

### 快速存取

- 將常用筆記釘選到收藏
- 使用快速鍵執行常見操作
- 在設定中自訂工具列
