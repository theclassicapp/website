# 備份與還原

使用 Classic 內建的備份系統保護您的筆記安全。再也不會遺失您的工作。

## 自動備份

### 運作方式

Classic 會定期自動建立筆記的備份：

- **預設頻率**：每 30 分鐘
- **備份位置**：與筆記相同的資料夾
- **保留**：保留最近 10 個備份
- **格式**：壓縮封存檔（`.cbak`）

### 配置自動備份

1. 開啟設定
2. 前往 **備份與還原**
3. 配置以下選項

| 設定 | 選項 | 預設值 |
| ---- | ---- | ------ |
| 啟用自動備份 | 開/關 | 開 |
| 備份頻率 | 15分鐘 / 30分鐘 / 1小時 / 4小時 | 30分鐘 |
| 保留備份數 | 5 / 10 / 20 / 50 | 10 |
| 備份位置 | 自訂路徑 | 筆記資料夾 |

## 手動備份

### 立即建立備份

1. 開啟命令面板（`Cmd/Ctrl+P`）
2. 輸入「備份」
3. 選擇 **立即建立備份**
4. 選擇位置（選用）
5. 備份即建立

### 備份檔案格式

備份儲存為 `.cbak` 檔案，包含：

- 所有筆記（Markdown 檔案）
- 附件
- 設定
- 元資料（標籤、連結等）

## 從備份還原

### 還原程序

1. 開啟設定
2. 前往 **備份與還原**
3. 點擊 **從備份還原**
4. 選擇備份檔案
5. 選擇還原選項
6. 確認還原

### 還原選項

| 選項 | 說明 |
| ---- | ---- |
| **全部取代** | 用備份完全取代當前筆記 |
| **合併** | 將備份與現有筆記合併 |
| **預覽** | 還原前查看備份內容 |

### 還原內容

- ✅ 所有筆記和資料夾
- ✅ 附件
- ✅ 標籤和元資料
- ✅ 筆記歷史
- ⚠️ 設定（選用）
- ❌ 應用程式偏好設定

## 備份位置

### 預設位置

```
~/Classic/backups/
├── backup-2024-01-15-10-30.cbak
├── backup-2024-01-15-11-00.cbak
└── backup-2024-01-15-11-30.cbak
```

### 自訂位置

1. 開啟設定
2. 前往備份與還原
3. 點擊「變更備份位置」
4. 選擇您偏好的資料夾

### 雲端備份

將備份位置指向雲端同步的資料夾：

- **iCloud**：`~/Library/Mobile Documents/com~apple~CloudDocs/Classic Backups`
- **Dropbox**：`~/Dropbox/Classic Backups`
- **Google Drive**：`~/Google Drive/Classic Backups`
- **OneDrive**：`~/OneDrive/Classic Backups`

## 最佳實踐

### 備份排程

- **自動備份**：保持啟用，間隔 30 分鐘
- **手動備份**：在重大變更前建立
- **異地備份**：定期將備份複製到外部儲存

### 備份驗證

定期驗證您的備份：

1. 建立測試還原
2. 檢查筆記內容
3. 確認附件
4. 確認元資料

### 備份輪替

- 在本地保留最近的備份
- 將較舊的備份封存到外部儲存
- 刪除非常舊的備份以節省空間

## 疑難排解

### 備份失敗

**可能原因**

- 磁碟空間不足
- 權限問題
- 檔案正在使用中

**解決方案**

1. 檢查可用磁碟空間
2. 確認資料夾權限
3. 關閉其他應用程式
4. 嘗試不同的位置

### 還原失敗

**可能原因**

- 備份檔案損壞
- 版本不相容
- 空間不足

**解決方案**

1. 嘗試較舊的備份
2. 檢查備份檔案完整性
3. 釋放磁碟空間
4. 聯繫支援

### 備份遺失

**可能原因**

- 自動備份已停用
- 備份位置已變更
- 備份已刪除

**解決方案**

1. 檢查備份設定
2. 確認備份位置
3. 啟用自動備份
4. 立即建立手動備份

## 進階選項

### 加密備份

為了增加安全性：

1. 開啟設定
2. 前往備份與還原
3. 啟用「加密備份」
4. 設定密碼
5. 請記住此密碼 - 無法復原

### 增量備份

使用增量備份節省空間：

- 僅儲存自上次備份以來的變更
- 完整還原可重建完整狀態
- 減少備份時間和大小

### 備份排除

從備份中排除特定內容：

1. 開啟設定
2. 前往備份與還原
3. 點擊「排除項目」
4. 新增要排除的資料夾或檔案
