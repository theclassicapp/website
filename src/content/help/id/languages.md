# Bahasa & Lokalisasi

Classic berbicara bahasa Anda. Gunakan aplikasi dalam bahasa pilihan Anda dan tulis catatan dalam bahasa apa pun yang Anda pilih.

## Bahasa yang Didukung

Antarmuka Classic tersedia dalam beberapa bahasa:

| Bahasa | Kode |
| ------ | ---- |
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| Dan lainnya... | |

## Mengubah Bahasa Antarmuka

### Melalui Pengaturan

1. Buka Pengaturan (`Cmd+,` atau `Ctrl+,`)
2. Buka **Umum** → **Bahasa**
3. Pilih bahasa pilihan Anda
4. Restart Classic untuk menerapkan

### Melalui Palet Perintah

1. Buka Palet Perintah (`Cmd+P` atau `Ctrl+P`)
2. Ketik "Bahasa"
3. Pilih bahasa pilihan Anda
4. Restart untuk menerapkan perubahan

## Menulis dalam Beberapa Bahasa

Classic mendukung menulis catatan dalam bahasa apa pun:

### Dukungan Unicode

- Dukungan Unicode penuh
- Bahasa RTL (Kanan-ke-Kiri) didukung
- Karakter CJK (China, Jepang, Korea) berfungsi sempurna
- Dukungan emoji

### Deteksi Bahasa

Classic dapat mendeteksi bahasa catatan Anda:

- Auto-detect untuk pemeriksaan ejaan
- Pengaturan bahasa per-catatan
- Dukungan bahasa campuran

## Pemeriksaan Ejaan

### Mengaktifkan Pemeriksaan Ejaan

1. Buka Pengaturan
2. Buka **Editor** → **Pemeriksaan Ejaan**
3. Aktifkan "Pemeriksaan Ejaan"

### Bahasa Pemeriksaan Ejaan

1. Buka Pengaturan
2. Buka **Editor** → **Pemeriksaan Ejaan**
3. Klik "Bahasa"
4. Pilih bahasa untuk diperiksa

### Menambahkan Kata ke Kamus

1. Klik kanan kata yang ditandai
2. Pilih "Tambahkan ke Kamus"
3. Kata tidak akan ditandai lagi

## Dukungan RTL

Untuk bahasa Arab, Ibrani, dan bahasa RTL lainnya:

### Deteksi RTL Otomatis

Classic mendeteksi konten RTL secara otomatis:

- Arah teks menyesuaikan
- Pergerakan kursor beradaptasi
- Perataan berubah

### Toggle RTL Manual

Paksa arah RTL atau LTR:

1. Buka Palet Perintah
2. Ketik "RTL" atau "LTR"
3. Pilih preferensi Anda

## Metode Input

Classic bekerja dengan metode input sistem Anda:

### Metode Input CJK

- Bekerja dengan Pinyin, Zhuyin, dll.
- Mendukung IME Jepang
- Metode input Korea didukung

### Karakter Khusus

Sisipkan karakter khusus dengan mudah:

- Gunakan pemilih karakter sistem Anda
- Salin/tempel dari peta karakter
- Gunakan input Unicode

## Format Tanggal & Waktu

Tanggal menyesuaikan dengan locale Anda:

### Opsi Format

| Pengaturan | Opsi |
| ---------- | ---- |
| Format Tanggal | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| Format Waktu | 12-jam, 24-jam |
| Hari Pertama Minggu | Minggu, Senin, Sabtu |

Konfigurasi di Pengaturan → Umum → Tanggal & Waktu

## Format Angka

Angka ditampilkan sesuai dengan locale Anda:

- **Pemisah desimal**: `.` atau `,`
- **Pemisah ribuan**: `,` atau `.` atau spasi
- **Simbol mata uang**: Berdasarkan wilayah

## Tata Letak Keyboard

Classic bekerja dengan tata letak keyboard apa pun:

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Tata letak kustom

## Pengaturan Lokalisasi

### Di Pengaturan → Umum → Bahasa

| Pengaturan | Deskripsi |
| ---------- | --------- |
| **Bahasa Antarmuka** | Bahasa UI aplikasi |
| **Format Tanggal** | Bagaimana tanggal muncul |
| **Format Waktu** | 12 atau 24 jam |
| **Hari Pertama Minggu** | Hari mulai kalender |
| **Format Angka** | Pemisah desimal/ribuan |

## Tips

### Catatan Bahasa Campuran

Tulis dalam beberapa bahasa dalam satu catatan:

```markdown
# Proyek Internasional

English: Selamat datang di proyek
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Font Khusus Bahasa

Beberapa bahasa terlihat lebih baik dengan font tertentu:

1. Buka Pengaturan
2. Buka Tampilan → Font
3. Setel font untuk skrip yang berbeda

### Beralih Bahasa dengan Cepat

Buat pintasan keyboard:

1. Buka Pengaturan → Pintasan
2. Temukan "Ganti Bahasa"
3. Setel pintasan pilihan Anda

## Pemecahan Masalah

### Bahasa Tidak Tersedia

- Periksa pembaruan aplikasi
- Bahasa mungkin sedang dalam pengembangan
- Kontribusikan terjemahan di GitHub

### Karakter Tidak Ditampilkan

- Instal font yang diperlukan
- Periksa pengaturan font
- Coba font yang berbeda

### Pemeriksaan Ejaan Tidak Berfungsi

- Verifikasi pemeriksaan ejaan diaktifkan
- Tambahkan bahasa ke pemeriksaan ejaan
- Restart Classic

### RTL Tidak Berfungsi dengan Benar

- Pastikan deteksi RTL aktif
- Coba toggle RTL manual
- Periksa masalah konten campuran

## Berkontribusi Terjemahan

Bantu terjemahkan Classic:

1. Kunjungi komunitas Classic
2. Bergabung dengan tim terjemahan
3. Kirim terjemahan
4. Bantu tingkatkan lokalisasi
