# Memahami Antarmuka

Antarmuka Classic dirancang agar bersih, intuitif, dan tidak mengganggu Anda. Berikut adalah tur komponen utama.

## Tata Letak Utama

Classic memiliki tata letak tiga panel:

1. **Bilah Samping** (kiri) - Navigasi dan organisasi catatan
2. **Editor** (tengah) - Ruang penulisan Anda
3. **Pratinjau/Properti** (kanan) - Panel opsional

## Bilah Samping

Bilah samping berisi beberapa bagian utama:

### Daftar Catatan

- **Semua Catatan**: Lihat semua catatan Anda dalam satu daftar
- **Catatan Harian**: Akses cepat ke jurnal hari ini
- **Favorit**: Tandai catatan yang sering digunakan
- **Folder**: Atur catatan dalam hierarki

### Navigasi

- **Bilah Pencarian**: Temukan catatan secara instan
- **Tag**: Jelajahi berdasarkan tag
- **Tautan Balik**: Lihat koneksi antar catatan

### Tindakan

- **Catatan Baru**: Buat catatan baru
- **Folder Baru**: Atur dengan folder
- **Impor**: Masukkan konten eksternal

## Editor

Editor adalah tempat Anda menulis dan memformat catatan.

### Fitur

- **Dukungan Markdown**: Sintaks Markdown lengkap
- **Pratinjau Langsung**: Lihat teks yang diformat saat Anda mengetik
- **Penyorotan Sintaks**: Blok kode disorot
- **Perintah Slash**: Ketik `/` untuk tindakan cepat

### Mode Editor

| Mode | Deskripsi |
|------|-----------|
| **Hanya Edit** | Lihat Markdown mentah |
| **Hanya Pratinjau** | Lihat hasil render |
| **Tampilan Terpisah** | Edit dan pratinjau berdampingan |

### Bilah Alat

Tindakan pemformatan cepat:

- Tebal, Miring, Coret
- Judul (H1-H6)
- Daftar (poin, bernomor, tugas)
- Tautan dan gambar
- Blok kode
- Tabel

## Bilah Status

Di bagian bawah editor:

- **Jumlah Kata/Karakter**: Lacak penulisan Anda
- **Terakhir Disimpan**: Kapan catatan terakhir disimpan
- **Lokasi Catatan**: Di mana catatan berada
- **Status Sinkronisasi**: Jika menggunakan sinkronisasi opsional

## Menu Konteks

Klik kanan untuk tindakan kontekstual:

### Menu Konteks Catatan

- Ganti nama
- Pindahkan ke folder
- Duplikat
- Hapus
- Ekspor
- Salin tautan

### Menu Konteks Editor

- Potong, Salin, Tempel
- Opsi pemformatan
- Sisipkan tautan/gambar
- Cari dan ganti

## Navigasi Keyboard

Navigasi antarmuka dengan keyboard Anda:

| Tindakan | Pintasan |
|----------|----------|
| Toggle Bilah Samping | `Cmd/Ctrl + M` |
| Fokus Pencarian | `Cmd/Ctrl + F` |
| Palet Perintah | `Cmd/Ctrl + P` |
| Catatan Baru | `Cmd/Ctrl + N` |

## Kustomisasi

Sesuaikan antarmuka sesuai keinginan Anda:

### Lebar Bilah Samping

1. Arahkan kursor ke tepi bilah samping
2. Seret untuk mengubah ukuran
3. Atau atur lebar yang tepat di Pengaturan

### Tema

Beralih antara mode Terang dan Gelap:

- Gunakan toggle tema di header
- Atau atur untuk mengikuti preferensi sistem Anda

### Ukuran Font

1. Buka Pengaturan
2. Buka Tampilan
3. Sesuaikan Ukuran Font Editor

## Tips

### Mode Fokus

Sembunyikan semua panel untuk penulisan tanpa gangguan:

1. Buka Palet Perintah
2. Ketik "Mode Fokus"
3. Tekan Enter

### Akses Cepat

- Sematkan catatan yang sering digunakan ke Favorit
- Gunakan pintasan keyboard untuk tindakan umum
- Kustomisasi bilah alat di Pengaturan
