# Perintah Slash

Perintah slash adalah cara ampuh untuk dengan cepat menyisipkan pemformatan dan konten khusus tanpa meninggalkan keyboard. Ketik `/` di editor untuk membuka palet perintah.

## Perintah yang Tersedia

### Judul

| Perintah | Hasil |
| -------- | ----- |
| `/h1` | Judul level 1 |
| `/h2` | Judul level 2 |
| `/h3` | Judul level 3 |
| `/h4` | Judul level 4 |
| `/h5` | Judul level 5 |
| `/h6` | Judul level 6 |

### Pemformatan Teks

| Perintah | Hasil |
| -------- | ----- |
| `/bold` | **Teks tebal** |
| `/italic` | *Teks miring* |
| `/strikethrough` | ~~Teks coret~~ |
| `/code` | Blok kode sebaris |
| `/quote` | Kutipan blok |

### Daftar

| Perintah | Hasil |
| -------- | ----- |
| `/bullet` | • Daftar poin |
| `/numbered` | 1. Daftar bernomor |
| `/todo` | - [ ] Item todo |
| `/check` | - [x] Item selesai |

### Sisipkan Konten

| Perintah | Hasil |
| -------- | ----- |
| `/image` | Sisipkan gambar dari URL atau file |
| `/link` | Sisipkan tautan |
| `/table` | Sisipkan tabel |
| `/hr` | Garis horizontal |
| `/code` | Blok kode dengan pemilihan bahasa |

### Blok Khusus

| Perintah | Hasil |
| -------- | ----- |
| `/mermaid` | Blok diagram Mermaid |
| `/katex` | Blok matematika KaTeX |
| `/callout` | Blok callout/admonition |

## Menggunakan Perintah Slash

1. Ketik `/` di editor
2. Menu akan muncul dengan semua perintah yang tersedia
3. Mulai mengetik untuk memfilter perintah
4. Tekan `Enter` atau klik untuk menyisipkan

## Tips Pro

- **Akses cepat**: Cukup ketik `/` dan mulai mengetik nama perintah
- **Pencarian fuzzy**: Anda tidak perlu mengetik nama perintah yang tepat - Classic akan menemukan kecocokan terdekat
- **Navigasi keyboard**: Gunakan tombol panah untuk menavigasi menu
- **Perintah terbaru**: Perintah yang sering digunakan muncul di bagian atas daftar
