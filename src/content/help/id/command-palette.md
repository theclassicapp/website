# Palet Perintah

Palet perintah adalah gerbang Anda ke setiap tindakan di Classic. Tekan pintasan keyboard sederhana dan akses fitur apa pun secara instan.

## Membuka Palet Perintah

| Platform | Pintasan |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Cara Menggunakan

1. Buka palet perintah (`Cmd+P` atau `Ctrl+P`)
2. Ketik untuk mencari perintah
3. Gunakan tombol panah untuk menavigasi
4. Tekan `Enter` untuk menjalankan

## Kategori Perintah

### Operasi File

- **Catatan Baru**: Buat catatan baru
- **Catatan Harian Baru**: Buat/buka catatan harian hari ini
- **Buka Catatan**: Cari dan buka catatan yang ada
- **Simpan Catatan**: Simpan catatan saat ini
- **Hapus Catatan**: Hapus catatan saat ini
- **Ganti Nama Catatan**: Ganti nama catatan saat ini

### Navigasi

- **Ke Catatan**: Loncat ke catatan apa pun
- **Ke Catatan Harian**: Buka tanggal tertentu
- **Ke Tag**: Jelajahi catatan berdasarkan tag
- **Kembali**: Navigasi kembali dalam riwayat
- **Maju**: Navigasi maju dalam riwayat

### Editor

- **Toggle Pratinjau**: Beralih antara mode edit dan pratinjau
- **Toggle Bilah Samping**: Tampilkan/sembunyikan bilah samping
- **Mode Fokus**: Masuk penulisan tanpa gangguan
- **Sisipkan Tanggal**: Tambahkan tanggal/waktu saat ini
- **Sisipkan Template**: Sisipkan template

### Pemformatan

- **Tebal**: Buat teks tebal
- **Miring**: Buat teks miring
- **Coret**: Coret teks
- **Kode**: Format sebagai kode
- **Tautan**: Sisipkan tautan
- **Heading**: Terapkan format heading

### Organisasi

- **Tambah Tag**: Tag catatan saat ini
- **Pindahkan ke Folder**: Pindahkan catatan ke folder
- **Arsipkan Catatan**: Arsipkan catatan saat ini
- **Sematkan Catatan**: Sematkan catatan ke atas daftar

### Tampilan

- **Toggle Tema**: Beralih antara terang/gelap
- **Zoom In**: Tingkatkan ukuran font
- **Zoom Out**: Kurangi ukuran font
- **Reset Zoom**: Reset ke ukuran default

## Tips

### Akses Cepat

- Perintah dicari secara fuzzy
- Perintah terbaru muncul pertama
- Sematkan perintah yang sering digunakan

### Navigasi Keyboard

- `↑`/`↓` - Navigasi hasil
- `Enter` - Jalankan perintah
- `Escape` - Tutup palet
- `Tab` - Berputar melalui kategori

### Perintah Kustom

1. Buka Pengaturan
2. Buka Palet Perintah
3. Tambahkan perintah kustom
4. Definisikan pintasan keyboard

## Perintah yang Sering Digunakan

Berikut beberapa perintah yang akan sering Anda gunakan:

| Perintah | Apa yang dilakukan |
| -------- | ------------------ |
| `Catatan Baru` | Buat catatan kosong |
| `Buka Catatan` | Cari semua catatan |
| `Catatan Harian Baru` | Jurnal hari ini |
| `Toggle Pratinjau` | Pratinjau vs edit |
| `Tambah Tag` | Tag catatan saat ini |
| `Tangkapan Cepat` | Tangkapan global |

## Mengkustomisasi Perintah

### Mengubah Pintasan

1. Buka Pengaturan (`Cmd+,` atau `Ctrl+,`)
2. Navigasi ke Pintasan Keyboard
3. Temukan perintah
4. Klik pintasan saat ini
5. Tekan pintasan baru Anda
6. Simpan

### Membuat Perintah Kustom

Pengguna tingkat lanjut dapat membuat perintah kustom:

1. Buka Pengaturan
2. Buka Perintah
3. Klik "Tambah Perintah Kustom"
4. Definisikan nama perintah
5. Tambahkan tindakan
6. Setel pintasan (opsional)
