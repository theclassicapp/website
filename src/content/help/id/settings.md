# Pengaturan

Kustomisasi Classic agar bekerja persis seperti yang Anda inginkan. Dari tampilan hingga perilaku editor, semuanya dapat dikonfigurasi.

## Membuka Pengaturan

### Pintasan Keyboard

| Platform | Pintasan |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Melalui Menu

- **macOS**: Classic → Pengaturan
- **Windows**: File → Pengaturan
- **Linux**: Edit → Pengaturan

### Melalui Palet Perintah

1. Buka Palet Perintah (`Cmd+P` atau `Ctrl+P`)
2. Ketik "Pengaturan"
3. Tekan Enter

## Kategori Pengaturan

### Umum

Pengaturan aplikasi dasar:

| Pengaturan | Deskripsi |
|-----------|-----------|
| **Bahasa** | Bahasa antarmuka |
| **Tema** | Terang, Gelap, atau Sistem |
| **Mulai Saat Login** | Luncurkan saat startup |
| **Periksa Pembaruan** | Perilaku auto-update |

### Tampilan

Kustomisasi tampilan dan nuansa:

| Pengaturan | Opsi |
|-----------|------|
| **Tema** | Terang / Gelap / Sistem |
| **Warna Aksen** | Biru, Ungu, Hijau, dll. |
| **Font Editor** | Font family dan ukuran |
| **Font UI** | Font antarmuka |

### Editor

Konfigurasi perilaku penulisan:

| Pengaturan | Deskripsi |
|-----------|-----------|
| **Pemeriksaan Ejaan** | Aktifkan/nonaktifkan pemeriksaan ejaan |

| **Ukuran Tab** | 2, 4, atau 8 spasi |

### Riwayat Versi

Kontrol pelacakan versi:

| Pengaturan | Opsi | Default |
|-----------|------|---------|
| **Aktifkan Riwayat** | Aktif/Mati | Aktif |
| **Periode Retensi** | 30-180 hari | 90 hari |
| **Versi Maks** | 50-500 | 100 |

### Backup & Restore

| Pengaturan | Deskripsi |
|-----------|-----------|

## Mencari Pengaturan

Tidak dapat menemukan pengaturan? Gunakan bilah pencarian:

1. Buka Pengaturan
2. Ketik di kotak pencarian
3. Pengaturan yang cocok muncul
4. Klik untuk loncat ke pengaturan itu

## Reset Pengaturan

### Reset Semua Pengaturan

Kembali ke default:

1. Buka Pengaturan
2. Gulir ke bawah
3. Klik "Reset Semua Pengaturan"
4. Konfirmasi tindakan

### Reset Kategori

Reset kategori tertentu:

1. Buka kategori
2. Klik "Reset ke Default"
3. Hanya kategori itu yang direset

## Mengekspor/Mengimpor Pengaturan

### Ekspor Pengaturan

Simpan konfigurasi Anda:

1. Buka Pengaturan
2. Klik "Ekspor Pengaturan"
3. Pilih lokasi
4. Simpan sebagai file JSON

### Impor Pengaturan

Muat konfigurasi yang tersimpan:

1. Buka Pengaturan
2. Klik "Impor Pengaturan"
3. Pilih file JSON
4. Konfirmasi impor

## Lokasi Pengaturan

Pengaturan disimpan secara lokal:

| Platform | Lokasi |
|----------|--------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Sinkronkan Pengaturan (Opsional)

Saat masuk, pengaturan dapat disinkronkan antar perangkat:

- **Sinkronkan**: Aktifkan di Pengaturan → Sinkron
- **Apa yang Disinkronkan**: Tema, font, pintasan
- **Apa yang Tidak Disinkronkan**: Jalur lokal, kunci enkripsi

## Kustomisasi Umum

### Tingkatkan Ukuran Font

1. Pengaturan → Tampilan
2. Ukuran Font Editor → Tingkatkan
3. Perubahan diterapkan segera

### Ubah Tema

1. Pengaturan → Tampilan
2. Tema → Pilih Terang/Gelap/Sistem
3. Aplikasi diperbarui secara instan

### Aktifkan Auto-Backup

1. Pengaturan → Backup & Restore
2. Aktifkan "Auto Backup"
3. Setel frekuensi dan lokasi

### Kustomisasi Pintasan

1. Pengaturan → Pintasan Keyboard
2. Temukan tindakan
3. Klik pintasan saat ini
4. Tekan tombol baru
5. Simpan

## Tips

### Akses Cepat

- Gunakan `Cmd/Ctrl+,` untuk membuka pengaturan dengan cepat
- Cari pengaturan berdasarkan nama
- Gunakan kategori untuk menavigasi

### Ekspor Sebelum Perubahan

Sebelum perubahan besar, ekspor pengaturan Anda:

- Mudah untuk restore jika ada yang salah
- Bagikan konfigurasi dengan orang lain

### Periksa Pengaturan Baru

Pembaruan mungkin menambahkan pengaturan baru:

- Periksa setelah memperbarui
- Jelajahi opsi baru

## Pemecahan Masalah

### Pengaturan Tidak Disimpan

- Periksa izin file
- Verifikasi ruang disk
- Coba reset dan konfigurasi ulang

### Pengaturan Reset Tidak Terduga

- Mungkin menunjukkan korupsi file
- Ekspor pengaturan secara teratur
- Periksa konflik aplikasi

### Tidak Dapat Menemukan Pengaturan

- Gunakan bilah pencarian
- Periksa kategori yang berbeda
- Mungkin di pengaturan Lanjutan
