# Membuat Catatan Pertama Anda

Mulai menggunakan Classic dengan membuat catatan pertama Anda. Ini sederhana dan intuitif.

## Membuat Catatan Baru

### Menggunakan Bilah Samping

1. Klik tombol **+** di bilah samping
2. Catatan baru dibuat
3. Mulai mengetik konten Anda

### Menggunakan Pintasan Keyboard

| Platform | Pintasan |
| -------- | -------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Menggunakan Palet Perintah

1. Buka Palet Perintah (`Cmd+P` atau `Ctrl+P`)
2. Ketik "Catatan Baru"
3. Tekan Enter

## Menulis Catatan Anda

### Pengeditan Dasar

Langsung mulai mengetik! Classic menyimpan otomatis saat Anda menulis.

### Menambahkan Judul

Baris pertama catatan Anda menjadi judul. Gunakan heading untuk hasil terbaik:

```markdown
# Catatan Pertama Saya

Ini adalah konten catatan saya.
```

### Memformat Teks

Gunakan Markdown untuk memformat teks Anda:

- **Tebal**: `**teks**` atau `Cmd/Ctrl+B`
- **Miring**: `*teks*` atau `Cmd/Ctrl+I`
- **Coret**: `~~teks~~`
- **Kode**: `` `kode` ``

## Menyimpan

Classic menyimpan catatan Anda secara otomatis saat Anda mengetik. Tidak perlu menyimpan secara manual.

### Simpan Manual

Jika diperlukan, Anda dapat memaksa penyimpanan:

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## Mengatur Catatan Anda

### Menambahkan Tag

Tag catatan Anda untuk memudahkan pencarian nanti:

1. Ketik `#` diikuti dengan nama tag Anda
2. Atau gunakan tombol tag di bilah alat
3. Beberapa tag didukung

Contoh:

```markdown
# Catatan Rapatan

Membahas perencanaan Q4 #kerja #rapat
```

### Memindahkan ke Folder

1. Klik kanan catatan di bilah samping
2. Pilih "Pindahkan ke Folder"
3. Pilih atau buat folder tujuan

### Menyematkan Catatan Penting

1. Klik kanan catatan
2. Pilih "Sematkan ke Atas"
3. Catatan muncul di bagian atas daftar Anda

## Pratinjau Catatan Anda

Lihat tampilan catatan yang diformat:

- Klik tombol **Pratinjau**
- Atau gunakan `Cmd/Ctrl+Shift+P`
- Beralih antara mode Edit dan Pratinjau

## Langkah Selanjutnya

Sekarang setelah Anda membuat catatan pertama:

- Pelajari lebih lanjut tentang [Markdown](/help/markdown)
- Atur dengan [Tag](/help/tags)
- Siapkan [Catatan Harian](/help/daily-notes)
- Jelajahi [Pintasan Keyboard](/help/keyboard-shortcuts)

## Tips

### Judul Catatan

- Buat judul deskriptif
- Baris pertama = judul catatan
- Gunakan H1 (`#`) untuk kejelasan

### Simpan Otomatis

- Tidak perlu khawatir kehilangan pekerjaan
- Perubahan disimpan secara otomatis
- Periksa bilah status untuk status penyimpanan

### Tangkapan Cepat

Perlu mencatat sesuatu dengan cepat?

- Gunakan Tangkapan Cepat (`Cmd/Ctrl+Shift+N`)
- Membuat catatan secara instan
- Bagus untuk pemikiran sesaat
