# Pintasan Keyboard

Bekerja lebih cepat dengan pintasan keyboard Classic. Semua pintasan dapat dikustomisasi di Pengaturan.

## Pintasan Global

Pintasan ini berfungsi di mana saja di aplikasi.

### Umum

| Pintasan (Windows/Linux) | Pintasan (macOS) | Tindakan |
| ------------------------ | ---------------- | -------- |
| `Ctrl+N` | `Cmd+N` | Catatan baru |
| `Ctrl+P` | `Cmd+P` | Palet perintah |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Tangkapan cepat global |
| `Ctrl+S` | `Cmd+S` | Simpan catatan |
| `Ctrl+F` | `Cmd+F` | Pencarian |
| `Ctrl+,` | `Cmd+,` | Pengaturan |

### Navigasi

| Pintasan (Windows/Linux) | Pintasan (macOS) | Tindakan |
| ------------------------ | ---------------- | -------- |
| `Ctrl+1` | `Cmd+1` | Ke catatan pertama |
| `Ctrl+2` | `Cmd+2` | Ke catatan kedua di daftar |
| `Ctrl+Left` | `Cmd+Left` | Navigasi kembali |
| `Ctrl+Right` | `Cmd+Right` | Navigasi maju |
| `Ctrl+Up` | `Cmd+Up` | Navigasi ke catatan induk |
| `Ctrl+Down` | `Cmd+Down` | Perluas/fokus subcatatan pertama |

| `Ctrl+I` | `Cmd+I` | Miring |
| `Ctrl+K` | `Cmd+K` | Sisipkan tautan |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | Coret |
| `Ctrl+`` | `Cmd+`` | Kode sebaris |

| `Tab` | `Tab` | Indentasi |

### Manajemen Jendela

| ------------------------ | ---------------- | -------- |

## Tangkapan Cepat

| Pintasan (Windows/Linux) | Pintasan (macOS) | Tindakan |
| ------------------------ | ---------------- | -------- |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Buka tangkapan cepat |
| `Escape` | `Escape` | Tutup tangkapan cepat |
| `Ctrl+Enter` | `Cmd+Enter` | Simpan dan tutup |

## Mengkustomisasi Pintasan

1. Buka **Pengaturan** (`Ctrl+,` atau `Cmd+,`)
2. Navigasi ke **Pintasan Keyboard**
3. Klik pintasan apa pun untuk mengubahnya
4. Tekan kombinasi tombol baru Anda
5. Klik **Simpan**

## Tips

- Sebagian besar pintasan dapat dikustomisasi
- Konflik dengan pintasan sistem akan disorot
- Reset ke default dengan mengklik tombol reset di samping setiap pintasan
