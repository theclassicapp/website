# Baki Sistem

Classic berada di baki sistem Anda untuk akses cepat. Tangkap pemikiran secara instan tanpa mengganggu alur kerja Anda.

## Apa Itu Baki Sistem?

Baki sistem (Windows/Linux) atau bilah menu (macOS) adalah tempat Classic berjalan di latar belakang, selalu siap untuk tangkapan cepat.

### Manfaat

- **Selalu tersedia**: Hanya satu klik jauhnya
- **Tangkapan cepat**: Catat pemikiran secara instan
- **Berjalan di latar belakang**: Tidak perlu menjaga aplikasi terbuka
- **Notifikasi**: Dapatkan pengingat tugas

## Mengaktifkan Baki Sistem

### macOS

1. Buka Classic
2. Buka Pengaturan → Umum
3. Aktifkan "Tampilkan di Bilah Menu"
4. Ikon Classic muncul di bilah menu

### Windows

1. Buka Classic
2. Buka Pengaturan → Umum
3. Aktifkan "Tampilkan di Baki Sistem"
4. Ikon Classic muncul di baki sistem

### Linux

1. Buka Classic
2. Buka Pengaturan → Umum
3. Aktifkan "Ikon Baki Sistem"
4. Tergantung pada lingkungan desktop

## Tangkapan Cepat

### Membuka Tangkapan Cepat

Klik ikon Classic di baki sistem:

- **macOS**: Ikon bilah menu
- **Windows**: Ikon baki sistem
- **Linux**: Ikon baki sistem

### Menggunakan Tangkapan Cepat

1. Klik ikon baki sistem
2. Jendela tangkapan cepat terbuka
3. Ketik catatan Anda
4. Tekan Enter untuk menyimpan
5. Jendela tertutup secara otomatis

### Pintasan Keyboard

Buka Tangkapan Cepat dari mana saja:

| Platform | Pintasan |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (global) |
| **Windows** | `Ctrl+Shift+N` (global) |
| **Linux** | `Ctrl+Shift+N` (global) |

## Menu Baki Sistem

Klik kanan ikon baki untuk opsi:

### Opsi Menu

| Opsi | Tindakan |
|------|----------|
| **Catatan Baru** | Buka Tangkapan Cepat |
| **Catatan Harian Baru** | Buka catatan harian hari ini |
| **Pencarian** | Buka jendela pencarian |
| **Buka Classic** | Buka jendela utama |
| **Pengaturan** | Buka pengaturan |
| **Keluar** | Keluar dari Classic |

## Pengaturan Baki

### Konfigurasi di Pengaturan

| Pengaturan | Deskripsi |
|-----------|-----------|
| **Tampilkan Ikon** | Toggle visibilitas ikon baki |
| **Mulai Diminimalkan** | Mulai Classic hanya di baki |
| **Tutup ke Baki** | Tetap berjalan saat jendela ditutup |
| **Notifikasi** | Aktifkan/nonaktifkan notifikasi |

### Mulai Saat Login

Membuat Classic mulai secara otomatis:

1. Buka Pengaturan
2. Buka Umum
3. Aktifkan "Mulai Saat Login"
4. Classic diluncurkan saat startup

## Notifikasi

### Aktifkan Notifikasi

1. Buka Pengaturan
2. Buka Notifikasi
3. Toggle "Aktifkan Notifikasi"

### Jenis Notifikasi

- **Pengingat**: Pengingat terjadwal
- **Catatan Harian**: Prompt catatan harian
- **Pembaruan**: Pembaruan aplikasi tersedia

## Perilaku Jendela

### Perilaku Tutup

Pilih apa yang terjadi saat Anda menutup jendela:

- **Minimalkan ke Baki**: Tetap berjalan, sembunyikan jendela
- **Keluar**: Keluar dari aplikasi
- **Tanya**: Tanyakan setiap kali

### Perilaku Minimalkan

- **Minimalkan ke Baki**: Sembunyikan di baki sistem
- **Minimalkan ke Taskbar**: Minimalkan normal

## Tips

### Tangkapan Cepat

Gunakan pintasan global untuk tangkapan tercepat:

1. Tekan `Cmd/Ctrl+Shift+N` di mana saja
2. Ketik pemikiran Anda
3. Tekan Enter
4. Lanjutkan pekerjaan Anda

### Akses Cepat

Klik ikon baki untuk melihat:

- Catatan terbaru
- Tindakan cepat
- Catatan harian hari ini

### Berjalan di Latar Belakang

Biarkan Classic berjalan untuk:

- Startup lebih cepat
- Tangkapan cepat
- Notifikasi
- Backup otomatis

## Pemecahan Masalah

### Ikon Baki Hilang

**macOS**

1. Periksa apakah ikon tersembunyi
2. Lihat di Control Center (macOS Big Sur+)
3. Restart Classic

**Windows**

1. Periksa ikon tersembunyi (panah di taskbar)
2. Seret ikon Classic ke area yang terlihat
3. Restart Classic

**Linux**

1. Periksa dukungan lingkungan desktop
2. Instal ekstensi baki sistem jika diperlukan
3. Restart Classic

### Pintasan Global Tidak Berfungsi

- Periksa konflik dengan aplikasi lain
- Aktifkan kembali pintasan di Pengaturan
- Restart Classic
- Periksa izin OS

### Notifikasi Tidak Muncul

**macOS**

1. System Preferences → Notifications
2. Temukan Classic
3. Aktifkan notifikasi

**Windows**

1. Settings → System → Notifications
2. Aktifkan notifikasi Classic

**Linux**

1. Periksa daemon notifikasi
2. Instal libnotify jika diperlukan

### Penggunaan Memori/CPU Tinggi

Jika ikon baki menggunakan terlalu banyak sumber daya:

1. Nonaktifkan animasi di Pengaturan
2. Kurangi frekuensi notifikasi
3. Periksa pembaruan aplikasi
