# Tema & Tampilan

Personalisasi tampilan Classic agar sesuai dengan gaya Anda dan mengurangi ketegangan mata.

## Opsi Tema

### Mode Terang

Tema default Classic memiliki fitur:

- Antarmuka bersih dan terang
- Optimal untuk lingkungan terang
- Kontras tinggi untuk keterbacaan

### Mode Gelap

Sempurna untuk begadang dan bekerja dengan cahaya redup:

- Mengurangi ketegangan mata
- Konsumsi baterai lebih rendah pada layar OLED
- Nyaman di mata di lingkungan gelap

### Sistem (Otomatis)

Ikuti tema sistem operasi Anda:

- Secara otomatis beralih berdasarkan pengaturan OS
- Tidak perlu pergantian manual
- Konsisten dengan aplikasi lain

## Mengganti Tema

### Melalui Pengaturan

1. Buka Pengaturan (`Cmd+,` atau `Ctrl+,`)
2. Navigasi ke **Tampilan**
3. Pilih tema pilihan Anda

### Melalui Palet Perintah

1. Buka Palet Perintah (`Cmd+P` atau `Ctrl+P`)
2. Ketik "Tema"
3. Pilih tema pilihan Anda

### Pintasan Keyboard

| Platform | Pintasan |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Berputar melalui: Terang → Gelap → Sistem

## Pengaturan Font

### Font Editor

Kustomisasi pengalaman menulis:

1. Buka Pengaturan
2. Buka **Tampilan** → **Editor**
3. Konfigurasi opsi font

| Pengaturan | Opsi | Default |
| ---------- | ---- | ------- |
| Font Family | Font sistem + kustom | System UI |
| Ukuran Font | 12-24px | 16px |
| Tinggi Baris | 1.0-2.5 | 1.75 |
| Spasi Huruf | -2 to 4 | 0 |

### Font UI

Sesuaikan font antarmuka:

1. Buka Pengaturan
2. Buka **Tampilan** → **Antarmuka**
3. Pilih font dan ukuran

### Font Monospace

Untuk blok kode dan konten teknis:

1. Buka Pengaturan
2. Buka **Tampilan** → **Kode**
3. Pilih font monospace

## Opsi Antarmuka

### Bilah Samping

Kustomisasi perilaku bilah samping:

| Pengaturan | Deskripsi |
| ---------- | --------- |
| **Auto-hide** | Sembunyikan bilah samping saat tidak digunakan |
| **Lebar** | Sesuaikan lebar bilah samping (200-400px) |
| **Tampilkan Ikon** | Toggle visibilitas ikon |

### Tata Letak Editor

Pilih pengalaman editing Anda:

- **Tampilan Terpisah**: Edit dan pratinjau berdampingan
- **Hanya Pratinjau**: Lihat hasil render saja
- **Hanya Edit**: Fokus pada Markdown mentah

### Daftar Catatan

Konfigurasi bagaimana catatan muncul:

- **Tampilan Daftar**: Ringkas, satu baris per catatan
- **Tampilan Kartu**: Kartu pratinjau dengan cuplikan
- **Tampilan Pohon**: Struktur folder hierarkis

## Warna Aksen

### Warna yang Tersedia

Pilih warna aksen Anda:

- Classic Blue (default)
- Purple
- Green
- Orange
- Red
- Pink
- Teal

### Mengubah Warna Aksen

1. Buka Pengaturan
2. Buka **Tampilan** → **Warna**
3. Pilih aksen pilihan Anda

Warna aksen mempengaruhi:

- Tautan dan tombol
- Sorotan seleksi
- Elemen aktif
- Indikator fokus

## Aksesibilitas

### Mode Kontras Tinggi

Untuk visibilitas yang lebih baik:

1. Buka Pengaturan
2. Buka **Tampilan** → **Aksesibilitas**
3. Aktifkan **Kontras Tinggi**

### Gerakan Berkurang

Minimalkan animasi:

1. Buka Pengaturan
2. Buka **Tampilan** → **Aksesibilitas**
3. Aktifkan **Gerakan Berkurang**

### Penyesuaian Font

- Tingkatkan ukuran font minimum
- Gunakan font ramah disleksia
- Sesuaikan rasio kontras

## CSS Kustom (Lanjutan)

Untuk pengguna power yang menginginkan kendali penuh:

1. Buka Pengaturan
2. Buka **Tampilan** → **Lanjutan**
3. Klik **Buka CSS Kustom**
4. Tambahkan kustomisasi Anda

### Contoh Kustomisasi

```css
/* Heading lebih besar */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Warna tautan kustom */
a { color: #3b82f6; }

/* Spasi baris lebih lebar */
.prose p { line-height: 2; }
```

## Sinkronisasi Tema

Saat Anda masuk (opsional), preferensi tema Anda disinkronkan antar perangkat:

- Sinkronisasi tema otomatis
- Sinkronisasi pengaturan font
- CSS kustom disinkronkan

## Tips

### Mencocokkan dengan Alur Kerja Anda

- Gunakan **Terang** saat siang, **Gelap** saat malam
- Mode **Sistem** menangani ini secara otomatis
- Pertimbangkan pencahayaan sekitar Anda

### Mengurangi Ketegangan Mata

- Gunakan mode Gelap dalam cahaya redup
- Tingkatkan ukuran font untuk sesi panjang
- Aktifkan Kontras Tinggi jika diperlukan
- Istirahat secara teratur

### Produktivitas

- Pelajari pintasan toggle tema
- Siapkan waktu pergantian otomatis
- Gunakan mode Sistem untuk konsistensi
