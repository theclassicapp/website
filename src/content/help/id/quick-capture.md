# Tangkapan Cepat

Tangkap ide secara instan tanpa mengganggu alur kerja Anda. Tangkapan Cepat membuka jendela pencatatan minimal dari mana saja di sistem Anda.

## Hotkey Global

Tekan hotkey yang ditentukan untuk membuka Tangkapan Cepat:

| Platform | Pintasan Default |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

Anda dapat mengkustomisasi pintasan ini di Pengaturan.

## Menggunakan Tangkapan Cepat

1. Tekan hotkey global (di mana saja di sistem Anda)
2. Ketik catatan Anda di jendela popup
3. Tekan `Enter` untuk menyimpan
4. Tekan `Escape` untuk membatalkan

## Fitur

### Antarmuka Minimal

- Jendela kecil dan fokus
- Dukungan Markdown
- Auto-fokus pada input
- Desain keyboard-first

### Tindakan Cepat

- `# ` + spasi = heading
- `#inbox` = tambahkan ke Inbox
- `#today` = tambahkan ke catatan hari ini
- Tag dengan `#namatag`

### Tujuan Default

Catatan Tangkapan Cepat masuk ke **Inbox** Anda secara default. Anda dapat mengubah ini:

1. Buka Pengaturan
2. Buka Tangkapan Cepat
3. Pilih tujuan default

## Tips

### Alur Kerja Tangkapan

1. **Tangkap** - Catat ide dengan cepat
2. **Proses** - Tinjau Inbox setiap hari
3. **Atur** - Pindahkan ke folder/catatan yang sesuai

### Praktik Terbaik

- Tetap singkat - satu ide per tangkapan
- Gunakan tag untuk konteks
- Jangan khawatir tentang pemformatan - perbaiki nanti
- Tinjau dan proses tangkapan secara teratur

### Integrasi

Tangkapan Cepat bekerja dengan baik dengan:

- **Catatan Harian**: Mulai setiap hari dengan meninjau tangkapan
- **Inbox**: Gunakan sebagai area penampungan
- **Tag**: Tambahkan konteks segera

## Pemecahan Masalah

### Hotkey Tidak Berfungsi?

**macOS**
1. Periksa System Preferences > Keyboard > Shortcuts
2. Cari konflik
3. Restart Classic

**Windows**
1. Jalankan Classic sebagai Administrator
2. Periksa pintasan yang konflik
3. Restart Classic

### Jendela Tidak Muncul?

- Periksa apakah Classic berjalan
- Verifikasi pintasan
- Periksa masalah fokus
- Restart Classic

### Tangkapan Tidak Disimpan?

- Periksa tujuan default
- Verifikasi izin tulis
- Periksa ruang disk
- Restart Classic
