# System Tray

Classic lives in your system tray for quick access. Capture thoughts instantly without interrupting your workflow.

## What Is the System Tray?

The system tray (Windows/Linux) or menu bar (macOS) is where Classic runs in the background, always ready for quick capture.

### Benefits

- **Always available**: One click away
- **Quick capture**: Jot thoughts instantly
- **Background running**: No need to keep app open
- **Notifications**: Get reminded of tasks

## Enabling System Tray

### macOS

1. Open Classic
2. Go to Settings → General
3. Enable "Show in Menu Bar"
4. Classic icon appears in menu bar

### Windows

1. Open Classic
2. Go to Settings → General
3. Enable "Show in System Tray"
4. Classic icon appears in system tray

### Linux

1. Open Classic
2. Go to Settings → General
3. Enable "System Tray Icon"
4. Depends on desktop environment

## Quick Capture

### Opening Quick Capture

Click the Classic icon in the system tray

- **macOS**: Menu bar icon
- **Windows**: System tray icon
- **Linux**: System tray icon

### Using Quick Capture

1. Click system tray icon
2. Quick capture window opens
3. Type your note
4. Press Enter to save
5. Window closes automatically

### Keyboard Shortcut

Open Quick Capture from anywhere

| Platform | Shortcut |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (global) |
| **Windows** | `Ctrl+Shift+N` (global) |
| **Linux** | `Ctrl+Shift+N` (global) |

## System Tray Menu

Right-click the tray icon for options

### Menu Options

| Option | Action |
|--------|--------|
| **New Note** | Open Quick Capture |
| **New Daily Note** | Open today's daily note |
| **Search** | Open search window |
| **Open Classic** | Open main window |
| **Settings** | Open settings |
| **Quit** | Exit Classic |

## Tray Settings

### Configure in Settings

| Setting | Description |
|---------|-------------|
| **Show Icon** | Toggle tray icon visibility |
| **Start Minimized** | Start Classic in tray only |
| **Close to Tray** | Keep running when window closed |
| **Notifications** | Enable/disable notifications |

### Start at Login

Have Classic start automatically

1. Open Settings
2. Go to General
3. Enable "Start at Login"
4. Classic launches on startup

## Notifications

### Enable Notifications

1. Open Settings
2. Go to Notifications
3. Toggle "Enable Notifications"

### Notification Types

- **Reminders**: Scheduled reminders
- **Daily Note**: Daily note prompts
- **Updates**: App update available

## Window Behavior

### Close Behavior

Choose what happens when you close the window

- **Minimize to Tray**: Keep running, hide window
- **Quit**: Exit the application
- **Ask**: Prompt each time

### Minimize Behavior

- **Minimize to Tray**: Hide in system tray
- **Minimize to Taskbar**: Minimize normally

## Troubleshooting

### Tray Icon Missing

**macOS**

1. Check if icon is hidden
2. Look in Control Center (macOS Big Sur+)
3. Restart Classic

**Windows**

1. Check hidden icons (arrow in taskbar)
2. Drag Classic icon to visible area
3. Restart Classic

**Linux**

1. Check desktop environment support
2. Install system tray extension if needed
3. Restart Classic

### Global Shortcut Not Working

- Check for conflicts with other apps
- Re-enable the shortcut in Settings
- Restart Classic
- Check OS permissions

### Notifications Not Showing

**macOS**

1. System Preferences → Notifications
2. Find Classic
3. Enable notifications

**Windows**

1. Settings → System → Notifications
2. Enable Classic notifications

**Linux**

1. Check notification daemon
2. Install libnotify if needed

### High Memory/CPU Usage

If tray icon uses too many resources

1. Disable animations in Settings
2. Reduce notification frequency
3. Check for app updates
