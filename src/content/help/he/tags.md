# תגיות

תגיות הן דרך חזקה לארגן ולסווג את ההערות שלכם ב-Classic. מצאו תוכן קשור באופן מיידי.

## יצירת תגיות

### תגיות בתוך השורה

הוסיפו תגיות ישירות בתוכן ההערה

```markdown
# הערות פגישה

דיון על מפת הדרכים של Q4 #עבודה #פגישות #תכנון
```

### באמצעות כפתור התגית

1. לחצו על סמל התגית בסרגל הכלים
2. הקלידו את שם התגית
3. לחצו Enter

### קיצור מקלדת

| פלטפורמה | קיצור מקלדת |
|----------|----------|
| **macOS** | `Cmd+T` |
| **Windows/Linux** | `Ctrl+T` |

## תחביר תגיות

### תגיות בסיסיות

```markdown
#פרויקט #עבודה #חשוב
```

### תגיות מקוננות

צרו היררכיות תגיות עם `/`

```markdown
#עבודה/פגישות
#עבודה/פרויקטים
#אישי/תחביבים/קריאה
```

### תגיות מרובות מילים

השתמשו במקפים או קווים תחתונים

```markdown
#פרויקט-אלפא
#הערות_פגישה
#תכנון-q4
```

## ניהול תגיות

### צפייה בכל התגיות

1. פתחו את סרגל הצד
2. מצאו את מקטע "תגיות"
3. ראו את כל התגיות שלכם

### ענן תגיות

דמיינו את התגיות בשימוש הכי תכוף

- תגיות גדולות יותר = יותר הערות
- לחצו על כל תגית לסינון
- ראו ספירת הערות לכל תגית

### שינוי שם תגיות

1. לחצו לחיצה ימנית על תגית
2. בחרו "שנה שם"
3. הזינו שם חדש
4. כל ההערות מתעדכנות אוטומטית

### מחיקת תגיות

1. לחצו לחיצה ימנית על תגית
2. בחרו "מחק"
3. אשרו מחיקה
4. התגית הוסרה מכל ההערות

## חיפוש לפי תגית

### חלונית תגיות

לחצו על תגית בסרגל הצד לסינון הערות

### תחביר חיפוש

השתמשו ב-`tag:` בחיפוש

```
tag:עבודה
tag:עבודה/פגישות
```

### שילוב תגיות

מצאו הערות עם תגיות מרובות

```
tag:עבודה tag:דחוף
```

### אי-כללת תגיות

מצאו הערות בלי תגית

```
-tag:ארכיון
```

## שיטות עבודה מומלצות לתגיות

### שמרו על פשטות

- השתמשו ב-3-5 תגיות לכל הערה
- היו עקביים עם שמות
- הימנעו מהתרבות תגיות

### השתמשו בהיררכיות

ארגנו תגיות קשורות

```
#עבודה
  #עבודה/פגישות
  #עבודה/פרויקטים
  #עבודה/תאריכים

#אישי
  #אישי/בריאות
  #אישי/כספים
```

### הקימו מוסכמות

הגדירו תקני תגיות לעצמכם

- **סטטוס**: `#לעשות` `#בתהליך` `#בוצע`
- **עדיפות**: `#דחוף` `#גבוה` `#נמוך`
- **סוג**: `#פגישה` `#הפניה` `#רעיון`

## הצעות תגיות

כשאתם מקלידים `#`, Classic מציע תגיות קיימות

- תגיות אחרונות מופיעות ראשונות
- נתמך התאמה מטושטשת
- לחצו Tab לקבלת הצעה

## סטטיסטיקות תגיות

צפו בשימוש בתגיות בהגדרות

- תגיות בשימוש הכי תכוף
- תגיות שאינן בשימוש
- מגמות תגיות לאורך זמן

## מקרי שימוש

### ניהול פרויקטים

```markdown
# התחלת פרויקט #פרויקט-אלפא #פגישה #2024-01-15

משתתפים: צוות אלפא
סטטוס: #לעשות
```

### בסיס ידע

```markdown
# סגורות ב-JavaScript #תכנות #javascript #הפניה #מתקדם

סגורות הן פונקציות שיש להן גישה ל...
```

### הערות יומיות

```markdown
# 2024-01-15

- סטנדאפ בוקר #עבודה #פגישות
- סקירת קוד #עבודה #פיתוח
- אימון בחדר כושר #אישי #בריאות
```

### הערות קריאה

```markdown
# הערות על הרגלים אטומיים #קריאה #ספרים #פרודוקטיביות

תובנות מפתח מהספר...
```

## קיצורי מקלדת

| פעולה | קיצור מקלדת |
|--------|----------|
| הוספת תגית | `Cmd/Ctrl+T` |
| חיפוש תגיות | `Cmd/Ctrl+F`, ואז הקלידו `tag:` |
| הצגת חלונית תגיות | `Cmd/Ctrl+Shift+T` |

## טיפים

### שמות עקביים

הקפידו על מוסכמה

```
✅ #הערות-פגישה
✅ #הערות_פגישה
❌ #הערות פגישה (רווחים לא מורשים)
❌ #הערותפגישה (לא עקבי)
```

### הימנעו מתיוג יתר

יותר מדי תגיות הופכות לבלתי ניתנות לניהול

```
✅ #עבודה #דחוף #פרויקט-אלפא
❌ #עבודה #דחוף #חשוב #היום #שני #משרד #שולחן #מחשב
```

### ניקוי תקופתי

סקרו ואחדו תגיות באופן קבוע

- מזגו תגיות דומות
- הסירו תגיות שאינן בשימוש
- עדכנו מוסכמות

## תכונות מתקדמות

### תיקיות חכמות מבוססות תגיות

צרו תיקיות שמתמלאות אוטומטית על סמך תגיות

1. צרו תיקייה חדשה
2. הגדירו מסנן: `tag:פרויקט`
3. כל ההערות המתויגות מופיעות

### תגיות ב-YAML

הוסיפו תגיות ל-frontmatter של ההערה

```yaml
---
tags:
  - עבודה
  - פגישות
  - פרויקט-אלפא
---
```

### כינויי תגיות

הגדירו כינויי תגיות בהגדרות

- `#לעשות` = `#משימה` = `#פעולה`
- אחדו תגיות דומות
