# פקודות סלאש

פקודות סלאש הן דרך חזקה להכניס במהירות עיצוב ותוכן מיוחד מבלי לעזוב את המקלדת. הקלידו `/` בעורך לפתיחת פלטת הפקודות.

## פקודות זמינות

### כותרות

| פקודה | תוצאה |
| ------- | ------ |
| `/h1` | כותרת רמה 1 |
| `/h2` | כותרת רמה 2 |
| `/h3` | כותרת רמה 3 |
| `/h4` | כותרת רמה 4 |
| `/h5` | כותרת רמה 5 |
| `/h6` | כותרת רמה 6 |

### עיצוב טקסט

| פקודה | תוצאה |
| ------- | ------ |
| `/bold` | **טקסט מודגש** |
| `/italic` | *טקסט נטוי* |
| `/strikethrough` | ~~קו חוצה~~ |
| `/code` | בלוק קוד בתוך השורה |
| `/quote` | ציטוט |

### רשימות

| פקודה | תוצאה |
| ------- | ------ |
| `/bullet` | • רשימת תבליטים |
| `/numbered` | 1. רשימה ממוספרת |
| `/todo` | - [ ] פריט משימה |
| `/check` | - [x] פריט שהושלם |

### הכנסת תוכן

| פקודה | תוצאה |
| ------- | ------ |
| `/image` | הכנסת תמונה מ-URL או קובץ |
| `/link` | הכנסת היפר-קישור |
| `/table` | הכנסת טבלה |
| `/hr` | קו אופקי |
| `/code` | בלוק קוד עם בחירת שפה |

### בלוקים מיוחדים

| פקודה | תוצאה |
| ------- | ------ |
| `/mermaid` | בלוק דיאגרמת Mermaid |
| `/katex` | בלוק מתמטיקה KaTeX |
| `/callout` | בלוק callout/אזהרה |

## שימוש בפקודות סלאש

1. הקלידו `/` בעורך
2. תופיע תפריט עם כל הפקודות הזמינות
3. התחילו להקליד לסינון פקודות
4. לחצו `Enter` או לחצו להכנסה

## טיפים למקצוענים

- **גישה מהירה**: פשוט הקלידו `/` והתחילו להקליד את שם הפקודה
- **חיפוש מטושטש**: אין צורך להקליד את שם הפקודה המדויק - Classic ימצא את ההתאמה הקרובה ביותר
- **ניווט במקלדת**: השתמשו במקשי החצים לניווט בתפריט
- **פקודות אחרונות**: פקודות בשימוש תכוף מופיעות בראש הרשימה
