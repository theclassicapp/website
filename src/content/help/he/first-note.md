# יצירת ההערה הראשונה שלכם

התחילו לעבוד עם Classic על ידי יצירת ההערה הראשונה שלכם. זה פשוט ואינטואיטיבי.

## יצירת הערה חדשה

### באמצעות סרגל הצד

1. לחצו על כפתור **+** בסרגל הצד
2. נוצרת הערה חדשה
3. התחילו להקליד את התוכן

### באמצעות קיצור מקלדת

| פלטפורמה | קיצור מקלדת |
| -------- | -------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### באמצעות פלטת פקודות

1. פתחו פלטת פקודות (`Cmd+P` או `Ctrl+P`)
2. הקלידו "הערה חדשה"
3. לחצו Enter

## כתיבת ההערה שלכם

### עריכה בסיסית

פשוט התחילו להקליד! Classic שומר אוטומטית בזמן שאתם כותבים.

### הוספת כותרת

השורה הראשונה של ההערה שלכם הופכת לכותרת. השתמשו בכותרת לתוצאות הטובות ביותר

```markdown
# ההערה הראשונה שלי

זהו התוכן של ההערה שלי.
```

### עיצוב טקסט

השתמשו ב-Markdown לעיצוב הטקסט

- **מודגש**: `**טקסט**` או `Cmd/Ctrl+B`
- **נטוי**: `*טקסט*` או `Cmd/Ctrl+I`
- **קו חוצה**: `~~טקסט~~`
- **קוד**: `` `קוד` ``

## שמירה

Classic שומר אוטומטית את ההערות שלכם בזמן שאתם מקלידים. אין צורך בשמירה ידנית.

### שמירה ידנית

אם צריך, אתם יכולים לאלץ שמירה

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## ארגון ההערה שלכם

### הוספת תגיות

תייגו את ההערה למציאה קלה בהמשך

1. הקלידו `#` ואחריו שם התגית
2. או השתמשו בכפתור התגית בסרגל הכלים
3. נתמכות תגיות מרובות

דוגמה

```markdown
# הערות פגישה

דיון על תכנון Q4 #עבודה #פגישות
```

### העברה לתיקייה

1. לחצו לחיצה ימנית על ההערה בסרגל הצד
2. בחרו "העבר לתיקייה"
3. בחרו או צרו תיקיית יעד

### הצמדת הערות חשובות

1. לחצו לחיצה ימנית על ההערה
2. בחרו "הצמד למעלה"
3. ההערה מופיעה בראש הרשימה

## תצוגה מקדימה של ההערה

ראו איך ההערה המעוצבת נראית

- לחצו על כפתור **תצוגה מקדימה**
- או השתמשו ב-`Cmd/Ctrl+Shift+P`
- עברו בין מצבי עריכה ותצוגה מקדימה

## הצעדים הבאים

עכשיו כשיצרתם את ההערה הראשונה

- למדו עוד על [Markdown](/help/markdown)
- ארגנו עם [תגיות](/help/tags)
- הגדירו [הערות יומיות](/help/daily-notes)
- חקרו [קיצורי מקלדת](/help/keyboard-shortcuts)

## טיפים

### כותרות הערות

- שמרו על כותרות תיאוריות
- שורה ראשונה = כותרת ההערה
- השתמשו ב-H1 (`#`) לבהירות

### שמירה אוטומטית

- לעולם אל תדאגו מאובדן עבודה
- שינויים נשמרים אוטומטית
- בדקו את שורת המצב לסטטוס שמירה

### לכידה מהירה

צריכים לרשום משהו מהר?

- השתמשו בלכידה מהירה (`Cmd/Ctrl+Shift+N`)
- יוצר הערה באופן מיידי
- מעולה למחשבות חולפות
