# ערכות נושא ומראה

התאימו אישית את המראה של Classic לסגנון שלכם והפחיתו עייפות עיניים.

## אפשרויות ערכת נושא

### מצב בהיר

ערכת הנושא הברירת מחדל של Classic כוללת

- ממשק נקי ובהיר
- אופטימלי לסביבות מוארות היטב
- ניגודיות גבוהה לקריאות

### מצב כהה

מושלם לינשופי לילה ועבודה בתאורה חלשה

- מפחית עייפות עיניים
- צריכת סוללה נמוכה יותר במסכי OLED
- קל יותר לעיניים בסביבות חשוכות

### מערכת (אוטומטי)

עקבו אחר ערכת הנושא של מערכת ההפעלה

- מחליף אוטומטית על סמך הגדרות מערכת ההפעלה
- אין צורך בהחלפה ידנית
- עקבי עם אפליקציות אחרות

## החלפת ערכות נושא

### דרך הגדרות

1. פתחו הגדרות (`Cmd+,` או `Ctrl+,`)
2. נווטו ל**מראה**
3. בחרו את ערכת הנושא המועדפת

### דרך פלטת פקודות

1. פתחו פלטת פקודות (`Cmd+P` או `Ctrl+P`)
2. הקלידו "ערכת נושא"
3. בחרו את ערכת הנושא המועדפת

### קיצור מקלדת

| פלטפורמה | קיצור מקלדת |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

מחזור: בהיר → כהה → מערכת

## הגדרות גופן

### גופן עורך

התאימו אישית את חוויית הכתיבה

1. פתחו הגדרות
2. עברו ל**מראה** → **עורך**
3. הגדירו אפשרויות גופן

| הגדרה | אפשרויות | ברירת מחדל |
| ------- | ------- | ------- |
| משפחת גופן | גופני מערכת + מותאם אישית | מערכת UI |
| גודל גופן | 12-24px | 16px |
| גובה שורה | 1.0-2.5 | 1.75 |
| ריווח אותיות | -2 עד 4 | 0 |

### גופן UI

התאימו את גופן הממשק

1. פתחו הגדרות
2. עברו ל**מראה** → **ממשק**
3. בחרו גופן וגודל

### גופן Monospace

לבלוקי קוד ותוכן טכני

1. פתחו הגדרות
2. עברו ל**מראה** → **קוד**
3. בחרו גופן monospace

## אפשרויות ממשק

### סרגל צד

התאימו אישית את התנהגות סרגל הצד

| הגדרה | תיאור |
| ------- | ----------- |
| **הסתרה אוטומטית** | הסתר סרגל צד כשלא בשימוש |
| **רוחב** | התא רוחב סרגל צד (200-400px) |
| **הצג סמלים** | החלף נראות סמלים |

### פריסת עורך

בחרו את חוויית העריכה שלכם

- **תצוגה מפוצלת**: עריכה ותצוגה מקדימה זה לצד זה
- **תצוגה מקדימה בלבד**: ראו פלט מעובד בלבד
- **עריכה בלבד**: התמקדו ב-Markdown גולמי

### רשימת הערות

הגדירו כיצד הערות מופיעות

- **תצוגת רשימה**: קומפקטית, שורה אחת לכל הערה
- **תצוגת כרטיסים**: כרטיסי תצוגה מקדימה עם קטעים
- **תצוגת עץ**: מבנה תיקיות היררכי

## צבעי הדגשה

### צבעים זמינים

בחרו את צבע ההדגשה שלכם

- כחול Classic (ברירת מחדל)
- סגול
- ירוק
- כתום
- אדום
- ורוד
- טורקיז

### שינוי צבע הדגשה

1. פתחו הגדרות
2. עברו ל**מראה** → **צבעים**
3. בחרו את ההדגשה המועדפת

צבע ההדגשה משפיע על

- קישורים וכפתורים
- הדגשות בחירה
- אלמנטים פעילים
- מחווני מיקוד

## נגישות

### מצב ניגודיות גבוהה

לראות טוב יותר

1. פתחו הגדרות
2. עברו ל**מראה** → **נגישות**
3. הפעילו **ניגודיות גבוהה**

### תנועה מופחתת

מזערו אנימציות

1. פתחו הגדרות
2. עברו ל**מראה** → **נגישות**
3. הפעילו **תנועה מופחתת**

### התאמות גופן

- הגדילו גודל גופן מינימלי
- השתמשו בגופנים ידידותיים לדיסלקסיה
- התאימו יחסי ניגודיות

## CSS מותאם אישית (מתקדם)

למשתמשי על שרוצים שליטה מלאה

1. פתחו הגדרות
2. עברו ל**מראה** → **מתקדם**
3. לחצו **פתח CSS מותאם אישית**
4. הוסיפו את ההתאמות האישיות שלכם

### דוגמאות התאמה אישית

```css
/* כותרות גדולות יותר */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* צבע קישור מותאם אישית */
a { color: #3b82f6; }

/* ריווח שורות רחב יותר */
.prose p { line-height: 2; }
```

## סנכרון ערכת נושא

כשאתם נכנסים (אופציונלי), העדפות ערכת הנושא שלכם מסתנכרנות בין מכשירים

- סנכרון ערכת נושא אוטומטי
- סנכרון הגדרות גופן
- CSS מותאם אישית מסתנכרן

## טיפים

### התאמה לזרימת העבודה

- השתמשו ב**בהיר** במהלך היום, **כהה** בלילה
- מצב **מערכת** מטפל בזה אוטומטית
- שקלו את תאורת הסביבה שלכם

### הפחתת עייפות עיניים

- השתמשו במצב כהה בתאורה חלשה
- הגדילו גודל גופן למפגשים ארוכים
- הפעילו ניגודיות גבוהה אם צריך
- קחו הפסקות קבועות

### פרודוקטיביות

- למדו את קיצור החלפת ערכת הנושא
- הגדירו זמני החלפה אוטומטית
- השתמשו במצב מערכת לעקביות
