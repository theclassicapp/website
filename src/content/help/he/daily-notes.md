# הערות יומיות

הערות יומיות יוצרות אוטומטית הערה חדשה לכל יום, מושלמות ליומן, תכנון יומי, או תיעוד מידע מבוסס זמן.

## איך זה עובד

### יצירה אוטומטית

- נוצרת הערה חדשה לכל יום
- הערות נקראות בפורמט תאריך (למשל, `2024-01-15.md`)
- הערות של ימים קודמים נשמרות וניתנות לחיפוש

### גישה להערות יומיות

1. לחצו על **הערות יומיות** בסרגל הצד
2. או לחצו `Ctrl/Cmd+D`
3. ההערה של היום נפתחת אוטומטית

## תבנית הערה יומית

התאימו אישית את התבנית להערות יומיות חדשות

### תבנית ברירת מחדל

```markdown
# {{date}}

## משימות
- [ ]
- [ ]

## הערות


## יומן

```

### התאמה אישית

1. פתחו הגדרות
2. עברו להערות יומיות
3. ערכו את התבנית
4. השתמשו ב-`{{date}}` לתאריך הנוכחי

### משתני תבנית

| משתנה | תיאור |
| -------- | ----------- |
| `{{date}}` | תאריך נוכחי (YYYY-MM-DD) |
| `{{time}}` | שעה נוכחית (HH:mm) |
| `{{weekday}}` | שם היום (ראשון, וכו') |

## תכונות

### ניווט בתאריכים

- **הקודם/הבא**: נווטו בין ימים
- **לוח שנה**: קפצו לתאריכים ספציפיים
- **חיפוש**: מצאו תוכן בכל ההערות היומיות

### קישור אוטומטי

- הערות שנוצרו באותו יום מקושרות אוטומטית
- קישורים חוזרים מציגים חיבורים בין ימים
- תגיות עובדות בכל ההערות היומיות

### אינטגרציה

- **לכידה מהירה**: הוסיפו להערה של היום עם `#היום`
- **תצוגת לוח שנה**: סקירה חזותית של ההערות היומיות
- **סטטיסטיקות**: עקבו אחרי רצפי הכתיבה שלכם

## שיטות עבודה מומלצות

### יומן יומי

```markdown
# 2024-01-15

## בוקר
- קמתי ב-6:30
- הרגשה: אנרגטי

## משימות
- [x] אימון בוקר
- [ ] סקירת PRs
- [ ] פגישת צוות ב-14:00

## למידות
- גיליתי קיצור מקלדת חדש
- למדתי על דיאגרמות Mermaid

## מחר
- התמקדות בתיעוד
- תזמון סקירת קוד
```

### הערות פגישה

```markdown
# 2024-01-15

## סטנדאפ צוות

**משתתפים**: אליס, בוב, קרול

### עדכונים
- אליס: סיימה אינטגרציית API
- בוב: עובד על רכיבי UI
- קרול: בדיקות בתהליך

### חסמים
- מחכים לסקירת עיצוב

### פריטי פעולה
- [ ] אליס: פריסה ל-staging
- [ ] בוב: השלמת ספריית רכיבים
- [ ] קרול: כתיבת בדיקות אינטגרציה
```

### יומן למידה

```markdown
# 2024-01-15

## היום למדתי

### דיאגרמות Mermaid
- יכול ליצור תרשימי זרימה ב-Markdown
- התחביר פשוט
- התצוגה המקדימה בזמן אמת עוזרת

### משאבים
- [תיעוד Mermaid](https://mermaid.js.org/)

## שאלות
- איך ליצור דיאגרמות מורכבות?
- שיטות עבודה מומלצות לארגון דיאגרמות?
```

## טיפים

### עקביות

- צרו הערות באותו זמן כל יום
- השתמשו בעיצוב עקבי
- סקרו ועדכנו לאורך היום

### ארגון

- השתמשו בכותרות למקטעים שונים
- הוסיפו תגיות לחיפוש קל
- קשרו להערות קשורות

### סקירה

- סקירה שבועית של הערות יומיות
- הוציאו פריטים חשובים להערות קבועות
- ארכבו או סכמו הערות ישנות
