# קיצורי מקלדת

עבדו מהר יותר עם קיצורי המקלדת של Classic. כל הקיצורים ניתנים להתאמה אישית בהגדרות.

## קיצורים גלובליים

אלה עובדים בכל מקום באפליקציה.

### כללי

| קיצור (Windows/Linux) | קיצור (macOS) | פעולה |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | הערה חדשה |
| `Ctrl+P` | `Cmd+P` | פלטת פקודות |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | לכידה מהירה גלובלית |
| `Ctrl+S` | `Cmd+S` | שמירת הערה |
| `Ctrl+F` | `Cmd+F` | חיפוש |
| `Ctrl+,` | `Cmd+,` | הגדרות |

### ניווט

| קיצור (Windows/Linux) | קיצור (macOS) | פעולה |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | עבור להערה הראשונה |
| `Ctrl+2` | `Cmd+2` | עבור להערה השנייה |
| `Ctrl+Left` | `Cmd+Left` | נווט אחורה |
| `Ctrl+Right` | `Cmd+Right` | נווט קדימה |
| `Ctrl+Up` | `Cmd+Up` | נווט להערת אב |
| `Ctrl+Down` | `Cmd+Down` | הרחב/מקד תת-הערה ראשונה |

## לכידה מהירה

| קיצור (Windows/Linux) | קיצור (macOS) | פעולה |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | פתיחת לכידה מהירה |
| `Escape` | `Escape` | סגירת לכידה מהירה |
| `Ctrl+Enter` | `Cmd+Enter` | שמירה וסגירה |

## התאמה אישית של קיצורים

1. פתחו **הגדרות** (`Ctrl+,` או `Cmd+,`)
2. נווטו ל**קיצורי מקלדת**
3. לחצו על כל קיצור לשינוי
4. לחצו את צירוף המקשים החדש
5. לחצו **שמור**

## טיפים

- רוב הקיצורים ניתנים להתאמה אישית
- קונפליקטים עם קיצורי מערכת יודגשו
- אפסו לברירת מחדל על ידי לחיצה על כפתור האיפוס ליד כל קיצור
