# הגדרות

התאימו אישית את Classic לעבוד בדיוק בדרך שאתם רוצים. ממראה להתנהגות עורך, הכל ניתן להגדרה.

## פתיחת הגדרות

### קיצור מקלדת

| פלטפורמה | קיצור מקלדת |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### דרך תפריט

- **macOS**: Classic → הגדרות
- **Windows**: קובץ → הגדרות
- **Linux**: עריכה → הגדרות

### דרך פלטת פקודות

1. פתחו פלטת פקודות (`Cmd+P` או `Ctrl+P`)
2. הקלידו "הגדרות"
3. לחצו Enter

## קטגוריות הגדרות

### כללי

הגדרות אפליקציה בסיסיות

| הגדרה | תיאור |
|---------|-------------|
| **שפה** | שפת ממשק |
| **ערכת נושא** | בהיר, כהה, או מערכת |
| **התחל בהתחברות** | הפעלה באתחול |
| **בדוק עדכונים** | התנהגות עדכון אוטומטי |

### מראה

התאימו אישית את המראה והתחושה

| הגדרה | אפשרויות |
|---------|---------|
| **ערכת נושא** | בהיר / כהה / מערכת |
| **צבע הדגשה** | כחול, סגול, ירוק וכו' |
| **גופן עורך** | משפחת גופן וגודל |
| **גופן UI** | גופן ממשק |
| **גופן Monospace** | גופן קוד |
| **רוחב סרגל צד** | 200-400px |
| **גובה שורה** | 1.0-2.5 |

### עורך

הגדירו התנהגות כתיבה

| הגדרה | תיאור |
|---------|-------------|
| **בדיקת איות** | הפעל/כבה בדיקת איות |
| **שמירה אוטומטית** | מרווח שמירה אוטומטית |
| **גלישת מילים** | הפעל/כבה גלישת מילים |
| **מספור שורות** | הצג/הסתר מספור שורות |
| **גודל Tab** | 2, 4, או 8 רווחים |
| **תיקון אוטומטי** | מרכאות חכמות, החלפה אוטומטית |

### היסטוריית גרסאות

שלטו במעקב גרסאות

| הגדרה | אפשרויות | ברירת מחדל |
|---------|---------|---------|
| **הפעל היסטוריה** | מופעל/מכובה | מופעל |
| **תקופת שמירה** | 30-180 יום | 90 יום |
| **מקסימום גרסאות** | 50-500 | 100 |

### גיבוי ושחזור

הגדרות הגנת נתונים

| הגדרה | תיאור |
|---------|-------------|
| **גיבוי אוטומטי** | הפעל גיבויים אוטומטיים |
| **תדירות גיבוי** | 15 דקות / 30 דקות / שעה / 4 שעות |
| **מיקום גיבוי** | נתיב מותאם אישית |
| **שמור גיבויים** | 5 / 10 / 20 / 50 |
| **הצפן גיבויים** | הגנת סיסמה |

### קיצורי מקלדת

צפו והתאימו אישית קיצורים

- ראו את כל הקיצורים הזמינים
- התאימו אישית כל קיצור
- אפס לברירות מחדל
- ייבוא/ייצוא קיצורים

### ייבוא וייצוא

הגדרות העברת נתונים

- ייבוא מאפליקציות אחרות
- אפשרויות פורמט ייצוא
- פעולות כמותיות

### פרטיות

הגדרות פרטיות ואבטחה

| הגדרה | תיאור |
|---------|-------------|
| **אנליטיקס** | הפעל/כבה אנליטיקס שימוש |
| **דוחות קריסה** | שלח דוחות קריסה |
| **נקה היסטוריה** | נקה חיפוש/היסטוריה |

### התראות

שלטו בהתראות

| הגדרה | תיאור |
|---------|---------|
| **הפעל התראות** | מופעל/מכובה |
| **תזכורת הערה יומית** | זמן לתזכורת |
| **צליל** | צליל התראה |

### מתקדם

הגדרות משתמש על

| הגדרה | תיאור |
|---------|-------------|
| **CSS מותאם אישית** | הוסף סגנונות מותאמים |
| **מצב איתור באגים** | הפעל רישום איתור באגים |
| **תכונות ניסיוניות** | תכונות בטא |
| **מיקום נתונים** | שנה תיקיית הערות |

## חיפוש הגדרות

לא מוצאים הגדרה? השתמשו בסרגל החיפוש

1. פתחו הגדרות
2. הקלידו בתיבת החיפוש
3. הגדרות תואמות מופיעות
4. לחצו לקפיצה לאותה הגדרה

## איפוס הגדרות

### איפוס כל ההגדרות

חזרו לברירות מחדל

1. פתחו הגדרות
2. גללו לתחתית
3. לחצו "אפס את כל ההגדרות"
4. אשרו את הפעולה

### איפוס קטגוריה

אפסו קטגוריה ספציפית

1. עברו לקטגוריה
2. לחצו "אפס לברירות מחדל"
3. רק אותה קטגוריה מתאפסת

## ייצוא/ייבוא הגדרות

### ייצוא הגדרות

שמרו את הקונפיגורציה שלכם

1. פתחו הגדרות
2. לחצו "ייצא הגדרות"
3. בחרו מיקום
4. שמרו כקובץ JSON

### ייבוא הגדרות

טענו קונפיגורציה שמורה

1. פתחו הגדרות
2. לחצו "ייבוא הגדרות"
3. בחרו את קובץ ה-JSON
4. אשרו ייבוא

## מיקום הגדרות

הגדרות מאוחסנות מקומית

| פלטפורמה | מיקום |
|----------|----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## סנכרון הגדרות (אופציונלי)

כשנכנסים, הגדרות יכולות להסתנכרן בין מכשירים

- **סנכרון**: הפעילו בהגדרות → סנכרון
- **מה מסתנכרן**: ערכת נושא, גופנים, קיצורים
- **מה לא מסתנכרן**: נתיבים מקומיים, מפתחות הצפנה

## התאמות אישיות נפוצות

### הגדלת גודל גופן

1. הגדרות → מראה
2. גודל גופן עורך → הגדילו
3. השינויים חלים מיידית

### שינוי ערכת נושא

1. הגדרות → מראה
2. ערכת נושא → בחרו בהיר/כהה/מערכת
3. האפליקציה מתעדכנת מיידית

### הפעלת גיבוי אוטומטי

1. הגדרות → גיבוי ושחזור
2. הפעילו "גיבוי אוטומטי"
3. הגדירו תדירות ומיקום

### התאמה אישית של קיצורים

1. הגדרות → קיצורי מקלדת
2. מצאו את הפעולה
3. לחצו על הקיצור הנוכחי
4. לחצו מקשים חדשים
5. שמרו

## טיפים

### גישה מהירה

- השתמשו ב-`Cmd/Ctrl+,` לפתיחת הגדרות במהירות
- חפשו הגדרות לפי שם
- השתמשו בקטגוריות לניווט

### ייצוא לפני שינויים

לפני שינויים משמעותיים, יצאו את ההגדרות שלכם

- קל לשחזר אם משהו משתבש
- שתפו קונפיגורציה עם אחרים

### בדקו הגדרות חדשות

עדכונים עשויים להוסיף הגדרות חדשות

- בדקו לאחר עדכון
- חקרו אפשרויות חדשות

## פתרון בעיות

### הגדרות לא נשמרות

- בדקו הרשאות קבצים
- ודאו מקום בדיסק
- נסו לאפס ולהגדיר מחדש

### הגדרות מתאפסות באופן בלתי צפוי

- עשוי להצביע על שחיתות קבצים
- יצאו הגדרות באופן קבוע
- בדקו קונפליקטים עם אפליקציות

### לא מוצאים הגדרה

- השתמשו בסרגל החיפוש
- בדקו קטגוריות שונות
- עשוי להיות בהגדרות מתקדמות
