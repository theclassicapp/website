# הבנת הממשק

הממשק של Classic נועד להיות נקי, אינטואיטיבי, ולא להפריע לכם. הנה סיור ברכיבים העיקריים.

## פריסה ראשית

ל-Classic יש פריסת שלושה חלונות

1. **סרגל צד** (שמאל) - ניווט וארגון הערות
2. **עורך** (מרכז) - מרחב הכתיבה שלכם
3. **תצוגה מקדימה/מאפיינים** (ימין) - חלונות אופציונליים

## סרגל צד

סרגל הצד מכיל מספר מקטעים מרכזיים

### רשימת הערות

- **כל ההערות**: צפו בכל ההערות שלכם ברשימה אחת
- **הערות יומיות**: גישה מהירה ליומן היומי
- **מועדפים**: סמנו הערות בשימוש תכוף בכוכב
- **תיקיות**: ארגנו הערות בהיררכיות

### ניווט

- **סרגל חיפוש**: מצאו הערות באופן מיידי
- **תגיות**: עיון לפי תגית
- **קישורים חוזרים**: ראו חיבורים בין הערות

### פעולות

- **הערה חדשה**: צרו הערה חדשה
- **תיקייה חדשה**: ארגנו עם תיקיות
- **ייבוא**: הביאו תוכן חיצוני

## העורך

העורך הוא המקום שבו אתם כותבים ומעצבים את ההערות שלכם.

### תכונות

- **תמיכה ב-Markdown**: תחביר Markdown מלא
- **תצוגה מקדימה בזמן אמת**: ראו טקסט מעוצב בזמן שאתם מקלידים
- **הדגשת תחביר**: בלוקי קוד מודגשים
- **פקודות סלאש**: הקלידו `/` לפעולות מהירות

### מצבי עורך

| מצב | תיאור |
|------|-------------|
| **עריכה בלבד** | ראו Markdown גולמי |
| **תצוגה מקדימה בלבד** | ראו פלט מעובד |
| **תצוגה מפוצלת** | עריכה ותצוגה מקדימה זה לצד זה |

### סרגל כלים

פעולות עיצוב מהירות

- מודגש, נטוי, קו חוצה
- כותרות (H1-H6)
- רשימות (תבליטים, ממוספרות, משימות)
- קישורים ותמונות
- בלוקי קוד
- טבלאות

## שורת מצב

בתחתית העורך

- **ספירת מילים/תווים**: עקבו אחרי הכתיבה שלכם
- **נשמר לאחרונה**: מתי ההערה נשמרה לאחרונה
- **נתיב ההערה**: היכן ההערה נמצאת
- **סטטוס סנכרון**: אם משתמשים בסנכרון אופציונלי

## תפריטי הקשר

לחיצה ימנית לפעולות הקשריות

### תפריט הקשר של הערה

- שינוי שם
- העברה לתיקייה
- שכפול
- מחיקה
- ייצוא
- העתקת קישור

### תפריט הקשר של העורך

- גזירה, העתקה, הדבקה
- אפשרויות עיצוב
- הוספת קישור/תמונה
- חיפוש והחלפה

## ניווט במקלדת

נווטו בממשק עם המקלדת

| פעולה | קיצור מקשים |
|--------|----------|
| החלפת סרגל צד | `Cmd/Ctrl + M` |
| מיקוד בחיפוש | `Cmd/Ctrl + F` |
| פלטת פקודות | `Cmd/Ctrl + P` |
| הערה חדשה | `Cmd/Ctrl + N` |

## התאמה אישית

התאימו את הממשק לטעמכם

### רוחב סרגל צד

1. רחפו מעל קצה סרגל הצד
2. גררו לשינוי גודל
3. או הגדירו רוחב מדויק בהגדרות

### ערכת נושא

עברו בין מצב בהיר לכהה

- השתמשו במתג ערכת הנושא בכותרת
- או הגדירו לעקוב אחר העדפת המערכת

### גודל גופן

1. פתחו הגדרות
2. עברו למראה
3. התאימו גודל גופן העורך

## טיפים

### מצב מיקוד

הסתירו את כל החלונות לכתיבה ללא הסחות דעת

1. פתחו פלטת פקודות
2. הקלידו "מצב מיקוד"
3. לחצו Enter

### גישה מהירה

- הצמידו הערות בשימוש תכוף למועדפים
- השתמשו בקיצורי מקלדת לפעולות נפוצות
- התאימו אישית את סרגל הכלים בהגדרות
