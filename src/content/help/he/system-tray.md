# מגש מערכת

Classic חי במגש המערכת שלכם לגישה מהירה. לכדו מחשבות באופן מיידי מבלי להפריע לזרימת העבודה שלכם.

## מהו מגש המערכת?

מגש המערכת (Windows/Linux) או שורת התפריטים (macOS) הוא המקום שבו Classic רץ ברקע, תמיד מוכן ללכידה מהירה.

### יתרונות

- **תמיד זמין**: רק לחיצה אחת משם
- **לכידה מהירה**: רשמו מחשבות באופן מיידי
- **ריצה ברקע**: אין צורך להחזיק את האפליקציה פתוחה
- **התראות**: קבלו תזכורות למשימות

## הפעלת מגש מערכת

### macOS

1. פתחו את Classic
2. עברו להגדרות → כללי
3. הפעילו "הצג בשורת התפריטים"
4. סמל Classic מופיע בשורת התפריטים

### Windows

1. פתחו את Classic
2. עברו להגדרות → כללי
3. הפעילו "הצג במגש המערכת"
4. סמל Classic מופיע במגש המערכת

### Linux

1. פתחו את Classic
2. עברו להגדרות → כללי
3. הפעילו "סמל מגש מערכת"
4. תלוי בסביבת שולחן העבודה

## לכידה מהירה

### פתיחת לכידה מהירה

לחצו על סמל Classic במגש המערכת

- **macOS**: סמל שורת התפריטים
- **Windows**: סמל מגש המערכת
- **Linux**: סמל מגש המערכת

### שימוש בלכידה מהירה

1. לחצו על סמל מגש המערכת
2. חלון לכידה מהירה נפתח
3. הקלידו את ההערה שלכם
4. לחצו Enter לשמירה
5. החלון נסגר אוטומטית

### קיצור מקלדת

פתחו לכידה מהירה מכל מקום

| פלטפורמה | קיצור מקלדת |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (גלובלי) |
| **Windows** | `Ctrl+Shift+N` (גלובלי) |
| **Linux** | `Ctrl+Shift+N` (גלובלי) |

## תפריט מגש מערכת

לחצו לחיצה ימנית על סמל המגש לאפשרויות

### אפשרויות תפריט

| אפשרות | פעולה |
|--------|--------|
| **הערה חדשה** | פתח לכידה מהירה |
| **הערה יומית חדשה** | פתח הערה יומית של היום |
| **חיפוש** | פתח חלון חיפוש |
| **פתח את Classic** | פתח חלון ראשי |
| **הגדרות** | פתח הגדרות |
| **יציאה** | צא מ-Classic |

## הגדרות מגש

### הגדר בהגדרות

| הגדרה | תיאור |
|---------|-------------|
| **הצג סמל** | החלפת נראות סמל המגש |
| **התחל ממוזער** | התחל את Classic במגש בלבד |
| **סגור למגש** | המשך לרוץ כשהחלון סגור |
| **התראות** | הפעל/כבה התראות |

### התחלה בהתחברות

גרמו ל-Classic להתחיל אוטומטית

1. פתחו הגדרות
2. עברו לכללי
3. הפעילו "התחל בהתחברות"
4. Classic מופעל בהפעלה

## התראות

### הפעלת התראות

1. פתחו הגדרות
2. עברו להתראות
3. החליפו "הפעל התראות"

### סוגי התראות

- **תזכורות**: תזכורות מתוזמנות
- **הערה יומית**: הנחיות הערה יומית
- **עדכונים**: עדכון אפליקציה זמין

## התנהגות חלון

### התנהגות סגירה

בחרו מה קורה כשאתם סוגרים את החלון

- **מזער למגש**: המשך לרוץ, הסתר חלון
- **יציאה**: צא מהאפליקציה
- **שאל**: שאל כל פעם

### התנהגות מזעור

- **מזער למגש**: הסתר במגש המערכת
- **מזער לשורת המשימות**: מזער נורמלית

## טיפים

### לכידה מהירה

השתמשו בקיצור גלובלי ללכידה המהירה ביותר

1. לחצו `Cmd/Ctrl+Shift+N` בכל מקום
2. הקלידו את המחשבה שלכם
3. לחצו Enter
4. המשיכו בעבודה שלכם

### גישה מהירה

לחצו על סמל המגש לראות

- הערות אחרונות
- פעולות מהירות
- ההערה היומית של היום

### ריצה ברקע

החזיקו את Classic רץ ל

- הפעלה מהירה יותר
- לכידה מהירה
- התראות
- גיבוי אוטומטי

## פתרון בעיות

### סמל מגש חסר

**macOS**

1. בדקו אם הסמל מוסתר
2. חפשו במרכז הבקרה (macOS Big Sur+)
3. הפעילו מחדש את Classic

**Windows**

1. בדקו סמלים מוסתרים (חץ בשורת המשימות)
2. גררו את סמל Classic לאזור הנראה
3. הפעילו מחדש את Classic

**Linux**

1. בדקו תמיכה בסביבת שולחן העבודה
2. התקינו הרחבת מגש מערכת אם צריך
3. הפעילו מחדש את Classic

### קיצור גלובלי לא עובד

- בדקו קונפליקטים עם אפליקציות אחרות
- הפעילו מחדש את הקיצור בהגדרות
- הפעילו מחדש את Classic
- בדקו הרשאות מערכת הפעלה

### התראות לא מופיעות

**macOS**

1. העדפות מערכת → התראות
2. מצאו את Classic
3. הפעילו התראות

**Windows**

1. הגדרות → מערכת → התראות
2. הפעילו התראות Classic

**Linux**

1. בדקו daemon התראות
2. התקינו libnotify אם צריך

### שימוש גבוה בזיכרון/CPU

אם סמל המגש משתמש ביותר מדי משאבים

1. כבו אנימציות בהגדרות
2. הפחיתו תדירות התראות
3. בדקו עדכוני אפליקציה
