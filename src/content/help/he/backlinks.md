# קישורים חוזרים

קישורים חוזרים מציגים חיבורים בין ההערות שלכם, ויוצרים רשת של רעיונות קשורים. גלו יחסים שלא ידעתם על קיומם.

## מהם קישורים חוזרים?

קישורים חוזרים מציגים אילו הערות מקשרות להערה הנוכחית. הם חיבורים בגרף הידע שלכם.

### יתרונות

- **גילוי חיבורים**: מצאו הערות קשורות אוטומטית
- **הקשר**: הבינו כיצד רעיונות קשורים
- **ניווט**: קפצו בין הערות מחוברות
- **גרף ידע**: דמיינו את החשיבה שלכם

## איך קישורים חוזרים עובדים

כשאתם יוצרים קישור להערה אחרת, Classic יוצר אוטומטית קישור חוזר.

```markdown
בהערה א':
ראו [[תכנון פרויקט]] לפרטים.

תוצאה:
- הערה א' מקשרת ל"תכנון פרויקט"
- "תכנון פרויקט" מציג קישור חוזר להערה א'
```

## צפייה בקישורים חוזרים

### חלונית קישורים חוזרים

פתחו את חלונית הקישורים החוזרים לראות את כל ההפניות

1. פתחו כל הערה
2. לחצו על סמל הקישורים החוזרים בסרגל הצד
3. ראו את כל ההערות המקשרות לזו

### קישורים חוזרים בתוך השורה

קישורים חוזרים מופיעים בתחתית כל הערה

- מציג שם הערה המקור
- מציג קטע הקשר
- לחצו לניווט

## יצירת קישורים

### תחביר ויקי-קישור

השתמשו בסוגריים מרובעים כפולים לקישור להערות

```markdown
[[שם הערה]]
[[הערה אחרת]]
```

### טקסט תצוגה

התאימו אישית טקסט קישור

```markdown
[[פרויקט אלפא|פרויקט האלפא]]
[[הערות פגישה|הפגישה של אתמול]]
```

### קישור לכותרות

קשרו למקטעים ספציפיים

```markdown
[[שם הערה#כותרת]]
[[שם הערה#מקטע|קפצו למקטע]]
```

## סוגי קישורים חוזרים

### קישורים ישירים

ויקי-קישורים מפורשים שאתם יוצרים

```markdown
קשור: [[הערה דומה]]
```

### אזכורים לא מקושרים

הערות שמזכירות את הכותרת אבל לא מקושרות

```markdown
הערה מזכירה "פרויקט אלפא" אבל לא מקושרת
```

Classic מציע להמיר אלה לקישורים.

## חלונית קישורים חוזרים

### תכונות

- **סינון**: חפשו בקישורים חוזרים
- **מיון**: לפי תאריך, שם, או רלוונטיות
- **תצוגה מקדימה**: רחפו לתצוגה מקדימה של תוכן
- **הקשר**: ראו טקסט מקיף

### פעולות

- לחצו לפתיחת הערה מקושרת
- לחיצה ימנית לאפשרויות
- המירו אזכורים לא מקושרים

## מקרי שימוש

### ניהול ידע

בנו ויקי אישי

```markdown
# למידת מכונה

קשור ל [[בינה מלאכותית]]
מבוסס על [[סטטיסטיקה]]
משמש ב [[עיבוד שפה טבעית]]
```

### תיעוד פרויקט

חברו מסמכים קשורים

```markdown
# עיצוב API

מפנה ל [[סכימת מסד נתונים]]
תלוי ב [[זרימת אימות]]
ראו גם [[אינטגרציית Frontend]]
```

### מחקר

קשרו מושגים ומקורות

```markdown
# הערות מחקר

מבוסס על [[מאמר מקור א']]
סותר את [[מאמר מקור ב']]
תומך ב [[השערה 1]]
```

### הערות פגישה

קשרו פגישות לפרויקטים

```markdown
# פגישה: תכנון Q4

פרויקט: [[פרויקט אלפא]]
קודם: [[סקירת Q3]]
הבא: [[התחלת Q4]]
```

## תצוגת גרף

דמיינו את הקישורים החוזרים שלכם כרשת

1. פתחו תצוגת גרף (`Cmd/Ctrl+G`)
2. ראו את כל ההערות כצמתים
3. קווים מציגים חיבורים
4. לחצו על צמתים לניווט

### בקרות גרף

- **זום**: גללו להתקרבות
- **פאן**: גררו להזזה
- **סינון**: הצג/הסתר לפי תגית
- **מיקוד**: הדגישו חיבורים של הערה

## טיפים

### קשרו בנדיבות

צרו קישורים בכל פעם שאתם מפנים למושג אחר

```markdown
✅ טוב:
דנו ב[[מפת הדרכים של Q4]] ו[[הקצאת תקציב]].

❌ הימנעו:
דנו במפת הדרכים של Q4 והקצאת תקציב.
```

### השתמשו בשמות עקביים

התאימו כותרות הערות בדיוק לקישורים אמינים

```markdown
✅ [[הערות פגישה]]
❌ [[הערות פגישה]] (רישיות משנה)
```

### סקרו קישורים חוזרים

בדקו קישורים חוזרים מדי פעם כדי

- לגלות חיבורים שנשכחו
- למצוא הערות למיזוג
- לזהות פערי ידע

### תצוגה מקדימה של קישורים

רחפו מעל קישורים לתצוגה מקדימה מבלי לפתוח

- ראו מספר שורות ראשונות
- בדקו אם זו ההערה הנכונה
- הפניה מהירה

## קיצורי מקלדת

| פעולה | קיצור מקלדת |
|--------|----------|
| פתיחת קישורים חוזרים | `Cmd/Ctrl+Shift+B` |
| יצירת קישור | `[[` (השלמה אוטומטית) |
| פתיחת תצוגת גרף | `Cmd/Ctrl+G` |
| ניווט בקישורים | `Tab` דרך קישורים |

## תכונות מתקדמות

### הטמעה

הטמיעו תוכן מהערות אחרות

```markdown
![[שם הערה]]
![[שם הערה#מקטע]]
```

### טרנסקלוזיה

כללו תוכן בתוך השורה

```markdown
## תוכן קשור
![[תבנית משותפת#כותרת]]
```

### צביעת קישורים

קישורים משנים צבע על סמך

- **כחול**: קישור שלא ביקרתם בו
- **סגול**: קישור שביקרתם בו
- **אדום**: קישור שבור (ההערה לא קיימת)

## פתרון בעיות

### קישורים חוזרים חסרים

- בדקו תחביר קישור `[[שם הערה]]`
- ודאו שכותרת ההערה תואמת בדיוק
- רעננו את חלונית הקישורים החוזרים

### קישורים שבורים

- ייתכן שההערה שונתה בשם
- ייתכן שההערה נמחקה
- השתמשו בחיפוש למציאת ההערה הנכונה

### קישורים כפולים

- אחדו הערות דומות
- השתמשו בהפניות להערות ששינו שם
- בדקו וריאציות כותרת
