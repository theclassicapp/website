# תת-הערות

תת-הערות מאפשרות לכם ליצור הערות מקוננות והיררכיות בתוך Classic. ארגנו נושאים מורכבים לחלקים ניתנים לניהול.

## מהן תת-הערות?

תת-הערות הן הערות בתוך הערות - דרך ליצור מבני תוכן היררכיים.

### יתרונות

- **ארגון**: פרקו נושאים מורכבים לחלקים קטנים יותר
- **קינון**: צרו רמות מרובות של היררכיה
- **ניווט**: קל לנווט בין הערות קשורות
- **גמישות**: הרחיבו וכווצו לפי הצורך

## יצירת תת-הערות

### שיטה 1: גרירה

צרו תת-הערות על ידי גרירת תוכן תחת הערת אב

1. צרו הערת אב
2. צרו הערה חדשה
3. גררו את ההערה החדשה על האב
4. היא הופכת לתת-הערה

### שיטה 2: תפריט הקשר

1. לחצו לחיצה ימנית על הערה
2. בחרו "תת-הערה חדשה"
3. נוצרת הערה מקוננת

### שיטה 3: קיצור מקלדת

1. בחרו הערה
2. לחצו `Cmd/Ctrl+Shift+N`
3. נוצרת תת-הערה

## ניהול תת-הערות

### הרחבה וכיווץ

לחצו על החץ ליד הערה עם תת-הערות

- **הרחבה**: הציגו את כל ההערות המקוננות
- **כיווץ**: הסתירו הערות מקוננות

### ניווט

- לחצו לפתיחת תת-הערה
- השתמשו במקשי החצים לניווט בהיררכיה
- פירורי לחם מציגים את המיקום שלכם

### העברת תת-הערות

גררו ושחררו לארגון מחדש

1. לחצו והחזיקו את תת-ההערה
2. גררו למיקום חדש
3. שחררו למיקום מחדש

### המרה להערה ראשית

קדמו תת-הערה לרמה העליונה

1. לחצו לחיצה ימנית על תת-ההערה
2. בחרו "העבר לשורש"
3. ההערה הופכת להערה ראשית

## עומק תת-הערות

Classic תומך ברמות מרובות של קינון

```
📁 פרויקט
  📄 סקירה כללית
  📁 מחקר
    📄 ראיונות משתמשים
    📄 תוצאות סקר
    📁 ניתוח
      📄 ממצאים מרכזיים
      📄 המלצות
  📁 תכנון
    📄 לוח זמנים
    📄 משאבים
```

### שיטות עבודה מומלצות

- **הגבילו עומק**: 3-4 רמות מקסימום לבהירות
- **קיבוץ לוגי**: קבצו תוכן קשור יחד
- **שמות ברורים**: השתמשו בשמות תיאוריים לתת-הערות

## קישור עם תת-הערות

### קישורים חוזרים

תת-הערות יוצרות אוטומטית קישורים חוזרים להערות אב

- ראו מקטע "הפניות מקושרות"
- נווטו בין הערות קשורות
- הבינו יחסים בין הערות

### ויקי-קישורים

קשרו לתת-הערות מכל מקום

```markdown
ראו [[פרויקט/מחקר/ניתוח]] לפרטים.
```

## טיפים

### מוסכמת שמות

השתמשו בשמות ברורים ועקביים

```
✅ טוב:
פרויקט/דרישות/פונקציונלי
פרויקט/דרישות/לא-פונקציונלי

❌ הימנעו:
פרויקט/דריש/פונק
פרויקט/דריש/לא-פונק
```

### ניווט בפירורי לחם

השתמשו בפירורי לחם להבנת המיקום שלכם

```
פרויקט > מחקר > ניתוח
```

### חיפוש

החיפוש כולל תת-הערות אוטומטית

- מצאו תוכן בכל הרמות
- סננו לפי עומק הערה
- ניווט מהיר

### תבניות

החילו תבניות על תת-הערות

1. צרו תת-הערה
2. השתמשו בפקודת סלאש `/template`
3. בחרו תבנית

## קיצורי מקלדת

| פעולה | קיצור מקלדת |
|--------|----------|
| תת-הערה חדשה | `Cmd/Ctrl+Shift+N` |
| הרחב הכל | `Cmd/Ctrl+Down` |
| כווץ הכל | `Cmd/Ctrl+Up` |
| העברה למעלה | `Opt/Alt+Up` |
| העברה למטה | `Opt/Alt+Down` |
| קידום | `Opt/Alt+Left` |
| הורדה | `Opt/Alt+Right` |

## פתרון בעיות

### תת-הערה לא מופיעה

- בדקו אם האב מכווץ
- ודאו שההערה לא הועברה
- רעננו את סרגל הצד

### לא ניתן ליצור תת-הערה

- ודאו שהערת אב קיימת
- בדקו הרשאות
- נסו את שיטת תפריט ההקשר

### תת-הערות אבודות

השתמשו בחיפוש למציאת הערות חסרות

1. פתחו חיפוש (`Cmd/Ctrl+F`)
2. הקלידו שם הערה
3. צפו בנתיב המלא בתוצאות
