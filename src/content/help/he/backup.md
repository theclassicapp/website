# גיבוי ושחזור

שמרו על ההערות שלכם בטוחות עם מערכת הגיבוי המובנית של Classic. לעולם אל תאבדו את העבודה שלכם שוב.

## גיבויים אוטומטיים

### איך זה עובד

Classic יוצר אוטומטית גיבויים של ההערות שלכם במרווחי זמן קבועים

- **תדירות ברירת מחדל**: כל 30 דקות
- **מיקום גיבוי**: אותה תיקייה כמו ההערות שלכם
- **שמירה**: שומר את 10 הגיבויים האחרונים
- **פורמט**: ארכיון דחוס (`.cbak`)

### הגדרת גיבוי אוטומטי

1. פתחו הגדרות
2. נווטו ל**גיבוי ושחזור**
3. הגדירו את הפרמטרים הבאים

| הגדרה | אפשרויות | ברירת מחדל |
| ------- | ------- | ------- |
| הפעל גיבוי אוטומטי | מופעל/מכובה | מופעל |
| תדירות גיבוי | 15 דקות / 30 דקות / שעה / 4 שעות | 30 דקות |
| שמור גיבויים | 5 / 10 / 20 / 50 | 10 |
| מיקום גיבוי | נתיב מותאם אישית | תיקיית ההערות |

## גיבוי ידני

### צרו גיבוי עכשיו

1. פתחו פלטת פקודות (`Cmd/Ctrl+P`)
2. הקלידו "גיבוי"
3. בחרו **צור גיבוי עכשיו**
4. בחרו מיקום (אופציונלי)
5. הגיבוי נוצר

### פורמט קובץ גיבוי

גיבויים מאוחסנים כקבצי `.cbak` המכילים

- כל ההערות (קבצי Markdown)
- קבצים מצורפים
- הגדרות
- מטא-נתונים (תגיות, קישורים וכו')

## שחזור מגיבוי

### תהליך שחזור

1. פתחו הגדרות
2. עברו ל**גיבוי ושחזור**
3. לחצו **שחזר מגיבוי**
4. בחרו את קובץ הגיבוי
5. בחרו אפשרויות שחזור
6. אשרו שחזור

### אפשרויות שחזור

| אפשרות | תיאור |
| ------ | ----------- |
| **החלף הכל** | החלף לחלוטין הערות נוכחיות עם גיבוי |
| **מזג** | שלב גיבוי עם הערות קיימות |
| **תצוגה מקדימה** | צפו בתוכן הגיבוי לפני השחזור |

### מה משוחזר

- ✅ כל ההערות והתיקיות
- ✅ קבצים מצורפים
- ✅ תגיות ומטא-נתונים
- ✅ היסטוריית הערות
- ⚠️ הגדרות (אופציונלי)
- ❌ העדפות אפליקציה

## מיקומי גיבוי

### מיקום ברירת מחדל

```
~/Classic/backups/
├── backup-2024-01-15-10-30.cbak
├── backup-2024-01-15-11-00.cbak
└── backup-2024-01-15-11-30.cbak
```

### מיקום מותאם אישית

1. פתחו הגדרות
2. עברו לגיבוי ושחזור
3. לחצו "שנה מיקום גיבוי"
4. בחרו את התיקייה המועדפת עליכם

### גיבוי בענן

הפנו את מיקום הגיבוי לתיקייה מסונכרנת בענן

- **iCloud**: `~/Library/Mobile Documents/com~apple~CloudDocs/Classic Backups`
- **Dropbox**: `~/Dropbox/Classic Backups`
- **Google Drive**: `~/Google Drive/Classic Backups`
- **OneDrive**: `~/OneDrive/Classic Backups`

## שיטות עבודה מומלצות

### לוח זמני גיבוי

- **גיבוי אוטומטי**: שמרו מופעל במרווחים של 30 דקות
- **גיבוי ידני**: צרו לפני שינויים משמעותיים
- **גיבוי מחוץ לאתר**: העתיקו גיבויים לאחסון חיצוני באופן קבוע

### אימות גיבויים

אמתו את הגיבויים שלכם מדי פעם

1. צרו שחזור בדיקה
2. בדקו תוכן הערות
3. ודאו קבצים מצורפים
4. אשרו מטא-נתונים

### רוטציית גיבויים

- שמרו גיבויים אחרונים מקומית
- ארכבו גיבויים ישנים יותר לאחסון חיצוני
- מחקו גיבויים ישנים מאוד לחיסכון במקום

## פתרון בעיות

### גיבוי נכשל

**גורמים אפשריים**

- אין מספיק מקום בדיסק
- בעיות הרשאות
- קובץ בשימוש

**פתרונות**

1. בדקו מקום פנוי בדיסק
2. ודאו הרשאות תיקייה
3. סגרו אפליקציות אחרות
4. נסו מיקום אחר

### שחזור נכשל

**גורמים אפשריים**

- קובץ גיבוי פגום
- גרסה לא תואמת
- אין מספיק מקום

**פתרונות**

1. נסו גיבוי ישן יותר
2. בדקו שלמות קובץ גיבוי
3. פנו מקום בדיסק
4. צרו קשר עם התמיכה

### גיבויים חסרים

**גורמים אפשריים**

- גיבוי אוטומטי מכובה
- מיקום גיבוי השתנה
- גיבויים נמחקו

**פתרונות**

1. בדקו הגדרות גיבוי
2. ודאו מיקום גיבוי
3. הפעילו גיבוי אוטומטי
4. צרו גיבוי ידני עכשיו

## אפשרויות מתקדמות

### גיבויים מוצפנים

לאבטחה מוגברת

1. פתחו הגדרות
2. עברו לגיבוי ושחזור
3. הפעילו "הצפן גיבויים"
4. הגדירו סיסמה
5. זכרו סיסמה זו - לא ניתן לשחזר אותה

### גיבויים מצטברים

חסכו מקום עם גיבויים מצטברים

- רק שינויים מאז הגיבוי האחרון מאוחסנים
- שחזור מלא משחזר מצב מלא
- מפחית זמן וגודל גיבוי

### אי-הכללות גיבוי

אל תכללו תוכן ספציפי בגיבויים

1. פתחו הגדרות
2. עברו לגיבוי ושחזור
3. לחצו "אי-הכללות"
4. הוסיפו תיקיות או קבצים לאי-הכללה
