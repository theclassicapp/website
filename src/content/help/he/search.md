# חיפוש

מצאו כל הערה באופן מיידי עם החיפוש החזק של Classic. חיפוש טקסט מלא, מסננים ואופרטורים עוזרים לכם לאתר בדיוק מה שאתם צריכים.

## חיפוש בסיסי

### פתיחת חיפוש

| פלטפורמה | קיצור מקלדת |
|----------|----------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

או לחצו על סרגל החיפוש בסרגל הצד.

### איך זה עובד

1. פתחו חיפוש
2. התחילו להקליד
3. תוצאות מופיעות באופן מיידי
4. לחצו לפתיחת הערה

## תכונות חיפוש

### חיפוש טקסט מלא

Classic מחפש דרך

- כותרות הערות
- תוכן ההערות
- תגיות
- שמות קבצים

### תוצאות בזמן אמת

תוצאות מתעדכנות בזמן שאתם מקלידים

- אין צורך ללחוץ Enter
- משוב מיידי
- התאמה מטושטשת מופעלת

### הדגשת חיפוש

מונחים שנמצאו מודגשים בתוצאות

- ראו הקשר סביב התאמות
- זהו הערות רלוונטיות במהירות

## אופרטורי חיפוש

### התאמה מדויקת

השתמשו במרכאות לביטויים מדויקים

```
"תכנון פרויקט"
"הערות פגישה"
```

### חיפוש לפי תגית

מצאו הערות עם תגיות ספציפיות

```
tag:עבודה
tag:פגישות
tag:דחוף
```

### אי-כללת מונחים

השתמשו ב-`-` לאי-כלילה

```
פרויקט -ארכיון
פגישה -מבוטל
```

### חיפוש OR

מצאו הערות עם כל מונח

```
פרויקט OR יוזמה
```

### טווח תאריכים

חפשו לפי תאריך

```
created:2024-01-01..2024-12-31
modified:last-week
```

### סוג קובץ

סננו לפי סיומת

```
ext:md
ext:txt
```

## חיפוש מתקדם

### שילוב אופרטורים

בנו שאילתות מורכבות

```
tag:עבודה -tag:ארכיון created:last-week
"פגישה" tag:פרויקט-אלפא
```

### חיפוש בנתיב

חפשו בתוך תיקיות

```
path:פרויקטים/
path:ארכיון/
```

### חיפוש בכותרות

חפשו רק כותרות

```
title:מפת-דרכים
title:פגישה
```

### חיפוש בתוכן

חפשו רק תוכן (ללא כותרות)

```
content:חשוב
content:תאריך-יעד
```

## מסנני חיפוש

### לפי תגית

1. הקלידו את החיפוש שלכם
2. לחצו על תגית לסינון
3. תוצאות מצטמצמות להערות מתויגות

### לפי תאריך

סננו לפי תאריך שינוי

- היום
- השבוע
- החודש
- טווח מותאם אישית

### לפי תיקייה

הגבילו חיפוש לתיקייה

1. לחצו לחיצה ימנית על תיקייה
2. בחרו "חפש בתיקייה"
3. תוצאות מוגבלות לאותה תיקייה

## תוצאות חיפוש

### מיון

מיינו תוצאות לפי

- **רלוונטיות** (ברירת מחדל)
- **תאריך שינוי** (החדש ביותר ראשון)
- **תאריך יצירה** (החדש ביותר ראשון)
- **כותרת** (אלפביתי)

### תצוגה מקדימה של תוצאות

רחפו מעל תוצאות לתצוגה מקדימה

- מוצגות מספר שורות ראשונות
- מונחים תואמים מודגשים
- תגיות מוצגות

### הקשר

ראו היכן ההתאמה התרחשה

- מציג טקסט מקיף
- מדגיש את ההתאמה
- לחצו לקפיצה למיקום

## היסטוריית חיפוש

### חיפושים אחרונים

גשו לחיפושים קודמים

1. לחצו בסרגל החיפוש
2. ראו חיפושים אחרונים
3. לחצו להרצה מחדש

### ניקוי היסטוריה

1. פתחו הגדרות
2. עברו לפרטיות
3. לחצו "נקה היסטוריית חיפוש"

## טיפים

### השתמשו במונחים ספציפיים

יותר ספציפי = תוצאות טובות יותר

```
✅ "תקציב שיווקי Q4"
❌ תקציב
```

### שלבו עם תגיות

צמצמו תוצאות במהירות

```
פרויקט tag:דחוף
```

### השתמשו באי-כללות

סננו רעש

```
פגישה -tag:ארכיון
```

### שמרו חיפושים תכופים

צרו חיפושים שמורים

1. הזינו את החיפוש שלכם
2. לחצו "שמור חיפוש"
3. תנו שם
4. גשו מסרגל הצד

## ניווט במקלדת

| פעולה | קיצור מקלדת |
|--------|----------|
| פתיחת חיפוש | `Cmd/Ctrl+F` |
| תוצאה הבאה | `Enter` או `Down` |
| תוצאה קודמת | `Shift+Enter` או `Up` |
| סגירת חיפוש | `Escape` |

## הגדרות חיפוש

### התאמה אישית בהגדרות

| הגדרה | אפשרויות |
|---------|---------|
| תלוי רישיות | מופעל/מכובה |
| חיפוש בכותרות | מופעל/מכובה |
| חיפוש בתוכן | מופעל/מכובה |
| חיפוש בתגיות | מופעל/מכובה |
| כולל ארכיון | מופעל/מכובה |

## מקרי שימוש

### מציאת הערות ישנות

```
"פגישה" date:last-year
```

### מחקר פרויקט

```
path:פרויקטים/ tag:מחקר
```

### פריטי פעולה

```
"TODO" OR "FIXME" -tag:בוצע
```

### עבודה אחרונה

```
modified:this-week
```

## פתרון בעיות

### לא נמצאו תוצאות

- בדקו איות
- הסירו מסננים
- נסו מונחים רחבים יותר
- בדקו אם ההערה בארכיון

### חיפוש איטי

- אוספי הערות גדולים עשויים לקחת יותר זמן
- כבו תלות ברישיות
- הפחיתו היקף חיפוש

### הערות חסרות

- ההערה עשויה להיות בתיקייה אחרת
- בדקו אם התגית נכונה
- ודאו שההערה לא בארכיון
