# מדריך Markdown

Classic תומך בתחביר Markdown מלא עם תצוגה מקדימה בזמן אמת. הנה מדריך מקיף לכל אפשרויות העיצוב הנתמכות.

## עיצוב בסיסי

| תחביר | תוצאה |
|-------|--------|
| `**מודגש**` | **מודגש** |
| `*נטוי*` | *נטוי* |
| `~~קו חוצה~~` | ~~קו חוצה~~ |
| `# כותרת 1` | כותרת 1 |
| `## כותרת 2` | ## כותרת 2 |
| `### כותרת 3` | ### כותרת 3 |

## קישורים

```markdown
[קישור בתוך השורה](https://classic.app)

[קישור בסגנון הפניה][https://classic.app]
```


## בלוקי קוד

קוד בתוך השורה `code`:

```javascript
const greeting = "Hello, World!";
console.log(greeting);
```

בלוק קוד עם שפה:

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("Classic"))
```

## תכונות מתקדמות

### דיאגרמות Mermaid

צרו דיאגרמות באמצעות תחביר Mermaid:

```mermaid
graph TD
    A[התחלה] --> B[תהליך]
    B --> C[סיום]
```

### משוואות מתמטיות

השתמשו ב-KaTeX לביטויים מתמטיים:

```markdown
$$E = mc^2$$
```

מתמטיקה בתוך השורה: $E = mc^2$
