# מתמטיקה ומשוואות

Classic תומך בסימון מתמטי באמצעות תחביר LaTeX. כתבו משוואות יפות בהערות שלכם.

## הפעלת מתמטיקה

תמיכה במתמטיקה מופעלת כברירת מחדל ב-Classic.

### מתמטיקה בתוך השורה

השתמשו בסימני דולר בודדים למשוואות בתוך השורה

```markdown
הנוסחה $E = mc^2$ שינתה את הפיזיקה לנצח.
```

מעובד כ: הנוסחה E=mc² שינתה את הפיזיקה לנצח.

### מתמטיקה בבלוק

השתמשו בסימני דולר כפולים למשוואות בלוק ממורכזות

```markdown
$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

## סימון נפוץ

### פעולות בסיסיות

| תחביר | תוצאה |
| ------ | ------ |
| `x + y` | x + y |
| `x - y` | x - y |
| `x \times y` | x × y |
| `x \div y` | x ÷ y |
| `x / y` | x/y |
| `x^2` | x² |
| `x_{n}` | xₙ |

### שברים

```markdown
$\frac{numerator}{denominator}$
```

דוגמאות

- `$\frac{1}{2}$` → ½
- `$\frac{a+b}{c+d}$` → (a+b)/(c+d)

### שורשים ריבועיים

```markdown
$\sqrt{x}$
$\sqrt[n]{x}$
```

דוגמאות

- `$\sqrt{2}$` → √2
- `$\sqrt[3]{8}$` → ³√8

### כתב תחתי ועילי

```markdown
$x_1$          → כתב תחתי
$x^2$          → כתב עילי
$x_1^2$        → שניהם
$x_{10}$       → כתב תחתי רב-תווי
$x^{n+1}$      → כתב עילי רב-תווי
```


### מטריצות

```markdown

$$
```

## טיפים

### רווח

השתמשו ב-`\,` `\;` `\:` `\quad` `\qquad` לרווח מותאם אישית

```markdown
$a \, b$     → רווח קטן
$a \quad b$  → רווח גדול יותר
```

### טקסט במתמטיקה

השתמשו ב-`\text{}` לטקסט רגיל בתוך משוואות

```markdown
$\text{כאשר } x > 0$
```

### צביעה

הוסיפו צבע לחלקים של משוואות

```markdown
$\textcolor{red}{חשוב}$
```

### יישור

השתמשו בסביבת `align` למשוואות מרובות שורות

```markdown
$$
\begin{align}
a &= b + c \\
d &= e + f
\end{align}
$$
```
