# ארכיטקטורת Local-First

Classic נבנה עם פרטיות כיסוד. הנתונים שלכם נשארים במכשיר שלכם, תמיד.

## מה זה אומר "Local-First"?

Local-first פירושו:

1. **הנתונים שלכם מאוחסנים מקומית** במחשב שלכם. כל ההערות, הקבצים המצורפים וההגדרות שלכם מאוחסנים בתיקייה במחשב.
2. **לא נדרש ענן** Classic עובד ללא חיבור לאינטרנט.
3. **אתם הבעלים של הנתונים שלכם** יש לכם שליטה מלאה על הנתונים שלכם.

## אחסון נתונים

### מיקום ברירת מחדל

הנתונים שלכם מאוחסנים בתיקייה ייעודית:

| פלטפורמה | מיקום |
| -------- | -------- |
| **macOS** | `~/Library/Application Support/Classic/notes` |
| **Windows** | `%APPDATA%\Classic\notes` |
| **Linux** | `~/.config/Classic/notes` |

### פורמט נתונים

- הערות מאוחסנות כקבצי טקסט פשוטים (`.md`)
- קבצים מצורפים מאוחסנים בתת-תיקיית `attachments`
- הגדרות מאוחסנות ב-`settings.json`

### מבנה קבצים

```
notes/
├── Inbox/
├── Daily Notes/
│   └── 2024-01-15.md
├── Projects/
│   ├── My Project.md
│   └── sub-note.md
└── attachments/
    └── image.png
settings.json
```

## יתרונות פרטיות

### לא נדרש חשבון

- לא צריך הרשמה
- לא צריך אימות דוא"ל
- לא נאסף מידע אישי

### ללא מעקב

- ללא אנליטיקס
- ללא טלמטריה
- ללא נתוני שימוש נשלחים לשרתים חיצוניים

### יכולת עבודה לא מקוונת

- עובד ללא אינטרנט
- מסתנכרן רק כשאתם בוחרים
- הנתונים שלכם תמיד נגישים

## אבטחת נתונים

### האחריות שלכם

- **גיבויים**: הגדירו גיבויים אוטומטיים בהגדרות
- **ייצוא**: יצאו את הנתונים שלכם באופן קבוע
- **עדכונים**: שמרו על Classic מעודכן לתיקוני אבטחה

### האחריות שלנו

- **אחסון מקומי**: אנחנו לעולם לא רואים או נוגעים בנתונים שלכם
- **ללא דלתות אחוריות**: אין איסוף נתונים נסתר
- **שקיפות**: פתוחים וכנים לגבי טיפול בנתונים

## סנכרון (אופציונלי)

Classic לא כולל סנכרון ענן מובנה. עם זאת, אתם יכולים לסנכרן את הנתונים שלכם:

### באמצעות iCloud/OneDrive/Dropbox

1. פתחו הגדרות Classic
2. שנו את מיקום תיקיית ההערות לתיקיית הענן שלכם
3. Classic יסתנכרן דרך ספק הענן שלכם

### באמצעות Git

1. אתחלו מאגר git בתיקיית ההערות שלכם
2. בצעו commit ו-push לשינויים ידנית או עם אוטומציה

## שחזור נתונים

### מגיבוי

1. פתחו הגדרות
2. עברו לגיבוי ושחזור
3. בחרו קובץ גיבוי
4. לחצו שחזר

### שחזור ידני

1. אתרו את תיקיית ההערות שלכם
2. העתיקו קבצים למיקום בטוח
3. ייבאו להתקנת Classic חדשה

## שאלות נפוצות

### האם הנתונים שלי באמת פרטיים?

**כן.** הנתונים שלכם לעולם לא עוזבים את המכשיר שלכם אלא אם כן אתם מסנכרנים אותם במפורש באמצעות שירות צד שלישי.

### האם אני יכול להעביר את הנתונים שלי למחשב אחר?

**כן.** פשוט העתיקו את תיקיית ההערות שלכם למחשב החדש והפנו את Classic אליה בהגדרות.

### מה קורה אם אני מסיר את ההתקנה של Classic?

**הנתונים שלכם נשארים.** הסרת התקנה של Classic לא מוחקת את תיקיית ההערות שלכם. אתם יכולים להסיר ולהתקין מחדש בבטחה מבלי לאבד נתונים.

### האם Classic יכול לגשת לנתונים שלי?

**לא.** Classic היא אפליקציה מקומית ללא קישוריות חיצונית לאיסוף נתונים.
