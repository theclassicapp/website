# שפות ולוקליזציה

Classic מדבר בשפה שלכם. השתמשו באפליקציה בשפה המועדפת עליכם וכתבו הערות בכל שפה שתבחרו.

## שפות נתמכות

הממשק של Classic זמין בשפות רבות

| שפה | קוד |
|----------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| ועוד... | |

## שינוי שפת הממשק

### דרך הגדרות

1. פתחו הגדרות (`Cmd+,` או `Ctrl+,`)
2. עברו ל**כללי** → **שפה**
3. בחרו את השפה המועדפת
4. הפעילו מחדש את Classic להחלה

### דרך פלטת פקודות

1. פתחו פלטת פקודות (`Cmd+P` או `Ctrl+P`)
2. הקלידו "שפה"
3. בחרו את השפה המועדפת
4. הפעילו מחדש להחלת השינויים

## כתיבה בשפות מרובות

Classic תומך בכתיבת הערות בכל שפה

### תמיכה ב-Unicode

- תמיכה מלאה ב-Unicode
- שפות RTL (מימין לשמאל) נתמכות
- תווי CJK (סינית, יפנית, קוריאנית) עובדים מושלם
- תמיכה באימוג'י

### זיהוי שפה

Classic יכול לזהות את שפת ההערות שלכם

- זיהוי אוטומטי לבדיקת איות
- הגדרת שפה לכל הערה
- תמיכה בשפות מעורבבות

## בדיקת איות

### הפעלת בדיקת איות

1. פתחו הגדרות
2. עברו ל**עורך** → **בדיקת איות**
3. הפעילו "בדיקת איות"

### שפות בדיקת איות

1. פתחו הגדרות
2. עברו ל**עורך** → **בדיקת איות**
3. לחצו "שפות"
4. בחרו שפות לבדיקה

### הוספת מילים למילון

1. לחצו לחיצה ימנית על מילה מסומנת
2. בחרו "הוסף למילון"
3. המילה לא תסומן שוב

## תמיכה ב-RTL

לערבית, עברית ושפות RTL אחרות

### זיהוי RTL אוטומטי

Classic מזהה תוכן RTL אוטומטית

- כיוון טקסט מתאים
- תנועת סמן מסתגלת
- יישור משתנה

### החלפת RTL ידנית

אלצו כיוון RTL או LTR

1. פתחו פלטת פקודות
2. הקלידו "RTL" או "LTR"
3. בחרו את העדפתכם

## שיטות קלט

Classic עובד עם שיטות הקלט של המערכת שלכם

### שיטות קלט CJK

- עובד עם Pinyin, Zhuyin וכו'
- תומך ב-Japanese IME
- שיטות קלט קוריאניות נתמכות

### תווים מיוחדים

הכניסו תווים מיוחדים בקלות

- השתמשו בבורר התווים של המערכת
- העתק/הדבק ממפות תווים
- השתמשו בקלט Unicode

## פורמטים תאריך ושעה

תאריכים מסתגלים ללוקל שלכם

### אפשרויות פורמט

| הגדרה | אפשרויות |
|---------|---------|
| פורמט תאריך | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| פורמט שעה | 12 שעות, 24 שעות |
| יום ראשון בשבוע | ראשון, שני, שבת |

הגדירו בהגדרות → כללי → תאריך ושעה

## פורמטים מספריים

מספרים מוצגים לפי הלוקל שלכם

- **מפריד עשרוני**: `.` או `,`
- **מפריד אלפים**: `,` או `.` או רווח
- **סמלי מטבע**: על סמך אזור

## פריסות מקלדת

Classic עובד עם כל פריסת מקלדת

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- פריסות מותאמות אישית

## הגדרות לוקליזציה

### בהגדרות → כללי → שפה

| הגדרה | תיאור |
|---------|-------------|
| **שפת ממשק** | שפת ממשק האפליקציה |
| **פורמט תאריך** | כיצד תאריכים מופיעים |
| **פורמט שעה** | 12 או 24 שעות |
| **יום ראשון בשבוע** | יום התחלת לוח שנה |
| **פורמט מספרים** | מפרידי עשרוני/אלפים |

## טיפים

### הערות בשפות מעורבבות

כתבו בשפות מרובות בהערה אחת

```markdown
# פרויקט בינלאומי

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
עברית: ברוכים הבאים לפרויקט
```

### גופנים ספציפיים לשפה

שפות מסוימות נראות טוב יותר עם גופנים ספציפיים

1. פתחו הגדרות
2. עברו למראה → גופנים
3. הגדירו גופנים לכתבים שונים

### החלפת שפות מהירה

צרו קיצור מקלדת

1. פתחו הגדרות → קיצורים
2. מצאו "החלף שפה"
3. הגדירו את הקיצור המועדף

## פתרון בעיות

### שפה לא זמינה

- בדקו עדכוני אפליקציה
- ייתכן שהשפה בפיתוח
- תרמו תרגומים ב-GitHub

### תווים לא מוצגים

- התקינו גופנים נדרשים
- בדקו הגדרות גופן
- נסו גופן אחר

### בדיקת איות לא עובדת

- ודאו שבדיקת איות מופעלת
- הוסיפו את השפה לבדיקת איות
- הפעילו מחדש את Classic

### RTL לא עובד כראוי

- ודאו שזיהוי RTL פועל
- נסו החלפת RTL ידנית
- בדקו בעיות תוכן מעורבב

## תרומת תרגומים

עזרו לתרגם את Classic

1. בקרו בקהילת Classic
2. הצטרפו לצוות התרגום
3. הגישו תרגומים
4. עזרו לשפר לוקליזציה
