# פלטת פקודות

פלטת הפקודות היא השער שלכם לכל פעולה ב-Classic. לחצו קיצור מקלדת פשוט וגשו לכל תכונה באופן מיידי.

## פתיחת פלטת הפקודות

| פלטפורמה | קיצור מקלדת |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## איך להשתמש

1. פתחו את פלטת הפקודות (`Cmd+P` או `Ctrl+P`)
2. הקלידו לחיפוש פקודה
3. השתמשו במקשי החצים לניווט
4. לחצו `Enter` להרצה

## קטגוריות פקודות

### פעולות קבצים

- **הערה חדשה**: צרו הערה חדשה
- **הערה יומית חדשה**: צרו/פתחו את ההערה היומית של היום
- **פתיחת הערה**: חפשו ופתחו הערות קיימות
- **שמירת הערה**: שמרו את ההערה הנוכחית
- **מחיקת הערה**: מחקו את ההערה הנוכחית
- **שינוי שם הערה**: שנו שם להערה הנוכחית

### ניווט

- **מעבר להערה**: קפצו לכל הערה
- **מעבר להערה יומית**: פתחו תאריך ספציפי
- **מעבר לתגית**: עיינו בהערות לפי תגית
- **חזרה אחורה**: נווטו אחורה בהיסטוריה
- **קדימה**: נווטו קדימה בהיסטוריה

### עורך

- **החלפת תצוגה מקדימה**: עברו בין מצבי עריכה ותצוגה מקדימה
- **החלפת סרגל צד**: הציגו/הסתירו את סרגל הצד
- **מצב מיקוד**: היכנסו לכתיבה ללא הסחות דעת
- **הוספת תאריך**: הוסיפו תאריך/שעה נוכחיים
- **הוספת תבנית**: הוסיפו תבנית

### עיצוב

- **מודגש**: הפכו טקסט למודגש
- **נטוי**: הפכו טקסט לנטוי
- **קו חוצה**: קו חוצה דרך טקסט
- **קוד**: עצבו כקוד
- **קישור**: הוסיפו קישור
- **כותרת**: החילו עיצוב כותרת

### ארגון

- **הוספת תגית**: תייגו את ההערה הנוכחית
- **העברה לתיקייה**: העבירו הערה לתיקייה
- **ארכוב הערה**: ארכבו את ההערה הנוכחית
- **הצמדת הערה**: הצמידו הערה לראש הרשימה

### תצוגה

- **החלפת ערכת נושא**: עברו בין בהיר/כהה
- **התקרבות**: הגדילו גודל גופן
- **התרחקות**: הקטינו גודל גופן
- **איפוס זום**: אפסו לגודל ברירת מחדל

## טיפים

### גישה מהירה

- פקודות מחופשות בצורה מטושטשת
- פקודות אחרונות מופיעות ראשונות
- הצמידו פקודות בשימוש תכוף

### ניווט במקלדת

- `↑`/`↓` - נווטו בתוצאות
- `Enter` - הריצו פקודה
- `Escape` - סגרו פלטה
- `Tab` - עברו בין קטגוריות

### פקודות מותאמות אישית

1. פתחו הגדרות
2. עברו לפלטת פקודות
3. הוסיפו פקודות מותאמות אישית
4. הגדירו קיצורי מקלדת

## פקודות בשימוש תכוף

הנה כמה פקודות שתשתמשו בהן לעתים קרובות

| פקודה | מה זה עושה |
| ------- | ------------- |
| `הערה חדשה` | צור הערה ריקה |
| `פתיחת הערה` | חפש את כל ההערות |
| `הערה יומית חדשה` | היומן של היום |
| `החלפת תצוגה מקדימה` | תצוגה מקדימה מול עריכה |
| `הוספת תגית` | תייג הערה נוכחית |
| `לכידה מהירה` | לכידה גלובלית |

## התאמה אישית של פקודות

### שינוי קיצורים

1. פתחו הגדרות (`Cmd+,` או `Ctrl+,`)
2. נווטו לקיצורי מקלדת
3. מצאו את הפקודה
4. לחצו על הקיצור הנוכחי
5. לחצו את הקיצור החדש
6. שמרו

### יצירת פקודות מותאמות אישית

משתמשים מתקדמים יכולים ליצור פקודות מותאמות אישית

1. פתחו הגדרות
2. עברו לפקודות
3. לחצו "הוסף פקודה מותאמת אישית"
4. הגדירו את שם הפקודה
5. הוסיפו את הפעולה/ות
6. הגדירו קיצור (אופציונלי)
