# ทางลัดคีย์บอร์ด

ทำงานได้เร็วขึ้นด้วยทางลัดคีย์บอร์ดของ Classic ทางลัดทั้งหมดสามารถปรับแต่งได้ในการตั้งค่า

## ทางลัดทั่วโลก

เหล่านี้ทำงานได้ทุกที่ในแอป

### ทั่วไป

| ทางลัด (Windows/Linux) | ทางลัด (macOS) | การดำเนินการ |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | สร้างบันทึกใหม่ |
| `Ctrl+P` | `Cmd+P` | Command palette |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Global quick capture |
| `Ctrl+S` | `Cmd+S` | บันทึกบันทึก |
| `Ctrl+F` | `Cmd+F` | ค้นหา |
| `Ctrl+,` | `Cmd+,` | การตั้งค่า |

## Quick Capture

| ทางลัด (Windows/Linux) | ทางลัด (macOS) | การดำเนินการ |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | เปิด quick capture |
| `Escape` | `Escape` | ปิด quick capture |
| `Ctrl+Enter` | `Cmd+Enter` | บันทึกและปิด |

## การปรับแต่งทางลัด

1. เปิด **Settings** (`Ctrl+,` หรือ `Cmd+,`)
2. ไปที่ **Keyboard Shortcuts**
3. คลิกที่ทางลัดใดๆ เพื่อเปลี่ยน
4. กดคีย์ผสมใหม่ของคุณ
5. คลิก **Save**

## เคล็ดลับ

- ทางลัดส่วนใหญ่สามารถปรับแต่งได้
- ความขัดแย้งกับทางลัดระบบจะถูกเน้น
- รีเซ็ตเป็นค่าเริ่มต้นโดยคลิกปุ่มรีเซ็ตข้างแต่ละทางลัด
