# ค้นหา

ค้นหาบันทึกใดๆ ได้ทันทีด้วยการค้นหาอันทรงพลังของ Classic การค้นหาแบบเต็มข้อความ ตัวกรอง และตัวดำเนินการช่วยคุณค้นหาสิ่งที่คุณต้องการได้อย่างแม่นยำ

## การค้นหาพื้นฐาน

### การเปิดการค้นหา

| แพลตฟอร์ม | ทางลัด |
|----------|----------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

หรือคลิกแถบค้นหาในแถบด้านข้าง

### มันทำงานอย่างไร

1. เปิดการค้นหา
2. เริ่มพิมพ์
3. ผลลัพธ์ปรากฏทันที
4. คลิกเพื่อเปิดบันทึก

## คุณสมบัติการค้นหา

### การค้นหาแบบเต็มข้อความ

Classic ค้นหาผ่าน

- ชื่อบันทึก
- เนื้อหาบันทึก
- แท็ก
- ชื่อไฟล์

### ผลลัพธ์แบบเรียลไทม์

ผลลัพธ์อัปเดตขณะคุณพิมพ์

- ไม่ต้องกด Enter
- ข้อเสนอแนะทันที
- เปิดใช้งานการจับคู่แบบ fuzzy

### การเน้นการค้นหา

คำที่พบถูกเน้นในผลลัพธ์

- ดูบริบทรอบๆ การจับคู่
- ระบุบันทึกที่เกี่ยวข้องได้อย่างรวดเร็ว

## ตัวดำเนินการการค้นหา

### การจับคู่แบบแน่นอน

ใช้เครื่องหมายคำพูดสำหรับวลีที่แน่นอน

```
"project planning"
"meeting notes"
```

### การค้นหาตามแท็ก

ค้นหาบันทึกที่มีแท็กเฉพาะ

```
tag:work
tag:meetings
tag:urgent
```

### การยกเว้นคำ

ใช้ `-` เพื่อยกเว้น

```
project -archived
meeting -cancelled
```

### การค้นหา OR

ค้นหาบันทึกที่มีคำใดคำหนึ่ง

```
project OR initiative
```

### ช่วงวันที่

ค้นหาตามวันที่

```
created:2024-01-01..2024-12-31
modified:last-week
```

### ประเภทไฟล์

กรองตามนามสกุล

```
ext:md
ext:txt
```

## การค้นหาขั้นสูง

### การรวมตัวดำเนินการ

สร้างคิวรีที่ซับซ้อน

```
tag:work -tag:archived created:last-week
"meeting" tag:project-alpha
```

### การค้นหาตามเส้นทาง

ค้นหาภายในโฟลเดอร์

```
path:Projects/
path:Archive/
```

### การค้นหาชื่อเรื่อง

ค้นหาเฉพาะชื่อ

```
title:Roadmap
title:Meeting
```

### การค้นหาเนื้อหา

ค้นหาเฉพาะเนื้อหา (ยกเว้นชื่อ)

```
content:important
content:deadline
```

## ตัวกรองการค้นหา

### ตามแท็ก

1. พิมพ์การค้นหาของคุณ
2. คลิกแท็กเพื่อกรอง
3. ผลลัพธ์แคบลงเป็นบันทึกที่แท็ก

### ตามวันที่

กรองตามวันที่แก้ไข

- วันนี้
- สัปดาห์นี้
- เดือนนี้
- ช่วงที่กำหนดเอง

### ตามโฟลเดอร์

จำกัดการค้นหาในโฟลเดอร์

1. คลิกขวาที่โฟลเดอร์
2. เลือก "Search in Folder"
3. ผลลัพธ์จำกัดเฉพาะโฟลเดอร์นั้น

## ผลลัพธ์การค้นหา

### การเรียงลำดับ

เรียงลำดับผลลัพธ์ตาม

- **ความเกี่ยวข้อง** (ค่าเริ่มต้น)
- **วันที่แก้ไข** (ใหม่ล่าสุดก่อน)
- **วันที่สร้าง** (ใหม่ล่าสุดก่อน)
- **ชื่อเรื่อง** (ตัวอักษร)

### ตัวอย่างผลลัพธ์

วางเมาส์เหนือผลลัพธ์เพื่อดูตัวอย่าง

- แสดงบรรทัดแรกๆ
- เน้นคำที่จับคู่
- แสดงแท็ก

### บริบท

ดูว่าการจับคู่เกิดขึ้นที่ไหน

- แสดงข้อความรอบๆ
- เน้นการจับคู่
- คลิกเพื่อกระโดดไปยังตำแหน่ง

## ประวัติการค้นหา

### การค้นหาล่าสุด

เข้าถึงการค้นหาก่อนหน้า

1. คลิกในแถบค้นหา
2. ดูการค้นหาล่าสุด
3. คลิกเพื่อเรียกใช้อีกครั้ง

### การล้างประวัติ

1. เปิดการตั้งค่า
2. ไปที่ Privacy
3. คลิก "Clear Search History"

## เคล็ดลับ

### ใช้คำเฉพาะ

เฉพาะเจาะจงมากขึ้น = ผลลัพธ์ที่ดีขึ้น

```
✅ "Q4 marketing budget"
❌ budget
```

### รวมกับแท็ก

แคบผลลัพธ์ได้อย่างรวดเร็ว

```
project tag:urgent
```

### ใช้การยกเว้น

กรองสิ่งรบกวน

```
meeting -tag:archived
```

### บันทึกการค้นหาบ่อย

สร้างการค้นหาที่บันทึกไว้

1. ป้อนการค้นหาของคุณ
2. คลิก "Save Search"
3. ตั้งชื่อ
4. เข้าถึงจากแถบด้านข้าง

## การนำทางด้วยคีย์บอร์ด

| การดำเนินการ | ทางลัด |
|--------|----------|
| เปิดการค้นหา | `Cmd/Ctrl+F` |
| ผลลัพธ์ถัดไป | `Enter` หรือ `Down` |
| ผลลัพธ์ก่อนหน้า | `Shift+Enter` หรือ `Up` |
| ปิดการค้นหา | `Escape` |

## การตั้งค่าการค้นหา

### ปรับแต่งในการตั้งค่า

| การตั้งค่า | ตัวเลือก |
|---------|---------|
| คำนึงถึงตัวพิมพ์ใหญ่เล็ก | เปิด/ปิด |
| ค้นหาชื่อ | เปิด/ปิด |
| ค้นหาเนื้อหา | เปิด/ปิด |
| ค้นหาแท็ก | เปิด/ปิด |
| รวมที่เก็บถาวร | เปิด/ปิด |

## กรณีการใช้งาน

### ค้นหาบันทึกเก่า

```
"meeting" date:last-year
```

### การวิจัยโครงการ

```
path:Projects/ tag:research
```

### รายการที่ต้องดำเนินการ

```
"TODO" OR "FIXME" -tag:done
```

### งานล่าสุด

```
modified:this-week
```

## การแก้ไขปัญหา

### ไม่พบผลลัพธ์

- ตรวจสอบการสะกด
- ลบตัวกรอง
- ลองคำที่กว้างขึ้น
- ตรวจสอบว่าบันทึกถูกเก็บถาวรหรือไม่

### การค้นหาช้า

- คอลเลกชันบันทึกขนาดใหญ่อาจใช้เวลานานขึ้น
- ปิดการใช้งานการคำนึงถึงตัวพิมพ์ใหญ่เล็ก
- ลดขอบเขตการค้นหา

### บันทึกหายไป

- บันทึกอาจอยู่ในโฟลเดอร์อื่น
- ตรวจสอบว่าแท็กถูกต้อง
- ตรวจสอบว่าบันทึกไม่ถูกเก็บถาวร
