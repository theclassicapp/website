# คณิตศาสตร์และสมการ

Classic รองรับสัญกรณ์ทางคณิตศาสตร์โดยใช้ไวยากรณ์ LaTeX เขียนสมการสวยงามในบันทึกของคุณ

## การเปิดใช้งานคณิตศาสตร์

การรองรับคณิตศาสตร์เปิดใช้งานโดยค่าเริ่มต้นใน Classic

### คณิตศาสตร์อินไลน์

ใช้เครื่องหมายดอลลาร์เดี่ยวสำหรับสมการอินไลน์

```markdown
The formula $E = mc^2$ changed physics forever.
```

แสดงผลเป็น: สูตร E=mc² เปลี่ยนฟิสิกส์ตลอดไป

### บล็อกคณิตศาสตร์

ใช้เครื่องหมายดอลลาร์คู่สำหรับสมการบล็อกที่จัดกึ่งกลาง

```markdown
$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

## สัญกรณ์ทั่วไป

### การดำเนินการพื้นฐาน

| ไวยากรณ์ | ผลลัพธ์ |
| ------ | ------ |
| `x + y` | x + y |
| `x - y` | x - y |
| `x \times y` | x × y |
| `x \div y` | x ÷ y |
| `x / y` | x/y |
| `x^2` | x² |
| `x_{n}` | xₙ |

### เศษส่วน

```markdown
$\frac{numerator}{denominator}$
```

ตัวอย่าง

- `$\frac{1}{2}$` → ½
- `$\frac{a+b}{c+d}$` → (a+b)/(c+d)

### รากที่สอง

```markdown
$\sqrt{x}$
$\sqrt[n]{x}$
```

ตัวอย่าง

- `$\sqrt{2}$` → √2
- `$\sqrt[3]{8}$` → ³√8

### ตัวห้อยและตัวยก

```markdown
$x_1$          → ตัวห้อย
$x^2$          → ตัวยก
$x_1^2$        → ทั้งสองอย่าง
$x_{10}$       → ตัวห้อยหลายตัวอักษร
$x^{n+1}$      → ตัวยกหลายตัวอักษร
```

## คณิตศาสตร์ขั้นสูง

### ผลรวม

```markdown
$$
\sum_{i=1}^{n} x_i
$$
```

### ปริพันธ์

```markdown
$$
\int_{a}^{b} f(x) \, dx
$$
```

### ลิมิต

```markdown
$$
\lim_{x \to \infty} f(x)
$$
```

### เมทริกซ์

```markdown
$$
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$
```

### ฟังก์ชันแบบเป็นชิ้นๆ

```markdown
$$
f(x) = \begin{cases}
  1 & \text{if } x > 0 \\
  0 & \text{if } x = 0 \\
  -1 & \text{if } x < 0
\end{cases}
$$
```

## เคล็ดลับ

### การเว้นวรรค

ใช้ `\,` `\;` `\:` `\quad` `\qquad` สำหรับการเว้นวรรคที่กำหนดเอง

```markdown
$a \, b$     → ช่องว่างเล็ก
$a \quad b$  → ช่องว่างใหญ่กว่า
```

### ข้อความในคณิตศาสตร์

ใช้ `\text{}` สำหรับข้อความปกติภายในสมการ

```markdown
$\text{when } x > 0$
```

### การใส่สี

เพิ่มสีให้ส่วนของสมการ

```markdown
$\textcolor{red}{important}$
```

### การจัดเรียง

ใช้สภาพแวดล้อม `align` สำหรับสมการหลายบรรทัด

```markdown
$$
\begin{align}
a &= b + c \\
d &= e + f
\end{align}
$$
```

## ตัวอย่างทั่วไป

### สูตรคูณ

```markdown
$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

### เอกลักษณ์ของออยเลอร์

```markdown
$$
e^{i\pi} + 1 = 0
$$
```

### ทฤษฎีบทพีทาโกรัส

```markdown
$$
a^2 + b^2 = c^2
$$
```

### การแจกแจงปกติ

```markdown
$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}
$$
```
