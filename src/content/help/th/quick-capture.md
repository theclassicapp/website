# Quick Capture

บันทึกแนวคิดได้ทันทีโดยไม่ขัดจังหวะเวิร์กโฟลว์ของคุณ Quick Capture เปิดหน้าต่างจดบันทึกขนาดเล็กจากที่ใดก็ได้ในระบบของคุณ

## Global Hotkey

กด hotkey ที่กำหนดเพื่อเปิด Quick Capture

| แพลตฟอร์ม | ทางลัดเริ่มต้น |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

คุณสามารถปรับแต่งทางลัดนี้ในการตั้งค่า

## การใช้ Quick Capture

1. กด global hotkey (ที่ใดก็ได้ในระบบของคุณ)
2. พิมพ์บันทึกของคุณในหน้าต่าง popup
3. กด `Enter` เพื่อบันทึก
4. กด `Escape` เพื่อยกเลิก

## คุณสมบัติ

### อินเทอร์เฟซขั้นต่ำ

- หน้าต่างขนาดเล็ก โฟกัส
- รองรับ Markdown
- โฟกัสอัตโนมัติที่การป้อนข้อมูล
- การออกแบบคีย์บอร์ดเป็นหลัก

### การดำเนินการด่วน

- `# ` + space = หัวข้อ
- `#inbox` = เพิ่มใน Inbox
- `#today` = เพิ่มในบันทึกของวันนี้
- แท็กด้วย `#tagname`

### ปลายทางเริ่มต้น

บันทึก Quick Capture ไปที่ **Inbox** ของคุณโดยค่าเริ่มต้น คุณสามารถเปลี่ยนได้

1. เปิดการตั้งค่า
2. ไปที่ Quick Capture
3. เลือกปลายทางเริ่มต้น

## เคล็ดลับ

### เวิร์กโฟลว์การบันทึก

1. **บันทึก** - จดแนวคิดได้อย่างรวดเร็ว
2. **ประมวลผล** - ตรวจสอบ Inbox รายวัน
3. **จัดระเบียบ** - ย้ายไปยังโฟลเดอร์/บันทึกที่เหมาะสม

### แนวทางปฏิบัติที่ดีที่สุด

- รักษาให้สั้น - หนึ่งแนวคิดต่อการบันทึก
- ใช้แท็กเพื่อบริบท
- ไม่ต้องกังวลเรื่องการจัดรูปแบบ - ปรับปรุงในภายหลัง
- ตรวจสอบและประมวลผลการบันทึกอย่างสม่ำเสมอ

### การรวม

Quick Capture ทำงานได้ดีกับ

- **Daily Notes**: เริ่มต้นแต่ละวันด้วยการตรวจสอบการบันทึก
- **Inbox**: ใช้เป็นพื้นที่รอ
- **แท็ก**: เพิ่มบริบททันที

## การแก้ไขปัญหา

### Hotkey ไม่ทำงาน?

**macOS**
1. ตรวจสอบ System Preferences > Keyboard > Shortcuts
2. มองหาความขัดแย้ง
3. รีสตาร์ท Classic

**Windows**
1. รัน Classic ในฐานะ Administrator
2. ตรวจสอบทางลัดที่ขัดแย้ง
3. รีสตาร์ท Classic

### หน้าต่างไม่ปรากฏ?

- ตรวจสอบว่า Classic ทำงานอยู่
- ตรวจสอบทางลัด
- ตรวจสอบปัญหาโฟกัส
- รีสตาร์ท Classic

### การบันทึกไม่บันทึก?

- ตรวจสอบปลายทางเริ่มต้น
- ตรวจสอบสิทธิ์การเขียน
- ตรวจสอบพื้นที่ดิสก์
- รีสตาร์ท Classic
