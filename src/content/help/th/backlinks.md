# แบ็คลิงก์

แบ็คลิงก์แสดงการเชื่อมโยงระหว่างบันทึกของคุณ สร้างเครือข่ายของแนวคิดที่เกี่ยวข้อง ค้นพบความสัมพันธ์ที่คุณไม่รู้จัก

## แบ็คลิงก์คืออะไร?

แบ็คลิงก์แสดงบันทึกที่เชื่อมโยงไปยังบันทึกปัจจุบัน พวกมันคือการเชื่อมต่อในกราฟความรู้ของคุณ

### ประโยชน์

- **ค้นพบการเชื่อมโยง**: ค้นหาบันทึกที่เกี่ยวข้องโดยอัตโนมัติ
- **บริบท**: เข้าใจว่าแนวคิดเกี่ยวข้องกันอย่างไร
- **การนำทาง**: กระโดดระหว่างบันทึกที่เชื่อมโยง
- **กราฟความรู้**: แสดงภาพการคิดของคุณ

## แบ็คลิงก์ทำงานอย่างไร

เมื่อคุณสร้างลิงก์ไปยังบันทึกอื่น Classic จะสร้างแบ็คลิงก์โดยอัตโนมัติ

```markdown
ในบันทึก A:
See [[Project Planning]] for details.

ผลลัพธ์:
- บันทึก A เชื่อมโยงไปยัง "Project Planning"
- "Project Planning" แสดงแบ็คลิงก์ไปยังบันทึก A
```

## การดูแบ็คลิงก์

### แผงแบ็คลิงก์

เปิดแผงแบ็คลิงก์เพื่อดูการอ้างอิงทั้งหมด

1. เปิดบันทึกใดๆ
2. คลิกไอคอนแบ็คลิงก์ในแถบด้านข้าง
3. ดูบันทึกทั้งหมดที่เชื่อมโยงมายังนี้

### แบ็คลิงก์อินไลน์

แบ็คลิงก์ปรากฏที่ด้านล่างของแต่ละบันทึก

- แสดงชื่อบันทึกต้นทาง
- แสดงสไนปเป็ตบริบท
- คลิกเพื่อนำทาง

## การสร้างลิงก์

### ไวยากรณ์ Wikilink

ใช้วงเล็บคู่เพื่อเชื่อมโยงไปยังบันทึก

```markdown
[[Note Name]]
[[Another Note]]
```

### ข้อความแสดง

ปรับแต่งข้อความลิงก์

```markdown
[[Project Alpha|the alpha project]]
[[Meeting Notes|yesterday's meeting]]
```

### เชื่อมโยงไปยังหัวข้อ

เชื่อมโยงไปยังส่วนเฉพาะ

```markdown
[[Note Name#Heading]]
[[Note Name#Section|jump to section]]
```

## ประเภทแบ็คลิงก์

### ลิงก์โดยตรง

Wikilinks ที่คุณสร้างโดยชัดเจน

```markdown
Related: [[Similar Note]]
```

### การกล่าวถึงที่ไม่ได้เชื่อมโยง

บันทึกที่กล่าวถึงชื่อเรื่องแต่ไม่ได้เชื่อมโยง

```markdown
Note mentions "Project Alpha" but isn't linked
```

Classic แนะนำให้แปลงเป็นลิงก์

## แผงแบ็คลิงก์

### คุณสมบัติ

- **กรอง**: ค้นหาผ่านแบ็คลิงก์
- **เรียงลำดับ**: ตามวันที่ ชื่อ หรือความเกี่ยวข้อง
- **ตัวอย่าง**: วางเมาส์เพื่อดูตัวอย่างเนื้อหา
- **บริบท**: ดูข้อความรอบๆ

### การดำเนินการ

- คลิกเพื่อเปิดบันทึกที่เชื่อมโยง
- คลิกขวาเพื่อดูตัวเลือก
- แปลงการกล่าวถึงที่ไม่ได้เชื่อมโยง

## กรณีการใช้งาน

### การจัดการความรู้

สร้างวิกิส่วนบุคคล

```markdown
# Machine Learning

Related to [[Artificial Intelligence]]
Builds on [[Statistics]]
Used in [[Natural Language Processing]]
```

### เอกสารโครงการ

เชื่อมโยงเอกสารที่เกี่ยวข้อง

```markdown
# API Design

References [[Database Schema]]
Depends on [[Authentication Flow]]
See also [[Frontend Integration]]
```

### การวิจัย

เชื่อมโยงแนวคิดและแหล่งข้อมูล

```markdown
# Research Notes

Based on [[Source Paper A]]
Contradicts [[Source Paper B]]
Supports [[Hypothesis 1]]
```

### บันทึกการประชุม

เชื่อมโยงการประชุมกับโครงการ

```markdown
# Meeting: Q4 Planning

Project: [[Project Alpha]]
Previous: [[Q3 Review]]
Next: [[Q4 Kickoff]]
```

## มุมมองกราฟ

แสดงภาพแบ็คลิงก์ของคุณเป็นเครือข่าย

1. เปิด Graph View (`Cmd/Ctrl+G`)
2. ดูบันทึกทั้งหมดเป็นโหนด
3. เส้นแสดงการเชื่อมโยง
4. คลิกโหนดเพื่อนำทาง

### การควบคุมกราฟ

- **ซูม**: เลื่อนเพื่อซูม
- **แพน**: ลากเพื่อเคลื่อนย้าย
- **กรอง**: แสดง/ซ่อนตามแท็ก
- **โฟกัส**: เน้นการเชื่อมโยงของบันทึก

## เคล็ดลับ

### เชื่อมโยงอย่างเต็มที่

สร้างลิงก์เมื่อใดก็ตามที่คุณอ้างอิงแนวคิดอื่น

```markdown
✅ ดี:
Discussed the [[Q4 Roadmap]] and [[Budget Allocation]].

❌ หลีกเลี่ยง:
Discussed the Q4 Roadmap and Budget Allocation.
```

### ใช้ชื่อที่สอดคล้อง

จับคู่ชื่อบันทึกให้ตรงสำหรับการเชื่อมโยงที่เชื่อถือได้

```markdown
✅ [[Meeting Notes]]
❌ [[meeting notes]] (ตัวพิมพ์ใหญ่มีผล)
```

### ตรวจสอบแบ็คลิงก์

ตรวจสอบแบ็คลิงก์เป็นระยะเพื่อ

- ค้นพบการเชื่อมโยงที่ลืม
- ค้นหาบันทึกที่จะรวม
- ระบุช่องว่างในความรู้

### ตัวอย่างลิงก์

วางเมาส์เหนือลิงก์เพื่อดูตัวอย่างโดยไม่ต้องเปิด

- ดูบรรทัดแรกๆ
- ตรวจสอบว่าเป็นบันทึกที่ถูกต้อง
- อ้างอิงด่วน

## ทางลัดคีย์บอร์ด

| การดำเนินการ | ทางลัด |
|--------|----------|
| เปิดแบ็คลิงก์ | `Cmd/Ctrl+Shift+B` |
| สร้างลิงก์ | `[[` (เติมอัตโนมัติ) |
| เปิด Graph View | `Cmd/Ctrl+G` |
| นำทางลิงก์ | `Tab` ผ่านลิงก์ |

## คุณสมบัติขั้นสูง

### การฝัง

ฝังเนื้อหาจากบันทึกอื่น

```markdown
![[Note Name]]
![[Note Name#Section]]
```

### การถ่ายทอด

รวมเนื้อหาอินไลน์

```markdown
## Related Content
![[Shared Template#Header]]
```

### การระบายสีลิงก์

ลิงก์เปลี่ยนสีตาม

- **น้ำเงิน**: ลิงก์ที่ไม่ได้เยี่ยมชม
- **ม่วง**: ลิงก์ที่เยี่ยมชมแล้ว
- **แดง**: ลิงก์เสีย (บันทึกไม่มีอยู่)

## การแก้ไขปัญหา

### แบ็คลิงก์หายไป

- ตรวจสอบไวยากรณ์ลิงก์ `[[Note Name]]`
- ตรวจสอบว่าชื่อบันทึกตรงทุกประการ
- รีเฟรชแผงแบ็คลิงก์

### ลิงก์เสีย

- บันทึกอาจถูกเปลี่ยนชื่อ
- บันทึกอาจถูกลบ
- ใช้การค้นหาเพื่อหาบันทึกที่ถูกต้อง

### ลิงก์ซ้ำ

- รวมบันทึกที่คล้ายกัน
- ใช้การเปลี่ยนเส้นทางสำหรับบันทึกที่เปลี่ยนชื่อ
- ตรวจสอบความแตกต่างของชื่อ
