# การสำรองและกู้คืนข้อมูล

รักษาบันทึกของคุณให้ปลอดภัยด้วยระบบสำรองข้อมูลในตัวของ Classic ไม่ต้องสูญเสียงานของคุณอีกต่อไป

## การสำรองข้อมูลอัตโนมัติ

### มันทำงานอย่างไร

Classic สร้างการสำรองข้อมูลบันทึกของคุณโดยอัตโนมัติในช่วงเวลาปกติ

- **ความถี่เริ่มต้น**: ทุก 30 นาที
- **ตำแหน่งสำรองข้อมูล**: โฟลเดอร์เดียวกับบันทึกของคุณ
- **การเก็บรักษา**: เก็บการสำรองข้อมูล 10 ครั้งล่าสุด
- **รูปแบบ**: ไฟล์เก็บถาวรบีบอัด (`.cbak`)

### การกำหนดค่า Auto Backup

1. เปิดการตั้งค่า
2. ไปที่ **Backup & Restore**
3. กำหนดค่าดังต่อไปนี้

| การตั้งค่า | ตัวเลือก | ค่าเริ่มต้น |
| ------- | ------- | ------- |
| เปิดใช้งาน Auto Backup | เปิด/ปิด | เปิด |
| ความถี่การสำรองข้อมูล | 15นาที / 30นาที / 1ชม. / 4ชม. | 30นาที |
| เก็บการสำรองข้อมูล | 5 / 10 / 20 / 50 | 10 |
| ตำแหน่งสำรองข้อมูล | เส้นทางที่กำหนดเอง | โฟลเดอร์บันทึก |

## การสำรองข้อมูลด้วยตนเอง

### สร้างการสำรองข้อมูลเดี๋ยวนี้

1. เปิด Command Palette (`Cmd/Ctrl+P`)
2. พิมพ์ "Backup"
3. เลือก **Create Backup Now**
4. เลือกตำแหน่ง (ไม่บังคับ)
5. การสำรองข้อมูลถูกสร้างขึ้น

### รูปแบบไฟล์สำรองข้อมูล

การสำรองข้อมูลถูกเก็บเป็นไฟล์ `.cbak` ที่มี

- บันทึกทั้งหมด (ไฟล์ Markdown)
- ไฟล์แนบ
- การตั้งค่า
- เมทาดาทา (แท็ก, ลิงก์, ฯลฯ)

## กู้คืนจากการสำรองข้อมูล

### กระบวนการกู้คืน

1. เปิดการตั้งค่า
2. ไปที่ **Backup & Restore**
3. คลิก **Restore from Backup**
4. เลือกไฟล์สำรองข้อมูล
5. เลือกตัวเลือกการกู้คืน
6. ยืนยันการกู้คืน

### ตัวเลือกการกู้คืน

| ตัวเลือก | คำอธิบาย |
| ------ | ----------- |
| **แทนที่ทั้งหมด** | แทนที่บันทึกปัจจุบันด้วยการสำรองข้อมูลอย่างสมบูรณ์ |
| **รวม** | รวมการสำรองข้อมูลกับบันทึกที่มีอยู่ |
| **ตัวอย่าง** | ดูเนื้อหาการสำรองข้อมูลก่อนกู้คืน |

### สิ่งที่ถูกกู้คืน

- ✅ บันทึกและโฟลเดอร์ทั้งหมด
- ✅ ไฟล์แนบ
- ✅ แท็กและเมทาดาทา
- ✅ ประวัติบันทึก
- ⚠️ การตั้งค่า (ไม่บังคับ)
- ❌ การตั้งค่าแอปพลิเคชัน

## ตำแหน่งสำรองข้อมูล

### ตำแหน่งเริ่มต้น

```
~/Classic/backups/
├── backup-2024-01-15-10-30.cbak
├── backup-2024-01-15-11-00.cbak
└── backup-2024-01-15-11-30.cbak
```

### ตำแหน่งที่กำหนดเอง

1. เปิดการตั้งค่า
2. ไปที่ Backup & Restore
3. คลิก "Change Backup Location"
4. เลือกโฟลเดอร์ที่คุณต้องการ

### การสำรองข้อมูลบนคลาวด์

ชี้ตำแหน่งสำรองข้อมูลไปยังโฟลเดอร์ที่ซิงค์กับคลาวด์

- **iCloud**: `~/Library/Mobile Documents/com~apple~CloudDocs/Classic Backups`
- **Dropbox**: `~/Dropbox/Classic Backups`
- **Google Drive**: `~/Google Drive/Classic Backups`
- **OneDrive**: `~/OneDrive/Classic Backups`

## แนวทางปฏิบัติที่ดีที่สุด

### กำหนดการสำรองข้อมูล

- **Auto Backup**: เปิดใช้งานที่ช่วง 30 นาที
- **การสำรองข้อมูลด้วยตนเอง**: สร้างก่อนการเปลี่ยนแปลงใหญ่
- **การสำรองข้อมูลนอกสถานที่**: คัดลอกการสำรองข้อมูลไปยังพื้นที่จัดเก็บภายนอกอย่างสม่ำเสมอ

### การตรวจสอบการสำรองข้อมูล

ตรวจสอบการสำรองข้อมูลของคุณเป็นระยะ

1. สร้างการกู้คืนทดสอบ
2. ตรวจสอบเนื้อหาบันทึก
3. ตรวจสอบไฟล์แนบ
4. ยืนยันเมทาดาทา

### การหมุนเวียนการสำรองข้อมูล

- เก็บการสำรองข้อมูลล่าสุดในเครื่อง
- เก็บการสำรองข้อมูลเก่าไปยังพื้นที่จัดเก็บภายนอก
- ลบการสำรองข้อมูลเก่ามากเพื่อประหยัดพื้นที่

## การแก้ไขปัญหา

### การสำรองข้อมูลล้มเหลว

**สาเหตุที่เป็นไปได้**

- พื้นที่ดิสก์ไม่เพียงพอ
- ปัญหาสิทธิ์
- ไฟล์กำลังถูกใช้งาน

**วิธีแก้ไข**

1. ตรวจสอบพื้นที่ดิสก์ที่มีอยู่
2. ตรวจสอบสิทธิ์โฟลเดอร์
3. ปิดแอปพลิเคชันอื่นๆ
4. ลองตำแหน่งอื่น

### การกู้คืนล้มเหลว

**สาเหตุที่เป็นไปได้**

- ไฟล์สำรองข้อมูลเสียหาย
- เวอร์ชันไม่เข้ากัน
- พื้นที่ไม่เพียงพอ

**วิธีแก้ไข**

1. ลองการสำรองข้อมูลที่เก่ากว่า
2. ตรวจสอบความสมบูรณ์ของไฟล์สำรองข้อมูล
3. เพิ่มพื้นที่ดิสก์ว่าง
4. ติดต่อฝ่ายสนับสนุน

### การสำรองข้อมูลหายไป

**สาเหตุที่เป็นไปได้**

- Auto backup ปิดอยู่
- ตำแหน่งสำรองข้อมูลเปลี่ยนแปลง
- การสำรองข้อมูลถูกลบ

**วิธีแก้ไข**

1. ตรวจสอบการตั้งค่าการสำรองข้อมูล
2. ตรวจสอบตำแหน่งสำรองข้อมูล
3. เปิดใช้งาน auto backup
4. สร้างการสำรองข้อมูลด้วยตนเองตอนนี้

## ตัวเลือกขั้นสูง

### การสำรองข้อมูลแบบเข้ารหัส

สำหรับความปลอดภัยเพิ่มเติม

1. เปิดการตั้งค่า
2. ไปที่ Backup & Restore
3. เปิดใช้งาน "Encrypt Backups"
4. ตั้งรหัสผ่าน
5. จดจำรหัสผ่านนี้ - ไม่สามารถกู้คืนได้

### การสำรองข้อมูลแบบเพิ่มทีละน้อย

ประหยัดพื้นที่ด้วยการสำรองข้อมูลแบบเพิ่มทีละน้อย

- เฉพาะการเปลี่ยนแปลงตั้งแต่การสำรองข้อมูลครั้งล่าสุดถูกเก็บไว้
- การกู้คืนแบบเต็มสร้างสถานะที่สมบูรณ์ใหม่
- ลดเวลาและขนาดการสำรองข้อมูล

### การยกเว้นการสำรองข้อมูล

ยกเว้นเนื้อหาเฉพาะจากการสำรองข้อมูล

1. เปิดการตั้งค่า
2. ไปที่ Backup & Restore
3. คลิก "Exclusions"
4. เพิ่มโฟลเดอร์หรือไฟล์ที่จะยกเว้น
