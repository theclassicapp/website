# แท็ก

แท็กเป็นวิธีที่มีประสิทธิภาพในการจัดระเบียบและจัดหมวดหมู่บันทึกของคุณใน Classic ค้นหาเนื้อหาที่เกี่ยวข้องได้ทันที

## การสร้างแท็ก

### แท็กอินไลน์

เพิ่มแท็กโดยตรงในเนื้อหาบันทึกของคุณ

```markdown
# Meeting Notes

Discussed the Q4 roadmap #work #meetings #planning
```

### ใช้ปุ่มแท็ก

1. คลิกไอคอนแท็กในแถบเครื่องมือ
2. พิมพ์ชื่อแท็กของคุณ
3. กด Enter

### ทางลัดคีย์บอร์ด

| แพลตฟอร์ม | ทางลัด |
|----------|----------|
| **macOS** | `Cmd+T` |
| **Windows/Linux** | `Ctrl+T` |

## ไวยากรณ์แท็ก

### แท็กพื้นฐาน

```markdown
#project #work #important
```

### แท็กซ้อน

สร้างลำดับชั้นของแท็กด้วย `/`

```markdown
#work/meetings
#work/projects
#personal/hobbies/reading
```

### แท็กหลายคำ

ใช้ขีดกลางหรือขีดล่าง

```markdown
#project-alpha
#meeting_notes
#q4-planning
```

## การจัดการแท็ก

### การดูแท็กทั้งหมด

1. เปิดแถบด้านข้าง
2. ค้นหาส่วน "Tags"
3. ดูแท็กทั้งหมดของคุณ

### แท็กคลาวด์

แสดงภาพแท็กที่ใช้มากที่สุด

- แท็กที่ใหญ่กว่า = บันทึกมากกว่า
- คลิกแท็กใดก็ได้เพื่อกรอง
- ดูจำนวนบันทึกต่อแท็ก

### การเปลี่ยนชื่อแท็ก

1. คลิกขวาที่แท็ก
2. เลือก "Rename"
3. ป้อนชื่อใหม่
4. บันทึกทั้งหมดอัปเดตอัตโนมัติ

### การลบแท็ก

1. คลิกขวาที่แท็ก
2. เลือก "Delete"
3. ยืนยันการลบ
4. แท็กถูกลบออกจากบันทึกทั้งหมด

## การค้นหาตามแท็ก

### แผงแท็ก

คลิกแท็กในแถบด้านข้างเพื่อกรองบันทึก

### ไวยากรณ์การค้นหา

ใช้ `tag:` ในการค้นหา

```
tag:work
tag:work/meetings
```

### การรวมแท็ก

ค้นหาบันทึกที่มีหลายแท็ก

```
tag:work tag:urgent
```

### การยกเว้นแท็ก

ค้นหาบันทึกที่ไม่มีแท็ก

```
-tag:archived
```

## แนวทางปฏิบัติที่ดีที่สุดสำหรับแท็ก

### รักษาความเรียบง่าย

- ใช้ 3-5 แท็กต่อบันทึก
- สอดคล้องกับการตั้งชื่อ
- หลีกเลี่ยงการแพร่กระจายของแท็ก

### ใช้ลำดับชั้น

จัดระเบียบแท็กที่เกี่ยวข้อง

```
#work
  #work/meetings
  #work/projects
  #work/deadlines

#personal
  #personal/health
  #personal/finance
```

### สร้างมาตรฐาน

กำหนดมาตรฐานแท็กสำหรับตัวคุณเอง

- **สถานะ**: `#todo` `#in-progress` `#done`
- **ความสำคัญ**: `#urgent` `#high` `#low`
- **ประเภท**: `#meeting` `#reference` `#idea`

## ข้อเสนอแนะแท็ก

เมื่อคุณพิมพ์ `#` Classic จะแนะนำแท็กที่มีอยู่

- แท็กล่าสุดปรากฏก่อน
- รองรับการจับคู่แบบ fuzzy
- กด Tab เพื่อยอมรับข้อเสนอแนะ

## สถิติแท็ก

ดูการใช้แท็กในการตั้งค่า

- แท็กที่ใช้มากที่สุด
- แท็กที่ไม่ได้ใช้
- แนวโน้มแท็กตามเวลา

## กรณีการใช้งาน

### การจัดการโครงการ

```markdown
# Project Kickoff #project-alpha #meeting #2024-01-15

Attendees: Team Alpha
Status: #todo
```

### ฐานความรู้

```markdown
# JavaScript Closures #programming #javascript #reference #advanced

Closures are functions that have access to...
```

### Daily Notes

```markdown
# 2024-01-15

- Morning standup #work #meetings
- Code review #work #development
- Gym session #personal #health
```

### บันทึกการอ่าน

```markdown
# Atomic Habits Notes #reading #books #productivity

Key takeaways from the book...
```

## ทางลัดคีย์บอร์ด

| การดำเนินการ | ทางลัด |
|--------|----------|
| เพิ่มแท็ก | `Cmd/Ctrl+T` |
| ค้นหาแท็ก | `Cmd/Ctrl+F`, จากนั้นพิมพ์ `tag:` |
| แสดงแผงแท็ก | `Cmd/Ctrl+Shift+T` |

## เคล็ดลับ

### การตั้งชื่อที่สอดคล้อง

ยึดติดกับหลักการ

```
✅ #meeting-notes
✅ #meeting_notes
❌ #Meeting Notes (ไม่อนุญาตช่องว่าง)
❌ #meetingNotes (ไม่สอดคล้อง)
```

### หลีกเลี่ยงการแท็กมากเกินไป

แท็กมากเกินไปจะจัดการไม่ได้

```
✅ #work #urgent #project-alpha
❌ #work #urgent #important #today #monday #office #desk #computer
```

### การทำความสะอาดเป็นระยะ

ตรวจสอบและรวมแท็กอย่างสม่ำเสมอ

- รวมแท็กที่คล้ายกัน
- ลบแท็กที่ไม่ได้ใช้
- อัปเดตมาตรฐาน

## คุณสมบัติขั้นสูง

### โฟลเดอร์อัจฉริยะตามแท็ก

สร้างโฟลเดอร์ที่เติมอัตโนมัติตามแท็ก

1. สร้างโฟลเดอร์ใหม่
2. ตั้งค่าตัวกรอง: `tag:project`
3. บันทึกที่แท็กทั้งหมดจะปรากฏ

### แท็กใน YAML

เพิ่มแท็กใน frontmatter ของบันทึก

```yaml
---
tags:
  - work
  - meetings
  - project-alpha
---
```

### นามแฝงแท็ก

ตั้งค่านามแฝงแท็กในการตั้งค่า

- `#todo` = `#task` = `#action`
- รวมแท็กที่คล้ายกัน
