# ภาษาและการแปลเป็นท้องถิ่น

Classic พูดภาษาของคุณ ใช้แอปในภาษาที่คุณต้องการและเขียนบันทึกในภาษาใดก็ได้ที่คุณเลือก

## ภาษาที่รองรับ

อินเทอร์เฟซของ Classic มีให้ในหลายภาษา

| ภาษา | รหัส |
|----------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| และอื่นๆ... | |

## การเปลี่ยนภาษาอินเทอร์เฟซ

### ผ่านการตั้งค่า

1. เปิดการตั้งค่า (`Cmd+,` หรือ `Ctrl+,`)
2. ไปที่ **General** → **Language**
3. เลือกภาษาที่คุณต้องการ
4. รีสตาร์ท Classic เพื่อใช้งาน

### ผ่าน Command Palette

1. เปิด Command Palette (`Cmd+P` หรือ `Ctrl+P`)
2. พิมพ์ "Language"
3. เลือกภาษาที่คุณต้องการ
4. รีสตาร์ทเพื่อใช้การเปลี่ยนแปลง

## การเขียนในหลายภาษา

Classic รองรับการเขียนบันทึกในภาษาใดก็ได้

### การรองรับ Unicode

- รองรับ Unicode เต็มรูปแบบ
- รองรับภาษา RTL (ขวาไปซ้าย)
- อักขระ CJK (จีน, ญี่ปุ่น, เกาหลี) ทำงานได้อย่างสมบูรณ์
- รองรับ Emoji

### การตรวจจับภาษา

Classic สามารถตรวจจับภาษาของบันทึกของคุณ

- ตรวจจับอัตโนมัติสำหรับการตรวจสอบการสะกด
- การตั้งค่าภาษาต่อบันทึก
- รองรับภาษาผสม

## การตรวจสอบการสะกด

### การเปิดใช้งานการตรวจสอบการสะกด

1. เปิดการตั้งค่า
2. ไปที่ **Editor** → **Spell Check**
3. เปิดใช้งาน "Spell Check"

### ภาษาการตรวจสอบการสะกด

1. เปิดการตั้งค่า
2. ไปที่ **Editor** → **Spell Check**
3. คลิก "Languages"
4. เลือกภาษาที่จะตรวจสอบ

### การเพิ่มคำลงในพจนานุกรม

1. คลิกขวาที่คำที่ถูกตั้งธง
2. เลือก "Add to Dictionary"
3. คำจะไม่ถูกตั้งธงอีก

## การรองรับ RTL

สำหรับภาษาอาหรับ, ฮีบรู และภาษา RTL อื่นๆ

### การตรวจจับ RTL อัตโนมัติ

Classic ตรวจจับเนื้อหา RTL โดยอัตโนมัติ

- ทิศทางข้อความปรับ
- การเคลื่อนไหวเคอร์เซอร์ปรับตัว
- การจัดเรียงเปลี่ยนแปลง

### การสลับ RTL ด้วยตนเอง

บังคับทิศทาง RTL หรือ LTR

1. เปิด Command Palette
2. พิมพ์ "RTL" หรือ "LTR"
3. เลือกความชอบของคุณ

## วิธีการป้อนข้อมูล

Classic ทำงานร่วมกับวิธีการป้อนข้อมูลของระบบของคุณ

### วิธีการป้อนข้อมูล CJK

- ทำงานร่วมกับ Pinyin, Zhuyin, ฯลฯ
- รองรับ Japanese IME
- รองรับวิธีการป้อนข้อมูลเกาหลี

### อักขระพิเศษ

แทรกอักขระพิเศษได้ง่าย

- ใช้ตัวเลือกอักขระของระบบของคุณ
- คัดลอก/วางจากแผนที่อักขระ
- ใช้การป้อนข้อมูล Unicode

## รูปแบบวันที่และเวลา

วันที่ปรับตาม locale ของคุณ

### ตัวเลือกรูปแบบ

| การตั้งค่า | ตัวเลือก |
|---------|---------|
| รูปแบบวันที่ | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| รูปแบบเวลา | 12 ชั่วโมง, 24 ชั่วโมง |
| วันแรกของสัปดาห์ | วันอาทิตย์, วันจันทร์, วันเสาร์ |

กำหนดค่าใน Settings → General → Date & Time

## รูปแบบตัวเลข

ตัวเลขแสดงตาม locale ของคุณ

- **ตัวคั่นทศนิยม**: `.` หรือ `,`
- **ตัวคั่นหลักพัน**: `,` หรือ `.` หรือช่องว่าง
- **สัญลักษณ์สกุลเงิน**: ขึ้นอยู่กับภูมิภาค

## เลย์เอาต์คีย์บอร์ด

Classic ทำงานกับเลย์เอาต์คีย์บอร์ดใดๆ

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- เลย์เอาต์ที่กำหนดเอง

## การตั้งค่าการแปลเป็นท้องถิ่น

### ใน Settings → General → Language

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **ภาษาอินเทอร์เฟซ** | ภาษา UI แอป |
| **รูปแบบวันที่** | วิธีแสดงวันที่ |
| **รูปแบบเวลา** | 12 หรือ 24 ชั่วโมง |
| **วันแรกของสัปดาห์** | วันเริ่มปฏิทิน |
| **รูปแบบตัวเลข** | ตัวคั่นทศนิยม/หลักพัน |

## เคล็ดลับ

### บันทึกภาษาผสม

เขียนในหลายภาษาในบันทึกเดียว

```markdown
# International Project

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### ฟอนต์เฉพาะภาษา

บางภาษาดูดีขึ้นด้วยฟอนต์เฉพาะ

1. เปิดการตั้งค่า
2. ไปที่ Appearance → Fonts
3. ตั้งค่าฟอนต์สำหรับสคริปต์ต่างๆ

### การสลับภาษาอย่างรวดเร็ว

สร้างทางลัดคีย์บอร์ด

1. เปิด Settings → Shortcuts
2. ค้นหา "Switch Language"
3. ตั้งค่าทางลัดที่คุณต้องการ

## การแก้ไขปัญหา

### ภาษาไม่พร้อมใช้งาน

- ตรวจสอบการอัปเดตแอป
- ภาษาอาจอยู่ระหว่างการพัฒนา
- มีส่วนร่วมในการแปลบน GitHub

### อักขระไม่แสดง

- ติดตั้งฟอนต์ที่จำเป็น
- ตรวจสอบการตั้งค่าฟอนต์
- ลองฟอนต์อื่น

### การตรวจสอบการสะกดไม่ทำงาน

- ตรวจสอบว่าการตรวจสอบการสะกดเปิดใช้งาน
- เพิ่มภาษาในการตรวจสอบการสะกด
- รีสตาร์ท Classic

### RTL ไม่ทำงานอย่างถูกต้อง

- ตรวจสอบให้แน่ใจว่าการตรวจจับ RTL เปิดอยู่
- ลองสลับ RTL ด้วยตนเอง
- ตรวจสอบปัญหาเนื้อหาผสม

## การมีส่วนร่วมในการแปล

ช่วยแปล Classic

1. เยี่ยมชมชุมชน Classic
2. เข้าร่วมทีมแปล
3. ส่งการแปล
4. ช่วยปรับปรุงการแปลเป็นท้องถิ่น
