# ทำความเข้าใจอินเทอร์เฟซ

อินเทอร์เฟซของ Classic ถูกออกแบบมาให้สะอาดตา ใช้งานง่าย และไม่รบกวนการทำงานของคุณ นี่คือทัวร์ชมส่วนประกอบหลัก

## เลย์เอาต์หลัก

Classic มีเลย์เอาต์แบบสามแผง

1. **แถบด้านข้าง** (ซ้าย) - การนำทางและการจัดระเบียบบันทึก
2. **ตัวแก้ไข** (กลาง) - พื้นที่เขียนของคุณ
3. **ตัวอย่าง/คุณสมบัติ** (ขวา) - แผงเสริมที่เลือกได้

## แถบด้านข้าง

แถบด้านข้างประกอบด้วยส่วนสำคัญหลายส่วน

### รายการบันทึก

- **All Notes**: ดูบันทึกทั้งหมดในรายการเดียว
- **Daily Notes**: เข้าถึงบันทึกประจำวันของวันนี้ได้อย่างรวดเร็ว
- **รายการโปรด**: ตั้งดาวบันทึกที่ใช้บ่อย
- **โฟลเดอร์**: จัดระเบียบบันทึกในลำดับชั้น

### การนำทาง

- **แถบค้นหา**: ค้นหาบันทึกได้ทันที
- **แท็ก**: เรียกดูตามแท็ก
- **แบ็คลิงก์**: ดูการเชื่อมโยงระหว่างบันทึก

### การดำเนินการ

- **New Note**: สร้างบันทึกใหม่
- **New Folder**: จัดระเบียบด้วยโฟลเดอร์
- **นำเข้า**: นำเนื้อหาจากภายนอกเข้ามา

## ตัวแก้ไข

ตัวแก้ไขคือที่ที่คุณเขียนและจัดรูปแบบบันทึกของคุณ

### คุณสมบัติ

- **รองรับ Markdown**: ไวยากรณ์ Markdown เต็มรูปแบบ
- **การแสดงตัวอย่างสด**: เห็นข้อความที่จัดรูปแบบขณะพิมพ์
- **การเน้นไวยากรณ์**: บล็อกโค้ดถูกเน้นสี
- **Slash Commands**: พิมพ์ `/` เพื่อใช้การดำเนินการด่วน

### โหมดตัวแก้ไข

| โหมด | คำอธิบาย |
|------|-------------|
| **แก้ไขอย่างเดียว** | เห็น Markdown ดิบ |
| **ตัวอย่างอย่างเดียว** | เห็นผลลัพธ์ที่แสดงผลแล้ว |
| **มุมมองแยก** | แก้ไขและดูตัวอย่างข้างกัน |

### แถบเครื่องมือ

การดำเนินการจัดรูปแบบด่วน

- หนา, ตัวเอียง, ขีดทับ
- หัวข้อ (H1-H6)
- รายการ (หัวข้อย่อย, ลำดับเลข, งาน)
- ลิงก์และรูปภาพ
- บล็อกโค้ด
- ตาราง

## แถบสถานะ

ที่ด้านล่างของตัวแก้ไข

- **จำนวนคำ/ตัวอักษร**: ติดตามการเขียนของคุณ
- **บันทึกล่าสุด**: เมื่อไหร่ที่บันทึกถูกบันทึกครั้งล่าสุด
- **เส้นทางบันทึก**: ตำแหน่งของบันทึก
- **สถานะซิงค์**: หากใช้การซิงค์เสริม

## เมนูตามบริบท

คลิกขวาเพื่อดูการดำเนินการตามบริบท

### เมนูบริบทบันทึก

- เปลี่ยนชื่อ
- ย้ายไปยังโฟลเดอร์
- ทำซ้ำ
- ลบ
- ส่งออก
- คัดลอกลิงก์

### เมนูบริบทตัวแก้ไข

- ตัด, คัดลอก, วาง
- ตัวเลือกจัดรูปแบบ
- แทรกลิงก์/รูปภาพ
- ค้นหาและแทนที่

## การนำทางด้วยคีย์บอร์ด

นำทางอินเทอร์เฟซด้วยคีย์บอร์ด

| การดำเนินการ | ทางลัด |
|--------|----------|
| เปิด/ปิดแถบด้านข้าง | `Cmd/Ctrl + M` |
| โฟกัสการค้นหา | `Cmd/Ctrl + F` |
| Command Palette | `Cmd/Ctrl + P` |
| สร้างบันทึกใหม่ | `Cmd/Ctrl + N` |

## การปรับแต่ง

ปรับอินเทอร์เฟซให้เหมาะกับคุณ

### ความกว้างแถบด้านข้าง

1. วางเมาส์เหนือขอบแถบด้านข้าง
2. ลากเพื่อปรับขนาด
3. หรือตั้งค่าความกว้างที่แน่นอนในการตั้งค่า

### ธีม

สลับระหว่างโหมดสว่างและมืด

- ใช้ปุ่มสลับธีมในส่วนหัว
- หรือตั้งค่าให้ตามการตั้งค่าระบบของคุณ

### ขนาดฟอนต์

1. เปิดการตั้งค่า
2. ไปที่ Appearance
3. ปรับ Editor Font Size

## เคล็ดลับ

### โหมดโฟกัส

ซ่อนแผงทั้งหมดเพื่อการเขียนโดยไม่มีสิ่งรบกวน

1. เปิด Command Palette
2. พิมพ์ "Focus Mode"
3. กด Enter

### การเข้าถึงด่วน

- ปักหมุดบันทึกที่ใช้บ่อยไปที่รายการโปรด
- ใช้ทางลัดคีย์บอร์ดสำหรับการดำเนินการทั่วไป
- ปรับแต่งแถบเครื่องมือในการตั้งค่า
