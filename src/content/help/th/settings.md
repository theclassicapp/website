# การตั้งค่า

ปรับแต่ง Classic ให้ทำงานตรงตามที่คุณต้องการ จากลักษณะที่ปรากฏไปจนถึงพฤติกรรมตัวแก้ไข ทุกอย่างสามารถกำหนดค่าได้

## การเปิดการตั้งค่า

### ทางลัดคีย์บอร์ด

| แพลตฟอร์ม | ทางลัด |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### ผ่านเมนู

- **macOS**: Classic → Settings
- **Windows**: File → Settings
- **Linux**: Edit → Settings

### ผ่าน Command Palette

1. เปิด Command Palette (`Cmd+P` หรือ `Ctrl+P`)
2. พิมพ์ "Settings"
3. กด Enter

## หมวดหมู่การตั้งค่า

### ทั่วไป

การตั้งค่าแอปพลิเคชันพื้นฐาน

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **Language** | ภาษาอินเทอร์เฟซ |
| **Theme** | สว่าง, มืด, หรือระบบ |
| **Start at Login** | เปิดตัวเมื่อเริ่มต้น |
| **Check for Updates** | พฤติกรรม auto-update |

### Appearance

ปรับแต่งรูปลักษณ์และความรู้สึก

| การตั้งค่า | ตัวเลือก |
|---------|---------|
| **Theme** | สว่าง / มืด / ระบบ |
| **Accent Color** | น้ำเงิน, ม่วง, เขียว, ฯลฯ |
| **Editor Font** | Font family และขนาด |
| **UI Font** | ฟอนต์อินเทอร์เฟซ |
| **Monospace Font** | ฟอนต์โค้ด |
| **Sidebar Width** | 200-400px |
| **Line Height** | 1.0-2.5 |

### Editor

กำหนดค่าพฤติกรรมการเขียน

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **Spell Check** | เปิด/ปิดการตรวจสอบการสะกด |
| **Auto-save** | ช่วงการบันทึกอัตโนมัติ |
| **Word Wrap** | เปิด/ปิด word wrap |
| **Line Numbers** | แสดง/ซ่อนหมายเลขบรรทัด |
| **Tab Size** | 2, 4, หรือ 8 ช่องว่าง |
| **Auto-correct** | Smart quotes, auto-replace |

### Version History

ควบคุมการติดตามเวอร์ชัน

| การตั้งค่า | ตัวเลือก | ค่าเริ่มต้น |
|---------|---------|---------|
| **Enable History** | เปิด/ปิด | เปิด |
| **Retention Period** | 30-180 วัน | 90 วัน |
| **Max Versions** | 50-500 | 100 |

### Backup & Restore

การตั้งค่าการป้องกันข้อมูล

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **Auto Backup** | เปิดใช้งานการสำรองข้อมูลอัตโนมัติ |
| **Backup Frequency** | 15นาที / 30นาที / 1ชม. / 4ชม. |
| **Backup Location** | เส้นทางที่กำหนดเอง |
| **Keep Backups** | 5 / 10 / 20 / 50 |
| **Encrypt Backups** | การป้องกันด้วยรหัสผ่าน |

### Keyboard Shortcuts

ดูและปรับแต่งทางลัด

- ดูทางลัดที่มีทั้งหมด
- ปรับแต่งทางลัดใดๆ
- รีเซ็ตเป็นค่าเริ่มต้น
- นำเข้า/ส่งออกทางลัด

### Import & Export

การตั้งค่าการย้ายข้อมูล

- นำเข้าจากแอปอื่น
- ตัวเลือกรูปแบบการส่งออก
- การดำเนินการแบบกลุ่ม

### Privacy

การตั้งค่าความเป็นส่วนตัวและความปลอดภัย

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **Analytics** | เปิด/ปิด analytics การใช้งาน |
| **Crash Reports** | ส่งรายงาน crash |
| **Clear History** | ล้างการค้นหา/ประวัติ |

### Notifications

ควบคุมการแจ้งเตือน

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **Enable Notifications** | เปิด/ปิด |
| **Daily Note Reminder** | เวลาเตือน |
| **Sound** | เสียงการแจ้งเตือน |

### Advanced

การตั้งค่าสำหรับผู้ใช้ขั้นสูง

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **Custom CSS** | เพิ่มสไตล์ที่กำหนดเอง |
| **Debug Mode** | เปิด debug logging |
| **Experimental Features** | คุณสมบัติ beta |
| **Data Location** | เปลี่ยนโฟลเดอร์บันทึก |

## การค้นหาการตั้งค่า

หาการตั้งค่าไม่เจอ? ใช้แถบค้นหา

1. เปิดการตั้งค่า
2. พิมพ์ในกล่องค้นหา
3. การตั้งค่าที่ตรงกันปรากฏขึ้น
4. คลิกเพื่อกระโดดไปยังการตั้งค่านั้น

## การรีเซ็ตการตั้งค่า

### รีเซ็ตการตั้งค่าทั้งหมด

กลับไปเป็นค่าเริ่มต้น

1. เปิดการตั้งค่า
2. เลื่อนไปที่ด้านล่าง
3. คลิก "Reset All Settings"
4. ยืนยันการดำเนินการ

### รีเซ็ตหมวดหมู่

รีเซ็ตหมวดหมู่เฉพาะ

1. ไปที่หมวดหมู่
2. คลิก "Reset to Defaults"
3. เฉพาะหมวดหมู่นั้นที่รีเซ็ต

## การส่งออก/นำเข้าการตั้งค่า

### ส่งออกการตั้งค่า

บันทึกการกำหนดค่าของคุณ

1. เปิดการตั้งค่า
2. คลิก "Export Settings"
3. เลือกตำแหน่ง
4. บันทึกเป็นไฟล์ JSON

### นำเข้าการตั้งค่า

โหลดการกำหนดค่าที่บันทึกไว้

1. เปิดการตั้งค่า
2. คลิก "Import Settings"
3. เลือกไฟล์ JSON
4. ยืนยันการนำเข้า

## ตำแหน่งการตั้งค่า

การตั้งค่าถูกเก็บไว้ในเครื่อง

| แพลตฟอร์ม | ตำแหน่ง |
|----------|----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## การซิงค์การตั้งค่า (ไม่บังคับ)

เมื่อลงชื่อเข้าใช้ การตั้งค่าสามารถซิงค์ระหว่างอุปกรณ์

- **ซิงค์**: เปิดใช้งานใน Settings → Sync
- **สิ่งที่ซิงค์**: Theme, fonts, shortcuts
- **สิ่งที่ไม่ซิงค์**: Local paths, encryption keys

## การปรับแต่งทั่วไป

### เพิ่มขนาดฟอนต์

1. Settings → Appearance
2. Editor Font Size → เพิ่ม
3. การเปลี่ยนแปลงใช้ทันที

### เปลี่ยนธีม

1. Settings → Appearance
2. Theme → เลือก Light/Dark/System
3. แอปอัปเดตทันที

### เปิดใช้งาน Auto-Backup

1. Settings → Backup & Restore
2. เปิดใช้งาน "Auto Backup"
3. ตั้งค่าความถี่และตำแหน่ง

### ปรับแต่งทางลัด

1. Settings → Keyboard Shortcuts
2. ค้นหาการดำเนินการ
3. คลิกทางลัดปัจจุบัน
4. กดคีย์ใหม่
5. บันทึก

## เคล็ดลับ

### การเข้าถึงด่วน

- ใช้ `Cmd/Ctrl+,` เพื่อเปิดการตั้งค่าอย่างรวดเร็ว
- ค้นหาการตั้งค่าตามชื่อ
- ใช้หมวดหมู่เพื่อนำทาง

### ส่งออกก่อนการเปลี่ยนแปลง

ก่อนการเปลี่ยนแปลงใหญ่ ส่งออกการตั้งค่าของคุณ

- ง่ายต่อการกู้คืนหากมีบางอย่างผิดพลาด
- แชร์การกำหนดค่ากับผู้อื่น

### ตรวจสอบการตั้งค่าใหม่

การอัปเดตอาจเพิ่มการตั้งค่าใหม่

- ตรวจสอบหลังจากอัปเดต
- สำรวจตัวเลือกใหม่

## การแก้ไขปัญหา

### การตั้งค่าไม่บันทึก

- ตรวจสอบสิทธิ์ไฟล์
- ตรวจสอบพื้นที่ดิสก์
- ลองรีเซ็ตและกำหนดค่าใหม่

### การตั้งค่ารีเซ็ตโดยไม่คาดคิด

- อาจบ่งชี้ถึงการเสียหายของไฟล์
- ส่งออกการตั้งค่าอย่างสม่ำเสมอ
- ตรวจสอบความขัดแย้งของแอป

### ไม่พบการตั้งค่า

- ใช้แถบค้นหา
- ตรวจสอบหมวดหมู่ต่างๆ
- อาจอยู่ในการตั้งค่า Advanced
