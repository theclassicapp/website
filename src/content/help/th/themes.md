# ธีมและลักษณะที่ปรากฏ

ปรับแต่งลักษณะที่ปรากฏของ Classic ให้ตรงกับสไตล์ของคุณและลดความเครียดต่อสายตา

## ตัวเลือกธีม

### โหมดสว่าง

ธีมเริ่มต้นของ Classic มีคุณสมบัติ

- อินเทอร์เฟซสะอาด สว่าง
- เหมาะสำหรับสภาพแวดล้อมที่สว่าง
- คอนทราสต์สูงสำหรับความสามารถในการอ่าน

### โหมดมืด

เหมาะสำหรับคนชอบกลางคืนและการทำงานในแสงน้อย

- ลดความเครียดต่อสายตา
- ใช้แบตเตอรี่น้อยลงบนหน้าจอ OLED
- ง่ายต่อสายตาในสภาพแวดล้อมที่มืด

### ระบบ (อัตโนมัติ)

ตามธีมระบบปฏิบัติการของคุณ

- สลับอัตโนมัติตามการตั้งค่า OS
- ไม่ต้องสลับด้วยตนเอง
- สอดคล้องกับแอปอื่นๆ

## การสลับธีม

### ผ่านการตั้งค่า

1. เปิดการตั้งค่า (`Cmd+,` หรือ `Ctrl+,`)
2. ไปที่ **Appearance**
3. เลือกธีมที่คุณต้องการ

### ผ่าน Command Palette

1. เปิด Command Palette (`Cmd+P` หรือ `Ctrl+P`)
2. พิมพ์ "Theme"
3. เลือกธีมที่คุณต้องการ

### ทางลัดคีย์บอร์ด

| แพลตฟอร์ม | ทางลัด |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

วนผ่าน: สว่าง → มืด → ระบบ

## การตั้งค่าฟอนต์

### ฟอนต์ตัวแก้ไข

ปรับแต่งประสบการณ์การเขียน

1. เปิดการตั้งค่า
2. ไปที่ **Appearance** → **Editor**
3. กำหนดค่าตัวเลือกฟอนต์

| การตั้งค่า | ตัวเลือก | ค่าเริ่มต้น |
| ------- | ------- | ------- |
| Font Family | ฟอนต์ระบบ + กำหนดเอง | System UI |
| Font Size | 12-24px | 16px |
| Line Height | 1.0-2.5 | 1.75 |
| Letter Spacing | -2 ถึง 4 | 0 |

### ฟอนต์ UI

ปรับฟอนต์อินเทอร์เฟซ

1. เปิดการตั้งค่า
2. ไปที่ **Appearance** → **Interface**
3. เลือกฟอนต์และขนาด

### ฟอนต์ Monospace

สำหรับบล็อกโค้ดและเนื้อหาทางเทคนิค

1. เปิดการตั้งค่า
2. ไปที่ **Appearance** → **Code**
3. เลือกฟอนต์ monospace

## ตัวเลือกอินเทอร์เฟซ

### แถบด้านข้าง

ปรับแต่งพฤติกรรมแถบด้านข้าง

| การตั้งค่า | คำอธิบาย |
| ------- | ----------- |
| **ซ่อนอัตโนมัติ** | ซ่อนแถบด้านข้างเมื่อไม่ใช้ |
| **ความกว้าง** | ปรับความกว้างแถบด้านข้าง (200-400px) |
| **แสดงไอคอน** | เปิด/ปิดการมองเห็นไอคอน |

### เลย์เอาต์ตัวแก้ไข

เลือกประสบการณ์การแก้ไขของคุณ

- **มุมมองแยก**: แก้ไขและดูตัวอย่างข้างกัน
- **ตัวอย่างอย่างเดียว**: ดูผลลัพธ์ที่แสดงผลแล้วเท่านั้น
- **แก้ไขอย่างเดียว**: โฟกัสที่ Markdown ดิบ

### รายการบันทึก

กำหนดค่าวิธีแสดงบันทึก

- **มุมมองรายการ**: กระชับ หนึ่งบรรทัดต่อบันทึก
- **มุมมองการ์ด**: การ์ดตัวอย่างพร้อมสไนปเป็ต
- **มุมมองต้นไม้**: โครงสร้างโฟลเดอร์แบบลำดับชั้น

## สีเน้น

### สีที่มี

เลือกสีเน้นของคุณ

- Classic Blue (ค่าเริ่มต้น)
- ม่วง
- เขียว
- ส้ม
- แดง
- ชมพู
- Teal

### การเปลี่ยนสีเน้น

1. เปิดการตั้งค่า
2. ไปที่ **Appearance** → **Colors**
3. เลือกสีเน้นที่คุณต้องการ

สีเน้นมีผลต่อ

- ลิงก์และปุ่ม
- การเน้นการเลือก
- องค์ประกอบที่ใช้งานอยู่
- ตัวบ่งชี้โฟกัส

## การช่วยการเข้าถึง

### โหมดคอนทราสต์สูง

สำหรับการมองเห็นที่ดีขึ้น

1. เปิดการตั้งค่า
2. ไปที่ **Appearance** → **Accessibility**
3. เปิดใช้งาน **High Contrast**

### ลดการเคลื่อนไหว

ลดภาพเคลื่อนไหวให้น้อยที่สุด

1. เปิดการตั้งค่า
2. ไปที่ **Appearance** → **Accessibility**
3. เปิดใช้งาน **Reduced Motion**

### การปรับฟอนต์

- เพิ่มขนาดฟอนต์ขั้นต่ำ
- ใช้ฟอนต์ที่เป็นมิตรกับ dyslexia
- ปรับอัตราส่วนคอนทราสต์

## CSS ที่กำหนดเอง (ขั้นสูง)

สำหรับผู้ใช้ขั้นสูงที่ต้องการควบคุมอย่างสมบูรณ์

1. เปิดการตั้งค่า
2. ไปที่ **Appearance** → **Advanced**
3. คลิก **Open Custom CSS**
4. เพิ่มการปรับแต่งของคุณ

### ตัวอย่างการปรับแต่ง

```css
/* หัวข้อใหญ่ขึ้น */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* สีลิงก์ที่กำหนดเอง */
a { color: #3b82f6; }

/* ระยะห่างบรรทัดกว้างขึ้น */
.prose p { line-height: 2; }
```

## การซิงค์ธีม

เมื่อคุณลงชื่อเข้าใช้ (ไม่บังคับ) การตั้งค่าธีมของคุณจะซิงค์ระหว่างอุปกรณ์

- การซิงค์ธีมอัตโนมัติ
- การตั้งค่าฟอนต์ซิงค์
- CSS ที่กำหนดเองซิงค์

## เคล็ดลับ

### การจับคู่เวิร์กโฟลว์ของคุณ

- ใช้ **สว่าง** ช่วงกลางวัน **มืด** ช่วงกลางคืน
- โหมด **ระบบ** จัดการสิ่งนี้โดยอัตโนมัติ
- พิจารณาแสงรอบข้างของคุณ

### การลดความเครียดต่อสายตา

- ใช้โหมดมืดในแสงน้อย
- เพิ่มขนาดฟอนต์สำหรับเซสชันยาว
- เปิดใช้งาน High Contrast หากจำเป็น
- หยุดพักสม่ำเสมอ

### ประสิทธิภาพ

- เรียนรู้ทางลัดสลับธีม
- ตั้งค่าเวลาสลับอัตโนมัติ
- ใช้โหมดระบบเพื่อความสอดคล้อง
