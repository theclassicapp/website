# Slash Commands

Slash commands เป็นวิธีที่มีประสิทธิภาพในการแทรกการจัดรูปแบบและเนื้อหาพิเศษได้อย่างรวดเร็วโดยไม่ต้องออกจากคีย์บอร์ด พิมพ์ `/` ในตัวแก้ไขเพื่อเปิด command palette

## คำสั่งที่มี

### หัวข้อ

| คำสั่ง | ผลลัพธ์ |
| ------- | ------ |
| `/h1` | หัวข้อระดับ 1 |
| `/h2` | หัวข้อระดับ 2 |
| `/h3` | หัวข้อระดับ 3 |
| `/h4` | หัวข้อระดับ 4 |
| `/h5` | หัวข้อระดับ 5 |
| `/h6` | หัวข้อระดับ 6 |

### การจัดรูปแบบข้อความ

| คำสั่ง | ผลลัพธ์ |
| ------- | ------ |
| `/bold` | **ข้อความหนา** |
| `/italic` | *ข้อความตัวเอียง* |
| `/strikethrough` | ~~ขีดทับ~~ |
| `/code` | บล็อกโค้ดอินไลน์ |
| `/quote` | บล็อกคำพูด |

### รายการ

| คำสั่ง | ผลลัพธ์ |
| ------- | ------ |
| `/bullet` | • รายการหัวข้อย่อย |
| `/numbered` | 1. รายการลำดับเลข |
| `/todo` | - [ ] รายการสิ่งที่ต้องทำ |
| `/check` | - [x] รายการที่เสร็จแล้ว |

### แทรกเนื้อหา

| คำสั่ง | ผลลัพธ์ |
| ------- | ------ |
| `/image` | แทรกรูปภาพจาก URL หรือไฟล์ |
| `/link` | แทรกไฮเปอร์ลิงก์ |
| `/table` | แทรกตาราง |
| `/hr` | เส้นคั่นแนวนอน |
| `/code` | บล็อกโค้ดพร้อมการเลือกภาษา |

### บล็อกพิเศษ

| คำสั่ง | ผลลัพธ์ |
| ------- | ------ |
| `/mermaid` | บล็อกไดอะแกรม Mermaid |
| `/katex` | บล็อกคณิตศาสตร์ KaTeX |
| `/callout` | บล็อก callout/admonition |

## การใช้ Slash Commands

1. พิมพ์ `/` ในตัวแก้ไข
2. เมนูจะปรากฏขึ้นพร้อมคำสั่งที่มีทั้งหมด
3. เริ่มพิมพ์เพื่อกรองคำสั่ง
4. กด `Enter` หรือคลิกเพื่อแทรก

## เคล็ดลับ Pro

- **การเข้าถึงด่วน**: เพียงพิมพ์ `/` และเริ่มพิมพ์ชื่อคำสั่ง
- **การค้นหาแบบ fuzzy**: คุณไม่จำเป็นต้องพิมพ์ชื่อคำสั่งที่แน่นอน - Classic จะหาที่ใกล้เคียงที่สุด
- **การนำทางด้วยคีย์บอร์ด**: ใช้ปุ่มลูกศรเพื่อนำทางในเมนู
- **คำสั่งล่าสุด**: คำสั่งที่ใช้บ่อยจะปรากฏที่ด้านบนของรายการ
