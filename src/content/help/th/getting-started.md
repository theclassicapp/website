# คู่มือเริ่มต้นใช้งาน

ยินดีต้อนรับสู่ Classic! คู่มือนี้จะช่วยให้คุณเริ่มต้นใช้งาน Classic ได้ภายในไม่กี่นาที

## ความต้องการของระบบ

- **macOS**: macOS 12 (Monterey) หรือใหม่กว่า
- **Windows**: Windows 10 หรือใหม่กว่า
- **Linux**: Ubuntu 20.04+ / Fedora 36+ หรือใหม่กว่า

## การติดตั้ง

1. ดาวน์โหลด Classic จาก[เว็บไซต์](/)
2. เปิดแอปพลิเคชัน
3. เริ่มต้นเขียนได้เลย!

## ประสบการณ์การใช้งานครั้งแรก

เมื่อคุณเปิด Classic ครั้งแรก คุณจะเห็นอินเทอร์เฟซที่สะอาดและเรียบง่าย ออกแบบมาเพื่อให้คุณโฟกัสกับการเขียน

![Classic App Screenshot]

### แถบด้านข้าง

ทางด้านซ้าย คุณจะพบกับ
- **Daily Notes** - บันทึกประจำวันอัตโนมัติของคุณ
- **ค้นหา** - ค้นหาอะไรก็ได้ในบันทึกทั้งหมดของคุณได้อย่างรวดเร็ว
- **Command Palette** - กด `Cmd/Ctrl+P` (Windows/Linux) หรือ `Cmd+Shift+P` (macOS) เพื่อเปิด command palette
- **เค้าร่าง** - เรียกดูและจัดระเบียบบันทึกของคุณในโครงสร้างแบบต้นไม้

### ตัวแก้ไข

พื้นที่ตรงกลางคือ **ตัวแก้ไข Markdown** พร้อมด้วย:
- การแสดงตัวอย่างแบบเรียลไทม์
- การเน้นไวยากรณ์
- เครื่องมือจัดรูปแบบที่หลากหลาย

### สร้างบันทึกแรกของคุณ

1. คลิกปุ่ม **New Note** ในแถบด้านข้าง หรือกด `Ctrl/Cmd+N` (Windows/Linux) หรือ `Cmd+N` (macOS)
2. ตั้งชื่อบันทึกของคุณ
3. เริ่มเขียนในรูปแบบ Markdown!

```markdown
# My First Note

Welcome to Classic! This is where I'll capture my thoughts and ideas.
```
