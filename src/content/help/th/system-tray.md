# System Tray

Classic อยู่ใน system tray ของคุณเพื่อการเข้าถึงด่วน บันทึกความคิดได้ทันทีโดยไม่ขัดจังหวะเวิร์กโฟลว์ของคุณ

## System Tray คืออะไร?

System tray (Windows/Linux) หรือ menu bar (macOS) คือที่ที่ Classic ทำงานในพื้นหลัง พร้อมเสมอสำหรับการบันทึกด่วน

### ประโยชน์

- **พร้อมใช้งานเสมอ**: ห่างแค่คลิกเดียว
- **การบันทึกด่วน**: จดความคิดได้ทันที
- **ทำงานในพื้นหลัง**: ไม่จำเป็นต้องเปิดแอปไว้
- **การแจ้งเตือน**: ได้รับการเตือนของงาน

## การเปิดใช้งาน System Tray

### macOS

1. เปิด Classic
2. ไปที่ Settings → General
3. เปิดใช้งาน "Show in Menu Bar"
4. ไอคอน Classic ปรากฏใน menu bar

### Windows

1. เปิด Classic
2. ไปที่ Settings → General
3. เปิดใช้งาน "Show in System Tray"
4. ไอคอน Classic ปรากฏใน system tray

### Linux

1. เปิด Classic
2. ไปที่ Settings → General
3. เปิดใช้งาน "System Tray Icon"
4. ขึ้นอยู่กับ desktop environment

## Quick Capture

### การเปิด Quick Capture

คลิกไอคอน Classic ใน system tray

- **macOS**: ไอคอน menu bar
- **Windows**: ไอคอน system tray
- **Linux**: ไอคอน system tray

### การใช้ Quick Capture

1. คลิกไอคอน system tray
2. หน้าต่าง quick capture เปิดขึ้น
3. พิมพ์บันทึกของคุณ
4. กด Enter เพื่อบันทึก
5. หน้าต่างปิดอัตโนมัติ

### ทางลัดคีย์บอร์ด

เปิด Quick Capture จากที่ไหนก็ได้

| แพลตฟอร์ม | ทางลัด |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (ทั่วโลก) |
| **Windows** | `Ctrl+Shift+N` (ทั่วโลก) |
| **Linux** | `Ctrl+Shift+N` (ทั่วโลก) |

## เมนู System Tray

คลิกขวาที่ไอคอน tray เพื่อดูตัวเลือก

### ตัวเลือกเมนู

| ตัวเลือก | การดำเนินการ |
|--------|--------|
| **New Note** | เปิด Quick Capture |
| **New Daily Note** | เปิดบันทึกประจำวันของวันนี้ |
| **ค้นหา** | เปิดหน้าต่างค้นหา |
| **เปิด Classic** | เปิดหน้าต่างหลัก |
| **การตั้งค่า** | เปิดการตั้งค่า |
| **ออก** | ออกจาก Classic |

## การตั้งค่า Tray

### กำหนดค่าในการตั้งค่า

| การตั้งค่า | คำอธิบาย |
|---------|-------------|
| **แสดงไอคอน** | เปิด/ปิดการมองเห็นไอคอน tray |
| **เริ่มต้นแบบย่อ** | เริ่ม Classic ใน tray เท่านั้น |
| **ปิดไปที่ Tray** | ทำงานต่อเมื่อปิดหน้าต่าง |
| **การแจ้งเตือน** | เปิด/ปิดการแจ้งเตือน |

### เริ่มต้นเมื่อ Login

ให้ Classic เริ่มต้นอัตโนมัติ

1. เปิดการตั้งค่า
2. ไปที่ General
3. เปิดใช้งาน "Start at Login"
4. Classic เปิดตัวเมื่อเริ่มต้นระบบ

## การแจ้งเตือน

### การเปิดใช้งานการแจ้งเตือน

1. เปิดการตั้งค่า
2. ไปที่ Notifications
3. เปิด/ปิด "Enable Notifications"

### ประเภทการแจ้งเตือน

- **การเตือนความจำ**: การเตือนความจำที่กำหนดเวลา
- **Daily Note**: การแจ้งเตือน daily note
- **อัปเดต**: มีอัปเดตแอปพร้อมใช้งาน

## พฤติกรรมหน้าต่าง

### พฤติกรรมการปิด

เลือกสิ่งที่เกิดขึ้นเมื่อคุณปิดหน้าต่าง

- **ย่อไปที่ Tray**: ทำงานต่อ ซ่อนหน้าต่าง
- **ออก**: ออกจากแอปพลิเคชัน
- **ถาม**: ถามทุกครั้ง

### พฤติกรรมการย่อ

- **ย่อไปที่ Tray**: ซ่อนใน system tray
- **ย่อไปที่ Taskbar**: ย่อตามปกติ

## เคล็ดลับ

### การบันทึกด่วน

ใช้ทางลัดทั่วโลกสำหรับการบันทึกที่เร็วที่สุด

1. กด `Cmd/Ctrl+Shift+N` ที่ไหนก็ได้
2. พิมพ์ความคิดของคุณ
3. กด Enter
4. ทำงานต่อ

### การเข้าถึงด่วน

คลิกไอคอน tray เพื่อดู

- บันทึกล่าสุด
- การดำเนินการด่วน
- Daily note ของวันนี้

### การทำงานในพื้นหลัง

รักษา Classic ทำงานสำหรับ

- การเริ่มต้นที่เร็วขึ้น
- การบันทึกด่วน
- การแจ้งเตือน
- การสำรองข้อมูลอัตโนมัติ

## การแก้ไขปัญหา

### ไอคอน Tray หายไป

**macOS**

1. ตรวจสอบว่าไอคอนถูกซ่อน
2. มองใน Control Center (macOS Big Sur+)
3. รีสตาร์ท Classic

**Windows**

1. ตรวจสอบไอคอนที่ซ่อน (ลูกศรใน taskbar)
2. ลากไอคอน Classic ไปยังพื้นที่ที่มองเห็น
3. รีสตาร์ท Classic

**Linux**

1. ตรวจสอบการรองรับ desktop environment
2. ติดตั้ง system tray extension หากจำเป็น
3. รีสตาร์ท Classic

### ทางลัดทั่วโลกไม่ทำงาน

- ตรวจสอบความขัดแย้งกับแอปอื่น
- เปิดใช้งานทางลัดใหม่ในการตั้งค่า
- รีสตาร์ท Classic
- ตรวจสอบสิทธิ์ OS

### การแจ้งเตือนไม่แสดง

**macOS**

1. System Preferences → Notifications
2. ค้นหา Classic
3. เปิดใช้งานการแจ้งเตือน

**Windows**

1. Settings → System → Notifications
2. เปิดใช้งานการแจ้งเตือน Classic

**Linux**

1. ตรวจสอบ notification daemon
2. ติดตั้ง libnotify หากจำเป็น
