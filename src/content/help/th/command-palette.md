# Command Palette

Command palette เป็นประตูสู่ทุกการดำเนินการใน Classic กดทางลัดคีย์บอร์ดง่ายๆ และเข้าถึงคุณสมบัติใดๆ ได้ทันที

## การเปิด Command Palette

| แพลตฟอร์ม | ทางลัด |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## วิธีใช้

1. เปิด command palette (`Cmd+P` หรือ `Ctrl+P`)
2. พิมพ์เพื่อค้นหาคำสั่ง
3. ใช้ปุ่มลูกศรเพื่อนำทาง
4. กด `Enter` เพื่อดำเนินการ

## หมวดหมู่คำสั่ง

### การดำเนินการไฟล์

- **New Note**: สร้างบันทึกใหม่
- **New Daily Note**: สร้าง/เปิดบันทึกประจำวันของวันนี้
- **Open Note**: ค้นหาและเปิดบันทึกที่มีอยู่
- **Save Note**: บันทึกบันทึกปัจจุบัน
- **Delete Note**: ลบบันทึกปัจจุบัน
- **Rename Note**: เปลี่ยนชื่อบันทึกปัจจุบัน

### การนำทาง

- **Go to Note**: กระโดดไปยังบันทึกใดๆ
- **Go to Daily Note**: เปิดวันที่เฉพาะเจาะจง
- **Go to Tag**: เรียกดูบันทึกตามแท็ก
- **Go Back**: นำทางย้อนกลับในประวัติ
- **Go Forward**: นำทางไปข้างหน้าในประวัติ

### ตัวแก้ไข

- **Toggle Preview**: สลับระหว่างโหมดแก้ไขและตัวอย่าง
- **Toggle Sidebar**: แสดง/ซ่อนแถบด้านข้าง
- **Focus Mode**: เข้าสู่การเขียนโดยไม่มีสิ่งรบกวน
- **Insert Date**: เพิ่มวันที่/เวลาปัจจุบัน
- **Insert Template**: แทรกเทมเพลต

### การจัดรูปแบบ

- **Bold**: ทำข้อความหนา
- **Italic**: ทำข้อความตัวเอียง
- **Strikethrough**: ขีดทับข้อความ
- **Code**: จัดรูปแบบเป็นโค้ด
- **Link**: แทรกลิงก์
- **Heading**: ใช้รูปแบบหัวข้อ

### การจัดระเบียบ

- **Add Tag**: แท็กบันทึกปัจจุบัน
- **Move to Folder**: ย้ายบันทึกไปยังโฟลเดอร์
- **Archive Note**: เก็บถาวรบันทึกปัจจุบัน
- **Pin Note**: ปักหมุดบันทึกไว้ด้านบนของรายการ

### มุมมอง

- **Toggle Theme**: สลับระหว่างสว่าง/มืด
- **Zoom In**: เพิ่มขนาดฟอนต์
- **Zoom Out**: ลดขนาดฟอนต์
- **Reset Zoom**: รีเซ็ตเป็นขนาดเริ่มต้น

## เคล็ดลับ

### การเข้าถึงด่วน

- คำสั่งถูกค้นหาแบบ fuzzy
- คำสั่งล่าสุดปรากฏก่อน
- ปักหมุดคำสั่งที่ใช้บ่อย

### การนำทางด้วยคีย์บอร์ด

- `↑`/`↓` - นำทางผลลัพธ์
- `Enter` - ดำเนินการคำสั่ง
- `Escape` - ปิด palette
- `Tab` - วนผ่านหมวดหมู่

### คำสั่งที่กำหนดเอง

1. เปิดการตั้งค่า
2. ไปที่ Command Palette
3. เพิ่มคำสั่งที่กำหนดเอง
4. กำหนดทางลัดคีย์บอร์ด

## คำสั่งที่ใช้บ่อย

นี่คือคำสั่งบางอย่างที่คุณจะใช้บ่อย

| คำสั่ง | มันทำอะไร |
| ------- | ------------- |
| `New Note` | สร้างบันทึกเปล่า |
| `Open Note` | ค้นหาบันทึกทั้งหมด |
| `New Daily Note` | บันทึกประจำวันของวันนี้ |
| `Toggle Preview` | ตัวอย่าง vs แก้ไข |
| `Add Tag` | แท็กบันทึกปัจจุบัน |
| `Quick Capture` | บันทึกทั่วโลก |

## การปรับแต่งคำสั่ง

### การเปลี่ยนทางลัด

1. เปิดการตั้งค่า (`Cmd+,` หรือ `Ctrl+,`)
2. ไปที่ Keyboard Shortcuts
3. ค้นหาคำสั่ง
4. คลิกทางลัดปัจจุบัน
5. กดทางลัดใหม่ของคุณ
6. บันทึก

### การสร้างคำสั่งที่กำหนดเอง

ผู้ใช้ขั้นสูงสามารถสร้างคำสั่งที่กำหนดเองได้

1. เปิดการตั้งค่า
2. ไปที่ Commands
3. คลิก "Add Custom Command"
4. กำหนดชื่อคำสั่ง
5. เพิ่มการดำเนินการ
6. ตั้งค่าทางลัด (ไม่บังคับ)
