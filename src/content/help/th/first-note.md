# สร้างบันทึกแรกของคุณ

เริ่มต้นใช้งาน Classic ด้วยการสร้างบันทึกแรกของคุณ ง่ายและใช้งานง่าย

## การสร้างบันทึกใหม่

### ใช้แถบด้านข้าง

1. คลิกปุ่ม **+** ในแถบด้านข้าง
2. บันทึกใหม่จะถูกสร้างขึ้น
3. เริ่มพิมพ์เนื้อหาของคุณ

### ใช้ทางลัดคีย์บอร์ด

| แพลตฟอร์ม | ทางลัด |
| -------- | -------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### ใช้ Command Palette

1. เปิด Command Palette (`Cmd+P` หรือ `Ctrl+P`)
2. พิมพ์ "New Note"
3. กด Enter

## การเขียนบันทึกของคุณ

### การแก้ไขพื้นฐาน

เริ่มพิมพ์ได้เลย! Classic บันทึกอัตโนมัติขณะคุณเขียน

### การเพิ่มชื่อเรื่อง

บรรทัดแรกของบันทึกจะเป็นชื่อเรื่อง ใช้หัวข้อเพื่อผลลัพธ์ที่ดีที่สุด

```markdown
# My First Note

This is the content of my note.
```

### การจัดรูปแบบข้อความ

ใช้ Markdown เพื่อจัดรูปแบบข้อความของคุณ

- **หนา**: `**text**` หรือ `Cmd/Ctrl+B`
- **ตัวเอียง**: `*text*` หรือ `Cmd/Ctrl+I`
- **ขีดทับ**: `~~text~~`
- **โค้ด**: `` `code` ``

## การบันทึก

Classic บันทึกบันทึกของคุณอัตโนมัติขณะคุณพิมพ์ ไม่ต้องบันทึกด้วยตนเอง

### การบันทึกด้วยตนเอง

หากจำเป็น คุณสามารถบังคับบันทึก

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## การจัดระเบียบบันทึกของคุณ

### การเพิ่มแท็ก

แท็กบันทึกของคุณเพื่อค้นหาง่ายในภายหลัง

1. พิมพ์ `#` ตามด้วยชื่อแท็กของคุณ
2. หรือใช้ปุ่มแท็กในแถบเครื่องมือ
3. รองรับหลายแท็ก

ตัวอย่าง

```markdown
# Meeting Notes

Discussed Q4 planning #work #meetings
```

### การย้ายไปยังโฟลเดอร์

1. คลิกขวาที่บันทึกในแถบด้านข้าง
2. เลือก "Move to Folder"
3. เลือกหรือสร้างโฟลเดอร์ปลายทาง

### การปักหมุดบันทึกสำคัญ

1. คลิกขวาที่บันทึก
2. เลือก "Pin to Top"
3. บันทึกจะปรากฏที่ด้านบนของรายการของคุณ

## ดูตัวอย่างบันทึกของคุณ

ดูว่าบันทึกที่จัดรูปแบบแล้วมีลักษณะอย่างไร

- คลิกปุ่ม **Preview**
- หรือใช้ `Cmd/Ctrl+Shift+P`
- สลับระหว่างโหมดแก้ไขและตัวอย่าง

## ขั้นตอนถัดไป

ตอนนี้คุณได้สร้างบันทึกแรกแล้ว

- เรียนรู้เพิ่มเติมเกี่ยวกับ [Markdown](/help/markdown)
- จัดระเบียบด้วย [แท็ก](/help/tags)
- ตั้งค่า [Daily Notes](/help/daily-notes)
- สำรวจ [ทางลัดคีย์บอร์ด](/help/keyboard-shortcuts)

## เคล็ดลับ

### ชื่อบันทึก

- ใช้ชื่อที่สื่อความหมาย
- บรรทัดแรก = ชื่อบันทึก
- ใช้ H1 (`#`) เพื่อความชัดเจน

### การบันทึกอัตโนมัติ

- ไม่ต้องกังวลเรื่องการสูญเสียงาน
- การเปลี่ยนแปลงถูกบันทึกอัตโนมัติ
- ตรวจสอบแถบสถานะสำหรับสถานะการบันทึก

### Quick Capture

ต้องการจดอะไรด่วน?

- ใช้ Quick Capture (`Cmd/Ctrl+Shift+N`)
- สร้างบันทึกทันที
- เหมาะสำหรับความคิดชั่วคราว
