# Command Palette

The command palette is your gateway to every action in Classic. Press a simple keyboard shortcut and access any feature instantly.

## Opening the Command Palette

| Platform | Shortcut |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## How to Use

1. Open the command palette (`Cmd+P` or `Ctrl+P`)
2. Type to search for a command
3. Use arrow keys to navigate
4. Press `Enter` to execute

## Command Categories

### File Operations

- **New Note**: Create a new note
- **New Daily Note**: Create/open today's daily note
- **Open Note**: Search and open existing notes
- **Save Note**: Save current note
- **Delete Note**: Delete current note
- **Rename Note**: Rename current note

### Navigation

- **Go to Note**: Jump to any note
- **Go to Daily Note**: Open a specific date
- **Go to Tag**: Browse notes by tag
- **Go Back**: Navigate back in history
- **Go Forward**: Navigate forward in history

### Editor

- **Toggle Preview**: Switch between edit and preview modes
- **Toggle Sidebar**: Show/hide the sidebar
- **Focus Mode**: Enter distraction-free writing
- **Insert Date**: Add current date/time
- **Insert Template**: Insert a template

### Formatting

- **Bold**: Make text bold
- **Italic**: Make text italic
- **Strikethrough**: Strike through text
- **Code**: Format as code
- **Link**: Insert a link
- **Heading**: Apply heading format

### Organization

- **Add Tag**: Tag the current note
- **Move to Folder**: Move note to a folder
- **Archive Note**: Archive current note
- **Pin Note**: Pin note to top of list

### View

- **Toggle Theme**: Switch between light/dark
- **Zoom In**: Increase font size
- **Zoom Out**: Decrease font size
- **Reset Zoom**: Reset to default size

## Tips

### Quick Access

- Commands are fuzzy-searched
- Recent commands appear first
- Pin frequently used commands

### Keyboard Navigation

- `↑`/`↓` - Navigate results
- `Enter` - Execute command
- `Escape` - Close palette
- `Tab` - Cycle through categories

### Custom Commands

1. Open Settings
2. Go to Command Palette
3. Add custom commands
4. Define keyboard shortcuts

## Customizing Commands

### Changing Shortcuts

1. Open Settings (`Cmd+,` or `Ctrl+,`)
2. Navigate to Keyboard Shortcuts
3. Find the command
4. Click the current shortcut
5. Press your new shortcut
6. Save

### Creating Custom Commands

Advanced users can create custom commands

1. Open Settings
2. Go to Commands
3. Click "Add Custom Command"
4. Define the command name
5. Add the action(s)
6. Set a shortcut (optional)
