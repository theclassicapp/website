# अपनी पहली नोट बनाना

अपनी पहली नोट बनाकर Classic के साथ आरंभ करें। यह सरल और सहज है।

## नई नोट बनाना

### साइडबार का उपयोग करके

1. साइडबार में **+** बटन पर क्लिक करें
2. एक नई नोट बन जाती है
3. अपनी सामग्री टाइप करना शुरू करें

### कीबोर्ड शॉर्टकट का उपयोग करके

| प्लेटफ़ॉर्म | शॉर्टकट |
| -------- | -------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### कमांड पैलेट का उपयोग करके

1. कमांड पैलेट खोलें (`Cmd+P` या `Ctrl+P`)
2. "New Note" टाइप करें
3. Enter दबाएं

## अपनी नोट लिखना

### बुनियादी संपादन

बस टाइप करना शुरू करें! Classic आप जैसे-जैसे लिखते हैं, स्वतः सहेजता है।

### शीर्षक जोड़ना

आपकी नोट की पहली पंक्ति शीर्षक बन जाती है। सर्वोत्तम परिणामों के लिए हेडिंग का उपयोग करें:

```markdown
# मेरी पहली नोट

यह मेरी नोट की सामग्री है।
```

### टेक्स्ट फ़ॉर्मेट करना

अपने टेक्स्ट को फ़ॉर्मेट करने के लिए Markdown का उपयोग करें:

- **बोल्ड**: `**टेक्स्ट**` या `Cmd/Ctrl+B`
- **इटैलिक**: `*टेक्स्ट*` या `Cmd/Ctrl+I`
- **स्ट्राइकथ्रू**: `~~टेक्स्ट~~`
- **कोड**: `` `कोड` ``

## सहेजना

Classic आप टाइप करते समय आपकी नोट्स को स्वतः सहेजता है। मैन्युअल रूप से सहेजने की आवश्यकता नहीं है।

### मैन्युअल सहेजना

यदि आवश्यक हो, तो आप जबरन सहेज सकते हैं:

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## अपनी नोट व्यवस्थित करना

### टैग जोड़ना

बाद में आसान खोज के लिए अपनी नोट को टैग करें:

1. `#` टाइप करें उसके बाद अपना टैग नाम
2. या टूलबार में टैग बटन का उपयोग करें
3. एकाधिक टैग समर्थित हैं

उदाहरण:

```markdown
# मीटिंग नोट्स

Q4 प्लानिंग पर चर्चा की गई #work #meetings
```

### फ़ोल्डर में ले जाना

1. साइडबार में नोट पर राइट-क्लिक करें
2. "Move to Folder" चुनें
3. गंतव्य फ़ोल्डर चुनें या बनाएं

### महत्वपूर्ण नोट्स पिन करना

1. नोट पर राइट-क्लिक करें
2. "Pin to Top" चुनें
3. नोट आपकी सूची के शीर्ष पर दिखाई देती है

## अपनी नोट प्रीव्यू करें

देखें कि आपकी फ़ॉर्मेट की गई नोट कैसी दिखती है:

- **प्रीव्यू** बटन पर क्लिक करें
- या `Cmd/Ctrl+Shift+P` का उपयोग करें
- संपादन और प्रीव्यू मोड के बीच टॉगल करें

## अगले कदम

अब जब आपने अपनी पहली नोट बना ली है:

- [Markdown](/help/markdown) के बारे में और जानें
- [टैग्स](/help/tags) से व्यवस्थित करें
- [डेली नोट्स](/help/daily-notes) सेट करें
- [कीबोर्ड शॉर्टकट](/help/keyboard-shortcuts) का अन्वेषण करें

## टिप्स

### नोट शीर्षक

- शीर्षक वर्णनात्मक रखें
- पहली पंक्ति = नोट शीर्षक
- स्पष्टता के लिए H1 (`#`) का उपयोग करें

### ऑटो-सेव

- काम खोने की कभी चिंता न करें
- बदलाव स्वचालित रूप से सहेजे जाते हैं
- सहेजने की स्थिति के लिए स्टेटस बार देखें

### त्वरित कैप्चर

कुछ जल्दी नोट करना है?

- त्वरित कैप्चर का उपयोग करें (`Cmd/Ctrl+Shift+N`)
- तुरंत एक नोट बनाता है
- क्षणिक विचारों के लिए उत्तम
