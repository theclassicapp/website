# स्लैश कमांड

स्लैश कमांड कीबोर्ड छोड़े बिना फ़ॉर्मेटिंग और विशेष सामग्री जल्दी से डालने का एक शक्तिशाली तरीका है। कमांड पैलेट खोलने के लिए एडिटर में `/` टाइप करें।

## उपलब्ध कमांड

### हेडिंग्स

| कमांड | परिणाम |
| ------- | ------ |
| `/h1` | लेवल 1 हेडिंग |
| `/h2` | लेवल 2 हेडिंग |
| `/h3` | लेवल 3 हेडिंग |
| `/h4` | लेवल 4 हेडिंग |
| `/h5` | लेवल 5 हेडिंग |
| `/h6` | लेवल 6 हेडिंग |

### टेक्स्ट फ़ॉर्मेटिंग

| कमांड | परिणाम |
| ------- | ------ |
| `/bold` | **बोल्ड टेक्स्ट** |
| `/italic` | *इटैलिक टेक्स्ट* |
| `/strikethrough` | ~~स्ट्राइकथ्रू~~ |
| `/code` | इनलाइन कोड ब्लॉक |
| `/quote` | ब्लॉकक्वोट |

### सूचियां

| कमांड | परिणाम |
| ------- | ------ |
| `/bullet` | • बुलेट सूची |
| `/numbered` | 1. नंबर्ड सूची |
| `/todo` | - [ ] टूडू आइटम |
| `/check` | - [x] पूर्ण आइटम |

### सामग्री डालें

| कमांड | परिणाम |
| ------- | ------ |
| `/image` | URL या फ़ाइल से चित्र डालें |
| `/link` | हाइपरलिंक डालें |
| `/table` | टेबल डालें |
| `/hr` | हॉरिजॉन्टल रूल |

### विशेष ब्लॉक्स

| ------- | ------ |

## स्लैश कमांड का उपयोग करना

1. एडिटर में `/` टाइप करें
2. सभी उपलब्ध कमांड के साथ एक मेनू दिखाई देगा
3. फ़िल्टर करने के लिए टाइप करना शुरू करें
4. डालने के लिए `Enter` दबाएं या क्लिक करें

## प्रो टिप्स

- **त्वरित पहुंच**: बस `/` टाइप करें और कमांड नाम टाइप करना शुरू करें
- **फज़ी खोज**: आपको सटीक कमांड नाम टाइप करने की आवश्यकता नहीं है - Classic सबसे करीबी मिलान ढूंढेगा
- **कीबोर्ड नेविगेशन**: मेनू को नेविगेट करने के लिए एरो कीज़ का उपयोग करें
- **हाल के कमांड**: अक्सर उपयोग किए जाने वाले कमांड सूची के शीर्ष पर दिखाई देते हैं
