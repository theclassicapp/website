# बैकअप और पुनर्स्थापन

Classic के अंतर्निहित बैकअप सिस्टम के साथ अपनी नोट्स को सुरक्षित रखें। अपना काम कभी न खोएं।

## स्वचालित बैकअप

### यह कैसे काम करता है

Classic नियमित अंतराल पर आपकी नोट्स के बैकअप स्वचालित रूप से बनाता है:

- **डिफ़ॉल्ट आवृत्ति**: हर 30 मिनट
- **बैकअप स्थान**: आपके नोट्स के समान फ़ोल्डर
- **प्रतिधारण**: अंतिम 10 बैकअप रखता है
- **प्रारूप**: कंप्रेस्ड आर्काइव (`.cbak`)

### ऑटो बैकअप कॉन्फ़िगर करना

1. सेटिंग्स खोलें
2. **Backup & Restore** पर नेविगेट करें
3. निम्नलिखित कॉन्फ़िगर करें:

| सेटिंग | विकल्प | डिफ़ॉल्ट |
| ------- | ------- | ------- |
| Enable Auto Backup | On/Off | On |
| Backup Frequency | 15min / 30min / 1hr / 4hr | 30min |
| Keep Backups | 5 / 10 / 20 / 50 | 10 |
| Backup Location | Custom path | Notes folder |

## मैन्युअल बैकअप

### अभी बैकअप बनाएं

1. कमांड पैलेट खोलें (`Cmd/Ctrl+P`)
2. "Backup" टाइप करें
3. **Create Backup Now** चुनें
4. स्थान चुनें (वैकल्पिक)
5. बैकअप बनाया जाता है

### बैकअप फ़ाइल प्रारूप

बैकअप `.cbak` फ़ाइलों के रूप में संग्रहीत होते हैं जिनमें शामिल हैं:

- सभी नोट्स (Markdown फ़ाइलें)
- अटैचमेंट्स
- सेटिंग्स
- मेटाडेटा (टैग्स, लिंक्स, आदि)

## बैकअप से पुनर्स्थापन

### पुनर्स्थापन प्रक्रिया

1. सेटिंग्स खोलें
2. **Backup & Restore** पर जाएं
3. **Restore from Backup** पर क्लिक करें
4. बैकअप फ़ाइल चुनें
5. पुनर्स्थापन विकल्प चुनें
6. पुनर्स्थापन की पुष्टि करें

### पुनर्स्थापन विकल्प

| विकल्प | विवरण |
| ------ | ----------- |
| **Replace All** | वर्तमान नोट्स को पूरी तरह से बैकअप से बदलें |
| **Merge** | बैकअप को मौजूदा नोट्स के साथ मिलाएं |
| **Preview** | पुनर्स्थापित करने से पहले बैकअप सामग्री देखें |

### क्या पुनर्स्थापित होता है

- ✅ सभी नोट्स और फ़ोल्डर्स
- ✅ अटैचमेंट्स
- ✅ टैग्स और मेटाडेटा
- ✅ नोट इतिहास
- ⚠️ सेटिंग्स (वैकल्पिक)
- ❌ एप्लिकेशन प्राथमिकताएं

## बैकअप स्थान

### डिफ़ॉल्ट स्थान

```
~/Classic/backups/
├── backup-2024-01-15-10-30.cbak
├── backup-2024-01-15-11-00.cbak
└── backup-2024-01-15-11-30.cbak
```

### कस्टम स्थान

1. सेटिंग्स खोलें
2. Backup & Restore पर जाएं
3. "Change Backup Location" पर क्लिक करें
4. अपना पसंदीदा फ़ोल्डर चुनें

### क्लाउड बैकअप

बैकअप स्थान को क्लाउड-सिंक्ड फ़ोल्डर की ओर इंगित करें:

- **iCloud**: `~/Library/Mobile Documents/com~apple~CloudDocs/Classic Backups`
- **Dropbox**: `~/Dropbox/Classic Backups`
- **Google Drive**: `~/Google Drive/Classic Backups`
- **OneDrive**: `~/OneDrive/Classic Backups`

## सर्वोत्तम अभ्यास

### बैकअप शेड्यूल

- **ऑटो बैकअप**: 30-मिनट के अंतराल पर सक्षम रखें
- **मैन्युअल बैकअप**: बड़े बदलावों से पहले बनाएं
- **ऑफसाइट बैकअप**: नियमित रूप से बैकअप को बाहरी स्टोरेज पर कॉपी करें

### बैकअप सत्यापन

समय-समय पर अपने बैकअप सत्यापित करें:

1. एक परीक्षण पुनर्स्थापन बनाएं
2. नोट सामग्री जांचें
3. अटैचमेंट्स सत्यापित करें
4. मेटाडेटा की पुष्टि करें

### बैकअप रोटेशन

- हाल के बैकअप स्थानीय रूप से रखें
- पुराने बैकअप को बाहरी स्टोरेज में संग्रहित करें
- स्थान बचाने के लिए बहुत पुराने बैकअप हटाएं

## समस्या निवारण

### बैकअप विफल

**संभावित कारण**

- अपर्याप्त डिस्क स्पेस
- अनुमति समस्याएं
- फ़ाइल उपयोग में है

**समाधान**

1. उपलब्ध डिस्क स्पेस जांचें
2. फ़ोल्डर अनुमतियां सत्यापित करें
3. अन्य एप्लिकेशन बंद करें
4. एक भिन्न स्थान आज़माएं

### पुनर्स्थापन विफल

**संभावित कारण**

- दूषित बैकअप फ़ाइल
- असंगत संस्करण
- अपर्याप्त स्पेस

**समाधान**

1. एक पुराना बैकअप आज़माएं
2. बैकअप फ़ाइल अखंडता जांचें
3. डिस्क स्पेस खाली करें
4. सहायता से संपर्क करें

### गायब बैकअप

**संभावित कारण**

- ऑटो बैकअप अक्षम
- बैकअप स्थान बदला गया
- बैकअप हटा दिए गए

**समाधान**

1. बैकअप सेटिंग्स जांचें
2. बैकअप स्थान सत्यापित करें
3. ऑटो बैकअप सक्षम करें
4. अभी मैन्युअल बैकअप बनाएं

## उन्नत विकल्प

### एन्क्रिप्टेड बैकअप

अतिरिक्त सुरक्षा के लिए:

1. सेटिंग्स खोलें
2. Backup & Restore पर जाएं
3. "Encrypt Backups" सक्षम करें
4. एक पासवर्ड सेट करें
5. इस पासवर्ड को याद रखें - यह पुनर्प्राप्त नहीं किया जा सकता

### वृद्धिशील बैकअप

वृद्धिशील बैकअप के साथ स्थान बचाएं:

- केवल पिछले बैकअप के बाद के बदलाव संग्रहीत किए जाते हैं
- पूर्ण पुनर्स्थापन पूर्ण स्थिति का पुनर्निर्माण करता है
- बैकअप समय और आकार कम होता है

### बैकअप बहिष्करण

बैकअप से विशिष्ट सामग्री बहिष्कृत करें:

1. सेटिंग्स खोलें
2. Backup & Restore पर जाएं
3. "Exclusions" पर क्लिक करें
4. बहिष्कृत करने के लिए फ़ोल्डर्स या फ़ाइलें जोड़ें
