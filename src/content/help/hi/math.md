# गणित और समीकरण

Classic LaTeX सिंटैक्स का उपयोग करके गणितीय नोटेशन का समर्थन करता है। अपनी नोट्स में सुंदर समीकरण लिखें।

## गणित सक्षम करना

गणित समर्थन Classic में डिफ़ॉल्ट रूप से सक्षम है।

### इनलाइन गणित

इनलाइन समीकरणों के लिए सिंगल डॉलर साइन का उपयोग करें:

```markdown
सूत्र $E = mc^2$ ने भौतिकी को हमेशा के लिए बदल दिया।
```

यह इस रूप में रेंडर होता है: सूत्र E=mc² ने भौतिकी को हमेशा के लिए बदल दिया।

### ब्लॉक गणित

सेंटर्ड ब्लॉक समीकरणों के लिए डबल डॉलर साइन का उपयोग करें:

```markdown
$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

## सामान्य नोटेशन

### बुनियादी ऑपरेशन

| सिंटैक्स | परिणाम |
| ------ | ------ |
| `x + y` | x + y |
| `x - y` | x - y |
| `x \times y` | x × y |
| `x \div y` | x ÷ y |
| `x / y` | x/y |
| `x^2` | x² |
| `x_{n}` | xₙ |

### भिन्न

```markdown
$\frac{numerator}{denominator}$
```

उदाहरण:

- `$\frac{1}{2}$` → ½
- `$\frac{a+b}{c+d}$` → (a+b)/(c+d)

### वर्गमूल

```markdown
$\sqrt{x}$
$\sqrt[n]{x}$
```

उदाहरण:

- `$\sqrt{2}$` → √2
- `$\sqrt[3]{8}$` → ³√8

### सबस्क्रिप्ट और सुपरस्क्रिप्ट

```markdown
$x_1$          → सबस्क्रिप्ट
$x^2$          → सुपरस्क्रिप्ट
$x_1^2$        → दोनों
$x_{10}$       → मल्टी-चार सबस्क्रिप्ट
$x^{n+1}$      → मल्टी-चार सुपरस्क्रिप्ट
```

## उन्नत गणित

### समष्टियां

```markdown
$$
\sum_{i=1}^{n} x_i
$$
```

### इंटीग्रल

```markdown
$$
\int_{a}^{b} f(x) \, dx
$$
```

### सीमाएं

```markdown
$$
\lim_{x \to \infty} f(x)
$$
```

### मैट्रिक्स

```markdown
$$
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$
```

### पीसवाइस फंक्शन्स

```markdown
$$
f(x) = \begin{cases}
  1 & \text{if } x > 0 \\
  0 & \text{if } x = 0 \\
  -1 & \text{if } x < 0
\end{cases}
$$
```

## टिप्स

### स्पेसिंग

कस्टम स्पेसिंग के लिए `\,` `\;` `\:` `\quad` `\qquad` का उपयोग करें:

```markdown
$a \, b$     → छोटा स्पेस
$a \quad b$  → बड़ा स्पेस
```

### गणित में टेक्स्ट

समीकरणों के अंदर नियमित टेक्स्ट के लिए `\text{}` का उपयोग करें:

```markdown
$\text{when } x > 0$
```

### रंग

समीकरणों के हिस्सों में रंग जोड़ें:

```markdown
$\textcolor{red}{important}$
```

### अलाइनमेंट

मल्टी-लाइन समीकरणों के लिए `align` एनवायरनमेंट का उपयोग करें:

```markdown
$$
\begin{align}
a &= b + c \\
d &= e + f
\end{align}
$$
```
