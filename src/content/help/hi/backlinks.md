# बैकलिंक्स

बैकलिंक्स आपकी नोट्स के बीच कनेक्शन दिखाते हैं, संबंधित विचारों का एक नेटवर्क बनाते हैं। ऐसे रिश्तों की खोज करें जिनके बारे में आपको पता नहीं था।

## बैकलिंक्स क्या हैं?

बैकलिंक्स दिखाते हैं कि कौन सी नोट्स वर्तमान नोट से लिंक करती हैं। ये आपके नॉलेज ग्राफ़ में कनेक्शन हैं।

### लाभ

- **कनेक्शन खोजें**: स्वचालित रूप से संबंधित नोट्स खोजें
- **संदर्भ**: समझें कि विचार कैसे संबंधित हैं
- **नेविगेशन**: कनेक्टेड नोट्स के बीच जाएं
- **नॉलेज ग्राफ़**: अपनी सोच की कल्पना करें

## बैकलिंक्स कैसे काम करते हैं

जब आप किसी अन्य नोट का लिंक बनाते हैं, तो Classic स्वचालित रूप से एक बैकलिंक बनाता है।

```markdown
नोट A में:
विवरण के लिए [[Project Planning]] देखें।

परिणाम:
- नोट A "Project Planning" से लिंक करती है
- "Project Planning" नोट A के लिए एक बैकलिंक दिखाती है
```

## बैकलिंक्स देखना

### बैकलिंक्स पैनल

सभी संदर्भ देखने के लिए बैकलिंक्स पैनल खोलें:

1. कोई भी नोट खोलें
2. साइडबार में बैकलिंक्स आइकन पर क्लिक करें
3. इससे लिंक करने वाली सभी नोट्स देखें

### इनलाइन बैकलिंक्स

बैकलिंक्स प्रत्येक नोट के नीचे दिखाई देते हैं:

- स्रोत नोट नाम दिखाता है
- संदर्भ स्निपेट प्रदर्शित करता है
- नेविगेट करने के लिए क्लिक करें

## लिंक्स बनाना

### विकिलिंक सिंटैक्स

नोट्स से लिंक करने के लिए डबल ब्रैकेट का उपयोग करें:

```markdown
[[Note Name]]
[[Another Note]]
```

### डिस्प्ले टेक्स्ट

लिंक टेक्स्ट कस्टमाइज़ करें:

```markdown
[[Project Alpha|the alpha project]]
[[Meeting Notes|yesterday's meeting]]
```

### हेडिंग्स से लिंक

विशिष्ट अनुभागों से लिंक:

```markdown
[[Note Name#Heading]]
[[Note Name#Section|jump to section]]
```

## बैकलिंक प्रकार

### डायरेक्ट लिंक्स

आपके द्वारा बनाए गए स्पष्ट विकिलिंक्स:

```markdown
संबंधित: [[Similar Note]]
```

### अनलिंक्ड मेंशंस

ऐसी नोट्स जो शीर्षक का उल्लेख करती हैं लेकिन लिंक नहीं हैं:

```markdown
Note mentions "Project Alpha" but isn't linked
```

Classic इन्हें लिंक्स में बदलने का सुझाव देता है।

## बैकलिंक्स पैनल

### विशेषताएं

- **फ़िल्टर**: बैकलिंक्स के माध्यम से खोजें
- **सॉर्ट**: तिथि, नाम या प्रासंगिकता द्वारा
- **प्रीव्यू**: सामग्री प्रीव्यू करने के लिए होवर करें
- **संदर्भ**: आसपास का टेक्स्ट देखें

### क्रियाएं

- लिंक की गई नोट खोलने के लिए क्लिक करें
- विकल्पों के लिए राइट-क्लिक करें
- अनलिंक्ड मेंशंस को बदलें

## उपयोग के मामले

### नॉलेज मैनेजमेंट

एक व्यक्तिगत विकी बनाएं:

```markdown
# Machine Learning

Related to [[Artificial Intelligence]]
Builds on [[Statistics]]
Used in [[Natural Language Processing]]
```

### प्रोजेक्ट डॉक्यूमेंटेशन

संबंधित दस्तावेज़ कनेक्ट करें:

```markdown
# API Design

References [[Database Schema]]
Depends on [[Authentication Flow]]
See also [[Frontend Integration]]
```

### अनुसंधान

अवधारणाओं और स्रोतों को लिंक करें:

```markdown
# Research Notes

Based on [[Source Paper A]]
Contradicts [[Source Paper B]]
Supports [[Hypothesis 1]]
```

### मीटिंग नोट्स

मीटिंग्स को प्रोजेक्ट्स से कनेक्ट करें:

```markdown
# Meeting: Q4 Planning

Project: [[Project Alpha]]
Previous: [[Q3 Review]]
Next: [[Q4 Kickoff]]
```

## ग्राफ़ व्यू

अपने बैकलिंक्स को एक नेटवर्क के रूप में विज़ुअलाइज़ करें:

1. ग्राफ़ व्यू खोलें (`Cmd/Ctrl+G`)
2. सभी नोट्स नोड्स के रूप में देखें
3. लाइन्स कनेक्शन दिखाती हैं
4. नेविगेट करने के लिए नोड्स पर क्लिक करें

### ग्राफ़ नियंत्रण

- **ज़ूम**: ज़ूम करने के लिए स्क्रॉल करें
- **पैन**: ले जाने के लिए ड्रैग करें
- **फ़िल्टर**: टैग द्वारा दिखाएं/छिपाएं
- **फोकस**: एक नोट के कनेक्शन हाइलाइट करें

## टिप्स

### उदारतापूर्वक लिंक करें

जब भी आप किसी अन्य अवधारणा का संदर्भ दें तो लिंक बनाएं:

```markdown
✅ अच्छा:
[[Q4 Roadmap]] और [[Budget Allocation]] पर चर्चा की गई।

❌ बचें:
Q4 Roadmap और Budget Allocation पर चर्चा की गई।
```

### सुसंगत नामों का उपयोग करें

विश्वसनीय लिंकिंग के लिए नोट शीर्षकों से सटीक मिलान करें:

```markdown
✅ [[Meeting Notes]]
❌ [[meeting notes]] (केस मायने रखता है)
```

### बैकलिंक्स की समीक्षा करें

समय-समय पर बैकलिंक्स जांचें:

- भूले हुए कनेक्शन खोजें
- मर्ज करने के लिए नोट्स खोजें
- नॉलेज गैप्स पहचानें

### लिंक प्रीव्यू

खोले बिना प्रीव्यू करने के लिए लिंक्स पर होवर करें:

- पहली कुछ पंक्तियां देखें
- जांचें कि यह सही नोट है
- त्वरित संदर्भ

## कीबोर्ड शॉर्टकट

| क्रिया | शॉर्टकट |
|--------|----------|
| बैकलिंक्स खोलें | `Cmd/Ctrl+Shift+B` |
| लिंक बनाएं | `[[` (ऑटो-कम्पलीट) |
| ग्राफ़ व्यू खोलें | `Cmd/Ctrl+G` |
| लिंक्स नेविगेट करें | लिंक्स के माध्यम से `Tab` |

## उन्नत विशेषताएं

### एम्बेडिंग

अन्य नोट्स से सामग्री एम्बेड करें:

```markdown
![[Note Name]]
![[Note Name#Section]]
```

### ट्रांसक्लूज़न

इनलाइन सामग्री शामिल करें:

```markdown
## Related Content
![[Shared Template#Header]]
```

### लिंक कलरिंग

लिंक्स निम्न के आधार पर रंग बदलते हैं:

- **ब्लू**: अनविज़िटेड लिंक
- **पर्पल**: विज़िटेड लिंक
- **रेड**: टूटी हुई लिंक (नोट मौजूद नहीं है)

## समस्या निवारण

### गायब बैकलिंक्स

- लिंक सिंटैक्स `[[Note Name]]` जांचें
- सत्यापित करें कि नोट शीर्षक सटीक मेल खाता है
- बैकलिंक्स पैनल रीफ्रेश करें

### टूटी हुई लिंक्स

- नोट का नाम बदला जा सकता है
- नोट हटाई जा सकती है
- सही नोट खोजने के लिए खोज का उपयोग करें

### डुप्लिकेट लिंक्स

- समान नोट्स समेकित करें
- नाम बदले गए नोट्स के लिए रीडायरेक्ट का उपयोग करें
- शीर्षक विविधताओं की जांच करें
