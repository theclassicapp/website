# कमांड पैलेट

कमांड पैलेट Classic में हर क्रिया का आपका प्रवेश द्वार है। एक सरल कीबोर्ड शॉर्टकट दबाएं और किसी भी विशेषता तक तुरंत पहुंचें।

## कमांड पैलेट खोलना

| प्लेटफ़ॉर्म | शॉर्टकट |
| -------- | -------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## उपयोग कैसे करें

1. कमांड पैलेट खोलें (`Cmd+P` या `Ctrl+P`)
2. कमांड खोजने के लिए टाइप करें
3. नेविगेट करने के लिए एरो कीज़ का उपयोग करें
4. निष्पादित करने के लिए `Enter` दबाएं

## कमांड श्रेणियां

### फ़ाइल ऑपरेशन

- **New Note**: एक नई नोट बनाएं
- **New Daily Note**: आज की डेली नोट बनाएं/खोलें
- **Open Note**: मौजूदा नोट्स खोजें और खोलें
- **Save Note**: वर्तमान नोट सहेजें
- **Delete Note**: वर्तमान नोट हटाएं
- **Rename Note**: वर्तमान नोट का नाम बदलें

### नेविगेशन

- **Go to Note**: किसी भी नोट पर जाएं
- **Go to Daily Note**: एक विशिष्ट तिथि खोलें
- **Go to Tag**: टैग द्वारा नोट्स ब्राउज़ करें
- **Go Back**: इतिहास में वापस नेविगेट करें
- **Go Forward**: इतिहास में आगे नेविगेट करें

### संपादक

- **Toggle Preview**: संपादन और प्रीव्यू मोड के बीच स्विच करें
- **Toggle Sidebar**: साइडबार दिखाएं/छिपाएं
- **Focus Mode**: विकर्षण-मुक्त लेखन में प्रवेश करें
- **Insert Date**: वर्तमान तिथि/समय जोड़ें
- **Insert Template**: एक टेम्प्लेट डालें

### फ़ॉर्मेटिंग

- **Bold**: टेक्स्ट को बोल्ड बनाएं
- **Italic**: टेक्स्ट को इटैलिक बनाएं
- **Strikethrough**: टेक्स्ट पर स्ट्राइक करें
- **Code**: कोड के रूप में फ़ॉर्मेट करें
- **Link**: एक लिंक डालें
- **Heading**: हेडिंग फ़ॉर्मेट लागू करें

### संगठन

- **Add Tag**: वर्तमान नोट को टैग करें
- **Move to Folder**: नोट को फ़ोल्डर में ले जाएं
- **Archive Note**: वर्तमान नोट संग्रहित करें
- **Pin Note**: नोट को सूची के शीर्ष पर पिन करें

### दृश्य

- **Toggle Theme**: लाइट/डार्क के बीच स्विच करें
- **Zoom In**: फ़ॉन्ट आकार बढ़ाएं
- **Zoom Out**: फ़ॉन्ट आकार घटाएं
- **Reset Zoom**: डिफ़ॉल्ट आकार पर रीसेट करें

## टिप्स

### त्वरित पहुंच

- कमांड फज़ी-खोजे जाते हैं
- हाल के कमांड पहले दिखाई देते हैं
- बार-बार उपयोग किए जाने वाले कमांड पिन करें

### कीबोर्ड नेविगेशन

- `↑`/`↓` - परिणाम नेविगेट करें
- `Enter` - कमांड निष्पादित करें
- `Escape` - पैलेट बंद करें
- `Tab` - श्रेणियों के माध्यम से साइकिल करें

### कस्टम कमांड

1. सेटिंग्स खोलें
2. Command Palette पर जाएं
3. कस्टम कमांड जोड़ें
4. कीबोर्ड शॉर्टकट परिभाषित करें

## बार-बार उपयोग किए जाने वाले कमांड

यहां कुछ कमांड हैं जो आप अक्सर उपयोग करेंगे:

| कमांड | यह क्या करता है |
| ------- | ------------- |
| `New Note` | रिक्त नोट बनाएं |
| `Open Note` | सभी नोट्स खोजें |
| `New Daily Note` | आज का जर्नल |
| `Toggle Preview` | प्रीव्यू बनाम संपादन |
| `Add Tag` | वर्तमान नोट टैग करें |
| `Quick Capture` | ग्लोबल कैप्चर |

## कमांड कस्टमाइज़ करना

### शॉर्टकट बदलना

1. सेटिंग्स खोलें (`Cmd+,` या `Ctrl+,`)
2. Keyboard Shortcuts पर नेविगेट करें
3. कमांड खोजें
4. वर्तमान शॉर्टकट पर क्लिक करें
5. अपना नया शॉर्टकट दबाएं
6. सहेजें

### कस्टम कमांड बनाना

उन्नत उपयोगकर्ता कस्टम कमांड बना सकते हैं:

1. सेटिंग्स खोलें
2. Commands पर जाएं
3. "Add Custom Command" पर क्लिक करें
4. कमांड नाम परिभाषित करें
5. क्रिया(यों) जोड़ें
6. एक शॉर्टकट सेट करें (वैकल्पिक)
