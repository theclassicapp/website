# कीबोर्ड शॉर्टकट

Classic के कीबोर्ड शॉर्टकट के साथ तेज़ी से काम करें। सभी शॉर्टकट सेटिंग्स में कस्टमाइज़ करने योग्य हैं।

## ग्लोबल शॉर्टकट

ये ऐप में कहीं भी काम करते हैं।

### सामान्य

| शॉर्टकट (Windows/Linux) | शॉर्टकट (macOS) | क्रिया |
| ------------------------ | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | नई नोट |
| `Ctrl+P` | `Cmd+P` | कमांड पैलेट |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | ग्लोबल त्वरित कैप्चर |
| `Ctrl+S` | `Cmd+S` | नोट सहेजें |
| `Ctrl+F` | `Cmd+F` | खोज |
| `Ctrl+,` | `Cmd+,` | सेटिंग्स |

### नेविगेशन

| शॉर्टकट (Windows/Linux) | शॉर्टकट (macOS) | क्रिया |
| ------------------------ | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | पहली नोट पर जाएं |
| `Ctrl+2` | `Cmd+2` | दूसरी नोट पर जाएं |
| `Ctrl+Left` | `Cmd+Left` | वापस नेविगेट करें |
| `Ctrl+Right` | `Cmd+Right` | आगे नेविगेट करें |
| `Ctrl+Up` | `Cmd+Up` | पैरेंट नोट पर नेविगेट करें |
| `Ctrl+Down` | `Cmd+Down` | पहली सबनोट विस्तार/फोकस करें |

### संपादक

| शॉर्टकट (Windows/Linux) | शॉर्टकट (macOS) | क्रिया |
| ------------------------ | ----------------- | ------ |
| `Ctrl+B` | `Cmd+B` | बोल्ड |
| `Ctrl+I` | `Cmd+I` | इटैलिक |
| `Ctrl+K` | `Cmd+K` | लिंक डालें |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | स्ट्राइकथ्रू |
| `Ctrl+`` | `Cmd+`` | इनलाइन कोड |
| `Ctrl+Shift+C` | `Cmd+Shift+C` | कोड ब्लॉक |
| `Ctrl+/` | `Cmd+/` | स्लैश कमांड |
| `Tab` | `Tab` | इंडेंट |
| `Shift+Tab` | `Shift+Tab` | अनइंडेंट |

### विंडो प्रबंधन

| शॉर्टकट (Windows/Linux) | शॉर्टकट (macOS) | क्रिया |
| ------------------------ | ----------------- | ------ |
| `Ctrl+W` | `Cmd+W` | वर्तमान नोट बंद करें |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | सभी नोट्स बंद करें |
| `F11` | `F11` | फुलस्क्रीन टॉगल करें |
| `Ctrl+M` | `Cmd+M` | साइडबार टॉगल करें |

## त्वरित कैप्चर

| शॉर्टकट (Windows/Linux) | शॉर्टकट (macOS) | क्रिया |
| ------------------------ | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | त्वरित कैप्चर खोलें |
| `Escape` | `Escape` | त्वरित कैप्चर बंद करें |
| `Ctrl+Enter` | `Cmd+Enter` | सहेजें और बंद करें |

## शॉर्टकट कस्टमाइज़ करना

1. **Settings** खोलें (`Ctrl+,` या `Cmd+,`)
2. **Keyboard Shortcuts** पर नेविगेट करें
3. बदलने के लिए किसी भी शॉर्टकट पर क्लिक करें
4. अपना नया की संयोजन दबाएं
5. **Save** पर क्लिक करें

## टिप्स

- अधिकांश शॉर्टकट कस्टमाइज़ करने योग्य हैं
- सिस्टम शॉर्टकट के साथ टकराव हाइलाइट किए जाएंगे
- प्रत्येक शॉर्टकट के बगल के रीसेट बटन पर क्लिक करके डिफ़ॉल्ट पर रीसेट करें
