# टैग्स

टैग्स Classic में अपनी नोट्स को व्यवस्थित और वर्गीकृत करने का एक शक्तिशाली तरीका है। संबंधित सामग्री तुरंत खोजें।

## टैग्स बनाना

### इनलाइन टैग्स

अपनी नोट सामग्री में सीधे टैग्स जोड़ें:

```markdown
# मीटिंग नोट्स

Q4 रोडमैप पर चर्चा की गई #work #meetings #planning
```

### टैग बटन का उपयोग करके

1. टूलबार में टैग आइकन पर क्लिक करें
2. अपना टैग नाम टाइप करें
3. Enter दबाएं

### कीबोर्ड शॉर्टकट

| प्लेटफ़ॉर्म | शॉर्टकट |
|----------|----------|
| **macOS** | `Cmd+T` |
| **Windows/Linux** | `Ctrl+T` |

## टैग सिंटैक्स

### बुनियादी टैग्स

```markdown
#project #work #important
```

### नेस्टेड टैग्स

`/` के साथ टैग पदानुक्रम बनाएं:

```markdown
#work/meetings
#work/projects
#personal/hobbies/reading
```

### मल्टी-वर्ड टैग्स

हाइफ़न या अंडरस्कोर का उपयोग करें:

```markdown
#project-alpha
#meeting_notes
#q4-planning
```

## टैग्स प्रबंधित करना

### सभी टैग्स देखना

1. साइडबार खोलें
2. "Tags" अनुभाग खोजें
3. अपने सभी टैग्स देखें

### टैग क्लाउड

अपने सबसे अधिक उपयोग किए जाने वाले टैग्स की कल्पना करें:

- बड़े टैग्स = अधिक नोट्स
- फ़िल्टर करने के लिए किसी भी टैग पर क्लिक करें
- प्रति टैग नोट गणना देखें

### टैग्स का नाम बदलना

1. एक टैग पर राइट-क्लिक करें
2. "Rename" चुनें
3. नया नाम दर्ज करें
4. सभी नोट्स स्वचालित रूप से अपडेट होती हैं

### टैग्स हटाना

1. एक टैग पर राइट-क्लिक करें
2. "Delete" चुनें
3. हटाने की पुष्टि करें
4. टैग सभी नोट्स से हटा दिया जाता है

## टैग द्वारा खोज

### टैग पैनल

नोट्स फ़िल्टर करने के लिए साइडबार में एक टैग पर क्लिक करें।

### खोज सिंटैक्स

खोज में `tag:` का उपयोग करें:

```
tag:work
tag:work/meetings
```

### टैग्स संयोजित करना

एकाधिक टैग्स वाली नोट्स खोजें:

```
tag:work tag:urgent
```

### टैग्स बहिष्कृत करना

टैग के बिना नोट्स खोजें:

```
-tag:archived
```

## टैग सर्वोत्तम अभ्यास

### सरल रखें

- प्रति नोट 3-5 टैग्स का उपयोग करें
- नामकरण में सुसंगत रहें
- टैग प्रसार से बचें

### पदानुक्रम का उपयोग करें

संबंधित टैग्स व्यवस्थित करें:

```
#work
  #work/meetings
  #work/projects
  #work/deadlines

#personal
  #personal/health
  #personal/finance
```

### परंपराएं स्थापित करें

अपने लिए टैग मानक परिभाषित करें:

- **स्थिति**: `#todo` `#in-progress` `#done`
- **प्राथमिकता**: `#urgent` `#high` `#low`
- **प्रकार**: `#meeting` `#reference` `#idea`

## टैग सुझाव

जैसे ही आप `#` टाइप करते हैं, Classic मौजूदा टैग्स सुझाता है:

- हाल के टैग्स पहले दिखाई देते हैं
- फज़ी मैचिंग समर्थित
- सुझाव स्वीकार करने के लिए Tab दबाएं

## टैग आंकड़े

सेटिंग्स में टैग उपयोग देखें:

- सबसे अधिक उपयोग किए जाने वाले टैग्स
- अप्रयुक्त टैग्स
- समय के साथ टैग रुझान

## उपयोग के मामले

### प्रोजेक्ट प्रबंधन

```markdown
# Project Kickoff #project-alpha #meeting #2024-01-15

Attendees: Team Alpha
Status: #todo
```

### नॉलेज बेस

```markdown
# JavaScript Closures #programming #javascript #reference #advanced

Closures are functions that have access to...
```

### डेली नोट्स

```markdown
# 2024-01-15

- Morning standup #work #meetings
- Code review #work #development
- Gym session #personal #health
```

### रीडिंग नोट्स

```markdown
# Atomic Habits Notes #reading #books #productivity

Key takeaways from the book...
```

## कीबोर्ड शॉर्टकट

| क्रिया | शॉर्टकट |
|--------|----------|
| टैग जोड़ें | `Cmd/Ctrl+T` |
| टैग्स खोजें | `Cmd/Ctrl+F`, फिर `tag:` टाइप करें |
| टैग पैनल दिखाएं | `Cmd/Ctrl+Shift+T` |

## टिप्स

### सुसंगत नामकरण

एक परंपरा का पालन करें:

```
✅ #meeting-notes
✅ #meeting_notes
❌ #Meeting Notes (स्पेस अनुमत नहीं)
❌ #meetingNotes (असंगत)
```

### अति-टैगिंग से बचें

बहुत अधिक टैग्स अप्रबंधनीय हो जाते हैं:

```
✅ #work #urgent #project-alpha
❌ #work #urgent #important #today #monday #office #desk #computer
```

### आवधिक सफाई

नियमित रूप से टैग्स की समीक्षा और समेकन करें:

- समान टैग्स मर्ज करें
- अप्रयुक्त टैग्स हटाएं
- परंपराएं अपडेट करें

## उन्नत विशेषताएं

### टैग-आधारित स्मार्ट फ़ोल्डर्स

टैग्स के आधार पर स्वचालित रूप से आबाद होने वाले फ़ोल्डर बनाएं:

1. नया फ़ोल्डर बनाएं
2. फ़िल्टर सेट करें: `tag:project`
3. सभी टैग की गई नोट्स दिखाई देती हैं

### YAML में टैग

नोट फ्रंटमैटर में टैग्स जोड़ें:

```yaml
---
tags:
  - work
  - meetings
  - project-alpha
---
```

### टैग उपनाम

सेटिंग्स में टैग उपनाम सेट करें:

- `#todo` = `#task` = `#action`
- समान टैग्स समेकित करें
