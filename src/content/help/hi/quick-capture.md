# त्वरित कैप्चर

अपने वर्कफ़्लो को बाधित किए बिना विचारों को तुरंत कैप्चर करें। त्वरित कैप्चर आपके सिस्टम के कहीं से भी एक न्यूनतम नोट-टेकिंग विंडो खोलता है।

## ग्लोबल हॉटकी

त्वरित कैप्चर खोलने के लिए निर्दिष्ट हॉटकी दबाएं:

| प्लेटफ़ॉर्म | डिफ़ॉल्ट शॉर्टकट |
| -------- | ---------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

आप सेटिंग्स में इस शॉर्टकट को कस्टमाइज़ कर सकते हैं।

## त्वरित कैप्चर का उपयोग

1. ग्लोबल हॉटकी दबाएं (आपके सिस्टम पर कहीं भी)
2. पॉपअप विंडो में अपनी नोट टाइप करें
3. सहेजने के लिए `Enter` दबाएं
4. रद्द करने के लिए `Escape` दबाएं

## विशेषताएं

### न्यूनतम इंटरफ़ेस

- छोटी, केंद्रित विंडो
- Markdown समर्थन
- इनपुट पर ऑटो-फोकस
- कीबोर्ड-फर्स्ट डिज़ाइन

### त्वरित क्रियाएं

- `# ` + स्पेस = हेडिंग
- `#inbox` = Inbox में जोड़ें
- `#today` = आज की नोट में जोड़ें
- `#tagname` के साथ टैग्स

### डिफ़ॉल्ट गंतव्य

त्वरित कैप्चर नोट्स डिफ़ॉल्ट रूप से आपके **Inbox** में जाती हैं। आप इसे बदल सकते हैं:

1. सेटिंग्स खोलें
2. Quick Capture पर जाएं
3. डिफ़ॉल्ट गंतव्य चुनें

## टिप्स

### कैप्चर वर्कफ़्लो

1. **कैप्चर** - विचार को जल्दी नोट करें
2. **प्रोसेस** - Inbox की दैनिक समीक्षा करें
3. **व्यवस्थित करें** - उपयुक्त फ़ोल्डर/नोट में ले जाएं

### सर्वोत्तम अभ्यास

- संक्षिप्त रखें - प्रति कैप्चर एक विचार
- संदर्भ के लिए टैग्स का उपयोग करें
- फ़ॉर्मेटिंग की चिंता न करें - बाद में परिष्कृत करें
- कैप्चर्स की नियमित रूप से समीक्षा और प्रोसेस करें

### एकीकरण

त्वरित कैप्चर इनके साथ बहुत अच्छा काम करता है:

- **डेली नोट्स**: प्रत्येक दिन कैप्चर्स की समीक्षा से शुरू करें
- **Inbox**: होल्डिंग एरिया के रूप में उपयोग करें
- **टैग्स**: तुरंत संदर्भ जोड़ें

## समस्या निवारण

### हॉटकी काम नहीं कर रहा?

**macOS**

1. System Preferences > Keyboard > Shortcuts जांचें
2. टकराव की तलाश करें
3. Classic पुनरारंभ करें

**Windows**

1. Classic को Administrator के रूप में चलाएं
2. टकराव वाले शॉर्टकट की जांच करें
3. Classic पुनरारंभ करें

### विंडो दिखाई नहीं दे रही?

- जांचें कि Classic चल रहा है या नहीं
- शॉर्टकट सत्यापित करें
- फोकस समस्याओं की जांच करें
- Classic पुनरारंभ करें

### कैप्चर्स सहेजी नहीं जा रहीं?

- डिफ़ॉल्ट गंतव्य जांचें
- लेखन अनुमतियां सत्यापित करें
- डिस्क स्पेस की जांच करें
- Classic पुनरारंभ करें
