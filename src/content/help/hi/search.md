# खोज

Classic की शक्तिशाली खोज के साथ कोई भी नोट तुरंत खोजें। फुल-टेक्स्ट खोज, फ़िल्टर और ऑपरेटर आपको वही खोजने में मदद करते हैं जो आपको चाहिए।

## बुनियादी खोज

### खोज खोलना

| प्लेटफ़ॉर्म | शॉर्टकट |
|----------|----------|
| **macOS** | `Cmd+F` |
| **Windows/Linux** | `Ctrl+F` |

या साइडबार में खोज बार पर क्लिक करें।

### यह कैसे काम करता है

1. खोज खोलें
2. टाइप करना शुरू करें
3. परिणाम तुरंत दिखाई देते हैं
4. नोट खोलने के लिए क्लिक करें

## खोज विशेषताएं

### फुल-टेक्स्ट खोज

Classic इनमें खोजता है:

- नोट शीर्षक
- नोट सामग्री
- टैग्स
- फ़ाइल नाम

### रियल-टाइम परिणाम

जैसे ही आप टाइप करते हैं परिणाम अपडेट होते हैं:

- Enter दबाने की आवश्यकता नहीं
- तत्काल प्रतिक्रिया
- फज़ी मैचिंग सक्षम

### खोज हाइलाइटिंग

पाए गए शब्द परिणामों में हाइलाइट किए जाते हैं:

- मैचों के आसपास संदर्भ देखें
- प्रासंगिक नोट्स जल्दी पहचानें

## खोज ऑपरेटर

### सटीक मिलान

सटीक वाक्यांशों के लिए उद्धरण चिह्नों का उपयोग करें:

```
"project planning"
"meeting notes"
```

### टैग खोज

विशिष्ट टैग्स वाली नोट्स खोजें:

```
tag:work
tag:meetings
tag:urgent
```

### शब्द बहिष्कृत करें

बहिष्कृत करने के लिए `-` का उपयोग करें:

```
project -archived
meeting -cancelled
```

### OR खोज

किसी भी शब्द वाली नोट्स खोजें:

```
project OR initiative
```

### तिथि सीमा

तिथि द्वारा खोजें:

```
created:2024-01-01..2024-12-31
modified:last-week
```

### फ़ाइल प्रकार

एक्सटेंशन द्वारा फ़िल्टर करें:

```
ext:md
ext:txt
```

## उन्नत खोज

### ऑपरेटर संयोजित करना

जटिल क्वेरी बनाएं:

```
tag:work -tag:archived created:last-week
"meeting" tag:project-alpha
```

### पाथ खोज

फ़ोल्डर्स के भीतर खोजें:

```
path:Projects/
path:Archive/
```

### शीर्षक खोज

केवल शीर्षक खोजें:

```
title:Roadmap
title:Meeting
```

### सामग्री खोज

केवल सामग्री खोजें (शीर्षक बहिष्कृत):

```
content:important
content:deadline
```

## खोज फ़िल्टर

### टैग द्वारा

1. अपनी खोज टाइप करें
2. फ़िल्टर करने के लिए एक टैग पर क्लिक करें
3. परिणाम टैग की गई नोट्स तक संकीर्ण होते हैं

### तिथि द्वारा

संशोधन तिथि द्वारा फ़िल्टर करें:

- आज
- इस सप्ताह
- इस महीने
- कस्टम सीमा

### फ़ोल्डर द्वारा

खोज को एक फ़ोल्डर तक सीमित करें:

1. एक फ़ोल्डर पर राइट-क्लिक करें
2. "Search in Folder" चुनें
3. परिणाम उस फ़ोल्डर तक सीमित

## खोज परिणाम

### सॉर्टिंग

इनके अनुसार परिणाम सॉर्ट करें:

- **प्रासंगिकता** (डिफ़ॉल्ट)
- **संशोधन तिथि** (नवीनतम पहले)
- **निर्माण तिथि** (नवीनतम पहले)
- **शीर्षक** (वर्णानुक्रमिक)

### परिणाम प्रीव्यू

प्रीव्यू करने के लिए परिणामों पर होवर करें:

- पहली कुछ पंक्तियां दिखाई देती हैं
- मिलान शब्द हाइलाइट किए जाते हैं
- टैग्स प्रदर्शित

### संदर्भ

देखें कि मैच कहाँ हुआ:

- आसपास का टेक्स्ट दिखाता है
- मैच को हाइलाइट करता है
- स्थान पर जाने के लिए क्लिक करें

## खोज इतिहास

### हाल की खोजें

पिछली खोजों तक पहुंचें:

1. खोज बार में क्लिक करें
2. हाल की खोजें देखें
3. पुनः चलाने के लिए क्लिक करें

### इतिहास साफ़ करना

1. सेटिंग्स खोलें
2. Privacy पर जाएं
3. "Clear Search History" पर क्लिक करें

## टिप्स

### विशिष्ट शब्दों का उपयोग करें

अधिक विशिष्ट = बेहतर परिणाम:

```
✅ "Q4 marketing budget"
❌ budget
```

### टैग्स के साथ संयोजित करें

परिणामों को जल्दी संकीर्ण करें:

```
project tag:urgent
```

### बहिष्करण का उपयोग करें

शोर फ़िल्टर करें:

```
meeting -tag:archived
```

### बार-बार की जाने वाली खोजें सहेजें

सहेजी गई खोजें बनाएं:

1. अपनी खोज दर्ज करें
2. "Save Search" पर क्लिक करें
3. इसे नाम दें
4. साइडबार से एक्सेस करें

## कीबोर्ड नेविगेशन

| क्रिया | शॉर्टकट |
|--------|----------|
| खोज खोलें | `Cmd/Ctrl+F` |
| अगला परिणाम | `Enter` या `Down` |
| पिछला परिणाम | `Shift+Enter` या `Up` |
| खोज बंद करें | `Escape` |

## खोज सेटिंग्स

### सेटिंग्स में कस्टमाइज़ करें

| सेटिंग | विकल्प |
|---------|---------|
| केस सेंसिटिव | चालू/बंद |
| शीर्षक खोजें | चालू/बंद |
| सामग्री खोजें | चालू/बंद |
| टैग्स खोजें | चालू/बंद |
| संग्रहीत शामिल करें | चालू/बंद |

## उपयोग के मामले

### पुरानी नोट्स खोजना

```
"meeting" date:last-year
```

### प्रोजेक्ट अनुसंधान

```
path:Projects/ tag:research
```

### एक्शन आइटम्स

```
"TODO" OR "FIXME" -tag:done
```

### हाल का काम

```
modified:this-week
```

## समस्या निवारण

### कोई परिणाम नहीं मिला

- वर्तनी जांचें
- फ़िल्टर हटाएं
- व्यापक शब्दों का प्रयास करें
- जांचें कि नोट संग्रहीत है या नहीं

### धीमी खोज

- बड़े नोट संग्रह में अधिक समय लग सकता है
- केस सेंसिटिविटी अक्षम करें
- खोज स्कोप कम करें

### गायब नोट्स

- नोट किसी अन्य फ़ोल्डर में हो सकती है
- जांचें कि टैग सही है या नहीं
- सत्यापित करें कि नोट संग्रहीत नहीं है
