# आयात और निर्यात

अन्य ऐप्स से अपनी नोट्स Classic में लाएं, या साझा करने या बैकअप के लिए निर्यात करें। Classic निर्बाध डेटा माइग्रेशन के लिए कई प्रारूपों का समर्थन करता है।

## नोट्स आयात करना

### समर्थित प्रारूप

| प्रारूप | एक्सटेंशन | नोट्स |
| ------ | --------- | ----- |
| Markdown | `.md` | ✅ पूर्ण समर्थन |
| Plain Text | `.txt` | ✅ नोट के रूप में आयातित |
| JSON | `.json` | ✅ Classic प्रारूप |
| HTML | `.html` | ⚠️ Markdown में परिवर्तित |
| Evernote | `.enex` | ⚠️ प्रायोगिक |
| Notion | Notion export | ⚠️ प्रायोगिक |
| Obsidian | Markdown export | ✅ पूर्ण समर्थन |

### आयात प्रक्रिया

1. **File** → **Import** खोलें (या `Cmd/Ctrl+I`)
2. आयात करने के लिए अपनी फ़ाइलें चुनें
3. गंतव्य फ़ोल्डर चुनें
4. आयात प्रीव्यू की समीक्षा करें
5. आयात की पुष्टि करें

### क्लिपबोर्ड से आयात

किसी भी स्रोत से त्वरित आयात:

1. क्लिपबोर्ड पर टेक्स्ट कॉपी करें
2. कमांड पैलेट खोलें
3. "Import from Clipboard" टाइप करें
4. नई नोट स्वचालित रूप से बनाई जाती है

### बल्क आयात

एक साथ कई फ़ाइलें आयात करें:

1. **File** → **Import** खोलें
2. **Import Folder** चुनें
3. नोट्स वाला फ़ोल्डर चुनें
4. सभी समर्थित फ़ाइलें आयात की जाती हैं

## नोट्स निर्यात करना

### निर्यात प्रारूप

| प्रारूप | उपयोग केस |
| ------ | ------- |
| **Markdown** | साझाकरण, बैकअप, अन्य ऐप्स |
| **HTML** | वेब प्रकाशन, प्रीव्यू |
| **PDF** | संग्रह, प्रिंटिंग |
| **Plain Text** | सरल साझाकरण |
| **JSON** | API एकीकरण |

### एकल नोट निर्यात

1. नोट खोलें
2. **File** → **Export** पर क्लिक करें
3. प्रारूप चुनें
4. स्थान चुनें
5. सहेजें

### एकाधिक नोट्स निर्यात

1. साइडबार में नोट्स चुनें (Cmd/Ctrl+click)
2. राइट-क्लिक → **Export Selected**
3. प्रारूप चुनें
4. स्थान चुनें
5. सहेजें

### सभी नोट्स निर्यात

1. **File** → **Export All** खोलें
2. प्रारूप चुनें
3. स्थान चुनें
4. निर्यात पूरा होने की प्रतीक्षा करें

## माइग्रेशन गाइड

### Obsidian से

Classic Obsidian वॉल्ट को सीधे आयात कर सकता है:

1. **File** → **Import** खोलें
2. **Obsidian Vault** चुनें
3. अपने वॉल्ट फ़ोल्डर पर नेविगेट करें
4. आयात करने के लिए चुनें

**क्या स्थानांतरित होता है**

- ✅ नोट्स और फ़ोल्डर्स
- ✅ टैग्स
- ✅ लिंक्स (बैकलिंक्स में परिवर्तित)
- ✅ मेटाडेटा
- ⚠️ प्लगइन्स (समर्थित नहीं)

### Notion से

1. Notion से Markdown के रूप में निर्यात करें
2. Classic में आयात करें
3. फ़ॉर्मेटिंग की समीक्षा करें
4. किसी भी टूटी हुई लिंक को ठीक करें

**Notion माइग्रेशन के लिए टिप्स**

- Notion के "Export as Markdown" का उपयोग करें
- चेकबॉक्स फ़ॉर्मेटिंग जांचें
- डेटाबेस संदर्भ सत्यापित करें
- लिंक्ड डेटाबेस मैन्युअल रूप से पुनः बनाएं

### Evernote से

1. Evernote से HTML के रूप में निर्यात करें
2. Classic के HTML आयात का उपयोग करें
3. परिवर्तित सामग्री की समीक्षा करें

**Evernote माइग्रेशन के लिए टिप्स**

- नोटबुक्स फ़ोल्डर्स बन जाते हैं
- टैग्स संरक्षित रहते हैं
- अटैचमेंट्स आयात किए जाते हैं
- फ़ॉर्मेटिंग जांचें

## सर्वोत्तम अभ्यास

### आयात करने से पहले

- मौजूदा नोट्स का बैकअप लें
- आयात प्रीव्यू की समीक्षा करें
- डुप्लिकेट्स की जांच करें
- फ़ॉर्मेटिंग सत्यापित करें

### आयात करने के बाद

- आयात की गई नोट्स की समीक्षा करें
- किसी भी फ़ॉर्मेटिंग समस्या को ठीक करें
- आंतरिक लिंक्स अपडेट करें
- आवश्यकतानुसार टैग्स जोड़ें

### नियमित निर्यात

- मासिक बैकअप निर्यात करें
- विभिन्न उद्देश्यों के लिए विभिन्न प्रारूपों का उपयोग करें
- तिथि के अनुसार निर्यात व्यवस्थित रखें
- ऑफसाइट बैकअप के लिए क्लाउड में संग्रहीत करें

## समस्या निवारण

### आयात विफल

**सामान्य कारण**

- असमर्थित प्रारूप
- दूषित फ़ाइल
- अनुमति समस्याएं
- मेमोरी सीमाएं

**समाधान**

1. एक भिन्न प्रारूप आज़माएं
2. फ़ाइल अखंडता जांचें
3. रीड अनुमतियां सत्यापित करें
4. एक समय में कम फ़ाइलें आयात करें

### फ़ॉर्मेटिंग खो गई

**सामान्य कारण**

- जटिल फ़ॉर्मेटिंग
- असमर्थित सिंटैक्स
- रूपांतरण सीमाएं

**समाधान**

1. स्रोत ऐप से Markdown निर्यात का उपयोग करें
2. आयात से पहले फ़ॉर्मेटिंग सरल बनाएं
3. आयात के बाद मैन्युअल रूप से ठीक करें

### अटैचमेंट्स गायब

**सामान्य कारण**

- आयात में अटैचमेंट्स शामिल नहीं
- गलत अटैचमेंट पाथ
- आकार सीमाएं

**समाधान**

1. अटैचमेंट्स के साथ पुनः आयात करें
2. अटैचमेंट फ़ोल्डर जांचें
3. अटैचमेंट्स अलग से आयात करें
