# सिस्टम ट्रे

Classic त्वरित पहुंच के लिए आपके सिस्टम ट्रे में रहता है। अपने वर्कफ़्लो को बाधित किए बिना विचारों को तुरंत कैप्चर करें।

## सिस्टम ट्रे क्या है?

सिस्टम ट्रे (Windows/Linux) या मेनू बार (macOS) वह जगह है जहां Classic बैकग्राउंड में चलता है, हमेशा त्वरित कैप्चर के लिए तैयार।

### लाभ

- **हमेशा उपलब्ध**: एक क्लिक दूर
- **त्वरित कैप्चर**: विचारों को तुरंत नोट करें
- **बैकग्राउंड में चलना**: ऐप खुला रखने की आवश्यकता नहीं
- **सूचनाएं**: टास्क्स की याद दिलाएं

## सिस्टम ट्रे सक्षम करना

### macOS

1. Classic खोलें
2. Settings → General पर जाएं
3. "Show in Menu Bar" सक्षम करें
4. Classic आइकन मेनू बार में दिखाई देता है

### Windows

1. Classic खोलें
2. Settings → General पर जाएं
3. "Show in System Tray" सक्षम करें
4. Classic आइकन सिस्टम ट्रे में दिखाई देता है

### Linux

1. Classic खोलें
2. Settings → General पर जाएं
3. "System Tray Icon" सक्षम करें
4. डेस्कटॉप एनवायरनमेंट पर निर्भर करता है

## त्वरित कैप्चर

### त्वरित कैप्चर खोलना

सिस्टम ट्रे में Classic आइकन पर क्लिक करें:

- **macOS**: मेनू बार आइकन
- **Windows**: सिस्टम ट्रे आइकन
- **Linux**: सिस्टम ट्रे आइकन

### त्वरित कैप्चर का उपयोग

1. सिस्टम ट्रे आइकन पर क्लिक करें
2. त्वरित कैप्चर विंडो खुलती है
3. अपनी नोट टाइप करें
4. सहेजने के लिए Enter दबाएं
5. विंडो स्वचालित रूप से बंद हो जाती है

### कीबोर्ड शॉर्टकट

कहीं से भी त्वरित कैप्चर खोलें:

| प्लेटफ़ॉर्म | शॉर्टकट |
|----------|----------|
| **macOS** | `Cmd+Shift+N` (ग्लोबल) |
| **Windows** | `Ctrl+Shift+N` (ग्लोबल) |
| **Linux** | `Ctrl+Shift+N` (ग्लोबल) |

## सिस्टम ट्रे मेनू

विकल्पों के लिए ट्रे आइकन पर राइट-क्लिक करें

### मेनू विकल्प

| विकल्प | क्रिया |
|--------|--------|
| **New Note** | त्वरित कैप्चर खोलें |
| **New Daily Note** | आज की डेली नोट खोलें |
| **Search** | खोज विंडो खोलें |
| **Open Classic** | मुख्य विंडो खोलें |
| **Settings** | सेटिंग्स खोलें |
| **Quit** | Classic से बाहर निकलें |

## ट्रे सेटिंग्स

### सेटिंग्स में कॉन्फ़िगर करें

| सेटिंग | विवरण |
|---------|-------------|
| **Show Icon** | ट्रे आइकन दृश्यता टॉगल करें |
| **Start Minimized** | केवल ट्रे में Classic प्रारंभ करें |
| **Close to Tray** | विंडो बंद होने पर चलता रहें |
| **Notifications** | सूचनाएं सक्षम/अक्षम करें |

### लॉगिन पर प्रारंभ करें

Classic को स्वचालित रूप से प्रारंभ करें:

1. सेटिंग्स खोलें
2. General पर जाएं
3. "Start at Login" सक्षम करें
4. Classic स्टार्टअप पर लॉन्च होता है

## सूचनाएं

### सूचनाएं सक्षम करें

1. सेटिंग्स खोलें
2. Notifications पर जाएं
3. "Enable Notifications" टॉगल करें

### सूचना प्रकार

- **अनुस्मारक**: निर्धारित अनुस्मारक
- **डेली नोट**: डेली नोट प्रॉम्प्ट्स
- **अपडेट**: ऐप अपडेट उपलब्ध

## विंडो व्यवहार

### बंद करने का व्यवहार

चुनें कि विंडो बंद करने पर क्या होता है:

- **Minimize to Tray**: चलता रहें, विंडो छिपाएं
- **Quit**: एप्लिकेशन से बाहर निकलें
- **Ask**: हर बार पूछें

### न्यूनतम करने का व्यवहार

- **Minimize to Tray**: सिस्टम ट्रे में छिपाएं
- **Minimize to Taskbar**: सामान्य रूप से न्यूनतम करें

## टिप्स

### तेज़ कैप्चर

सबसे तेज़ कैप्चर के लिए ग्लोबल शॉर्टकट का उपयोग करें:

1. कहीं भी `Cmd/Ctrl+Shift+N` दबाएं
2. अपना विचार टाइप करें
3. Enter दबाएं
4. अपना काम जारी रखें

### त्वरित पहुंच

ट्रे आइकन पर क्लिक करके देखें:

- हाल की नोट्स
- त्वरित क्रियाएं
- आज की डेली नोट

### बैकग्राउंड में चलना

इनके लिए Classic चलाते रहें:

- तेज़ स्टार्टअप
- त्वरित कैप्चर
- सूचनाएं
- ऑटो-बैकअप

## समस्या निवारण

### ट्रे आइकन गायब

**macOS**

1. जांचें कि आइकन छिपा है या नहीं
2. Control Center देखें (macOS Big Sur+)
3. Classic पुनरारंभ करें

**Windows**

1. छिपे हुए आइकन जांचें (टास्कबार में तीर)
2. Classic आइकन को दृश्य क्षेत्र में खींचें
3. Classic पुनरारंभ करें

**Linux**

1. डेस्कटॉप एनवायरनमेंट समर्थन जांचें
2. यदि आवश्यक हो तो सिस्टम ट्रे एक्सटेंशन इंस्टॉल करें
3. Classic पुनरारंभ करें

### ग्लोबल शॉर्टकट काम नहीं कर रहा

- अन्य ऐप्स के साथ टकराव जांचें
- सेटिंग्स में शॉर्टकट पुनः सक्षम करें
- Classic पुनरारंभ करें
- OS अनुमतियां जांचें

### सूचनाएं नहीं दिख रहीं

**macOS**

1. System Preferences → Notifications
2. Classic खोजें
3. सूचनाएं सक्षम करें

**Windows**

1. Settings → System → Notifications
2. Classic सूचनाएं सक्षम करें

**Linux**

1. नोटिफिकेशन डेमन जांचें
2. यदि आवश्यक हो तो libnotify इंस्टॉल करें

### उच्च मेमोरी/CPU उपयोग

यदि ट्रे आइकन बहुत अधिक संसाधनों का उपयोग करता है:

1. सेटिंग्स में एनिमेशन अक्षम करें
2. सूचना आवृत्ति कम करें
3. ऐप अपडेट की जांच करें
