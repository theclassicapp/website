# सेटिंग्स

Classic को वैसे ही काम करने के लिए कस्टमाइज़ करें जैसे आप चाहते हैं। उपस्थिति से लेकर संपादक व्यवहार तक, सब कुछ कॉन्फ़िगर करने योग्य है।

## सेटिंग्स खोलना

### कीबोर्ड शॉर्टकट

| प्लेटफ़ॉर्म | शॉर्टकट |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### मेन्यू के माध्यम से

- **macOS**: Classic → Settings
- **Windows**: File → Settings
- **Linux**: Edit → Settings

### कमांड पैलेट के माध्यम से

1. कमांड पैलेट खोलें (`Cmd+P` या `Ctrl+P`)
2. "Settings" टाइप करें
3. Enter दबाएं

## सेटिंग्स श्रेणियां

### सामान्य

बुनियादी एप्लिकेशन सेटिंग्स:

| सेटिंग | विवरण |
|---------|-------------|
| **Language** | इंटरफ़ेस भाषा |
| **Theme** | Light, Dark, या System |
| **Start at Login** | स्टार्टअप पर लॉन्च |
| **Check for Updates** | ऑटो-अपडेट व्यवहार |

### उपस्थिति

लुक और फील कस्टमाइज़ करें:

| सेटिंग | विकल्प |
|---------|---------|
| **Theme** | Light / Dark / System |
| **Accent Color** | Blue, Purple, Green, आदि |
| **Editor Font** | फ़ॉन्ट परिवार और आकार |
| **UI Font** | इंटरफ़ेस फ़ॉन्ट |
| **Monospace Font** | कोड फ़ॉन्ट |
| **Sidebar Width** | 200-400px |
| **Line Height** | 1.0-2.5 |

### संपादक

लेखन व्यवहार कॉन्फ़िगर करें:

| सेटिंग | विवरण |
|---------|-------------|
| **Spell Check** | स्पेल चेकिंग सक्षम/अक्षम |
| **Auto-save** | स्वचालित सहेजने का अंतराल |
| **Word Wrap** | वर्ड रैप सक्षम/अक्षम |
| **Line Numbers** | लाइन नंबर दिखाएं/छिपाएं |
| **Tab Size** | 2, 4, या 8 स्पेस |
| **Auto-correct** | स्मार्ट क्वोट्स, ऑटो-रिप्लेस |

### संस्करण इतिहास

संस्करण ट्रैकिंग नियंत्रित करें:

| सेटिंग | विकल्प | डिफ़ॉल्ट |
|---------|---------|---------|
| **Enable History** | On/Off | On |
| **Retention Period** | 30-180 दिन | 90 दिन |
| **Max Versions** | 50-500 | 100 |

### बैकअप और पुनर्स्थापन

डेटा सुरक्षा सेटिंग्स:

| सेटिंग | विवरण |
|---------|-------------|
| **Auto Backup** | स्वचालित बैकअप सक्षम करें |
| **Backup Frequency** | 15min / 30min / 1hr / 4hr |
| **Backup Location** | कस्टम पाथ |
| **Keep Backups** | 5 / 10 / 20 / 50 |
| **Encrypt Backups** | पासवर्ड सुरक्षा |

### कीबोर्ड शॉर्टकट

शॉर्टकट देखें और कस्टमाइज़ करें:

- सभी उपलब्ध शॉर्टकट देखें
- किसी भी शॉर्टकट को कस्टमाइज़ करें
- डिफ़ॉल्ट पर रीसेट करें
- शॉर्टकट आयात/निर्यात करें

### आयात और निर्यात

डेटा माइग्रेशन सेटिंग्स:

- अन्य ऐप्स से आयात करें
- निर्यात प्रारूप विकल्प
- बल्क ऑपरेशन

### गोपनीयता

गोपनीयता और सुरक्षा सेटिंग्स:

| सेटिंग | विवरण |
|---------|-------------|

| **Crash Reports** | क्रैश रिपोर्ट भेजें |
| **Clear History** | खोज/इतिहास साफ़ करें |

### सूचनाएं

सूचनाएं नियंत्रित करें:

| सेटिंग | विवरण |
|---------|---------|
| **Enable Notifications** | On/Off |
| **Daily Note Reminder** | याद दिलाने का समय |
| **Sound** | सूचना ध्वनि |

### उन्नत

पावर यूज़र सेटिंग्स:

| सेटिंग | विवरण |
|---------|-------------|
| **Custom CSS** | कस्टम स्टाइल जोड़ें |
| **Debug Mode** | डिबग लॉगिंग सक्षम करें |
| **Experimental Features** | बीटा विशेषताएं |
| **Data Location** | नोट्स फ़ोल्डर बदलें |

## सेटिंग्स खोजना

सेटिंग नहीं मिल रही? सर्च बार का उपयोग करें:

1. सेटिंग्स खोलें
2. सर्च बॉक्स में टाइप करें
3. मिलान सेटिंग्स दिखाई देती हैं
4. उस सेटिंग पर जाने के लिए क्लिक करें

## सेटिंग्स रीसेट करना

### सभी सेटिंग्स रीसेट करें

डिफ़ॉल्ट पर वापस लौटें:

1. सेटिंग्स खोलें
2. नीचे स्क्रॉल करें
3. "Reset All Settings" पर क्लिक करें
4. क्रिया की पुष्टि करें

### श्रेणी रीसेट करें

विशिष्ट श्रेणी रीसेट करें:

1. श्रेणी पर जाएं
2. "Reset to Defaults" पर क्लिक करें
3. केवल वह श्रेणी रीसेट होती है

## सेटिंग्स निर्यात/आयात

### सेटिंग्स निर्यात

अपना कॉन्फ़िगरेशन सहेजें:

1. सेटिंग्स खोलें
2. "Export Settings" पर क्लिक करें
3. स्थान चुनें
4. JSON फ़ाइल के रूप में सहेजें

### सेटिंग्स आयात

सहेजे गए कॉन्फ़िगरेशन को लोड करें:

1. सेटिंग्स खोलें
2. "Import Settings" पर क्लिक करें
3. JSON फ़ाइल चुनें
4. आयात की पुष्टि करें

## सेटिंग्स स्थान

सेटिंग्स स्थानीय रूप से संग्रहीत हैं:

| प्लेटफ़ॉर्म | स्थान |
|----------|----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## सेटिंग्स सिंक (वैकल्पिक)

साइन इन होने पर, सेटिंग्स डिवाइसों में सिंक हो सकती हैं:

- **Sync**: Settings → Sync में सक्षम करें
- **क्या सिंक होता है**: थीम, फ़ॉन्ट्स, शॉर्टकट
- **क्या सिंक नहीं होता**: स्थानीय पाथ, एन्क्रिप्शन कीज़

## सामान्य कस्टमाइज़ेशन

### फ़ॉन्ट आकार बढ़ाएं

1. Settings → Appearance
2. Editor Font Size → बढ़ाएं
3. बदलाव तुरंत लागू होते हैं

### थीम बदलें

1. Settings → Appearance
2. Theme → Light/Dark/System चुनें
3. ऐप तुरंत अपडेट होता है

### ऑटो-बैकअप सक्षम करें

1. Settings → Backup & Restore
2. "Auto Backup" सक्षम करें
3. आवृत्ति और स्थान सेट करें

### शॉर्टकट कस्टमाइज़ करें

1. Settings → Keyboard Shortcuts
2. क्रिया खोजें
3. वर्तमान शॉर्टकट पर क्लिक करें
4. नई कीज़ दबाएं
5. सहेजें

## टिप्स

### त्वरित पहुंच

- सेटिंग्स जल्दी खोलने के लिए `Cmd/Ctrl+,` का उपयोग करें
- नाम से सेटिंग्स खोजें
- नेविगेट करने के लिए श्रेणियों का उपयोग करें

### बदलावों से पहले निर्यात करें

बड़े बदलावों से पहले, अपनी सेटिंग्स निर्यात करें:

- यदि कुछ गलत हो जाए तो आसानी से पुनर्स्थापित करें
- दूसरों के साथ कॉन्फ़िगरेशन साझा करें

### नई सेटिंग्स की जांच करें

अपडेट नई सेटिंग्स जोड़ सकते हैं:

- अपडेट के बाद जांचें
- नए विकल्पों का अन्वेषण करें

## समस्या निवारण

### सेटिंग्स सहेजी नहीं जा रहीं

- फ़ाइल अनुमतियां जांचें
- डिस्क स्पेस सत्यापित करें
- रीसेट करके पुनः कॉन्फ़िगर करने का प्रयास करें

### सेटिंग्स अनपेक्षित रूप से रीसेट

- फ़ाइल भ्रष्टाचार का संकेत हो सकता है
- नियमित रूप से सेटिंग्स निर्यात करें
- ऐप टकराव की जांच करें

### सेटिंग नहीं मिल रही

- सर्च बार का उपयोग करें
- विभिन्न श्रेणियां जांचें
- Advanced सेटिंग्स में हो सकती है
