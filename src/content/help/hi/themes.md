# थीम्स और उपस्थिति

Classic की उपस्थिति को अपनी शैली से मिलाने के लिए वैयक्तिकृत करें और आंखों के तनाव को कम करें।

## थीम विकल्प

### लाइट मोड

Classic का डिफ़ॉल्ट थीम विशेषताएं:

- साफ, चमकदार इंटरफ़ेस
- अच्छी रोशनी वाले वातावरण के लिए इष्टतम
- पठनीयता के लिए उच्च कंट्रास्ट

### डार्क मोड

रात के उल्लू और कम रोशनी वाले काम के लिए उत्तम:

- आंखों के तनाव को कम करता है
- OLED स्क्रीन पर कम बैटरी खपत
- अंधेरे वातावरण में आंखों पर आसान

### सिस्टम (ऑटो)

अपने ऑपरेटिंग सिस्टम के थीम का पालन करें:

- OS सेटिंग्स के आधार पर स्वचालित रूप से स्विच करता है
- मैन्युअल स्विचिंग की आवश्यकता नहीं
- अन्य ऐप्स के साथ सुसंगत

## थीम्स स्विच करना

### सेटिंग्स के माध्यम से

1. सेटिंग्स खोलें (`Cmd+,` या `Ctrl+,`)
2. **Appearance** पर नेविगेट करें
3. अपना पसंदीदा थीम चुनें

### कमांड पैलेट के माध्यम से

1. कमांड पैलेट खोलें (`Cmd+P` या `Ctrl+P`)
2. "Theme" टाइप करें
3. अपना पसंदीदा थीम चुनें

### कीबोर्ड शॉर्टकट

| प्लेटफ़ॉर्म | शॉर्टकट |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

साइकिल: Light → Dark → System

## फ़ॉन्ट सेटिंग्स

### संपादक फ़ॉन्ट

लेखन अनुभव को कस्टमाइज़ करें:

1. सेटिंग्स खोलें
2. **Appearance** → **Editor** पर जाएं
3. फ़ॉन्ट विकल्प कॉन्फ़िगर करें

| सेटिंग | विकल्प | डिफ़ॉल्ट |
| ------- | ------- | ------- |
| Font Family | सिस्टम फ़ॉन्ट्स + कस्टम | System UI |
| Font Size | 12-24px | 16px |
| Line Height | 1.0-2.5 | 1.75 |
| Letter Spacing | -2 to 4 | 0 |

### UI फ़ॉन्ट

इंटरफ़ेस फ़ॉन्ट समायोजित करें:

1. सेटिंग्स खोलें
2. **Appearance** → **Interface** पर जाएं
3. फ़ॉन्ट और आकार चुनें

### मोनोस्पेस फ़ॉन्ट

कोड ब्लॉक्स और तकनीकी सामग्री के लिए:

1. सेटिंग्स खोलें
2. **Appearance** → **Code** पर जाएं
3. मोनोस्पेस फ़ॉन्ट चुनें

## इंटरफ़ेस विकल्प

### साइडबार

साइडबार व्यवहार कस्टमाइज़ करें:

| सेटिंग | विवरण |
| ------- | ----------- |
| **Auto-hide** | उपयोग में नहीं होने पर साइडबार छिपाएं |
| **Width** | साइडबार चौड़ाई समायोजित करें (200-400px) |
| **Show Icons** | आइकन दृश्यता टॉगल करें |

### संपादक लेआउट

अपना संपादन अनुभव चुनें:

- **Split View**: साथ-साथ संपादित करें और प्रीव्यू करें
- **Preview Only**: केवल रेंडर किया गया आउटपुट देखें
- **Edit Only**: कच्चे Markdown पर ध्यान केंद्रित करें

### नोट सूची

कॉन्फ़िगर करें कि नोट्स कैसे दिखाई देती हैं:

- **List View**: कॉम्पैक्ट, प्रति नोट एक लाइन
- **Card View**: स्निपेट के साथ प्रीव्यू कार्ड्स
- **Tree View**: पदानुक्रमित फ़ोल्डर संरचना

## एक्सेंट कलर्स

### उपलब्ध रंग

अपना एक्सेंट कलर चुनें:

- Classic Blue (डिफ़ॉल्ट)
- Purple
- Green
- Orange
- Red
- Pink
- Teal

### एक्सेंट कलर बदलना

1. सेटिंग्स खोलें
2. **Appearance** → **Colors** पर जाएं
3. अपना पसंदीदा एक्सेंट चुनें

एक्सेंट कलर प्रभावित करता है:

- लिंक्स और बटन
- चयन हाइलाइट्स
- सक्रिय तत्व
- फोकस इंडिकेटर

## एक्सेसिबिलिटी

### हाई कंट्रास्ट मोड

बेहतर दृश्यता के लिए:

1. सेटिंग्स खोलें
2. **Appearance** → **Accessibility** पर जाएं
3. **High Contrast** सक्षम करें

### मोशन कम करें

एनिमेशन कम करें:

1. सेटिंग्स खोलें
2. **Appearance** → **Accessibility** पर जाएं
3. **Reduced Motion** सक्षम करें

### फ़ॉन्ट समायोजन

- न्यूनतम फ़ॉन्ट आकार बढ़ाएं
- डिस्लेक्सिया-फ्रेंडली फ़ॉन्ट्स का उपयोग करें
- कंट्रास्ट अनुपात समायोजित करें

## कस्टम CSS (उन्नत)

पूर्ण नियंत्रण चाहने वाले पावर यूज़र्स के लिए:

1. सेटिंग्स खोलें
2. **Appearance** → **Advanced** पर जाएं
3. **Open Custom CSS** पर क्लिक करें
4. अपने कस्टमाइज़ेशन जोड़ें

### कस्टमाइज़ेशन उदाहरण

```css
/* Larger headings */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Custom link color */
a { color: #3b82f6; }

/* Wider line spacing */
.prose p { line-height: 2; }
```

## थीम सिंक्रोनाइज़ेशन

जब आप साइन इन करते हैं (वैकल्पिक), आपकी थीम प्राथमिकताएं डिवाइसों में सिंक होती हैं:

- स्वचालित थीम सिंक
- फ़ॉन्ट सेटिंग्स सिंक
- कस्टम CSS सिंक

## टिप्स

### अपने वर्कफ़्लो से मिलान

- दिन में **Light**, रात में **Dark** का उपयोग करें
- **System** मोड यह स्वचालित रूप से संभालता है
- अपनी परिवेश प्रकाश पर विचार करें

### आंखों के तनाव को कम करना

- कम रोशनी में डार्क मोड का उपयोग करें
- लंबे सत्रों के लिए फ़ॉन्ट आकार बढ़ाएं
- यदि आवश्यक हो तो हाई कंट्रास्ट सक्षम करें
- नियमित ब्रेक लें

### उत्पादकता

- थीम टॉगल शॉर्टकट सीखें
- ऑटो-स्विचिंग समय सेट करें
- सुसंगतता के लिए सिस्टम मोड का उपयोग करें
