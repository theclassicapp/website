# लोकल-फर्स्ट आर्किटेक्चर

Classic गोपनीयता को आधार के रूप में लेकर बनाया गया है। आपका डेटा हमेशा आपके डिवाइस पर रहता है।

## "लोकल-फर्स्ट" का क्या मतलब है?

लोकल-फर्स्ट का मतलब है:

1. **आपका डेटा स्थानीय रूप से संग्रहीत है** आपके कंप्यूटर पर। आपकी सभी नोट्स, अटैचमेंट और सेटिंग्स आपके कंप्यूटर पर एक फ़ोल्डर में संग्रहीत हैं।
2. **क्लाउड की आवश्यकता नहीं** Classic इंटरनेट कनेक्शन के बिना काम करता है।
3. **आप अपने डेटा के मालिक हैं** आपके पास अपने डेटा पर पूर्ण नियंत्रण है।

## डेटा स्टोरेज

### डिफ़ॉल्ट स्थान

आपका डेटा एक समर्पित फ़ोल्डर में संग्रहीत है:

| प्लेटफ़ॉर्म | स्थान |
| -------- | -------- |
| **macOS** | `~/Library/Application Support/Classic/notes` |
| **Windows** | `%APPDATA%\Classic\notes` |
| **Linux** | `~/.config/Classic/notes` |

### डेटा प्रारूप

- नोट्स प्लेन टेक्स्ट फ़ाइलों (`.md`) के रूप में संग्रहीत हैं
- अटैचमेंट एक `attachments` सबफ़ोल्डर में संग्रहीत हैं
- सेटिंग्स `settings.json` में संग्रहीत हैं

### फ़ाइल संरचना

```
notes/
├── Inbox/
├── Daily Notes/
│   └── 2024-01-15.md
├── Projects/
│   ├── My Project.md
│   └── sub-note.md
└── attachments/
    └── image.png
settings.json
```

## गोपनीयता लाभ

### खाते की आवश्यकता नहीं

- साइन-अप की आवश्यकता नहीं
- ईमेल सत्यापन नहीं
- कोई व्यक्तिगत जानकारी एकत्र नहीं

### कोई ट्रैकिंग नहीं

- कोई एनालिटिक्स नहीं
- कोई टेलीमेट्री नहीं
- बाहरी सर्वर पर कोई उपयोग डेटा नहीं भेजा जाता

### ऑफ़लाइन सक्षम

- इंटरनेट के बिना काम करता है
- केवल तभी सिंक करता है जब आप चुनें
- आपका डेटा हमेशा सुलभ है

## डेटा सुरक्षा

### आपकी जिम्मेदारी

- **बैकअप**: सेटिंग्स में स्वचालित बैकअप सेट करें
- **निर्यात**: नियमित रूप से अपना डेटा निर्यात करें
- **अपडेट**: सुरक्षा फिक्सेस के लिए Classic को अपडेट रखें

### हमारी जिम्मेदारी

- **स्थानीय स्टोरेज**: हम आपका डेटा कभी नहीं देखते या छूते नहीं
- **कोई बैकडोर नहीं**: कोई छिपी हुई डेटा संग्रह नहीं
- **पारदर्शिता**: डेटा हैंडलिंग के बारे में खुले और ईमानदार

## सिंक (वैकल्पिक)

Classic में अंतर्निहित क्लाउड सिंक शामिल नहीं है। हालांकि, आप अपने डेटा को सिंक कर सकते हैं:

### iCloud/OneDrive/Dropbox का उपयोग करके

1. Classic सेटिंग्स खोलें
2. नोट्स फ़ोल्डर स्थान को अपने क्लाउड फ़ोल्डर में बदलें
3. Classic आपके क्लाउड प्रोवाइडर के माध्यम से सिंक करेगा

### Git का उपयोग करके

1. अपने नोट्स फ़ोल्डर में एक git रेपो इनिशियलाइज़ करें
2. स्वचालन के साथ मैन्युअल रूप से बदलाव कमिट और पुश करें

## डेटा पुनर्प्राप्ति

### बैकअप से

1. सेटिंग्स खोलें
2. Backup & Restore पर जाएं
3. एक बैकअप फ़ाइल चुनें
4. Restore पर क्लिक करें

### मैन्युअल पुनर्प्राप्ति

1. अपना नोट्स फ़ोल्डर खोजें
2. फ़ाइलों को सुरक्षित स्थान पर कॉपी करें
3. एक नई Classic इंस्टॉलेशन में आयात करें

## अक्सर पूछे जाने वाले प्रश्न

### क्या मेरा डेटा वास्तव में निजी है?

**हाँ।** आपका डेटा तब तक आपके डिवाइस से नहीं निकलता जब तक आप स्पष्ट रूप से तृतीय-पक्ष सेवा का उपयोग करके इसे सिंक नहीं करते।

### क्या मैं अपना डेटा दूसरे कंप्यूटर पर ले जा सकता हूं?

**हाँ।** बस अपने नोट्स फ़ोल्डर को नए कंप्यूटर पर कॉपी करें और सेटिंग्स में Classic को उसकी ओर इंगित करें।

### यदि मैं Classic को अनइंस्टॉल कर दूं तो क्या होगा?

**आपका डेटा बना रहता है।** Classic को अनइंस्टॉल करने से आपका नोट्स फ़ोल्डर नहीं हटता। आप डेटा खोए बिना सुरक्षित रूप से अनइंस्टॉल और पुनः इंस्टॉल कर सकते हैं।

### क्या Classic मेरे डेटा तक पहुंच सकता है?

**नहीं।** Classic डेटा संग्रह के लिए बाहरी कनेक्टिविटी के बिना एक स्थानीय एप्लिकेशन है।
