# भाषाएं और स्थानीयकरण

Classic आपकी भाषा बोलता है। अपनी पसंदीदा भाषा में ऐप का उपयोग करें और किसी भी भाषा में नोट्स लिखें।

## समर्थित भाषाएं

Classic का इंटरफ़ेस कई भाषाओं में उपलब्ध है:

| भाषा | कोड |
|----------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| और अधिक... | |

## इंटरफ़ेस भाषा बदलना

### सेटिंग्स के माध्यम से

1. सेटिंग्स खोलें (`Cmd+,` या `Ctrl+,`)
2. **General** → **Language** पर जाएं
3. अपनी पसंदीदा भाषा चुनें
4. लागू करने के लिए Classic पुनरारंभ करें

### कमांड पैलेट के माध्यम से

1. कमांड पैलेट खोलें (`Cmd+P` या `Ctrl+P`)
2. "Language" टाइप करें
3. अपनी पसंदीदा भाषा चुनें
4. बदलाव लागू करने के लिए पुनरारंभ करें

## कई भाषाओं में लिखना

Classic किसी भी भाषा में नोट्स लिखने का समर्थन करता है।

### यूनिकोड समर्थन

- पूर्ण यूनिकोड समर्थन
- RTL (राइट-टू-लेफ्ट) भाषाएं समर्थित
- CJK (चीनी, जापानी, कोरियाई) वर्ण पूरी तरह से काम करते हैं
- इमोजी समर्थन

### भाषा पहचान

Classic आपकी नोट्स की भाषा का पता लगा सकता है:

- स्पेल चेकिंग के लिए ऑटो-डिटेक्ट
- प्रति-नोट भाषा सेटिंग
- मिश्रित भाषा समर्थन

## स्पेल चेकिंग

### स्पेल चेक सक्षम करना

1. सेटिंग्स खोलें
2. **Editor** → **Spell Check** पर जाएं
3. "Spell Check" सक्षम करें

### स्पेल चेक भाषाएं

1. सेटिंग्स खोलें
2. **Editor** → **Spell Check** पर जाएं
3. "Languages" पर क्लिक करें
4. जांचने के लिए भाषाएं चुनें

### डिक्शनरी में शब्द जोड़ना

1. फ्लैग किए गए शब्द पर राइट-क्लिक करें
2. "Add to Dictionary" चुनें
3. शब्द फिर से फ्लैग नहीं होगा

## RTL समर्थन

अरबी, हिब्रू और अन्य RTL भाषाओं के लिए।

### स्वचालित RTL पहचान

Classic RTL सामग्री को स्वचालित रूप से पहचानता है:

- टेक्स्ट डायरेक्शन समायोजित होता है
- कर्सर मूवमेंट अनुकूलित होता है
- अलाइनमेंट बदलता है

### मैन्युअल RTL टॉगल

RTL या LTR डायरेक्शन जबरन लागू करें:

1. कमांड पैलेट खोलें
2. "RTL" या "LTR" टाइप करें
3. अपनी प्राथमिकता चुनें

## इनपुट मेथड्स

Classic आपके सिस्टम के इनपुट मेथड्स के साथ काम करता है।

### CJK इनपुट मेथड्स

- पिनयिन, झुयिन, आदि के साथ काम करता है
- जापानी IME का समर्थन करता है
- कोरियाई इनपुट मेथड्स समर्थित

### विशेष वर्ण

आसानी से विशेष वर्ण डालें:

- अपने सिस्टम के कैरेक्टर पिकर का उपयोग करें
- कैरेक्टर मैप्स से कॉपी/पेस्ट करें
- यूनिकोड इनपुट का उपयोग करें

## तिथि और समय प्रारूप

तिथियां आपके लोकेल के अनुसार अनुकूलित होती हैं।

### प्रारूप विकल्प

| सेटिंग | विकल्प |
|---------|---------|
| Date Format | MM/DD/YYYY, DD/MM/YYYY, YYYY-MM-DD |
| Time Format | 12-घंटे, 24-घंटे |
| First Day of Week | रविवार, सोमवार, शनिवार |

Settings → General → Date & Time में कॉन्फ़िगर करें

## नंबर प्रारूप

नंबर आपके लोकेल के अनुसार प्रदर्शित होते हैं:

- **दशमलव विभाजक**: `.` या `,`
- **हजार विभाजक**: `,` या `.` या स्पेस
- **मुद्रा प्रतीक**: क्षेत्र के आधार पर

## कीबोर्ड लेआउट

Classic किसी भी कीबोर्ड लेआउट के साथ काम करता है:

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- कस्टम लेआउट

## स्थानीयकरण सेटिंग्स

### Settings → General → Language में

| सेटिंग | विवरण |
|---------|-------------|
| **Interface Language** | ऐप UI भाषा |
| **Date Format** | तिथियां कैसे दिखती हैं |
| **Time Format** | 12 या 24 घंटे |
| **First Day of Week** | कैलेंडर प्रारंभ दिन |
| **Number Format** | दशमलव/हजार विभाजक |

## टिप्स

### मिश्रित भाषा नोट्स

एक नोट में कई भाषाओं में लिखें:

```markdown
# International Project

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### भाषा-विशिष्ट फ़ॉन्ट्स

कुछ भाषाएं विशिष्ट फ़ॉन्ट्स के साथ बेहतर दिखती हैं:

1. सेटिंग्स खोलें
2. Appearance → Fonts पर जाएं
3. विभिन्न स्क्रिप्ट्स के लिए फ़ॉन्ट्स सेट करें

### जल्दी भाषा स्विच करना

एक कीबोर्ड शॉर्टकट बनाएं:

1. Settings → Shortcuts खोलें
2. "Switch Language" खोजें
3. अपना पसंदीदा शॉर्टकट सेट करें

## समस्या निवारण

### भाषा उपलब्ध नहीं

- ऐप अपडेट की जांच करें
- भाषा विकास में हो सकती है
- GitHub पर अनुवाद योगदान करें

### वर्ण प्रदर्शित नहीं हो रहे

- आवश्यक फ़ॉन्ट्स इंस्टॉल करें
- फ़ॉन्ट सेटिंग्स जांचें
- एक भिन्न फ़ॉन्ट आज़माएं

### स्पेल चेक काम नहीं कर रहा

- सत्यापित करें कि स्पेल चेक सक्षम है
- स्पेल चेक में भाषा जोड़ें
- Classic पुनरारंभ करें

### RTL सही से काम नहीं कर रहा

- सुनिश्चित करें कि RTL पहचान चालू है
- मैन्युअल RTL टॉगल आज़माएं
- मिश्रित सामग्री समस्याओं की जांच करें

## अनुवाद योगदान

Classic का अनुवाद करने में मदद करें:

1. Classic कम्युनिटी पर जाएं
2. अनुवाद टीम से जुड़ें
3. अनुवाद जमा करें
4. स्थानीयकरण में सुधार में मदद करें
