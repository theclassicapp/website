# Schrägstrich-Befehle

Schrägstrich-Befehle sind eine leistungsstarke Möglichkeit, Formatierungen und spezielle Inhalte schnell einzufügen, ohne die Tastatur zu verlassen. Geben Sie `/` im Editor ein, um die Befehlspalette zu öffnen.

## Verfügbare Befehle

### Überschriften

| Befehl | Ergebnis |
| ------- | -------- |
| `/h1` | Überschrift Ebene 1 |
| `/h2` | Überschrift Ebene 2 |
| `/h3` | Überschrift Ebene 3 |
| `/h4` | Überschrift Ebene 4 |
| `/h5` | Überschrift Ebene 5 |
| `/h6` | Überschrift Ebene 6 |

### Textformatierung

| Befehl | Ergebnis |
| ------- | -------- |
| `/bold` | **Fetter Text** |
| `/italic` | *Kursiver Text* |
| `/strikethrough` | ~~Durchgestrichen~~ |
| `/code` | Inline-Codeblock |
| `/quote` | Blockzitat |

### Listen

| Befehl | Ergebnis |
| ------- | -------- |
| `/bullet` | • Aufzählungsliste |
| `/numbered` | 1. Nummerierte Liste |
| `/todo` | - [ ] Aufgabenelement |
| `/check` | - [x] Abgeschlossenes Element |

### Inhalte einfügen

| Befehl | Ergebnis |
| ------- | -------- |
| `/image` | Bild von URL oder Datei einfügen |
| `/link` | Hyperlink einfügen |
| `/table` | Tabelle einfügen |
| `/hr` | Horizontale Linie |
| `/code` | Codeblock mit Sprachauswahl |

### Spezialblöcke

| Befehl | Ergebnis |
| ------- | -------- |
| `/mermaid` | Mermaid-Diagrammblock |
| `/katex` | KaTeX-Mathematikblock |
| `/callout` | Hinweis-/Warnungsblock |

## Schrägstrich-Befehle verwenden

1. Geben Sie `/` im Editor ein
2. Ein Menü mit allen verfügbaren Befehlen erscheint
3. Beginnen Sie zu tippen, um Befehle zu filtern
4. Drücken Sie `Enter` oder klicken Sie zum Einfügen

## Profi-Tipps

- **Schneller Zugriff**: Geben Sie einfach `/` ein und beginnen Sie mit dem Befehlsnamen
- **Fuzzy-Suche**: Sie müssen nicht den exakten Befehlsnamen eingeben - Classic findet die beste Übereinstimmung
- **Tastaturnavigation**: Verwenden Sie die Pfeiltasten zur Navigation im Menü
- **Zuletzt verwendete Befehle**: Häufig verwendete Befehle erscheinen oben in der Liste
