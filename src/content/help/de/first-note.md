# Ihre erste Notiz erstellen

Starten Sie mit Classic, indem Sie Ihre erste Notiz erstellen. Es ist einfach und intuitiv.

## Eine neue Notiz erstellen

### Über die Seitenleiste

1. Klicken Sie auf die Schaltfläche **+** in der Seitenleiste
2. Eine neue Notiz wird erstellt
3. Beginnen Sie mit der Eingabe Ihres Inhalts

### Über Tastaturkürzel

| Plattform | Tastenkombination |
| --------- | ----------------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Über die Befehlspalette

1. Befehlspalette öffnen (`Cmd+P` oder `Ctrl+P`)
2. "Neue Notiz" eingeben
3. Eingabetaste drücken

## Ihre Notiz schreiben

### Grundlegende Bearbeitung

Fangen Sie einfach an zu tippen! Classic speichert automatisch während Sie schreiben.

### Einen Titel hinzufügen

Die erste Zeile Ihrer Notiz wird zum Titel. Verwenden Sie für beste Ergebnisse eine Überschrift:

```markdown
# Meine erste Notiz

Dies ist der Inhalt meiner Notiz.
```

### Text formatieren

Verwenden Sie Markdown, um Ihren Text zu formatieren:

- **Fett**: `**Text**` oder `Cmd/Ctrl+B`
- **Kursiv**: `*Text*` oder `Cmd/Ctrl+I`
- **Durchgestrichen**: `~~Text~~`
- **Code**: `` `Code` ``

## Speichern

Classic speichert Ihre Notizen automatisch während Sie tippen. Kein manuelles Speichern erforderlich.

### Manuelles Speichern

Falls nötig, können Sie das Speichern erzwingen:

- **macOS**: `Cmd+S`
- **Windows/Linux**: `Ctrl+S`

## Ihre Notiz organisieren

### Tags hinzufügen

Taggen Sie Ihre Notiz für späteres leichteres Finden:

1. `#` gefolgt von Ihrem Tag-Namen eingeben
2. Oder die Tag-Schaltfläche in der Symbolleiste verwenden
3. Mehrere Tags werden unterstützt

Beispiel:

```markdown
# Besprechungsnotizen

Q4-Planung besprochen #arbeit #besprechungen
```

### In einen Ordner verschieben

1. Rechtsklick auf die Notiz in der Seitenleiste
2. "In Ordner verschieben" auswählen
3. Zielordner auswählen oder erstellen

### Wichtige Notizen anheften

1. Rechtsklick auf die Notiz
2. "Oben anheften" auswählen
3. Die Notiz erscheint oben in Ihrer Liste

## Ihre Notiz anzeigen

Sehen Sie, wie Ihre formatierte Notiz aussieht:

- Klicken Sie auf die Schaltfläche **Vorschau**
- Oder verwenden Sie `Cmd/Ctrl+Shift+P`
- Zwischen Bearbeitungs- und Vorschaumodus wechseln

## Nächste Schritte

Jetzt, wo Sie Ihre erste Notiz erstellt haben:

- Erfahren Sie mehr über [Markdown](/help/markdown)
- Organisieren Sie mit [Tags](/help/tags)
- Richten Sie [Tägliche Notizen](/help/daily-notes) ein
- Entdecken Sie [Tastaturkürzel](/help/keyboard-shortcuts)

## Tipps

### Notiztitel

- Halten Sie Titel beschreibend
- Erste Zeile = Notiztitel
- Verwenden Sie H1 (`#`) für Klarheit

### Automatisches Speichern

- Keine Sorge über verlorene Arbeit
- Änderungen werden automatisch gespeichert
- Speicherstatus in der Statusleiste überprüfen

### Schnellerfassung

Müssen Sie etwas schnell notieren?

- Verwenden Sie Schnellerfassung (`Cmd/Ctrl+Shift+N`)
- Erstellt sofort eine Notiz
- Perfekt für flüchtige Gedanken
