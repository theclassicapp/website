# Schnellstart-Anleitung

Willkommen bei Classic! Diese Anleitung hilft Ihnen, in wenigen Minuten mit Classic loszulegen.

## Systemanforderungen

- **macOS**: macOS 12 (Monterey) oder neuer
- **Windows**: Windows 10 oder neuer
- **Linux**: Ubuntu 20.04+ / Fedora 36+ oder neuer

## Installation

1. Laden Sie Classic von der [Webseite](/) herunter.
2. Öffnen Sie die Anwendung
3. Beginnen Sie zu schreiben!

## Erste Schritte

Wenn Sie Classic zum ersten Mal öffnen, sehen Sie eine saubere, minimalistische Benutzeroberfläche, die Ihnen hilft, sich auf Ihr Schreiben zu konzentrieren.

![Classic App Screenshot]

### Die Seitenleiste

Auf der linken Seite finden Sie:
- **Tägliche Notizen** - Ihr automatisches Tagebuch
- **Suche** - Finden Sie schnell alles in all Ihren Notizen
- **Befehlspalette** - Drücken Sie `Cmd/Ctrl+P` (Windows/Linux) oder `Cmd+Shift+P` (macOS), um die Befehlspalette zu öffnen
- **Gliederung** - Durchsuchen und organisieren Sie Ihre Notizen in einer Baumstruktur

### Der Editor

Der mittlere Bereich ist der **Markdown-Editor** mit:
- Live-Vorschau beim Tippen
- Syntaxhervorhebung
- Umfangreiche Formatierungswerkzeuge

### Ihre erste Notiz erstellen

1. Klicken Sie auf die Schaltfläche **Neue Notiz** in der Seitenleiste oder drücken Sie `Ctrl/Cmd+N` (Windows/Linux) oder `Cmd+N` (macOS)
2. Geben Sie Ihrer Notiz einen Titel
3. Beginnen Sie in Markdown zu schreiben!

```markdown
# Meine erste Notiz

Willkommen bei Classic! Hier werde ich meine Gedanken und Ideen festhalten.
```

## Nächste Schritte

- Erfahren Sie mehr über [Markdown-Syntax](/help/markdown)
- Entdecken Sie [Schrägstrich-Befehle](/help/slash-commands)
- Richten Sie [Tägliche Notizen](/help/daily-notes) ein
