# Einstellungen

Passen Sie Classic so an, dass es genau so funktioniert, wie Sie es möchten. Von Erscheinungsbild bis Editor-Verhalten ist alles konfigurierbar.

## Einstellungen öffnen

### Tastaturkürzel

| Plattform | Tastenkombination |
|----------|-------------------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Über Menü

- **macOS**: Classic → Einstellungen
- **Windows**: Datei → Einstellungen
- **Linux**: Bearbeiten → Einstellungen

### Über Befehlspalette

1. Befehlspalette öffnen (`Cmd+P` oder `Ctrl+P`)
2. "Einstellungen" eingeben
3. Eingabetaste drücken

## Einstellungen durchsuchen

Eine Einstellung nicht gefunden? Verwenden Sie die Suchleiste:

1. Einstellungen öffnen
2. In das Suchfeld tippen
3. Passende Einstellungen erscheinen
4. Klicken, um zu dieser Einstellung zu springen

## Einstellungen zurücksetzen

### Alle Einstellungen zurücksetzen

Auf Standardwerte zurückkehren:

1. Einstellungen öffnen
2. Nach unten scrollen
3. "Alle Einstellungen zurücksetzen" klicken
4. Aktion bestätigen

### Kategorie zurücksetzen

Bestimmte Kategorie zurücksetzen:

1. Zur Kategorie gehen
2. "Auf Standardwerte zurücksetzen" klicken
3. Nur diese Kategorie wird zurückgesetzt

## Einstellungen exportieren/importieren

### Einstellungen exportieren

Konfiguration speichern:

1. Einstellungen öffnen
2. "Einstellungen exportieren" klicken
3. Speicherort auswählen
4. Als JSON-Datei speichern

### Einstellungen importieren

Gespeicherte Konfiguration laden:

1. Einstellungen öffnen
2. "Einstellungen importieren" klicken
3. JSON-Datei auswählen
4. Import bestätigen

## Einstellungen-Speicherort

Einstellungen werden lokal gespeichert:

| Plattform | Speicherort |
|----------|-------------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Einstellungen synchronisieren (Optional)

Wenn angemeldet, können Einstellungen zwischen Geräten synchronisiert werden:

- **Synchronisieren**: In Einstellungen → Sync aktivieren
- **Was synchronisiert wird**: Design, Schriften, Kurzbefehle
- **Was nicht synchronisiert wird**: Lokale Pfade, Verschlüsselungsschlüssel

## Häufige Anpassungen

### Schriftgröße erhöhen

1. Einstellungen → Erscheinungsbild
2. Editorschriftgröße → Erhöhen
3. Änderungen werden sofort angewendet

### Design ändern

1. Einstellungen → Erscheinungsbild
2. Design → Hell/Dunkel/System auswählen
3. App aktualisiert sofort

### Auto-Backup aktivieren

1. Einstellungen → Backup & Wiederherstellung
2. "Auto-Backup" aktivieren
3. Häufigkeit und Speicherort festlegen

### Kurzbefehle anpassen

1. Einstellungen → Tastaturkürzel
2. Die Aktion finden
3. Aktuelles Kürzel klicken
4. Neue Tasten drücken
5. Speichern

## Tipps

### Schneller Zugriff

- `Cmd/Ctrl+,` verwenden, um Einstellungen schnell zu öffnen
| Nach Einstellungen nach Namen suchen
- Kategorien zur Navigation verwenden

### Vor Änderungen exportieren

Vor großen Änderungen Einstellungen exportieren:

- Einfach wiederherzustellen, falls etwas schief geht
| Konfiguration mit anderen teilen

### Nach neuen Einstellungen suchen

Updates können neue Einstellungen hinzufügen:

- Nach dem Aktualisieren überprüfen
| Neue Optionen erkunden

## Fehlerbehebung

### Einstellungen werden nicht gespeichert

- Dateiberechtigungen überprüfen
| Speicherplatz verifizieren
- Zurücksetzen und neu konfigurieren versuchen

### Einstellungen unerwartet zurückgesetzt

- Könnte auf Dateibeschädigung hinweisen
| Einstellungen regelmäßig exportieren
- Auf App-Konflikte überprüfen

### Einstellung nicht gefunden

- Die Suchleiste verwenden
| Unterschiedliche Kategorien überprüfen
- Könnte in erweiterten Einstellungen sein
