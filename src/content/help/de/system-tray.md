# System-Tray

Classic befindet sich in Ihrem System-Tray für schnellen Zugriff. Erfassen Sie Gedanken sofort, ohne Ihren Workflow zu unterbrechen.

## Was ist der System-Tray?

Der System-Tray (Windows/Linux) oder die Menüleiste (macOS) ist der Ort, an dem Classic im Hintergrund läuft und immer für die Schnellerfassung bereit ist.

### Vorteile

- **Immer verfügbar**: Nur einen Klick entfernt
- **Schnellerfassung**: Gedanken sofort notieren
- **Hintergrundbetrieb**: App muss nicht geöffnet bleiben
- **Benachrichtigungen**: An Aufgaben erinnert werden

## System-Tray aktivieren

### macOS

1. Classic öffnen
2. Zu Einstellungen → Allgemein gehen
3. "In Menüleiste anzeigen" aktivieren
4. Classic-Symbol erscheint in der Menüleiste

### Windows

1. Classic öffnen
2. Zu Einstellungen → Allgemein gehen
3. "Im System-Tray anzeigen" aktivieren
4. Classic-Symbol erscheint im System-Tray

### Linux

1. Classic öffnen
2. Zu Einstellungen → Allgemein gehen
3. "System-Tray-Symbol" aktivieren
4. Hängt von der Desktop-Umgebung ab

## Schnellerfassung

### Schnellerfassung öffnen

Klicken Sie auf das Classic-Symbol im System-Tray:

- **macOS**: Menüleisten-Symbol
- **Windows**: System-Tray-Symbol
- **Linux**: System-Tray-Symbol

### Schnellerfassung verwenden

1. Auf System-Tray-Symbol klicken
2. Schnellerfassungsfenster öffnet sich
3. Notiz eingeben
4. Eingabetaste zum Speichern drücken
5. Fenster schließt sich automatisch

### Tastaturkürzel

Schnellerfassung von überall öffnen:

| Plattform | Tastenkombination |
|----------|-------------------|
| **macOS** | `Cmd+Shift+N` (global) |
| **Windows** | `Ctrl+Shift+N` (global) |
| **Linux** | `Ctrl+Shift+N` (global) |

## System-Tray-Menü

Rechtsklick auf das Tray-Symbol für Optionen:

### Menüoptionen

| Option | Aktion |
|--------|--------|
| **Neue Notiz** | Schnellerfassung öffnen |
| **Neue tägliche Notiz** | Heutige tägliche Notiz öffnen |
| **Suche** | Suchfenster öffnen |
| **Classic öffnen** | Hauptfenster öffnen |
| **Einstellungen** | Einstellungen öffnen |
| **Beenden** | Classic beenden |

## Tray-Einstellungen

### In Einstellungen konfigurieren

| Einstellung | Beschreibung |
|-------------|--------------|
| **Symbol anzeigen** | Tray-Symbol-Sichtbarkeit umschalten |
| **Minimiert starten** | Classic nur im Tray starten |
| **In Tray schließen** | Weiterlaufen beim Schließen des Fensters |
| **Benachrichtigungen** | Benachrichtigungen aktivieren/deaktivieren |

### Bei Anmeldung starten

Lassen Sie Classic automatisch starten:

1. Einstellungen öffnen
2. Zu Allgemein gehen
3. "Bei Anmeldung starten" aktivieren
4. Classic startet beim Hochfahren

## Benachrichtigungen

### Benachrichtigungen aktivieren

1. Einstellungen öffnen
2. Zu Benachrichtigungen gehen
3. "Benachrichtigungen aktivieren" umschalten

### Benachrichtigungstypen

- **Erinnerungen**: Geplante Erinnerungen
- **Tägliche Notiz**: Aufforderungen für tägliche Notizen
- **Updates**: App-Update verfügbar

## Fensterverhalten

### Schließverhalten

Wählen Sie, was beim Schließen des Fensters passiert:

- **In Tray minimieren**: Weiterlaufen, Fenster ausblenden
- **Beenden**: Anwendung beenden
- **Fragen**: Jedes Mal nachfragen

### Minimierverhalten

- **In Tray minimieren**: Im System-Tray ausblenden
- **In Taskleiste minimieren**: Normal minimieren

## Tipps

### Schnelle Erfassung

Verwenden Sie das globale Tastaturkürzel für schnellste Erfassung:

1. `Cmd/Ctrl+Shift+N` überall drücken
2. Gedanken eingeben
3. Eingabetaste drücken
4. Mit der Arbeit fortfahren

### Schneller Zugriff

Klicken Sie auf das Tray-Symbol, um zu sehen:

- Zuletzt verwendete Notizen
- Schnellaktionen
- Heutige tägliche Notiz

### Hintergrundbetrieb

Lassen Sie Classic laufen für:

- Schnelleren Start
- Schnellerfassung
- Benachrichtigungen
- Auto-Backup

## Fehlerbehebung

### Tray-Symbol fehlt

**macOS**

1. Überprüfen, ob Symbol ausgeblendet ist
2. Im Kontrollzentrum suchen (macOS Big Sur+)
3. Classic neu starten

**Windows**

1. Verborgene Symbole überprüfen (Pfeil in Taskleiste)
2. Classic-Symbol in sichtbaren Bereich ziehen
3. Classic neu starten

**Linux**

1. Desktop-Umgebungs-Unterstützung überprüfen
2. System-Tray-Erweiterung installieren falls nötig
3. Classic neu starten

### Globales Tastaturkürzel funktioniert nicht

- Auf Konflikte mit anderen Apps überprüfen
- Tastaturkürzel in Einstellungen wieder aktivieren
- Classic neu starten
- OS-Berechtigungen überprüfen

### Benachrichtigungen werden nicht angezeigt

**macOS**

1. Systemeinstellungen → Benachrichtigungen
2. Classic finden
3. Benachrichtigungen aktivieren

**Windows**

1. Einstellungen → System → Benachrichtigungen
2. Classic-Benachrichtigungen aktivieren

**Linux**

1. Benachrichtigungs-Daemon überprüfen
2. libnotify installieren falls nötig

### Hoher Speicher-/CPU-Verbrauch

Wenn das Tray-Symbol zu viele Ressourcen verbraucht:

1. Animationen in Einstellungen deaktivieren
2. Benachrichtigungshäufigkeit reduzieren
3. Nach App-Updates suchen
