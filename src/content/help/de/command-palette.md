# Befehlspalette

Die Befehlspalette ist Ihr Zugang zu jeder Aktion in Classic. Drücken Sie ein einfaches Tastaturkürzel und greifen Sie sofort auf jede Funktion zu.

## Befehlspalette öffnen

| Plattform | Tastenkombination |
| --------- | ----------------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Verwendung

1. Befehlspalette öffnen (`Cmd+P` oder `Ctrl+P`)
2. Tippen, um nach einem Befehl zu suchen
3. Pfeiltasten zur Navigation verwenden
4. `Eingabetaste` zum Ausführen drücken

## Befehlskategorien

### Dateioperationen

- **Neue Notiz**: Eine neue Notiz erstellen
- **Neue tägliche Notiz**: Heutige tägliche Notiz erstellen/öffnen
- **Notiz öffnen**: Vorhandene Notizen suchen und öffnen
- **Notiz speichern**: Aktuelle Notiz speichern
- **Notiz löschen**: Aktuelle Notiz löschen
- **Notiz umbenennen**: Aktuelle Notiz umbenennen

### Navigation

- **Zu Notiz springen**: Zu einer beliebigen Notiz springen
- **Zu täglicher Notiz springen**: Ein bestimmtes Datum öffnen
- **Zu Tag springen**: Notizen nach Tag durchsuchen
- **Zurück navigieren**: Im Verlauf zurück navigieren
- **Vorwärts navigieren**: Im Verlauf vorwärts navigieren

### Editor

- **Vorschau umschalten**: Zwischen Bearbeitungs- und Vorschaumodus wechseln
- **Seitenleiste umschalten**: Seitenleiste ein-/ausblenden
- **Fokusmodus**: Ablenkungsfreies Schreiben aktivieren
- **Datum einfügen**: Aktuelles Datum/Uhrzeit einfügen
- **Vorlage einfügen**: Eine Vorlage einfügen

### Formatierung

- **Fett**: Text fett formatieren
- **Kursiv**: Text kursiv formatieren
- **Durchgestrichen**: Text durchstreichen
- **Code**: Als Code formatieren
- **Link**: Einen Link einfügen
- **Überschrift**: Überschriftenformat anwenden

### Organisation

- **Tag hinzufügen**: Aktuelle Notiz taggen
- **In Ordner verschieben**: Notiz in einen Ordner verschieben
- **Notiz archivieren**: Aktuelle Notiz archivieren
- **Notiz anheften**: Notiz oben an der Liste anheften

### Ansicht

- **Design umschalten**: Zwischen Hell/Dunkel wechseln
- **Vergrößern**: Schriftgröße erhöhen
- **Verkleinern**: Schriftgröße verringern
- **Zoom zurücksetzen**: Auf Standardgröße zurücksetzen

## Tipps

### Schneller Zugriff

- Befehle werden mit Fuzzy-Suche durchsucht
| Zuletzt verwendete Befehle erscheinen zuerst
- Häufig verwendete Befehle anheften

### Tastaturnavigation

- `↑`/`↓` - Ergebnisse navigieren
- `Eingabetaste` - Befehl ausführen
- `Escape` - Palette schließen
- `Tab` - Durch Kategorien blättern

### Benutzerdefinierte Befehle

1. Einstellungen öffnen
2. Zu Befehlspalette gehen
3. Benutzerdefinierte Befehle hinzufügen
4. Tastaturkürzel definieren

## Häufig verwendete Befehle

Hier sind einige Befehle, die Sie oft verwenden werden:

| Befehl | Was es tut |
| ------- | ---------- |
| `Neue Notiz` | Leere Notiz erstellen |
| `Notiz öffnen` | Alle Notizen durchsuchen |
| `Neue tägliche Notiz` | Heutiges Tagebuch |
| `Vorschau umschalten` | Vorschau vs. Bearbeitung |
| `Tag hinzufügen` | Aktuelle Notiz taggen |
| `Schnellerfassung` | Globale Erfassung |

## Befehle anpassen

### Tastaturkürzel ändern

1. Einstellungen öffnen (`Cmd+,` oder `Ctrl+,`)
2. Zu Tastaturkürzel navigieren
3. Den Befehl finden
4. Auf das aktuelle Tastaturkürzel klicken
5. Neues Tastaturkürzel drücken
6. Speichern

### Benutzerdefinierte Befehle erstellen

Fortgeschrittene Benutzer können benutzerdefinierte Befehle erstellen:

1. Einstellungen öffnen
2. Zu Befehle gehen
3. "Benutzerdefinierten Befehl hinzufügen" klicken
4. Befehlsnamen definieren
5. Die Aktion(en) hinzufügen
6. Ein Tastaturkürzel festlegen (optional)
