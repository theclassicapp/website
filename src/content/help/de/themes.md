# Designs & Erscheinungsbild

Passen Sie Classics Erscheinungsbild an Ihren Stil an und reduzieren Sie Augenbelastung.

## Design-Optionen

### Heller Modus

Classics Standard-Design bietet:

- Saubere, helle Benutzeroberfläche
- Optimal für gut beleuchtete Umgebungen
- Hoher Kontrast für Lesbarkeit

### Dunkler Modus

Perfekt für Nachteulen und Arbeit bei schlechtem Licht:

- Reduziert Augenbelastung
- Niedrigerer Batterieverbrauch bei OLED-Bildschirmen
- Angenehm für die Augen in dunklen Umgebungen

### System (Auto)

Folgt dem Design Ihres Betriebssystems:

- Wechselt automatisch basierend auf OS-Einstellungen
| Kein manuelles Umschalten nötig
- Konsistent mit anderen Apps

## Designs wechseln

### Über Einstellungen

1. Einstellungen öffnen (`Cmd+,` oder `Ctrl+,`)
2. Zu **Erscheinungsbild** navigieren
3. Bevorzugtes Design auswählen

### Über Befehlspalette

1. Befehlspalette öffnen (`Cmd+P` oder `Ctrl+P`)
2. "Design" eingeben
3. Bevorzugtes Design auswählen

### Tastaturkürzel

| Plattform | Tastenkombination |
| --------- | ----------------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Zyklus durch: Hell → Dunkel → System

## Schrifteinstellungen

### Editorschrift

Das Schreiberlebnis anpassen:

1. Einstellungen öffnen
2. Zu **Erscheinungsbild** → **Editor** gehen
3. Schrift-Optionen konfigurieren

| Einstellung | Optionen | Standard |
| ----------- | -------- | -------- |
| Schriftfamilie | Systemschriften + benutzerdefiniert | System UI |
| Schriftgröße | 12-24px | 16px |
| Zeilenhöhe | 1.0-2.5 | 1.75 |
| Zeichenabstand | -2 bis 4 | 0 |

### UI-Schrift

Schrift der Benutzeroberfläche anpassen:

1. Einstellungen öffnen
2. Zu **Erscheinungsbild** → **Benutzeroberfläche** gehen
3. Schrift und Größe auswählen

### Schriftbreite

Für Codeblöcke und technische Inhalte:

1. Einstellungen öffnen
2. Zu **Erscheinungsbild** → **Code** gehen
3. Schriftbreite auswählen

## Benutzeroberflächen-Optionen

### Seitenleiste

Verhalten der Seitenleiste anpassen:

| Einstellung | Beschreibung |
| ----------- | ------------ |
| **Auto-Ausblenden** | Seitenleiste ausblenden, wenn nicht verwendet |
| **Breite** | Seitenleistenbreite anpassen (200-400px) |
| **Symbole anzeigen** | Symbol-Sichtbarkeit umschalten |

### Editor-Layout

Wählen Sie Ihr Bearbeitungserlebnis:

- **Geteilte Ansicht**: Bearbeiten und Vorschau nebeneinander
- **Nur Vorschau**: Nur gerenderte Ausgabe sehen
- **Nur Bearbeiten**: Fokus auf rohes Markdown

### Notizliste

Konfigurieren, wie Notizen erscheinen:

- **Listenansicht**: Kompakt, eine Zeile pro Notiz
- **Kartenansicht**: Vorschaukarten mit Snippets
- **Baumansicht**: Hierarchische Ordnerstruktur

## Akzentfarben

### Verfügbare Farben

Wählen Sie Ihre Akzentfarbe:

- Klassisch Blau (Standard)
- Lila
- Grün
- Orange
- Rot
- Rosa
- Türkis

### Akzentfarbe ändern

1. Einstellungen öffnen
2. Zu **Erscheinungsbild** → **Farben** gehen
3. Bevorzugten Akzent auswählen

Die Akzentfarbe beeinflusst:

- Links und Schaltflächen
| Auswahl-Hervorhebungen
- Aktive Elemente
| Fokus-Indikatoren

## Barrierefreiheit

### Modus mit hohem Kontrast

Für bessere Sichtbarkeit:

1. Einstellungen öffnen
2. Zu **Erscheinungsbild** → **Barrierefreiheit** gehen
3. **Hoher Kontrast** aktivieren

### Reduzierte Bewegung

Animationen minimieren:

1. Einstellungen öffnen
2. Zu **Erscheinungsbild** → **Barrierefreiheit** gehen
3. **Reduzierte Bewegung** aktivieren

### Schriftanpassungen

- Minimale Schriftgröße erhöhen
- Legasthenie-freundliche Schriften verwenden
| Kontrastverhältnisse anpassen

## Benutzerdefiniertes CSS (Fortgeschritten)

Für Power-User, die vollständige Kontrolle wollen:

1. Einstellungen öffnen
2. Zu **Erscheinungsbild** → **Erweitert** gehen
3. **Benutzerdefiniertes CSS öffnen** klicken
4. Anpassungen hinzufügen

### Beispiel-Anpassungen

```css
/* Größere Überschriften */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Benutzerdefinierte Link-Farbe */
a { color: #3b82f6; }

/* Breiterer Zeilenabstand */
.prose p { line-height: 2; }
```

## Design-Synchronisierung

Wenn Sie sich anmelden (optional), werden Ihre Design-Einstellungen zwischen Geräten synchronisiert:

- Automatische Design-Synchronisierung
- Schrift-Einstellungen synchronisieren
- Benutzerdefiniertes CSS synchronisiert

## Tipps

### An Ihren Workflow anpassen

- **Hell** tagsüber, **Dunkel** nachts verwenden
- **System**-Modus erledigt dies automatisch
| Beleuchtung Ihrer Umgebung berücksichtigen

### Augenbelastung reduzieren

- Dunkelmodus bei schlechtem Licht verwenden
| Schriftgröße für lange Sitzungen erhöhen
- Hohen Kontrast aktivieren falls nötig
| Regelmäßige Pausen einlegen

### Produktivität

- Das Design-Umschalt-Tastaturkürzel lernen
- Auto-Umschaltzeiten einrichten
| System-Modus für Konsistenz verwenden
