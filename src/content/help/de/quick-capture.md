# Schnellerfassung

Erfassen Sie Ideen sofort, ohne Ihren Workflow zu unterbrechen. Schnellerfassung öffnet ein minimales Notizfenster von überall in Ihrem System.

## Globales Tastaturkürzel

Drücken Sie das festgelegte Tastaturkürzel, um Schnellerfassung zu öffnen:

| Plattform | Standard-Tastenkombination |
| --------- | -------------------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

Sie können dieses Tastaturkürzel in den Einstellungen anpassen.

## Schnellerfassung verwenden

1. Globales Tastaturkürzel drücken (überall in Ihrem System)
2. Notiz im Popup-Fenster eingeben
3. `Eingabetaste` zum Speichern drücken
4. `Escape` zum Abbrechen drücken

## Funktionen

### Minimale Benutzeroberfläche

- Kleines, fokussiertes Fenster
- Markdown-Unterstützung
| Auto-Fokus auf Eingabe
- Tastatur-First-Design

### Schnellaktionen

- `# ` + Leerzeichen = Überschrift
- `#inbox` = Zu Inbox hinzufügen
- `#heute` = Zur heutigen Notiz hinzufügen
- Tags mit `#tagname`

### Standardziel

Schnellerfassungs-Notizen gehen standardmäßig in Ihren **Posteingang**. Sie können dies ändern:

1. Einstellungen öffnen
2. Zu Schnellerfassung gehen
3. Standardziel auswählen

## Tipps

### Erfassungs-Workflow

1. **Erfassen** - Die Idee schnell notieren
2. **Verarbeiten** - Inbox täglich überprüfen
3. **Organisieren** - In entsprechenden Ordner/Notiz verschieben

### Best Practices

- Kurz halten - eine Idee pro Erfassung
- Tags für Kontext verwenden
- Keine Sorge über Formatierung - später verfeinern
- Erfassungen regelmäßig überprüfen und verarbeiten

### Integration

Schnellerfassung funktioniert gut mit:

- **Tägliche Notizen**: Jeden Tag mit Überprüfung der Erfassungen beginnen
- **Inbox**: Als Zwischenablage verwenden
- **Tags**: Sofort Kontext hinzufügen

## Fehlerbehebung

### Tastaturkürzel funktioniert nicht?

**macOS**
1. Systemeinstellungen > Tastatur > Kurzbefehle überprüfen
2. Nach Konflikten suchen
3. Classic neu starten

**Windows**
1. Classic als Administrator ausführen
2. Auf konflikterzeugende Kurzbefehle überprüfen
3. Classic neu starten

### Fenster erscheint nicht?

- Überprüfen, ob Classic läuft
- Das Tastaturkürzel verifizieren
- Auf Fokus-Probleme überprüfen
- Classic neu starten

### Erfassungen werden nicht gespeichert?

- Standardziel überprüfen
- Schreibberechtigungen verifizieren
- Auf Speicherplatz überprüfen
- Classic neu starten
