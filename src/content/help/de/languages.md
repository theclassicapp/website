# Sprachen & Lokalisierung

Classic spricht Ihre Sprache. Verwenden Sie die App in Ihrer bevorzugten Sprache und schreiben Sie Notizen in einer beliebigen Sprache Ihrer Wahl.

## Unterstützte Sprachen

Classics Benutzeroberfläche ist in mehreren Sprachen verfügbar:

| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |

| Und mehr... | |

## Sprache der Benutzeroberfläche ändern

### Über Einstellungen

1. Einstellungen öffnen (`Cmd+,` oder `Ctrl+,`)
2. Zu **Allgemein** → **Sprache** gehen
3. Bevorzugte Sprache auswählen
4. Classic neu starten zum Anwenden

### Über Befehlspalette

1. Befehlspalette öffnen (`Cmd+P` oder `Ctrl+P`)
2. "Sprache" eingeben
3. Bevorzugte Sprache auswählen
4. Neustarten zum Anwenden der Änderungen

## In mehreren Sprachen schreiben

Classic unterstützt das Schreiben von Notizen in jeder Sprache:

### Unicode-Unterstützung

- Vollständige Unicode-Unterstützung
- RTL-Sprachen (Right-to-Left) unterstützt
- CJK-Zeichen (Chinesisch, Japanisch, Koreanisch) funktionieren perfekt
- Emoji-Unterstützung

### Spracherkennung

Classic kann die Sprache Ihrer Notizen erkennen:

- Auto-Erkennung für Rechtschreibprüfung
| Sprach-Einstellung pro Notiz
- Unterstützung für gemischte Sprachen

## Rechtschreibprüfung

### Rechtschreibprüfung aktivieren

1. Einstellungen öffnen
2. Zu **Editor** → **Rechtschreibprüfung** gehen
3. "Rechtschreibprüfung" aktivieren

### Sprachen für Rechtschreibprüfung

1. Einstellungen öffnen
2. Zu **Editor** → **Rechtschreibprüfung** gehen
3. "Sprachen" klicken
4. Zu überprüfende Sprachen auswählen

### Wörter zum Wörterbuch hinzufügen

1. Rechtsklick auf ein markiertes Wort
2. "Zum Wörterbuch hinzufügen" auswählen
3. Wort wird nicht mehr markiert

## RTL-Unterstützung

Für Arabisch, Hebräisch und andere RTL-Sprachen:

### Automatische RTL-Erkennung

Classic erkennt RTL-Inhalte automatisch:

- Textrichtung passt sich an
| Cursor-Bewegung passt sich an
- Ausrichtung ändert sich

### Manueller RTL-Umschalter

RTL- oder LTR-Richtung erzwingen:

1. Befehlspalette öffnen
2. "RTL" oder "LTR" eingeben
3. Ihre Präferenz auswählen

| Zeichenauswahl Ihres Systems verwenden

## Datums- & Zeitformate

Daten passen sich Ihrem Gebietsschema an:

### Format-Optionen

| Einstellung | Optionen |
|-------------|----------|
| Datumsformat | MM/TT/JJJJ, TT.MM.JJJJ, JJJJ-MM-TT |
| Zeitformat | 12-Stunden, 24-Stunden |
| Erster Tag der Woche | Sonntag, Montag, Samstag |

In Einstellungen → Allgemein → Datum & Zeit konfigurieren

## Zahlenformate

Zahlen werden gemäß Ihrem Gebietsschema angezeigt:

- **Dezimaltrennzeichen**: `.` oder `,`
- **Tausendertrennzeichen**: `,` oder `.` oder Leerzeichen
| **Währungssymbole**: Basierend auf Region

## Tastaturlayouts

Classic funktioniert mit jedem Tastaturlayout:

- QWERTY, AZERTY, QWERTZ
- Dvorak, Colemak
- Benutzerdefinierte Layouts

## Lokalisierungseinstellungen

### In Einstellungen → Allgemein → Sprache

| Einstellung | Beschreibung |
|-------------|--------------|
| **Benutzeroberflächen-Sprache** | App-UI-Sprache |
| **Datumsformat** | Wie Daten erscheinen |
| **Zeitformat** | 12 oder 24 Stunden |
| **Erster Tag der Woche** | Kalender-Starttag |
| **Zahlenformat** | Dezimal-/Tausendertrennzeichen |

## Tipps

### Notizen in gemischten Sprachen

In mehreren Sprachen in einer Notiz schreiben:

```markdown
# Internationales Projekt

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Sprachspezifische Schriften

Manche Sprachen sehen mit bestimmten Schriften besser aus:

1. Einstellungen öffnen
2. Zu Erscheinungsbild → Schriften gehen
3. Schriften für verschiedene Schriften festlegen

### Schnelles Sprachwechseln

Erstellen Sie ein Tastaturkürzel:

1. Einstellungen → Kurzbefehle öffnen
2. "Sprache wechseln" finden
3. Bevorzugtes Tastaturkürzel festlegen

## Fehlerbehebung

### Sprache nicht verfügbar

- Nach App-Updates suchen
| Sprache könnte in Entwicklung sein
- Übersetzungen auf GitHub beisteuern

### Zeichen werden nicht angezeigt

- Erforderliche Schriften installieren
| Schrift-Einstellungen überprüfen
- Andere Schrift versuchen

### Rechtschreibprüfung funktioniert nicht

- Verifizieren, dass Rechtschreibprüfung aktiviert ist
| Sprache zur Rechtschreibprüfung hinzufügen
- Classic neu starten

### RTL funktioniert nicht korrekt

- Sicherstellen, dass RTL-Erkennung aktiv ist
| Manuellen RTL-Umschalter versuchen
| Auf Probleme mit gemischten Inhalten überprüfen
