# Settings

Customize Classic to work exactly the way you want. From appearance to editor behavior, everything is configurable.

## Opening Settings

### Keyboard Shortcut

| Platform | Shortcut |
|----------|----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Via Menu

- **macOS**: Classic → Settings
- **Windows**: File → Settings
- **Linux**: Edit → Settings

### Via Command Palette

1. Open Command Palette (`Cmd+P` or `Ctrl+P`)
2. Type "Settings"
3. Press Enter

## Settings Categories

### General

Basic application settings

| Setting | Description |
|---------|-------------|
| **Language** | Interface language |
| **Theme** | Light, Dark, or System |
| **Start at Login** | Launch on startup |
| **Check for Updates** | Auto-update behavior |

### Appearance

Customize the look and feel

| Setting | Options |
|---------|---------|
| **Theme** | Light / Dark / System |
| **Accent Color** | Blue, Purple, Green, etc. |
| **Editor Font** | Font family and size |
| **UI Font** | Interface font |
| **Monospace Font** | Code font |
| **Sidebar Width** | 200-400px |
| **Line Height** | 1.0-2.5 |

### Editor

Configure writing behavior

| Setting | Description |
|---------|-------------|
| **Spell Check** | Enable/disable spell checking |
| **Auto-save** | Automatic saving interval |
| **Word Wrap** | Enable/disable word wrap |
| **Line Numbers** | Show/hide line numbers |
| **Tab Size** | 2, 4, or 8 spaces |
| **Auto-correct** | Smart quotes, auto-replace |

### Version History

Control version tracking

| Setting | Options | Default |
|---------|---------|---------|
| **Enable History** | On/Off | On |
| **Retention Period** | 30-180 days | 90 days |
| **Max Versions** | 50-500 | 100 |

### Backup & Restore

Data protection settings

| Setting | Description |
|---------|-------------|
| **Auto Backup** | Enable automatic backups |
| **Backup Frequency** | 15min / 30min / 1hr / 4hr |
| **Backup Location** | Custom path |
| **Keep Backups** | 5 / 10 / 20 / 50 |
| **Encrypt Backups** | Password protection |

### Keyboard Shortcuts

View and customize shortcuts

- See all available shortcuts

- Reset to defaults
- Import/export shortcuts

### Import & Export

Data migration settings

### Advanced

## Searching Settings

Can't find a setting? Use the search bar

1. Open Settings
2. Type in the search box
3. Matching settings appear
4. Click to jump to that setting

## Resetting Settings

### Reset All Settings

Return to defaults

1. Open Settings
2. Scroll to bottom
3. Click "Reset All Settings"
4. Confirm the action

### Reset Category

Reset specific category

1. Go to the category
2. Click "Reset to Defaults"
3. Only that category resets

## Exporting/Importing Settings

### Export Settings

Save your configuration

1. Open Settings
2. Click "Export Settings"
3. Choose location
4. Save as JSON file

### Import Settings

Load a saved configuration

1. Open Settings
2. Click "Import Settings"
3. Select the JSON file
4. Confirm import

## Settings Location

Settings are stored locally

| Platform | Location |
|----------|----------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Sync Settings (Optional)

When signed in, settings can sync across devices

- **Sync**: Enable in Settings → Sync
- **What Syncs**: Theme, fonts, shortcuts
- **What Doesn't Sync**: Local paths, encryption keys

## Common Customizations

### Increase Font Size

1. Settings → Appearance
2. Editor Font Size → Increase
3. Changes apply immediately

### Change Theme

1. Settings → Appearance
2. Theme → Select Light/Dark/System
3. App updates instantly

### Enable Auto-Backup

1. Settings → Backup & Restore
2. Enable "Auto Backup"
3. Set frequency and location

### Customize Shortcuts

1. Settings → Keyboard Shortcuts
2. Find the action
3. Click current shortcut
4. Press new keys
5. Save

## Tips

### Quick Access

- Use `Cmd/Ctrl+,` to open settings quickly
- Search for settings by name
- Use categories to navigate

### Export Before Changes

Before major changes, export your settings

- Easy to restore if something goes wrong
- Share configuration with others

### Check for New Settings

Updates may add new settings

- Check after updating
- Explore new options

## Troubleshooting

### Settings Not Saving

- Check file permissions
- Verify disk space
- Try resetting and reconfiguring

### Settings Reset Unexpectedly

- May indicate file corruption
- Export settings regularly
- Check for app conflicts

### Can't Find a Setting

- Use the search bar
- Check different categories
- May be in Advanced settings
