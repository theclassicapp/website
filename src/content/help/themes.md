# Themes & Appearance

Personalize Classic's appearance to match your style and reduce eye strain.

## Theme Options

### Light Mode

Classic's default theme features

- Clean, bright interface
- Optimal for well-lit environments
- High contrast for readability

### Dark Mode

Perfect for night owls and low-light work

- Reduces eye strain
- Lower battery consumption on OLED screens
- Easy on the eyes in dark environments

### System (Auto)

Follow your operating system's theme

- Automatically switches based on OS settings
- No manual switching needed
- Consistent with other apps

## Switching Themes

### Via Settings

1. Open Settings (`Cmd+,` or `Ctrl+,`)
2. Navigate to **Appearance**
3. Select your preferred theme

### Via Command Palette

1. Open Command Palette (`Cmd+P` or `Ctrl+P`)
2. Type "Theme"
3. Select your preferred theme

### Keyboard Shortcut

| Platform | Shortcut |
| -------- | -------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Cycles through: Light → Dark → System

## Font Settings

### Editor Font

Customize the writing experience

1. Open Settings
2. Go to **Appearance** → **Editor**
3. Configure font options

| Setting | Options | Default |
| ------- | ------- | ------- |
| Font Family | System fonts + custom | System UI |
| Font Size | 12-24px | 16px |
| Line Height | 1.0-2.5 | 1.75 |
| Letter Spacing | -2 to 4 | 0 |

### UI Font

Adjust the interface font

1. Open Settings
2. Go to **Appearance** → **Interface**
3. Choose font and size

### Monospace Font

For code blocks and technical content

1. Open Settings
2. Go to **Appearance** → **Code**
3. Select monospace font

## Interface Options

### Sidebar

Customize the sidebar behavior

| Setting | Description |
| ------- | ----------- |
| **Auto-hide** | Hide sidebar when not in use |
| **Width** | Adjust sidebar width (200-400px) |
| **Show Icons** | Toggle icon visibility |

### Editor Layout

Choose your editing experience

- **Split View**: Edit and preview side by side
- **Preview Only**: See rendered output only
- **Edit Only**: Focus on raw Markdown

### Note List

Configure how notes appear

- **List View**: Compact, one-line per note
- **Card View**: Preview cards with snippets
- **Tree View**: Hierarchical folder structure

## Accessibility

### High Contrast Mode

For better visibility

1. Open Settings
2. Go to **Appearance** → **Accessibility**
3. Enable **High Contrast**

### Reduced Motion

Minimize animations

1. Open Settings
2. Go to **Appearance** → **Accessibility**
3. Enable **Reduced Motion**

### Font Adjustments

- Increase minimum font size
- Use dyslexia-friendly fonts
- Adjust contrast ratios

## Custom CSS (Advanced)

For power users who want complete control

1. Open Settings
2. Go to **Appearance** → **Advanced**
3. Click **Open Custom CSS**
4. Add your customizations

### Example Customizations

```css
/* Larger headings */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Custom link color */
a { color: #3b82f6; }

/* Wider line spacing */
.prose p { line-height: 2; }
```

## Tips

### Matching Your Workflow

- Use **Light** during day, **Dark** at night
- **System** mode handles this automatically
- Consider your ambient lighting

### Reducing Eye Strain

- Use Dark mode in low light
- Increase font size for long sessions
- Enable High Contrast if needed
- Take regular breaks

### Productivity

- Learn the theme toggle shortcut
- Set up auto-switching times
- Use System mode for consistency
