# Langues et localisation

Classic parle votre langue. Utilisez l'application dans votre langue préférée et écrivez des notes dans la langue de votre choix.

## Langues prises en charge

L'interface de Classic est disponible en plusieurs langues :

| Langue | Code |
|--------|------|
| English | `en` |
| 简体中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |
| 日本語 | `ja` |
| 한국어 | `ko` |
| Español | `es` |
| Français | `fr` |
| Deutsch | `de` |
| Italiano | `it` |
| Português | `pt` |
| Русский | `ru` |
| العربية | `ar` |
| Et plus... | |

## Changer la langue de l'interface

### Via les Paramètres

1. Ouvrez les Paramètres (`Cmd+,` ou `Ctrl+,`)
2. Allez dans **Général** → **Langue**
3. Sélectionnez votre langue préférée
4. Redémarrez Classic pour appliquer

### Via la Palette de commandes

1. Ouvrez la Palette de commandes (`Cmd+P` ou `Ctrl+P`)
2. Tapez "Langue"
3. Sélectionnez votre langue préférée
4. Redémarrez pour appliquer les changements

## Écrire dans plusieurs langues

Classic prend en charge l'écriture de notes dans n'importe quelle langue.

### Support Unicode

- Support Unicode complet
- Langues RTL (de droite à gauche) prises en charge
- Caractères CJK (chinois, japonais, coréen) fonctionnent parfaitement
- Support des emojis

### Détection de langue

Classic peut détecter la langue de vos notes :

- Détection automatique pour la vérification orthographique
- Paramètre de langue par note
- Support des langues mixtes

## Vérification orthographique

### Activer la vérification orthographique

1. Ouvrez les Paramètres
2. Allez dans **Éditeur** → **Vérification orthographique**
3. Activez "Vérification orthographique"

### Langues de vérification orthographique

1. Ouvrez les Paramètres
2. Allez dans **Éditeur** → **Vérification orthographique**
3. Cliquez sur "Langues"
4. Sélectionnez les langues à vérifier

### Ajouter des mots au dictionnaire

1. Faites un clic droit sur un mot signalé
2. Sélectionnez "Ajouter au dictionnaire"
3. Le mot ne sera plus signalé

## Support RTL

Pour l'arabe, l'hébreu et autres langues RTL.

### Détection RTL automatique

Classic détecte le contenu RTL automatiquement :

- La direction du texte s'ajuste
- Le mouvement du curseur s'adapte
- L'alignement change

### Basculement RTL manuel

Forcez la direction RTL ou LTR :

1. Ouvrez la Palette de commandes
2. Tapez "RTL" ou "LTR"
3. Sélectionnez votre préférence

## Méthodes de saisie

Classic fonctionne avec les méthodes de saisie de votre système.

### Méthodes de saisie CJK

- Fonctionne avec Pinyin, Zhuyin, etc.
- Supporte l'IME japonais
- Méthodes de saisie coréennes prises en charge

### Caractères spéciaux

Insérez facilement des caractères spéciaux :

- Utilisez le sélecteur de caractères de votre système
- Copiez/collez depuis des tables de caractères
- Utilisez la saisie Unicode

## Formats de date et d'heure

Les dates s'adaptent à votre locale.

### Options de format

| Paramètre | Options |
|-----------|---------|
| Format de date | JJ/MM/AAAA, MM/JJ/AAAA, AAAA-MM-JJ |
| Format d'heure | 12 heures, 24 heures |
| Premier jour de la semaine | Dimanche, Lundi, Samedi |

Configurez dans Paramètres → Général → Date et heure

## Formats numériques

Les nombres s'affichent selon votre locale :

- **Séparateur décimal** : `.` ou `,`
- **Séparateur de milliers** : `,` ou `.` ou espace
- **Symboles monétaires** : Selon la région

## Dispositions clavier

Classic fonctionne avec n'importe quelle disposition clavier :

- AZERTY, QWERTZ, QWERTY
- Dvorak, Colemak
- Dispositions personnalisées

## Paramètres de localisation

### Dans Paramètres → Général → Langue

| Paramètre | Description |
|-----------|-------------|
| **Langue de l'interface** | Langue de l'interface utilisateur |
| **Format de date** | Comment les dates apparaissent |
| **Format d'heure** | 12 ou 24 heures |
| **Premier jour de la semaine** | Jour de début du calendrier |
| **Format numérique** | Séparateurs décimal/millier |

## Conseils

### Notes multilingues

Écrivez dans plusieurs langues dans une seule note :

```markdown
# Projet international

English: Welcome to the project
中文: 欢迎来到这个项目
日本語: プロジェクトへようこそ
한국어: 프로젝트에 오신 것을 환영합니다
```

### Polices spécifiques à une langue

Certaines langues sont plus belles avec des polices spécifiques :

1. Ouvrez les Paramètres
2. Allez dans Apparence → Polices
3. Définissez des polices pour différents scripts

### Changer de langue rapidement

Créez un raccourci clavier :

1. Ouvrez Paramètres → Raccourcis
2. Trouvez "Changer de langue"
3. Définissez votre raccourci préféré

## Dépannage

### Langue non disponible

- Vérifiez les mises à jour de l'application
- La langue peut être en cours de développement
- Contribuez aux traductions sur GitHub

### Caractères non affichés

- Installez les polices requises
- Vérifiez les paramètres de police
- Essayez une police différente

### Vérification orthographique ne fonctionne pas

- Vérifiez que la vérification orthographique est activée
- Ajoutez la langue à la vérification orthographique
- Redémarrez Classic

### RTL ne fonctionne pas correctement

- Assurez-vous que la détection RTL est activée
- Essayez le basculement RTL manuel
- Vérifiez les problèmes de contenu mixte
