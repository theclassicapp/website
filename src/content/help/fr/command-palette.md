# Palette de commandes

La palette de commandes est votre passerelle vers chaque action dans Classic. Appuyez sur un simple raccourci clavier et accédez instantanément à n'importe quelle fonctionnalité.

## Ouvrir la Palette de commandes

| Plateforme | Raccourci |
| ---------- | --------- |
| **macOS** | `Cmd+P` |
| **Windows/Linux** | `Ctrl+P` |

## Comment utiliser

1. Ouvrez la palette de commandes (`Cmd+P` ou `Ctrl+P`)
2. Tapez pour rechercher une commande
3. Utilisez les touches fléchées pour naviguer
4. Appuyez sur `Entrée` pour exécuter

## Catégories de commandes

### Opérations de fichiers

- **Nouvelle note** : Créez une nouvelle note
- **Nouvelle note quotidienne** : Créez/ouvrez la note quotidienne du jour
- **Ouvrir une note** : Recherchez et ouvrez des notes existantes
- **Sauvegarder la note** : Sauvegardez la note actuelle
- **Supprimer la note** : Supprimez la note actuelle
- **Renommer la note** : Renommez la note actuelle

### Navigation

- **Aller à la note** : Accédez à n'importe quelle note
- **Aller à la note quotidienne** : Ouvrez une date spécifique
- **Aller au tag** : Parcourez les notes par tag
- **Retour** : Naviguez en arrière dans l'historique
- **Avancer** : Naviguez en avant dans l'historique

### Éditeur

- **Basculer l'aperçu** : Basculez entre les modes édition et aperçu
- **Basculer la barre latérale** : Afficher/masquer la barre latérale
- **Mode focus** : Entrez en écriture sans distraction
- **Insérer la date** : Ajoutez la date/heure actuelle
- **Insérer un modèle** : Insérez un modèle

### Formatage

- **Gras** : Mettez le texte en gras
- **Italique** : Mettez le texte en italique
- **Barré** : Barrez le texte
- **Code** : Formatez comme code
- **Lien** : Insérez un lien
- **Titre** : Appliquez le format de titre

### Organisation

- **Ajouter un tag** : Taguez la note actuelle
- **Déplacer vers un dossier** : Déplacez la note vers un dossier
- **Archiver la note** : Archivez la note actuelle
- **Épingler la note** : Épinglez la note en haut de la liste

### Affichage

- **Basculer le thème** : Basculez entre clair/sombre
- **Zoom avant** : Augmentez la taille de police
- **Zoom arrière** : Diminuez la taille de police
- **Réinitialiser le zoom** : Réinitialisez à la taille par défaut

## Conseils

### Accès rapide

- Les commandes sont recherchées de manière floue
- Les commandes récentes apparaissent en premier
- Épinglez les commandes fréquemment utilisées

### Navigation clavier

- `↑`/`↓` - Naviguer dans les résultats
- `Entrée` - Exécuter la commande
- `Echap` - Fermer la palette
- `Tab` - Parcourir les catégories

### Commandes personnalisées

1. Ouvrez les Paramètres
2. Allez dans Palette de commandes
3. Ajoutez des commandes personnalisées
4. Définissez des raccourcis clavier

## Commandes fréquemment utilisées

Voici quelques commandes que vous utiliserez souvent :

| Commande | Ce qu'elle fait |
| -------- | --------------- |
| `Nouvelle note` | Crée une note vide |
| `Ouvrir une note` | Recherche toutes les notes |
| `Nouvelle note quotidienne` | Journal du jour |
| `Basculer l'aperçu` | Aperçu vs édition |
| `Ajouter un tag` | Tague la note actuelle |
| `Capture rapide` | Capture globale |

## Personnaliser les commandes

### Changer les raccourcis

1. Ouvrez les Paramètres (`Cmd+,` ou `Ctrl+,`)
2. Naviguez vers Raccourcis clavier
3. Trouvez la commande
4. Cliquez sur le raccourci actuel
5. Appuyez sur votre nouveau raccourci
6. Sauvegardez

### Créer des commandes personnalisées

Les utilisateurs avancés peuvent créer des commandes personnalisées :

1. Ouvrez les Paramètres
2. Allez dans Commandes
3. Cliquez sur "Ajouter une commande personnalisée"
4. Définissez le nom de la commande
5. Ajoutez l'action ou les actions
6. Définissez un raccourci (optionnel)
