# Comprendre l'interface

L'interface de Classic est conçue pour être épurée, intuitive et ne pas vous gêner. Voici une visite des principaux composants.

## Disposition principale

Classic dispose d'une disposition à trois panneaux :

1. **Barre latérale** (gauche) - Navigation et organisation des notes
2. **Éditeur** (centre) - Votre espace d'écriture
3. **Aperçu/Propriétés** (droite) - Panneaux optionnels

## Barre latérale

La barre latérale contient plusieurs sections clés :

### Liste des notes

- **Toutes les notes** : Affichez toutes vos notes dans une seule liste
- **Notes quotidiennes** : Accès rapide au journal du jour
- **Favoris** : Étoilez les notes fréquemment utilisées
- **Dossiers** : Organisez les notes en hiérarchies

### Navigation

- **Barre de recherche** : Trouvez des notes instantanément
- **Tags** : Parcourez par tag
- **Rétroliens** : Voyez les connexions entre les notes

### Actions

- **Nouvelle note** : Créez une nouvelle note
- **Nouveau dossier** : Organisez avec des dossiers
- **Importer** : Importez du contenu externe

## Éditeur

L'éditeur est l'endroit où vous écrivez et formatez vos notes.

### Fonctionnalités

- **Support Markdown** : Syntaxe Markdown complète
- **Aperçu en direct** : Voyez le texte formaté pendant que vous tapez
- **Coloration syntaxique** : Les blocs de code sont colorés
- **Commandes slash** : Tapez `/` pour des actions rapides

### Modes de l'éditeur

| Mode | Description |
|------|-------------|
| **Édition uniquement** | Voyez le Markdown brut |
| **Aperçu uniquement** | Voyez le rendu |
| **Vue fractionnée** | Éditez et prévisualisez côte à côte |

### Barre d'outils

Actions de formatage rapides :

- Gras, Italique, Barré
- Titres (H1-H6)
- Listes (puces, numérotées, tâches)
- Liens et images
- Blocs de code
- Tableaux

## Barre d'état

En bas de l'éditeur :

- **Nombre de mots/caractères** : Suivez votre écriture
- **Dernière sauvegarde** : Quand la note a été sauvegardée pour la dernière fois
- **Chemin de la note** : Où la note est située
- **État de synchronisation** : Si vous utilisez la synchronisation optionnelle

## Menus contextuels

Clic droit pour les actions contextuelles :

### Menu contextuel de note

- Renommer
- Déplacer vers un dossier
- Dupliquer
- Supprimer
- Exporter
- Copier le lien

### Menu contextuel de l'éditeur

- Couper, Copier, Coller
- Options de formatage
- Insérer un lien/image
- Rechercher et remplacer

## Navigation au clavier

Naviguez dans l'interface avec votre clavier :

| Action | Raccourci |
|--------|----------|
| Basculer la barre latérale | `Cmd/Ctrl + M` |
| Focus sur la recherche | `Cmd/Ctrl + F` |
| Palette de commandes | `Cmd/Ctrl + P` |
| Nouvelle note | `Cmd/Ctrl + N` |

## Personnalisation

Ajustez l'interface à votre convenance :

### Largeur de la barre latérale

1. Survolez le bord de la barre latérale
2. Faites glisser pour redimensionner
3. Ou définissez une largeur exacte dans les Paramètres

### Thème

Basculez entre le mode Clair et Sombre :

- Utilisez le bouton de thème dans l'en-tête
- Ou configurez-le pour suivre la préférence de votre système

### Taille de police

1. Ouvrez les Paramètres
2. Allez dans Apparence
3. Ajustez la taille de police de l'éditeur

## Conseils

### Mode focus

Masquez tous les panneaux pour une écriture sans distraction :

1. Ouvrez la Palette de commandes
2. Tapez "Mode Focus"
3. Appuyez sur Entrée

### Accès rapide

- Épinglez les notes fréquemment utilisées aux Favoris
- Utilisez les raccourcis clavier pour les actions courantes
- Personnalisez la barre d'outils dans les Paramètres
