# Thèmes et apparence

Personnalisez l'apparence de Classic pour correspondre à votre style et réduire la fatigue oculaire.

## Options de thème

### Mode clair

Le thème par défaut de Classic propose :

- Interface épurée et lumineuse
- Optimal pour les environnements bien éclairés
- Contraste élevé pour la lisibilité

### Mode sombre

Parfait pour les oiseaux de nuit et le travail en faible luminosité :

- Réduit la fatigue oculaire
- Consommation d'énergie inférieure sur les écrans OLED
- Doux pour les yeux dans les environnements sombres

### Système (Auto)

Suit le thème de votre système d'exploitation :

- Bascule automatiquement selon les paramètres du système d'exploitation
- Pas de basculement manuel nécessaire
- Cohérent avec les autres applications

## Changer de thème

### Via les Paramètres

1. Ouvrez les Paramètres (`Cmd+,` ou `Ctrl+,`)
2. Naviguez vers **Apparence**
3. Sélectionnez votre thème préféré

### Via la Palette de commandes

1. Ouvrez la Palette de commandes (`Cmd+P` ou `Ctrl+P`)
2. Tapez "Thème"
3. Sélectionnez votre thème préféré

### Raccourci clavier

| Plateforme | Raccourci |
| ---------- | --------- |
| **macOS** | `Cmd+Shift+T` |
| **Windows/Linux** | `Ctrl+Shift+T` |

Cycle : Clair → Sombre → Système

## Paramètres de police

### Police de l'éditeur

Personnalisez l'expérience d'écriture :

1. Ouvrez les Paramètres
2. Allez dans **Apparence** → **Éditeur**
3. Configurez les options de police

| Paramètre | Options | Par défaut |
| --------- | ------- | ---------- |
| Famille de police | Polices système + personnalisées | Interface système |
| Taille de police | 12-24px | 16px |
| Hauteur de ligne | 1.0-2.5 | 1.75 |
| Espacement des lettres | -2 à 4 | 0 |

### Police de l'interface

Ajustez la police de l'interface :

1. Ouvrez les Paramètres
2. Allez dans **Apparence** → **Interface**
3. Choisissez la police et la taille

### Police à chasse fixe

Pour les blocs de code et le contenu technique :

1. Ouvrez les Paramètres
2. Allez dans **Apparence** → **Code**
3. Sélectionnez la police à chasse fixe

## Options de l'interface

### Barre latérale

Personnalisez le comportement de la barre latérale :

| Paramètre | Description |
| --------- | ----------- |
| **Masquage automatique** | Masquer la barre latérale lorsqu'elle n'est pas utilisée |
| **Largeur** | Ajuster la largeur de la barre latérale (200-400px) |
| **Afficher les icônes** | Basculer la visibilité des icônes |

### Disposition de l'éditeur

Choisissez votre expérience d'édition :

- **Vue fractionnée** : Éditez et prévisualisez côte à côte
- **Aperçu uniquement** : Voir uniquement le rendu
- **Édition uniquement** : Concentrez-vous sur le Markdown brut

### Liste de notes

Configurez l'apparence des notes :

- **Vue liste** : Compacte, une ligne par note
- **Vue carte** : Cartes de prévisualisation avec extraits
- **Vue arborescente** : Structure de dossiers hiérarchique

## Couleurs d'accentuation

### Couleurs disponibles

Choisissez votre couleur d'accentuation :

- Bleu Classic (par défaut)
- Violet
- Vert
- Orange
- Rouge
- Rose
- Sarcelle

### Changer la couleur d'accentuation

1. Ouvrez les Paramètres
2. Allez dans **Apparence** → **Couleurs**
3. Sélectionnez votre accent préféré

La couleur d'accentuation affecte :

- Liens et boutons
- Surlignages de sélection
- Éléments actifs
- Indicateurs de focus

## Accessibilité

### Mode contraste élevé

Pour une meilleure visibilité :

1. Ouvrez les Paramètres
2. Allez dans **Apparence** → **Accessibilité**
3. Activez **Contraste élevé**

### Mouvement réduit

Minimisez les animations :

1. Ouvrez les Paramètres
2. Allez dans **Apparence** → **Accessibilité**
3. Activez **Mouvement réduit**

### Ajustements de police

- Augmentez la taille de police minimale
- Utilisez des polices adaptées à la dyslexie
- Ajustez les ratios de contraste

## CSS personnalisé (Avancé)

Pour les utilisateurs expérimentés qui veulent un contrôle complet :

1. Ouvrez les Paramètres
2. Allez dans **Apparence** → **Avancé**
3. Cliquez sur **Ouvrir le CSS personnalisé**
4. Ajoutez vos personnalisations

### Exemples de personnalisations

```css
/* Titres plus grands */
.prose h1 { font-size: 2.5rem; }
.prose h2 { font-size: 2rem; }

/* Couleur de lien personnalisée */
a { color: #3b82f6; }

/* Espacement de ligne plus large */
.prose p { line-height: 2; }
```

## Synchronisation des thèmes

Lorsque vous vous connectez (optionnel), vos préférences de thème se synchronisent sur tous les appareils :

- Synchronisation automatique du thème
- Synchronisation des paramètres de police
- Le CSS personnalisé se synchronise

## Conseils

### Adapter à votre flux de travail

- Utilisez **Clair** pendant la journée, **Sombre** la nuit
- Le mode **Système** gère cela automatiquement
- Considérez votre éclairage ambiant

### Réduire la fatigue oculaire

- Utilisez le mode Sombre en faible luminosité
- Augmentez la taille de police pour les longues sessions
- Activez le Contraste élevé si nécessaire
- Prenez des pauses régulières

### Productivité

- Apprenez le raccourci de basculement de thème
- Configurez des heures de basculement automatique
- Utilisez le mode Système pour la cohérence
