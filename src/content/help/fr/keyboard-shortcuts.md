# Raccourcis clavier

Travaillez plus vite avec les raccourcis clavier de Classic. Tous les raccourcis sont personnalisables dans les Paramètres.

## Raccourcis globaux

Ceux-ci fonctionnent n'importe où dans l'application.

### Général

| Raccourci (Windows/Linux) | Raccourci (macOS) | Action |
| ------------------------- | ----------------- | ------ |
| `Ctrl+N` | `Cmd+N` | Nouvelle note |
| `Ctrl+P` | `Cmd+P` | Palette de commandes |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Capture rapide globale |
| `Ctrl+S` | `Cmd+S` | Sauvegarder la note |
| `Ctrl+F` | `Cmd+F` | Recherche |
| `Ctrl+,` | `Cmd+,` | Paramètres |

### Navigation

| Raccourci (Windows/Linux) | Raccourci (macOS) | Action |
| ------------------------- | ----------------- | ------ |
| `Ctrl+1` | `Cmd+1` | Aller à la première note |
| `Ctrl+2` | `Cmd+2` | Aller à la deuxième note |
| `Ctrl+Gauche` | `Cmd+Gauche` | Naviguer en arrière |
| `Ctrl+Droite` | `Cmd+Droite` | Naviguer en avant |
| `Ctrl+Haut` | `Cmd+Haut` | Naviguer vers la note parente |
| `Ctrl+Bas` | `Cmd+Bas` | Développer/focaliser la première sous-note |

### Éditeur

| Raccourci (Windows/Linux) | Raccourci (macOS) | Action |
| ------------------------- | ----------------- | ------ |
| `Ctrl+B` | `Cmd+B` | Gras |
| `Ctrl+I` | `Cmd+I` | Italique |
| `Ctrl+K` | `Cmd+K` | Insérer un lien |
| `Ctrl+Shift+K` | `Cmd+Shift+K` | Barré |
| `Ctrl+`` | `Cmd+`` | Code en ligne |
| `Ctrl+Shift+C` | `Cmd+Shift+C` | Bloc de code |
| `Ctrl+/` | `Cmd+/` | Commandes slash |
| `Tab` | `Tab` | Indenter |
| `Shift+Tab` | `Shift+Tab` | Désindenter |

### Gestion des fenêtres

| Raccourci (Windows/Linux) | Raccourci (macOS) | Action |
| ------------------------- | ----------------- | ------ |
| `Ctrl+W` | `Cmd+W` | Fermer la note actuelle |
| `Ctrl+Shift+W` | `Cmd+Shift+W` | Fermer toutes les notes |
| `F11` | `F11` | Basculer le plein écran |
| `Ctrl+M` | `Cmd+M` | Basculer la barre latérale |

## Capture rapide

| Raccourci (Windows/Linux) | Raccourci (macOS) | Action |
| ------------------------- | ----------------- | ------ |
| `Ctrl+Shift+P` | `Cmd+Shift+P` | Ouvrir la capture rapide |
| `Echap` | `Echap` | Fermer la capture rapide |
| `Ctrl+Entrée` | `Cmd+Entrée` | Sauvegarder et fermer |

## Personnaliser les raccourcis

1. Ouvrez **Paramètres** (`Ctrl+,` ou `Cmd+,`)
2. Naviguez vers **Raccourcis clavier**
3. Cliquez sur n'importe quel raccourci pour le changer
4. Appuyez sur votre nouvelle combinaison de touches
5. Cliquez sur **Sauvegarder**

## Conseils

- La plupart des raccourcis sont personnalisables
- Les conflits avec les raccourcis système seront surlignés
- Réinitialisez aux valeurs par défaut en cliquant sur le bouton de réinitialisation à côté de chaque raccourci
