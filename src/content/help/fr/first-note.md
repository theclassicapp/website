# Créer votre première note

Commencez avec Classic en créant votre première note. C'est simple et intuitif.

## Créer une nouvelle note

### Via la barre latérale

1. Cliquez sur le bouton **+** dans la barre latérale
2. Une nouvelle note est créée
3. Commencez à taper votre contenu

### Via le raccourci clavier

| Plateforme | Raccourci |
| ---------- | --------- |
| **macOS** | `Cmd+N` |
| **Windows/Linux** | `Ctrl+N` |

### Via la Palette de commandes

1. Ouvrez la Palette de commandes (`Cmd+P` ou `Ctrl+P`)
2. Tapez "Nouvelle note"
3. Appuyez sur Entrée

## Écrire votre note

### Édition de base

Commencez simplement à taper ! Classic sauvegarde automatiquement pendant que vous écrivez.

### Ajouter un titre

La première ligne de votre note devient le titre. Utilisez un titre pour de meilleurs résultats :

```markdown
# Ma première note

Ceci est le contenu de ma note.
```

### Formater le texte

Utilisez Markdown pour formater votre texte :

- **Gras** : `**texte**` ou `Cmd/Ctrl+B`
- **Italique** : `*texte*` ou `Cmd/Ctrl+I`
- **Barré** : `~~texte~~`
- **Code** : `` `code` ``

## Sauvegarde

Classic sauvegarde automatiquement vos notes pendant que vous tapez. Pas besoin de sauvegarder manuellement.

### Sauvegarde manuelle

Si nécessaire, vous pouvez forcer une sauvegarde :

- **macOS** : `Cmd+S`
- **Windows/Linux** : `Ctrl+S`

## Organiser votre note

### Ajouter des tags

Taguez votre note pour la retrouver facilement plus tard :

1. Tapez `#` suivi du nom de votre tag
2. Ou utilisez le bouton de tag dans la barre d'outils
3. Plusieurs tags sont supportés

Exemple :

```markdown
# Notes de réunion

Discussion sur la planification T4 #travail #reunions
```

### Déplacer vers un dossier

1. Faites un clic droit sur la note dans la barre latérale
2. Sélectionnez "Déplacer vers un dossier"
3. Choisissez ou créez un dossier de destination

### Épingler les notes importantes

1. Faites un clic droit sur la note
2. Sélectionnez "Épingler en haut"
3. La note apparaît en haut de votre liste

## Prévisualiser votre note

Voyez à quoi ressemble votre note formatée :

- Cliquez sur le bouton **Aperçu**
- Ou utilisez `Cmd/Ctrl+Shift+P`
- Basculez entre les modes Édition et Aperçu

## Prochaines étapes

Maintenant que vous avez créé votre première note :

- En savoir plus sur [Markdown](/help/markdown)
- Organiser avec les [Tags](/help/tags)
- Configurer les [Notes quotidiennes](/help/daily-notes)
- Explorer les [Raccourcis clavier](/help/keyboard-shortcuts)

## Conseils

### Titres de notes

- Gardez les titres descriptifs
- Première ligne = titre de la note
- Utilisez H1 (`#`) pour plus de clarté

### Sauvegarde automatique

- Ne vous souciez jamais de perdre votre travail
- Les modifications sont sauvegardées automatiquement
- Vérifiez la barre d'état pour l'état de sauvegarde

### Capture rapide

Besoin de noter quelque chose rapidement ?

- Utilisez la Capture rapide (`Cmd/Ctrl+Shift+N`)
- Crée une note instantanément
- Parfait pour les pensées fugaces
