# Capture rapide

Capturez des idées instantanément sans interrompre votre flux de travail. La Capture rapide ouvre une fenêtre de prise de notes minimale depuis n'importe où sur votre système.

## Raccourci global

Appuyez sur le raccourci désigné pour ouvrir la Capture rapide :

| Plateforme | Raccourci par défaut |
| ---------- | ------------------- |
| **macOS** | `Cmd+Shift+P` |
| **Windows** | `Ctrl+Shift+P` |
| **Linux** | `Ctrl+Shift+P` |

Vous pouvez personnaliser ce raccourci dans les Paramètres.

## Utiliser la Capture rapide

1. Appuyez sur le raccourci global (n'importe où sur votre système)
2. Tapez votre note dans la fenêtre popup
3. Appuyez sur `Entrée` pour sauvegarder
4. Appuyez sur `Echap` pour annuler

## Fonctionnalités

### Interface minimale

- Fenêtre petite et focalisée
- Support Markdown
- Focus automatique sur la saisie
- Conception clavier d'abord

### Actions rapides

- `# ` + espace = titre
- `#inbox` = ajouter à la Boîte de réception
- `#today` = ajouter à la note du jour
- Tags avec `#nomdetag`

### Destination par défaut

Les notes de Capture rapide vont dans votre **Boîte de réception** par défaut. Vous pouvez changer cela :

1. Ouvrez les Paramètres
2. Allez dans Capture rapide
3. Sélectionnez la destination par défaut

## Conseils

### Flux de capture

1. **Capturer** - Notez l'idée rapidement
2. **Traiter** - Examinez la Boîte de réception quotidiennement
3. **Organiser** - Déplacez vers le dossier/note approprié

### Meilleures pratiques

- Restez bref - une idée par capture
- Utilisez des tags pour le contexte
- Ne vous souciez pas du formatage - affinez plus tard
- Examinez et traitez les captures régulièrement

### Intégration

La Capture rapide fonctionne bien avec :

- **Notes quotidiennes** : Commencez chaque jour en examinant les captures
- **Boîte de réception** : Utilisez comme zone de stockage
- **Tags** : Ajoutez du contexte immédiatement

## Dépannage

### Le raccourci ne fonctionne pas ?

**macOS**
1. Vérifiez Préférences Système > Clavier > Raccourcis
2. Recherchez les conflits
3. Redémarrez Classic

**Windows**
1. Exécutez Classic en tant qu'administrateur
2. Vérifiez les raccourcis en conflit
3. Redémarrez Classic

### La fenêtre n'apparaît pas ?

- Vérifiez si Classic est en cours d'exécution
- Vérifiez le raccourci
- Vérifiez les problèmes de focus
- Redémarrez Classic

### Les captures ne sont pas sauvegardées ?

- Vérifiez la destination par défaut
- Vérifiez les permissions d'écriture
- Vérifiez l'espace disque
- Redémarrez Classic
