# Paramètres

Personnalisez Classic pour qu'il fonctionne exactement comme vous le souhaitez. De l'apparence au comportement de l'éditeur, tout est configurable.

## Ouvrir les Paramètres

### Raccourci clavier

| Plateforme | Raccourci |
|------------|-----------|
| **macOS** | `Cmd+,` |
| **Windows/Linux** | `Ctrl+,` |

### Via le menu

- **macOS** : Classic → Paramètres
- **Windows** : Fichier → Paramètres
- **Linux** : Édition → Paramètres

### Via la Palette de commandes

1. Ouvrez la Palette de commandes (`Cmd+P` ou `Ctrl+P`)
2. Tapez "Paramètres"
3. Appuyez sur Entrée

## Catégories de paramètres

### Général

Paramètres de base de l'application :

| Paramètre | Description |
|-----------|-------------|
| **Langue** | Langue de l'interface |
| **Thème** | Clair, Sombre ou Système |
| **Démarrer à la connexion** | Lancer au démarrage |
| **Vérifier les mises à jour** | Comportement de mise à jour automatique |

### Apparence

Personnalisez l'apparence :

| Paramètre | Options |
|-----------|---------|
| **Thème** | Clair / Sombre / Système |
| **Couleur d'accentuation** | Bleu, Violet, Vert, etc. |
| **Police de l'éditeur** | Famille et taille de police |
| **Police de l'interface** | Police de l'interface |
| **Police à chasse fixe** | Police de code |
| **Largeur de la barre latérale** | 200-400px |
| **Hauteur de ligne** | 1.0-2.5 |

### Éditeur

Configurez le comportement d'écriture :

| Paramètre | Description |
|-----------|-------------|
| **Vérification orthographique** | Activer/désactiver la vérification orthographique |
| **Sauvegarde automatique** | Intervalle de sauvegarde automatique |
| **Retour à la ligne** | Activer/désactiver le retour à la ligne |
| **Numéros de ligne** | Afficher/masquer les numéros de ligne |
| **Taille de tabulation** | 2, 4 ou 8 espaces |
| **Correction automatique** | Guillemets intelligents, remplacement automatique |

### Historique des versions

Contrôlez le suivi des versions :

| Paramètre | Options | Par défaut |
|-----------|---------|------------|
| **Activer l'historique** | On/Off | On |
| **Période de rétention** | 30-180 jours | 90 jours |
| **Versions max** | 50-500 | 100 |

### Sauvegarde et restauration

Paramètres de protection des données :

| Paramètre | Description |
|-----------|-------------|
| **Sauvegarde automatique** | Activer les sauvegardes automatiques |
| **Fréquence de sauvegarde** | 15min / 30min / 1h / 4h |
| **Emplacement de sauvegarde** | Chemin personnalisé |
| **Conserver les sauvegardes** | 5 / 10 / 20 / 50 |
| **Chiffrer les sauvegardes** | Protection par mot de passe |

### Raccourcis clavier

Voir et personnaliser les raccourcis :

- Voir tous les raccourcis disponibles
- Personnaliser n'importe quel raccourci
- Réinitialiser aux valeurs par défaut

### Import et export

Paramètres de migration de données :

- Importer depuis d'autres applications

### Confidentialité

Paramètres de confidentialité et de sécurité :

| Paramètre | Description |
|-----------|-------------|

| **Rapports de plantage** | Envoyer les rapports de plantage |

### Notifications

Contrôler les notifications :

| Paramètre | Description |
|-----------|-------------|
| **Activer les notifications** | On/Off |
| **Rappel de note quotidienne** | Heure de rappel |

### Avancé

Paramètres pour utilisateurs expérimentés :

| Paramètre | Description |
|-----------|-------------|
| **CSS personnalisé** | Ajouter des styles personnalisés |

## Rechercher dans les paramètres

Vous ne trouvez pas un paramètre ? Utilisez la barre de recherche :

1. Ouvrez les Paramètres
2. Tapez dans la zone de recherche
3. Les paramètres correspondants apparaissent
4. Cliquez pour accéder à ce paramètre

## Réinitialiser les paramètres

### Réinitialiser tous les paramètres

Retour aux valeurs par défaut :

1. Ouvrez les Paramètres
2. Faites défiler vers le bas
3. Cliquez sur "Réinitialiser tous les paramètres"
4. Confirmez l'action

### Réinitialiser une catégorie

Réinitialiser une catégorie spécifique :

1. Allez dans la catégorie
2. Cliquez sur "Réinitialiser aux valeurs par défaut"
3. Seule cette catégorie est réinitialisée

## Exporter/Importer les paramètres

### Exporter les paramètres

Sauvegardez votre configuration :

1. Ouvrez les Paramètres
2. Cliquez sur "Exporter les paramètres"
3. Choisissez l'emplacement
4. Sauvegardez en tant que fichier JSON

### Importer les paramètres

Chargez une configuration sauvegardée :

1. Ouvrez les Paramètres
2. Cliquez sur "Importer les paramètres"
3. Sélectionnez le fichier JSON
4. Confirmez l'importation

## Emplacement des paramètres

Les paramètres sont stockés localement :

| Plateforme | Emplacement |
|------------|-------------|
| **macOS** | `~/Library/Application Support/Classic/` |
| **Windows** | `%APPDATA%/Classic/` |
| **Linux** | `~/.config/Classic/` |

## Synchroniser les paramètres (Optionnel)

Lorsque vous êtes connecté, les paramètres peuvent se synchroniser sur tous les appareils :

- **Synchroniser** : Activez dans Paramètres → Sync
- **Ce qui se synchronise** : Thème, polices, raccourcis
- **Ce qui ne se synchronise pas** : Chemins locaux, clés de chiffrement

## Personnalisations courantes

### Augmenter la taille de police

1. Paramètres → Apparence
2. Taille de police de l'éditeur → Augmenter
3. Les changements s'appliquent immédiatement

### Changer le thème

1. Paramètres → Apparence
2. Thème → Sélectionner Clair/Sombre/Système
3. L'application se met à jour instantanément

### Activer la sauvegarde automatique

1. Paramètres → Sauvegarde et restauration
2. Activez "Sauvegarde automatique"
3. Définissez la fréquence et l'emplacement

### Personnaliser les raccourcis

1. Paramètres → Raccourcis clavier
2. Trouvez l'action
3. Cliquez sur le raccourci actuel
4. Appuyez sur les nouvelles touches
5. Sauvegardez

## Conseils

### Accès rapide

- Utilisez `Cmd/Ctrl+,` pour ouvrir rapidement les paramètres
- Recherchez des paramètres par nom
- Utilisez les catégories pour naviguer

### Exporter avant les changements

Avant les changements majeurs, exportez vos paramètres :

- Facile à restaurer si quelque chose ne va pas
- Partagez la configuration avec d'autres

### Vérifier les nouveaux paramètres

Les mises à jour peuvent ajouter de nouveaux paramètres :

- Vérifiez après la mise à jour
- Explorez les nouvelles options

## Dépannage

### Les paramètres ne se sauvegardent pas

- Vérifiez les permissions de fichier
- Vérifiez l'espace disque
- Essayez de réinitialiser et reconfigurer

### Les paramètres se réinitialisent de manière inattendue

- Peut indiquer une corruption de fichier
- Exportez les paramètres régulièrement
- Vérifiez les conflits d'applications

### Impossible de trouver un paramètre

- Utilisez la barre de recherche
- Vérifiez différentes catégories
- Peut être dans les paramètres Avancés
