# Guide de démarrage rapide

Bienvenue dans Classic ! Ce guide vous aidera à prendre en main Classic en quelques minutes.

## Configuration requise

- **macOS** : macOS 12 (Monterey) ou version ultérieure
- **Windows** : Windows 10 ou version ultérieure
- **Linux** : Ubuntu 20.04+ / Fedora 36+ ou équivalent

## Installation

1. Téléchargez Classic depuis le [site web](/).
2. Ouvrez l'application
3. Commencez à écrire !

## Première expérience de lancement

Lorsque vous ouvrez Classic pour la première fois, vous verrez une interface épurée et minimaliste conçue pour vous permettre de vous concentrer sur votre écriture.

![Capture d'écran de l'application Classic]

### La barre latérale

Sur le côté gauche, vous pouvez trouver vos :

- **Notes quotidiennes** - Votre journal quotidien automatique
- **Recherche** - Trouvez rapidement n'importe quoi dans toutes vos notes
- **Palette de commandes** - Appuyez sur `Cmd/Ctrl+P` (Windows/Linux) ou `Cmd+Shift+P` (macOS) pour ouvrir la palette de commandes
- **Plan** - Parcourez et organisez vos notes en structure arborescente

### L'éditeur

La zone centrale est l'**éditeur Markdown** avec :
- Aperçu en direct
- Coloration syntaxique
- Outils de formatage riches

### Créer votre première note

1. Cliquez sur le bouton **Nouvelle note** dans la barre latérale, ou appuyez sur `Ctrl/Cmd+N` (Windows/Linux) ou `Cmd+N` (macOS)
2. Donnez un titre à votre note
3. Commencez à écrire en Markdown !

```markdown
# Ma première note

Bienvenue dans Classic ! C'est ici que je vais capturer mes pensées et mes idées.
```
