# Commandes slash

Les commandes slash sont un moyen puissant d'insérer rapidement du formatage et du contenu spécial sans quitter le clavier. Tapez `/` dans l'éditeur pour ouvrir la palette de commandes.

## Commandes disponibles

### Titres

| Commande | Résultat |
| -------- | -------- |
| `/h1` | Titre de niveau 1 |
| `/h2` | Titre de niveau 2 |
| `/h3` | Titre de niveau 3 |
| `/h4` | Titre de niveau 4 |
| `/h5` | Titre de niveau 5 |
| `/h6` | Titre de niveau 6 |

### Formatage de texte

| Commande | Résultat |
| -------- | -------- |
| `/bold` | **Texte en gras** |
| `/italic` | *Texte en italique* |
| `/strikethrough` | ~~Texte barré~~ |
| `/code` | Bloc de code en ligne |
| `/quote` | Citation |

### Listes

| Commande | Résultat |
| -------- | -------- |
| `/bullet` | • Liste à puces |
| `/numbered` | 1. Liste numérotée |
| `/todo` | - [ ] Élément à faire |
| `/check` | - [x] Élément terminé |

### Insérer du contenu

| Commande | Résultat |
| -------- | -------- |
| `/image` | Insérer une image depuis une URL ou un fichier |
| `/link` | Insérer un lien hypertexte |
| `/table` | Insérer un tableau |
| `/hr` | Ligne horizontale |
| `/code` | Bloc de code avec sélection de langage |

### Blocs spéciaux

| Commande | Résultat |
| -------- | -------- |
| `/mermaid` | Bloc de diagramme Mermaid |
| `/katex` | Bloc mathématique KaTeX |
| `/callout` | Bloc d'encadré/avertissement |

## Utiliser les commandes slash

1. Tapez `/` dans l'éditeur
2. Un menu apparaîtra avec toutes les commandes disponibles
3. Commencez à taper pour filtrer les commandes
4. Appuyez sur `Entrée` ou cliquez pour insérer

## Conseils avancés

- **Accès rapide** : Tapez simplement `/` et commencez à taper le nom de la commande
- **Recherche floue** : Vous n'avez pas besoin de taper le nom exact de la commande - Classic trouvera la correspondance la plus proche
- **Navigation clavier** : Utilisez les touches fléchées pour naviguer dans le menu
- **Commandes récentes** : Les commandes fréquemment utilisées apparaissent en haut de la liste
