# Barre d'état système

Classic réside dans votre barre d'état système pour un accès rapide. Capturez des pensées instantanément sans interrompre votre flux de travail.

## Qu'est-ce que la barre d'état système ?

La barre d'état système (Windows/Linux) ou la barre de menus (macOS) est l'endroit où Classic s'exécute en arrière-plan, toujours prêt pour une capture rapide.

### Avantages

- **Toujours disponible** : À un clic de distance
- **Capture rapide** : Notez des pensées instantanément
- **Exécution en arrière-plan** : Pas besoin de garder l'application ouverte
- **Notifications** : Soyez rappelé des tâches

## Activer la barre d'état système

### macOS

1. Ouvrez Classic
2. Allez dans Paramètres → Général
3. Activez "Afficher dans la barre de menus"
4. L'icône Classic apparaît dans la barre de menus

### Windows

1. Ouvrez Classic
2. Allez dans Paramètres → Général
3. Activez "Afficher dans la barre d'état système"
4. L'icône Classic apparaît dans la barre d'état système

### Linux

1. Ouvrez Classic
2. Allez dans Paramètres → Général
3. Activez "Icône de la barre d'état système"
4. Dépend de l'environnement de bureau

## Capture rapide

### Ouvrir la Capture rapide

Cliquez sur l'icône Classic dans la barre d'état système :

- **macOS** : Icône de la barre de menus
- **Windows** : Icône de la barre d'état système
- **Linux** : Icône de la barre d'état système

### Utiliser la Capture rapide

1. Cliquez sur l'icône de la barre d'état système
2. La fenêtre de capture rapide s'ouvre
3. Tapez votre note
4. Appuyez sur Entrée pour sauvegarder
5. La fenêtre se ferme automatiquement

### Raccourci clavier

Ouvrez la Capture rapide de n'importe où :

| Plateforme | Raccourci |
|------------|-----------|
| **macOS** | `Cmd+Shift+N` (global) |
| **Windows** | `Ctrl+Shift+N` (global) |
| **Linux** | `Ctrl+Shift+N` (global) |

## Menu de la barre d'état système

Faites un clic droit sur l'icône de la barre d'état pour les options :

### Options du menu

| Option | Action |
|--------|--------|
| **Nouvelle note** | Ouvrir la Capture rapide |
| **Nouvelle note quotidienne** | Ouvrir la note quotidienne du jour |
| **Recherche** | Ouvrir la fenêtre de recherche |
| **Ouvrir Classic** | Ouvrir la fenêtre principale |
| **Paramètres** | Ouvrir les paramètres |
| **Quitter** | Quitter Classic |

## Paramètres de la barre d'état

### Configurer dans les Paramètres

| Paramètre | Description |
|-----------|-------------|
| **Afficher l'icône** | Basculer la visibilité de l'icône de la barre d'état |
| **Démarrer minimisé** | Démarrer Classic uniquement dans la barre d'état |
| **Fermer vers la barre d'état** | Maintenir en cours d'exécution lors de la fermeture de la fenêtre |
| **Notifications** | Activer/désactiver les notifications |

### Démarrer à la connexion

Faites démarrer Classic automatiquement :

1. Ouvrez les Paramètres
2. Allez dans Général
3. Activez "Démarrer à la connexion"
4. Classic se lance au démarrage

## Notifications

### Activer les notifications

1. Ouvrez les Paramètres
2. Allez dans Notifications
3. Basculez "Activer les notifications"

### Types de notifications

- **Rappels** : Rappels planifiés
- **Note quotidienne** : Invites de note quotidienne
- **Mises à jour** : Mise à jour de l'application disponible

## Comportement des fenêtres

### Comportement de fermeture

Choisissez ce qui se passe lorsque vous fermez la fenêtre :

- **Réduire vers la barre d'état** : Maintenir en cours d'exécution, masquer la fenêtre
- **Quitter** : Quitter l'application
- **Demander** : Inviter à chaque fois

### Comportement de réduction

- **Réduire vers la barre d'état** : Masquer dans la barre d'état système
- **Réduire vers la barre des tâches** : Réduire normalement

## Conseils

### Capture rapide

Utilisez le raccourci global pour la capture la plus rapide :

1. Appuyez sur `Cmd/Ctrl+Shift+N` n'importe où
2. Tapez votre pensée
3. Appuyez sur Entrée
4. Continuez votre travail

### Accès rapide

Cliquez sur l'icône de la barre d'état pour voir :

- Notes récentes
- Actions rapides
- Note quotidienne du jour

### Exécution en arrière-plan

Maintenez Classic en cours d'exécution pour :

- Démarrage plus rapide
- Capture rapide
- Notifications
- Sauvegarde automatique

## Dépannage

### Icône de la barre d'état manquante

**macOS**

1. Vérifiez si l'icône est masquée
2. Regardez dans le Centre de contrôle (macOS Big Sur+)
3. Redémarrez Classic

**Windows**

1. Vérifiez les icônes masquées (flèche dans la barre des tâches)
2. Faites glisser l'icône Classic vers la zone visible
3. Redémarrez Classic

**Linux**

1. Vérifiez le support de l'environnement de bureau
2. Installez l'extension de barre d'état système si nécessaire
3. Redémarrez Classic

### Raccourci global ne fonctionne pas

- Vérifiez les conflits avec d'autres applications
- Réactivez le raccourci dans les Paramètres
- Redémarrez Classic
- Vérifiez les permissions du système d'exploitation

### Notifications ne s'affichant pas

**macOS**

1. Préférences Système → Notifications
2. Trouvez Classic
3. Activez les notifications

**Windows**

1. Paramètres → Système → Notifications
2. Activez les notifications Classic

**Linux**

1. Vérifiez le démon de notification
2. Installez libnotify si nécessaire

### Utilisation élevée de mémoire/CPU

Si l'icône de la barre d'état utilise trop de ressources :

1. Désactivez les animations dans les Paramètres
2. Réduisez la fréquence des notifications
3. Vérifiez les mises à jour de l'application
